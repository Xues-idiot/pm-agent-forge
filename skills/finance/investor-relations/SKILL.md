# 投资者关系 (Investor Relations)

## 触发条件
- 投资者沟通
- 信息披露
- 股东管理
- 市场维护

## 输入
- 公司类型
- 股东结构
- 投资者类型
- 沟通目标

## 输出
- IR策略
- 沟通计划
- 信息披露
- 危机处理

## Prompt模板
```
请制定[公司]投资者关系方案。

背景：
- 公司类型：[类型]
- 股东结构：[结构]
- 投资者类型：[类型]
- 沟通目标：[目标]

请输出：
1. IR战略定位
2. 投资者画像
3. 沟通策略
4. 信息披露
5. 活动计划
6. 危机处理
7. 股东回报
8. 市场监控
```

## 示例
### 示例1
- 输入：上市公司，需要提升机构投资者关注
- 输出：IR方案，含路演计划、分析师关系、沟通策略

## 注意事项
- 要主动透明
- 要长期维护
- 要危机预案
