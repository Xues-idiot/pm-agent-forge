# 融资管理 (Raising Capital)

## 触发条件
- 融资规划
- 投资人关系
- 融资谈判
- 估值定价

## 输入
- 融资阶段
- 目标金额
- 估值预期
- 时间要求

## 输出
- 融资策略
- 材料准备
- 投资人名单
- 谈判方案

## Prompt模板
```
请制定[公司]融资方案。

背景：
- 融资阶段：[阶段]
- 目标金额：[金额]
- 估值预期：[估值]
- 时间要求：[要求]

请输出：
1. 融资战略
2. 投资人画像
3. 材料准备
4. 投资人名单
5. 接触策略
6. 估值分析
7. 谈判要点
8. 交割安排
```

## 示例
### 示例1
- 输入：B轮融资，目标2亿，估值10亿
- 输出：融资方案，含投资人名单、路演材料、谈判策略

## 注意事项
- 要充分准备
- 要多方比较
- 要保护利益
