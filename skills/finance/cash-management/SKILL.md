# 资金管理 (Cash Management)

## 触发条件
- 资金计划
- 流动性管理
- 资金预测
- 投资配置

## 输入
- 业务特征
- 资金规模
- 风险偏好
- 市场环境

## 输出
- 资金方案
- 投资策略
- 风险控制
- 应急计划

## Prompt模板
```
请优化[公司]资金管理。

背景：
- 业务特征：[特征]
- 资金规模：[规模]
- 风险偏好：[偏好]
- 市场环境：[环境]

请输出：
1. 资金状况诊断
2. 流动性管理策略
3. 资金预测模型
4. 投资配置策略
5. 银行关系管理
6. 汇率风险控制
7. 应急流动性
8. 成本优化
```

## 示例
### 示例1
- 输入：跨国企业，多币种资金，需要集中管理
- 输出：资金方案，含现金池、外汇对冲、投资配置

## 注意事项
- 要安全优先
- 要流动性充足
- 要收益平衡
