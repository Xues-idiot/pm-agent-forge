# 商业计划 (Business Planning)

## 触发条件
- 商业计划书
- 融资计划
- 战略规划
- 预算规划

## 输入
- 业务模式
- 发展目标
- 资源需求
- 目标读者

## 输出
- 商业计划书
- 财务预测
- 里程碑
- 风险分析

## Prompt模板
```
请撰写[公司/项目]的商业计划书。

背景：
- 业务模式：[模式]
- 发展目标：[目标]
- 资源需求：[需求]
- 目标读者：[读者]

请输出：
1. 执行摘要
2. 问题与解决方案
3. 市场分析
4. 业务模式
5. 竞争优势
6. 团队介绍
7. 发展计划
8. 财务预测
9. 融资需求
10. 风险分析
```

## 示例
### 示例1
- 输入：AI创业公司天使轮融资，面向VC
- 输出：完整商业计划书，含市场分析、竞争壁垒、财务预测

## 注意事项
- 要逻辑清晰
- 要数据支撑
- 要突出亮点
