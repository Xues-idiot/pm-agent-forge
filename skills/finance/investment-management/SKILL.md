# Investment Management

投资管理，评估投资机会和管理投资组合。

## 何时使用

- 投资机会评估
- 投资决策
- 投资组合管理
- 风险评估
- 投后管理

## 投资分析

### 投资类型

```markdown
## 投资类型

### 按资产类别
| 类型 | 说明 | 风险 |
|------|------|------|
| 股权 | 股票投资 | 高 |
| 债券 | 债权投资 | 中低 |
| 房产 | 不动产 | 中 |
| PE/VC | 私募股权 | 高 |
| 现金 | 货币基金 | 低 |

### 按投资策略
| 策略 | 说明 | 适用 |
|------|------|------|
| 价值投资 | 低估值买入 | 长期 |
| 成长投资 | 高增长公司 | 中期 |
| 指数投资 | 被动跟踪 | 长期 |
| 主动管理 | 择时择股 | 专业 |
```

### 投资决策流程

```markdown
## 投资决策

### 标准流程
1. 筛选 → 2. 初审 → 3. 尽调 → 4. 投决 → 5. 执行

### 各阶段输出
- 筛选：投资机会清单
- 初审：立项报告
- 尽调：尽调报告
- 投决：投资备忘录
- 执行：投资协议
```

## 估值方法

### 估值模型

```markdown
## 估值方法

### 相对估值
| 方法 | 适用 | 指标 |
|------|------|------|
| P/E | 盈利稳定 | 市盈率 |
| P/B | 金融 | 市净率 |
| P/S | 收入稳定 | 市销率 |
| EV/EBITDA | 重资产 | 企业价值 |

### 绝对估值
| 方法 | 适用 | 要点 |
|------|------|------|
| DCF | 现金流稳定 | 折现率 |
| DDM | 分红稳定 | 分红率 |
| NAV | 资产密集 | 净资产 |

### 估值示例
```markdown
## DCF估值

### 参数
- 预测期：5年
- 永续增长率：3%
- 折现率：10%

### 公式
DCF = Σ CFt/(1+r)^t + TV/(1+r)^n
```
```

```markdown
## 尽职调查

### 业务尽调
| 事项 | 内容 |
|------|------|
| 商业模式 | 如何赚钱 |

| 事项 | 内容 |
|------|------|
| 财务报表 | 真实可靠 |
| 盈利质量 | 现金流 |
| 税务 | 合规性 |
| 负债 | 或有负债 |

### 法律尽调
| 事项 | 内容 |
|------|------|
| 股权结构 | 清晰合法 |
| 合同 | 重大合同 |
| 诉讼 | 潜在风险 |
| 知识产权 | 归属完整 |
```

### 尽调报告

```markdown
## 尽调报告

4. 财务分析

6. 风险分析

- 估值不严谨

- 条款不保护

- 退出不积极
