# 资金中心优化 (Treasury Optimization)

## 触发条件
- 现金集中
- 内部借贷
- 资金效率
- 汇率风险

## 输入
- 集团结构
- 资金分布
- 银行关系
- 风险偏好

## 输出
- 优化方案
- 集中策略
- 内部定价
- 系统支撑

## Prompt模板
```
请优化[集团]资金管理。

背景：
- 集团结构：[结构]
- 资金分布：[分布]
- 银行关系：[关系]
- 风险偏好：[偏好]

请输出：
1. 现金集中策略
2. 内部借贷机制
3. 转移定价
4. 银行关系管理
5. 资金预测
6. 投资策略
7. 汇率风险
8. 系统平台
```

## 示例
### 示例1
- 输入：跨国集团，10个子公司，货币多元化
- 输出：优化方案，含Cash Pooling、外汇对冲、内部银行

## 注意事项
- 要合规第一
- 要效率优先
- 要风险可控
