# Finance Compliance Check Skill

检查金融合规要求，确保业务符合监管规定。

## 何时使用

- KYC（了解客户）检查
- AML（反洗钱）筛查
- 合规审查
- 监管报告
- 内部控制检查
- 合规培训

## 合规检查流程

```
1. 明确监管要求 → 2. 收集证据 → 3. 执行检查 → 4. 问题识别 → 5. 风险评估 → 6. 整改跟踪
```

### 1. 明确监管要求

**常见金融合规领域：**

```markdown
## 合规领域

### KYC/AML
| 要求 | 说明 | 法规依据 |
|------|------|---------|
| 客户识别 | 验证客户身份 | 反洗钱法 |
| 可疑交易 | 识别异常交易 | 大额交易报告 |
| 风险分级 | 客户风险评估 | 风险分级指引 |
| 名单筛查 |制裁/黑名单 | 联合国/本国制裁 |

### 消费者保护
| 要求 | 说明 | 法规依据 |
|------|------|---------|
| 产品适当性 | 产品与客户匹配 | 适当性指引 |
| 信息披露 | 产品信息透明 | 信息披露规范 |
| 营销合规 | 宣传材料合规 | 广告法/理财新规 |
| 投诉处理 | 投诉处理机制 | 投诉处理指引 |

### 数据合规
| 要求 | 说明 | 法规依据 |
|------|------|---------|
| 数据保护 | 客户信息保护 | 数据安全法 |
| 跨境传输 | 数据出境合规 | 跨境传输规则 |
| 留存期限 | 交易记录保存 | 反洗钱留存要求 |

### 运营合规
| 要求 | 说明 | 法规依据 |
|------|------|---------|
| 资本充足 | 最低资本要求 | 资本管理办法 |
| 流动性 | 流动性比例 | 流动性指引 |
| 风控指标 | 风险指标合规 | 风控指引 |
```

### 3. 执行检查

**合规检查方法：**

```markdown
## 合规检查方法

### 文件检查
- 审阅制度文件
- 检查流程记录
- 核对签名签章

### 系统检查
- 系统权限设置
- 操作日志审查
- 数据完整性验证

### 访谈检查
- 员工访谈
- 客户回访
- 第三方确认

### 抽样检查
- 交易抽样
- 账户抽样
- 影像资料抽查
```

**检查抽样标准：**

| 检查类型 | 抽样比例 | 最低数量 |
|---------|---------|---------|
| 高风险客户 | 100% | - |
| 中风险客户 | 30% | 50 |
| 低风险客户 | 10% | 100 |
| 大额交易 | 100% | - |
| 可疑交易 | 100% | - |

### 5. 风险评估

**合规风险评估：**

```markdown
## 合规风险评估

### 风险矩阵
| 可能性 → | 低 | 中 | 高 |
|---------|---|---|---|
| **影响 ↓** | | | |
| 严重 | 低 | 中 | 高 |
| 一般 | 低 | 低 | 中 |
| 轻微 | 低 | 低 | 低 |

### 风险评级
| 风险等级 | 定义 | 处置方式 |
|---------|------|---------|
| 高风险 | 高可能性+严重/一般影响 | 立即整改+升级报告 |
| 中风险 | 中可能性+严重影响 | 限时整改+持续监测 |
| 低风险 | 低可能性+轻微影响 | 改进流程 |

### 监管处罚风险
| 违规类型 | 处罚类型 | 参考案例 |
|---------|---------|---------|
| KYC缺失 | 罚款+整改 | 某银行X百万 |
| 洗钱疏漏 | 罚款+暂停业务 | 某机构X千万 |
| 信息披露违规 | 警告+罚款 | 某公司X百万 |
```

**消费者保护检查清单：**
- [ ] 产品适当性评估已执行
- [ ] 信息披露符合要求
- [ ] 营销宣传合规
- [ ] 投诉处理机制有效
- [ ] 客户资产隔离

## 红牌警告

- 客户身份验证缺失
- 制裁名单未筛查
- 可疑交易未上报
- 记录保存期限不足
- 内控缺陷未整改
- 监管报告延误
- 重大违规瞒报
- 监管处罚未整改
