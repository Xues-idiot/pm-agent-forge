# Finance Investment Analysis Skill

分析投资机会，评估资产价值，制定投资策略。

## 何时使用

- 评估股票/债券投资
- 分析基金/理财产品
- 评估项目投资
- 构建投资组合
- 撰写投资报告
- 做出投资建议

## 投资分析流程

```
1. 明确投资目标 → 2. 收集信息 → 3. 财务分析 → 4. 估值分析 → 5. 风险评估 → 6. 投资建议
```

### 4. 估值分析

**估值方法选择：**

| 估值方法 | 适用场景 | 优缺点 |
|---------|---------|--------|
| DCF | 内在价值评估 | 精确但依赖假设 |
| PE/PB | 相对估值 | 简单但易受市场影响 |
| PS | 成长型公司 | 适合早期 |
| EV/EBITDA | 成熟期公司 | 排除折旧差异 |
| NAV | 资产业务 | 适合金融/地产 |

**DCF 估值模板：**

```python
def dcf_valuation(fcf_forecast, wacc, terminal_growth, shares):
    """
    DCF 估值模型

    参数:
        fcf_forecast: 未来5年FCF预测 [fcf1, fcf2, ..., fcf5]
        wacc: 加权平均资本成本
        terminal_growth: 永续增长率
        shares: 流通股数
    """
    # 预测期现金流现值
    pv = 0
    for i, fcf in enumerate(fcf_forecast, 1):
        pv += fcf / (1 + wacc) ** i

    # 永续价值
    terminal_value = fcf_forecast[-1] * (1 + terminal_growth) / (wacc - terminal_growth)
    pv_terminal = terminal_value / (1 + wacc) ** 5

    # 企业价值
    ev = pv + pv_terminal

    # 股权价值
    equity_value = ev - net_debt

    # 每股价值
    per_share_value = equity_value / shares

    return per_share_value
```
