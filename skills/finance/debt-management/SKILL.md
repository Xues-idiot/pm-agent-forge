# 债务管理 (Debt Management)

## 触发条件
- 债务结构优化
- 融资策略
- 偿债计划
- 成本控制

## 输入
- 债务现状
- 融资能力
- 利率环境
- 评级情况

## 输出
- 债务策略
- 融资方案
- 偿债计划
- 风险控制

## Prompt模板
```
请优化[公司]债务管理。

背景：
- 债务现状：[现状]
- 融资能力：[能力]
- 利率环境：[环境]
- 评级情况：[评级]

请输出：
1. 债务状况诊断
2. 债务结构优化
3. 融资渠道策略
4. 利率风险管理
5. 偿债计划
6. 评级管理
7. 流动性保障
8. 成本控制
```

## 示例
### 示例1
- 输入：地产企业，高负债率，需要优化债务结构
- 输出：债务方案，含债务置换、期限优化、评级提升

## 注意事项
- 要控制风险
- 要降低成本
- 要持续监控
