# Treasury Management Skill

管理资金运营，优化现金流，处理企业资金职能。

## 何时使用

- 资金计划
- 现金流管理
- 资金调度
- 银行关系
- 资金风险管理
- 资金效率优化

## 资金管理流程

```
1. 资金规划 → 2. 日常运营 → 3. 现金流管理 → 4. 资金调度 → 5. 风险管理
```

### 2. 日常运营

**收付款管理：**

```markdown
## 收付款管理

### 收款管理
| 类型 | 方式 | 到账时间 | 手续费 |
|------|------|----------|--------|
| 银行转账 | 对公转账 | 1-2天 | 低 |
| 票据收款 | 承兑汇票 | 3-6月 | 无 |
| 第三方支付 | 支付宝等 | 即时/次日 | 中 |
| 保理 | 应收账款保理 | 1-3天 | 中 |

### 付款管理
```markdown
## 付款优先级

### 优先级排序
1. 员工工资（必须按时）
2. 税金社保（滞纳金高）
3. 银行到期贷款（影响信用）
4. 核心供应商（关系维护）
5. 一般供应商（按合同）
6. 其他（视情况）
```

### 审批流程
| 付款金额 | 审批人 | 审批时间 |
|----------|--------|----------|
| <5万 | 部门负责人 | 即时 |
| 5-50万 | 总监 | 1天 |
| 50-200万 | VP | 2天 |
| >200万 | CFO+CEO | 3天 |
```

### 5. 风险管理

**资金风险识别：**

```markdown
## 资金风险

### 风险类型
| 风险 | 描述 | 影响 | 可能性 |
|------|------|------|--------|
| 流动性风险 | 资金不足无法付款 | 高 | 中 |
| 信用风险 | 对方违约 | 中 | 中 |
| 市场风险 | 汇率/利率波动 | 中 | 中 |
| 操作风险 | 付款错误/欺诈 | 高 | 低 |

### 流动性风险
```markdown
## 流动性管理

### 流动性指标
| 指标 | 计算 | 警戒值 | 安全值 |
|------|------|--------|--------|
| 流动比率 | 流动资产/流动负债 | >1.2 | >1.5 |
| 速动比率 | 速动资产/流动负债 | >0.8 | >1.0 |
| 现金比率 | 现金/流动负债 | >0.2 | >0.3 |

### 应急措施
- [ ] 银行授信启用
- [ ] 加快应收账款
- [ ] 延后非必要支出
- [ ] 股权融资
```
```

**资金安全：**

```markdown
## 资金安全

### 内控措施
| 控制点 | 措施 |
|--------|------|
| 权限分离 | 管钱的不管账 |
| 双人审批 | 大额双人确认 |
| 银行对账 | 每日核对 |
| 不相容职务 | 申请/审批/执行分离 |

### 防欺诈
- [ ] 收款方一致性校验
- [ ] 大额付款复核
- [ ] 异常交易告警
- [ ] U盾/密码分开保管
```

## 资金管理检查清单

- [ ] 账户体系清晰
- [ ] 资金计划完备
- [ ] 收付款规范
- [ ] 现金流可控
- [ ] 调度合理

- [ ] 风险识别到位
- [ ] 内控完善

## 红牌警告

- 流动性危机
- 银行账户透支
- 大额付款错误
- 资金被挪用
- 银行关系断裂
- 融资渠道枯竭
- 汇率损失
- 内部贪腐
