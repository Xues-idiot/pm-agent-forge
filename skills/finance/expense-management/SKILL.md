# 费用管理 (Expense Management)

## 触发条件
- 费用控制
- 预算管理
- 报销流程
- 成本优化

## 输入
- 费用类型
- 费用规模
- 管理目标
- 员工体验

## 输出
- 管理方案
- 预算标准
- 报销流程
- 控制机制

## Prompt模板
```
请优化[组织]费用管理。

背景：
- 费用类型：[类型]
- 费用规模：[规模]
- 管理目标：[目标]
- 员工体验：[体验]

请输出：
1. 费用诊断
2. 预算标准
3. 审批流程
4. 报销规范
5. 控制机制
6. 违规处理
7. 系统支撑
8. 持续优化
```

## 示例
### 示例1
- 输入：企业差旅费用高，需要控制
- 输出：管理方案，含差旅标准、在线审批、集中采购

## 注意事项
- 要标准明确
- 要流程简化
- 要合规透明
