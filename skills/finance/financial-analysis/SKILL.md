# Financial Analysis

财务分析，分析财务报表和财务表现。

## 何时使用

- 财务报表分析
- 财务指标分析
- 现金流分析
- 财务预测
- 投资分析

## 财务基础

### 财务报表

```markdown
## 财务报表

### 三大报表
| 报表 | 内容 | 用途 |
|------|------|------|
| 资产负债表 | 资产/负债/权益 | 财务状况 |
| 利润表 | 收入/成本/利润 | 盈利能力 |
| 现金流量表 | 现金流入/流出 | 流动性 |

### 报表关系
```markdown
## 报表关系

资产负债表是"照片"：
- 某一时点的财务状况

利润表是"录像"：
- 某一时期的经营成果

现金流量表是"流水账"：
- 某一时期的现金变动
```
```

### DCF估值

```markdown
## DCF估值

### 步骤
1. 预测未来现金流（5-10年）
2. 估计永续增长率
3. 计算折现率（WACC）
4. 计算预测期现金流现值
5. 计算永续价值现值
6. 汇总为企业价值

### 公式
```
企业价值 = Σ FCFt/(1+WACC)t + TV/(1+WACC)n

其中：
- FCFt: 第t年自由现金流
- TV: 永续价值 = FCFn×(1+g)/(WACC-g)
- WACC: 加权平均资本成本
```
```
