# 并购顾问 (M&A Advisory)

## 触发条件
- 并购标的筛选
- 尽职调查
- 交易结构
- 整合规划

## 输入
- 战略目标
- 标的类型
- 预算范围
- 时间要求

## 输出
- 标的分析
- 尽职调查报告
- 交易方案
- 整合计划

## Prompt模板
```
请为[收购方]提供并购顾问服务。

背景：
- 战略目标：[目标]
- 标的类型：[类型]
- 预算范围：[预算]
- 时间要求：[要求]

请输出：
1. 并购战略建议
2. 标的筛选标准
3. 初筛分析
4. 尽职调查清单
5. 估值分析
6. 交易结构设计
7. 风险识别
8. 谈判策略
9. 整合规划
10. 时间表
```

## 示例
### 示例1
- 输入：上市公司，想收购一家AI创业公司，估值5亿以内
- 输出：顾问方案，含标的筛选、尽调清单、交易结构、整合计划

## 注意事项
- 要战略导向
- 要充分尽职调查
- 要重视整合
