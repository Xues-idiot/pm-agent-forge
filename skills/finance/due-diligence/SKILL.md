# 尽职调查 (Due Diligence)

## Overview

尽职调查是在投资、并购、合作前对目标进行全方位调查评估的技能。涵盖业务尽职调查、财务尽职调查、法律尽职调查等核心环节。

## 何时使用

- 股权投资前
- 并购交易前
- 重大合作前
- 供应商准入评估

## 流程

1. **调查准备**：制定调查计划
2. **业务DD**：调查商业模式和竞争力
3. **财务DD**：核查财务数据和真实性
4. **法律DD**：核查法律风险和合规性
5. **技术DD**：评估技术能力和知识产权
6. **团队DD**：评估核心团队背景
7. **报告整合**：形成尽调报告

## 检查清单

- [ ] 调查范围覆盖全面
- [ ] 核查方法科学有效
- [ ] 重大风险已识别
- [ ] 数据来源可靠
- [ ] 报告结论有依据
- [ ] 保密措施到位

## 红牌警告

- 调查范围过窄遗漏重大风险
- 数据来源不可靠
- 重大风险未识别
- 报告结论与数据矛盾
- 信息泄露造成损失
