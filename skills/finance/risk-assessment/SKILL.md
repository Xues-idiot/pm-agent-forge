# Finance Risk Assessment Skill

评估各类金融风险，为决策提供风险依据。

## 何时使用

- 评估贷款/信用风险
- 评估投资风险
- 评估市场风险
- 评估操作风险
- 制定风控策略
- 风险报告撰写

## 风险评估流程

```
1. 风险识别 → 2. 风险分析 → 3. 风险量化 → 4. 风险评估 → 5. 风控建议 → 6. 监控机制
```

### 1. 风险识别

**风险分类体系：**

| 风险类型 | 描述 | 主要来源 |
|---------|------|---------|
| 信用风险 | 违约风险 | 借款人/交易对手 |
| 市场风险 | 价格波动风险 | 利率/汇率/股价 |
| 操作风险 | 流程/系统风险 | 内部流程/人员/系统 |
| 流动性风险 | 变现能力风险 | 资产流动性 |
| 合规风险 | 法规违反风险 | 监管要求 |

### 3. 风险量化

**量化指标：**

```markdown
## 风险量化指标

### 信用风险
| 指标 | 计算公式 | 阈值 |
|------|---------|------|
| 违约概率(PD) | 历史违约率/模型 | <5% |
| 违约损失率(LGD) | 1-回收率 | <50% |
| 违约敞口(EAD) | 风险敞口 | - |
| 预期损失(EL) | PD×LGD×EAD | - |

### 市场风险
| 指标 | 计算公式 | 阈值 |
|------|---------|------|
| VaR | 分位数法/历史模拟 | <2% |
| ES | 条件尾部期望 | <3% |
| 波动率 | 历史/隐含 | <20% |
| Beta | 相对基准 | <1.5 |

### 操作风险
| 指标 | 计算公式 | 阈值 |
|------|---------|------|
| 损失频率 | 事件数/时间 | - |
| 损失幅度 | 平均损失金额 | - |
| 操作风险资本 | LDA模型 | - |
```

**VaR 计算示例：**

```python
import numpy as np

def calculate_var(returns, confidence=0.95):
    """计算 Value at Risk"""
    return np.percentile(returns, (1 - confidence) * 100)

def calculate_es(returns, confidence=0.95):
    """计算 Expected Shortfall"""
    var = calculate_var(returns, confidence)
    return returns[returns <= var].mean()

# 使用
returns = np.array([...])  # 历史收益
var_95 = calculate_var(returns, 0.95)
es_95 = calculate_es(returns, 0.95)
```

### 4. 风险评估

**风险矩阵：**

```markdown
## 风险评估矩阵

### 风险等级
| 等级 | 风险程度 | PD范围 | VaR范围 | 处置 |
|------|---------|--------|--------|------|
| R1 | 很低 | <1% | <1% | 接受 |
| R2 | 低 | 1-3% | 1-2% | 监测 |
| R3 | 中 | 3-8% | 2-5% | 控制 |
| R4 | 高 | 8-15% | 5-10% | 缓释 |
| R5 | 很高 | >15% | >10% | 规避 |

```markdown

|---------|------|---------|

| 市场风险 | R3 | 波动率偏高 |
| 操作风险 | R2 | 流程较完善 |
```

### 5. 风控建议

**风控措施：**

| 风险类型 | 控制措施 | 缓释工具 |
|---------|---------|---------|
| 信用风险 | 准入标准、额度控制 | 抵押、担保、保险 |
| 市场风险 | 止损线、仓位限制 | 对冲、期权 |
| 操作风险 | 流程控制、权限管理 | 保险、业务连续性计划 |
| 流动性风险 | 流动性储备、资产分级 | 央行借贷便利 |

- 风险识别有明显遗漏
- 量化模型假设不合理
- 历史数据样本偏差
- 压力测试场景不充分
- 风险缓释措施不可行
- 风险报告数据不一致
- 监管合规要求未满足
