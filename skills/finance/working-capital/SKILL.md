# 营运资金 (Working Capital Management)

## 触发条件
- 营运资金优化
- 现金周期
- 应收账款
- 应付账款

## 输入
- 业务模式
- 资金现状
- 行业特征
- 优化目标

## 输出
- 诊断报告
- 优化方案
- 指标目标
- 实施路径

## Prompt模板
```
请优化[公司]营运资金管理。

背景：
- 业务模式：[模式]
- 资金现状：[现状]
- 行业特征：[特征]
- 优化目标：[目标]

请输出：
1. 现金周期诊断
2. 应收账款管理
3. 应付账款管理
4. 存货优化
5. 现金预测
6. 优化目标
7. 实施方案
8. 效果评估
```

## 示例
### 示例1
- 输入：工程企业，应收账款周转90天
- 输出：优化方案，含信用管理、催收策略、保理

## 注意事项
- 要全链路优化
- 要平衡各方
- 要持续监控
