# 成本优化 (Cost Optimization)

## 触发条件
- 成本分析
- 成本降低
- 效率提升
- 预算控制

## 输入
- 成本结构
- 优化目标
- 业务约束
- 组织情况

## 输出
- 成本分析
- 优化方案
- 实施路径
- 评估机制

## Prompt模板
```
请为[公司/部门]制定成本优化方案。

背景：
- 成本结构：[结构]
- 优化目标：[目标]
- 业务约束：[约束]
- 组织情况：[情况]

请输出：
1. 成本结构分析
2. 成本分类（固定/可变/可控）
3. 成本优化机会识别
4. 各成本项优化措施
5. 供应商优化
6. 流程优化
7. 技术降本
8. 组织效率提升
9. 实施计划和责任分工
10. 效果评估
```

## 示例
### 示例1
- 输入：制造企业，材料成本占60%，需要降低10%成本
- 输出：优化方案，含供应商整合、工艺优化、设计降本

## 注意事项
- 要系统性分析
- 要避免牺牲质量
- 要持续跟踪
