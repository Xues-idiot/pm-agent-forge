# 审计管理 (Audit Management)

## 触发条件
- 审计计划
- 风险审计
- 合规审计
- 内控评估

## 输入
- 组织规模
- 业务复杂度
- 监管要求
- 审计资源

## 输出
- 审计计划
- 审计方案
- 整改跟踪
- 评估报告

## Prompt模板
```
请制定[组织]审计管理方案。

背景：
- 组织规模：[规模]
- 业务复杂度：[复杂度]
- 监管要求：[要求]
- 审计资源：[资源]

请输出：
1. 审计战略
2. 风险评估
3. 审计计划
4. 审计方案
5. 审计执行
6. 发现整改
7. 报告机制
8. 质量控制
```

## 示例
### 示例1
- 输入：上市公司，面临严格的监管审计
- 输出：审计方案，含年度计划、重点领域、整改跟踪

## 注意事项
- 要独立客观
- 要风险导向
- 要持续跟进
