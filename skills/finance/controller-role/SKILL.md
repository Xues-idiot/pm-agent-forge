# 财务控制 (Controller Role)

## 触发条件
- 财务控制
- 流程管控
- 合规监督
- 风险管理

## 输入
- 组织类型
- 财务流程
- 控制目标
- 审计要求

## 输出
- 控制框架
- 流程控制
- 监督机制
- 改进建议

## Prompt模板
```
请设计[组织]财务控制方案。

背景：
- 组织类型：[类型]
- 财务流程：[流程]
- 控制目标：[目标]
- 审计要求：[要求]

请输出：
1. 控制环境
2. 控制框架
3. 采购到付款
4. 销售到收款
5. 费用管理
6. 财务报告
7. 监督机制
8. 持续改进
```

## 示例
### 示例1
- 输入：集团企业，需要强化财务管控
- 输出：控制方案，含审批流程、权限分离、审计跟踪

## 注意事项
- 要控制风险
- 要流程优化
- 要合规优先
