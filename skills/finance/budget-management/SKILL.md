# Budget Management Skill

管理预算，规划财务资源，控制部门支出。

## 何时使用

- 年度预算编制
- 季度预算调整
- 部门预算控制
- 预算执行分析
- 超支预警

## 预算管理流程

```
1. 目标确认 → 2. 预算编制 → 3. 执行监控 → 4. 差异分析 → 5. 滚动调整
```

### 预算编制方法

```markdown
## 预算方法

### 增量预算
- 基于上年实际 + 合理增长
- 适用：稳定业务

### 零基预算
- 从零开始重新评估每项支出
- 适用：新业务/重大调整

### 作业预算
- 基于业务量 × 作业成本率
- 适用：制造业/服务业

### 滚动预算
- 季度更新，保持12个月视野
- 适用：变化快的业务
```

### 差异分解
```markdown
## 差异分解

### 收入差异
- 销量差异 = (实际销量-预算销量) × 预算单价
- 价格差异 = (实际单价-预算单价) × 实际销量
- 结构差异 = 组合变化带来的差异

### 成本差异
- 效率差异 = (实际用量-标准用量) × 标准价格
- 价格差异 = (实际价格-标准价格) × 实际用量
```
```

## 红牌警告

- 预算脱离实际
- 执行率严重滞后
- 超预算无审批
- 差异原因不清
- 预警机制失效
- 频繁追加预算
- 资金闲置浪费
- 预算造假
