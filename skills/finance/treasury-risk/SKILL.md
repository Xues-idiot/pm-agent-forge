# 资金风险 (Treasury Risk)

## 触发条件
- 资金风险
- 汇率风险
- 利率风险
- 流动性风险

## 输入
- 风险类型
- 敞口规模
- 风险偏好
- 对冲工具

## 输出
- 风险评估
- 对冲策略
- 监控机制
- 应急预案

## Prompt模板
```
请管理[资金]风险。

背景：
- 风险类型：[类型]
- 敞口规模：[敞口]
- 风险偏好：[偏好]
- 对冲工具：[工具]

请输出：
1. 风险识别
2. 风险评估
3. 对冲策略
4. 工具选择
5. 监控机制
6. 应急预案
```

## 示例
### 示例1
- 输入：跨国企业，有大额美元敞口
- 输出：风险方案，含远期锁汇、外汇期权、对冲比例

## 注意事项
- 要风险可控
- 要成本效益
- 要持续监控
