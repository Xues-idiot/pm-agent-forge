# Finance Financial Report Analysis Skill

分析财务报表，评估企业财务健康状况。

## 何时使用

- 分析公司财务状况
- 评估投资标的
- 信用评估
- 并购尽职调查
- 定期财务审查
- 财务尽调

## 财务分析流程

```
1. 收集报表 → 2. 报表结构分析 → 3. 比率分析 → 4. 趋势分析 → 5. 综合评估 → 6. 分析报告
```

### 3. 比率分析

**核心财务比率：**

```markdown
## 财务比率分析

### 盈利能力
| 比率 | 计算公式 | 行业均值 | 公司值 | 评价 |
|------|---------|---------|-------|------|
| 毛利率 | (营收-成本)/营收 | 30% | | |
| 净利率 | 净利润/营收 | 10% | | |
| ROE | 净利润/净资产 | 15% | | |
| ROA | 净利润/总资产 | 8% | | |

### 偿债能力
| 比率 | 计算公式 | 安全值 | 公司值 | 评价 |
|------|---------|-------|-------|------|
| 资产负债率 | 负债/资产 | <60% | | |
| 流动比率 | 流动/流动负债 | >1.5 | | |
| 速动比率 | (流动-存货)/流动负债 | >1.0 | | |
| 利息保障倍数 | EBIT/利息 | >3 | | |

### 运营能力
| 比率 | 计算公式 | 行业均值 | 公司值 | 评价 |
|------|---------|---------|-------|------|

| 比率 | 计算公式 | 近3年均值 |
```

## 财务分析检查清单

- [ ] 报表数据完整
- [ ] 数字勾稽关系正确
- [ ] 比率计算准确
- [ ] 趋势分析客观
- [ ] 与行业对比合理
- [ ] 风险提示充分
- [ ] 结论有数据支撑

## 红牌警告

- 报表数据来源不明
- 数字勾稽关系不符
- 异常波动未解释
- 行业对比不恰当
- 关键风险未识别
- 结论与数据矛盾
- 审计意见被忽略
