# 财务报告 (Financial Reporting)

## 触发条件
- 财务报告优化
- 报告质量提升
- 报告流程
- 信息披露

## 输入
- 报告类型
- 用户需求
- 现有流程
- 合规要求

## 输出
- 报告体系
- 流程优化
- 质量控制
- 披露管理

## Prompt模板
```
请优化[公司]的财务报告。

背景：
- 报告类型：[类型]
- 用户需求：[需求]
- 现有流程：[流程]
- 合规要求：[要求]

请输出：
1. 财务报告体系设计
2. 报告使用者分析
3. 报告内容优化
4. 报告流程优化
5. 数据质量保障
6. 自动化方案
7. 合规控制
8. 管理层报告设计
9. 披露管理
```

## 示例
### 示例1
- 输入：上市公司，需要提升季报质量，缩短报告周期
- 输出：优化方案，含报告模板、流程优化、系统自动化

## 注意事项
- 要满足用户需求
- 要保证准确性
- 要及时性
