# HR Employee Training Skill

规划员工培训，识别培训需求，设计学习路径。

## 何时使用

- 新员工培训规划
- 员工能力提升计划
- 团队培训需求分析
- 培训效果评估
- 培训体系搭建

## 培训规划流程

```
1. 需求分析 → 2. 制定计划 → 3. 资源准备 → 4. 培训执行 → 5. 效果评估
```

**培训形式选择：**

| 形式 | 适用场景 | 优缺点 |
|------|---------|--------|
| 线下集中 | 技能实操、团建 | 效果好但成本高 |
| 线上课程 | 知识学习、通用技能 | 灵活但互动少 |
| 在岗实践 | 业务技能、经验积累 | 实用但难系统化 |
| 导师制 | 领导力、职业发展 | 个性化但资源有限 |
| 外部培训 | 专业认证、高端培训 | 质量高但成本高 |

### 5. 效果评估

**培训效果评估模型（柯氏四级）：**

| 层级 | 评估内容 | 方法 | 时机 |
|------|---------|------|------|
| L1 反应层 | 学员满意度 | 问卷 | 培训结束时 |
| L2 学习层 | 知识/技能掌握 | 考试/实操 | 培训结束时 |
| L3 行为层 | 行为改变 | 观察/上级反馈 | 30-60天后 |
| L4 结果层 | 业务影响 | 绩效数据 | 6-12月后 |

## 培训体系建设

**培训体系框架：**

```markdown
## 培训体系

### 新员工培训
| 阶段 | 内容 | 时长 | 形式 |
|------|------|------|------|
| 入职第1周 | 公司文化、制度 | 2天 | 集中培训 |
| 入职1个月 | 岗位基础技能 | - | 在岗学习 |
| 入职3个月 | 独立工作能力 | - | 在岗实践 |
| 转正评估 | 综合评估 | - | 考核 |

### 专业技能培训
| 层级 | 培训内容 | 周期 |
|------|---------|------|
| 初级 | 基础技能 | 入职1年内 |
| 中级 | 进阶技能 | 1-3年 |
| 高级 | 专家技能 | 3年+ |

### 管理培训
| 层级 | 培训内容 | 对象 |
|------|---------|------|
| 储备干部 | 管理基础 | 高潜员工 |
| 新任管理者 | 管理技能 | 新晋管理者 |
| 中层管理者 | 战略管理 | 中层管理者 |
| 高层管理者 | 领导力 | 高层管理者 |
```

## 红牌警告

- 培训需求未经调研
- 培训计划流于形式
- 培训内容与需求脱节
- 培训效果未跟踪
- 培训成本过高
- 员工参与度低
- 培训后无实际应用
