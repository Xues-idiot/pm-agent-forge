# HR Performance Evaluation Skill

评估员工绩效，制定目标，反馈表现。

## 何时使用

- 季度/年度绩效评估
- 试用期评估
- 项目结束评估
- 晋升评估
- 绩效改进计划

## 绩效评估流程

```
1. 明确评估目的 → 2. 收集评估信息 → 3. 绩效评级 → 4. 反馈准备 → 5. 绩效面谈 → 6. 发展规划
```

### 1. 明确评估目的

**评估类型：**

| 类型 | 周期 | 权重 | 用途 |
|------|------|------|------|
| 年度评估 | 1年 | 100% | 年终奖/晋升 |
| 半年度评估 | 6个月 | 50% | 调薪/晋升 |
| 季度评估 | 3个月 | 30% | 持续反馈 |
| 试用期评估 | 3个月 | - | 转正决策 |

### 3. 绩效评级

**绩效等级定义：**

| 等级 | 定义 | 比例控制 | 描述 |
|------|------|---------|------|
| S | 卓越 | ≤10% | 超出期望，大幅超越 |
| A | 优秀 | ≤20% | 超出期望 |
| B+ | 良好 | - | 超出部分期望 |
| B | 合格 | 目标区间 | 达到期望 |
| C | 待改进 | - | 部分达到 |
| D | 不合格 | - | 未达到 |

### 5. 绩效面谈

**面谈流程：**

```markdown
## 绩效面谈

### 面谈前准备
- [ ] 提前通知员工
- [ ] 收集完整绩效数据
- [ ] 准备面谈材料
- [ ] 选择安静的面谈环境

### 面谈结构
| 时间 | 内容 |
|------|------|
| 5min | 开场寒暄 |
| 10min | 员工自我评价 |
| 15min | 绩效反馈 |
| 10min | 讨论改进计划 |
| 10min | 设定下期目标 |
| 5min | 总结收尾 |

### 面谈技巧
- 先听后说
- 具体事例支撑
- 双向沟通
- 聚焦行为而非人格
```

## 绩效评估检查清单

- [ ] 评估目的明确
- [ ] 评估标准一致
- [ ] 数据收集完整
- [ ] 评估结果有依据
- [ ] 反馈具体有事例
- [ ] 改进计划可执行
- [ ] 员工签字确认

## 红牌警告

- 评估标准不一致
- 评估结果无事例支撑
- 只谈缺点不谈优点
- 绩效结果未提前通知
- 晋升决策缺乏数据
- 绩效改进计划不具体
- 绩效面谈走过场
