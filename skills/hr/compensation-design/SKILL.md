# Compensation Design Skill

设计薪酬体系，规划薪资结构，管理员工激励方案。

## 何时使用

- 薪酬体系设计
- 薪资结构调整
- 奖金方案制定
- 股权激励设计
- 福利方案优化
- 薪酬调研分析

## 薪酬管理流程

```
1. 薪酬调研 → 2. 体系设计 → 3. 方案制定 → 4. 实施沟通 → 5. 效果评估
```

## 薪酬管理检查清单

- [ ] 市场调研充分
- [ ] 职级体系清晰
- [ ] 薪酬结构合理
- [ ] 奖金方案公平
- [ ] 长期激励到位
- [ ] 沟通策略有效
- [ ] 成本可控
- [ ] 合规合法

## 红牌警告

- 薪酬严重低于市场
- 内部严重倒挂
- 薪酬不透明导致不满
- 绩效奖金平均主义
- 长期激励不兑现
- 福利成本失控
- 薪酬歧视
- 偷税漏税
