# 人员分析 (People Analytics)

## 触发条件
- 人力数据分析
- 组织诊断
- 效能分析
- 预测模型

## 输入
- 数据范围
- 分析目标
- 组织阶段
- 可视化需求

## 输出
- 分析报告
- 预测模型
- 决策建议
- 看板设计

## Prompt模板
```
请对[组织]进行人员分析。

背景：
- 数据范围：[范围]
- 分析目标：[目标]
- 组织阶段：[阶段]
- 可视化需求：[需求]

请输出：
1. 数据概览
2. 组织结构分析
3. 人员效率分析
4. 流失分析
5. 关键驱动因素
6. 预测模型
7. 建议和行动
8. 可视化看板
```

## 示例
### 示例1
- 输入：500人互联网公司，需要预测下季度离职率
- 输出：分析报告，含流失预测模型、关键因素、干预建议

## 注意事项
- 要数据驱动
- 要洞察驱动
- 要行动导向
