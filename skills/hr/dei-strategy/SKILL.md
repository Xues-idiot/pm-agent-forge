# 多元化与包容 (DEI Strategy)

## 触发条件
- DEI战略制定
- 包容文化
- 招聘多元化
- 员工体验

## 输入
- 组织现状
- 目标设定
- 行业实践
- 资源投入

## 输出
- DEI战略
- 目标指标
- 实施方案
- 评估机制

## Prompt模板
```
请为[组织]制定DEI战略。

背景：
- 组织现状：[现状]
- 目标设定：[目标]
- 行业实践：[实践]
- 资源投入：[资源]

请输出：
1. DEI现状评估
2. DEI战略目标
3. 招聘多元化
4. 晋升公平性
5. 薪酬公平性
6. 包容文化
7. 员工资源组
8. 培训和发展
9. 评估和报告
```

## 示例
### 示例1
- 输入：跨国科技公司，需要提升女性管理层比例
- 输出：DEI方案，含招聘目标、导师计划、包容文化

## 注意事项
- 要领导承诺
- 要数据驱动
- 要持续跟进
