# Employee Onboarding Skill

设计员工入职项目，创建迎新流程，管理新员工体验。

## 何时使用

- 入职流程设计
- 新员工培训
- 导师制安排
- 试用期管理
- 入职体验优化

## 入职管理流程

```
1. 入职准备 → 2. 第一天 → 3. 第一周 → 4. 试用期管理 → 5. 转正评估
```

## 入职体验优化

**入职体验指标：**

```markdown
## 入职体验指标

### NPS 收集
- 入职7天：发送入职体验调研
- 入职30天：再次调研
- 入职90天：转正调研

### 调研内容
```markdown
## 入职体验调研

1. 入职第一天体验如何？
2. 培训内容有帮助吗？
3. 导师支持到位吗？
4. 有什么建议？

评分：1-10分
NPS = 推荐者% - 贬损者%
```
```

## 入职管理检查清单

- [ ] 入职前准备充分
- [ ] 第一天安排紧凑
- [ ] 手续办理高效
- [ ] 导师匹配合适
- [ ] 培训内容实用
- [ ] 评估节点清晰
- [ ] 转正流程规范
- [ ] 体验持续改进

## 红牌警告

- 入职手续不全
- 合同条款违规
- 试用期内不管理
- 导师形同虚设
- 评估标准模糊
- 转正决策拖延
- 新员工快速离职
- 负面舆情未处理
