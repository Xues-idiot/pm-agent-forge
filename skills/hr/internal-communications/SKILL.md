# 内部沟通 (Internal Communications)

## 触发条件
- 沟通策略
- 员工沟通
- 变革沟通
- 文化传播

## 输入
- 员工规模
- 沟通目标
- 现有渠道
- 文化特点

## 输出
- 沟通策略
- 渠道规划
- 内容策略
- 效果评估

## Prompt模板
```
请制定[组织]内部沟通策略。

背景：
- 员工规模：[规模]
- 沟通目标：[目标]
- 现有渠道：[渠道]
- 文化特点：[文化]

请输出：
1. 沟通目标
2. 沟通原则
3. 沟通渠道
4. 内容策略
5. 关键信息
6. 变革沟通
7. 危机沟通
8. 效果评估
```

## 示例
### 示例1
- 输入：大型企业，员工分散，需要提升凝聚力
- 输出：沟通策略，含多渠道、内容矩阵、员工参与

## 注意事项
- 要双向沟通
- 要一致透明
- 要持续改进
