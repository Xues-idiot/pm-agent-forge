# 远程办公 (Remote Work)

## 触发条件
- 远程办公政策
- 远程团队管理
- 协作效率
- 文化维护

## 输入
- 组织类型
- 远程比例
- 员工特征
- 业务性质

## 输出
- 远程政策
- 管理机制
- 协作工具
- 评估体系

## Prompt模板
```
请设计[组织]远程办公方案。

背景：
- 组织类型：[类型]
- 远程比例：[比例]
- 员工特征：[特征]
- 业务性质：[性质]

请输出：
1. 远程办公政策
2. 适用岗位
3. 工作时间灵活安排
4. 协作和沟通机制
5. 绩效管理调整
6. 信息安全
7. 员工支持
8. 线下聚集
9. 效果评估
```

## 示例
### 示例1
- 输入：互联网公司，想让全员每周远程2天
- 输出：远程方案，含协作工具、考核调整、信息安全

## 注意事项
- 要信任员工
- 要结果导向
- 要保持连接
