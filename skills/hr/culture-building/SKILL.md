# 企业文化建设 (Culture Building)

## 触发条件
- 企业文化塑造
- 文化落地推行
- 文化评估诊断
- 跨文化管理

## 输入
- 企业发展阶段
- 现有文化
- 战略目标
- 员工构成

## 输出
- 企业文化体系
- 文化落地方案
- 行为规范
- 评估机制

## Prompt模板
```
请为[公司]设计企业文化体系。

背景：
- 发展阶段：[阶段]
- 现有文化：[文化]
- 战略目标：[目标]
- 员工构成：[构成]

请输出：
1. 企业文化诊断
2. 愿景使命价值观设计
3. 文化内涵诠释
4. 行为规范体系
5. 文化落地机制（制度/活动/故事）
6. 文化传播和培训
7. 管理者文化示范
8. 文化评估和迭代
9. 跨文化管理（如涉及）
```

## 示例
### 示例1
- 输入：并购后企业，两个团队文化差异大，需要融合
- 输出：文化融合方案，含核心价值统一、文化整合策略

## 注意事项
- 文化要真实而非口号
- 要从上往下推行
- 要持续强化
