# Offboarding Management Skill

离职管理，规范员工离职流程，沉淀组织经验。

## 何时使用

- 员工离职处理
- 离职面谈
- 离职流程管理
- 知识传承
- 离职分析改进

## 离职管理流程

```
1. 离职申请 → 2. 挽留评估 → 3. 离职办理 → 4. 知识交接 → 5. 离职分析
```

### 1. 离职申请

**离职类型：**

```markdown
## 离职类型

### 自愿离职
| 类型 | 说明 | 处理优先级 |
|------|------|----------|
| 个人发展 | 创业/深造 | 标准流程 |
| 家庭原因 | 照顾家庭 | 标准流程 |
| 职业倦怠 | 工作动力下降 | 关注原因 |
| 薪酬不满 | 薪资低于预期 | 评估调整可能 |

### 非自愿离职
| 类型 | 说明 | 处理优先级 |
|------|------|----------|
| 裁员 | 业务调整 | 合规优先 |
| 绩效优化 | 不胜任 | 关怀优先 |
| 违纪解除 | 违规行为 | 合规处理 |

### 离职申请规范
```markdown
## 申请流程

### 提前通知
- 试用期员工：3天
- 正式员工：30天
- 管理层：60-90天

### 离职申请内容
- 离职原因
- 预计离职日期
- 工作交接安排
```
```

**离职申请处理：**

```markdown
## 申请处理

### 审批流程
```markdown
## 审批节点

员工提交 → 直属上级 → HR → 分管领导 → 审批完成

### 各节点职责
| 节点 | 职责 |
|------|------|
| 直属上级 | 挽留评估、交接安排 |
| HR | 离职手续、风险评估 |
| 分管领导 | 最终审批 |
```
```

### 2. 挽留评估

**挽留价值评估：**

```markdown
## 挽留评估

### 挽留价值矩阵
```markdown
## 挽留决策

| 员工价值 | 离职原因 | 建议 |
|----------|----------|------|
| 高 | 可解决 | 优先挽留 |
| 高 | 难解决 | 尽量挽留 |
| 低 | - | 友好分手 |
```
```

### 挽留策略：
```markdown
## 挽留策略

### 薪酬调整
适用：薪酬低于市场
方案：调薪/奖金承诺
风险：内部平衡

### 晋升发展
适用：有潜力员工
方案：晋升/新项目
风险：承诺需兑现

### 工作调整
适用：工作倦怠
方案：岗位调整/轮岗
风险：需合理安排

### 情感挽留
适用：忠诚老员工
方案：领导沟通
风险：效果有限
```

**挽留面谈：**

```markdown
## 挽留面谈

### 面谈准备
- [ ] 了解员工背景
- [ ] 明确挽留方案
- [ ] 准备面谈环境
- [ ] 预约合适时间

### 面谈话术
```markdown
## 挽留话术

### 开场
"收到你的离职申请，我想和你好好聊聊，看看有没有什么我们可以一起解决的。"

### 倾听了解
"能告诉我你决定离开的具体原因吗？"
"你对现在的岗位/团队/公司有什么想法？"

### 回应解决
"关于你提到的[原因]，我们来看看能不能解决..."

### 挽留提出
"基于你的贡献，我想和你讨论一个方案..."

### 结果确认
"你考虑一下吧，我们明天再聊？"
```
```

### 5. 离职分析

**离职面谈：**

```markdown
## 离职面谈

### 面谈时机
- 审批通过后尽快安排
- 避免最后一天才谈

### 面谈问题
```markdown
## 面谈话术

### 开放性问题
"你离开的真正原因是什么？"
"这段时间对公司有什么建议？"
"你对团队管理有什么看法？"
"如果你重新选择，会留下来吗？"

### 敏感问题
（根据情况谨慎询问）
"是什么让你最终决定离开？"
"有没有什么我们本可以做得更好？"
```
```

## 红牌警告

- 强制离职不赔偿
- 扣押证件扣押工资
- 恶意不批准离职
- 忽视核心人才流失
- 离职面谈谈废话
- 离职原因不分析
- 改进措施不执行
- 恶意挖角不防范
