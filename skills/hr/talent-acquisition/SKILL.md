# 人才获取策略 (Talent Acquisition Strategy)

## 触发条件
- 招聘策略制定
- 招聘渠道优化
- 雇主品牌建设
- 人才库建设

## 输入
- 招聘需求
- 人才市场
- 竞争对手
- 预算情况

## 输出
- 招聘策略
- 渠道组合
- 雇主品牌
- 评估体系

## Prompt模板
```
请为[公司/团队]制定人才获取策略。

背景：
- 招聘需求：[需求]
- 人才市场：[市场]
- 竞争对手：[对手]
- 预算情况：[预算]

请输出：
1. 人才获取目标和原则
2. 岗位分析和人才画像
3. 招聘渠道组合策略
4. 招聘流程优化
5. 雇主品牌建设
6. 人才库运营策略
7. 猎头和外包策略
8. 招聘效果评估体系
9. 成本控制策略
```

## 示例
### 示例1
- 输入：AI创业公司，急招算法工程师，预算有限
- 输出：招聘策略，含技术社区运营、工程师推荐、内部培养

## 注意事项
- 要精准而非多投
- 要重视雇主品牌
- 要持续运营人才库
