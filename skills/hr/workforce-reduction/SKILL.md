# 裁员管理 (Workforce Reduction)

## 触发条件
- 组织优化
- 裁员计划
- 员工安置
- 沟通管理

## 输入
- 裁员规模
- 裁员原因
- 员工构成
- 时间要求

## 输出
- 裁员方案
- 补偿方案
- 安置措施
- 沟通计划

## Prompt模板
```
请制定[组织]裁员管理方案。

背景：
- 裁员规模：[规模]
- 裁员原因：[原因]
- 员工构成：[构成]
- 时间要求：[要求]

请输出：
1. 裁员背景和必要性
2. 裁员标准和选择
3. 补偿方案
4. 安置措施
5. 沟通策略
6. 时间表
7. 风险控制
8. 后续跟进
```

## 示例
### 示例1
- 输入：业务调整，需要裁减20%员工
- 输出：裁员方案，含选择标准、补偿方案、沟通计划

## 注意事项
- 要合法合规
- 要有尊严
- 要充分沟通
