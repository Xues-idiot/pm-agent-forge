# Recruitment Management Skill

管理端到端招聘流程，从人力规划到候选人入职。

## 何时使用

- 招聘流程管理
- 人才获取策略
- 面试流程设计
- 候选人体验
- 招聘效率优化
- 雇主品牌

## 招聘管理流程

```
1. 人力规划 → 2. 人才获取 → 3. 筛选评估 → 4. 面试决策 → 5. 入职管理
```

- [ ] 部分匹配

### 不通过 → 婉拒

- [ ] 经验不足

## 招聘管理检查清单

- [ ] 需求分析清晰
- [ ] 渠道选择合适
- [ ] 筛选标准明确
- [ ] 面试流程规范
- [ ] 决策机制高效
- [ ] Offer有竞争力
- [ ] 入职体验良好
- [ ] 数据追踪完善

## 红牌警告

- 招聘需求不清晰
- 薪资严重偏离市场
- 试用期大量淘汰
- 关键人才流失
- 歧视性招聘
- 背景调查流于形式
- 信息不透明
- 承诺不兑现
