# 人力规划 (Workforce Planning)

## Overview

人力规划是根据业务战略规划未来人力资源需求的技能。涵盖需求预测、人才盘点、梯队建设、成本预算等核心环节。

## 何时使用

- 制定年度人力预算
- 业务扩张需要增编
- 组织架构调整
- 人才队伍建设

## 流程

1. **战略对齐**：理解业务战略和目标
2. **现状盘点**：盘点现有人力资源
3. **需求预测**：预测未来人力需求
4. **缺口分析**：分析人才缺口
5. **获取策略**：制定人才获取计划
6. **成本预算**：制定人力成本预算
7. **执行监控**：监控执行情况

## 红牌警告

- 人力规划与业务脱节
- 盲目扩张导致人效下降
- 成本预算严重超支
- 关键岗位招聘不力
