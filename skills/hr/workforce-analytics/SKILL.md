# 人力数据分析 (Workforce Analytics)

## 触发条件
- 人力数据分析
- 人员效率评估
- 流失预测
- 成本分析

## 输入
- 数据范围
- 分析需求
- 组织结构
- 历史数据

## 输出
- 分析报告
- 预测模型
- 决策建议
- 监控看板

## Prompt模板
```
请对[组织]进行人力数据分析。

背景：
- 员工规模：[规模]
- 数据范围：[范围]
- 分析需求：[需求]
- 历史数据：[数据]

请输出：
1. 人力数据概览
2. 组织效能分析
3. 人员结构分析（部门/层级/年龄）
4. 人才流动分析（流入/流出/晋升）
5. 人效指标分析
6. 流失预测模型
7. 成本分析
8. 关键发现和建议
9. 监控看板设计
```

## 示例
### 示例1
- 输入：千人互联网公司，需要分析人员流失原因
- 输出：分析报告，含流失率趋势、流失原因、预测模型

## 注意事项
- 数据要准确
- 分析要深入
- 建议要可执行
