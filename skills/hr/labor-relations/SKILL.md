# 劳动关系 (Labor Relations)

## 触发条件
- 工会管理
- 集体协商
- 员工参与
- 争议处理

## 输入
- 员工规模
- 工会情况
- 法规环境
- 历史关系

## 输出
- 管理策略
- 沟通机制
- 协商流程
- 争议处理

## Prompt模板
```
请优化[组织]劳动关系管理。

背景：
- 员工规模：[规模]
- 工会情况：[情况]
- 法规环境：[环境]
- 历史关系：[历史]

请输出：
1. 劳动关系诊断
2. 管理策略
3. 沟通机制
4. 工会管理
5. 集体协商
6. 员工参与
7. 争议处理
8. 预防机制
```

## 示例
### 示例1
- 输入：制造企业，有工会，需要维护和谐关系
- 输出：管理方案，含沟通机制、集体协商、员工参与

## 注意事项
- 要合规管理
- 要平等协商
- 要预防争议
