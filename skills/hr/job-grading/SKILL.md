# 岗位评估 (Job Grading)

## 触发条件
- 岗位体系
- 职级序列
- 薪酬对标
- 晋升通道

## 输入
- 组织架构
- 岗位分布
- 市场定位
- 管理诉求

## 输出
- 评估方案
- 级别体系
- 对标结果
- 应用建议

## Prompt模板
```
请设计[组织]岗位评估方案。

背景：
- 组织架构：[架构]
- 岗位分布：[分布]
- 市场定位：[定位]
- 管理诉求：[诉求]

请输出：
1. 岗位分类
2. 评估维度
3. 评估方法
4. 级别体系
5. 市场对标
6. 结果应用
7. 实施计划
```

## 示例
### 示例1
- 输入：科技公司，需要统一职级体系
- 输出：评估方案，含评估维度、级别带宽、市场对标

## 注意事项
- 要科学公正
- 要市场导向
- 要应用明确
