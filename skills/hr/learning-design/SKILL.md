# 学习设计 (Learning Design)

## 触发条件
- 课程设计
- 学习体验
- 教学方法
- 效果评估

## 输入
- 学习目标
- 学员特点
- 内容类型
- 交付形式

## 输出
- 学习方案
- 内容设计
- 活动设计
- 评估方案

## Prompt模板
```
请设计[主题]学习方案。

背景：
- 学习目标：[目标]
- 学员特点：[特点]
- 内容类型：[类型]
- 交付形式：[形式]

请输出：
1. 学习目标
2. 内容架构
3. 教学方法
4. 学习活动
5. 案例设计
6. 练习设计
7. 效果评估
```

## 示例
### 示例1
- 输入：领导力培训，中层管理者，混合式学习
- 输出：学习方案，含课堂+实践+辅导混合设计

## 注意事项
- 要以学员为中心
- 要知行合一
- 要效果导向
