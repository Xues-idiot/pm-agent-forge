# Talent Assessment Skill

人才评估，支撑绩效管理和人才决策。

## 何时使用

- 人才盘点
- 绩效评估
- 晋升评审
- 潜力评估
- 继任计划

## 评估流程

```
1. 评估准备 → 2. 数据收集 → 3. 多维度评估 → 4. 综合判断 → 5. 结果应用
```

### 1. 评估准备

**明确评估目的：**

```markdown
## 评估目的

### 评估类型
| 类型 | 频率 | 侧重点 |
|------|------|--------|
| 年度评估 | 每年 | 综合表现 |
| 晋升评估 | 按申请 | 能力潜力 |
| 期中评估 | 每半年 | 反馈改进 |
| 试用期评估 | 转正前 | 是否录用 |

### 评估维度
```markdown
## 通用评估维度

### 业绩维度
- 目标完成情况
- 工作质量
- 工作效率

### 能力维度
- 专业能力
- 管理能力（如适用）
- 通用能力

### 态度维度
- 责任心
- 协作性
- 学习成长
```
```

**评估周期：**

```markdown
## 评估周期

### 年度评估
```markdown

| 阶段 | 时间 | 动作 |
|------|------|------|

```

```markdown
## 评估数据

|----------|------|------|

```

### 3. 多维度评估

**绩效评估：**

```markdown
## 绩效评估

### 评估维度
| 维度 | 权重 | 评分标准 |
|------|------|----------|
| 目标完成 | 40% | 0-100分 |
| 工作质量 | 30% | 0-100分 |
| 专业能力 | 20% | 0-100分 |
| 协作态度 | 10% | 0-100分 |

### 评分标准
```markdown
## 评分定义

| 等级 | 分值 | 定义 |
|------|------|------|
| A | 90-100 | 远超预期，做出突出贡献 |
| B | 80-89 | 超出预期，全面达成目标 |
| C | 70-79 | 达到预期，较好完成工作 |
| D | 60-69 | 基本达到，有待改进 |
| E | <60 | 未达到，需改进或淘汰 |
```
```

```markdown

### 4. 综合判断

**校准会议：**

```markdown
## 校准会

### 参会人
- HR负责人
- 部门负责人
- 各小组负责人

### 校准流程
```markdown
## 校准流程

1. 数据汇总 → 2. 分组讨论 → 3. 部门校准 → 4. 结果确认

### 讨论要点
- [ ] 高绩效/低绩效员工
- [ ] 绩效分布合理性
- [ ] 强制分布比例
- [ ] 晋升/调薪建议
- [ ] 改进计划
```
```

**评估结果：**

```markdown
## 评估结果

### 结果分布（强制分布）
```markdown
## 比例要求

| 等级 | 比例 | 说明 |
|------|------|------|
| A | 10-15% | 优秀 |
| B | 25-30% | 良好 |
| C | 40-50% | 合格 |
| D | 10-15% | 待改进 |
| E | 0-5% | 不合格 |
```
```

### 5. 结果应用

**应用场景：**

```markdown
## 结果应用

### 调薪/奖金
| 绩效 | 调薪幅度 | 奖金系数 |
|------|----------|----------|
| A | 15-25% | 2-3个月 |
| B | 8-15% | 1.5-2个月 |
| C | 3-8% | 1个月 |
| D | 0-3% | 0.5个月 |
| E | 0% | 0 |

### 晋升/发展
| 绩效 | 潜力 | 建议 |
|------|------|------|
| B+ | 高 | 优先晋升 |
| B+ | 中 | 培养晋升 |
| B | 高 | 加速培养 |
| C | - | 本级发展 |
| D | - | 绩效改进 |

### 培训发展
```markdown
## 发展计划

| 等级 | 培训重点 |
|------|----------|
| A | 领导力/战略思维 |
| B | 能力短板提升 |
| C | 岗位技能强化 |
| D | 针对性改进 |
```
```

## 人才评估检查清单

- [ ] 评估目的明确
- [ ] 评估维度全面
- [ ] 数据来源可靠
- [ ] 评分标准一致
- [ ] 校准流程规范
- [ ] 结果应用合理
- [ ] 反馈及时到位
- [ ] 改进计划跟踪

## 红牌警告

- 评估标准不清晰
- 打分主观随意
- 强制分布不合理
- 绩效结果不透明
- 反馈空洞敷衍
- 改进计划不落地
- 评估结果不应用
- 连续低绩效不处理
