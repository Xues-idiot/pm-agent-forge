# 薪酬策略 (Compensation Strategy)

## 触发条件
- 薪酬体系设计
- 薪酬结构调整
- 市场竞争性分析
- 薪酬激励设计

## 输入
- 公司战略
- 市场定位
- 财务约束
- 员工结构

## 输出
- 薪酬策略
- 薪酬结构设计
- 市场对标
- 激励方案

## Prompt模板
```
请为[公司]制定薪酬策略。

背景：
- 公司阶段：[阶段]
- 市场定位：[定位]
- 财务约束：[约束]
- 员工结构：[结构]

请输出：
1. 薪酬战略定位
2. 薪酬目标（内部公平/外部竞争）
3. 薪酬结构设计（固浮比/级别宽带）
4. 市场对标分析
5. 薪酬水平策略（领先/跟随/保守）
6. 奖金激励方案
7. 长期激励设计
8. 薪酬调整机制
```

## 示例
### 示例1
- 输入：互联网创业公司，核心员工竞争激烈，需要吸引高端人才
- 输出：薪酬策略，含现金+股权组合、竞争性薪酬、成长机会

## 注意事项
- 要外部有竞争
- 要内部公平
- 要激励效果
