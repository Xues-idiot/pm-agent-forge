# 学习文化建设 (Learning Culture)

## 触发条件
- 学习型组织建设
- 学习氛围营造
- 知识共享机制
- 持续学习激励

## 输入
- 组织类型
- 员工特征
- 现有文化
- 发展目标

## 输出
- 学习型组织方案
- 学习机制
- 激励设计
- 评估体系

## Prompt模板
```
请为[组织]建设学习文化。

背景：
- 组织类型：[类型]
- 员工特征：[特征]
- 现有文化：[文化]
- 发展目标：[目标]

请输出：
1. 学习文化愿景
2. 学习生态系统设计
3. 知识共享机制
4. 导师制度
5. 学习激励机制
6. 实践社区运营
7. 学习成果转化
8. 效果评估
9. 持续迭代
```

## 示例
### 示例1
- 输入：科技公司，工程师爱自学，需要加强知识沉淀
- 输出：学习文化方案，含内部分享、技术读书会、知识库

## 注意事项
- 要领导示范
- 要与工作结合
- 要持续强化
