# 雇佣政策 (Employment Policy)

## 触发条件
- 政策制定
- 合规管理
- 制度更新
- 员工手册

## 输入
- 组织类型
- 员工规模
- 法规要求
- 文化特点

## 输出
- 政策框架
- 制度文档
- 合规清单
- 更新计划

## Prompt模板
```
请制定[组织]雇佣政策。

背景：
- 组织类型：[类型]
- 员工规模：[规模]
- 法规要求：[要求]
- 文化特点：[特点]

请输出：
1. 政策框架
2. 入职管理
3. 考勤假期
4. 薪酬福利
5. 绩效管理
6. 晋升发展
7. 离职管理
8. 纪律处分
9. 合规检查
```

## 示例
### 示例1
- 输入：跨国公司，需要制定统一雇佣政策
- 输出：政策框架，含全球标准+本地合规、员工手册

## 注意事项
- 要合法合规
- 要清晰明确
- 要公平一致
