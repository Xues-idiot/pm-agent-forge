# 组织变革管理 (Organizational Change Management)

## 触发条件
- 组织架构调整
- 业务流程重组
- 变革转型推进
- 变革阻力化解

## 输入
- 变革背景
- 变革范围
- 利益相关方
- 风险容忍度

## 输出
- 变革管理方案
- 沟通计划
- 阻力分析
- 实施路径

## Prompt模板
```
请为[变革项目]制定管理方案。

背景：
- 变革类型：[类型]
- 变革范围：[范围]
- 涉及部门：[部门]
- 时间要求：[要求]

请输出：
1. 变革愿景和目标
2. 影响分析（组织/流程/人员）
3. 利益相关方分析和策略
4. 阻力识别和应对
5. 沟通和参与计划
6. 培训和能力提升
7. 实施路径和里程碑
8. 风险管理和预案
9. 变革效果评估
```

## 示例
### 示例1
- 输入：传统企业数字化转型，涉及全员，需在6个月内完成
- 输出：变革管理方案，含沟通策略、培训计划、阻力应对

## 注意事项
- 沟通要持续
- 参与要广泛
- 阻力要正视
