# 员工福祉 (Employee Wellbeing)

## 触发条件
- 福利体系设计
- 员工体验
- 健康管理
- 满意度提升

## 输入
- 员工规模
- 福利现状
- 预算情况
- 员工特征

## 输出
- 福利策略
- 福利方案
- 健康管理
- 评估机制

## Prompt模板
```
请设计[组织]员工福祉方案。

背景：
- 员工规模：[规模]
- 福利现状：[现状]
- 预算情况：[预算]
- 员工特征：[特征]

请输出：
1. 员工福祉战略
2. 薪酬外福利
3. 健康管理
4. 工作生活平衡
5. 心理健康
6. 职业发展
7. 认可和奖励
8. 实施计划
9. 效果评估
```

## 示例
### 示例1
- 输入：互联网公司，员工压力大，流失率高
- 输出：福祉方案，含弹性福利、健康保险、心理支持

## 注意事项
- 要员工导向
- 要个性化
- 要持续改进
