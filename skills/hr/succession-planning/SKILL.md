# 继任计划 (Succession Planning)

## 触发条件
- 关键岗位继任
- 人才梯队建设
- 领导力发展
- 人才储备

## 输入
- 关键岗位
- 人才现状
- 发展周期
- 风险识别

## 输出
- 继任计划
- 人才梯队
- 发展方案
- 评估机制

## Prompt模板
```
请为[组织]设计继任计划。

背景：
- 关键岗位：[岗位]
- 人才现状：[现状]
- 发展周期：[周期]
- 风险识别：[风险]

请输出：
1. 继任计划目标
2. 关键岗位识别
3. 人才盘点
4. 继任者画像
5. 发展路径设计
6. 发展计划（IDP）
7. 过渡期设计
8. 评估和调整机制
9. 沟通和保密
```

## 示例
### 示例1
- 输入：CEO两年后退休，需要培养接班人
- 输出：继任计划，含人选评估、发展方案、过渡计划

## 注意事项
- 要提前规划
- 要保密
- 要动态调整
