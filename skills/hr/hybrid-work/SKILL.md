# 混合办公设计 (Hybrid Work Design)

## 触发条件
- 混合办公模式设计
- 远程工作政策
- 协作效率提升
- 办公空间优化

## 输入
- 业务特点
- 员工类型
- 办公设施
- 管理模式

## 输出
- 混合办公方案
- 政策制度
- 协作工具
- 空间规划

## Prompt模板
```
请为[公司]设计混合办公模式。

背景：
- 业务类型：[类型]
- 员工分布：[分布]
- 办公设施：[设施]
- 管理层态度：[态度]

请输出：
1. 混合办公模式选择（远程/ office/ hybrid）
2. 适用岗位和员工范围
3. 工作时间和地点政策
4. 协作和沟通机制
5. 绩效管理模式调整
6. 办公空间优化方案
7. 技术和工具支持
8. 福利和合规考虑
9. 试点和推广计划
```

## 示例
### 示例1
- 输入：科技公司，工程师岗位为主，分布在全国多地
- 输出：混合办公方案，含异步协作、核心协作时间、居家补贴

## 注意事项
- 要平衡效率和协作
- 要公平对待远程和办公室员工
- 要有明确的政策
