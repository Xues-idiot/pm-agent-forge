# 员工关系管理 (Employee Relations)

## 触发条件
- 员工关系维护
- 员工满意度提升
- 劳动风险预防
- 离职管理

## 输入
- 员工规模
- 劳动关系现状
- 历史问题
- 管理诉求

## 输出
- 员工关系策略
- 沟通机制
- 风险预防
- 离职管理

## Prompt模板
```
请为[公司]设计员工关系管理方案。

背景：
- 员工规模：[规模]
- 劳动关系现状：[现状]
- 历史问题：[问题]
- 管理诉求：[诉求]

请输出：
1. 员工关系目标和原则
2. 员工沟通机制设计
3. 员工满意度调查
4. 劳动风险预防
5. 申诉和投诉处理
6. 离职管理
7. 核心员工保留
8. 文化建设
9. 突发事件的应对
```

## 示例
### 示例1
- 输入：劳动密集型企业，员工1000人，有过劳动仲裁
- 输出：员工关系方案，含沟通机制、风险预防、留存激励

## 注意事项
- 要主动沟通
- 要预防而非救火
- 要合法合规
