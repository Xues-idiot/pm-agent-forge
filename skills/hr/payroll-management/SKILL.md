# 薪酬发放 (Payroll Management)

## 触发条件
- 薪资核算
- 税务计算
- 发放管理
- 合规要求

## 输入
- 员工规模
- 薪酬结构
- 税务要求
- 发放频率

## 输出
- 核算方案
- 税务处理
- 发放流程
- 异常处理

## Prompt模板
```
请优化[组织]薪酬发放。

背景：
- 员工规模：[规模]
- 薪酬结构：[结构]
- 税务要求：[要求]
- 发放频率：[频率]

请输出：
1. 薪酬核算流程
2. 税务计算
3. 社保公积金
4. 发放流程
5. 异常处理
6. 保密管理
7. 系统支撑
```

## 示例
### 示例1
- 输入：跨国企业，多地发薪，需要统一管理
- 输出：发放方案，含多地发薪、税务合规、系统集成

## 注意事项
- 要准确及时
- 要合规保密
- 要持续优化
