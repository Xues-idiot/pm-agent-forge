# 绩效辅导 (Performance Coaching)

## 触发条件
- 绩效面谈
- 员工辅导
- 绩效改进
- 目标追踪

## 输入
- 绩效类型
- 员工情况
- 管理目标
- 辅导风格

## 输出
- 辅导计划
- 面谈策略
- 改进方案
- 跟进机制

## Prompt模板
```
请设计[员工]绩效辅导方案。

背景：
- 绩效类型：[类型]
- 员工情况：[情况]
- 管理目标：[目标]
- 辅导风格：[风格]

请输出：
1. 绩效诊断
2. 辅导目标
3. 面谈策略
4. 反馈技巧
5. 改进计划
6. 资源支持
7. 跟进机制
8. 效果评估
```

## 示例
### 示例1
- 输入：试用期员工，绩效不达标
- 输出：辅导方案，含绩效分析、改进计划、每周跟进

## 注意事项
- 要具体客观
- 要发展导向
- 要持续跟进
