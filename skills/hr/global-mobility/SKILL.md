# 全球 Mobility (Global Mobility)

## 触发条件
- 跨国派遣
- 海外用工
- 员工调动
- 签证合规

## 输入
- 派遣类型
- 目的地国家
- 员工情况
- 成本预算

## 输出
- 派遣方案
- 签证安排
- 薪酬设计
- 合规要求

## Prompt模板
```
请设计[员工]全球派遣方案。

背景：
- 派遣类型：[类型]
- 目的地：[目的地]
- 员工情况：[情况]
- 成本预算：[预算]

请输出：
1. 派遣政策
2. 签证安排
3. 税务筹划
4. 薪酬方案
5. 福利设计
6. 家属安排
7. 回国安置
8. 合规要求
```

## 示例
### 示例1
- 输入：高管从中国派往美国3年
- 输出：派遣方案，含签证、税务、住房、子女教育

## 注意事项
- 要合规先行
- 要成本控制
- 要员工体验
