# 药品管理 (Drug Management)

## 触发条件
- 药品采购管理
- 药品库存管理
- 用药安全
- 合理用药

## 输入
- 药品类型
- 库存现状
- 使用场景
- 管理目标

## 输出
- 采购方案
- 库存策略
- 用药规范
- 评估体系

## Prompt模板
```
请为[医疗机构/药房]设计药品管理方案。

背景：
- 药品类型：[类型]
- 库存现状：[现状]
- 使用场景：[场景]
- 管理目标：[目标]

请输出：
1. 药品分类管理
2. 采购流程优化
3. 库存管理策略
4. 药品养护规范
5. 用药安全管理
6. 合理用药监控
7. 药品追溯系统
8. 成本控制
9. 供应商管理
```

## 示例
### 示例1
- 输入：社区医院，药品300种，需要降低库存成本
- 输出：管理方案，含ABC分类、重点药品管理、零库存策略

## 注意事项
- 要确保用药安全
- 要平衡库存和成本
- 要符合法规
