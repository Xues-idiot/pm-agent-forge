# Health Data Analysis

健康数据分析，分析健康指标和医疗数据。

## 何时使用

- 健康指标分析
- 疾病分析
- 治疗效果分析
- 医疗数据可视化
- 健康预警

## 健康数据基础

### 健康数据类型

```markdown
## 健康数据类型

### 个人健康数据
| 类型 | 内容 | 数据源 |
|------|------|--------|
| 基本信息 | 性别/年龄/身高体重 | 体检 |
| 生命体征 | 血压/心率/体温 | 监测设备 |
| 检验数据 | 血常规/尿常规/生化 | 检验科 |
| 影像数据 | X光/CT/MRI | 放射科 |
| 病历数据 | 诊断/处方/手术 | 门诊/住院 |

### 群体健康数据
| 类型 | 内容 | 用途 |
|------|------|------|
| 发病率 | 疾病发生率 | 疾病负担 |
| 患病率 | 疾病现患率 | 卫生规划 |
| 死亡率 | 疾病死亡率 | 公共卫生 |
| 就诊数据 | 门诊/住院 | 服务利用 |
```

### 健康指标

```markdown
## 健康指标

### 身体健康指标
| 指标 | 正常范围 | 说明 |
|------|----------|------|
| BMI | 18.5-24 | 体质指数 |
| 血压 | <140/90 | 收缩压/舒张压 |
| 心率 | 60-100 | 次/分 |
| 空腹血糖 | <7.0 mmol/L | 糖尿病筛查 |
| 胆固醇 | <5.2 mmol/L | 总胆固醇 |
| 甘油三酯 | <1.7 mmol/L | 心血管风险 |

### 生活方式指标
| 指标 | 推荐值 | 说明 |
|------|--------|------|
| 运动 | 每周150分钟 | 中等强度 |
| 睡眠 | 7-8小时/天 | 成人 |
| 饮食 | 均衡多样 | 膳食指南 |
| 吸烟 | 不吸烟 | 最好 |
| 饮酒 | 限量 | 男<25g/女<15g |
```

### 疾病模式

```markdown
## 疾病模式

### 时间趋势
| 模式 | 特点 | 示例 |
|------|------|------|
| 散发 | 零星发生 | 罕见病 |
| 聚集 | 小范围集中 | 流感爆发 |
| 流行 | 超过平常 | 季节性流感 |
| 大流行 | 全球蔓延 | 新冠 |

### 空间分布
| 分布 | 特点 | 影响因素 |
|------|------|----------|
| 聚集性 | 某些地区高发 | 环境/遗传 |
| 梯度性 | 沿海→内陆 | 经济发展 |
| 城乡差异 | 城市vs农村 | 生活方式 |

### 人群特征
| 特征 | 相关疾病 |
|------|----------|
| 年龄 | 心脑血管↑ |
| 性别 | 男↑心脏病,女↑自身免疫 |
| 职业 | 粉尘→尘肺 |
| 地区 | 南方→风湿,北方→心脑血管 |
```

## 检查清单

- [ ] 数据来源可靠
- [ ] 指标定义清晰
- [ ] 分析方法正确

## 红牌警告

- 数据来源不可靠
- 指标定义混乱
- 分析方法错误
- 可视化误导
- 预警过度/不足
- 干预无效果评估
- 隐私泄露风险
