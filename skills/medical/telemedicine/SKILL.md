# Telemedicine Skill

开展远程医疗咨询，管理虚拟诊所，提供远程医疗服务。

## 何时使用

- 远程问诊
- 在线复诊
- 健康咨询
- 患者随访
- 远程会诊

## 远程医疗流程

```
1. 预约管理 → 2. 问诊前准备 → 3. 问诊执行 → 4. 处方流转 → 5. 随访管理
```

### 1. 预约管理

**预约系统：**

```markdown
## 预约管理

### 预约渠道
| 渠道 | 适用场景 | 开放时间 |
|------|----------|----------|
| APP | 年轻患者 | 24小时 |
| 小程序 | 便捷用户 | 24小时 |
| 电话 | 老年患者 | 8:00-20:00 |
| 短信 | 复诊提醒 | 自动 |

### 预约时段
| 时段 | 开放号源 | 医生数量 |
|------|----------|----------|
| 上午 8:00-12:00 | 50个/医生 | 10人 |
| 下午 14:00-18:00 | 40个/医生 | 8人 |
| 晚上 19:00-21:00 | 20个/医生 | 5人 |

### 预约规则
- [ ] 每人每天限约1次
- [ ] 预约需实名制
- [ ] 取消需提前2小时
- [ ] 迟到需重新排队

### 问诊模板
```markdown
## 问诊模板

### 标准问诊流程
1. 确认患者身份（姓名+身份证）
2. 核实主要症状
3. 了解发病时间
4. 询问伴随症状
5. 了解既往史/过敏史
6. 查看检查报告
7. 形成诊断意见

**问诊类型：**

```markdown

| 项目 | 说明 |
|------|------|

| 适合场景 | 非急性问题 |
| 沟通次数 | 有限制（10次） |
| 局限 | 无法直接观察 |

### 视频问诊
| 项目 | 说明 |
|------|------|
| 时长 | 10-15分钟 |
| 适合场景 | 需要观察患者 |
| 优势 | 可观察体征 |

### 电话问诊
| 项目 | 说明 |
|------|------|
| 时长 | 5-10分钟 |
| 适合场景 | 网络不佳 |
| 优势 | 便捷 |
| 局限 | 无法看报告 |

**满意度调查：**

```markdown
## 满意度调查

### 评价维度
| 维度 | 问题 |
|------|------|
| 响应速度 | 问诊响应是否及时 |
| 医生专业性 | 医生是否专业 |
| 沟通效果 | 沟通是否清晰 |
| 整体满意度 | 总体满意度 |

### 评分标准
- 1分：不满意
- 2分：一般
- 3分：满意
- 4分：很满意
- 5分：非常满意

### NPS
推荐意愿：0-10分
NPS = 推荐者% - 贬损者%
```

## 红牌警告

- 无资质开展诊疗
- 处方不规范
- 用药错误
- 误诊漏诊
- 隐私泄露
- 应急处理不当
- 超范围执业
- 虚假宣传
