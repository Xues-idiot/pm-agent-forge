# Medical Insurance Settlement Skill

处理医疗保险理赔，处理报销，管理医保结算流程。

## 何时使用

- 医保结算
- 商业保险理赔
- 费用报销
- 医保审核
- 定点医院管理
- 医保政策咨询

## 医保结算流程

```
1. 资格确认 → 2. 费用结算 → 3. 审核报销 → 4. 资金拨付 → 5. 档案管理
```

### 1. 资格确认

**医保资格：**

```markdown
## 医保资格确认

### 医保类型
| 类型 | 适用人群 | 报销方式 |
|------|----------|----------|
| 城镇职工 | 在职/退休职工 | 直接结算 |
| 城乡居民 | 非职工 | 直接结算 |
| 新型农村合作医疗 | 农村居民 | 直接结算 |

### 资格核查
```markdown
## 资格核查

### 必查项目
- [ ] 社会保障卡有效
- [ ] 医保参保状态正常
- [ ] 就诊医院为定点医院
- [ ] 当日就诊次数未超限
- [ ] 药品/检查在医保目录

### 异常情况处理
| 异常 | 处理方式 |
|------|----------|
| 卡无效 | 自费就诊，后补报销 |
| 欠费停保 | 缴清后报销 |
| 医院非定点 | 转院或自费 |
| 超次限额 | 审核后处理 |
```
```

### 2. 费用结算

**费用分类：**

```markdown
## 费用分类

### 医保目录内费用
| 类别 | 报销比例 | 说明 |
|------|----------|------|
| 甲类药品 | 100% | 基本全额报销 |
| 乙类药品 | 80-90% | 部分自付 |
| 丙类药品 | 0% | 全额自费 |
| 甲类诊疗 | 100% | 基本全额报销 |
| 乙类诊疗 | 70-90% | 部分自付 |

### 费用计算
```markdown
## 费用计算

### 计算公式
```
医保报销 = (总费用 - 丙类自费 - 乙类自付×比例 - 起付线) × 报销比例
个人自付 = 总费用 - 医保报销
```

### 示例
- 总费用：1000元
- 丙类自费：100元
- 乙类自付：200元（自付比例10%）
- 乙类自付金额：200 × 10% = 20元
- 起付线：500元（三甲医院）
- 报销比例：80%

医保报销 = (1000 - 100 - 20 - 500) × 80% = 304元
个人自付 = 1000 - 304 = 696元
```
```

**结算方式：**

```markdown
## 结算方式

### 实时结算
| 适用 | 流程 |
|------|------|
| 医保定点医院 | 就诊→刷卡→直接报销→个人自付 |

### 事后报销
| 适用 | 流程 |
|------|------|
| 急诊/异地就医 | 全额垫付→收集材料→提交报销→资金到账 |

### 异地就医
```markdown
## 异地就医

### 备案要求
- [ ] 异地安置退休人员
- [ ] 异地长期居住人员
- [ ] 常驻异地工作人员
- [ ] 异地转诊人员

### 报销规则
- 备案后：执行参保地政策
- 未备案：降低报销比例或不予报销
```
```

### 3. 审核报销

**审核要点：**

```markdown
## 审核要点

### 真实性审核
- [ ] 就诊记录与费用明细一致
- [ ] 诊断与治疗相符
- [ ] 药品与诊断相符
- [ ] 检查合理必要

### 规范性审核
- [ ] 诊疗项目编码正确
- [ ] 药品编码对应
- [ ] 服务项目在目录内
- [ ] 数量金额准确

### 违规类型
| 类型 | 表现 | 处理 |
|------|------|------|
| 挂床住院 | 无实际治疗 | 拒付+处罚 |
| 虚记费用 | 多记未做项目 | 拒付+处罚 |
| 分解住院 | 多次住院分解 | 拒付+处罚 |
| 过度检查 | 检查套餐化 | 拒付 |
```

### 4. 资金拨付

**拨付流程：**

```markdown
## 资金拨付

### 拨付流程
```markdown
## 拨付流程

医院申报 → 医保审核 → 金额确认 → 拨款申请 → 资金拨付 → 到账通知
```

### 拨付周期
| 类型 | 周期 | 说明 |
|------|------|------|
| 城镇职工 | 月度结算 | 次月拨付 |
| 城乡居民 | 季度结算 | 次季度拨付 |
| 医疗救助 | 季度结算 | 次季度拨付 |

### 差错处理
| 类型 | 处理方式 |
|------|----------|
| 医院多收 | 扣回多付资金 |
| 医院少收 | 下期补付 |
| 系统错误 | 核实后调整 |
```
