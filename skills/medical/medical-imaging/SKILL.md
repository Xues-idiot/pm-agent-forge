# 医学影像分析 (Medical Imaging Analysis)

## Overview

医学影像分析是通过AI技术辅助医生分析CT、MRI、X光等医学影像的技能。涵盖影像质控、标注管理、模型训练、临床验证等核心环节。

## 何时使用

- 影像科医生工作量过大
- 需要辅助诊断提升效率
- 基层医院缺乏影像专科医生
- 早筛早诊需求

## 流程

1. **临床需求**：确定分析任务和场景
2. **数据收集**：收集符合要求的影像数据
3. **影像标注**：专业医生进行标注
4. **模型训练**：训练AI分析模型
5. **性能评估**：评估准确率、敏感性、特异性
6. **临床验证**：在医院进行临床验证
7. **迭代优化**：根据反馈持续优化


## 检查清单

- [ ] 影像数据符合隐私保护要求
- [ ] 标注质量有保障
- [ ] 模型性能满足临床需求
- [ ] 有可解释性分析
- [ ] 通过临床验证
- [ ] 有NMPA/FDA注册证

## 红牌警告

- 使用未授权影像数据
- 标注质量差影响模型效果
- 模型泛化能力不足
- 缺乏临床验证直接上线
- 过度宣传AI诊断能力
