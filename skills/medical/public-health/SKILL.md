# 公共卫生 (Public Health)

## 触发条件
- 疾病预防
- 健康教育
- 应急响应
- 社区健康

## 输入
- 区域特点
- 疾病谱
- 资源能力
- 政策环境

## 输出
- 防控策略
- 健康教育
- 应急方案
- 评估机制

## Prompt模板
```
请制定[区域]公共卫生方案。

背景：
- 区域特点：[特点]
- 疾病谱：[疾病]
- 资源能力：[资源]
- 政策环境：[政策]

请输出：
1. 公共卫生评估
2. 疾病防控策略
3. 健康教育
4. 疫苗接种
5. 应急响应
6. 社区健康
7. 监督评估
```

## 示例
### 示例1
- 输入：城市社区，需要提升慢病管理
- 输出：公卫方案，含健康档案、慢病管理、健康教育

## 注意事项
- 要预防为主
- 要社区为基础
- 要多方协作
