# Medical Diagnostic Assistance Skill

辅助临床诊断，生成鉴别诊断，分析症状，推荐检查。

## 何时使用

- 辅助诊断决策
- 鉴别诊断生成
- 检查项目推荐
- 疑难病例分析
- 诊断标准核对
- 罕见病识别提示

## 诊断辅助流程

```
1. 症状收集 → 2. 初步分析 → 3. 鉴别诊断 → 4. 检查推荐 → 5. 诊断建议
```

## 诊断辅助决策支持

### 诊断思维框架

```markdown
## 诊断思维

### 经典七步法
1. 收集完整病史
2. 分析主要症状
3. 提出鉴别诊断
4. 针对性体检
5. 选择检查项目
6. 归纳诊断依据
7. 给出诊断结论

### 常用鉴别方法
| 方法 | 适用场景 |
|------|---------|
| 排除法 | 症状不典型 |
| 特征法 | 有典型表现 |
| 随访法 | 诊断困难 |
| 会诊法 | 疑难病例 |
```

## 诊断辅助检查清单

- [ ] 症状收集完整
- [ ] 体征检查到位
- [ ] 鉴别诊断全面
- [ ] 检查推荐合理
- [ ] 诊断依据充分
- [ ] 病情评估准确
- [ ] 建议明确可行

## 红牌警告

- 症状收集不完整
- 重要阴性体征未记录
- 鉴别诊断过于局限
- 过度依赖辅助检查
- 忽略常见病
- 诊断标准未核对
- 病情严重程度评估错误
- 紧急情况未识别
