# Pharmacy Management Skill

管理药房运营、药品库存，或医疗机构的药品供应链。

## 何时使用

- 药房运营管理
- 药品采购
- 库存管理
- 用药安全
- 合理用药监控
- 药品供应链

## 药房管理流程

```
1. 采购管理 → 2. 库存管理 → 3. 药品调配 → 4. 用药安全 → 5. 质量控制
```

### 1. 采购管理

**药品采购计划：**

```markdown
## 药品采购

### 采购方式
| 方式 | 适用 | 特点 |
|------|------|------|
| 集中采购 | 医保目录 | 价格低，周期长 |
| 挂网采购 | 省级平台 | 阳光采购 |
| 议价采购 | 非挂网药品 | 灵活 |
| 应急采购 | 短缺药品 | 快速 |

## 供应商资质

### 评估指标
| 指标 | 权重 | 评分 |
|------|------|------|
| 价格 | 30% | |
| 质量 | 30% | |

| 服务 | 20% | |
```
```

**库存预警：**

```markdown
## 库存预警

### 预警规则
| 类型 | 触发条件 | 响应 |
|------|----------|------|
| 库存不足 | <7天用量 | 立即采购 |
| 临期预警 | 距到期<6月 | 优先使用 |
| 积压预警 | >3月用量 | 停止采购 |
| 效期预警 | 距到期<3月 | 退货/处理 |
```

### 2. 库存管理

**药品分类管理：**

```markdown
## ABC分类管理

### A类（高价值）
| 特点 | 占比 | 管理方式 |
|------|------|----------|
| 金额>60% | 数量<10% | 精确管理，每日盘点 |

### B类（中等价值）
| 特点 | 占比 | 管理方式 |
|------|------|----------|
| 金额20-30% | 数量20-30% | 定期盘点，周度管理 |

### C类（低价值）
| 特点 | 占比 | 管理方式 |
|------|------|----------|
| 金额<20% | 数量>60% | 简化管理，月度盘点 |
```

**效期管理：**

```markdown
## 效期管理

### 先进先出原则
- [ ] 按生产日期/效期入库
- [ ] 先产先出
- [ ] 有效期近的放在前面

### 效期追踪
| 效期 | 剩余时间 | 处理方式 |
|------|----------|----------|
| 绿标 | >12月 | 正常销售 |
| 黄标 | 6-12月 | 优先使用 |
| 橙标 | 3-6月 | 监控使用 |
| 红标 | <3月 | 禁止销售 |

### 报损管理
```markdown
## 药品报损

### 报损条件
- [ ] 过期失效
- [ ] 破损污染
- [ ] 退回无法再销售

### 报损流程
1. 填写报损单
2. 原因说明
3. 质量部门确认
4. 领导审批
5. 实物销毁/退货

```
```

### 3. 药品调配

**处方审核：**

```markdown
## 处方审核

### 审核内容
| 审核项 | 要求 |
|--------|------|
| 处方格式 | 完整、规范 |
| 药品名称 | 通用名/商品名 |
| 剂量 | 成人/儿童/肝肾功能 |
| 给药途径 | 正确 |
| 用法用量 | 合理 |
| 配伍禁忌 | 无相互作用 |
| 皮试要求 | 已做/已注明 |

### 拦截级别
| 级别 | 类型 | 处理 |
|------|------|------|
| 严重 | 绝对禁忌 | 拦截+拦截 |
| 中度 | 需确认 | 提醒+确认 |
| 轻度 | 注意 | 提醒 |
```

**调配流程：**

```markdown
## 调配流程

### 标准流程
1. 接收处方
2. 审核处方
3. 调配药品
4. 核对药品
5. 用药交代
6. 发药

### 特殊药品管理
```markdown
## 特殊药品

### 麻醉药品
- [ ] 五专管理：专人/专柜/专锁/专账/专方
- [ ] 空安瓿回收
- [ ] 双人核对

### 精神药品
- [ ] 一类：同麻管理
- [ ] 二类：专柜存放

### 贵重药品
- [ ] 逐日盘点
- [ ] 冷链管理（如需）

### 冷链药品
- [ ] 2-8℃储存
- [ ] 全程温度记录
- [ ] 交接核查
```
```

### 4. 用药安全

**合理用药监控：**

```markdown
## 合理用药

### 监控指标
| 指标 | 计算 | 标准 |
|------|------|------|
| 处方合格率 | 合格处方/总处方 | >95% |
| 抗菌药使用率 | 抗菌药处方/总处方 | <50% |
| 注射剂使用率 | 注射处方/总处方 | <30% |
| 基本药物使用 | 基药金额/总金额 | >50% |

### 用药禁忌
```markdown
## 配伍禁忌

### 药理拮抗
- 抗生素+杀菌剂（部分情况）

### 物理相容
- 不宜混合的注射液

### 化学相容
- 酸碱中和
- 沉淀反应
```

### 不良反应监测：

```markdown
## ADR监测

### 报告流程
1. 发现不良反应
2. 初步评估
3. 填写报告表
4. 上报不良反应中心
5. 分析评价

### 报告要求
- [ ] 新的/严重不良反应：15日内报告
- [ ] 一般不良反应：季度报告
- [ ] 死亡病例：立即报告
```

## 红牌警告

- 假药/劣药入库
- 过期药品销售
- 处方审核缺失
- 特殊药品管理失控
- 库存账实不符
- 温控记录造假
- 人员无证上岗
- GSP严重违规
