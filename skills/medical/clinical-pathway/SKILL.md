# 临床路径设计 (Clinical Pathway Design)

## 触发条件
- 制定临床路径
- 诊疗流程标准化
- 医疗质量控制
- 成本效益分析

## 输入
- 疾病类型
- 诊疗现状
- 质控目标
- 资源约束

## 输出
- 临床路径文档
- 诊疗标准
- 变异管理
- 评估机制

## Prompt模板
```
请为[疾病/手术]设计临床路径。

背景：
- 疾病/手术：[类型]
- 诊疗现状：[现状]
- 质控目标：[目标]
- 资源约束：[约束]

请输出：
1. 临床路径目标和范围
2. 患者入组标准
3. 标准诊疗流程（时间节点）
4. 检查和治疗安排
5. 用药方案
6. 变异标准和处理
7. 出院标准和随访
8. 质量监控指标
9. 效果评估和改进
```

## 示例
### 示例1
- 输入：膝关节置换手术，需要缩短住院时间、控制感控
- 输出：临床路径，含术前检查、手术安排、康复计划、出院标准

## 注意事项
- 要循证依据
- 要考虑变异
- 要持续改进
