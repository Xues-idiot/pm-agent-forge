# 保险设计 (Insurance Design)

## 触发条件
- 保险产品设计
- 保障方案
- 定价策略
- 销售渠道

## 输入
- 产品类型
- 目标人群
- 保障需求
- 监管要求

## 输出
- 产品方案
- 保障设计
- 定价模型
- 推广策略

## Prompt模板
```
请设计[类型]保险产品。

背景：
- 产品类型：[类型]
- 目标人群：[人群]
- 保障需求：[需求]
- 监管要求：[要求]

请输出：
1. 产品定位
2. 保障范围设计
3. 保险条款
4. 定价模型
5. 赔付流程
6. 销售渠道
7. 核保规则
8. 再保安排
9. 合规审查
```

## 示例
### 示例1
- 输入：百万医疗险，面向年轻人，健康告知严格
- 输出：产品方案，含保障责任、费率表、健康告知

## 注意事项
- 要合规
- 要可持续
- 要客户友好
