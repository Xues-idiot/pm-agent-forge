# 临床试验 (Clinical Trial)

## 触发条件
- 试验设计
- 伦理审批
- 试验执行
- 数据管理

## 输入
- 试验阶段
- 适应症
- 目标人群
- 监管要求

## 输出
- 试验方案
- 伦理文件
- 执行计划
- 质量控制

## Prompt模板
```
请设计[产品]临床试验方案。

背景：
- 试验阶段：[阶段]
- 适应症：[适应症]
- 目标人群：[人群]
- 监管要求：[要求]

请输出：
1. 试验目的
2. 试验设计
3. 入选排除标准
4. 试验流程
5. 终点指标
6. 样本量计算
7. 统计分析
8. 伦理要求
9. 质量控制
```

## 示例
### 示例1
- 输入：肿瘤新药进入III期临床
- 输出：试验方案，含随机对照、样本量、终点指标

## 注意事项
- 要科学严谨
- 要伦理合规
- 要数据可靠
