# Medical Literature Search Skill

搜索医学文献，查找临床指南，研究治疗方法。

## 何时使用

- 查找临床指南
- 文献检索支持
- 治疗方案研究
- 最新医学进展
- 药物信息查询
- 疑难病例文献支持

## 文献检索流程

```
1. 明确检索目的 → 2. 选择检索资源 → 3. 构建检索策略 → 4. 筛选评估 → 5. 结果整理
```

### 1. 明确检索目的

**检索需求分类：**

```markdown
## 检索需求

### 需求类型
- [ ] 临床指南查询
- [ ] 治疗方案研究
- [ ] 诊断标准查找
- [ ] 药物信息查询
- [ ] 流行病学数据
- [ ] 最新研究进展
- [ ] 病例报道参考

### 检索问题框架 (PICO)
| 要素 | 内容 |
|------|------|
| P (Patient) | [患者特征/疾病] |
| I (Intervention) | [干预措施] |
| C (Comparison) | [对比/对照] |
| O (Outcome) | [结局指标] |

### 检索约束
- 语言：[中文/英文/不限]
- 时间范围：[近5年/近10年/不限]
- 文献类型：[指南/综述/RCT/病例]
```

### 2. 选择检索资源

**常用医学数据库：**

| 数据库 | 资源类型 | 适用场景 |
|--------|---------|---------|
| PubMed | 英文文献 | 基础研究、临床研究 |
| Embase | 英文文献 | 药理学、欧洲研究 |
| Cochrane Library | 系统评价 | 高质量证据 |
| CNKI | 中文文献 | 中文研究 |
| 万方 | 中文文献 | 中文研究 |
| UpToDate | 临床知识 | 临床决策 |
| DynaMed | 临床知识 | 临床决策 |
| ClinicalTrials | 临床试验 | 试验注册 |

**专业指南资源：**

```markdown
## 指南资源

### 国际指南
- [NICE Guidelines](https://www.nice.org.uk)
- [AHRQ Guidelines](https://www.ahrq.gov)
- [ESC Guidelines](https://www.escardio.org)
- [ADA Standards](https://diabetes.org)

### 中文指南
- [中华医学会指南](http://www.cma.org.cn)
- [中国临床指南协作网](http://www.guidelines.gov.cn)
```

### 3. 构建检索策略

**检索词选择：**

```markdown
## 检索词

### 核心检索词
- 主要疾病词：
- 主要治疗词：

### 同义词/扩展词
| 核心词 | 同义词 |
|--------|--------|
| | |

### 检索词组合
- AND：同时包含
- OR：包含任一
- NOT：排除

### 检索式示例
```search
(Diabetes OR "Type 2 Diabetes") AND
(Metformin OR "First-line therapy") AND
(Guideline OR "Treatment algorithm")
```
```

### 4. 筛选评估

**文献筛选流程：**

```markdown
## 文献筛选

### 初筛（标题+摘要）
| 文献 | 年份 | 相关性 | 主要发现 | 是否纳入 |
|------|------|---------|---------|---------|
| | | 高/中/低 | | 是/否/待定 |

### 复筛（全文）
| 文献 | 方法学质量 | 样本量 | 主要结论 | 证据等级 |
|------|----------|--------|---------|---------|
| | | | | |

### 证据等级评估
| 等级 | 研究类型 | 说明 |
|------|---------|------|
| I | 大样本RCT | 金标准 |
| II | 小样本RCT | 需谨慎解读 |
| III | 队列研究 | |
| IV | 病例对照 | |
| V | 专家意见 | |
```

**质量评估工具：**

```markdown
## 质量评估

### RCT 评估 (Cochrane RoB)
- [ ] 随机序列生成
- [ ] 分配隐藏
- [ ] 盲法
- [ ] 完整性
- [ ] 选择性报告
- [ ] 其他偏倚

### 观察性研究评估 (NOS)
- [ ] 暴露组和非暴露组可比
- [ ] 暴露测量
- [ ] 结局发生
- [ ] 研究控制
```

- [ ] 检索目的明确

## 红牌警告

- 检索词过于局限
- 数据库选择不当
- 忽略重要同义词
- 筛选标准不明确
- 证据质量未评估
- 引用信息过时
- 结论外推过度
- 利益冲突未披露
