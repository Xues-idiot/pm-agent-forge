# Medical Record Analysis Skill

分析病历资料，提取关键信息，总结患者健康状况。

## 何时使用

- 病历摘要生成
- 患者历史回顾
- 临床文档审核
- 医学影像报告解读辅助
- 出院小结撰写
- 病历质量审核

## 病历分析流程

```
1. 资料收集 → 2. 信息提取 → 3. 症状分析 → 4. 诊断核对 → 5. 总结输出
```

### 1. 资料收集

**病历资料清单：**

```markdown
## 病历资料

### 基础信息
- [ ] 患者基本信息
- [ ] 主诉
- [ ] 现病史
- [ ] 既往史
- [ ] 过敏史
- [ ] 个人史
- [ ] 家族史

### 检验检查
- [ ] 血常规
- [ ] 生化检查
- [ ] 影像报告
- [ ] 心电图
- [ ] 其他特殊检查

### 诊疗记录
- [ ] 入院记录
- [ ] 病程记录
- [ ] 手术记录
- [ ] 出院小结
```

## 病历分析检查清单

- [ ] 病历资料完整
- [ ] 关键信息提取准确
- [ ] 诊断有依据
- [ ] 用药记录清晰
- [ ] 病历书写规范
- [ ] 随访要求明确

## 红牌警告

- 病历关键信息遗漏
- 诊断缺乏依据
- 药物过敏未标注
- 检查结果异常未关注
- 病历存在逻辑矛盾
- 重要病史未记录
