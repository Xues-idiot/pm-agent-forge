# Patient Experience

患者体验，管理患者体验和服务质量。

## 何时使用

- 患者体验提升
- 就诊流程优化
- 服务设计
- 投诉处理
- 满意度管理

## 患者体验基础

### 体验维度

```markdown
## 患者体验

### 体验维度
| 维度 | 内容 |
|------|------|
| 就诊前 | 预约、挂号 |
| 就诊中 | 问诊、检查、治疗 |
| 就诊后 | 随访、康复 |

### 关键时刻
| 时刻 | 重要性 |
|------|--------|

| 问诊沟通 | 信任建立 |
| 检查治疗 | 核心价值 |
| 取药离院 | 最后印象 |
```

```markdown

|------|------|------|

### 沟通技巧
```markdown
## 沟通技巧

### 问诊技巧
- 开放性问题："您哪里不舒服？"
- 追问细节："这种感觉多久了？"
- 总结确认："我理解您是..."

### 解释技巧
- 用比喻解释
- 用图示说明
- 书面材料补充
```

## 投诉处理

### 投诉分类

```markdown
## 投诉分类

### 按类型
| 类型 | 示例 | 处理 |
|------|------|------|
| 服务态度 | 态度冷漠 | 道歉+改进 |
| 等候时间 | 等待过长 | 优化流程 |
| 医疗质量 | 治疗效果差 | 调查处理 |
| 收费问题 | 费用不透明 | 解释说明 |
| 隐私问题 | 信息泄露 | 严肃处理 |

### 按严重度
| 级别 | 定义 | 处理时效 |
|------|------|----------|
| P0 | 重大医疗事故 | 立即 |
| P1 | 严重投诉 | 24h |
| P2 | 一般投诉 | 48h |
| P3 | 建议反馈 | 72h |
```

### 处理流程

```markdown
## 投诉处理

### 处理步骤
1. 接收投诉：倾听记录
2. 初步响应：道歉+感谢
3. 调查核实：了解情况
4. 处理解决：提出方案
5. 回复患者：告知结果
6. 改进预防：总结复盘

### 处理话术
```markdown
## 处理话术

### 接待
"您好，非常抱歉给您带来不好的体验，请您详细说说情况，我会认真记录并跟进。"

### 回复
"您好，关于您反映的问题，我们经过调查了解到...，我们已经采取了...措施，感谢您的反馈。"
```
```
