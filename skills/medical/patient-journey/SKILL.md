# 患者旅程 (Patient Journey)

## 触发条件
- 患者体验优化
- 就医流程再造
- 触点管理
- 满意度提升

## 输入
- 病种类型
- 患者特征
- 就医场景
- 改进目标

## 输出
- 患者旅程地图
- 触点分析
- 优化方案
- 评估指标

## Prompt模板
```
请分析[患者类型]的患者旅程。

背景：
- 病种类型：[病种]
- 患者特征：[特征]
- 就医场景：[场景]
- 改进目标：[目标]

请输出：
1. 患者旅程地图（认知-就诊-治疗-随访）
2. 各阶段触点分析
3. 患者需求和痛点
4. 关键决策点
5. 优化机会
6. 改善方案
7. 效果评估
```

## 示例
### 示例1
- 输入：慢病患者，从确诊到长期用药，需要优化体验
- 输出：患者旅程，含挂号、确诊、取药、复诊全流程优化

## 注意事项
- 要以患者为中心
- 要关注全旅程
- 要持续改进
