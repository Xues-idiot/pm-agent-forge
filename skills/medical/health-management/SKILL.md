# Medical Health Management Skill

管理患者健康，制定健康管理计划，监测慢性病。

## 何时使用

- 慢病管理计划制定
- 健康体检报告解读
- 健康风险评估
- 用药管理
- 健康生活方式指导
- 随访计划制定

## 健康管理流程

```
1. 健康评估 → 2. 风险识别 → 3. 计划制定 → 4. 干预实施 → 5. 随访监测
```

### 5. 随访监测

**随访计划模板：**

```markdown

### 短期随访（1-3个月）
| 随访项目 | 时间点 | 重点关注 |
```

## 慢性病管理

### 高血压管理

```markdown
## 高血压管理

### 血压控制目标
| 人群 | 目标血压 |
|------|---------|
| 一般患者 | <140/90 mmHg |
| 糖尿病患者 | <130/80 mmHg |
| 老年人 | <150/90 mmHg |

### 管理要点
- [ ] 家庭血压监测
- [ ] 低盐饮食
- [ ] 规律运动
- [ ] 限酒
- [ ] 规律用药
- [ ] 定期随访
```

### 糖尿病管理

```markdown
## 糖尿病管理

### 控制目标
| 指标 | 目标值 |
|------|-------|
| 空腹血糖 | 4.4-7.0 mmol/L |
| 餐后2h血糖 | <10.0 mmol/L |
| HbA1c | <7.0% |
| 血压 | <130/80 mmHg |
| LDL | <2.6 mmol/L |

### 管理要点
- [ ] 饮食控制
- [ ] 运动治疗
- [ ] 药物治疗
- [ ] 血糖监测
- [ ] 健康教育
- [ ] 并发症筛查
```

## 健康管理检查清单

- [ ] 健康评估完整
- [ ] 风险识别准确
- [ ] 目标设定合理
- [ ] 计划具体可执行
- [ ] 用药管理规范
- [ ] 随访计划明确
- [ ] 患者依从性高

## 红牌警告

- 风险评估不准确
- 控制目标设定不合理
- 随访间隔过长
- 异常指标未及时处理
- 用药安全隐患未识别
- 患者教育不足
- 依从性问题未解决
- 转诊指征未把握
