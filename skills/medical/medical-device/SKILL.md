# 医疗器械 (Medical Device)

## 触发条件
- 器械注册
- 质量体系
- 临床评价
- 上市后监管

## 输入
- 器械类型
- 风险等级
- 目标市场
- 注册路径

## 输出
- 注册方案
- 质量体系
- 临床评价
- 监管策略

## Prompt模板
```
请制定[医疗器械]注册方案。

背景：
- 器械类型：[类型]
- 风险等级：[等级]
- 目标市场：[市场]
- 注册路径：[路径]

请输出：
1. 器械分类
2. 适用标准
3. 注册路径
4. 技术文档
5. 质量体系
6. 临床评价
7. 审评审批
8. 上市后监管
```

## 示例
### 示例1
- 输入：三类医疗器械，进入中国市场
- 输出：注册方案，含NMPA审批、临床试验、质量体系
