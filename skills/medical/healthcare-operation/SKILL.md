# 医疗运营 (Healthcare Operations)

## 触发条件
- 医院运营优化
- 就诊流程改进
- 效率提升
- 成本控制

## 输入
- 机构类型
- 运营现状
- 改进目标
- 资源情况

## 输出
- 运营优化方案
- 流程改进
- 效率指标
- 实施方案

## Prompt模板
```
请优化[医疗机构]的运营。

背景：
- 机构类型：[类型]
- 运营现状：[现状]
- 改进目标：[目标]
- 资源情况：[资源]

请输出：
1. 运营诊断
2. 就诊流程优化
3. 科室协同机制
4. 资源调度优化
5. 效率指标体系
6. 成本控制方案
7. 质量控制
8. 患者体验提升
9. 数字化支撑
```

## 示例
### 示例1
- 输入：三甲医院，日门诊量1万，需要缩短平均就诊时间
- 输出：优化方案，含预约分诊、智能排班、流程再造

## 注意事项
- 要以患者为中心
- 要平衡效率和质量
- 要持续改进
