# 健康促进 (Health Promotion)

## 触发条件
- 健康干预
- 疾病预防
- 健康管理
- 生活方式

## 输入
- 目标人群
- 健康问题
- 干预资源
- 政策支持

## 输出
- 干预方案
- 健康管理
- 效果评估
- 推广计划

## Prompt模板
```
请设计[人群]健康促进方案。

背景：
- 目标人群：[人群]
- 健康问题：[问题]
- 干预资源：[资源]
- 政策支持：[支持]

请输出：
1. 健康需求评估
2. 干预策略
3. 健康教育
4. 行为改变
5. 环境支持
6. 效果评估
7. 推广计划
```

## 示例
### 示例1
- 输入：企业员工，需要改善亚健康
- 输出：健康方案，含体检分析、健康干预、激励制度

## 注意事项
- 要科学依据
- 要个性化
- 要持续跟进
