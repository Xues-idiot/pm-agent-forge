# 医疗保险 (Health Insurance)

## 触发条件
- 医保政策
- 费用报销
- 医保管理
- 合规使用

## 输入
- 保险类型
- 覆盖范围
- 政策环境
- 管理目标

## 输出
- 管理方案
- 费用控制
- 合规要求
- 服务优化

## Prompt模板
```
请优化[机构]医疗保险管理。

背景：
- 保险类型：[类型]
- 覆盖范围：[范围]
- 政策环境：[环境]
- 管理目标：[目标]

请输出：
1. 医保政策分析
2. 费用管理方案
3. 审核机制
4. 合规要求
5. 信息化支撑
6. 服务优化
7. 风险控制
```

## 示例
### 示例1
- 输入：医院，需要控制医保基金使用
- 输出：管理方案，含费用审核、监控体系、合规控制

## 注意事项
- 要合规使用
- 要控费合理
- 要服务患者
