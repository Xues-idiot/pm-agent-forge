# User Stratification

私域用户分层体系，用RFM模型和综合分层框架优化运营。

## 何时使用

- 分析用户价值
- 设计用户分层策略
- 制定差异化运营方案
- 提升转化效率
- 优化资源配置

## 核心分层模型

### RFM模型

| 维度 | 说明 | 含义 |
|------|------|------|
| **R (Recency)** | 最近一次消费时间 | 越近越有价值 |
| **F (Frequency)** | 消费频率 | 越高越忠诚 |
| **M (Monetary)** | 消费金额 | 越高越有价值 |

### RFM评分标准

```
评分规则（1-5分）：
R: 1-7天=5, 8-14天=4, 15-30天=3, 31-60天=2, 60天+=1
F: 月均5次+=5, 3-4次=4, 2次=3, 1次=2, 0次=1
M: 月均1000+=5, 500-999=4, 200-499=3, 100-199=2, <100=1
```

### 综合分层矩阵

```
价值维度 × 活跃维度
```

|  | 高活跃 | 中活跃 | 低活跃/沉默 |
|--|--------|--------|------------|
| **高价值** | 钻石用户 | 黄金用户 | VIP沉默⚠️ |
| **中价值** | 黄金用户 | 白银用户 | 待激活 |
| **低价值** | 潜力用户 | 休眠用户 | 流失用户❌ |

## 各分层用户特征与策略

### 钻石用户
**特征**: 高消费 + 高活跃
**指标**: RFM>444, 活跃天数>20/月

**运营策略**:
- VIP专属服务、限量优先
- 会长福利、专属顾问
- 新品优先体验
- 生日/节日专属关怀

### VIP沉默
**特征**: 高消费 + 低活跃
**指标**: 消费高但30天无活跃

**运营策略**:
- 专属召回、1V1触达
- 专享优惠、限时福利
- 了解沉默原因
- 个性化召回方案

### 待激活
**特征**: 中消费 + 低活跃
**指标**: 消费一般，活跃不稳定

**运营策略**:
- 激活活动、限时福利
- 多次触达、教育引导
- 积分/权益诱惑
- 了解阻碍因素

### 流失用户
**特征**: 低消费 + 沉默
**指标**: 60天无消费、无活跃

**运营策略**:
- 低成本召回、问卷调研
- 大额优惠触发
- 放弃标记（节省资源）
- 偶尔触达维护品牌印象

## 分层运营策略模板

```markdown
## 分层运营方案

### 钻石用户运营
**目标**: 维护忠诚度，提升LTV
**触达方式**: 专属顾问+VIP群
**频率**: 每周1-2次
**内容**: 新品+专属权益+生日关怀

### VIP沉默召回
**目标**: 激活沉默高价值用户
**触达方式**: 1V1企微+专享券
**频率**: 连续3天触达
**内容**: 专属召回文案+限时优惠

### 待激活用户
**目标**: 提升活跃度和消费
**触达方式**: 群发+活动
**频率**: 每周2-3次
**内容**: 激活活动+积分任务

### 流失用户
**目标**: 低成本召回或标记放弃
**触达方式**: 大额优惠/问卷
**频率**: 每月1次
**内容**: 超级优惠/离因调研
```

## 红牌警告

- 所有用户用同一策略
- 从不分层运营
- 忽视VIP沉默用户
- 流失用户继续投入资源

## 好/坏对比

**好**：
```
分析发现10%用户贡献50%收入
→ 钻石用户用1V1专属服务
→ VIP沉默用定向召回
→ 低价值用户减少投入
→ ROI大幅提升
```

**坏**：
```
对所有用户发同样促销信息
→ 高价值用户觉得太频繁
→ 低价值用户转化率低
→ 资源浪费、效果差
```
