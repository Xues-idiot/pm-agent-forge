## 红牌警告

- 定位模糊
- 入口单一
- 体验差
- 转化路径长
- 不做用户召回
- 不看数据

## 好/坏对比

**好**：
```
小程序运营：
- 定位清晰
- 多入口布局
- 体验流畅
- 裂变有效
- 召回机制完善
→ 持续增长、高转化
```

**坏**：
```
小程序运营：
- 定位模糊
- 体验差
- 入口单一
- 没有裂变
- 不管留存
→ 增长停滞、用户流失
```
