### 权益设计原则

```markdown
## 权益设计原则

### 低等级会员
- 门槛低、易获得
- 基础优惠为主
- 引导升级钩子

### 中等级会员
- 明显差异感
- 服务升级
- 性价比最优

### 高等级会员
- 稀缺感和专属感
- 极致服务
- 社交荣誉
```

## 会员体系检查清单

- [ ] 等级门槛合理
- [ ] 权益差异化明显
- [ ] 积分体系可持续
- [ ] 升级路径清晰
- [ ] 技术实现可行
- [ ] 成本可控
- [ ] 运营SOP到位
- [ ] 数据监控完善

## 红牌警告

- 等级门槛过高过低
- 权益无差异化
- 积分通货膨胀
- 成本失控
- 系统漏洞被套利
- 只增不减（升降级）
- 用户无法感知价值
- 与运营脱节
