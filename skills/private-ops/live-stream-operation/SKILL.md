# Live Stream Operation

直播运营，策划直播活动和直播带货。

## 何时使用

- 直播活动策划
- 直播带货运营
- 主播培训管理
- 直播数据分析
- 直播团队协作

## 检查清单

- [ ] 选品分层合理
- [ ] 脚本设计完整
- [ ] 团队分工明确
- [ ] 话术准备充分
- [ ] 设备网络测试
- [ ] 数据监控就绪

## 红牌警告

- 选品没有梯度
- 话术太硬逼单
- 团队配合混乱
- 虚假刷单
- 虚假宣传

## 好/坏对比

**好**：
```
直播准备：
- 引流款+主推款+利润款分层
- 脚本时间精确到分钟
- 话术经过演练
- 团队配合默契
→ 直播GMV高、用户好评
```

**坏**：
```
直播准备：
- 随便选品，没有规划
- 想到什么说什么
- 团队各干各的
→ 直播翻车、转化低
```
