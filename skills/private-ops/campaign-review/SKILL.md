# Campaign Review Skill

复盘营销活动效果，分析活动成果，创建活动总结报告。

## 何时使用

- 活动复盘
- 效果分析
- ROI计算
- 经验沉淀
- 改进建议

## 复盘管理流程

```
1. 数据收集 → 2. 效果评估 → 3. 分析洞察 → 4. 经验沉淀 → 5. 改进计划
```

### ROI计算
```markdown
ROI = (总收入 - 总投入) / 总投入 × 100%

例：
投入 = 100万
收入 = 300万
ROI = (300-100)/100 = 200%
```
```
