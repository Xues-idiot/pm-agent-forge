# Fission Campaign Design Skill

策划裂变活动，设计邀请机制，执行增长黑客营销。

## 何时使用

- 用户增长活动
- 邀请裂变策划
- 病毒传播设计
- 红包/奖励活动
- 拼团/助力活动

## 裂变活动流程

```
1. 目标设定 → 2. 机制设计 → 3. 物料准备 → 4. 执行监控 → 5. 复盘优化
```

### 2. 机制设计

**裂变模型：**

```markdown
## 裂变模型

### 常见模型对比
| 模型 | 原理 | 优点 | 缺点 | 适用场景 |
|------|------|------|------|----------|
| 邀请有礼 | 邀请双方都得奖励 | 动力强 | 成本高 | 快速获客 |
| 拼团 | N人成团享优惠 | 社交属性强 | 成团率低 | 电商 |
| 助力 | 邀请助力解锁奖励 | 传播广 | 转化低 | 品牌传播 |
| 抽奖 | 邀请增加抽奖机会 | 趣味强 | 精准度低 | 活跃为主 |
| 任务系统 | 完成邀请任务得奖励 | 可控 | 参与门槛高 | 高价值用户 |

### 奖励设计
```markdown
## 奖励设计

### 邀请人奖励
| 成功邀请人数 | 奖励 | 累计奖励 |
|--------------|------|----------|
| 1人 | 5元优惠券 | 5元 |
| 3人 | 20元优惠券 | 25元 |
| 5人 | 50元优惠券+礼品 | 75元+礼品 |
| 10人 | 100元+专属称号 | 175元+称号 |

### 被邀请人奖励
| 奖励类型 | 内容 | 门槛 |
|----------|------|------|
| 新人礼包 | X元优惠券 | 注册即得 |
| 首单奖励 | 首单立减X元 | 首单 |
| 专属奖励 | 额外X元 | 首单满X元 |

### 奖励规则
- [ ] 每人邀请上限：X人
- [ ] 奖励发放时间：即时/次月
- [ ] 提现门槛：X元
- [ ] 风控规则：设备/账号限制
```

**裂变路径设计：**

```markdown
## 裂变路径

### 典型路径
```
用户A看到活动
    ↓
分享到朋友圈/群/好友
    ↓
用户B看到邀请
    ↓
用户B点击参与
    ↓
用户A获得奖励通知
    ↓
用户B完成首单（可选）
    ↓
用户A获得额外奖励（可选）
```

### 关键节点
| 节点 | 优化点 | 提升手段 |
|------|--------|----------|
| 分享点击 | 分享→点击 | 文案/配图 |
| 落地页转化 | 曝光→参与 | 落地页优化 |
| 被邀请转化 | 参与→注册 | 奖励力度 |
| 留存 | 注册→首单 | 后续运营 |
```

- 奖励预算失控
