# Private Traffic Acquisition Skill

获取私域流量，管理微信/社群渠道增长，优化引流策略。

## 何时使用

- 私域流量获取
- 渠道引流
- 裂变增长
- 跨平台导流
- 流量成本优化
- 私域用户沉淀

## 流量获取流程

```
1. 渠道分析 → 2. 引流设计 → 3. 执行落地 → 4. 数据监控 → 5. 优化迭代
```

### 2. 引流设计

**引流路径：**

```markdown
## 引流路径

### 典型路径
```markdown
## 公域→私域路径

公域平台（抖音/小红书）
    ↓ 内容触达
用户兴趣
    ↓ 主页/评论区
引流钩子（福利/资料）
    ↓ 点击
微信/社群
    ↓ 承接
私域沉淀（企微/社群）
```

### 引流钩子设计
| 钩子类型 | 适用场景 | 转化率 | 成本 |
|----------|----------|--------|------|
| 实物礼品 | 高客单 | 高 | 中 |
| 虚拟资料 | 低客单 | 中 | 低 |
| 优惠券 | 电商 | 高 | 中 |
| 体验装 | 美妆/食品 | 高 | 中 |
| 咨询/服务 | 服务行业 | 中 | 低 |
| 会员权益 | 高忠诚 | 中 | 低 |
```

**各平台引流策略：**

```markdown
## 各平台引流

### 抖音引流私域
```markdown
## 抖音→私域

### 方式1：评论区引流
- 评论区置顶：关注公众号/加我微信
- 小号引导：回复"666"获取

### 方式2：主页引流
- 主页简介：公众号名称
- 主页背景图：二维码
- 组件：微信/加群

### 方式3：直播间引流
- 直播口播：引流到社群
- 小程序：跳转微信
```

## 红牌警告

- 引流被平台限制/封号
- 引流成本过高
- 转化率过低
- 虚假流量
- 违规引流
- 私域承接不足
- 用户流失高
- 隐私合规风险
