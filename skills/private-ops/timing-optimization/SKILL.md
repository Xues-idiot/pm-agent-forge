## 人群-时机匹配

### 职场人群
**特征**: 白天工作忙，晚上有时间
**最佳时机**: 午间(12-13点)、晚间(19-21点)
**不适宜**: 上午9-12点、下午2-5点

### 自由职业
**特征**: 时间灵活，碎片化
**最佳时机**: 早间(8-10点)、下午(14-17点)
**不适宜**: 无特别限制

### 宝妈群体
**特征**: 上午/午间孩子上学/休息
**最佳时机**: 上午(9-11点)、午间(13-15点)
**不适宜**: 早上7-9点、下午3-5点

### 学生群体
**特征**: 课余时间，晚间活跃
**最佳时机**: 晚间(19-22点)、周末
**不适宜**: 上午8-12点(上课)

### 银发人群
**特征**: 早起早睡，习惯传统媒体
**最佳时机**: 早间(7-9点)、午间(11-13点)
**不适宜**: 晚间21点后

## 分层用户时机策略

```markdown
## 分层用户时机策略

### 高价值用户(钻石/VIP)
**策略**: 优先发送，保证触达
**最佳时机**: 晚间黄金时段 19-21点
**发送频率**: 可适当提高(但不骚扰)

### 中价值用户(黄金/白银)
**策略**: 常规运营，测试优化
**最佳时机**: 午间/晚间
**发送频率**: 适中

### 低价值用户(潜力/休眠)
**策略**: 控制成本，测试低成本时段
**最佳时机**: 午间(成本低)
**发送频率**: 较低

### 沉默用户(待激活/流失)
**策略**: 大规模、低成本召回
**最佳时机**: 周末/节假日
**发送频率**: 低
```

## 发送频率策略

### 频率分层

```markdown
## 发送频率策略

### 按用户价值
| 分层 | 日均上限 | 周上限 |
|------|----------|--------|
| 钻石用户 | 2-3条 | 10-15条 |
| 黄金用户 | 1-2条 | 7-10条 |
| 普通用户 | 1条 | 5-7条 |
| 沉默用户 | 1条/周 | 2-3条 |

### 按内容类型
| 内容类型 | 建议频率 |
|----------|----------|
| 促销优惠 | 每周1-2次 |
| 内容推送 | 每周2-3次 |
| 互动活动 | 每月2-4次 |
| 会员关怀 | 每月1-2次 |

### 频率控制原则
✓ 用户主动互动后48小时内不发送
✓ 连续3次无打开后降低频率
✓ 重大节日/活动可适当增加
✗ 同一天不超过3条
✗ 不在用户休息时段发送
```

## 红牌警告

- 所有用户同一时间发送
- 不考虑用户生活习惯
- 过度发送导致用户疲劳
- 夜间发送打扰用户
- 不监控发送效果

## 好/坏对比

**好**：
```
分析发现:
- 职场用户19-21点打开率最高
- 宝妈群体9-11点响应最好
→ 分层分时段发送
→ 打开率提升30%
```

**坏**：
```
所有用户统一晚上8点发送
→ 用户疲劳
→ 打开率持续下降
→ 被投诉
```
