# Private Domain Conversion Optimization Skill

优化私域转化漏斗，提升转化率，管理转化实验。

## 何时使用

- 私域转化分析
- 转化率提升
- 转化漏斗优化
- 用户激活
- 付费转化
- 复购促进

## 私域转化流程

```
1. 漏斗诊断 → 2. 用户激活 → 3. 信任建立 → 4. 转化收割 → 5. 复购运营
```

### 1. 漏斗诊断

**私域转化漏斗：**

```markdown
## 私域转化漏斗

### 标准漏斗
```markdown
私域用户 → 活跃用户 → 意向用户 → 付费用户 → 复购用户
   100%          30%          15%          5%          3%
```

### 各环节定义
| 阶段 | 定义 | 关键动作 |
|------|------|----------|
| 私域用户 | 添加企微/进群 | 完成添加 |
| 活跃用户 | 7天内有互动 | 浏览/点击/发言 |
| 意向用户 | 表达购买意向 | 询价/收藏/加购 |
| 付费用户 | 完成首次购买 | 支付成功 |
| 复购用户 | 30天内复购 | 再次支付 |

### 转化率健康值
| 环节 | 健康值 | 预警值 |
|------|--------|--------|
| 添加→活跃 | >30% | <20% |
| 活跃→意向 | >15% | <10% |
| 意向→付费 | >30% | <20% |
| 付费→复购 | >50% | <30% |
```

**转化诊断：**

```markdown
## 转化诊断

### 问题识别
| 问题环节 | 表现 | 可能原因 |
|----------|------|----------|
| 添加→活跃 | 流失>70% | 无欢迎引导 |
| 活跃→意向 | 流失>85% | 内容无价值 |
| 意向→付费 | 流失>70% | 价格/信任问题 |
| 付费→复购 | 流失>50% | 产品/服务差 |

### 流失用户画像
```markdown
## 流失用户特征

### 添加后沉默用户
- 渠道：[主要来源]
- 特征：[用户标签]
- 原因：[可能原因]

### 咨询后未购买
- 流失环节：[哪个环节]
- 流失原因：[可能原因]
```
```

✅ [价值1]

### 3. 信任建立

**信任体系：**

```markdown
## 信任建立

### 信任要素
| 要素 | 具体动作 | 效果 |
|------|----------|------|
| 专业形象 | 朋友圈专业内容 | 展示专业 |
| 案例背书 | 成功案例分享 | 增强信任 |
| 用户评价 | 真实反馈展示 | 社会证明 |
| 及时响应 | 快速回复消息 | 安全感 |

### 内容策略
```markdown
## 信任内容

### 朋友圈内容比例
- 专业内容：40%
- 案例分享：25%
- 用户好评：20%
- 个人生活：15%

### 发布节奏
| 时间 | 内容类型 |
|------|----------|
| 早7-8点 | 早安问候+金句 |
| 上午10点 | 干货知识 |
| 下午2点 | 案例分享 |
| 下午5点 | 用户好评 |
| 晚8点 | 限时活动 |
```
```

### 4. 转化收割

**转化策略：**

```markdown
## 转化策略

### 转化时机
| 时机 | 触发条件 | 转化方式 |
|------|----------|----------|
| 新人转化 | 添加3天内 | 新人专享价 |
| 活动转化 | 限时活动 | 紧迫感+优惠 |
| 场景转化 | 用户表达需求 | 个性化推荐 |
| 节日转化 | 节日/生日 | 专属优惠 |

### 转化话术
```markdown
## 转化话术

### 直接转化
"我们的[产品]特别适合你，现在有限时优惠，点击链接即可购买：[链接]"

### 场景转化
"看到你关注[产品]，和[你的需求]很匹配，我们刚好有活动，帮你争取到了[优惠]"

### 限时转化
"这个优惠今晚12点截止，需要的话抓紧哦~"
```

### 转化承接
- [ ] 转化路径清晰（<3步）
- [ ] 支付方式便捷
- [ ] 优惠叠加说明
- [ ] 购买保障
```

## 红牌警告

- 转化链路过长
- 虚假转化数据
- 过度营销扰民
- 产品/服务不过关
- 承诺不兑现
- 用户隐私泄露
- 违反平台规则
- 转化成本高于LTV
