# Member Lifecycle

会员生命周期管理，设计会员旅程、优化参与度和留存。

## 何时使用

- 设计会员体系
- 优化会员旅程
- 提升会员留存
- 设计会员升级/降级策略
- 分析会员流失

## 会员生命周期阶段

```
引入期 → 成长期 → 成熟期 → 休眠期 → 流失期
```

### 各阶段特征与策略

```markdown
## 会员生命周期

### 引入期 (0-7天)
**特征**：新加入，活跃度最高但不稳定
**策略**：快速激活，建立认知

| 动作 | 内容 | 时间 |
|------|------|------|
| 欢迎 | 欢迎语+权益说明 | 即时 |
| 引导 | 功能介绍+新手任务 | 1-3天 |
| 首单激励 | 首单优惠 | 3-7天 |

### 成长期 (7-30天)
**特征**：开始使用，活跃度上升
**策略**：培养习惯，提供价值

| 动作 | 内容 | 频率 |
|------|------|------|
| 价值推送 | 个性化内容 | 每日 |
| 任务激励 | 完成任务获积分 | 每周 |
| 社群引导 | 加入社群 | 1次 |

### 成熟期 (30-180天)
**特征**：高频使用，稳定性强
**策略**：提升LTV，促进复购

| 动作 | 内容 | 频率 |
|------|------|------|
| 专属权益 | VIP专享 | 持续 |
| 升级激励 | 达到下一级的好处 | 适时 |
| 复购提醒 | 补充购买 | 按周期 |

### 休眠期 (180-365天)
**特征**：活跃度下降，有流失风险
**策略**：定向激活，防止流失

| 动作 | 内容 | 频率 |
|------|------|------|
| 沉默提醒 | 使用提醒 | 每周 |
| 召回激励 | 大额优惠 | 2周/次 |
| 调研了解 | 流失原因 | 适时 |

### 流失期 (365天+)
**特征**：长期不活跃，几乎无价值
**策略**：低成本维护或放弃

| 动作 | 内容 | 频率 |
|------|------|------|
| 低成本触达 | 节日问候 | 偶尔 |
| 超级召回 | 极限优惠 | 大促时 |
| 流失登记 | 记录流失原因 | 永久 |
```

## 会员旅程设计

### 会员旅程模板

```markdown
## 会员旅程

### 阶段1：认知
- 触点：公众号、广告、朋友推荐
- 行为：了解、比较
- 痛点：信任度不够
- 机会：展示真实用户评价

### 阶段2：加入
- 触点：注册页、落地页
- 行为：注册、领权益
- 痛点：流程复杂
- 机会：简化流程、即时权益

### 阶段3：激活
- 触点：新手引导、欢迎语
- 行为：首次使用核心功能
- 痛点：不知道有什么用
- 机会：引导完成关键动作

### 阶段4：转化
- 触点：商品页、活动页
- 行为：首次购买
- 痛点：价格顾虑
- 机会：新人专享价

### 阶段5：留存
- 触点：推送、群发、社群
- 行为：持续使用、复购
- 痛点：价值感知弱
- 机会：定期价值提醒
```

### 关键动作定义

```markdown
## 关键动作 (Key Actions)

### Aha Moment
用户第一次感受到产品价值的时刻

| 产品类型 | Aha Moment |
|----------|-----------|
| 电商 | 收到货/完成首单 |
| 内容 | 找到想要的内容 |
| 工具 | 第一次解决问题 |
| 社交 | 第一次互动 |

### 关键行为指标
| 行为 | 定义 | 健康值 |
|------|------|--------|
| 激活率 | 完成Aha Moment的用户占比 | >40% |
| 首单时间 | 注册到首单的平均天数 | <7天 |
| 7日留存 | 7天内回访的用户占比 | >30% |
```

### 升级/降级规则

```markdown
## 升级规则

### 年度计算
- 升级：年度消费达到门槛即升级
- 降级：年度消费低于门槛则降级

### 权益生效
- 升级：即时生效，立即享有新权益
- 降级：到期后降级，给1个月缓冲期

### 升降级示例
| 当前等级 | 年度消费 | 新等级 | 生效时间 |
|----------|----------|--------|----------|
| 银卡 | ¥3000 | 金卡 | 即时 |
| 金卡 | ¥800 | 银卡 | 缓冲后 |
```

## 留存分析

### 留存曲线

```markdown
## 留存曲线分析

### 留存率表格
| 时间 | 新用户留存率 | 活跃用户留存率 |
|------|-------------|---------------|
| 次日 | 40% | 80% |
| 7日 | 20% | 60% |
| 30日 | 10% | 40% |
| 90日 | 5% | 25% |

### 留存问题诊断
- 次日留存低 → 激活流程有问题
- 7日留存低 → 价值不够明显
- 30日留存低 → 产品粘性不够
```

### 流失预警

```markdown
## 流失预警

### 预警信号
| 阶段 | 信号 | 应对 |
|------|------|------|
| 轻度流失 | 30天未活跃 | 普通推送 |
| 中度流失 | 60天未活跃 | 优惠召回 |
| 重度流失 | 90天未活跃 | 大额激励 |
| 流失 | 180天未活跃 | 放弃标记 |

### 干预策略
- 推送频率异常下降
- 使用功能突然减少
- 客服咨询减少
→ 触发流失预警
```

## 检查清单

- [ ] 生命周期各阶段策略明确
- [ ] 关键动作清晰定义
- [ ] 权益体系有差异化
- [ ] 升级降级规则透明
- [ ] 流失预警机制建立

## 红牌警告

- 所有会员同一策略
- 没有Aha Moment设计
- 权益无差异化
- 流失不干预

## 好/坏对比

**好**：
```
新会员加入后：
1. 即时发放欢迎权益
2. 引导完成关键动作
3. 7天内多次触达
4. 促进首单转化
→ 提升激活率
```

**坏**：
```
新会员加入后：
没有引导、没有激励
等用户自己发现价值
→ 激活率低、流失率高
```
