# Community Operation Skill

运营社群，管理群互动，构建私域用户粘性。

## 何时使用

- 社群搭建
- 用户活跃度提升
- 社群内容运营
- 社群问题处理
- 社群转化

## 社群运营流程

```
1. 社群搭建 → 2. 内容规划 → 3. 互动管理 → 4. 活跃提升 → 5. 转化变现
```

### 5. 转化变现

**转化路径：**

```markdown
## 转化设计

### 转化漏斗
```
社群曝光 → 入群 → 活跃 → 信任 → 购买 → 复购
   ↓        ↓      ↓      ↓      ↓      ↓
  100%    60%    30%    20%    10%    5%
```

### 转化策略
| 阶段 | 转化方式 | 触发条件 |
|------|----------|----------|
| 信任建立 | 专业知识输出 | 入群3天内 |
| 需求激发 | 案例分享 | 有需求迹象 |
| 购买引导 | 限时优惠 | 信任度达标 |
| 复购促进 | 会员专享 | 首次购买后 |

### 转化话术
```markdown
## 转化话术

### 产品推荐
"应大家要求，这次团购来了！
[产品名] 限时优惠XX元
比平时便宜XX元，需要的扣1"

### 限时促销
"这波福利只到今晚12点
过了就没了，需要的抓紧~
我私聊发你购买链接"

### 复购引导
"上次买过的老客户
这次会员专属价
私信"复购"获取专属链接"
```

## 红牌警告

- 社群无人管理变广告群
- 只发广告不发价值
- 忽视用户问题
- 频繁刷屏打扰用户
- 承诺不兑现
- 转化过于频繁
- 社群负面舆情失控
- 群规朝令夕改
