# Activity Operation

## 何时使用

- 活动策划
- 活动执行
- 活动推广
- 活动复盘
- 活动迭代

## 检查清单

- [ ] 目标明确
- [ ] 规则清晰
- [ ] 奖励有吸引力
- [ ] 执行准备充分
- [ ] 监控到位
- [ ] 复盘总结

## 红牌警告

- 目标不明确
- 规则复杂难懂
- 奖励无吸引力
- 执行准备不足
- 监控不到位
- 不做复盘

## 好/坏对比

**好**：
```
活动运营：
- 目标清晰
- 规则简单
- 奖励有力
- 执行到位
- 复盘总结
→ 达成目标、用户满意
```

**坏**：
```
活动运营：
- 目标模糊
- 规则复杂
- 奖励鸡肋
- 执行混乱
- 从不复盘
→ 效果差、浪费资源
```
