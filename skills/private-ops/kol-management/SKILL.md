# KOL Management Skill

管理KOL合作，达人商务合作，关键意见领袖关系维护。

## 何时使用

- KOL筛选评估
- 合作方案制定
- KOL关系维护
- 合作效果分析
- 达人矩阵建设

## KOL管理流程

```
1. KOL筛选 → 2. 合作洽谈 → 3. 合作执行 → 4. 效果评估 → 5. 关系维护
```

## KOL管理检查清单

- [ ] KOL评估维度全面
- [ ] 报价合理有依据
- [ ] 合同条款清晰
- [ ] 执行节点明确
- [ ] 内容审核到位
- [ ] 数据追踪完整
- [ ] 效果评估客观
- [ ] 关系维护持续

## 红牌警告

- KOL数据造假
- 合同条款模糊
- 承诺不兑现
- 沟通响应慢
- 效果数据不透明
- 竞品同台
- 法律风险（虚假宣传）
- 过度依赖单一KOL
