# Moments Marketing

关键时点营销，捕捉用户关键时刻和触发式营销。

## 何时使用

- 设计用户关键时刻触达
- 规划触发式营销
- 优化用户生命周期触点
- 提升用户活跃度
- 促进转化和复购

## 关键时刻类型

### 用户行为时刻


## 红牌警告

- 过度打扰用户
- 触发时机错误
- 内容不相关
- 优惠过于频繁
- 没有用户授权

## 好/坏对比

**好**：
```
时刻营销：
- 用户浏览后及时降价提醒
- 内容相关、文案有吸引力
- 触发时机恰当
→ 召回成功，转化率高
```

**坏**：
```
时刻营销：
- 用户刚看完又发同样内容
- 一天发5条打扰
- 内容和用户无关
→ 用户反感，取关
```
