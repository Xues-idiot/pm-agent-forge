# User Lifecycle

用户生命周期，设计成员旅程和优化用户阶段。

## 何时使用

- 用户生命周期设计
- 阶段转化优化
- 用户旅程梳理
- 生命周期营销
- 会员阶段管理

## 用户生命周期

### 生命周期阶段

```markdown
## 用户生命周期

### 五阶段模型
```
认知 → 了解 → 体验 → 付费 → 忠诚
  ↓       ↓       ↓      ↓     ↓
 获取    激活    转化   变现   留存/推荐
```

### 各阶段定义
| 阶段 | 定义 | 核心目标 |
|------|------|----------|
| 认知期 | 知道品牌 | 获取关注 |
| 了解期 | 感兴趣 | 产生意向 |
| 体验期 | 尝试使用 | 完成首用 |
| 付费期 | 首次购买 | 完成转化 |
| 忠诚期 | 持续使用 | 复购+推荐 |

### 指标定义
```markdown
## 生命周期指标

| 阶段 | 核心指标 | 健康值 |
|------|----------|--------|
| 认知 | 曝光量、加粉率 | >5% |
| 了解 | 内容互动率 | >10% |
| 体验 | 首用完成率 | >30% |
| 付费 | 首购转化率 | >5% |
| 忠诚 | 复购率、NPS | >40% |
```
```

## 用户分群

### 按生命周期分

## 阶段转化

### 转化路径

```markdown
## 转化路径

### 典型路径
```
曝光 → 点击 → 注册 → 首次使用 → 首次付费 → 复购
```

### 各环节转化率
| 环节 | 转化率 | 优化空间 |
|------|--------|----------|
| 曝光→点击 | 3-5% | CTR优化 |
| 点击→注册 | 5-10% | 注册流程 |
| 注册→首用 | 30-50% | 引导体验 |
| 首用→首购 | 5-15% | 转化促进 |
| 首购→复购 | 20-30% | 留存运营 |

### 转化漏斗
```markdown
## 转化漏斗

          曝光(1000)
             ↓
          点击(50)    5%
             ↓
          注册(5)     10%
             ↓
         首用(2)      40%
             ↓
         首购(0.2)    10%
```
```

### 转化优化

```markdown
## 转化优化

### 关键节点
| 节点 | 问题 | 优化策略 |
|------|------|----------|
| 注册 | 门槛高 | 简化流程 |
| 首用 | 不知道怎么用 | 引导教程 |
| 首购 | 决策难 | 优惠激励 |
| 复购 | 没有理由 | 会员权益 |

- 简化流程
- 减少步骤
- 默认选项

- 消息提醒
- 场景触发
- 个性化推送

| 时刻 | 定义 | 运营重点 |

- 临门一脚优惠

- 信任背书

### 回忆时刻（口碑传播）
- 邀请评价
- 分享激励

## 检查清单

- [ ] 生命周期阶段清晰定义
- [ ] 用户分群合理
- [ ] 转化路径优化
- [ ] 用户旅程完整
- [ ] 关键时刻策略明确
- [ ] 自动触达机制建立

## 红牌警告

- 生命周期定义模糊
- 阶段转化没有监测
- 只关注获取忽视其他阶段
- 用户旅程缺失
- 过度营销打扰

## 好/坏对比

**好**：
```
生命周期运营：
- 阶段定义清晰
- 各阶段策略明确
- 转化路径优化
- 用户旅程完整
- 自动化触达
→ 转化率高、留存好
```

**坏**：
```
生命周期运营：
- 阶段定义模糊
- 只管获取不管留存
- 没有转化监测
- 用户旅程缺失
- 过度打扰
→ 流失高、价值低
```
