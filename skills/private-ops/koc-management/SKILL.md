# KOC Management

KOC（关键意见消费者）管理，培养草根社群拥护者。

## 何时使用

- KOC招募和管理
- 制定KOC激励计划
- 设计分销机制
- 运营KOC社群
- 评估KOC效果

## KOC vs KOL

```markdown
## KOC vs KOL

| 维度 | KOL | KOC |
|------|-----|-----|
| 粉丝量 | 10万+ | 1000-10万 |
| 影响力 | 大众认知 | 小众信任 |
| 商业化 | 付费合作 | 佣金合作 |
| 内容风格 | 专业推广 | 真实分享 |
| 适用场景 | 品牌曝光 | 种草转化 |
```

### 招募标准

```markdown
## KOC筛选标准

### 基本条件
- [ ] 是产品真实用户
- [ ] 活跃在目标社群
- [ ] 愿意分享真实体验
- [ ] 无负面记录

### 加分项
- [ ] 有一定粉丝基础
- [ ] 内容质量高
- [ ] 互动率高
- [ ] 与品牌调性相符

### KOC分层
| 等级 | 粉丝/影响力 | 要求 | 权益 |
|------|-------------|------|------|
| 试用KOC | 0 | 申请通过 | 试用产品 | 佣金比例 |
| 正式KOC | 1000+ | 内容≥3篇 | 产品+佣金 | 10% |
| 核心KOC | 5000+ | 月产出≥4篇 | 专属权益 | 15% |
| 超级KOC | 10000+ | 专属合作 | 全方位支持 | 20% |
```

## 红牌警告

- KOC全是刷量来的
- 激励太低无动力
- 无差异化权益
- 只追求数量不管质量

## 好/坏对比

**好**：
```
KOC计划：
- 招募真实用户
- 分层权益激励
- 专业内容培训
- 定期社群互动
→ 真实种草、转化稳定
```

**坏**：
```
KOC计划：
- 买来的粉丝
- 统一佣金无差异
- 没有运营支持
- 只看数量不管质量
→ 虚假繁荣、转化差
```
