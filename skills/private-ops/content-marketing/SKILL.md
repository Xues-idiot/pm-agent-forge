# Content Marketing

私域内容营销，规划内容策略、制定内容日历。

## 何时使用

- 制定月度内容计划
- 内容选题策划
- 创作内容模板
- 内容分发策略
- 内容效果分析

## 内容策略框架

### 内容金字塔

```markdown
## 内容金字塔

### 顶层：品牌内容 (10%)
- 品牌故事、创始人故事
- 使命愿景价值观
- 目的：建立品牌认知

### 中层：教育内容 (30%)
- 产品教程、使用技巧
- 行业知识、解决方案
- 目的：建立专业信任

### 底层：促销内容 (60%)
- 产品介绍、活动信息
- 用户评价、案例展示
- 目的：直接促进转化
```

### 内容支柱

```markdown
## 内容支柱

### 确定内容支柱的步骤
1. 分析目标用户需求
2. 结合产品核心价值
3. 差异化于竞品
4. 确定3-5个支柱

### 内容支柱示例
| 支柱 | 内容方向 | 适合的品牌 |
|------|----------|-----------|
| 产品教程 | 使用技巧、攻略 | 工具类产品 |
| 行业洞察 | 数据报告、趋势 | B2B服务 |
| 用户故事 | 真实案例、评价 | 消费品 |
| 生活方式 | 场景植入、理念 | 生活方式品牌 |
```

## 检查清单

- [ ] 内容支柱明确
- [ ] 月度日历制定
- [ ] 每周节奏稳定
- [ ] 选题来源多样
- [ ] 内容模板标准化
- [ ] 数据复盘机制

## 红牌警告

- 全部是促销内容
- 没有用户视角内容
- 复制粘贴不调整
- 发了不追踪效果

## 好/坏对比

**好**：
```
内容策略：
- 干货60% + 促销40%
- 用户视角真实分享
- 有节日借势规划
- 定期数据分析优化
→ 用户信任、转化稳定
```

**坏**：
```
内容策略：
- 100%促销
- 自嗨式宣传
- 没有用户内容
- 发完不分析
→ 用户疲劳、信任流失
```
