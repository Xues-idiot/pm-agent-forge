# Intelligent QA Skill

开展客服智能质检，分析对话质量，自动化服务审计。

## 何时使用

- 客服质量检测
- 智能质检系统
- 服务审计自动化
- 客服培训需求识别
- 服务问题诊断
- 质检效率提升

## 智能质检流程

```
1. 质检体系设计 → 2. 质检规则配置 → 3. 智能检测 → 4. 问题分析 → 5. 改进闭环
```

### 1. 质检体系设计

**质检维度：**

```markdown
## 质检维度

### 服务态度（20分）
| 检查项 | 扣分标准 |
|--------|----------|
| 礼貌用语 | 未使用"您好/请问"等 -2分 |
| 态度友好 | 冷淡/不耐烦 -3分 |
| 耐心程度 | 催促/不耐烦 -2分 |
| 情绪稳定 | 情绪化表达 -5分 |

### 服务规范（30分）
| 检查项 | 扣分标准 |
|--------|----------|
| 称呼规范 | 未正确称呼客户 -1分 |
| 响应规范 | 响应超时 -2分 |
| 话术规范 | 未使用标准话术 -2分 |
| 操作规范 | 系统操作错误 -3分 |

### 问题解决（40分）
| 检查项 | 扣分标准 |
|--------|----------|
| 问题理解 | 未正确理解问题 -5分 |
| 方案提供 | 未提供有效方案 -5分 |
| 解决率 | 未解决问题 -10分 |
| 升级处理 | 应升级未升级 -5分 |

### 合规要求（10分）
| 检查项 | 扣分标准 |
|--------|----------|
| 隐私保护 | 泄露客户信息 -10分 |
| 敏感词 | 使用违规话术 -5分 |
| 虚假承诺 | 承诺未兑现 -5分 |
```

**质检等级：**

```markdown
## 质检等级

| 等级 | 分数 | 标准 | 占比目标 |
|------|------|------|----------|
| A（优秀） | 95-100 | 全面超越预期 | 20% |
| B（良好） | 85-94 | 全面达到预期 | 60% |
| C（合格） | 70-84 | 基本达到预期 | 15% |
| D（不合格） | <70 | 未达到预期 | <5% |
```
