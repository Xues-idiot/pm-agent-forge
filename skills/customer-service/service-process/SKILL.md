# 服务流程设计 (Service Process Design)

## 触发条件
- 设计客服流程
- 优化服务流程
- 跨部门流程协同
- 服务标准化

## 输入
- 服务场景
- 用户需求
- 现有流程
- 部门协作

## 输出
- 流程设计文档
- SOP手册
- 流程图
- 异常处理机制

## Prompt模板
```
请为[服务场景]设计客服流程。

背景：
- 服务类型：[类型]
- 用户需求：[需求]
- 涉及部门：[部门]
- 服务目标：[目标]

请输出：
1. 流程总体设计（主流程+分支流程）
2. 各环节标准动作
3. 部门协作机制
4. 异常处理流程
5. 升级机制
6. 流程KPI（时效/质量/满意度）
7. 流程优化触发条件
```

## 示例
### 示例1
- 输入：电商退货退款流程，涉及客服/仓库/财务
- 输出：退货退款SOP，含退款审核/仓库收货/财务退款全流程

## 注意事项
- 流程要简单高效
- 要明确责任边界
- 要有异常处理机制
