# 服务指标体系 (Service Metrics System)

## 触发条件
- 设计客服KPI体系
- 服务质量评估
- 客服绩效管理
- 服务监控看板

## 输入
- 服务模式
- 业务目标
- 现有指标
- 管理诉求

## 输出
- 指标体系文档
- 指标定义说明
- 监控看板设计
- 改进闭环机制

## Prompt模板
```
请为[客服团队]设计服务指标体系。

背景：
- 客服规模：[规模]
- 服务渠道：[渠道]
- 业务目标：[目标]
- 管理诉求：[诉求]

请输出：
1. 指标体系设计（数量/质量/效率/满意度）
2. 各指标定义和计算方式
3. 指标目标值设定
4. 监控看板设计
5. 数据采集方案
6. 改进闭环机制
7. 考核权重建议
```

## 示例
### 示例1
- 输入：电商客服团队，100人，关心响应速度和解决率
- 输出：客服指标体系，含响应速度/解决率/满意度/成本四大维度

## 注意事项
- 指标要可衡量
- 要平衡短期和长期
- 要与业务目标对齐
