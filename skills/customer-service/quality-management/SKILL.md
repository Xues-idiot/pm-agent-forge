# 服务质量管理 (Service Quality Management)

## 触发条件
- 服务质量体系建设
- 服务标准制定
- 服务质量监控
- 质量持续改进

## 输入
- 服务模式
- 服务规模
- 质量现状
- 改进目标

## 输出
- 质量管理体系
- 服务标准
- 监控方案
- 改进计划

## Prompt模板
```
请为[客服团队]设计服务质量管理体系。

背景：
- 服务模式：[模式]
- 服务规模：[规模]
- 质量现状：[现状]
- 改进目标：[目标]

请输出：
1. 服务质量目标和原则
2. 服务标准体系（响应速度/解决质量/服务态度）
3. 质检方案（抽检比例/质检维度/评分标准）
4. 服务质量监控看板
5. 质量问题分析和改进机制
6. 质量与绩效挂钩方案
7. 培训和能力提升计划
8. 质量文化建设
```

## 示例
### 示例1
- 输入：电商客服团队200人，当前满意度85%，目标90%
- 输出：质量管理体系，含智能质检、改进闭环、激励方案

## 注意事项
- 标准要可衡量
- 监控要实时
- 改进要闭环
