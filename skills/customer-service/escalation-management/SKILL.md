# Escalation Management Skill

处理客户投诉，管理升级案例，解决关键服务故障。

## 何时使用

- 客户投诉升级
- 重大服务故障
- VIP客户问题
- 媒体曝光风险
- 法律诉讼预警

## 升级处理流程

```
1. 接收升级 → 2. 情况评估 → 3. 方案制定 → 4. 客户沟通 → 5. 跟进闭环
```

## 红牌警告

- 未及时响应导致升级
- 承诺无法兑现
- 补偿超出授权
- 忽视客户核心诉求
- 内部推诿踢皮球
- 未经审批私下承诺
- 问题反复复发
- 媒体曝光未预案
