# 智能客服设计 (Intelligent Service Design)

## 触发条件
- 设计智能客服系统
- 规划智能客服能力
- 人机协作设计
- 智能化转型

## 输入
- 业务场景
- 常见问题
- 人工客服能力
- 技术限制

## 输出
- 智能客服设计方案
- 人机协作方案
- 知识库规划
- 效果评估指标

## Prompt模板
```
请为[业务/产品]设计智能客服方案。

背景：
- 业务类型：[类型]
- 常见问题：[问题列表]
- 人工能力：[能力现状]
- 技术约束：[约束]

请输出：
1. 智能客服定位（辅助/替代/增强）
2. 核心能力设计（FAQ/意图识别/多轮对话/知识推荐）
3. 人机协作方案（机器人→人工边界/协作模式）
4. 知识库架构设计
5. 训练数据规划
6. 效果评估指标
7. 分阶段实施计划
```

## 示例
### 示例1
- 输入：在线教育平台，常见问题有课程咨询/退款申请/技术问题
- 输出：智能客服方案，含课程咨询机器人+人机协作流程

## 注意事项
- 智能客服不是万能的
- 要设计好人工接管机制
- 知识库要持续运营
