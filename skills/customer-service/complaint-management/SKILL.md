# 投诉管理 (Complaint Management)

## 触发条件
- 处理重大投诉
- 设计投诉流程
- 投诉预防机制
- 投诉复盘改进

## 输入
- 投诉类型
- 投诉量级
- 影响程度
- 现有流程

## 输出
- 投诉处理方案
- 预防机制
- 升级策略
- 复盘模板

## Prompt模板
```
请设计[业务]的投诉管理体系。

背景：
- 业务类型：[类型]
- 投诉类型：[类型]
- 月投诉量：[量]
- 现有处理方式：[方式]

请输出：
1. 投诉分类分级标准
2. 处理流程（受理-调查-处理-回复-回访）
3. 各级别处理权限和升级机制
4. 重大投诉应对策略
5. 投诉预防机制
6. 投诉数据分析维度
7. 典型案例复盘模板
```

## 示例
### 示例1
- 输入：金融产品，投诉类型主要是服务和费率，用户量100万
- 输出：投诉管理体系，含分级标准、快速响应、预防机制

## 注意事项
- 处理要及时
- 要找到根本原因
- 预防比处理更重要
