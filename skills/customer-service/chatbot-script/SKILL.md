# Chatbot Script Design Skill

设计聊天机器人对话流程，编写对话脚本，优化聊天体验。

## 何时使用

- 聊天机器人搭建
- 对话流程设计
- 机器人话术撰写
- 意图识别配置
- 对话效果优化

## 对话设计流程

```
1. 场景分析 → 2. 意图设计 → 3. 流程绘制 → 4. 话术撰写 → 5. 测试优化
```

### 4. 话术撰写

**话术模板库：**

```markdown
## 寒暄话术
```markdown
**开场白：**
您好，我是XX客服机器人，请问有什么可以帮您？

**再次访问：**
欢迎回来！请问是查询之前的订单吗？
```

## 确认话术
```markdown
**信息确认：**
您的订单号是[order_id]，下单时间是[date]，请问对吗？

**操作确认：**
确认为您办理退款吗？退款将在1-3个工作日内到账。
```

## 引导话术
```markdown
**信息收集：**
请提供您的[需要的信息]，以便我帮您查询。

**操作引导：**
请点击[按钮名称]继续操作
```

## 结束话术
```markdown
**正常结束：**
感谢您的咨询，祝您生活愉快！

**转人工：**
您的问题已转接给人工客服，请稍候。
```

## 错误话术
```markdown
**无法理解：**
抱歉，我没有理解您的问题。请换个方式描述，或输入"转人工"咨询人工客服。

**系统错误：**
系统繁忙，请稍后再试。您也可以拨打客服热线 400-xxx-xxxx
```
```

**风格指南：**

| 场景 | 语气 | 示例 |
|------|------|------|
| 问候 | 友好热情 | "您好！" |
| 询问 | 耐心引导 | "请您告诉我..." |
| 确认 | 清晰准确 | "您的订单是..." |
| 错误 | 诚恳歉意 | "抱歉让您久等了" |
| 结束 | 温暖祝福 | "祝您生活愉快！" |

## 对话设计检查清单

- [ ] 核心意图全覆盖
- [ ] 对话流程无死循环
- [ ] 转人工条件合理
- [ ] 话术风格一致
- [ ] 异常处理完善
- [ ] 测试用例完整
- [ ] 效果指标可衡量

## 红牌警告

- 意图识别率过低
- 死循环无法退出
- 承诺机器人无法实现的功能
- 敏感操作无确认
- 转人工条件过严
- 话术风格不一致
- 未考虑无障碍访问
- 缺乏日志记录
