# 反垄断合规 (Antitrust Compliance)

## 触发条件
- 反垄断合规
- 竞争法
- 垄断风险
- 经营者集中

## 输入
- 业务类型
- 市场地位
- 交易情况
- 竞争对手

## 输出
- 合规方案
- 风险评估
- 审查流程
- 整改措施

## Prompt模板
```
请评估[业务]反垄断合规。

背景：
- 业务类型：[类型]
- 市场地位：[地位]
- 交易情况：[交易]
- 竞争对手：[对手]

请输出：
1. 相关市场界定
2. 市场地位评估
3. 合规风险识别
4. 经营者集中审查
5. 合规制度
6. 日常运营注意
7. 风险应对
```

## 示例
### 示例1
- 输入：互联网平台，计划收购竞争对手
- 输出：合规方案，含集中审查、整改措施、合规制度

## 注意事项
- 要主动合规
- 要关注执法趋势
- 要控制风险
