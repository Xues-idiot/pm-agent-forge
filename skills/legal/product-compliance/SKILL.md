# 产品合规 (Product Compliance)

## 触发条件
- 产品认证
- 标准符合
- 安全合规
- 市场准入

## 输入
- 产品类型
- 目标市场
- 认证要求
- 时间要求

## 输出
- 合规方案
- 认证计划
- 测试安排
- 风险控制

## Prompt模板
```
请评估[产品]合规方案。

背景：
- 产品类型：[类型]
- 目标市场：[市场]
- 认证要求：[要求]
- 时间要求：[要求]

请输出：
1. 适用法规
2. 标准要求
3. 认证清单
4. 测试计划
5. 型式认可
6. 市场准入
7. 标识要求
8. 持续合规
```

## 示例
### 示例1
- 输入：电子产品出口欧盟，需要CE认证
- 输出：合规方案，含适用指令、测试安排、CE标识

## 注意事项
- 要目标市场明确
- 要提前规划
- 要持续合规
