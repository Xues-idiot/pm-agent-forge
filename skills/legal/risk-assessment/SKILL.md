# Legal Risk Assessment Skill

评估法律风险，识别潜在法律责任，评估诉讼风险，制定风险应对策略。

## 何时使用

- 法律风险识别
- 潜在诉讼评估
- 商业决策法律评估
- 合作伙伴风险评估
- 法律尽职调查
- 风险管理策略制定

## 法律风险评估流程

```
1. 风险识别 → 2. 风险分析 → 3. 风险评估 → 4. 风险应对 → 5. 监控报告
```

- 重大法律风险未识别
- 风险评估明显低估
- 忽视监管政策变化

- 监控机制失效
- 重大风险未上报
- 违规行为未整改
- 法律意见被忽视
