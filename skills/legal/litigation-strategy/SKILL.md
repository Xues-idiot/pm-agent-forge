# 诉讼策略 (Litigation Strategy)

## 触发条件
- 诉讼方案制定
- 诉讼风险评估
- 证据策略
- 判决执行

## 输入
- 案件类型
- 争议金额
- 证据情况
- 时间要求

## 输出
- 诉讼方案
- 风险评估
- 证据策略
- 执行预案

## Prompt模板
```
请制定[案件]诉讼策略。

背景：
- 案件类型：[类型]
- 争议金额：[金额]
- 证据情况：[证据]
- 时间要求：[要求]

请输出：
1. 案件分析
2. 诉讼请求设计
3. 法律依据
4. 证据策略
5. 风险评估
6. 程序规划
7. 调解预案
8. 判决执行
9. 成本估算
```

## 示例
### 示例1
- 输入：合同违约纠纷，我方是原告，证据充分
- 输出：诉讼策略，含诉请设计、证据清单、赔偿计算

## 注意事项
- 要充分评估
- 要准备预案
- 要控制成本
