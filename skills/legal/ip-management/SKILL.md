# Intellectual Property Management Skill

管理知识产权，处理专利/版权问题，保护商标和商业秘密。

## 何时使用

- 专利申请
- 商标保护
- 版权管理
- 软件著作权
- 商业秘密保护
- IP 风险评估

## IP 管理流程

```
1. IP 盘点 → 2. 保护策略 → 3. 申请管理 → 4. 风险监控 → 5. 维权保护
```

### 2. 保护策略

**专利策略：**

```markdown
## 专利策略

### 专利类型选择
| 类型 | 保护客体 | 审查周期 | 保护期限 | 费用 | 适用场景 |
|------|----------|----------|----------|------|----------|
| 发明专利 | 技术方案 | 2-3年 | 20年 | 高 | 核心技术创新 |
| 实用新型 | 形状/结构 | 6-8月 | 10年 | 中 | 改进型创新 |
| 外观设计 | 产品外观 | 4-6月 | 15年 | 中 | 产品设计 |

### 专利申请决策
```markdown
## 申请决策矩阵

| 场景 | 建议 | 理由 |
|------|------|------|
| 核心技术，竞争对手难以绕开 | 申请发明专利 | 最强保护 |
| 改进型创新，防御性使用 | 申请实用新型 | 快速获权 |
| 产品外观，营销价值大 | 申请外观设计 | 保护产品形象 |
| 迭代快速，易被绕过 | 商业秘密保护 | 保密优于公开 |
| 开源组件 | 不申请专利 | 避免专利流氓 |
```

### 4. 风险监控

**IP 风险识别：**

```markdown
## IP 风险监控

### 内部风险
| 风险 | 描述 | 防范措施 |
|------|------|----------|
| 技术泄露 | 员工离职带走技术 | 竞业协议+保密协议 |
| 职务发明 | 权属争议 | 明确权属+奖励机制 |
| 侵权风险 | 使用未授权软件/代码 | 正版化+代码审计 |
| 商标淡化 | 商标被通用化 | 规范使用+防御注册 |

### 外部风险
| 风险 | 描述 | 防范措施 |
|------|------|----------|
| 被侵权 | 技术被抄袭 | 监测+快速维权 |
| 被抢注 | 商标被抢注 | 监测+异议+无效 |
| 侵权诉讼 | 产品侵犯他人IP | FTO分析+避让设计 |
| 恶意投诉 | 竞争对手恶意投诉 | 法律应对+申诉 |
```

### 5. 维权保护

**维权策略：**

```markdown
## 维权策略

### 维权方式选择
| 方式 | 适用场景 | 优点 | 缺点 | 成本 |
|------|----------|------|------|------|
| 协商和解 | 轻微侵权 | 快速 | 效果弱 | 低 |
| 行政投诉 | 商标/专利侵权 | 高效 | 赔偿低 | 中 |
| 民事诉讼 | 重大侵权 | 赔偿高 | 周期长 | 高 |
| 刑事报案 | 恶意侵权 | 威慑力强 | 门槛高 | 中 |

### 维权流程
```markdown
## 维权流程

1. 证据保全（公证/鉴定）
2. 侵权分析（对比+取证）
3. 发律师函（视情况）
4. 协商/调解（如可能）
5. 提起诉讼（必要时）
6. 执行判决
```
```

## 红牌警告

- 核心 IP 未申请保护
- 侵权风险未排查
- 商标被抢注
- 员工泄露商业秘密
- 使用未授权软件
- 专利申请超期
- 权利过期未续
- IP 资产流失
