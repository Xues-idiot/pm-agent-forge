# Legal Research Skill

法律研究，查找相关法规，分析判例，支持法律决策。

## 何时使用

- 法律问题研究
- 法规查询
- 判例分析
- 合规依据
- 法律风险评估
- 诉讼/仲裁准备

## 法律研究流程

```
1. 明确问题 → 2. 确定研究范围 → 3. 检索法律资源 → 4. 分析法律依据 → 5. 形成结论
```

## 法律检索技巧

### 关键词检索

```markdown
## 关键词技巧

### 精确匹配
- 使用双引号："合同解除"

### OR 检索
- 合同 OR 契约

### AND 检索
- 违约 AND 赔偿

### NOT 检索
- 劳动 AND NOT 劳务
```

### 引用检索

```markdown
## 引用检索

### 正向检索
- 查找引用了某法规的文件

### 反向检索
- 查找某法规引用的文件
```

## 法律研究检查清单

- [ ] 问题定义清晰
- [ ] 研究范围明确
- [ ] 法律资源权威
- [ ] 检索词全面
- [ ] 法律依据准确
- [ ] 判例参照适当
- [ ] 结论有依据
- [ ] 建议可行

## 红牌警告

- 法律依据已失效
- 检索范围过窄
- 忽略重要法律规定
- 误解法律条款含义
- 判例不具参照性
- 结论过于确定
- 忽略地方差异
- 忽略时效性
