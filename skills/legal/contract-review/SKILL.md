# Legal Contract Review Skill

审查合同条款，识别法律风险，提出修改建议。

## 何时使用

- 合同审查
- 合同起草
- 合同谈判支持
- 合同风险评估
- 供应商合同审查
- 劳动合同审查

## 合同审查流程

```
1. 合同收集 → 2. 基础信息 → 3. 条款分析 → 4. 风险识别 → 5. 修改建议 → 6. 审查报告
```

```markdown
## 合同风险识别

| 风险点 | 描述 | 风险等级 | 建议 |
|--------|------|---------|------|
| | | 高 | |

### 中风险项
| 风险点 | 描述 | 风险等级 | 建议 |
|--------|------|---------|------|
| | | 中 | |

### 低风险项
| 风险点 | 描述 | 风险等级 | 建议 |
|--------|------|---------|------|
| | | 低 | |

### 风险类型分类
| 风险类型 | 具体风险 | 应对策略 |
|---------|---------|---------|
| 主体风险 | 对方履约能力 | 担保/预付款 |
| 条款风险 | 条款不公平 | 协商修改 |
| 履行风险 | 履行能力不足 | 分期履行 |
| 终止风险 | 提前终止条款 | 明确条件 |
```

### 5. 修改建议

**条款修改建议模板：**

```markdown
## 条款修改建议

### 建议1：[条款名称]
**当前条款：**
[原文]

**风险分析：**
[风险描述]

**建议修改为：**
[修改建议]

**修改理由：**
[理由说明]

### 建议2：
...
```

**谈判优先级：**

| 优先级 | 条款类型 | 说明 |
|--------|---------|------|
| 必须争取 | 核心权益条款 | 必须修改 |
| 尽量争取 | 重要利益条款 | 尽力修改 |
| 可以接受 | 一般条款 | 可接受现有表述 |
| 可以让步 | 非核心条款 | 可以让步 |
```

## 常见合同类型审查要点

### 采购合同

```markdown
## 采购合同审查要点

### 必备条款
- [ ] 产品规格型号
- [ ] 质量标准
- [ ] 交货方式
- [ ] 验收标准
- [ ] 付款方式
- [ ] 质量保证期

### 重点关注
- 质量不达标处理
- 延迟交货违约责任
- 售后服务条款
- 知识产权归属
```

### 劳动合同

```markdown
## 劳动合同审查要点

### 必备条款
- [ ] 合同期限
- [ ] 工作内容
- [ ] 工作地点
- [ ] 工作时间
- [ ] 休息休假
- [ ] 劳动报酬
- [ ] 社会保险

### 重点关注
- 试用期条款
- 竞业限制
- 保密条款
- 解除条件
- 违约金条款
```

## 合同审查检查清单

- [ ] 合同主体适格
- [ ] 必备条款完整
- [ ] 条款表述清晰
- [ ] 权利义务对等
- [ ] 违约责任明确
- [ ] 争议解决条款
- [ ] 生效条件明确
- [ ] 签署手续完备

## 红牌警告

- 合同主体不具备履约资质
- 关键条款模糊不清
- 违约责任明显不对等
- 缺少争议解决条款
- 签署手续不完整
- 超越经营范围的条款
- 违反法律法规的条款
- 明显不公平的格式条款
