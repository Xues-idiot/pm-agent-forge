# 政策倡导 (Policy Advocacy)

## 触发条件
- 政策影响
- 行业倡议
- 政府关系
- 公共事务

## 输入
- 政策领域
- 企业诉求
- 行业状况
- 资源能力

## 输出
- 倡导策略
- 沟通计划
- 联盟建设
- 效果评估

## Prompt模板
```
请制定[政策]倡导策略。

背景：
- 政策领域：[领域]
- 企业诉求：[诉求]
- 行业状况：[状况]
- 资源能力：[资源]

请输出：
1. 政策环境分析
2. 倡导目标
3. 核心信息
4. 策略选择
5. 沟通计划
6. 联盟建设
7. 效果评估
```

## 示例
### 示例1
- 输入：新能源行业，希望争取更多政策支持
- 输出：倡导策略，含行业联盟、政策研究、公众沟通

## 注意事项
- 要合法合规
- 要长期经营
- 要多元参与
