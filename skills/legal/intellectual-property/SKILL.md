# 知识产权战略 (Intellectual Property Strategy)

## 触发条件
- 知识产权战略制定
- 专利布局
- 商标保护
- IP运营

## 输入
- 业务类型
- 技术领域
- 竞争环境
- 发展阶段

## 输出
- IP战略
- 专利布局
- 保护方案
- 运营策略

## Prompt模板
```
请为[公司/产品]制定知识产权战略。

背景：
- 业务类型：[类型]
- 技术领域：[领域]
- 竞争环境：[环境]
- 发展阶段：[阶段]

请输出：
1. 知识产权现状诊断
2. IP战略目标
3. 专利布局策略（核心/外围/防御）
4. 商标保护方案
5. 商业秘密保护
6. 版权管理
7. IP风险防范
8. IP运营和变现策略
9. 竞争对手IP分析
10. 实施计划和预算
```

## 示例
### 示例1
- 输入：AI创业公司，核心算法有创新，正处于融资阶段
- 输出：IP战略，含专利布局、保密措施、融资IP尽调

## 注意事项
- 要有前瞻性布局
- 要平衡保护和运营
- 要关注竞争对手
