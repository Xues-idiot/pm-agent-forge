# 数据合规 (Data Compliance)

## 触发条件
- 数据合规体系建设
- 合规风险评估
- 数据安全规划
- 监管应对

## 输入
- 数据类型
- 业务场景
- 法规要求
- 现有措施

## 输出
- 合规评估报告
- 合规整改方案
- 数据安全策略
- 实施路线图

## Prompt模板
```
请为[公司/业务]设计数据合规方案。

背景：
- 数据类型：[类型]
- 业务场景：[场景]
- 适用法规：[法规]
- 现有措施：[措施]

请输出：
1. 数据合规现状评估
2. 法规要求对照检查表
3. 合规差距分析
4. 数据分类分级方案
5. 合规整改措施
6. 数据安全技术方案
7. 组织保障和培训计划
8. 合规运营机制
9. 监管应对预案
```

## 示例
### 示例1
- 输入：金融科技公司，处理大量用户金融数据，需符合等保2.0和个人信息保护法
- 输出：数据合规方案，含分类分级、安全措施、监管应对

## 注意事项
- 要覆盖主要法规要求
- 技术和管理措施并重
- 要建立持续合规机制
