# 监管合规 (Regulatory Compliance)

## 触发条件
- 行业监管分析
- 合规体系建设
- 监管应对
- 合规培训

## 输入
- 行业类型
- 监管机构
- 业务范围
- 合规现状

## 输出
- 合规框架
- 合规清单
- 整改方案
- 培训计划

## Prompt模板
```
请为[公司/行业]设计监管合规方案。

背景：
- 行业类型：[行业]
- 监管机构：[机构]
- 业务范围：[范围]
- 合规现状：[现状]

请输出：
1. 监管环境分析
2. 合规要求和清单
3. 合规差距分析
4. 合规管理体系设计
5. 合规制度和流程
6. 合规技术支撑
7. 合规团队建设
8. 监管应对策略
9. 合规培训计划
10. 合规效果评估
```

## 示例
### 示例1
- 输入：金融科技公司，涉及央行和银保监双重监管
- 输出：合规方案，含双牌照合规、数据安全、监管沟通

## 注意事项
- 要关注政策变化
- 要主动沟通监管
- 要持续合规而非一次性
