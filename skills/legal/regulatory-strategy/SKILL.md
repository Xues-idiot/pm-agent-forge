# 监管策略 (Regulatory Strategy)

## 触发条件
- 监管关系
- 合规策略
- 政策影响
- 监管预判

## 输入
- 行业监管
- 政策环境
- 企业定位
- 风险偏好

## 输出
- 监管策略
- 合规规划
- 政策影响
- 关系维护

## Prompt模板
```
请制定[企业]监管策略。

背景：
- 行业监管：[监管]
- 政策环境：[环境]
- 企业定位：[定位]
- 风险偏好：[偏好]

请输出：
1. 监管环境分析
2. 监管策略定位
3. 合规规划
4. 政策影响评估
5. 监管关系维护
6. 危机应对
7. 持续跟踪
```

## 示例
### 示例1
- 输入：金融科技公司，需要应对多牌照监管
- 输出：监管策略，含牌照规划、合规路径、关系维护

## 注意事项
- 要主动合规
- 要关系维护
- 要前瞻布局
