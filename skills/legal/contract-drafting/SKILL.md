# 合同起草 (Contract Drafting)

## 触发条件
- 合同文本起草
- 条款设计
- 风险条款
- 模板制定

## 输入
- 合同类型
- 交易结构
- 双方地位
- 风险偏好

## 输出
- 合同文本
- 条款说明
- 风险提示
- 修改记录

## Prompt模板
```
请起草[类型]合同。

背景：
- 合同类型：[类型]
- 交易结构：[结构]
- 双方地位：[地位]
- 风险偏好：[偏好]

请输出：
1. 合同基本信息
2. 合同正文（完整条款）
3. 核心条款说明
4. 风险条款
5. 签署页
6. 附件清单
```

## 示例
### 示例1
- 输入：软件开发合同，委托方强势，预付70%
- 输出：合同文本，含详细功能范围、验收标准、知识产权归属

## 注意事项
- 条款要清晰
- 权责要对等
- 要留有弹性
