# 金融科技合规 (FinTech Compliance)

## 触发条件
- 金融牌照
- 监管科技
- 合规系统
- 风控模型

## 输入
- 业务类型
- 牌照需求
- 监管环境
- 技术能力

## 输出
- 合规方案
- 牌照申请
- 系统设计
- 风险控制

## Prompt模板
```
请设计[金融科技]合规方案。

背景：
- 业务类型：[类型]
- 牌照需求：[牌照]
- 监管环境：[环境]
- 技术能力：[能力]

请输出：
1. 监管环境分析
2. 牌照申请策略
3. 合规系统设计
4. 风控模型
5. 反欺诈
6. 合规流程
7. 监管报送
8. 持续监控
```

## 示例
### 示例1
- 输入：互联网贷款平台，需要符合金融持牌要求
- 输出：合规方案，含牌照申请、风控系统、监管报送

## 注意事项
- 要主动合规
- 要技术赋能
- 要持续跟进
