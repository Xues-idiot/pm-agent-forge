# Legal Compliance Check Skill

合规检查，验证业务活动符合法律法规要求。

## 何时使用

- 业务合规检查
- 资质证照审查
- 合规风险评估
- 内部合规审查
- 合作伙伴合规
- 定期合规审计

## 合规检查流程

```
1. 明确业务范围 → 2. 确定法规要求 → 3. 收集证据 → 4. 执行检查 → 5. 风险评估 → 6. 整改建议
```

## 红牌警告

- 无证照经营
- 证照已过期
- 业务超出许可范围
- 存在重大安全隐患
- 税务严重违规
- 劳动者权益严重侵害
- 数据泄露
- 涉嫌违法犯罪
