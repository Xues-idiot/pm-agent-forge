# 公司治理 (Corporate Governance)

## 触发条件
- 治理结构设计
- 治理制度完善
- 董事会运作优化
- 决策机制优化

## 输入
- 公司类型
- 发展阶段
- 股权结构
- 治理现状

## 输出
- 治理诊断报告
- 治理结构方案
- 制度优化建议
- 决策流程

## Prompt模板
```
请为[公司]设计/优化公司治理方案。

背景：
- 公司类型：[类型]
- 发展阶段：[阶段]
- 股权结构：[结构]
- 现有治理：[现状]

请输出：
1. 公司治理现状诊断
2. 治理结构设计（三会一层）
3. 董事会/管理层职责分工
4. 决策流程和权限表
5. 信息披露机制
6. 风险管控机制
7. 关联交易管理
8. 中小股东保护措施
```

## 示例
### 示例1
- 输入：民营企业，准备上市，股权较集中
- 输出：治理优化方案，含引入独立董事、完善董事会专门委员会

## 注意事项
- 要符合监管要求
- 要平衡效率和风险
- 要考虑发展阶段
