# 交易支持 (Transaction Support)

## 触发条件
- 合同谈判支持
- 交易结构设计
- 法律尽职调查
- 交割执行

## 输入
- 交易类型
- 交易金额
- 交易对手
- 时间要求

## 输出
- 交易方案
- 法律意见
- 风险提示
- 交割清单

## Prompt模板
```
请为[交易类型]提供法律支持。

背景：
- 交易类型：[类型]
- 交易金额：[金额]
- 交易对手：[对手]
- 时间要求：[要求]

请输出：
1. 交易结构和方案
2. 法律尽职调查清单
3. 法律风险识别和评估
4. 交易文件清单
5. 核心条款设计
6. 风险缓解措施
7. 审批流程
8. 交割条件清单
9. 交割后义务
10. 时间表和责任分工
```

## 示例
### 示例1
- 输入：股权融资交易，投前估值1亿，需在4周内完成
- 输出：法律支持方案，含尽调清单、投资协议核心条款、交割清单

## 注意事项
- 要平衡效率和风险
- 要关注税务影响
- 要确保交割完整
