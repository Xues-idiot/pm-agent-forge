# 公司治理 (Corporate Governance)

## 触发条件
- 治理结构完善
- 董事会运作
- 股东权益
- 合规上市

## 输入
- 公司类型
- 股权结构
- 发展需求
- 监管要求

## 输出
- 治理方案
- 制度框架
- 实施计划
- 评估机制

## Prompt模板
```
请优化[公司]公司治理。

背景：
- 公司类型：[类型]
- 股权结构：[结构]
- 发展需求：[需求]
- 监管要求：[要求]

请输出：
1. 公司治理现状评估
2. 治理结构设计
3. 三会一层职责
4. 议事规则
5. 信息披露
6. 关联交易管理
7. 风险管控
8. 合规计划
```

## 示例
### 示例1
- 输入：民营企业准备上市，需要规范治理
- 输出：治理方案，含引入独立董事、完善董事会、关联交易管理

## 注意事项
- 要合规优先
- 要高效决策
- 要保护股东权益
