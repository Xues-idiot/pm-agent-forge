# Contract Lifecycle Management Skill

管理合同生命周期，跟踪合同执行，处理合同续签和终止。

## 何时使用

- 合同生命周期管理
- 合同执行跟踪
- 合同续签管理
- 合同变更管理
- 合同归档
- 合同风险监控

## 合同管理流程

```
1. 合同签订 → 2. 合同执行 → 3. 履约监控 → 4. 变更管理 → 5. 归档管理
```

### 3. 履约监控

**履约预警：**

```markdown
## 履约预警

### 预警规则
| 预警级别 | 触发条件 | 响应时间 |
|----------|----------|----------|
| 红色预警 | 付款逾期>30天 | 立即处理 |
| 橙色预警 | 付款逾期>7天 | 24小时 |
| 黄色预警 | 付款/交付日前7天 | 提前提醒 |

### 风险监控
| 风险类型 | 识别方法 | 应对措施 |
|----------|----------|----------|
| 对方履约能力 | 经营异常/舆情 | 增加担保/加速验收 |
| 交付延期 | 进度跟踪 | 督促/索赔 |
| 质量不达标 | 验收检查 | 整改/扣款 |
| 合同纠纷 | 沟通记录 | 协商/法律 |
```

## 红牌警告

- 合同台账不完整
- 审批流程缺失
- 履约跟踪失控
- 逾期付款未处理
- 重大变更未审批
- 合同丢失
- 授权过期
- 印章管理混乱
