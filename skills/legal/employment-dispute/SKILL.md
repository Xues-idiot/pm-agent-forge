# Employment Dispute Management Skill

处理劳动争议，进行劳动仲裁，管理员工离职冲突。

## 何时使用

- 劳动争议处理
- 仲裁/诉讼代理
- 离职谈判
- 薪酬纠纷
- 解除劳动合同
- 竞业限制争议

## 劳动争议处理流程

```
1. 争议预防 → 2. 内部协商 → 3. 调解处理 → 4. 仲裁/诉讼 → 5. 执行跟踪
```

### 1. 争议预防

**合规管理体系：**

```markdown
## 劳动合规体系

### 核心制度
| 制度 | 要求 | 检查频率 |
|------|------|----------|
| 劳动合同 | 100%签订 | 入职时 |
| 社保公积金 | 依法缴纳 | 每月 |
| 工资支付 | 按时足额 | 每月 |
| 工作时间 | 符合工时规定 | 持续 |
| 休息休假 | 依法安排 | 持续 |

### 风险自检清单
```markdown
## 合规自检

### 入职管理
- [ ] 入职当月签订劳动合同
- [ ] 入职当月缴纳社保
- [ ] 告知员工规章制度
- [ ] 收取员工证件复印件

### 4. 仲裁/诉讼

**仲裁准备：**

```markdown

- 劳动争议申请仲裁时效：1年

### 仲裁管辖
- 用人单位所在地
- 劳动合同履行地

**常见争议类型：**

```markdown
## 常见争议处理

### 解除劳动合同争议
| 情形 | 补偿/赔偿 | 法律依据 |
|------|-----------|----------|
| 协商解除 | N个月工资 | 劳动合同法第36条 |
| 员工过错解除 | 无补偿 | 劳动合同法第39条 |
| 员工非过错解除 | N+1或N | 劳动合同法第40条 |
| 经济性裁员 | N+1 | 劳动合同法第41条 |
| 违法解除 | 2N | 劳动合同法第87条 |

### 工资争议
| 类型 | 处理方式 |
|------|----------|
| 克扣工资 | 补发+25%经济补偿 |
| 拖欠工资 | 补发+50%-100%赔偿金 |
| 加班工资 | 平日1.5倍/休息日2倍/法定3倍 |

### 工伤待遇
| 伤残等级 | 一次性伤残补助金 | 伤残津贴 |
|----------|------------------|----------|
| 一级 | 27个月本人工资 | 本人工资90% |
| 二级 | 25个月本人工资 | 本人工资85% |
| ... | | |
