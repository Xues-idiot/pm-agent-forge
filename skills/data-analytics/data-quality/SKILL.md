### 异常值检测
```python
## 异常值检测方法

### 统计方法
- Z-score: |z| > 3 为异常
- IQR: Q1-1.5*IQR ~ Q3+1.5*IQR 外为异常

### 业务规则
- 年龄: 0-120
- 收入: 0-10亿
- 订单金额: >0

### 检测示例
```python
import numpy as np

def detect_outliers_iqr(data, column):
    Q1 = data[column].quantile(0.25)
    Q3 = data[column].quantile(0.75)
    IQR = Q3 - Q1
    lower = Q1 - 1.5 * IQR
    upper = Q3 + 1.5 * IQR
    outliers = data[(data[column] < lower) | (data[column] > upper)]
    return outliers
```

### 空值处理
| 策略 | 适用场景 | 方法 |
|------|----------|------|

### 格式标准化
| 类型 | 规则 | 示例 |
|------|------|------|
| 手机号 | 去86/去横线 | 86138... → 138... |
| 日期 | 统一格式 | 统一 YYYY-MM-DD |
| 金额 | 统一单位 | 统一元 |
| 地址 | 结构化 | 省-市-区-详细 |
```

### 清洗代码模板

```python
## 数据清洗模板

import pandas as pd
import numpy as np

def clean_data(df):
    df_clean = df.copy()

    # 1. 去重
    df_clean = df_clean.drop_duplicates()

    # 2. 空值处理
    df_clean['phone'].fillna('unknown', inplace=True)
    df_clean['age'].fillna(df_clean['age'].median(), inplace=True)

    # 3. 格式标准化
    df_clean['phone'] = df_clean['phone'].str.replace(r'[^0-9]', '', regex=True)
    df_clean['phone'] = df_clean['phone'].str[-11:]  # 取后11位

    # 4. 异常值处理
    df_clean = df_clean[(df_clean['age'] >= 0) & (df_clean['age'] <= 120)]

    # 5. 数据类型转换
    df_clean['date'] = pd.to_datetime(df_clean['date'])

    return df_clean
```
