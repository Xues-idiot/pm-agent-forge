# 异常检测 (Anomaly Detection)

## Overview

异常检测是识别数据中异常模式/行为的技能。涵盖统计方法、机器学习、实时检测、报警处理等核心环节。

## 何时使用

- 业务数据异常监控
- 欺诈行为检测
- 系统故障预警
- 质量异常检测

## 流程

1. **场景分析**：确定异常类型和检测目标
2. **数据准备**：收集和预处理数据
3. **方法选择**：选择合适的检测方法
4. **模型训练**：训练异常检测模型
5. **阈值设定**：设定报警阈值
6. **实时检测**：部署实时检测系统
7. **报警处理**：建立报警处理流程

## 红牌警告

- 异常定义不准确
- 阈值过严导致误报过多
- 阈值过松导致漏报
- 忽视业务周期性问题
- 检测不及时导致损失扩大
