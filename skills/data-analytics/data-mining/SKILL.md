## 相关性分析

### 相关分析

```markdown
## 相关性分析

### 相关系数
| 系数 | 范围 | 适用 |
|------|------|------|
| Pearson | -1~1 | 线性/连续 |
| Spearman | -1~1 | 单调/有序 |
| Kendall | -1~1 | 有序/小样本 |

### 相关性解读
| 相关系数 | 强度 | 业务含义 |
|----------|------|----------|
| 0.8-1.0 | 极强 | A导致B? |
| 0.6-0.8 | 强 | 强关联 |
| 0.4-0.6 | 中 | 中度关联 |
| 0.2-0.4 | 弱 | 弱关联 |
| 0-0.2 | 极弱/无 | 无关联 |

### 注意事项
- 相关不等于因果
- 隐藏变量
- 线性假设
- 数据范围
```
