# Cohort Analysis

队列分析，分析用户群体随时间的行为。

## 何时使用

- 留存分析
- 用户分群分析
- 生命周期分析
- 行为趋势分析
- 效果评估

```markdown

## 留存分析

```markdown
## 留存定义

### 留存概念
| 概念 | 说明 | 计算 |

### 留存计算

```markdown
## 留存计算

### 计算公式
N日留存率 = 第N天回访的Cohort数 / Cohort总人数

示例：
- Cohort：2024-01-01注册用户1000人
- 第7天回访：300人
- 7日留存率：300/1000 = 30%

## 流失分析

### 流失定义

```markdown
## 流失定义

### 流失类型
| 类型 | 定义 | 判断 |
|------|------|------|
| 绝对流失 | 明确不再回来 | 长期不回 |
| 相对流失 | 活跃下降 | 符合定义 |
| 沉默流失 | 无行为沉默 | 无任何行为 |

### 流失周期
| 周期 | 定义 | 适用 |
|------|------|------|
| 短期流失 | 7天内 | 高频产品 |
| 中期流失 | 7-30天 | 中频产品 |
| 长期流失 | 30天+ | 低频产品 |

## 流失特征

### 行为特征
- 活跃度下降

|------|----------|------|

### 生命周期价值

```markdown
## LTV预测

### LTV计算
| 方法 | 说明 | 适用 |
|------|------|------|
| 历史法 | 历史数据均值 | 稳定产品 |
| 趋势法 | 趋势外推 | 增长产品 |
| 模型法 | 预测模型 | 精准预测 |

### LTV公式
```markdown
## LTV公式

LTV = Σ (月ARPU × 留存率 × 毛利率) / 折现率

示例：
- 月ARPU: 50元
- 月留存率: 90%
- 毛利率: 70%
- 折现率: 10%/12

月1: 50 × 0.9 × 0.7 / (1+0.1/12) = 31.5
月2: 50 × 0.9² × 0.7 / (1+0.1/12)² = ...
...
```
```
