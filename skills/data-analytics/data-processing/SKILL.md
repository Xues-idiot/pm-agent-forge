# Data Analytics Data Processing Skill

数据清洗和转换，为分析做准备。

## 何时使用

- 原始数据需要清洗
- 数据格式需要转换
- 存在缺失值或异常值
- 需要合并多个数据源
- 数据需要标准化

## 数据处理流程

```
1. 数据理解 → 2. 数据清洗 → 3. 数据转换 → 4. 特征工程 → 5. 数据验证 → 6. 输出
```

### 2. 数据清洗

**缺失值处理：**

| 策略 | 适用场景 | 方法 |
|------|---------|------|
| 删除 | 缺失比例 > 50% | 删除行/列 |
| 填充 | 缺失比例 < 30% | 均值/中位数/众数/插值 |
| 保留 | 缺失有业务含义 | 标记为"未知" |

```python
# 缺失值处理示例
df['字段'].fillna(df['字段'].mean(), inplace=True)  # 均值填充
df['字段'].fillna('未知', inplace=True)  # 类别变量
```

**异常值处理：**

```markdown
## 异常值检测

### 方法
- 3σ原则：超过均值±3标准差
- IQR方法：超过 Q1-1.5*IQR 或 Q3+1.5*IQR
- 业务规则：不符合业务逻辑的值

### 处理策略
| 异常类型 | 处理方式 |
|---------|---------|
| 输入错误 | 修正 |
| 真实极端值 | 保留或缩尾 |
| 无效数据 | 删除 |
```

### 3. 数据转换

**常用转换：**

| 转换类型 | 目的 | 示例 |
|---------|------|------|
| 标准化 | 消除量纲影响 | MinMax / Z-score |
| 编码 | 类别变量转数值 | OneHot / Label |
| 对数 | 处理偏态分布 | log(x+1) |
| 分箱 | 连续转离散 | 等频/等距 |

```python
# 标准化示例
from sklearn.preprocessing import StandardScaler
scaler = StandardScaler()
df['标准化字段'] = scaler.fit_transform(df[['原始字段']])

# 分箱示例
df['年龄段'] = pd.cut(df['年龄'], bins=[0, 18, 35, 60, 100], labels=['未成年', '青年', '中年', '老年'])
```

### 4. 特征工程

**特征构建：**
```markdown
## 特征工程

### 衍生特征
- 日期特征：年、月、日、星期、季度
- 时间差：距某个日期的天数
- 聚合特征：count, sum, mean, max, min
- 交叉特征：字段1 × 字段2

### 特征选择
| 方法 | 说明 |
|------|------|
| 方差阈值 | 删除低方差特征 |
| 相关系数 | 删除高相关特征 |
| 重要性 | 基于模型的特征重要性 |
```

## 红牌警告

- 缺失值直接删除不评估影响

- 类别编码顺序混乱
- 处理后数据与原始数据不一致
- 未记录数据处理过程
