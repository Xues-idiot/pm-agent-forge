# Funnel Analysis

漏斗分析，分析用户转化和流失。

## 何时使用

- 转化漏斗设计
- 流失分析
- 用户行为分析
- 优化点识别
- AARRR转化分析

## 漏斗基础

### 漏斗概念

```markdown
## 漏斗概念

### 漏斗定义
| 要素 | 说明 | 示例 |
|------|------|------|
| 步骤 | 转化节点 | 注册→下单→支付 |
| 流失 | 每步离开 | 注册后未下单 |
| 转化 | 成功前进 | 完成支付 |
| 转化率 | 成功比例 | 50% |

### 漏斗特点
```markdown
## 漏斗特点

### 特性
- 递减性：每步递减
- 比例性：流失有比例
- 路径性：有序步骤

### 常用漏斗
| 类型 | 场景 | 核心指标 |
|------|------|----------|
| 获客漏斗 | 获取用户 | CPA |
| 转化漏斗 | 业务流程 | 整体转化 |
| 留存漏斗 | 用户留存 | 留存率 |
| 变现漏斗 | 收入转化 | 付费率 |
```
```

### 漏斗模型

```markdown
## 漏斗模型

### AARRR模型
```
获客(Acquisition)
    ↓
激活(Activation)
    ↓
留存(Retention)
    ↓
变现(Revenue)
    ↓

```

```
访问 → 注册 → 认证 → 首次使用 → 活跃 → 付费 → 推荐
```

### 电商转化漏斗
```
浏览 → 加购 → 下单 → 支付 → 确认收货
```
```

## 转化分析

### 转化指标

```markdown
## 转化指标

### 核心指标
| 指标 | 计算 | 说明 |
|------|------|------|
| 整体转化率 | 最终完成/起点 | 整体效率 |
| 步骤转化率 | 本步完成/上步 | 单步效率 |
| 流失率 | 流失/上步 | 单步流失 |
| 流失量 | 上步-本步 | 绝对流失 |

### 辅助指标
| 指标 | 计算 | 说明 |
|------|------|------|
| 平均转化时长 | 平均完成时间 | 转化速度 |
| 转化中位数 | 中位完成时间 | 典型时间 |
| 完成路径长度 | 实际路径 | 路径复杂度 |

### 转化计算
```markdown
## 转化计算

### 步骤转化率
步骤转化率 = 步骤N完成数 / 步骤N-1完成数

示例：
- 访问用户：10000
- 注册用户：2000
- 注册转化率：2000/10000 = 20%

### 累计转化率
累计转化率 = 步骤N完成数 / 漏斗起点

示例：
- 访问用户：10000
- 支付用户：500
- 累计转化率：500/10000 = 5%
```
```

### 分析示例
```markdown
## 分析示例

漏斗：浏览→加购→下单→支付

| 步骤 | 用户数 | 转化率 | 流失率 | 流失量 |
|------|--------|--------|--------|--------|
| 浏览 | 10000 | - | - | - |
| 加购 | 3000 | 30% | 70% | 7000 |
| 下单 | 1500 | 50% | 50% | 1500 |
| 支付 | 750 | 50% | 50% | 750 |

分析：
- 浏览→加购流失严重（70%）
- 加购→下单相对健康（50%）
- 优化重点：浏览→加购环节
```
```

## 流失分析

### 流失识别

```markdown
## 流失识别

### 流失定义
| 类型 | 定义 | 时间 |

| 方法 | 说明 | 适用 |
|------|------|------|

| 行为序列 | 流失前行为 | 归因分析 |

### 流失特征

## 漏斗优化

### 优化框架

```markdown
## 漏斗优化

### 优化步骤
1. 识别瓶颈 → 最大流失点
2. 分析原因 → 流失归因
3. 提出假设 → 优化方案
4. 验证效果 → A/B测试
5. 迭代优化 → 持续改进

### 优化优先级
| 因素 | 评估 | 优先级 |
|------|------|--------|
| 流失量 | 大→小 | 先大 |
| 优化难度 | 难→易 | 先易 |
| 预期效果 | 大→小 | 先大 |
| 成本 | 低→高 | 先低 |

### 优化收益计算
优化收益 = 流失量 × 转化提升 × 单用户价值
```
```

### 优化策略

```markdown
## 优化策略

### 流程优化
| 策略 | 方法 | 适用 |
|------|------|------|
| 简化流程 | 减少步骤 | 步骤过多 |
| 合并步骤 | 一步多能 | 功能类似 |
| 并行处理 | 同时进行 | 独立步骤 |
| 跳过非必要 | 直接跳过 | 价值低步骤 |

### 体验优化
| 策略 | 方法 | 适用 |
|------|------|------|
| 减少输入 | 默认值/自动填 | 表单复杂 |
| 进度提示 | 步骤条/百分比 | 步骤多 |
| 实时反馈 | 即时校验 | 需确认 |
| 错误恢复 | 草稿/返回 | 易出错步骤 |

### 引导优化
| 策略 | 方法 | 适用 |
|------|------|------|
| 例子展示 | 示例结果 | 不理解 |
| 利益强调 | 突出好处 | 犹豫不决 |
| 社会证明 | 用户评价 | 信任不足 |
| 紧迫感 | 限时/限量 | 决策慢 |
```
```

## 检查清单

- [ ] 漏斗步骤定义清晰
- [ ] 步骤间互不重叠
- [ ] 转化率计算准确
- [ ] 流失原因分析到位

## 红牌警告

- 步骤定义混乱
- 流失归因不准确
- 只看整体不看细节
- 优化无数据支撑
- 不做A/B验证
- 优化后不复盘
- 忽视负面优化
