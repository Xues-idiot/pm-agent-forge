# 定价优化 (Pricing Optimization)

## 触发条件
- 动态定价
- 价格测试
- 价格监控
- 收益管理

## 输入
- 产品类型
- 定价目标
- 竞争环境
- 数据能力

## 输出
- 定价策略
- 测试方案
- 监控系统
- 优化机制

## Prompt模板
```
请优化[产品]定价。

背景：
- 产品类型：[类型]
- 定价目标：[目标]
- 竞争环境：[环境]
- 数据能力：[能力]

请输出：
1. 定价诊断
2. 价格敏感度
3. 定价策略
4. 价格结构
5. 动态定价
6. 测试方案
7. 监控系统
8. 优化机制
```

## 示例
### 示例1
- 输入：酒店，需要优化收益管理
- 输出：定价方案，含动态定价、渠道管理、收益分析

## 注意事项
- 要数据驱动
- 要持续优化
- 要客户价值
