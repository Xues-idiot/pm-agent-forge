# 内容营销 (Content Marketing)

## Overview

内容营销是通过创作和分发有价值的内容，吸引和留住目标用户的技能。涵盖内容策略、内容创作、内容分发、效果评估等核心环节。

## 何时使用

- 需要建立品牌专业形象
- 希望获取搜索引擎流量
- 需要长期沉淀内容资产
- 产品决策周期长需要持续教育用户

## 流程

1. **内容策略**：制定内容定位和方向
2. **选题规划**：确定内容主题和形式
3. **内容创作**：撰写文章、制作视频等
4. **内容分发**：在多平台发布内容
5. **SEO优化**：优化内容搜索排名
6. **数据监测**：追踪阅读量、留资、转化
7. **迭代优化**：基于数据优化内容方向

## 模板

### 内容营销规划模板
```
## 内容定位
- 品牌角色: [专家/朋友/导师]
- 内容调性: [专业/亲和/有趣]
- 差异化: [独特价值主张]

## 内容矩阵
| 内容类型 | 占比 | 目的 | 发布渠道 |
|----------|------|------|----------|
| 干货文章 | 40% | SEO+留资 | 官网+公众号 |
| 案例分享 | 30% | 信任背书 | 官网+知乎 |
| 视频内容 | 20% | 品牌曝光 | B站+抖音 |
| 用户故事 | 10% | 口碑传播 | 社媒 |

## 核心选题
1. [主题1] - [目的]
2. [主题2] - [目的]
3. [主题3] - [目的]

## KPI目标
- 月产出: [X篇]
- SEO关键词: [X个]
- 留资量: [X个/月]
```

## 检查清单

- [ ] 内容定位清晰，与品牌一致
- [ ] 选题覆盖用户决策全链路
- [ ] 内容质量稳定，有编辑规范
- [ ] SEO基础优化到位
- [ ] 内容分发渠道覆盖目标用户

## 红牌警告

- 内容同质化缺乏差异化
- 追求流量忽视内容质量
- 忽视SEO导致内容无搜索流量
- 内容产出与分发渠道脱节
