# 定价策略 (Pricing Strategy)

## 触发条件
- 产品定价
- 价格调整
- 竞争定价
- 促销活动定价

## 输入
- 产品成本
- 市场竞争
- 用户价值感知
- 定价目标

## 输出
- 定价策略报告
- 价格体系
- 定价模型
- 价格调整方案

## Prompt模板
```
请为[产品/服务]制定定价策略。

背景：
- 产品信息：[信息]
- 成本结构：[成本]
- 市场竞争：[竞争]
- 定价目标：[目标]

请输出：
1. 定价目标和原则
2. 成本分析和底线价格
3. 市场竞争定价分析
4. 用户价值感知分析
5. 定价策略选择（成本加成/价值定价/竞争定价/渗透定价）
6. 价格体系和结构
7. 促销活动定价策略
8. 价格调整机制
```

## 示例
### 示例1
- 输入：SaaS产品，竞品定价$99/月，目标高端市场
- 输出：定价策略，含价值定价、高中低版本设计、年度优惠

## 注意事项
- 价格要反映价值
- 要考虑竞争
- 要有弹性空间
