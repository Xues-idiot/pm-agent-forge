# 账户营销 (Account-Based Marketing)

## 触发条件
- 大客户营销
- 个性化营销
- B2B营销
- 精准获客

## 输入
- 目标账户
- 业务类型
- 销售周期
- 竞争环境

## 输出
- 账户策略
- 个性化内容
- 多渠道触达
- 效果评估

## Prompt模板
```
请为[目标账户]制定账户营销策略。

背景：
- 目标账户：[账户]
- 业务类型：[类型]
- 销售周期：[周期]
- 竞争环境：[竞争]

请输出：
1. 账户分析
2. 账户分层
3. 个性化内容策略
4. 多渠道触达
5. 销售协同
6. 时间节点
7. 效果评估
```

## 示例
### 示例1
- 输入：大型企业采购，需切入5家目标客户
- 输出：ABM策略，含账户分析、个性化内容、多渠道触达

## 注意事项
- 要精准
- 要个性化
- 要销售协同
