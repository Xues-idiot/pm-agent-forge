# 客户管理 (Account Management)

## 触发条件
- 大客户管理
- 客户关系
- 销售扩展
- 客户满意度

## 输入
- 客户分层
- 服务模式
- 销售目标
- 竞争环境

## 输出
- 管理策略
- 服务标准
- 扩展方案
- 评估机制

## Prompt模板
```
请设计[客户]管理方案。

背景：
- 客户分层：[分层]
- 服务模式：[模式]
- 销售目标：[目标]
- 竞争环境：[环境]

请输出：
1. 客户战略定位
2. 分层服务标准
3. 客户经理角色
4. 关系管理
5. 销售扩展策略
6. 客户满意度
7. 风险管理
8. 效果评估
```

## 示例
### 示例1
- 输入：企业级客户，年度合同，需要提升ARPU
- 输出：管理方案，含分层服务、扩展销售、关系维护

## 注意事项
- 要以客户为中心
- 要个性化服务
- 要持续价值交付
