# 营销组合策略 (Marketing Mix)

## 触发条件
- 制定产品上市策略
- 优化渠道组合
- 定价策略调整
- 整合营销方案

## 输入
- 产品信息
- 市场环境
- 竞争对手
- 目标用户

## 输出
- 4P策略方案
- 营销组合优化建议
- 资源配置方案

## Prompt模板
```
请为[产品]制定营销组合策略（4P）。

产品信息：
- 产品名称：[名称]
- 产品特点：[核心卖点]
- 价格区间：[价格范围]
- 现有渠道：[渠道]
- 主要竞品：[竞品]

请输出完整的4P策略：
1. Product（产品）：产品定位、产品线规划、差异化策略
2. Price（价格）：定价策略、价格体系、价格调整机制
3. Place（渠道）：渠道策略、渠道管理、渠道创新
4. Promotion（促销）：促销策略、内容策略、整合营销
```

## 示例
### 示例1
- 输入：新款智能手表，价格1999-2999，竞品Apple Watch
- 输出：4P完整策略方案

## 注意事项
- 4P各要素要协调一致
- 要基于用户需求制定
- 要考虑竞争策略
