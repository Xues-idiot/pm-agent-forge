# 品牌策略 (Brand Strategy)

## 触发条件
- 制定品牌发展战略
- 进行品牌升级
- 品牌定位调整
- 新品牌创建

## 输入
- 企业愿景和使命
- 目标市场
- 竞争格局
- 品牌现状

## 输出
- 品牌战略报告
- 品牌定位陈述
- 品牌架构规划
- 品牌识别系统建议

## Prompt模板
```
作为品牌战略专家，请为[公司/产品]制定品牌发展战略。

企业背景：
- 愿景：[愿景]
- 业务：[核心业务]
- 市场：[目标市场]
- 竞品：[主要竞争者]

请输出：
1. 品牌诊断（现有品牌资产和问题）
2. 品牌定位（核心价值主张）
3. 品牌架构（单一/多品牌策略）
4. 品牌个性与调性
5. 品牌传播策略
```

## 示例
### 示例1
- 输入：新能源汽车品牌，定位高端市场，竞品是特斯拉
- 输出：品牌战略报告，包括"科技豪华"定位

## 注意事项
- 品牌定位要清晰差异化
- 要考虑长远发展
- 品牌架构要支撑业务战略
