# 社交媒体策略 (Social Media Strategy)

## 触发条件
- 制定社交媒体策略
- 社交账号矩阵规划
- KOL合作策略
- 社交内容规划

## 输入
- 品牌定位
- 目标平台
- 目标用户
- 竞争环境

## 输出
- 社交媒体策略
- 账号矩阵规划
- 内容策略
- KOL合作策略

## Prompt模板
```
请为[品牌]制定社交媒体策略。

背景：
- 品牌定位：[定位]
- 目标平台：[平台]
- 目标用户：[用户]
- 竞品社交：[竞品]

请输出：
1. 社交媒体目标设定
2. 平台选择和定位
3. 账号矩阵规划
4. 内容策略（内容 pillars/形式/频率）
5. KOL/KOC合作策略
6. 社群运营策略
7. 用户互动策略
8. 数据监测和效果评估
```

## 示例
### 示例1
- 输入：美妆品牌，目标用户是18-25岁女性，竞品在抖音/小红书表现好
- 输出：社交媒体策略，含双平台布局、达人种草矩阵、私域社群

## 注意事项
- 平台要选择适合的
- 内容要有差异化
- 数据要持续监测
