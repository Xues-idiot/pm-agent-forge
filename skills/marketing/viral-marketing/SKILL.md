# 病毒式营销 (Viral Marketing)

## Overview

病毒式营销是通过设计传播机制，让用户自愿分享传播，从而实现爆发式增长的技能。涵盖传播机制设计、激励体系、裂变路径、传播追踪等环节。

## 何时使用

- 产品冷启动需要快速增长
- 有高传播性的产品或内容
- 希望降低获客成本
- 竞品在增长黑客方面表现活跃

## 流程

1. **传播机制**：设计分享激励和传播路径
2. **裂变路径**：规划用户→新用户的转化路径
3. **激励设计**：设计邀请人和被邀请人的奖励
4. **素材准备**：制作传播素材和分享话术
5. **上线测试**：小规模测试验证传播效果
6. **数据监测**：追踪传播系数、裂变率
7. **优化迭代**：优化传播机制提升K因子

## 检查清单

- [ ] 传播机制简单明了，用户易理解
- [ ] 激励力度有吸引力但不过度
- [ ] 防刷机制健全
- [ ] 传播路径流畅，无断层
- [ ] 数据追踪完善，能计算K因子

## 红牌警告

- 奖励过度导致羊毛党泛滥
- 传播机制复杂用户不愿参与
- 缺乏防刷机制导致成本失控
- 忽视用户体验造成口碑反噬
