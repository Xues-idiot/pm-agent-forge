# 付费广告投放 (Paid Advertising)

## Overview

付费广告投放是通过搜索引擎、社交媒体、信息流等平台购买广告位，以获取精准流量的营销技能。涵盖投放策略、创意优化、竞价管理、效果归因等核心环节。

## 何时使用

- 需要快速获取精准流量时
- 产品冷启动阶段
- 促销活动推广
- 竞品关键词抢夺
- 品牌曝光扩大

## 流程

1. **目标确认**：明确KPI（ROI、CPA、GMV）
2. **平台选择**：根据目标用户画像选择投放平台
3. **投放策略**：制定定向、预算、出价策略
4. **创意制作**：设计广告文案和素材
5. **投放执行**：设置投放计划，上线广告
6. **效果监测**：实时监控数据，优化调整
7. **归因分析**：分析各渠道贡献度

## 检查清单

- [ ] 投放目标明确，KPI可衡量
- [ ] 人群定向精准，用户画像清晰
- [ ] 创意素材符合平台规范
- [ ] 落地页与广告创意一致
- [ ] 设置了转化跟踪和归因模型
- [ ] 预算分配合理，有梯度策略
- [ ] 竞品词包已覆盖

## 红牌警告

- 未设置转化跟踪导致效果无法衡量
- 投放平台与目标用户不匹配
- 落地页加载速度过慢影响转化
- 忽视广告疲劳度，创意不迭代
