# 内容营销策略 (Content Marketing Strategy)

## 触发条件
- 制定内容营销规划
- 内容矩阵规划
- 内容主题规划
- 内容差异化策略

## 输入
- 品牌定位
- 目标用户
- 业务目标
- 现有内容资产

## 输出
- 内容策略文档
- 内容主题日历
- 内容矩阵规划
- KPI体系

## Prompt模板
```
请为[品牌]制定内容营销策略。

背景信息：
- 品牌定位：[定位]
- 目标用户：[用户画像]
- 业务目标：[目标]
- 现有内容：[已有内容]

请输出：
1. 内容使命和核心信息
2. 内容 pillars（内容支柱）规划
3. 内容形式选择（图文/视频/直播等）
4. 内容日历规划（季度）
5. 各平台内容差异策略
6. 内容效果衡量指标
```

## 示例
### 示例1
- 输入：美妆品牌，目标用户是年轻女性，业务目标是提升品牌认知
- 输出：美妆内容营销策略，含"教程/评测/种草"三大内容支柱

## 注意事项
- 内容要有价值而非纯粹广告
- 要建立内容差异化
- 要形成内容资产积累
