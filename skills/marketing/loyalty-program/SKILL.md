# 会员忠诚度 (Loyalty Program)

## 触发条件
- 会员体系设计
- 积分策略
- 会员运营
- 会员价值提升

## 输入
- 业务类型
- 客户特征
- 竞争环境
- 运营能力

## 输出
- 会员体系
- 积分规则
- 会员权益
- 运营策略

## Prompt模板
```
请为[品牌]设计会员忠诚度方案。

背景：
- 业务类型：[类型]
- 客户特征：[特征]
- 竞争环境：[竞争]
- 运营能力：[能力]

请输出：
1. 会员体系定位
2. 会员等级设计
3. 积分机制
4. 会员权益设计
5. 升级保级规则
6. 会员获取策略
7. 会员运营策略
8. 数据分析体系
9. 技术支撑
```

## 示例
### 示例1
- 输入：连锁咖啡品牌，有100万普通用户，需要提升复购
- 输出：会员方案，含银金钻等级、积分兑换、生日权益

## 注意事项
- 权益要差异化
- 升级要可感知
- 成本要可控
