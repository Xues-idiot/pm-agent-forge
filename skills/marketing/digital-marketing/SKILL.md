# 数字营销 (Digital Marketing)

## 触发条件
- 制定数字营销策略
- 整合数字渠道
- 效果营销优化
- 数据驱动营销

## 输入
- 营销目标
- 目标用户
- 预算范围
- 现有渠道

## 输出
- 数字营销策略
- 渠道组合方案
- 投放计划
- 效果评估体系

## Prompt模板
```
请为[品牌/产品]制定数字营销策略。

背景：
- 营销目标：[目标]
- 目标用户：[用户]
- 预算范围：[预算]
- 现有数字渠道：[渠道]

请输出：
1. 数字营销目标拆解
2. 目标用户数字画像
3. 渠道组合策略（SEM/SEO/社交/内容/DSP等）
4. 各渠道预算分配
5. 投放策略和创意方向
6. 数据监测和归因方案
7. 效果评估KPI体系
8. 优化策略和迭代计划
```

## 示例
### 示例1
- 输入：新品上市，ToB产品，预算100万，首次进入中国市场
- 输出：数字营销策略，含SEM、本地化社交媒体、行业垂直平台

## 注意事项
- 渠道要整合而非孤立
- 数据要打通
- 效果要可衡量
