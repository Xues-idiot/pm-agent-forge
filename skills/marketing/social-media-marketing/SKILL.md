# 社交媒体营销 (Social Media Marketing)

## Overview

社交媒体营销是利用微博、微信、抖音、小红书等社交平台进行品牌推广、用户互动、内容传播的技能。涵盖平台选择、内容规划、KOL合作、数据运营等环节。

## 何时使用

- 品牌需要建立社交媒体影响力
- 产品需要通过社交渠道推广
- 需要与用户建立互动关系
- 竞品在社交平台表现活跃
- 需要UGC内容沉淀

## 流程

1. **平台选择**：根据目标用户选择核心平台
2. **账号规划**：确定账号定位、人设、内容方向
3. **内容规划**：制定月度内容日历
4. **内容制作**：产出图文、短视频、直播等内容
5. **发布运营**：保持发布频率，互动回复
6. **数据复盘**：分析阅读量、互动率、转化率

## 检查清单

- [ ] 账号定位清晰，内容风格统一
- [ ] 内容日历完整，发布时间规律
- [ ] 视觉风格符合品牌调性
- [ ] 互动话术有特色，回复及时
- [ ] KOL合作有效果评估机制
- [ ] 数据复盘周期性执行

## 红牌警告

- 内容风格混乱，缺乏统一人设
- 发布频率不稳定
- 只发不管，忽视用户评论互动
- 数据复盘流于形式
