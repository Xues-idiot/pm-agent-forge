# 客户体验 (Customer Experience)

## 触发条件
- 体验战略
- 触点优化
- 体验设计
- 反馈闭环

## 输入
- 客户旅程
- 触点现状
- 体验目标
- 资源投入

## 输出
- 体验战略
- 触点设计
- 监测体系
- 改进机制

## Prompt模板
```
请设计[业务]客户体验方案。

背景：
- 客户旅程：[旅程]
- 触点现状：[现状]
- 体验目标：[目标]
- 资源投入：[资源]

请输出：
1. 体验战略
2. 客户旅程
3. 触点设计
4. 体验标准
5. 反馈机制
6. 监测体系
7. 持续改进
```

## 示例
### 示例1
- 输入：电商平台，需要提升购物体验
- 输出：体验方案，含旅程优化、界面设计、NPS监测

## 注意事项
- 要客户视角
- 要全链路
- 要持续改进
