# 渠道策略 (Channel Strategy)

## 触发条件
- 渠道规划
- 渠道管理
- 渠道激励
- 渠道优化

## 输入
- 产品类型
- 渠道现状
- 目标市场
- 竞争环境

## 输出
- 渠道策略
- 渠道设计
- 激励方案
- 管理机制

## Prompt模板
```
请制定[产品]渠道策略。

背景：
- 产品类型：[类型]
- 渠道现状：[现状]
- 目标市场：[市场]
- 竞争环境：[环境]

请输出：
1. 渠道目标
2. 渠道模式选择
3. 渠道结构设计
4. 渠道伙伴招募
5. 激励政策
6. 渠道管理
7. 渠道支持
8. 渠道评估
```

## 示例
### 示例1
- 输入：消费品新品，需要快速覆盖全国
- 输出：渠道策略，含经销商+终端+电商组合

## 注意事项
- 要适配产品
- 要利益平衡
- 要持续优化
