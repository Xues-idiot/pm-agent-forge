# 需求生成 (Demand Generation)

## 触发条件
- 获客增长
- 营销漏斗
- 线索管理
- ROI优化

## 输入
- 目标客户
- 营销渠道
- 线索目标
- 转化路径

## 输出
- 获客策略
- 渠道组合
- 线索管理
- 效果分析

## Prompt模板
```
请制定[产品]需求生成策略。

背景：
- 目标客户：[客户]
- 营销渠道：[渠道]
- 线索目标：[目标]
- 转化路径：[路径]

请输出：
1. 获客目标
2. 渠道策略
3. 内容营销
4. 全渠道整合
5. 线索管理
6. 营销技术
7. 效果分析
```

## 示例
### 示例1
- 输入：SaaS产品，需要增加MQL
- 输出：获客策略，含内容营销、SEO、付费线索

## 注意事项
- 要精准获客
- 要渠道整合
- 要持续优化
