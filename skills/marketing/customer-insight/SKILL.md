# 客户洞察 (Customer Insight)

## 触发条件
- 理解目标用户
- 用户研究分析
- 消费者行为研究
- 用户画像构建

## 输入
- 市场调研数据
- 用户行为数据
- 访谈/调研原始材料
- 用户反馈

## 输出
- 用户画像报告
- 消费者行为洞察
- 用户需求分析
- 机会点建议

## Prompt模板
```
请基于以下数据进行客户洞察分析：

数据来源：
- 用户调研：[调研内容]
- 行为数据：[行为数据摘要]
- 用户反馈：[反馈摘要]

请输出：
1. 用户画像（人口属性/心理属性/行为特征）
2. 用户旅程地图（认知-考虑-决策-使用-推荐）
3. 用户核心需求和痛点
4. 用户决策因素优先级
5. 潜在机会点和建议
```

## 示例
### 示例1
- 输入：电商平台用户数据，Z世代消费者
- 输出：Z世代用户画像和消费洞察

## 注意事项
- 洞察要基于数据而非假设
- 要区分显性需求和隐性需求
- 要关注用户决策链路
