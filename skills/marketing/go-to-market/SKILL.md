# 市场进入GTM (Go-To-Market)

## 触发条件
- 新品上市
- 新市场进入
- 新业务推广
- 快速增长

## 输入
- 产品定位
- 目标客户
- 竞争优势
- 资源投入

## 输出
- GTM策略
- 执行计划
- 里程碑
- 指标体系

## Prompt模板
```
请制定[产品]GTM策略。

背景：
- 产品定位：[定位]
- 目标客户：[客户]
- 竞争优势：[优势]
- 资源投入：[资源]

请输出：
1. 市场定位
2. 目标客户
3. 价值主张
4. 渠道策略
5. 营销策略
6. 销售策略
7. 指标体系
8. 实施路径
```

## 示例
### 示例1
- 输入：企业级SaaS产品，需要快速获客
- 输出：GTM策略，含ICP、获客渠道、销售赋能

## 注意事项
- 要精准定位
- 要高效执行
- 要快速迭代
