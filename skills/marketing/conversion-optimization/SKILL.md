# 转化优化 (Conversion Optimization)

## 触发条件
- 转化率提升
- 漏斗优化
- 用户体验优化
- A/B测试

## 输入
- 转化漏斗
- 当前转化率
- 优化目标
- 技术能力

## 输出
- 优化方案
- 测试计划
- 优先级排序
- 效果预测

## Prompt模板
```
请优化[转化漏斗]。

背景：
- 漏斗阶段：[阶段]
- 当前转化率：[比例]
- 优化目标：[目标]
- 技术能力：[能力]

请输出：
1. 漏斗诊断
2. 用户行为分析
3. 优化机会识别
4. 优化方案设计
5. A/B测试计划
6. 技术实现方案
7. 优先级排序
8. 效果跟踪
```

## 示例
### 示例1
- 输入：电商下单漏斗，从加购到付款转化率60%
- 输出：优化方案，含结算页优化、支付方式、物流选项优化

## 注意事项
- 要数据驱动
- 要小步快跑
- 要关注全链路
