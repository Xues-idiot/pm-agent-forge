# 邮件营销 (Email Marketing)

## Overview

邮件营销是通过电子邮件向目标用户传递价值信息、实现转化目标的技能。涵盖用户分群、邮件设计、自动化流程、A/B测试等核心环节。

## 何时使用

- 用户生命周期运营
- 沉默用户唤醒
- 大促活动通知
- 新产品上线推广
- 会员权益通知

## 流程

1. **用户分群**：根据用户属性和行为进行分群
2. **内容策划**：设计邮件主题和内容
3. **模板设计**：制作邮件视觉模板
4. **自动化流程**：设置触发规则和发送流程
5. **发送执行**：控制发送时间、数量
6. **效果监测**：追踪打开率、点击率、转化率
7. **优化迭代**：基于数据优化内容和策略

## 模板

### 邮件营销规划模板
```
## 用户分群
| 人群 | 特征 | 发送策略 |
|------|------|----------|
| 新用户 | 注册7天内 | 欢迎引导 |
| 活跃用户 | 30天内有行为 | 活动通知 |
| 沉睡用户 | 90天无行为 | 唤醒刺激 |

## 邮件类型
- 欢迎邮件: 触发条件[注册成功]
- 促活邮件: 触发条件[30天未登录]
- 召回邮件: 触发条件[60天未消费]
- 大促邮件: 触发条件[活动前3天]

## 核心指标
- 送达率目标: >95%
- 打开率目标: >20%
- 点击率目标: >3%
- 转化率目标: >1%
```

## 检查清单

- [ ] 邮件标题有吸引力，避免垃圾邮件关键词
- [ ] 发送频率合理，不过度打扰用户
- [ ] 落地页与邮件内容一致
- [ ] 退订链接清晰，符合法规
- [ ] 自动化流程触发条件明确
- [ ] 做好数据追踪和效果评估

## 红牌警告

- 发送频率过高导致用户反感
- 邮件内容与用户预期不符
- 忽视移动端适配
- 触发规则设置错误导致错发
