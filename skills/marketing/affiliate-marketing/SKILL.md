# 联盟营销 (Affiliate Marketing)

## Overview

联盟营销是通过招募分销商、CPS合作、渠道分成等方式，借助合作伙伴资源实现销售增长的技能。涵盖伙伴招募、分佣设计、渠道管理、数据追踪等环节。

## 何时使用

- 需要快速扩大销售渠道
- 自身流量有限需借助外部渠道
- 希望降低获客成本，按效果付费
- 产品有足够毛利空间支持分佣

## 流程

1. **分佣策略**：确定佣金比例和结算方式
2. **伙伴招募**：招募分销商、KOL、渠道商
3. **物料提供**：提供推广链接、文案素材
4. **渠道管理**：培训、指导、激励渠道
5. **数据追踪**：追踪各渠道销售数据
6. **佣金结算**：按效果结算佣金
7. **优化迭代**：优化分佣策略和渠道结构

## 检查清单

- [ ] 分佣比例有竞争力，能吸引优质渠道
- [ ] 推广物料齐全，方便渠道使用
- [ ] 数据追踪系统完善，佣金计算透明
- [ ] 渠道培训到位，推广方式规范
- [ ] 防止刷单和作弊机制健全

## 红牌警告

- 分佣比例过高导致亏损
- 缺乏防作弊机制导致资源浪费
- 结算周期不清晰引发渠道纠纷
