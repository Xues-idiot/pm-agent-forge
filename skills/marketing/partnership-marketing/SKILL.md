# 合作营销 (Partnership Marketing)

## 触发条件
- 品牌合作
- 跨界营销
- 渠道合作
- 联名活动

## 输入
- 合作目标
- 伙伴类型
- 合作形式
- 资源投入

## 输出
- 合作策略
- 伙伴筛选
- 合作方案
- 效果评估

## Prompt模板
```
请设计[品牌]合作营销方案。

背景：
- 合作目标：[目标]
- 伙伴类型：[类型]
- 合作形式：[形式]
- 资源投入：[投入]

请输出：
1. 合作战略定位
2. 伙伴画像
3. 合作形式设计
4. 价值交换
5. 权益分配
6. 执行计划
7. 风险控制
8. 效果评估
```

## 示例
### 示例1
- 输入：茶饮品牌，想与奶茶联名推广
- 输出：合作方案，含联名产品、渠道共享、传播计划

## 注意事项
- 要门当户对
- 要互补共赢
- 要执行到位
