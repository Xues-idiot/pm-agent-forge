# 营销活动设计 (Campaign Design)

## 触发条件
- 制定季度/年度营销计划
- 设计新品上市活动
- 规划节日营销活动
- 设计促销方案

## 输入
- 活动目标（品牌/转化/获客）
- 目标用户画像
- 预算范围
- 时间周期

## 输出
- 活动策划方案文档
- 渠道投放计划
- 时间节点表
- 效果预估

## Prompt模板
```
作为营销策划专家，请为[产品/品牌]设计一场[活动主题]营销活动。

活动背景：
- 目标：[具体目标]
- 目标用户：[用户画像]
- 预算：[X万元]
- 时间：[周期]

请输出完整的活动策划案，包括：
1. 活动主题和核心创意
2. 目标拆解（曝光/互动/转化）
3. 活动玩法设计
4. 渠道策略和预算分配
5. 节奏安排和时间节点
6. 预期效果和衡量指标
```

## 示例
### 示例1
- 输入：双11电商大促活动，预算500万，目标GMV 5000万
- 输出：完整双11活动策划案，含预热/爆发/返场三阶段

## 注意事项
- 创意要有差异化，避免同质化
- 预算分配要合理，聚焦核心渠道
- 效果指标要可衡量
