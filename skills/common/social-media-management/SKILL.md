# 社交媒体管理 (Social Media Management)

## 触发条件
- 社交运营
- 内容发布
- 粉丝互动
- 品牌建设

## 输入
- 社交平台
- 内容策略
- 发布计划
- 互动资源

## 输出
- 内容规划
- 发布日历
- 互动策略
- 效果评估

## Prompt模板
```
请管理社交媒体。

背景：
- 社交平台：[平台]
- 内容策略：[策略]
- 发布计划：[计划]
- 互动资源：[资源]

请输出：
1. 内容规划
2. 发布日历
3. 互动策略
4. 效果评估
5. 危机处理
```

## 示例
### 示例1
- 输入：品牌社交
- 输出：社交管理，内容矩阵、KOL合作、粉丝互动

## 注意事项
- 要内容质量
- 要持续更新
- 要粉丝互动
- 要数据监控
