# 问题升级 (Issue Escalation)

## 触发条件
- 问题识别
- 升级标准
- 升级流程
- 解决机制

## 输入
- 问题类型
- 影响程度
- 升级标准
- 处理能力

## 输出
- 问题分类
- 升级标准
- 升级流程
- 解决机制

## Prompt模板
```
请设计问题升级机制。

背景：
- 问题类型：[类型]
- 影响程度：[程度]
- 升级标准：[标准]
- 处理能力：[能力]

请输出：
1. 问题分类
2. 升级标准
3. 升级流程
4. 处理机制
5. 闭环验证
```

## 示例
### 示例1
- 输入：客服体系，需要升级机制
- 输出：升级机制，含分级标准、升级路径、处理时效

## 注意事项
- 要标准清晰
- 要及时升级
- 要闭环处理
