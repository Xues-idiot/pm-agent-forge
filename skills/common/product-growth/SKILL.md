# 产品增长 (Product Growth)

## 触发条件
- 用户增长
- 活跃度
- 留存提升
- 商业变现

## 输入
- 增长目标
- 当前漏斗
- 用户行为
- 增长资源

## 输出
- 增长策略
- 实验计划
- 指标体系
- 迭代路径

## Prompt模板
```
请制定产品增长策略。

背景：
- 增长目标：[目标]
- 当前漏斗：[漏斗]
- 用户行为：[行为]
- 增长资源：[资源]

请输出：
1. 增长策略
2. 实验计划
3. 指标体系
4. 迭代路径
5. 资源需求
```

## 示例
### 示例1
- 输入：互联网产品
- 输出：产品增长，获客、激活、留存、变现、推荐

## 注意事项
- 要数据驱动
- 要快速实验
- 要全漏斗
- 要持续优化
