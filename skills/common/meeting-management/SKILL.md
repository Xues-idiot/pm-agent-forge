# 会议管理 (Meeting Management)

## 触发条件
- 会议效率
- 决策质量
- 时间控制
- 会议组织

## 输入
- 会议目的
- 参会人员
- 议程设计
- 时间安排

## 输出
- 会议设计
- 议程安排
- 主持要点
- 跟进机制

## Prompt模板
```
请管理会议。

背景：
- 会议目的：[目的]
- 参会人员：[人员]
- 议程设计：[议程]
- 时间安排：[时间]

请输出：
1. 会议设计
2. 议程安排
3. 主持要点
4. 决策记录
5. 跟进机制
```

## 示例
### 示例1
- 输入：周例会
- 输出：会议管理，明确议程、时间控制、决策记录、任务跟进

## 注意事项
- 要有明确目的
- 要有议程
- 要控制时间
- 要有行动项
