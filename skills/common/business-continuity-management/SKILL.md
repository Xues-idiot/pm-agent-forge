# 业务连续性管理 (Business Continuity Management)

## 触发条件
- 业务中断
- 灾难应对
- 恢复计划
- 韧性建设

## 输入
- 关键业务
- 威胁分析
- 恢复需求
- 资源状况

## 输出
- BCM框架
- 业务影响
- 恢复策略
- 测试演练

## Prompt模板
```
请进行业务连续性管理。

背景：
- 关键业务：[业务]
- 威胁分析：[威胁]
- 恢复需求：[需求]
- 资源状况：[资源]

请输出：
1. BCM框架
2. 业务影响分析
3. 恢复策略
4. 测试演练
5. 维护机制
```

## 示例
### 示例1
- 输入：企业BCM
- 输出：业务连续性管理， BIA、BCP、演练计划

## 注意事项
- 要高层支持
- 要全员参与
- 要定期演练
- 要持续改进
