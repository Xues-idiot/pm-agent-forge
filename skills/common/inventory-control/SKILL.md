# 库存控制 (Inventory Control)

## 触发条件
- 库存管理
- 资金占用
- 缺货风险
- 损耗控制

## 输入
- 库存结构
- 周转率
- 需求预测
- 补货策略

## 输出
- 库存策略
- 补货计划
- 风险预警
- 成本优化

## Prompt模板
```
请控制库存。

背景：
- 库存结构：[结构]
- 周转率：[周转]
- 需求预测：[预测]
- 补货策略：[策略]

请输出：
1. 库存策略
2. 补货计划
3. 风险预警
4. 成本优化
5. 监控指标
```

## 示例
### 示例1
- 输入：零售库存
- 输出：库存控制，ABC分类、安全库存、JIT补货

## 注意事项
- 要供需平衡
- 要分类管理
- 要数据驱动
- 要持续优化
