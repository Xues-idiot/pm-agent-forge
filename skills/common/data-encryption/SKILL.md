# 数据加密 (Data Encryption)

## 触发条件
- 数据保护
- 隐私合规
- 密钥管理
- 传输安全

## 输入
- 数据类型
- 保护级别
- 合规要求
- 性能影响

## 输出
- 加密策略
- 算法选择
- 密钥管理
- 实施方案

## Prompt模板
```
请设计数据加密。

背景：
- 数据类型：[类型]
- 保护级别：[级别]
- 合规要求：[要求]
- 性能影响：[性能]

请输出：
1. 加密策略
2. 算法选择
3. 密钥管理
4. 实施方案
5. 监控机制
```

## 示例
### 示例1
- 输入：敏感数据
- 输出：数据加密，传输加密、存储加密、密钥轮换

## 注意事项
- 要分层加密
- 要密钥安全
- 要性能考虑
- 要合规要求
