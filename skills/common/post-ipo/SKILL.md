# 上市后管理 (Post-IPO Management)

## 触发条件
- 上市公司
- 投资者关系
- 业绩披露
- 股价管理

## 输入
- 投资者
- 业绩目标
- 披露要求
- 市场预期

## 输出
- IR策略
- 披露管理
- 预期管理
- 股价沟通

## Prompt模板
```
请管理上市公司。

背景：
- 投资者：[投资者]
- 业绩目标：[目标]
- 披露要求：[披露]
- 市场预期：[预期]

请输出：
1. IR策略
2. 披露管理
3. 预期管理
4. 股价沟通
5. 合规要求
```

## 示例
### 示例1
- 输入：上市公司
- 输出：上市后管理，季度业绩、投资者路演、分析师沟通

## 注意事项
- 要透明披露
- 要预期管理
- 要长期价值
- 要合规要求
