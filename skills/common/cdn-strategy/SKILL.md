# CDN策略 (CDN Strategy)

## 触发条件
- 静态资源
- 访问加速
- 成本优化
- 全球分发

## 输入
- 内容类型
- 访问特点
- 覆盖区域
- 成本预算

## 输出
- CDN架构
- 缓存策略
- 刷新机制
- 成本优化

## Prompt模板
```
请制定CDN策略。

背景：
- 内容类型：[类型]
- 访问特点：[特点]
- 覆盖区域：[区域]
- 成本预算：[预算]

请输出：
1. CDN架构
2. 缓存策略
3. 刷新机制
4. 防盗链
5. 成本优化
```

## 示例
### 示例1
- 输入：视频网站
- 输出：CDN策略，含多CDN回源、热点缓存、防盗链、流量费用优化

## 注意事项
- 要合理缓存
- 要及时刷新
- 要防盗链
- 要成本控制
