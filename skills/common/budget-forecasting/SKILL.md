# 预算预测 (Budget Forecasting)

## 触发条件
- 预算编制
- 资源规划
- 目标设定
- 绩效考核

## 输入
- 历史数据
- 业务计划
- 假设条件
- 约束因素

## 输出
- 预算方案
- 分配计划
- 差异分析
- 监控机制

## Prompt模板
```
请编制预算预测。

背景：
- 历史数据：[历史]
- 业务计划：[计划]
- 假设条件：[假设]
- 约束因素：[约束]

请输出：
1. 预算方案
2. 分配计划
3. 差异分析
4. 监控机制
5. 调整流程
```

## 示例
### 示例1
- 输入：年度预算
- 输出：预算预测，收入预算、成本预算、利润预算

## 注意事项
- 要上下结合
- 要合理假设
- 要弹性设计
- 要动态调整
