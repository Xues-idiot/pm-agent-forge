# 指标看板 (Metric Dashboard)

## 触发条件
- KPI监控
- 目标追踪
- 异常预警
- 团队协同

## 输入
- 核心指标
- 目标值
- 当前状态
- 预警阈值

## 输出
- 指标看板
- 目标进度
- 趋势分析
- 行动建议

## Prompt模板
```
请设计指标看板。

背景：
- 核心指标：[指标]
- 目标值：[目标]
- 当前状态：[状态]
- 预警阈值：[阈值]

请输出：
1. 指标看板
2. 目标进度
3. 趋势分析
4. 行动建议
5. 责任归属
```

## 示例
### 示例1
- 输入：销售团队
- 输出：指标看板，目标完成率、区域排名、趋势对比

## 注意事项
- 要核心指标
- 要实时更新
- 要可视化
- 要行动导向
