# 战略联盟 (Strategic Alliance)

## 触发条件
- 合作伙伴
- 资源整合
- 市场拓展
- 风险分担

## 输入
- 合作目标
- 潜在伙伴
- 合作模式
- 资源互补

## 输出
- 合作策略
- 伙伴选择
- 合作模式
- 治理机制

## Prompt模板
```
请制定战略联盟策略。

背景：
- 合作目标：[目标]
- 潜在伙伴：[伙伴]
- 合作模式：[模式]
- 资源互补：[互补]

请输出：
1. 合作目标
2. 伙伴选择标准
3. 合作模式设计
4. 治理机制
5. 退出条款
```

## 示例
### 示例1
- 输入：技术合作
- 输出：战略联盟，股权合作、合资公司、战略合作三种模式

## 注意事项
- 要目标一致
- 要资源互补
- 要利益共享
- 要风险共担
