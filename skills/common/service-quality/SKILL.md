# 服务质量 (Service Quality)

## 触发条件
- 客户服务
- 满意度提升
- 服务标准
- 体验优化

## 输入
- 服务流程
- 客户反馈
- 服务指标
- 行业标杆

## 输出
- 服务标准
- 监控体系
- 改进方案
- 培训计划

## Prompt模板
```
请提升服务质量。

背景：
- 服务流程：[流程]
- 客户反馈：[反馈]
- 服务指标：[指标]
- 行业标杆：[标杆]

请输出：
1. 服务标准
2. 监控指标
3. 改进方案
4. 培训计划
5. 考核机制
```

## 示例
### 示例1
- 输入：客服部门
- 输出：服务质量方案，含响应时效、解决率、满意度、首次解决率

## 注意事项
- 要客户视角
- 要可衡量
- 要持续跟进
- 要奖励优秀
