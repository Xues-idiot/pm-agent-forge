# API战略 (API Strategy)

## 触发条件
- 开放平台
- 生态建设
- 开发者生态
- 商业模式

## 输入
- API资源
- 开发者需求
- 生态目标
- 商业模式

## 输出
- API设计
- 开发者体验
- 生态治理
- 盈利模式

## Prompt模板
```
请制定API战略。

背景：
- API资源：[资源]
- 开发者需求：[需求]
- 生态目标：[目标]
- 商业模式：[模式]

请输出：
1. API设计原则
2. 开发者体验
3. 生态治理
4. 盈利模式
5. 演进规划
```

## 示例
### 示例1
- 输入：支付API
- 输出：API战略，开放支付能力、开发者门户、分成模式

## 注意事项
- 要开发者导向
- 要安全可靠
- 要持续演进
- 要商业模式清晰
