# 应急规划 (Emergency Planning)

## 触发条件
- 应急预案
- 灾害应对
- 人员安全
- 快速响应

## 输入
- 风险类型
- 影响范围
- 响应资源
- 疏散需求

## 输出
- 应急预案
- 响应流程
- 疏散路线
- 资源调配

## Prompt模板
```
请进行应急规划。

背景：
- 风险类型：[类型]
- 影响范围：[范围]
- 响应资源：[资源]
- 疏散需求：[需求]

请输出：
1. 应急预案
2. 响应流程
3. 疏散路线
4. 资源调配
5. 演练计划
```

## 示例
### 示例1
- 输入：自然灾害
- 输出：应急规划，预警机制、疏散方案、救援资源

## 注意事项
- 要定期演练
- 要更新维护
- 要人员培训
- 要社区联动
