# 故障复盘 (Postmortem)

## 触发条件
- 故障总结
- 根因分析
- 改进措施
- 经验传承

## 输入
- 故障过程
- 影响范围
- 处理时效
- 团队反馈

## 输出
- 时间线
- 根因分析
- 改进措施
- 行动项

## Prompt模板
```
请进行故障复盘。

背景：
- 故障过程：[过程]
- 影响范围：[范围]
- 处理时效：[时效]
- 团队反馈：[反馈]

请输出：
1. 故障概述
2. 时间线
3. 根因分析
4. 改进措施
5. 行动项跟踪
```

## 示例
### 示例1
- 输入：重大故障复盘
- 输出：复盘报告，含故障过程、5Why分析、短期和长期改进、责任人

## 注意事项
- 要不追责
- 要找根因
- 要改进实际
- 要文档沉淀
