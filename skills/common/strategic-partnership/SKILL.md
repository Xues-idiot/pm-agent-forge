# 战略合作 (Strategic Partnership)

## 触发条件
- 合作伙伴
- 合作模式
- 价值创造
- 关系管理

## 输入
- 合作目标
- 伙伴类型
- 资源互补
- 合作深度

## 输出
- 合作策略
- 伙伴选择
- 合作模式
- 关系管理

## Prompt模板
```
请设计[合作方]战略合作方案。

背景：
- 合作目标：[目标]
- 伙伴类型：[类型]
- 资源互补：[互补]
- 合作深度：[深度]

请输出：
1. 合作战略定位
2. 伙伴画像
3. 合作模式
4. 价值分配
5. 治理机制
6. 风险管理
```

## 示例
### 示例1
- 输入：科技公司，与云厂商战略合作
- 输出：合作方案，含技术集成、市场推广、收入分成

## 注意事项
- 要互补共赢
- 要长期导向
- 要治理清晰
