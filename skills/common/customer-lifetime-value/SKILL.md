# 客户生命周期价值 (Customer Lifetime Value)

## 触发条件
- LTV计算
- 客户价值评估
- 资源分配
- 投资回报

## 输入
- 客户数据
- 购买行为
- 留存周期
- 成本结构

## 输出
- LTV计算模型
- 客户分层
- 资源策略
- 优化建议

## Prompt模板
```
请计算客户生命周期价值。

背景：
- 客户数据：[数据]
- 购买行为：[行为]
- 留存周期：[周期]
- 成本结构：[成本]

请输出：
1. LTV计算模型
2. 客户价值分层
3. 获取成本建议
4. 资源分配策略
5. 提升LTV路径
```

## 示例
### 示例1
- 输入：电商平台，客户复购数据
- 输出：LTV模型，含平均订单价值×年购买频次×平均生命周期×毛利率

## 注意事项
- 要准确核算成本
- 要考虑留存率
- 要动态计算
- 要指导资源配置
