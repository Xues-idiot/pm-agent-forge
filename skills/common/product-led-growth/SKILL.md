# 产品驱动增长 (Product-Led Growth)

## 触发条件
- PLG策略
- 产品营销
- 用户口碑
- 病毒传播

## 输入
- 产品特点
- 用户体验
- 增长指标
- 资源投入

## 输出
- PLG策略
- 获客机制
- 转化策略
- 留存策略

## Prompt模板
```
请实施产品驱动增长。

背景：
- 产品特点：[特点]
- 用户体验：[体验]
- 增长指标：[指标]
- 资源投入：[资源]

请输出：
1. PLG策略
2. 获客机制
3. 激活策略
4. 留存策略
5. 商业转化
```

## 示例
### 示例1
- 输入：SaaS产品
- 输出：PLG策略，免费试用、内置推荐、团队协作、升级转化

## 注意事项
- 要产品优先
- 要用户导向
- 要数据驱动
- 要持续优化
