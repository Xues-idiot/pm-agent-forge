# 合资经营 (Joint Venture)

## 触发条件
- 战略合作
- 资源整合
- 风险分担
- 市场进入

## 输入
- 合作目标
- 合作方
- 出资比例
- 治理结构

## 输出
- 合资策略
- 股权设计
- 治理结构
- 退出机制

## Prompt模板
```
请制定合资策略。

背景：
- 合作目标：[目标]
- 合作方：[合作方]
- 出资比例：[比例]
- 治理结构：[治理]

请输出：
1. 合资策略
2. 股权设计
3. 治理结构
4. 运营机制
5. 退出机制
```

## 示例
### 示例1
- 输入：中外合资
- 输出：合资经营，50/50股比，董事会领导，管理层执行

## 注意事项
- 要互利共赢
- 要治理清晰
- 要风险共担
- 要退出机制
