# 数据可视化 (Data Visualization)

## 触发条件
- 数据展示
- 报告制作
- 故事讲述
- 决策支持

## 输入
- 数据内容
- 受众需求
- 展示场景
- 图表类型

## 输出
- 可视化方案
- 图表选择
- 布局设计
- 交互设计

## Prompt模板
```
请设计数据可视化。

背景：
- 数据内容：[内容]
- 受众需求：[受众]
- 展示场景：[场景]
- 图表类型：[类型]

请输出：
1. 可视化方案
2. 图表选择
3. 布局设计
4. 交互设计
5. 技术选型
```

## 示例
### 示例1
- 输入：销售报表
- 输出：数据可视化，趋势图、对比图、地图热力图

## 注意事项
- 要受众导向
- 要简洁清晰
- 要故事线
- 要适合场景
