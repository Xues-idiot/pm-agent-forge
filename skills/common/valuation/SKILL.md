# 估值分析 (Valuation)

## 触发条件
- 估值方法
- 投资决策
- 融资定价
- 并购交易

## 输入
- 财务数据
- 增长预期
- 风险因素
- 市场环境

## 输出
- 估值方法
- 估值结果
- 估值区间
- 投资建议

## Prompt模板
```
请进行估值分析。

背景：
- 财务数据：[数据]
- 增长预期：[增长]
- 风险因素：[风险]
- 市场环境：[市场]

请输出：
1. 估值方法选择
2. 估值模型
3. 估值结果
4. 估值区间
5. 投资建议
```

## 示例
### 示例1
- 输入：科技公司
- 输出：估值分析，DCF 10亿、PS 10倍、市值区间8-12亿

## 注意事项
- 要方法适配
- 要假设合理
- 要多方法验证
- 要动态调整
