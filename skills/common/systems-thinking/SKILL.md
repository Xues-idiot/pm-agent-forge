# 系统思维 (Systems Thinking)

## 触发条件
- 复杂问题分析
- 因果关系分析
- 反馈循环识别
- 杠杆点分析

## 输入
- 系统类型
- 问题表现
- 涉及要素
- 时间范围

## 输出
- 系统地图
- 因果关系
- 反馈循环
- 干预建议

## Prompt模板
```
请用系统思维分析[系统/问题]。

背景：
- 系统类型：[类型]
- 问题表现：[表现]
- 涉及要素：[要素]
- 时间范围：[范围]

请输出：
1. 系统边界定义
2. 系统要素识别
3. 因果关系图
4. 反馈循环分析
5. 系统行为模式
6. 杠杆点识别
7. 干预建议
```

## 示例
### 示例1
- 输入：城市交通拥堵问题
- 输出：系统分析，含利益相关方、反馈循环、干预点

## 注意事项
- 要看整体
- 要看长远
- 要识别反馈
