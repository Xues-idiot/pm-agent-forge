# 合作策略 (Partnership Strategy)

## 触发条件
- 合作伙伴
- 生态建设
- 资源整合
- 互惠共赢

## 输入
- 合作目标
- 潜在伙伴
- 合作模式
- 资源互补

## 输出
- 合作策略
- 伙伴分级
- 合作模式
- 落地计划

## Prompt模板
```
请制定合作策略。

背景：
- 合作目标：[目标]
- 潜在伙伴：[伙伴]
- 合作模式：[模式]
- 资源互补：[互补]

请输出：
1. 合作策略
2. 伙伴分级体系
3. 合作模式设计
4. 落地计划
5. 评估机制
```

## 示例
### 示例1
- 输入：企业软件与硬件厂商合作
- 输出：合作策略，含技术集成、联合营销、渠道分成、联合创新

## 注意事项
- 要互惠互利
- 要明确权责
- 要风险共担
- 要持续经营
