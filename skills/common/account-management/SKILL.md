# 大客户管理 (Account Management)

## 触发条件
- 大客户维护
- 客户深耕
- 增购机会
- 风险客户

## 输入
- 客户价值
- 客户关系
- 购买历史
- 组织架构

## 输出
- 客户策略
- 关系维护计划
- 增购机会
- 风险预警

## Prompt模板
```
请制定大客户管理策略。

背景：
- 客户价值：[价值]
- 客户关系：[关系]
- 购买历史：[历史]
- 组织架构：[架构]

请输出：
1. 客户策略
2. 关键决策人关系
3. 增购机会识别
4. 风险预警机制
5. 客户成功计划
```

## 示例
### 示例1
- 输入：企业软件大客户，年消费100万+
- 输出：大客户策略，含季度业务回顾、年度战略对齐、联合创新计划

## 注意事项
- 要深入了解客户
- 要建立信任关系
- 要创造额外价值
- 要提前识别风险
