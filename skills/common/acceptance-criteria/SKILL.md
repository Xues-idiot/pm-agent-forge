# 验收标准 (Acceptance Criteria)

## 触发条件
- 交付验收
- 质量标准
- 验收流程
- 双方确认

## 输入
- 交付物
- 质量要求
- 验收流程
- 验收时间

## 输出
- 验收标准
- 检查清单
- 验收流程
- 缺陷处理

## Prompt模板
```
请制定验收标准。

背景：
- 交付物：[交付]
- 质量要求：[要求]
- 验收流程：[流程]
- 验收时间：[时间]

请输出：
1. 验收标准
2. 检查清单
3. 验收流程
4. 缺陷处理
5. 验收签字
```

## 示例
### 示例1
- 输入：软件交付
- 输出：验收标准，功能测试、性能指标、安全扫描、用户文档

## 注意事项
- 要明确具体
- 要可测试
- 要双方确认
- 要严格执行
