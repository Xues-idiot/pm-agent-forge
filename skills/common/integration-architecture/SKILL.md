# 集成架构 (Integration Architecture)

## 触发条件
- 系统集成
- 数据流转
- 服务编排
- 架构解耦

## 输入
- 系统清单
- 数据需求
- 集成模式
- 技术选型

## 输出
- 集成架构
- 集成模式
- 数据流设计
- 实施路径

## Prompt模板
```
请设计集成架构。

背景：
- 系统清单：[清单]
- 数据需求：[需求]
- 集成模式：[模式]
- 技术选型：[选型]

请输出：
1. 集成架构
2. 集成模式
3. 数据流设计
4. 实施路径
5. 监控设计
```

## 示例
### 示例1
- 输入：企业系统集成
- 输出：集成架构，ESB、微服务、API网关

## 注意事项
- 要松耦合
- 要可扩展
- 要标准化
- 要可监控
