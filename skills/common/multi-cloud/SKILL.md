# 多云战略 (Multi-cloud Strategy)

## 触发条件
- 多云部署
- 供应商选择
- 风险分散
- 成本优化

## 输入
- 业务需求
- 云服务商
- 工作负载
- 管理能力

## 输出
- 多云策略
- 负载分布
- 统一管理
- 成本优化

## Prompt模板
```
请制定多云战略。

背景：
- 业务需求：[需求]
- 云服务商：[服务商]
- 工作负载：[负载]
- 管理能力：[能力]

请输出：
1. 多云策略
2. 负载分布
3. 统一管理
4. 成本优化
5. 风险控制
```

## 示例
### 示例1
- 输入：企业多云
- 输出：多云战略，AWS/Azure/阿里云，按负载分配

## 注意事项
- 要避免锁定
- 要统一管理
- 要成本最优
- 要灾备设计
