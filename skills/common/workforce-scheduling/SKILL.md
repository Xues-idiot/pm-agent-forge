# 排班管理 (Workforce Scheduling)

## 触发条件
- 排班计划
- 劳动力优化
- 成本控制
- 服务保障

## 输入
- 业务需求
- 人员配置
- 法规限制
- 成本预算

## 输出
- 排班方案
- 人员匹配
- 成本分析
- 应急方案

## Prompt模板
```
请管理排班。

背景：
- 业务需求：[需求]
- 人员配置：[配置]
- 法规限制：[限制]
- 成本预算：[预算]

请输出：
1. 排班方案
2. 人员匹配
3. 成本分析
4. 应急方案
5. 评估机制
```

## 示例
### 示例1
- 输入：客服排班
- 输出：排班管理，高峰班次、弹性排班、备份人员

## 注意事项
- 要业务匹配
- 要法规合规
- 要成本效益
- 要员工满意
