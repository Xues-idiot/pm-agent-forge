# 库存优化 (Inventory Optimization)

## 触发条件
- 库存成本
- 库存周转
- 缺货风险
- 存储能力

## 输入
- 库存现状
- 需求特征
- 补货策略
- 成本结构

## 输出
- 优化方案
- 安全库存
- 补货计划
- 成本分析

## Prompt模板
```
请优化库存。

背景：
- 库存现状：[现状]
- 需求特征：[特征]
- 补货策略：[策略]
- 成本结构：[成本]

请输出：
1. 优化方案
2. 安全库存
3. 补货计划
4. 成本分析
5. KPI设计
```

## 示例
### 示例1
- 输入：多品类库存
- 输出：库存优化，ABC分类、EOQ模型、JIT策略

## 注意事项
- 要平衡成本
- 要保障供应
- 要数据驱动
- 要持续改进
