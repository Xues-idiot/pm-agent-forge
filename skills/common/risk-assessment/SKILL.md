# 风险评估 (Risk Assessment)

## 触发条件
- 风险管理
- 风险识别
- 风险量化
- 应对策略

## 输入
- 业务场景
- 风险类型
- 历史数据
- 风险偏好

## 输出
- 风险清单
- 风险量化
- 应对策略
- 监控机制

## Prompt模板
```
请进行风险评估。

背景：
- 业务场景：[场景]
- 风险类型：[类型]
- 历史数据：[数据]
- 风险偏好：[偏好]

请输出：
1. 风险识别
2. 风险量化
3. 风险优先级
4. 应对策略
5. 监控机制
```

## 示例
### 示例1
- 输入：海外市场拓展
- 输出：风险评估，含汇率风险、政治风险、竞争风险、法律合规风险

## 注意事项
- 要系统识别
- 要量化评估
- 要动态监控
- 要及时应对
