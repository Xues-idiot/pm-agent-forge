# 焦点小组 (Focus Group)

## 触发条件
- 小组讨论
- 意见收集
- 概念测试
- 创意发散

## 输入
- 讨论主题
- 参与者筛选
- 小组数量
- 讨论大纲

## 输出
- 讨论引导
- 观点汇总
- 共识形成
- 洞察提炼

## Prompt模板
```
请组织焦点小组。

背景：
- 讨论主题：[主题]
- 参与者筛选：[筛选]
- 小组数量：[数量]
- 讨论大纲：[大纲]

请输出：
1. 讨论引导
2. 观点汇总
3. 共识形成
4. 洞察提炼
5. 后续验证
```

## 示例
### 示例1
- 输入：新概念测试
- 输出：焦点小组，用户反应、讨论共识、改进方向

## 注意事项
- 要筛选参与者
- 要引导讨论
- 要控制节奏
- 要归纳洞察
