# RFM分析 (RFM Analysis)

## 触发条件
- 客户分层
- 价值评估
- 精准营销
- 会员管理

## 输入
- 最近购买
- 购买频率
- 消费金额
- 分层标准

## 输出
- RFM评分
- 客户分层
- 营销策略
- 优先级

## Prompt模板
```
请进行RFM分析。

背景：
- 最近购买：[最近]
- 购买频率：[频率]
- 消费金额：[金额]
- 分层标准：[标准]

请输出：
1. RFM评分
2. 客户分层
3. 营销策略
4. 优先级
5. 预期效果
```

## 示例
### 示例1
- 输入：零售客户
- 输出：RFM分析，高价值、潜力用户、流失风险、沉睡用户

## 注意事项
- 要动态更新
- 要业务匹配
- 要精准营销
- 要持续运营
