# 控制评估 (Control Assessment)

## 触发条件
- 内部控制
- 合规检查
- 风险控制
- 改进建议

## 输入
- 控制目标
- 控制措施
- 测试证据
- 缺陷发现

## 输出
- 评估结论
- 缺陷清单
- 风险评级
- 改进建议

## Prompt模板
```
请评估控制。

背景：
- 控制目标：[目标]
- 控制措施：[措施]
- 测试证据：[证据]
- 缺陷发现：[发现]

请输出：
1. 评估结论
2. 缺陷清单
3. 风险评级
4. 改进建议
5. 跟踪验证
```

## 示例
### 示例1
- 输入：内控评估
- 输出：控制评估，有效控制、重大缺陷、一般建议

## 注意事项
- 要客观独立
- 要证据充分
- 要风险导向
- 要持续跟踪
