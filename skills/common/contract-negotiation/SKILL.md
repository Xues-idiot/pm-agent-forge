# 合同谈判 (Contract Negotiation)

## 触发条件
- 合同签订
- 商务条款
- 风险分配
- 合作达成

## 输入
- 合同标的
- 对方诉求
- 我方底线
- 市场惯例

## 输出
- 谈判策略
- 关键条款
- 让步空间
- 备选方案

## Prompt模板
```
请制定合同谈判策略。

背景：
- 合同标的：[标的]
- 对方诉求：[诉求]
- 我方底线：[底线]
- 市场惯例：[惯例]

请输出：
1. 谈判策略
2. 关键条款优先级
3. 让步空间
4. 备选方案
5. 谈判话术
```

## 示例
### 示例1
- 输入：企业软件采购合同谈判
- 输出：谈判策略，含价格折扣、付款方式、服务范围、违约责任、续约条款

## 注意事项
- 要充分准备
- 要明确底线
- 要创造双赢
- 要书面确认
