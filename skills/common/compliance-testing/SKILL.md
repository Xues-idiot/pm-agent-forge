# 合规测试 (Compliance Testing)

## 触发条件
- 合规验证
- 控制测试
- 审计准备
- 缺陷整改

## 输入
- 合规要求
- 测试范围
- 测试方法
- 历史发现

## 输出
- 测试计划
- 测试结果
- 缺陷报告
- 整改建议

## Prompt模板
```
请进行合规测试。

背景：
- 合规要求：[要求]
- 测试范围：[范围]
- 测试方法：[方法]
- 历史发现：[历史]

请输出：
1. 测试计划
2. 测试结果
3. 缺陷报告
4. 整改建议
5. 验证报告
```

## 示例
### 示例1
- 输入：SOX合规
- 输出：合规测试，控制测试、文件审核、访谈验证

## 注意事项
- 要独立性
- 要证据充分
- 要问题导向
- 要整改验证
