# 项目复盘 (Project Review)

## 触发条件
- 项目总结
- 经验教训
- 流程改进
- 团队成长

## 输入
- 项目目标
- 实际结果
- 过程数据
- 团队反馈

## 输出
- 结果评估
- 原因分析
- 经验总结
- 改进行动

## Prompt模板
```
请进行项目复盘。

背景：
- 项目目标：[目标]
- 实际结果：[结果]
- 过程数据：[数据]
- 团队反馈：[反馈]

请输出：
1. 结果评估
2. 成功因素
3. 问题分析
4. 经验教训
5. 改进行动
```

## 示例
### 示例1
- 输入：营销活动，效果未达预期
- 输出：复盘结果，含目标人群偏差、创意测试不足、执行细节疏忽

## 注意事项
- 要客观真实
- 要鼓励发言
- 要聚焦改进
- 要文档沉淀
