# 战略思考 (Strategic Thinking)

## 触发条件
- 制定战略规划
- 分析竞争态势
- 业务模式设计
- 长期规划

## 输入
- 企业愿景
- 市场环境
- 竞争格局
- 自身资源

## 输出
- 战略分析报告
- 战略选择
- 战略路径
- 执行要点

## Prompt模板
```
请进行[业务/公司]的战略规划。

背景：
- 愿景使命：[愿景]
- 市场环境：[环境]
- 竞争格局：[格局]
- 自身资源：[资源]

请使用战略分析框架输出：
1. 宏观环境分析（PEST）
2. 行业分析（波特五力）
3. 竞争分析（竞争对手/替代品）
4. 自身能力分析（SWOT）
5. 战略选择（增长/维持/收缩）
6. 战略目标和时间表
7. 关键成功因素
8. 战略路径和里程碑
9. 资源需求和配置
10. 风险和应对
```

## 示例
### 示例1
- 输入：传统零售企业转型，目标线上线下融合
- 输出：数字化转型战略，含全渠道战略、能力建设路径

## 注意事项
- 要有全局视角
- 要考虑长期和短期平衡
- 战略要可执行
