# 业务复盘 (Business Review)

## 触发条件
- 阶段总结
- 经验沉淀
- 问题分析
- 改进提升

## 输入
- 业务阶段
- 目标达成
- 数据表现
- 问题挑战

## 输出
- 复盘报告
- 经验总结
- 问题分析
- 改进行动

## Prompt模板
```
请进行业务复盘。

背景：
- 业务阶段：[阶段]
- 目标达成：[达成]
- 数据表现：[数据]
- 问题挑战：[问题]

请输出：
1. 目标回顾
2. 结果评估
3. 原因分析
4. 经验总结
5. 改进行动
6. 下步计划
```

## 示例
### 示例1
- 输入：季度业务复盘，目标完成70%
- 输出：复盘报告，含完成分析、问题根因、下季度改进行动

## 注意事项
- 要实事求是
- 要追根溯源
- 要落地改进
