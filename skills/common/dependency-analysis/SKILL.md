# 依赖分析 (Dependency Analysis)

## 触发条件
- 项目规划
- 风险识别
- 进度管理
- 资源协调

## 输入
- 任务列表
- 依赖关系
- 资源约束
- 时间要求

## 输出
- 依赖图谱
- 关键路径
- 风险点
- 协调计划

## Prompt模板
```
请分析依赖关系。

背景：
- 任务列表：[任务]
- 依赖关系：[关系]
- 资源约束：[约束]
- 时间要求：[要求]

请输出：
1. 依赖图谱
2. 关键路径
3. 风险识别
4. 协调计划
5. 进度建议
```

## 示例
### 示例1
- 输入：大型项目
- 输出：依赖分析，含内部依赖、外部依赖、关键链、风险缓冲

## 注意事项
- 要识别隐性依赖
- 要动态更新
- 要及时沟通
- 要关键路径优先
