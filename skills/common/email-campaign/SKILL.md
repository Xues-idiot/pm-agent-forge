# 邮件营销 (Email Campaign)

## 触发条件
- 邮件推广
- 用户触达
- 活动营销
- 自动化

## 输入
- 营销目标
- 用户分群
- 邮件内容
- 发送时间

## 输出
- 邮件策略
- 内容规划
- 分群策略
- 效果评估

## Prompt模板
```
请策划邮件营销。

背景：
- 营销目标：[目标]
- 用户分群：[分群]
- 邮件内容：[内容]
- 发送时间：[时间]

请输出：
1. 邮件策略
2. 内容规划
3. 分群策略
4. 效果评估
5. 合规要求
```

## 示例
### 示例1
- 输入：促销邮件
- 输出：邮件营销，欢迎邮件、促销邮件、挽回邮件序列

## 注意事项
- 要用户分群
- 要内容相关
- 要时机恰当
- 要效果追踪
