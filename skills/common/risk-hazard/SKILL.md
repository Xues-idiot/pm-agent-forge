# 风险与危害 (Risk & Hazard)

## 触发条件
- 危害识别
- 风险评估
- 控制措施
- 法规合规

## 输入
- 危害清单
- 暴露程度
- 后果严重
- 控制措施

## 输出
- 危害评估
- 风险矩阵
- 控制方案
- 监测计划

## Prompt模板
```
请评估风险与危害。

背景：
- 危害清单：[清单]
- 暴露程度：[暴露]
- 后果严重：[后果]
- 控制措施：[措施]

请输出：
1. 危害评估
2. 风险矩阵
3. 控制方案
4. 监测计划
5. 评审机制
```

## 示例
### 示例1
- 输入：危险作业
- 输出：风险与危害，HAZOP分析、安全间距、防护装备

## 注意事项
- 要全面识别
- 要科学评估
- 要有效控制
- 要持续监测
