# 内部收益率 (Internal Rate of Return)

## 触发条件
- 投资评估
- 资本预算
- 项目排序
- 价值评估

## 输入
- 现金流序列
- 初始投资
- 项目周期
- 比较基准

## 输出
- IRR计算
- 项目排序
- 风险调整
- 决策建议

## Prompt模板
```
请计算内部收益率。

背景：
- 现金流序列：[现金流]
- 初始投资：[投资]
- 项目周期：[周期]
- 比较基准：[基准]

请输出：
1. IRR计算
2. 项目排序
3. 风险调整
4. 决策建议
5. 敏感性分析
```

## 示例
### 示例1
- 输入：项目投资
- 输出：IRR计算，IRR 25%、NPV正值、投资可行

## 注意事项
- 要完整现金流
- 要假设合理
- 要风险调整
- 要结合NPV
