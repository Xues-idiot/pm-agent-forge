# 再营销 (Remarketing)

## 触发条件
- 流失召回
- 精准触达
- 转化提升
- 成本控制

## 输入
- 用户行为
- 再营销人群
- 广告创意
- 投放策略

## 输出
- 人群策略
- 创意设计
- 投放方案
- 效果评估

## Prompt模板
```
请实施再营销。

背景：
- 用户行为：[行为]
- 再营销人群：[人群]
- 广告创意：[创意]
- 投放策略：[策略]

请输出：
1. 人群策略
2. 创意设计
3. 投放方案
4. 效果评估
5. 优化方向
```

## 示例
### 示例1
- 输入：购物车放弃
- 输出：再营销，浏览未购买、加购未支付、历史访客

## 注意事项
- 要精准人群
- 要创意吸引
- 要频次控制
- 要效果追踪
