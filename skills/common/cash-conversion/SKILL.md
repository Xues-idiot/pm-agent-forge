# 现金循环 (Cash Conversion Cycle)

## 触发条件
- 资金效率
- 周转优化
- 流动性
- 经营效率

## 输入
- 存货周期
- 应收账款
- 应付账款
- 业务模式

## 输出
- CCC计算
- 周期分析
- 优化空间
- 改进建议

## Prompt模板
```
请分析现金循环。

背景：
- 存货周期：[存货]
- 应收账款：[应收]
- 应付账款：[应付]
- 业务模式：[模式]

请输出：
1. CCC计算
2. 周期分析
3. 优化空间
4. 改进建议
5. 监控指标
```

## 示例
### 示例1
- 输入：制造企业
- 输出：现金循环，采购-生产-销售-回款周期、优化方向

## 注意事项
- 要周期管理
- 要各环节优化
- 要资金效率
- 要持续监控
