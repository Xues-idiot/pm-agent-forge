# 合规管理 (Compliance Management)

## 触发条件
- 合规检查
- 法规遵循
- 风险防控
- 审计应对

## 输入
- 法规要求
- 业务范围
- 现有控制
- 审计发现

## 输出
- 合规框架
- 控制措施
- 评估机制
- 改进计划

## Prompt模板
```
请管理合规。

背景：
- 法规要求：[要求]
- 业务范围：[范围]
- 现有控制：[控制]
- 审计发现：[发现]

请输出：
1. 合规框架
2. 控制措施
3. 评估机制
4. 改进计划
5. 培训计划
```

## 示例
### 示例1
- 输入：数据合规
- 输出：合规管理，GDPR合规、数据保护影响评估、隐私政策

## 注意事项
- 要法规跟踪
- 要全员意识
- 要持续监控
- 要及时改进
