# 访问控制 (Access Control)

## 触发条件
- 权限管理
- 身份认证
- 最小权限
- 合规审计

## 输入
- 资产分类
- 用户角色
- 认证方式
- 合规要求

## 输出
- 权限模型
- 认证策略
- 访问策略
- 审计机制

## Prompt模板
```
请设计访问控制。

背景：
- 资产分类：[分类]
- 用户角色：[角色]
- 认证方式：[方式]
- 合规要求：[要求]

请输出：
1. 权限模型
2. 角色定义
3. 认证策略
4. 访问策略
5. 审计机制
```

## 示例
### 示例1
- 输入：企业应用系统
- 输出：访问控制，含RBAC角色模型、多因素认证、定期权限审计

## 注意事项
- 要最小权限
- 要职责分离
- 要定期审计
- 要及时回收
