# 干系人登记 (Stakeholder Register)

## 触发条件
- 干系人管理
- 沟通规划
- 期望管理
- 关系维护

## 输入
- 干系人类型
- 利益影响
- 沟通需求
- 权力级别

## 输出
- 干系人清单
- 分类分析
- 沟通策略
- 管理计划

## Prompt模板
```
请管理干系人。

背景：
- 干系人类型：[类型]
- 利益影响：[影响]
- 沟通需求：[需求]
- 权力级别：[权力]

请输出：
1. 干系人清单
2. 利益影响分析
3. 权力/兴趣矩阵
4. 沟通策略
5. 维护计划
```

## 示例
### 示例1
- 输入：大型项目
- 输出：干系人登记，高层管理 Sponsor、业务部门 关键用户、团队成员

## 注意事项
- 要全面识别
- 要分析需求
- 要制定策略
- 要持续维护
