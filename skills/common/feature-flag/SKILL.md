# 特性开关 (Feature Flag)

## 触发条件
- 灰度发布
- A/B测试
- 快速关闭
- 灵活控制

## 输入
- 功能特点
- 发布策略
- 用户群体
- 监控需求

## 输出
- 开关设计
- 灰度策略
- 规则配置
- 监控方案

## Prompt模板
```
请设计特性开关。

背景：
- 功能特点：[特点]
- 发布策略：[策略]
- 用户群体：[群体]
- 监控需求：[需求]

请输出：
1. 开关设计
2. 灰度策略
3. 规则配置
4. 监控方案
5. 清理机制
```

## 示例
### 示例1
- 输入：新功能上线
- 输出：特性开关，含按用户比例灰度、内测用户、白名单、降级开关

## 注意事项
- 要集中管理
- 要实时生效
- 要有降级方案
- 要定期清理
