# 学习发展 (Learning & Development)

## 触发条件
- 培训计划
- 能力提升
- 职业发展
- 知识管理

## 输入
- 培训需求
- 学习资源
- 预算限制
- 业务优先级

## 输出
- 培训体系
- 课程设计
- 学习路径
- 效果评估

## Prompt模板
```
请设计学习发展体系。

背景：
- 培训需求：[需求]
- 学习资源：[资源]
- 预算限制：[预算]
- 业务优先级：[优先级]

请输出：
1. 培训体系
2. 课程设计
3. 学习路径
4. 效果评估
5. 成本分析
```

## 示例
### 示例1
- 输入：员工培训
- 输出：学习发展，在线学习、面授课程、导师辅导

## 注意事项
- 要需求导向
- 要多样化
- 要效果评估
- 要持续改进
