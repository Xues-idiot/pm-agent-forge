# 谈判技巧 (Negotiation Skills)

## 触发条件
- 商业谈判准备
- 合同谈判
- 采购谈判
- 跨部门协调

## 输入
- 谈判对手
- 谈判目标
- 可用资源
- BATNA

## 输出
- 谈判策略
- 筹码分析
- 方案选项
- 预案

## Prompt模板
```
请为[谈判场景]制定谈判策略。

背景：
- 谈判主题：[主题]
- 对方：[对手]
- 我方目标：[目标]
- 底线：[底线]

请输出：
1. 谈判目标和优先级
2. 对方分析（立场/利益/底线）
3. 我方筹码和劣势
4. BATNA（最佳替代方案）
5. 谈判策略选择
6. 分阶段谈判方案
7. 可能的问题和应对
8. 让步策略
9. 签约条件设计
```

## 示例
### 示例1
- 输入：SaaS采购谈判，对方是大型企业，我方是创业公司
- 输出：谈判策略，含价值主张、定制化让步、分阶段合作

## 注意事项
- 要了解对方需求
- 要准备多个方案
- 要知道自己的底线
