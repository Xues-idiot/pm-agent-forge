# 应用安全 (Application Security)

## 触发条件
- 安全开发
- 漏洞防护
- 代码审计
- 安全测试

## 输入
- 应用架构
- 安全需求
- 攻击向量
- 合规要求

## 输出
- 安全设计
- 威胁建模
- 安全编码
- 测试方案

## Prompt模板
```
请保障应用安全。

背景：
- 应用架构：[架构]
- 安全需求：[需求]
- 攻击向量：[向量]
- 合规要求：[要求]

请输出：
1. 安全设计
2. 威胁建模
3. 安全编码
4. 测试方案
5. 监控机制
```

## 示例
### 示例1
- 输入：Web应用
- 输出：应用安全，OWASP Top 10、安全编码、渗透测试

## 注意事项
- 要安全设计
- 要安全编码
- 要安全测试
- 要持续监控
