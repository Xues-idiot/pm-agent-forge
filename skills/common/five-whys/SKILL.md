# 五个为什么 (Five Whys)

## 触发条件
- 快速分析
- 根本原因
- 问题解决
- 持续改进

## 输入
- 问题表象
- 直接原因
- 层层追问
- 最终根因

## 输出
- 原因链
- 根因确定
- 解决方案
- 预防措施

## Prompt模板
```
请使用五个为什么分析。

背景：
- 问题表象：[表象]
- 直接原因：[原因]
- 层层追问：[追问]
- 最终根因：[根因]

请输出：
1. 问题陈述
2. Why1-Why5
3. 根因确定
4. 解决方案
5. 预防措施
```

## 示例
### 示例1
- 输入：会议迟到
- 输出：Why分析，迟到→闹钟没响→睡过头→晚睡→加班太晚→任务过多

## 注意事项
- 要追根溯源
- 要逻辑清晰
- 要团队共识
- 要落到行动
