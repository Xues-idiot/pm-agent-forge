# A/B测试 (A/B Testing)

## 触发条件
- 实验设计
- 效果验证
- 数据驱动
- 迭代优化

## 输入
- 实验假设
- 用户群体
- 样本量
- 评估指标

## 输出
- 实验设计
- 分流策略
- 样本计算
- 结果分析

## Prompt模板
```
请设计A/B测试。

背景：
- 实验假设：[假设]
- 用户群体：[群体]
- 样本量：[量]
- 评估指标：[指标]

请输出：
1. 实验设计
2. 分流策略
3. 样本计算
4. 评估方法
5. 结果分析
```

## 示例
### 示例1
- 输入：按钮颜色优化
- 输出：A/B测试，含等比分流、最小样本量计算、统计显著性分析

## 注意事项
- 要假设清晰
- 要样本足够
- 要分流均匀
- 要结论审慎
