# 绩效评估 (Performance Review)

## 触发条件
- 绩效考核
- 目标达成
- 人才发展
- 激励反馈

## 输入
- 岗位职责
- 目标设定
- 评估标准
- 历史表现

## 输出
- 评估体系
- 评估流程
- 反馈策略
- 发展建议

## Prompt模板
```
请制定绩效评估方案。

背景：
- 岗位职责：[职责]
- 目标设定：[目标]
- 评估标准：[标准]
- 历史表现：[表现]

请输出：
1. 评估维度
2. 评估标准
3. 评估流程
4. 反馈策略
5. 结果应用
```

## 示例
### 示例1
- 输入：销售团队季度评估
- 输出：评估体系，含业绩60%、过程行为20%、团队协作20%

## 注意事项
- 要客观公正
- 要及时反馈
- 要发展导向
- 要与激励挂钩
