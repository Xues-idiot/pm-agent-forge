# 平台策略 (Marketplace Strategy)

## 触发条件
- 双边市场
- 平台运营
- 网络效应
- 生态构建

## 输入
- 买卖双方
- 平台定位
- 收费模式
- 增长策略

## 输出
- 平台设计
- 供需平衡
- 增长策略
- 风险控制

## Prompt模板
```
请制定平台策略。

背景：
- 买卖双方：[双方]
- 平台定位：[定位]
- 收费模式：[收费]
- 增长策略：[策略]

请输出：
1. 平台设计
2. 供需平衡
3. 增长策略
4. 风险控制
5. 生态治理
```

## 示例
### 示例1
- 输入：交易平台
- 输出：平台策略，撮合机制、抽佣模式、供需调控

## 注意事项
- 要双边平衡
- 要网络效应
- 要质量控制
- 要持续创新
