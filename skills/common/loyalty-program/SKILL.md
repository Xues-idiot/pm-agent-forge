# 忠诚度计划 (Loyalty Program)

## 触发条件
- 用户留存
- 会员运营
- 消费激励
- 品牌粘性

## 输入
- 会员等级
- 积分规则
- 兑换权益
- 升级条件

## 输出
- 会员体系
- 积分机制
- 权益设计
- 运营策略

## Prompt模板
```
请设计忠诚度计划。

背景：
- 会员等级：[等级]
- 积分规则：[规则]
- 兑换权益：[权益]
- 升级条件：[条件]

请输出：
1. 会员体系
2. 积分机制
3. 权益设计
4. 运营策略
5. 成本核算
```

## 示例
### 示例1
- 输入：零售品牌
- 输出：忠诚度计划，金卡/银卡/普卡、积分抵现、生日礼遇

## 注意事项
- 要价值感知
- 要升级激励
- 要差异化权益
- 要持续运营
