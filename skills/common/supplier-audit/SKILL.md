# 供应商审计 (Supplier Audit)

## 触发条件
- 供应商评估
- 质量控制
- 风险管控
- 持续改进

## 输入
- 供应商清单
- 审核标准
- 审核资源
- 风险等级

## 输出
- 审核计划
- 审核发现
- 风险评估
- 改进要求

## Prompt模板
```
请进行供应商审计。

背景：
- 供应商清单：[清单]
- 审核标准：[标准]
- 审核资源：[资源]
- 风险等级：[风险]

请输出：
1. 审核计划
2. 审核发现
3. 风险评估
4. 改进要求
5. 跟踪验证
```

## 示例
### 示例1
- 输入：供应商质量
- 输出：供应商审计，现场审核、文件审核、能力评估

## 注意事项
- 要标准明确
- 要客观评估
- 要问题跟踪
- 要持续监控
