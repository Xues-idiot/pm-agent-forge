# 联盟营销 (Affiliate Program)

## 触发条件
- 联盟推广
- 佣金激励
- 渠道拓展
- 效果追踪

## 输入
- 产品服务
- 佣金比例
- 联盟伙伴
- 追踪系统

## 输出
- 联盟策略
- 佣金设计
- 招募计划
- 管理机制

## Prompt模板
```
请设计联盟营销。

背景：
- 产品服务：[产品]
- 佣金比例：[比例]
- 联盟伙伴：[伙伴]
- 追踪系统：[系统]

请输出：
1. 联盟策略
2. 佣金设计
3. 招募计划
4. 管理机制
5. 合规要求
```

## 示例
### 示例1
- 输入：电商联盟
- 输出：联盟营销，CPS佣金、联盟招募、素材提供

## 注意事项
- 要佣金合理
- 要追踪精准
- 要伙伴管理
- 要持续优化
