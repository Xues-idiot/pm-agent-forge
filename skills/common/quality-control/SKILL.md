# 质量控制 (Quality Control)

## 触发条件
- 质量检查
- 缺陷管理
- 质量改进
- 验收标准

## 输入
- 质量标准
- 检查方法
- 缺陷数据
- 质量目标

## 输出
- 检查计划
- 缺陷分析
- 改进措施
- 验收结果

## Prompt模板
```
请进行质量控制。

背景：
- 质量标准：[标准]
- 检查方法：[方法]
- 缺陷数据：[缺陷]
- 质量目标：[目标]

请输出：
1. 检查计划
2. 检查执行
3. 缺陷分析
4. 改进措施
5. 验收结果
```

## 示例
### 示例1
- 输入：生产质量
- 输出：质量控制，抽检方案、不合格率分析、原因改进

## 注意事项
- 要标准明确
- 要方法得当
- 要持续改进
- 要全流程覆盖
