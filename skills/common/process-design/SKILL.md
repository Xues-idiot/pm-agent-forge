# 流程设计 (Process Design)

## 触发条件
- 流程优化
- 标准化
- 效率提升
- 最佳实践

## 输入
- 业务需求
- 现有流程
- 瓶颈问题
- 改进目标

## 输出
- 流程设计
- 步骤定义
- 角色职责
- KPI设定

## Prompt模板
```
请设计业务流程。

背景：
- 业务需求：[需求]
- 现有流程：[流程]
- 瓶颈问题：[瓶颈]
- 改进目标：[目标]

请输出：
1. 流程设计
2. 步骤分解
3. 角色职责
4. 流转规则
5. KPI设定
```

## 示例
### 示例1
- 输入：订单履约流程
- 输出：流程设计，含订单确认→库存锁定→生产调度→质检→发货→签收

## 注意事项
- 要简洁高效
- 要端到端
- 要明确责任
- 要持续优化
