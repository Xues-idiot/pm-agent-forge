# IT治理 (IT Governance)

## 触发条件
- IT管理
- 投资决策
- 价值交付
- 风险管控

## 输入
- IT战略
- 业务需求
- 技术趋势
- 资源预算

## 输出
- 治理框架
- 决策机制
- 评估体系
- 改进计划

## Prompt模板
```
请建立IT治理框架。

背景：
- IT战略：[战略]
- 业务需求：[需求]
- 技术趋势：[趋势]
- 资源预算：[预算]

请输出：
1. 治理框架
2. 决策机制
3. 评估体系
4. 报告机制
5. 改进计划
```

## 示例
### 示例1
- 输入：大型企业IT部门
- 输出：IT治理，含IT委员会、优先级评审、投资回报评估、服务目录

## 注意事项
- 要对齐业务
- 要明确责任
- 要量化评估
- 要持续改进
