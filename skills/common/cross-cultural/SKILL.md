# 跨文化沟通 (Cross-cultural Communication)

## 触发条件
- 跨国沟通
- 文化差异
- 多元团队
- 国际合作

## 输入
- 文化背景
- 沟通习惯
- 禁忌事项
- 沟通目的

## 输出
- 文化意识
- 沟通策略
- 礼仪建议
- 关系建立

## Prompt模板
```
请提升跨文化沟通能力。

背景：
- 文化背景：[背景]
- 沟通习惯：[习惯]
- 禁忌事项：[禁忌]
- 沟通目的：[目的]

请输出：
1. 文化差异意识
2. 沟通策略
3. 语言礼仪
4. 非语言沟通
5. 关系建立
```

## 示例
### 示例1
- 输入：中西方合作
- 输出：跨文化沟通，直接vs委婉、时间观念、沟通方式差异

## 注意事项
- 要了解差异
- 要尊重文化
- 要开放包容
- 要灵活适应
