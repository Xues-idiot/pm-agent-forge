# A/B测试设计 (A/B Test Design)

## 触发条件
- 实验设计
- 效果验证
- 数据驱动
- 优化迭代

## 输入
- 测试假设
- 样本选择
- 分组策略
- 评估指标

## 输出
- 测试方案
- 样本量
- 统计功效
- 结果分析

## Prompt模板
```
请设计A/B测试。

背景：
- 测试假设：[假设]
- 样本选择：[样本]
- 分组策略：[分组]
- 评估指标：[指标]

请输出：
1. 测试方案
2. 样本量计算
3. 统计功效
4. 结果分析
5. 决策建议
```

## 示例
### 示例1
- 输入：页面改版
- 输出：A/B测试，转化率对比、统计显著性、置信区间

## 注意事项
- 要假设明确
- 要样本足够
- 要随机分组
- 要结论谨慎
