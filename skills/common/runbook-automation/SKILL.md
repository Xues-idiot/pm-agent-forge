# 运维手册自动化 (Runbook Automation)

## 触发条件
- 运维自动化
- 故障自愈
- 效率提升
- 标准化

## 输入
- 运维场景
- 故障类型
- 处理流程
- 技术能力

## 输出
- 自动化场景
- 脚本设计
- 触发机制
- 验证方案

## Prompt模板
```
请实现运维手册自动化。

背景：
- 运维场景：[场景]
- 故障类型：[类型]
- 处理流程：[流程]
- 技术能力：[能力]

请输出：
1. 自动化场景
2. 脚本设计
3. 触发机制
4. 验证方案
5. 监控告警
```

## 示例
### 示例1
- 输入：服务宕机自愈
- 输出：自动化方案，含健康检查→自动重启→流量切换→通知告警

## 注意事项
- 要场景明确
- 要可回滚
- 要充分测试
- 要监控验证
