# 可用性测试 (Usability Testing)

## 触发条件
- 产品测试
- 用户体验
- 问题发现
- 改进建议

## 输入
- 测试目标
- 测试任务
- 典型用户
- 评估标准

## 输出
- 测试方案
- 任务设计
- 问题发现
- 改进建议

## Prompt模板
```
请进行可用性测试。

背景：
- 测试目标：[目标]
- 测试任务：[任务]
- 典型用户：[用户]
- 评估标准：[标准]

请输出：
1. 测试方案
2. 任务设计
3. 评估指标
4. 问题发现
5. 改进建议
```

## 示例
### 示例1
- 输入：App下单流程
- 输出：可用性测试，含任务完成率、时间、错误率、满意度评分

## 注意事项
- 要任务明确
- 要真实场景
- 要及时记录
- 要客观分析
