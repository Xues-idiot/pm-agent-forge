# 灾难恢复 (Disaster Recovery)

## 触发条件
- 数据备份
- 系统恢复
- 灾难预案
- RTO/RPO

## 输入
- 关键系统
- 恢复目标
- 灾难场景
- 资源投入

## 输出
- 恢复策略
- 备份方案
- 灾难预案
- 测试计划

## Prompt模板
```
请制定灾难恢复计划。

背景：
- 关键系统：[系统]
- 恢复目标：[目标]
- 灾难场景：[场景]
- 资源投入：[资源]

请输出：
1. 恢复策略
2. 备份方案
3. 灾难预案
4. 切换流程
5. 测试计划
```

## 示例
### 示例1
- 输入：核心数据库
- 输出：DR方案，含同城双活、异地备份、RTO 2小时、RPO 15分钟

## 注意事项
- 要业务导向
- 要定期测试
- 要文档化
- 要持续改进
