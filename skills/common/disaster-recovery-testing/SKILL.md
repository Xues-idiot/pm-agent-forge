# 灾难恢复测试 (Disaster Recovery Testing)

## 触发条件
- 灾备验证
- RTO/RPO
- 恢复流程
- 团队能力

## 输入
- 恢复方案
- 测试场景
- 测试范围
- 参与人员

## 输出
- 测试计划
- 测试结果
- 问题记录
- 改进建议

## Prompt模板
```
请进行灾难恢复测试。

背景：
- 恢复方案：[方案]
- 测试场景：[场景]
- 测试范围：[范围]
- 参与人员：[人员]

请输出：
1. 测试计划
2. 测试结果
3. 问题记录
4. 改进建议
5. 认证报告
```

## 示例
### 示例1
- 输入：数据中心灾备
- 输出：灾难恢复测试，切换演练、数据验证、时间记录

## 注意事项
- 要定期测试
- 要全面覆盖
- 要记录结果
- 要持续改进
