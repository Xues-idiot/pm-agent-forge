# 公众演讲 (Public Speaking)

## 触发条件
- 演讲汇报
- 舞台表现
- 听众互动
- 紧张管理

## 输入
- 演讲场合
- 听众特征
- 演讲内容
- 时间限制

## 输出
- 演讲结构
- 内容设计
- 表达技巧
- 互动方法

## Prompt模板
```
请提升公众演讲能力。

背景：
- 演讲场合：[场合]
- 听众特征：[听众]
- 演讲内容：[内容]
- 时间限制：[时间]

请输出：
1. 演讲结构
2. 开场设计
3. 内容展开
4. 结尾升华
5. 互动技巧
```

## 示例
### 示例1
- 输入：产品发布会
- 输出：公众演讲，故事开场、亮点展示、互动演示、号召行动

## 注意事项
- 要充分准备
- 要练习多次
- 要管理紧张
- 要与听众连接
