# 合规评估 (Compliance Assessment)

## 触发条件
- 合规检查
- 法规要求
- 风险差距
- 整改计划

## 输入
- 法规清单
- 业务范围
- 现有控制
- 风险偏好

## 输出
- 合规框架
- 差距分析
- 风险评估
- 整改计划

## Prompt模板
```
请进行合规评估。

背景：
- 法规清单：[清单]
- 业务范围：[范围]
- 现有控制：[控制]
- 风险偏好：[偏好]

请输出：
1. 合规框架
2. 差距分析
3. 风险评估
4. 整改计划
5. 优先级排序
```

## 示例
### 示例1
- 输入：数据安全法规
- 输出：合规评估，等保2.0、数据安全法、个人信息保护法合规差距

## 注意事项
- 要了解法规
- 要全面评估
- 要优先级排序
- 要持续监控
