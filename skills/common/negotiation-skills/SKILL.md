# 谈判技巧 (Negotiation Skills)

## 触发条件
- 商务谈判
- 利益争取
- 协议达成
- 关系维护

## 输入
- 谈判目标
- 对方诉求
- 底线设定
- 备选方案

## 输出
- 谈判策略
- 议题排序
- 让步方案
- 协议框架

## Prompt模板
```
请提升谈判技巧。

背景：
- 谈判目标：[目标]
- 对方诉求：[诉求]
- 底线设定：[底线]
- 备选方案：[备选]

请输出：
1. 谈判策略
2. 议题排序
3. 让步方案
4. 应对策略
5. 协议框架
```

## 示例
### 示例1
- 输入：合同谈判
- 输出：谈判技巧，双赢思维、分议题谈判、交换让步

## 注意事项
- 要充分准备
- 要底线明确
- 要创造价值
- 要关系维护
