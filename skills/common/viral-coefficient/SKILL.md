# 病毒系数 (Viral Coefficient)

## 触发条件
- 病毒传播
- 推荐增长
- 口碑营销
- 增长优化

## 输入
- 邀请行为
- 转化率
- 分享动机
- 用户基础

## 输出
- K因子计算
- 传播分析
- 优化策略
- 预期效果

## Prompt模板
```
请分析病毒系数。

背景：
- 邀请行为：[邀请]
- 转化率：[转化]
- 分享动机：[动机]
- 用户基础：[基础]

请输出：
1. K因子计算
2. 传播分析
3. 优化策略
4. 预期效果
5. 成本对比
```

## 示例
### 示例1
- 输入：邀请注册
- 输出：病毒分析，K=0.3，每位用户邀请0.3人，病毒传播缓慢

## 注意事项
- 要K>1才有病毒
- 要优化关键环节
- 要激励机制
- 要持续跟踪
