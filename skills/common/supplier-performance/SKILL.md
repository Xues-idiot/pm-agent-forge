# 供应商绩效 (Supplier Performance)

## 触发条件
- 供应商评估
- 绩效管理
- 持续改进
- 合作关系

## 输入
- 评估指标
- 绩效数据
- 反馈信息
- 合作目标

## 输出
- 绩效评分
- 问题分析
- 改进计划
- 激励措施

## Prompt模板
```
请评估供应商绩效。

背景：
- 评估指标：[指标]
- 绩效数据：[数据]
- 反馈信息：[反馈]
- 合作目标：[目标]

请输出：
1. 绩效评分
2. 问题分析
3. 改进计划
4. 激励措施
5. 合作建议
```

## 示例
### 示例1
- 输入：供应商考核
- 输出：供应商绩效，交期、质量、服务、价格综合评分

## 注意事项
- 要客观公正
- 要数据说话
- 要持续改进
- 要奖惩分明
