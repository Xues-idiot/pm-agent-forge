# 安索夫矩阵 (Ansoff Matrix)

## 触发条件
- 市场策略
- 增长战略
- 产品规划
- 风险评估

## 输入
- 现有产品
- 新产品
- 现有市场
- 新市场

## 输出
- 增长策略
- 风险评估
- 实施建议
- 资源配置

## Prompt模板
```
请用安索夫矩阵制定增长策略。

背景：
- 现有产品：[产品]
- 新产品：[新产品]
- 现有市场：[市场]
- 新市场：[新市场]

请输出：
1. 市场渗透策略
2. 产品开发策略
3. 市场开发策略
4. 多元化策略
5. 风险评估
6. 实施建议
```

## 示例
### 示例1
- 输入：快消品公司寻找增长机会
- 输出：增长策略，含市场渗透、产品升级、新市场拓展

## 注意事项
- 要风险匹配
- 要能力支撑
- 要循序渐进
