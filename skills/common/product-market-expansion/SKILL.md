# 产品市场扩展 (Product-Market Expansion)

## 触发条件
- 产品延伸
- 市场拓展
- 组合策略
- 协同效应

## 输入
- 现有产品
- 新产品机会
- 现有市场
- 新市场机会

## 输出
- 扩展策略
- 产品组合
- 市场选择
- 资源配置

## Prompt模板
```
请制定产品市场扩展策略。

背景：
- 现有产品：[产品]
- 新产品机会：[机会]
- 现有市场：[市场]
- 新市场机会：[新市场]

请输出：
1. 扩展策略
2. 产品组合
3. 市场选择
4. 协同效应
5. 资源配置
```

## 示例
### 示例1
- 输入：手机产品
- 输出：产品市场扩展，手机→平板→手表→耳机生态

## 注意事项
- 要聚焦核心
- 要利用协同
- 要资源配置
- 要风险控制
