# 碳足迹 (Carbon Footprint)

## 触发条件
- 碳排放
- 气候变化
- 减排目标
- 碳交易

## 输入
- 排放源
- 活动数据
- 排放因子
- 计算边界

## 输出
- 碳排放量
- 排放清单
- 减排机会
- 认证路径

## Prompt模板
```
请计算碳足迹。

背景：
- 排放源：[源]
- 活动数据：[数据]
- 排放因子：[因子]
- 计算边界：[边界]

请输出：
1. 碳排放量
2. 排放清单
3. 减排机会
4. 认证路径
5. 报告机制
```

## 示例
### 示例1
- 输入：企业碳核算
- 输出：碳足迹，范围1/2/3排放、减排路径、碳中和

## 注意事项
- 要全面核算
- 要数据可靠
- 要减排优先
- 要持续跟踪
