# 客户成功 (Customer Success)

## 触发条件
- 客户留存
- 价值实现
- 客户健康
- 续费扩展

## 输入
- 客户类型
- 价值目标
- 生命周期
- 资源情况

## 输出
- CS策略
- 健康度模型
- 价值实现
- 运营流程

## Prompt模板
```
请设计[产品/服务]客户成功方案。

背景：
- 客户类型：[类型]
- 价值目标：[目标]
- 生命周期：[周期]
- 资源情况：[资源]

请输出：
1. CS战略定位
2. 客户分层
3. 健康度模型
4. 价值实现路径
5. 主动服务
6. 客户教育
7. 续费扩展
8. CS工具支撑
```

## 示例
### 示例1
- 输入：SaaS产品，100家客户，需要提升续费率
- 输出：CS方案，含健康度监控、主动触达、价值报告

## 注意事项
- 要价值导向
- 要主动服务
- 要数据驱动
