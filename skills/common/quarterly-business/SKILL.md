# 季度业务评审 (Quarterly Business Review)

## 触发条件
- 业务复盘
- 绩效评估
- 策略调整
- 规划对齐

## 输入
- 业务数据
- 目标达成
- 市场变化
- 竞争态势

## 输出
- 业务评估
- 目标达成
- 策略调整
- 下季计划

## Prompt模板
```
请进行季度业务评审。

背景：
- 业务数据：[数据]
- 目标达成：[达成]
- 市场变化：[变化]
- 竞争态势：[态势]

请输出：
1. 业务评估
2. 目标达成分析
3. 市场分析
4. 策略调整
5. 下季计划
```

## 示例
### 示例1
- 输入：Q1业务数据
- 输出：QBR，目标达成80%、市场份额提升、策略调整、下季重点

## 注意事项
- 要数据详实
- 要分析深入
- 要策略对齐
- 要计划具体
