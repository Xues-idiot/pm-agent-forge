# 异常检测 (Anomaly Detection)

## 触发条件
- 数据质量
- 风险预警
- 问题诊断
- 监控告警

## 输入
- 数据特征
- 正常模式
- 异常定义
- 检测方法

## 输出
- 检测方案
- 阈值设计
- 预警机制
- 应对流程

## Prompt模板
```
请设计异常检测。

背景：
- 数据特征：[特征]
- 正常模式：[模式]
- 异常定义：[定义]
- 检测方法：[方法]

请输出：
1. 检测方案
2. 阈值设计
3. 预警机制
4. 应对流程
```

## 示例
### 示例1
- 输入：交易金额异常检测
- 输出：检测方案，含统计方法、阈值设置、告警流程

## 注意事项
- 要准确识别
- 要减少误报
- 要及时响应
