# 创新策略 (Innovation Strategy)

## 触发条件
- 创新规划
- 技术布局
- 研发方向
- 竞争壁垒

## 输入
- 技术趋势
- 市场需求
- 竞争态势
- 研发能力

## 输出
- 创新路线图
- 技术布局
- 研发优先级
- 风险评估

## Prompt模板
```
请制定创新策略。

背景：
- 技术趋势：[趋势]
- 市场需求：[需求]
- 竞争态势：[态势]
- 研发能力：[能力]

请输出：
1. 创新方向
2. 技术布局
3. 研发优先级
4. 时间规划
5. 风险评估
```

## 示例
### 示例1
- 输入：传统制造企业数字化转型
- 输出：创新策略，含智能制造、供应链数字化、新商业模式探索

## 注意事项
- 要面向未来
- 要结合实际
- 要资源聚焦
- 要容错机制
