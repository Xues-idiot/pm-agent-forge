# 行动项跟踪 (Action Item Tracking)

## 触发条件
- 任务分配
- 进度跟踪
- 责任落实
- 闭环管理

## 输入
- 行动项
- 责任人
- 完成标准
- 截止日期

## 输出
- 跟踪清单
- 完成状态
- 偏差分析
- 提醒机制

## Prompt模板
```
请跟踪行动项。

背景：
- 行动项：[行动]
- 责任人：[责任人]
- 完成标准：[标准]
- 截止日期：[日期]

请输出：
1. 跟踪清单
2. 完成状态
3. 偏差分析
4. 提醒机制
5. 升级流程
```

## 示例
### 示例1
- 输入：会议行动项
- 输出：行动项跟踪，周会回顾、状态更新、逾期提醒

## 注意事项
- 要明确具体
- 要责任到人
- 要定期检查
- 要及时关闭
