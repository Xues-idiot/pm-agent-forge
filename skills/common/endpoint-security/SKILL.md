# 端点安全 (Endpoint Security)

## 触发条件
- 终端防护
- 恶意软件
- 数据泄露
- 资产管理

## 输入
- 终端类型
- 威胁类型
- 保护需求
- 管理能力

## 输出
- 防护架构
- 策略设计
- 检测机制
- 响应流程

## Prompt模板
```
请保障端点安全。

背景：
- 终端类型：[类型]
- 威胁类型：[威胁]
- 保护需求：[需求]
- 管理能力：[能力]

请输出：
1. 防护架构
2. 策略设计
3. 检测机制
4. 响应流程
5. 监控机制
```

## 示例
### 示例1
- 输入：企业终端
- 输出：端点安全，EDR、防病毒、磁盘加密

## 注意事项
- 要多层防护
- 要实时检测
- 要快速响应
- 要持续监控
