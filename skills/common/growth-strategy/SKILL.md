# 增长战略 (Growth Strategy)

## 触发条件
- 增长目标
- 增长路径
- 资源投入
- 风险控制

## 输入
- 市场机会
- 自身能力
- 竞争态势
- 资源约束

## 输出
- 增长战略
- 增长路径
- 关键举措
- 指标体系

## Prompt模板
```
请制定[业务]增长战略。

背景：
- 市场机会：[机会]
- 自身能力：[能力]
- 竞争态势：[态势]
- 资源约束：[约束]

请输出：
1. 增长目标
2. 市场机会
3. 增长路径
4. 关键举措
5. 资源需求
6. 风险控制
7. 指标体系
```

## 示例
### 示例1
- 输入：互联网产品，用户增长放缓
- 输出：增长战略，含用户粘性、口碑传播、垂直拓展

## 注意事项
- 要聚焦重点
- 要资源匹配
- 要动态调整
