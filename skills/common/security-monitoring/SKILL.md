# 安全监控 (Security Monitoring)

## 触发条件
- 安全威胁
- 实时监控
- 异常检测
- 应急响应

## 输入
- 监控范围
- 日志来源
- 威胁情报
- 响应能力

## 输出
- 监控方案
- 日志分析
- 告警机制
- 响应流程

## Prompt模板
```
请进行安全监控。

背景：
- 监控范围：[范围]
- 日志来源：[来源]
- 威胁情报：[情报]
- 响应能力：[能力]

请输出：
1. 监控方案
2. 日志分析
3. 告警机制
4. 响应流程
5. 评估机制
```

## 示例
### 示例1
- 输入：企业安全
- 输出：安全监控，SIEM集成、实时告警、安全运营中心

## 注意事项
- 要全面覆盖
- 要实时告警
- 要快速响应
- 要持续优化
