# 元数据管理 (Metadata Management)

## 触发条件
- 数据目录
- 数据发现
- 血缘追踪
- 标准定义

## 输入
- 数据资产
- 技术元数据
- 业务元数据
- 管理元数据

## 输出
- 元模型
- 数据目录
- 血缘关系
- 标准定义

## Prompt模板
```
请管理元数据。

背景：
- 数据资产：[资产]
- 技术元数据：[技术]
- 业务元数据：[业务]
- 管理元数据：[管理]

请输出：
1. 元模型
2. 数据目录
3. 血缘关系
4. 标准定义
5. 访问控制
```

## 示例
### 示例1
- 输入：数据仓库
- 输出：元数据管理，表结构、血缘关系、业务定义

## 注意事项
- 要自动化
- 要业务易懂
- 要持续更新
- 要工具支撑
