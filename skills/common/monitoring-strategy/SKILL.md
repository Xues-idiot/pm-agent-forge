# 监控策略 (Monitoring Strategy)

## 触发条件
- 系统监控
- 性能管理
- 告警优化
- 稳定性保障

## 输入
- 系统架构
- 监控需求
- 告警现状
- 工具能力

## 输出
- 监控体系
- 指标设计
- 告警规则
- 响应流程

## Prompt模板
```
请制定监控策略。

背景：
- 系统架构：[架构]
- 监控需求：[需求]
- 告警现状：[现状]
- 工具能力：[工具]

请输出：
1. 监控体系
2. 指标设计
3. 告警规则
4. 响应流程
5. 可视化
```

## 示例
### 示例1
- 输入：分布式系统
- 输出：监控策略，含基础设施、应用层、业务层监控，告警分级、On-Call机制

## 注意事项
- 要全面覆盖
- 要告警有效
- 要持续优化
- 要全天候保障
