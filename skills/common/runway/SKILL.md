# 资金跑道 (Runway Management)

## 触发条件
- 资金管理
- 烧钱率
- 融资节奏
- 风险预警

## 输入
- 现金储备
- 月烧钱率
- 融资能力
- 风险偏好

## 输出
- 跑道分析
- 烧钱率
- 融资规划
- 风险预警

## Prompt模板
```
请管理资金跑道。

背景：
- 现金储备：[现金]
- 月烧钱率：[烧钱率]
- 融资能力：[融资能力]
- 风险偏好：[风险]

请输出：
1. 跑道分析
2. 烧钱率
3. 融资规划
4. 风险预警
5. 应急方案
```

## 示例
### 示例1
- 输入：创业公司
- 输出：跑道分析，现金1000万、月烧钱100万、跑道10个月

## 注意事项
- 要密切监控
- 要提前融资
- 要控制成本
- 要风险管理
