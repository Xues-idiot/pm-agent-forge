# 收益实现 (Benefit Realization)

## 触发条件
- 价值交付
- ROI追踪
- 项目评估
- 持续改进

## 输入
- 预期收益
- 衡量指标
- 基线数据
- 实现时间

## 输出
- 收益追踪
- 指标对比
- 实现评估
- 调整建议

## Prompt模板
```
请追踪收益实现。

背景：
- 预期收益：[预期]
- 衡量指标：[指标]
- 基线数据：[基线]
- 实现时间：[时间]

请输出：
1. 收益追踪
2. 指标对比
3. 实现评估
4. 偏差分析
5. 调整建议
```

## 示例
### 示例1
- 输入：数字化转型项目
- 输出：收益实现，效率提升20%、成本降低15%、客户满意度提升

## 注意事项
- 要量化收益
- 要定期追踪
- 要偏差分析
- 要持续改进
