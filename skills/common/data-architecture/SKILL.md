# 数据架构 (Data Architecture)

## 触发条件
- 数据平台
- 架构设计
- 技术选型
- 能力规划

## 输入
- 业务需求
- 数据规模
- 技术趋势
- 成本预算

## 输出
- 架构蓝图
- 技术选型
- 实施路径
- 能力规划

## Prompt模板
```
请设计数据架构。

背景：
- 业务需求：[需求]
- 数据规模：[规模]
- 技术趋势：[趋势]
- 成本预算：[预算]

请输出：
1. 架构蓝图
2. 技术选型
3. 实施路径
4. 能力规划
5. 风险评估
```

## 示例
### 示例1
- 输入：数据平台
- 输出：数据架构，Lakehouse、数据湖、数据仓库分层

## 注意事项
- 要业务导向
- 要弹性扩展
- 要成本效益
- 要技术前瞻
