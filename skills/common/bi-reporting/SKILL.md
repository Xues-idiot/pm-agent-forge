# BI报表 (BI Reporting)

## 触发条件
- 报表需求
- 数据可视化
- 自助分析
- 决策支持

## 输入
- 业务需求
- 数据来源
- 用户群体
- 使用场景

## 输出
- 报表架构
- 报表设计
- 权限管理
- 发布流程

## Prompt模板
```
请设计BI报表。

背景：
- 业务需求：[需求]
- 数据来源：[来源]
- 用户群体：[群体]
- 使用场景：[场景]

请输出：
1. 报表架构
2. 指标体系
3. 报表设计
4. 权限管理
5. 发布流程
```

## 示例
### 示例1
- 输入：销售管理报表
- 输出：BI报表，含销售看板、区域分析、产品排名、预测报表

## 注意事项
- 要需求驱动
- 要指标统一
- 要性能优化
- 要易于使用
