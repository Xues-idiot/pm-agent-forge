# 三点估算 (Three-point Estimation)

## 触发条件
- 不确定性
- 风险管理
- 进度预测
- 成本估算

## 输入
- 乐观估计
- 最可能
- 悲观估计
- 分布假设

## 输出
- 期望值
- 标准差
- 置信区间
- 风险量化

## Prompt模板
```
请进行三点估算。

背景：
- 乐观估计：[乐观]
- 最可能估计：[最可能]
- 悲观估计：[悲观]
- 分布假设：[分布]

请输出：
1. 期望值计算
2. 标准差
3. 置信区间
4. 风险量化
5. 决策建议
```

## 示例
### 示例1
- 输入：任务工期估算
- 输出：三点估算，O=5天、M=7天、P=12天，期望6.2天

## 注意事项
- 要基于经验
- 要考虑风险
- 要动态调整
- 要持续改进
