# 数字化转型 (Digital Transformation)

## 触发条件
- 数字化战略
- 业务数字化
- 技术升级
- 组织变革

## 输入
- 业务现状
- 数字化目标
- 技术基础
- 投资预算

## 输出
- 转型战略
- 实施路径
- 技术方案
- 组织变革

## Prompt模板
```
请制定[组织]数字化转型战略。

背景：
- 业务现状：[现状]
- 数字化目标：[目标]
- 技术基础：[基础]
- 投资预算：[预算]

请输出：
1. 数字化成熟度评估
2. 转型愿景
3. 转型战略
4. 应用场景
5. 技术架构
6. 数据战略
7. 组织变革
8. 实施路径
9. 投资回报
```

## 示例
### 示例1
- 输入：传统制造企业，需要数字化转型提升效率
- 输出：转型战略，含智能制造、数字供应链、数据驱动

## 注意事项
- 要业务导向
- 要一把手工程
- 要持续投入
