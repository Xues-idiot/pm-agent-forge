# 服务级别协议 (Service Level Agreement)

## 触发条件
- SLA定义
- 服务标准
- 客户承诺
- 运维管理

## 输入
- 服务范围
- 客户需求
- 服务能力
- 行业标准

## 输出
- SLA定义
- 指标体系
- 监控机制
- 违约处理

## Prompt模板
```
请制定服务级别协议。

背景：
- 服务范围：[范围]
- 客户需求：[需求]
- 服务能力：[能力]
- 行业标准：[标准]

请输出：
1. 服务范围定义
2. SLA指标
3. 服务标准
4. 监控机制
5. 违约处理
```

## 示例
### 示例1
- 输入：企业SaaS服务
- 输出：SLA定义，含99.9%可用性、4小时响应、7×24监控

## 注意事项
- 要务实可达成
- 要明确衡量标准
- 要建立监控
- 要定期回顾
