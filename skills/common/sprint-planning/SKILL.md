# Sprint规划 (Sprint Planning)

## 触发条件
- 敏捷开发
- 迭代管理
- 任务分配
- 目标承诺

## 输入
- 产品待办
- 团队能力
- Sprint目标
- 历史速度

## 输出
- Sprint目标
- 任务分配
- 工作量评估
- 风险识别

## Prompt模板
```
请规划Sprint。

背景：
- 产品待办：[待办]
- 团队能力：[能力]
- Sprint目标：[目标]
- 历史速度：[速度]

请输出：
1. Sprint目标
2. 任务选择
3. 工作量评估
4. 任务分配
5. 风险识别
```

## 示例
### 示例1
- 输入：两周Sprint
- 输出：Sprint规划，含目标设定、故事点分配、任务细化、风险识别

## 注意事项
- 要团队参与
- 要容量匹配
- 要目标明确
- 要持续改进
