# 数据库选型 (Database Selection)

## 触发条件
- 数据库选择
- 技术选型
- 成本评估
- 性能需求

## 输入
- 数据特点
- 访问模式
- 一致性要求
- 规模预期

## 输出
- 选型分析
- 成本评估
- 迁移方案
- 风险评估

## Prompt模板
```
请选择数据库。

背景：
- 数据特点：[特点]
- 访问模式：[模式]
- 一致性要求：[要求]
- 规模预期：[预期]

请输出：
1. 选型分析
2. 成本评估
3. 架构设计
4. 迁移方案
5. 风险评估
```

## 示例
### 示例1
- 输入：社交应用
- 输出：数据库选型，含MySQL用户主数据、Redis缓存、MongoDB内容存储

## 注意事项
- 要场景匹配
- 要成本效益
- 要团队能力
- 要渐进迁移
