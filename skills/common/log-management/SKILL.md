# 日志管理 (Log Management)

## 触发条件
- 日志收集
- 问题排查
- 安全审计
- 合规要求

## 输入
- 日志类型
- 数据量级
- 存储需求
- 分析需求

## 输出
- 日志架构
- 收集方案
- 分析策略
- 存储规划

## Prompt模板
```
请管理日志。

背景：
- 日志类型：[类型]
- 数据量级：[量级]
- 存储需求：[需求]
- 分析需求：[需求]

请输出：
1. 日志架构
2. 收集方案
3. 分析策略
4. 存储规划
5. 保留策略
```

## 示例
### 示例1
- 输入：互联网应用
- 输出：日志管理，含集中式日志平台、实时分析、安全审计、合规保留

## 注意事项
- 要规范格式
- 要集中存储
- 要分级管理
- 要成本控制
