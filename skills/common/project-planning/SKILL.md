# 项目规划 (Project Planning)

## 触发条件
- 制定项目计划
- 项目分解结构
- 资源规划
- 风险规划

## 输入
- 项目目标
- 时间约束
- 资源限制
- 干系人

## 输出
- 项目计划书
- WBS分解
- 里程碑计划
- 风险登记册

## Prompt模板
```
请为[项目]制定项目计划。

背景：
- 项目目标：[目标]
- 时间约束：[约束]
- 资源限制：[资源]
- 干系人：[干系人]

请输出：
1. 项目概述和目标
2. 项目范围定义
3. WBS工作分解结构
4. 里程碑计划
5. 详细进度计划
6. 资源需求和分配
7. 成本估算
8. 风险识别和应对
9. 沟通管理计划
10. 质量保证措施
```

## 示例
### 示例1
- 输入：ERP系统实施项目，预算500万，6个月，SAP系统
- 输出：项目计划书，含WBS、里程碑、资源计划

## 注意事项
- 目标要SMART
- 分解要适度
- 风险要提前识别
