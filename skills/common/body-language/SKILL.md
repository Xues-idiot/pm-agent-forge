# 肢体语言 (Body Language)

## 触发条件
- 非语言沟通
- 形象表现
- 演讲展示
- 日常沟通

## 输入
- 沟通场景
- 目标效果
- 肢体习惯
- 文化背景

## 输出
- 肢体技巧
- 面部表情
- 空间利用
- 文化适配

## Prompt模板
```
请提升肢体语言。

背景：
- 沟通场景：[场景]
- 目标效果：[目标]
- 肢体习惯：[习惯]
- 文化背景：[文化]

请输出：
1. 肢体姿态
2. 面部表情
3. 眼神交流
4. 空间利用
5. 文化适配
```

## 示例
### 示例1
- 输入：演讲展示
- 输出：肢体语言，开放姿态、眼神交流、手势自然、走动适度

## 注意事项
- 要自然得体
- 要与语言一致
- 要文化适配
- 要情境合适
