# 容量规划 (Capacity Planning)

## 触发条件
- 资源规划
- 性能优化
- 成本控制
- 扩展性

## 输入
- 当前负载
- 增长预测
- 资源能力
- 成本预算

## 输出
- 容量模型
- 扩展计划
- 监控指标
- 优化建议

## Prompt模板
```
请进行容量规划。

背景：
- 当前负载：[负载]
- 增长预测：[预测]
- 资源能力：[能力]
- 成本预算：[预算]

请输出：
1. 容量模型
2. 增长预测
3. 扩展计划
4. 监控指标
5. 成本优化
```

## 示例
### 示例1
- 输入：云服务资源
- 输出：容量规划，含按需扩展、预留实例、成本优化、自动伸缩

## 注意事项
- 要提前规划
- 要监控趋势
- 要成本效益
- 要弹性扩展
