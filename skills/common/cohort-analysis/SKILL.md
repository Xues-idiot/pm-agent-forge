# 队列分析 (Cohort Analysis)

## 触发条件
- 用户行为
- 留存分析
- 收入分析
- 生命周期

## 输入
- 用户分组
- 时间维度
- 行为指标
- 分析目的

## 输出
- 队列设计
- 留存曲线
- 收入贡献
- 洞察发现

## Prompt模板
```
请进行队列分析。

背景：
- 用户分组：[分组]
- 时间维度：[时间]
- 行为指标：[指标]
- 分析目的：[目的]

请输出：
1. 队列设计
2. 留存曲线
3. 收入贡献
4. 关键洞察
5. 行动建议
```

## 示例
### 示例1
- 输入：按月分析用户留存
- 输出：队列分析，含各月留存率、付费转化、生命周期价值

## 注意事项
- 要时间序列
- 要分组对比
- 要洞察驱动
