# 兼并收购 (Mergers & Acquisitions)

## 触发条件
- 并购机会
- 标的评估
- 尽职调查
- 整合规划

## 输入
- 战略目标
- 标的情况
- 估值预期
- 资源能力

## 输出
- 标的筛选
- 评估框架
- 风险分析
- 整合计划

## Prompt模板
```
请评估兼并收购机会。

背景：
- 战略目标：[目标]
- 标的情况：[标的]
- 估值预期：[预期]
- 资源能力：[能力]

请输出：
1. 标的筛选标准
2. 评估框架
3. 估值方法
4. 风险分析
5. 整合计划
```

## 示例
### 示例1
- 输入：大型企业收购互补型公司
- 输出：评估框架，含战略匹配度、财务状况、人力资源、技术产品、市场协同

## 注意事项
- 要战略驱动
- 要充分尽职调查
- 要重视文化匹配
- 要规划整合
