# 体验设计 (Experience Design)

## 触发条件
- 用户体验
- 界面优化
- 流程改进
- 满意度提升

## 输入
- 用户研究
- 现有体验
- 设计目标
- 技术约束

## 输出
- 设计方案
- 原型验证
- 迭代计划
- 评估指标

## Prompt模板
```
请设计体验。

背景：
- 用户研究：[研究]
- 现有体验：[体验]
- 设计目标：[目标]
- 技术约束：[约束]

请输出：
1. 设计方案
2. 原型验证
3. 迭代计划
4. 评估指标
5. 实施方案
```

## 示例
### 示例1
- 输入：App优化
- 输出：体验设计，信息架构、交互流程、视觉设计

## 注意事项
- 要用户中心
- 要持续迭代
- 要数据驱动
- 要跨职能协作
