# Epic拆分 (Epic Breaking)

## 触发条件
- 需求拆分
- 任务细化
- 工作量评估
- Sprint规划

## 输入
- Epic描述
- 业务目标
- 团队能力
- 拆分标准

## 输出
- 拆分方案
- 用户故事
- 依赖关系
- 优先级

## Prompt模板
```
请拆分Epic。

背景：
- Epic描述：[描述]
- 业务目标：[目标]
- 团队能力：[能力]
- 拆分标准：[标准]

请输出：
1. 拆分原则
2. 用户故事列表
3. 验收标准
4. 依赖关系
5. 优先级排序
```

## 示例
### 示例1
- 输入：大型Epic
- 输出：Epic拆分，含按用户角色、用户任务、验收条件拆分多个Story

## 注意事项
- 要用户价值
- 要可测试
- 要独立部署
- 要团队参与
