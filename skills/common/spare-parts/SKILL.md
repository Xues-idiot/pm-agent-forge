# 备件管理 (Spare Parts Management)

## 触发条件
- 备件库存
- 维修保障
- 资金占用
- 服务时效

## 输入
- 设备清单
- 故障率
- 备件成本
- 供应周期

## 输出
- 备件策略
- 库存方案
- 采购计划
- 成本优化

## Prompt模板
```
请管理备件。

背景：
- 设备清单：[清单]
- 故障率：[故障率]
- 备件成本：[成本]
- 供应周期：[周期]

请输出：
1. 备件策略
2. 库存方案
3. 采购计划
4. 成本优化
5. 监控机制
```

## 示例
### 示例1
- 输入：工业设备
- 输出：备件管理，核心备件库存、供应商寄售、JIT采购

## 注意事项
- 要保障服务
- 要控制库存
- 要供应商协同
- 要持续优化
