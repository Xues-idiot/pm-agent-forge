# 技术路线图 (Technology Roadmap)

## 触发条件
- 技术规划
- 研发优先级
- 技术债务
- 架构演进

## 输入
- 业务需求
- 技术趋势
- 现状评估
- 资源能力

## 输出
- 技术路线图
- 里程碑规划
- 资源需求
- 风险评估

## Prompt模板
```
请制定技术路线图。

背景：
- 业务需求：[需求]
- 技术趋势：[趋势]
- 现状评估：[评估]
- 资源能力：[能力]

请输出：
1. 技术目标
2. 路线图规划
3. 里程碑
4. 资源需求
5. 风险与应对
```

## 示例
### 示例1
- 输入：SaaS企业，技术栈老旧
- 输出：技术路线图，含微服务改造、云原生迁移、自动化运维

## 注意事项
- 要匹配业务战略
- 要考虑技术趋势
- 要平衡新功能和重构
- 要持续迭代
