# 投资回收期 (Payback Period)

## 触发条件
- 投资决策
- 回报评估
- 风险分析
- 资本预算

## 输入
- 投资金额
- 预期收益
- 现金流
- 折现率

## 输出
- 回收期计算
- 现金流分析
- 风险评估
- 决策建议

## Prompt模板
```
请计算投资回收期。

背景：
- 投资金额：[金额]
- 预期收益：[收益]
- 现金流：[现金流]
- 折现率：[折现率]

请输出：
1. 回收期计算
2. 现金流分析
3. 风险评估
4. 决策建议
5. 敏感性分析
```

## 示例
### 示例1
- 输入：项目投资
- 输出：回收期，静态回收期、动态回收期、投资回报率

## 注意事项
- 要现金流
- 要考虑折现
- 要风险调整
- 要多方案对比
