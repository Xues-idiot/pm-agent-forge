# 里程碑跟踪 (Milestone Tracking)

## 触发条件
- 项目进度
- 阶段交付
- 目标达成
- 风险预警

## 输入
- 项目计划
- 里程碑定义
- 完成状态
- 依赖关系

## 输出
- 里程碑进度
- 偏差分析
- 风险预警
- 纠偏建议

## Prompt模板
```
请跟踪项目里程碑。

背景：
- 项目计划：[计划]
- 里程碑定义：[定义]
- 完成状态：[状态]
- 依赖关系：[依赖]

请输出：
1. 里程碑进度
2. 偏差分析
3. 风险预警
4. 纠偏建议
5. 干系人沟通
```

## 示例
### 示例1
- 输入：软件开发
- 输出：里程碑跟踪，设计评审、编码完成、测试通过、上线

## 注意事项
- 要明确定义
- 要定期检查
- 要及时预警
- 要主动沟通
