# 客服机器人 (Chatbot Design)

## 触发条件
- 智能客服
- 自动回复
- 7×24服务
- 成本优化

## 输入
- 业务场景
- 知识库
- 交互设计
- 升级机制

## 输出
- 对话设计
- 意图识别
- 知识覆盖
- 升级策略

## Prompt模板
```
请设计客服机器人。

背景：
- 业务场景：[场景]
- 知识库：[知识]
- 交互设计：[交互]
- 升级机制：[升级]

请输出：
1. 对话设计
2. 意图识别
3. 知识覆盖
4. 升级策略
5. 效果评估
```

## 示例
### 示例1
- 输入：电商客服
- 输出：Chatbot设计，常见问题→订单查询→退换货→人工升级

## 注意事项
- 要覆盖高频
- 要自然对话
- 要及时升级
- 要持续学习
