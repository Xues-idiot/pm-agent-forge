# 项目复盘 (Project Retrospective)

## 触发条件
- 项目结束
- 经验总结
- 过程改进
- 知识沉淀

## 输入
- 项目目标
- 达成情况
- 成功因素
- 改进机会

## 输出
- 目标达成
- 经验总结
- 教训学习
- 改进建议

## Prompt模板
```
请进行项目复盘。

背景：
- 项目目标：[目标]
- 达成情况：[达成]
- 成功因素：[成功]
- 改进机会：[机会]

请输出：
1. 目标达成分析
2. 经验总结
3. 教训学习
4. 改进建议
5. 知识沉淀
```

## 示例
### 示例1
- 输入：项目结束
- 输出：项目复盘，成功因素、失败原因、改进计划

## 注意事项
- 要客观公正
- 要团队参与
- 要形成文档
- 要持续跟踪
