# 产品发布管理 (Product Launch)

## 触发条件
- 新品发布
- 发布计划
- 协同执行
- 市场节奏

## 输入
- 产品信息
- 目标市场
- 竞争情况
- 资源配置

## 输出
- 发布计划
- 协同流程
- 风险预案
- 效果评估

## Prompt模板
```
请管理产品发布。

背景：
- 产品信息：[信息]
- 目标市场：[市场]
- 竞争情况：[竞争]
- 资源配置：[资源]

请输出：
1. 发布计划
2. 协同流程
3. 部门职责
4. 风险预案
5. 效果评估
```

## 示例
### 示例1
- 输入：企业软件新版本发布
- 输出：发布计划，含内测→公测→正式发布，配合营销活动和客户培训

## 注意事项
- 要充分准备
- 要跨部门协同
- 要节奏把控
- 要复盘总结
