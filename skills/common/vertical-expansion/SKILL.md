# 垂直整合 (Vertical Expansion)

## 触发条件
- 产业链
- 价值链
- 控制权
- 利润率

## 输入
- 产业链位置
- 上下游
- 整合机会
- 资源需求

## 输出
- 整合策略
- 目标选择
- 进入方式
- 协同效应

## Prompt模板
```
请制定垂直整合策略。

背景：
- 产业链位置：[位置]
- 上下游：[上下游]
- 整合机会：[机会]
- 资源需求：[资源]

请输出：
1. 整合策略
2. 整合目标
3. 进入方式
4. 协同效应
5. 风险评估
```

## 示例
### 示例1
- 输入：电商平台
- 输出：垂直整合，自建物流→自建支付→自有品牌

## 注意事项
- 要战略价值
- 要资源匹配
- 要风险可控
- 要协同创造
