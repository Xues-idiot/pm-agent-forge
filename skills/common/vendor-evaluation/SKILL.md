# 供应商评估 (Vendor Evaluation)

## 触发条件
- 供应商选择
- 采购决策
- 合作管理
- 风险控制

## 输入
- 采购需求
- 供应商列表
- 评估标准
- 预算约束

## 输出
- 评估体系
- 供应商分级
- 选择建议
- 合作框架

## Prompt模板
```
请评估供应商。

背景：
- 采购需求：[需求]
- 供应商列表：[列表]
- 评估标准：[标准]
- 预算约束：[预算]

请输出：
1. 评估维度
2. 评估权重
3. 供应商分级
4. 选择建议
5. 合作框架
```

## 示例
### 示例1
- 输入：企业软件采购，多家供应商
- 输出：评估体系，含功能匹配度、技术先进性、服务能力、成本效益、风险评估

## 注意事项
- 要多维度评估
- 要参考市场口碑
- 要试用验证
- 要考虑长期合作
