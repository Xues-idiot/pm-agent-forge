# 敏捷转型 (Agile Transformation)

## 触发条件
- 敏捷变革
- 方法导入
- 文化转变
- 效率提升

## 输入
- 现状评估
- 转型目标
- 组织准备
- 变革阻力

## 输出
- 转型策略
- 实施路径
- 培训计划
- 监控机制

## Prompt模板
```
请推动敏捷转型。

背景：
- 现状评估：[评估]
- 转型目标：[目标]
- 组织准备：[准备]
- 变革阻力：[阻力]

请输出：
1. 转型策略
2. 实施路径
3. 培训计划
4. 监控机制
5. 成功标准
```

## 示例
### 示例1
- 输入：传统企业
- 输出：敏捷转型，试点团队、敏捷培训、持续改进

## 注意事项
- 要高层支持
- 要小步快跑
- 要持续改进
- 要全员参与
