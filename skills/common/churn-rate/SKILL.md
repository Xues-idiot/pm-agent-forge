# 流失率分析 (Churn Rate)

## 触发条件
- 客户流失
- 留存分析
- 流失预防
- 价值保护

## 输入
- 流失用户
- 留存用户
- 时间周期
- 流失原因

## 输出
- 流失率
- 流失原因
- 流失预警
- 预防策略

## Prompt模板
```
请分析流失率。

背景：
- 流失用户：[流失]
- 留存用户：[留存]
- 时间周期：[周期]
- 流失原因：[原因]

请输出：
1. 流失率计算
2. 流失原因分析
3. 流失特征
4. 流失预警
5. 预防策略
```

## 示例
### 示例1
- 输入：订阅服务月流失
- 输出：流失分析，月流失率5%，主动取消占60%，被动过期占40%

## 注意事项
- 要细分流失
- 要早发现
- 要分析原因
- 要主动预防
