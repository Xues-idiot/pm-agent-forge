# 导师制度 (Mentorship)

## 触发条件
- 人才培养
- 知识传承
- 职业辅导
- 文化传递

## 输入
- 导师资源
- 学员需求
- 匹配方案
- 目标设定

## 输出
- 导师匹配
- 辅导计划
- 评估机制
- 激励措施

## Prompt模板
```
请设计导师制度。

背景：
- 导师资源：[资源]
- 学员需求：[需求]
- 匹配方案：[匹配]
- 目标设定：[目标]

请输出：
1. 导师匹配
2. 辅导目标
3. 辅导计划
4. 评估机制
5. 激励措施
```

## 示例
### 示例1
- 输入：新员工导师
- 输出：导师制度，导师匹配、辅导目标、月度沟通、效果评估

## 注意事项
- 要目标明确
- 要自愿参与
- 要定期沟通
- 要效果评估
