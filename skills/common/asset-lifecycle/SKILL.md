# 资产生命周期 (Asset Lifecycle)

## 触发条件
- 资产管理
- 采购决策
- 运维成本
- 处置规划

## 输入
- 资产类型
- 采购时间
- 运维状态
- 折旧情况

## 输出
- 生命周期
- 维护计划
- 更新策略
- 处置方案

## Prompt模板
```
请管理资产生命周期。

背景：
- 资产类型：[类型]
- 采购时间：[时间]
- 运维状态：[状态]
- 折旧情况：[折旧]

请输出：
1. 生命周期阶段
2. 维护计划
3. 更新策略
4. 处置方案
5. 成本分析
```

## 示例
### 示例1
- 输入：IT设备
- 输出：资产生命周期，采购→部署→运维→淘汰→处置

## 注意事项
- 要全流程
- 要成本效益
- 要及时更新
- 要合规处置
