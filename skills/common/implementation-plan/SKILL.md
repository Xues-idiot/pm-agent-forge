# 实施规划 (Implementation Plan)

## 触发条件
- 项目实施
- 上线计划
- 切换策略
- 风险管理

## 输入
- 项目范围
- 资源情况
- 时间约束
- 风险容忍

## 输出
- 实施计划
- 里程碑
- 切换策略
- 风险预案

## Prompt模板
```
请制定实施规划。

背景：
- 项目范围：[范围]
- 资源情况：[资源]
- 时间约束：[约束]
- 风险容忍：[容忍]

请输出：
1. 项目范围
2. 实施策略
3. 里程碑
4. 资源计划
5. 切换策略
6. 风险管理
```

## 示例
### 示例1
- 输入：ERP系统上线，需要切换旧系统
- 输出：实施规划，含分阶段上线、数据迁移、并行运行

## 注意事项
- 要风险可控
- 要平滑切换
- 要回滚预案
