# 趋势分析 (Trend Analysis)

## 触发条件
- 数据分析
- 趋势预测
- 模式识别
- 决策支持

## 输入
- 历史数据
- 时间范围
- 分析维度
- 预测目标

## 输出
- 趋势识别
- 模式分析
- 预测模型
- 建议行动

## Prompt模板
```
请进行趋势分析。

背景：
- 历史数据：[数据]
- 时间范围：[范围]
- 分析维度：[维度]
- 预测目标：[目标]

请输出：
1. 趋势识别
2. 模式分析
3. 预测模型
4. 建议行动
5. 置信区间
```

## 示例
### 示例1
- 输入：销售数据
- 输出：趋势分析，季节性波动、增长趋势、预测值

## 注意事项
- 要数据质量
- 要多维度
- 要模型验证
- 要结合业务
