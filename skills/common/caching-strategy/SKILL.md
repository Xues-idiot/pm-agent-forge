# 缓存策略 (Caching Strategy)

## 触发条件
- 性能优化
- 访问加速
- 成本降低
- 一致性

## 输入
- 数据特征
- 访问模式
- 缓存类型
- 一致性要求

## 输出
- 缓存设计
- 过期策略
- 更新机制
- 容量规划

## Prompt模板
```
请设计缓存策略。

背景：
- 数据特征：[特征]
- 访问模式：[模式]
- 缓存类型：[类型]
- 一致性要求：[一致性]

请输出：
1. 缓存设计
2. 过期策略
3. 更新机制
4. 容量规划
5. 监控指标
```

## 示例
### 示例1
- 输入：热点数据
- 输出：缓存策略，Redis缓存、LRU淘汰、延迟双删

## 注意事项
- 要热点优先
- 要过期合理
- 要一致性与性能平衡
- 要容量规划
