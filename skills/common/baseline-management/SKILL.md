# 基线管理 (Baseline Management)

## 触发条件
- 基准控制
- 变更评估
- 绩效对比
- 偏差分析

## 输入
- 范围基线
- 进度基线
- 成本基线
- 变更请求

## 输出
- 基线对比
- 偏差分析
- 变更影响
- 恢复计划

## Prompt模板
```
请管理基线。

背景：
- 范围基线：[范围]
- 进度基线：[进度]
- 成本基线：[成本]
- 变更请求：[变更]

请输出：
1. 基线对比
2. 偏差分析
3. 变更影响
4. 恢复计划
5. 基线更新
```

## 示例
### 示例1
- 输入：项目执行中
- 输出：基线管理，与基线对比、偏差分析、变更影响、恢复计划

## 注意事项
- 要严格执行
- 要评估变更
- 要偏差可控
- 要规范更新
