# 网红营销 (Influencer Marketing)

## 触发条件
- KOL合作
- 种草推广
- 品牌曝光
- 效果评估

## 输入
- 营销目标
- 目标KOL
- 合作形式
- 预算范围

## 输出
- KOL策略
- 合作方案
- 内容规划
- 效果评估

## Prompt模板
```
请策划网红营销。

背景：
- 营销目标：[目标]
- 目标KOL：[KOL]
- 合作形式：[形式]
- 预算范围：[预算]

请输出：
1. KOL策略
2. 合作方案
3. 内容规划
4. 效果评估
5. 风险控制
```

## 示例
### 示例1
- 输入：品牌推广
- 输出：网红营销，头部KOL+腰部KOL组合、种草+直播

## 注意事项
- 要选对KOL
- 要内容真实
- 要效果可量化
- 要合规管理
