# 流程优化 (Process Optimization)

## 触发条件
- 流程诊断
- 流程再造
- 效率提升
- 标准化

## 输入
- 流程现状
- 优化目标
- 约束条件
- 涉及部门

## 输出
- 流程诊断
- 优化方案
- 实施方案
- 效果评估

## Prompt模板
```
请优化[流程/业务]。

背景：
- 流程现状：[现状]
- 优化目标：[目标]
- 约束条件：[约束]
- 涉及部门：[部门]

请输出：
1. 流程现状描述
2. 流程诊断（耗时/瓶颈/浪费）
3. 优化目标设定
4. 优化方案设计
5. 流程再设计
6. 自动化机会
7. 实施计划
8. 变革管理
9. 效果评估
```

## 示例
### 示例1
- 输入：订单处理流程，当前5个节点，周期3天
- 输出：优化方案，含流程简化、自动化、周期缩短到1天

## 注意事项
- 要以终为始
- 要消除浪费
- 要持续改进
