# 运营评审 (Operational Review)

## 触发条件
- 运营绩效
- 效率评估
- 成本分析
- 改进机会

## 输入
- 运营数据
- 效率指标
- 成本数据
- 改进机会

## 输出
- 运营评估
- 效率分析
- 成本评估
- 改进建议

## Prompt模板
```
请进行运营评审。

背景：
- 运营数据：[数据]
- 效率指标：[指标]
- 成本数据：[成本]
- 改进机会：[机会]

请输出：
1. 运营评估
2. 效率分析
3. 成本评估
4. 风险分析
5. 改进建议
```

## 示例
### 示例1
- 输入：运营月度数据
- 输出：运营评审，效率提升10%、成本降低5%、改进机会

## 注意事项
- 要数据分析
- 要趋势分析
- 要对标优化
- 要持续改进
