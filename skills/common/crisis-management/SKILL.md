# 危机管理 (Crisis Management)

## 触发条件
- 危机预案
- 危机应对
- 危机沟通
- 危机恢复

## 输入
- 危机类型
- 影响范围
- 利益相关方
- 资源情况

## 输出
- 危机预案
- 应对方案
- 沟通策略
- 恢复计划

## Prompt模板
```
请设计[危机类型]的危机管理方案。

背景：
- 危机类型：[类型]
- 影响范围：[范围]
- 利益相关方：[相关方]
- 资源情况：[资源]

请输出：
1. 危机情景假设
2. 危机应对小组
3. 危机分级标准
4. 应对流程
5. 沟通策略
6. 媒体应对
7. 利益相关方沟通
8. 危机恢复计划
9. 事后复盘
10. 预防机制
```

## 示例
### 示例1
- 输入：食品企业，产品被曝光安全问题
- 输出：危机方案，含响应小组、真相公布、渠道召回、媒体沟通

## 注意事项
- 要快速响应
- 要诚实透明
- 要持续沟通
