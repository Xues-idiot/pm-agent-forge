# 监控设计 (Monitoring Design)

## 触发条件
- 系统监控
- 性能管理
- 故障检测
- 容量规划

## 输入
- 监控对象
- 监控指标
- 告警阈值
- 可视化需求

## 输出
- 监控架构
- 指标体系
- 告警配置
- 仪表盘

## Prompt模板
```
请设计监控系统。

背景：
- 监控对象：[对象]
- 监控指标：[指标]
- 告警阈值：[阈值]
- 可视化需求：[需求]

请输出：
1. 监控架构
2. 指标体系
3. 告警配置
4. 仪表盘
5. 报告机制
```

## 示例
### 示例1
- 输入：IT基础设施
- 输出：监控设计，基础设施、应用层、业务层监控

## 注意事项
- 要全面覆盖
- 要层次清晰
- 要及时告警
- 要便于排查
