# 知识传承 (Knowledge Transfer)

## 触发条件
- 经验沉淀
- 岗位交接
- 新人培训
- 能力复制

## 输入
- 知识来源
- 知识类型
- 受众需求
- 传递方式

## 输出
- 知识体系
- 传递计划
- 工具支持
- 效果评估

## Prompt模板
```
请进行知识传承。

背景：
- 知识来源：[来源]
- 知识类型：[类型]
- 受众需求：[需求]
- 传递方式：[方式]

请输出：
1. 知识梳理
2. 传递计划
3. 工具/平台
4. 评估机制
5. 持续更新
```

## 示例
### 示例1
- 输入：核心员工离职，岗位交接
- 输出：知识传承，含文档整理、操作手册、视频教程、交接期辅导

## 注意事项
- 要结构化沉淀
- 要实践验证
- 要持续更新
- 要激励参与
