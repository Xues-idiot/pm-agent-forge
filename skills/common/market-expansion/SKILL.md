# 市场扩张 (Market Expansion)

## 触发条件
- 新市场进入
- 地域扩展
- 渠道扩张
- 增长战略

## 输入
- 现有市场
- 目标市场
- 竞争优势
- 资源能力

## 输出
- 扩张策略
- 市场选择
- 进入模式
- 资源配置

## Prompt模板
```
请制定市场扩张策略。

背景：
- 现有市场：[现有]
- 目标市场：[目标]
- 竞争优势：[优势]
- 资源能力：[能力]

请输出：
1. 扩张策略
2. 市场选择
3. 进入模式
4. 资源配置
5. 风险评估
```

## 示例
### 示例1
- 输入：区域品牌
- 输出：市场扩张，全国一线城市→新一线城市→下沉市场

## 注意事项
- 要渐进入市
- 要本地适配
- 要资源配置
- 要风险控制
