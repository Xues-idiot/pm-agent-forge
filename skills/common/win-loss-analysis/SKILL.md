# 赢单输单分析 (Win-Loss Analysis)

## 触发条件
- 销售复盘
- 竞争分析
- 输单原因
- 策略优化

## 输入
- 赢单案例
- 输单案例
- 竞争对比
- 客户反馈

## 输出
- 赢单因素
- 输单原因
- 竞争洞察
- 策略建议

## Prompt模板
```
请进行赢单输单分析。

背景：
- 赢单案例：[赢单]
- 输单案例：[输单]
- 竞争对比：[对比]
- 客户反馈：[反馈]

请输出：
1. 赢单关键因素
2. 输单主要原因
3. 竞争优劣势
4. 策略改进建议
5. 销售话术优化
```

## 示例
### 示例1
- 输入：季度销售数据，赢单率35%
- 输出：分析结果，含赢单在产品功能匹配度胜出，输单在价格和服务响应

## 注意事项
- 要系统收集数据
- 要多维度分析
- 要与团队分享
- 要转化为行动
