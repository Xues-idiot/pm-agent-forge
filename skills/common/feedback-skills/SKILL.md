# 反馈技巧 (Feedback Skills)

## 触发条件
- 绩效反馈
- 行为反馈
- 建设性反馈
- 接受反馈

## 输入
- 反馈目的
- 反馈对象
- 反馈内容
- 反馈时机

## 输出
- 反馈策略
- 反馈内容
- 表达方式
- 跟进建议

## Prompt模板
```
请提升反馈技巧。

背景：
- 反馈目的：[目的]
- 反馈对象：[对象]
- 反馈内容：[内容]
- 反馈时机：[时机]

请输出：
1. 反馈策略
2. 具体内容
3. 表达方式
4. 接收反应
5. 跟进建议
```

## 示例
### 示例1
- 输入：行为反馈
- 输出：反馈技巧 SBI模型、具体行为、发展建议

## 注意事项
- 要具体行为
- 要及时反馈
- 要建设性
- 要双向沟通
