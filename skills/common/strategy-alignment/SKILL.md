# 战略对齐 (Strategy Alignment)

## 触发条件
- 目标分解
- 上下对齐
- 执行落地
- 绩效挂钩

## 输入
- 公司战略
- 部门目标
- 个人目标
- 资源分配

## 输出
- 对齐框架
- 目标分解
- 协同机制
- 评估机制

## Prompt模板
```
请实现战略对齐。

背景：
- 公司战略：[战略]
- 部门目标：[部门]
- 个人目标：[个人]
- 资源分配：[资源]

请输出：
1. 对齐框架
2. 目标分解
3. 协同机制
4. 评估机制
5. 调整流程
```

## 示例
### 示例1
- 输入：公司战略到部门
- 输出：战略对齐，战略→部门OKR→个人OKR→绩效评估

## 注意事项
- 要纵向对齐
- 要横向协同
- 要动态调整
- 要评估反馈
