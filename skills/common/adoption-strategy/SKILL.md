# 采用策略 (Adoption Strategy)

## 触发条件
- 用户采用
- 行为改变
- 培训推广
- 持续使用

## 输入
- 产品类型
- 目标用户
- 采用障碍
- 推广资源

## 输出
- 采用策略
- 推广计划
- 培训方案
- 激励机制

## Prompt模板
```
请设计采用策略。

背景：
- 产品类型：[类型]
- 目标用户：[用户]
- 采用障碍：[障碍]
- 推广资源：[资源]

请输出：
1. 采用目标
2. 障碍分析
3. 采用策略
4. 推广计划
5. 培训方案
6. 激励机制
```

## 示例
### 示例1
- 输入：新系统上线，员工不愿使用
- 输出：采用策略，含分批培训、激励政策、超级用户

## 注意事项
- 要用户视角
- 要循序渐进
- 要持续引导
