# 价值流分析 (Value Stream Mapping)

## 触发条件
- 流程优化
- 交付效率
- 库存管理
- 成本控制

## 输入
- 流程步骤
- 时间消耗
- 价值贡献
- 等待浪费

## 输出
- 价值流图
- 时间分析
- 浪费识别
- 改善机会

## Prompt模板
```
请进行价值流分析。

背景：
- 流程步骤：[步骤]
- 时间消耗：[时间]
- 价值贡献：[贡献]
- 等待浪费：[浪费]

请输出：
1. 当前状态图
2. 时间分析
3. 浪费识别
4. 改善机会
5. 未来状态设计
```

## 示例
### 示例1
- 输入：订单履行流程，需要缩短交付周期
- 输出：价值流分析，含各环节时间、浪费点、改善机会

## 注意事项
- 要端到端
- 要数据驱动
- 要持续改善
