# 决策日志 (Decision Log)

## 触发条件
- 决策记录
- 责任追溯
- 经验积累
- 团队对齐

## 输入
- 决策内容
- 决策依据
- 参与人
- 预期结果

## 输出
- 决策记录
- 依据文档
- 责任分配
- 效果追踪

## Prompt模板
```
请管理决策日志。

背景：
- 决策内容：[内容]
- 决策依据：[依据]
- 参与人：[参与]
- 预期结果：[结果]

请输出：
1. 决策记录
2. 决策依据
3. 责任分配
4. 效果追踪
5. 复盘时间
```

## 示例
### 示例1
- 输入：技术选型决策
- 输出：决策日志，微服务架构、依据性能需求、责任人CTO、效果追踪Q2

## 注意事项
- 要及时记录
- 要清晰依据
- 要责任明确
- 要效果追踪
