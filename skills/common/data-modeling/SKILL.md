# 数据建模 (Data Modeling)

## 触发条件
- 数据库设计
- 数据结构
- 查询优化
- 扩展性

## 输入
- 业务需求
- 数据特点
- 查询模式
- 性能要求

## 输出
- 概念模型
- 逻辑模型
- 物理模型
- 优化建议

## Prompt模板
```
请进行数据建模。

背景：
- 业务需求：[需求]
- 数据特点：[特点]
- 查询模式：[模式]
- 性能要求：[要求]

请输出：
1. 概念模型
2. 逻辑模型
3. 物理模型
4. 索引设计
5. 优化建议
```

## 示例
### 示例1
- 输入：订单系统
- 输出：数据模型，含订单主表、订单明细、商品表、用户表及关系设计

## 注意事项
- 要规范化设计
- 要考虑查询
- 要预留扩展
- 要性能优先
