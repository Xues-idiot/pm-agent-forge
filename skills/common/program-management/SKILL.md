# 项目集管理 (Program Management)

## 触发条件
- 多项目管理
- 项目协调
- 资源优化
- 收益实现

## 输入
- 项目组合
- 组织结构
- 战略目标
- 资源约束

## 输出
- 管理框架
- 协调机制
- 资源规划
- 收益管理

## Prompt模板
```
请设计[组织]项目集管理方案。

背景：
- 项目组合：[组合]
- 组织结构：[结构]
- 战略目标：[目标]
- 资源约束：[约束]

请输出：
1. 项目集目标
2. 组件项目关系
3. 治理结构
4. 协调机制
5. 资源管理
6. 收益管理
7. 风险管理
8. 进度管理
9. 沟通管理
```

## 示例
### 示例1
- 输入：集团同时推进10个IT项目
- 输出：管理方案，含项目优先级、资源池、收益管理

## 注意事项
- 要对齐战略
- 要协调资源
- 要管理依赖
