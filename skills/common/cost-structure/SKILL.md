# 成本结构 (Cost Structure)

## 触发条件
- 成本分析
- 效率提升
- 预算编制
- 定价决策

## 输入
- 成本分类
- 成本习性
- 规模效应
- 行业对标

## 输出
- 成本结构
- 成本分解
- 优化空间
- 改进路径

## Prompt模板
```
请分析成本结构。

背景：
- 成本分类：[分类]
- 成本习性：[习性]
- 规模效应：[效应]
- 行业对标：[对标]

请输出：
1. 成本结构
2. 成本分解
3. 优化空间
4. 改进路径
5. 监控指标
```

## 示例
### 示例1
- 输入：企业成本
- 输出：成本结构，固定/可变/半可变，优化机会

## 注意事项
- 要分类清晰
- 要习性明确
- 要对标分析
- 要持续优化
