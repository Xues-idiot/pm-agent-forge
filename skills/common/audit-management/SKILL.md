# 审计管理 (Audit Management)

## 触发条件
- 内部审计
- 合规审计
- 审计整改
- 风险防控

## 输入
- 审计范围
- 审计标准
- 历史发现
- 整改情况

## 输出
- 审计计划
- 审计发现
- 整改建议
- 跟踪报告

## Prompt模板
```
请管理审计工作。

背景：
- 审计范围：[范围]
- 审计标准：[标准]
- 历史发现：[发现]
- 整改情况：[整改]

请输出：
1. 审计计划
2. 审计范围
3. 审计发现
4. 整改建议
5. 跟踪报告
```

## 示例
### 示例1
- 输入：年度内部审计
- 输出：审计报告，含财务审计、运营审计、合规审计三大领域发现

## 注意事项
- 要独立客观
- 要全面覆盖
- 要跟踪整改
- 要持续改进
