# 公司治理 (Corporate Governance)

## 触发条件
- 治理结构
- 董事会
- 内部控制
- 合规要求

## 输入
- 股权结构
- 董事会构成
- 治理政策
- 法规要求

## 输出
- 治理框架
- 董事会结构
- 决策机制
- 监督机制

## Prompt模板
```
请优化公司治理。

背景：
- 股权结构：[结构]
- 董事会构成：[构成]
- 治理政策：[政策]
- 法规要求：[要求]

请输出：
1. 治理框架
2. 董事会结构
3. 决策机制
4. 监督机制
5. 信息披露
```

## 示例
### 示例1
- 输入：上市公司
- 输出：公司治理，董事会、监事会、战略委员会

## 注意事项
- 要合规运营
- 要透明披露
- 要内部控制
- 要持续完善
