# 容器策略 (Container Strategy)

## 触发条件
- 容器化
- 容器编排
- 运维效率
- 弹性扩展

## 输入
- 应用架构
- 团队能力
- 运维需求
- 成本预算

## 输出
- 容器化方案
- 编排策略
- 存储方案
- 网络设计

## Prompt模板
```
请制定容器策略。

背景：
- 应用架构：[架构]
- 团队能力：[能力]
- 运维需求：[需求]
- 成本预算：[预算]

请输出：
1. 容器化方案
2. 编排策略
3. 镜像管理
4. 存储方案
5. 网络设计
```

## 示例
### 示例1
- 输入：微服务应用
- 输出：容器策略，含Kubernetes编排、Harbor镜像仓库、CSI存储、Cilium网络

## 注意事项
- 要标准化
- 要安全扫描
- 要资源隔离
- 要成本控制
