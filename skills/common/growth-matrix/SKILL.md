# 增长矩阵 (Growth Matrix)

## 触发条件
- 增长策略
- 产品矩阵
- 市场机会
- 资源配置

## 输入
- 现有业务
- 增长机会
- 资源能力
- 风险偏好

## 输出
- 增长矩阵
- 业务组合
- 资源配置
- 优先级

## Prompt模板
```
请使用增长矩阵。

背景：
- 现有业务：[现有]
- 增长机会：[机会]
- 资源能力：[能力]
- 风险偏好：[风险]

请输出：
1. 增长矩阵
2. 各业务定位
3. 资源配置
4. 优先级排序
5. 风险评估
```

## 示例
### 示例1
- 输入：多元化企业
- 输出：增长矩阵，现有核心、相关扩展、新兴业务、创新探索

## 注意事项
- 要组合平衡
- 要资源配置
- 要动态调整
- 要风险控制
