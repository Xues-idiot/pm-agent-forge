# 采购订单 (Purchase Order)

## 触发条件
- 订单创建
- 供应商确认
- 交付跟踪
- 发票核对

## 输入
- 采购内容
- 供应商信息
- 价格条款
- 交付要求

## 输出
- 订单信息
- 确认回执
- 交付状态
- 收货确认

## Prompt模板
```
请管理采购订单。

背景：
- 采购内容：[内容]
- 供应商信息：[信息]
- 价格条款：[条款]
- 交付要求：[要求]

请输出：
1. 订单信息
2. 确认回执
3. 交付状态
4. 收货确认
5. 付款申请
```

## 示例
### 示例1
- 输入：采购订单
- 输出：订单管理，订单创建、确认、交货、收货、付款

## 注意事项
- 要信息准确
- 要及时确认
- 要跟踪交付
- 要验收把关
