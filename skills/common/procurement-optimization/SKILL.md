# 采购优化 (Procurement Optimization)

## 触发条件
- 采购效率
- 成本降低
- 流程改进
- 供应商整合

## 输入
- 采购支出
- 供应市场
- 现有流程
- 优化目标

## 输出
- 优化方案
- 成本分析
- 流程改进
- 供应商策略

## Prompt模板
```
请优化采购。

背景：
- 采购支出：[支出]
- 供应市场：[市场]
- 现有流程：[流程]
- 优化目标：[目标]

请输出：
1. 优化方案
2. 成本分析
3. 流程改进
4. 供应商策略
5. 实施计划
```

## 示例
### 示例1
- 输入：企业采购
- 输出：采购优化，集中采购、供应商整合、电子采购

## 注意事项
- 要总成本最低
- 要流程高效
- 要供应商协同
- 要持续改进
