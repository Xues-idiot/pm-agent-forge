# 优先级框架 (Priority Framework)

## 触发条件
- 任务排序
- 资源分配
- 时间管理
- 价值最大化

## 输入
- 任务列表
- 资源约束
- 目标优先级
- 时间限制

## 输出
- 优先级标准
- 排序方法
- 资源分配
- 动态调整

## Prompt模板
```
请设计优先级框架。

背景：
- 任务列表：[列表]
- 资源约束：[约束]
- 目标优先级：[目标]
- 时间限制：[限制]

请输出：
1. 优先级维度
2. 评分标准
3. 排序方法
4. 资源分配
5. 动态调整
```

## 示例
### 示例1
- 输入：产品经理，需求多时间少
- 输出：优先级框架，含价值-复杂度矩阵、MoSCoW

## 注意事项
- 要价值导向
- 要灵活调整
- 要透明沟通
