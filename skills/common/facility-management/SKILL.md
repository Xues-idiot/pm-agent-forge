# 设施管理 (Facility Management)

## 触发条件
- 设施运维
- 空间管理
- 安保管理
- 成本控制

## 输入
- 设施类型
- 规模面积
- 使用需求
- 管理目标

## 输出
- 运维方案
- 空间规划
- 安保体系
- 成本模型

## Prompt模板
```
请设计[设施]管理方案。

背景：
- 设施类型：[类型]
- 规模面积：[面积]
- 使用需求：[需求]
- 管理目标：[目标]

请输出：
1. 设施管理范围
2. 运维体系设计
3. 预防性维护
4. 空间管理
5. 安保管理
6. 环境卫生
7. 能源管理
8. 成本控制
9. 应急管理
10. 数字化平台
```

## 示例
### 示例1
- 输入：商业写字楼，10万平方米，需要提升运维效率
- 输出：管理方案，含FM系统、预防维护、能源优化

## 注意事项
- 要安全第一
- 要预防为主
- 要持续优化
