# 文化转型 (Culture Transformation)

## 触发条件
- 企业文化
- 价值观
- 行为改变
- 组织氛围

## 输入
- 现有文化
- 目标文化
- 转型阻力
- 领导示范

## 输出
- 文化诊断
- 转型策略
- 落地计划
- 评估机制

## Prompt模板
```
请推动文化转型。

背景：
- 现有文化：[现有]
- 目标文化：[目标]
- 转型阻力：[阻力]
- 领导示范：[示范]

请输出：
1. 文化诊断
2. 转型策略
3. 落地计划
4. 评估机制
5. 激励机制
```

## 示例
### 示例1
- 输入：创新文化
- 输出：文化转型，容错机制、创新奖励、跨部门协作

## 注意事项
- 要领导示范
- 要制度配套
- 要持续强化
- 要耐心坚持
