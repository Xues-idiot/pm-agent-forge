# 收入预测 (Revenue Forecast)

## 触发条件
- 财务规划
- 目标设定
- 资源分配
- 绩效评估

## 输入
- 历史收入
- 增长趋势
- 业务pipeline
- 市场环境

## 输出
- 预测模型
- 预测结果
- 假设条件
- 风险评估

## Prompt模板
```
请预测收入。

背景：
- 历史收入：[历史]
- 增长趋势：[趋势]
- 业务pipeline：[pipeline]
- 市场环境：[环境]

请输出：
1. 预测模型
2. 预测结果
3. 假设条件
4. 风险评估
5. 敏感性分析
```

## 示例
### 示例1
- 输入：企业收入
- 输出：收入预测，月度预测、季度预测、年度预测

## 注意事项
- 要数据支撑
- 要假设合理
- 要情景分析
- 要动态更新
