# 影响评估 (Impact Assessment)

## 触发条件
- 变更影响
- 风险评估
- 决策支持
- 资源规划

## 输入
- 变更内容
- 影响范围
- 历史数据
- 评估标准

## 输出
- 影响分析
- 风险评估
- 缓解措施
- 决策建议

## Prompt模板
```
请评估变更影响。

背景：
- 变更内容：[内容]
- 影响范围：[范围]
- 历史数据：[数据]
- 评估标准：[标准]

请输出：
1. 影响范围
2. 影响程度
3. 风险评估
4. 缓解措施
5. 决策建议
```

## 示例
### 示例1
- 输入：核心系统架构升级
- 输出：影响评估，含业务影响、技术影响、迁移风险、缓解措施

## 注意事项
- 要全面分析
- 要量化评估
- 要提前沟通
- 要持续跟踪
