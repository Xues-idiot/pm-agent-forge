# 速度追踪 (Velocity Tracking)

## 触发条件
- 团队绩效
- 预测能力
- 容量规划
- 过程改进

## 输入
- 历史速度
- 当前Sprint
- 团队配置
- 预测目标

## 输出
- 速度分析
- 趋势预测
- 容量规划
- 改进建议

## Prompt模板
```
请追踪团队速度。

背景：
- 历史速度：[历史]
- 当前Sprint：[当前]
- 团队配置：[配置]
- 预测目标：[目标]

请输出：
1. 速度分析
2. 趋势预测
3. 容量规划
4. 预测模型
5. 改进建议
```

## 示例
### 示例1
- 输入：敏捷团队
- 输出：速度追踪，含故事点趋势、预测Sprint完成、容量规划、能力提升

## 注意事项
- 要数据准确
- 要趋势分析
- 要容量匹配
- 要持续跟踪
