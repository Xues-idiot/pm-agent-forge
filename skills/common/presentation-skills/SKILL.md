# 演示技巧 (Presentation Skills)

## 触发条件
- 演讲展示
- 汇报呈现
- 说服沟通
- 舞台表现

## 输入
- 听众分析
- 内容结构
- 演示材料
- 时间控制

## 输出
- 演示结构
- 幻灯片设计
- 表达技巧
- 应对问答

## Prompt模板
```
请提升演示技巧。

背景：
- 听众分析：[听众]
- 内容结构：[结构]
- 演示材料：[材料]
- 时间控制：[时间]

请输出：
1. 演示结构
2. 幻灯片设计
3. 表达技巧
4. 时间控制
5. 应对问答
```

## 示例
### 示例1
- 输入：业务汇报
- 输出：演示技巧，观点先行、数据支撑、故事讲述、互动问答

## 注意事项
- 要了解听众
- 要结构清晰
- 要练习充分
- 要自然自信
