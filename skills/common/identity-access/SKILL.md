# 身份与访问 (Identity & Access)

## 触发条件
- 身份认证
- 权限管理
- 最小权限
- 合规审计

## 输入
- 用户群体
- 资源类型
- 认证需求
- 合规要求

## 输出
- IAM架构
- 认证策略
- 授权模型
- 审计机制

## Prompt模板
```
请设计身份与访问。

背景：
- 用户群体：[群体]
- 资源类型：[类型]
- 认证需求：[需求]
- 合规要求：[要求]

请输出：
1. IAM架构
2. 认证策略
3. 授权模型
4. 审计机制
5. 监控机制
```

## 示例
### 示例1
- 输入：企业访问
- 输出：身份与访问，SSO、MFA、基于角色访问控制

## 注意事项
- 要最小权限
- 要强认证
- 要持续验证
- 要全面审计
