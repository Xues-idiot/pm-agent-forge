# 安全框架 (Security Frameworks)

## 触发条件
- 安全建设
- 合规要求
- 最佳实践
- 评估标准

## 输入
- 安全目标
- 行业要求
- 现有能力
- 资源限制

## 输出
- 框架选择
- 控制措施
- 实施路径
- 评估机制

## Prompt模板
```
请构建安全框架。

背景：
- 安全目标：[目标]
- 行业要求：[要求]
- 现有能力：[能力]
- 资源限制：[限制]

请输出：
1. 框架选择
2. 控制措施
3. 实施路径
4. 评估机制
5. 认证计划
```

## 示例
### 示例1
- 输入：企业安全
- 输出：安全框架，ISO 27001、NIST CSF、安全控制措施

## 注意事项
- 要适配业务
- 要符合法规
- 要分步实施
- 要持续改进
