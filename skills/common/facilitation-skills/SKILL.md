# 引导技巧 (Facilitation Skills)

## 触发条件
- 会议引导
- 团队讨论
- 决策引导
- 参与促进

## 输入
- 讨论目标
- 参与者
- 讨论形式
- 时间限制

## 输出
- 引导设计
- 讨论方法
- 决策流程
- 产出定义

## Prompt模板
```
请提升引导技巧。

背景：
- 讨论目标：[目标]
- 参与者：[参与者]
- 讨论形式：[形式]
- 时间限制：[时间]

请输出：
1. 引导设计
2. 讨论方法
3. 提问技巧
4. 决策流程
5. 产出定义
```

## 示例
### 示例1
- 输入：头脑风暴
- 输出：引导技巧，发散思维、分类整理、优先级投票、行动计划

## 注意事项
- 要中立引导
- 要确保参与
- 要聚焦目标
- 要掌控时间
