# 本地化策略 (Localization Strategy)

## 触发条件
- 区域扩张
- 海外市场
- 文化适配
- 运营落地

## 输入
- 目标市场
- 产品特点
- 竞争格局
- 运营资源

## 输出
- 本地化策略
- 产品适配
- 运营规划
- 资源配置

## Prompt模板
```
请制定本地化策略。

背景：
- 目标市场：[市场]
- 产品特点：[特点]
- 竞争格局：[格局]
- 运营资源：[资源]

请输出：
1. 本地化策略
2. 产品适配方案
3. 运营规划
4. 团队搭建
5. 风险与应对
```

## 示例
### 示例1
- 输入：国内App进入东南亚
- 输出：本地化策略，含语言翻译、文化适配、支付集成、线下推广

## 注意事项
- 要尊重当地文化
- 要适配当地习惯
- 要控制成本
- 要渐进式推进
