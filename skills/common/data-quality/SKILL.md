# 数据质量 (Data Quality)

## 触发条件
- 数据质量
- 质量监控
- 问题追踪
- 质量改进

## 输入
- 数据类型
- 质量规则
- 问题数据
- 业务影响

## 输出
- 质量框架
- 监控规则
- 质量报告
- 改进流程

## Prompt模板
```
请管理数据质量。

背景：
- 数据类型：[类型]
- 质量规则：[规则]
- 问题数据：[问题]
- 业务影响：[影响]

请输出：
1. 质量框架
2. 质量维度
3. 监控规则
4. 质量报告
5. 改进流程
```

## 示例
### 示例1
- 输入：客户主数据
- 输出：数据质量，含完整性、准确性、一致性、时效性监控规则

## 注意事项
- 要规则明确
- 要持续监控
- 要及时整改
- 要业务认可
