# 问题跟踪 (Issue Tracking)

## 触发条件
- 问题管理
- 缺陷跟踪
- 进度监控
- 质量控制

## 输入
- 问题列表
- 优先级
- 责任人
- 截止日期

## 输出
- 跟踪体系
- 处理流程
- 升级机制
- 统计分析

## Prompt模板
```
请建立问题跟踪机制。

背景：
- 问题列表：[列表]
- 优先级：[优先级]
- 责任人：[责任人]
- 截止日期：[日期]

请输出：
1. 问题分类
2. 跟踪流程
3. 优先级定义
4. 升级机制
5. 统计分析
```

## 示例
### 示例1
- 输入：软件开发项目
- 输出：跟踪体系，含Bug严重等级、责任人分配、Sprint内闭环、逾期预警

## 注意事项
- 要及时记录
- 要明确责任
- 要定期回顾
- 要持续改进
