# 现金管理 (Cash Management)

## 触发条件
- 现金储备
- 流动性
- 收益率
- 风险控制

## 输入
- 现金需求
- 收支预测
- 投资渠道
- 风险偏好

## 输出
- 现金策略
- 投资方案
- 预警机制
- 收益分析

## Prompt模板
```
请管理现金。

背景：
- 现金需求：[需求]
- 收支预测：[预测]
- 投资渠道：[渠道]
- 风险偏好：[偏好]

请输出：
1. 现金策略
2. 投资方案
3. 预警机制
4. 收益分析
5. 流动性保障
```

## 示例
### 示例1
- 输入：企业现金
- 输出：现金管理，短期理财、货币基金、银行存款

## 注意事项
- 要流动性优先
- 要安全第一
- 要收益平衡
- 要合规要求
