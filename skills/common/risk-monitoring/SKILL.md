# 风险监控 (Risk Monitoring)

## 触发条件
- 风险跟踪
- 预警指标
- 动态评估
- 报告机制

## 输入
- 风险清单
- 监控指标
- 预警阈值
- 报告频率

## 输出
- 监控体系
- 预警机制
- 风险报告
- 响应建议

## Prompt模板
```
请监控风险。

背景：
- 风险清单：[清单]
- 监控指标：[指标]
- 预警阈值：[阈值]
- 报告频率：[频率]

请输出：
1. 监控体系
2. 预警机制
3. 风险报告
4. 响应建议
5. 升级机制
```

## 示例
### 示例1
- 输入：项目风险
- 输出：风险监控，红黄蓝预警、定期报告、动态更新

## 注意事项
- 要实时监控
- 要及时预警
- 要动态更新
- 要报告透明
