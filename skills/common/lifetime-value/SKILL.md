# 终身价值 (Lifetime Value)

## 触发条件
- 用户价值
- 投资回报
- 客户分层
- 运营决策

## 输入
- 用户行为
- 消费金额
- 留存周期
- 成本结构

## 输出
- LTV计算
- 用户分层
- 获取上限
- 运营建议

## Prompt模板
```
请计算终身价值。

背景：
- 用户行为：[行为]
- 消费金额：[金额]
- 留存周期：[周期]
- 成本结构：[成本]

请输出：
1. LTV计算
2. 用户分层
3. 获取上限
4. 运营建议
5. 监控指标
```

## 示例
### 示例1
- 输入：游戏产品
- 输出：LTV，玩家生命周期、付费深度、留存曲线

## 注意事项
- 要动态预测
- 要成本完整
- 要用户分层
- 要业务匹配
