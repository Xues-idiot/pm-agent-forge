# 配置管理 (Configuration Management)

## 触发条件
- 资产配置
- 关系管理
- 变更追踪
- 运维自动化

## 输入
- 配置项
- 依赖关系
- 变更历史
- 运维需求

## 输出
- 配置数据库
- 关系图谱
- 变更流程
- 自动化方案

## Prompt模板
```
请建立配置管理。

背景：
- 配置项：[项]
- 依赖关系：[关系]
- 变更历史：[历史]
- 运维需求：[需求]

请输出：
1. 配置数据库
2. 关系图谱
3. 变更管理
4. 监控集成
5. 自动化方案
```

## 示例
### 示例1
- 输入：微服务架构
- 输出：配置管理，含服务注册发现、配置中心、依赖关系图、自动化部署

## 注意事项
- 要准确完整
- 要及时更新
- 要可视化
- 要自动化
