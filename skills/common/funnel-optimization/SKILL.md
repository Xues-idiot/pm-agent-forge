# 漏斗优化 (Funnel Optimization)

## 触发条件
- 漏斗分析
- 流失分析
- 转化提升
- 用户旅程

## 输入
- 漏斗阶段
- 流失数据
- 用户行为
- 优化资源

## 输出
- 漏斗诊断
- 流失原因
- 优化策略
- 优先级

## Prompt模板
```
请优化漏斗。

背景：
- 漏斗阶段：[阶段]
- 流失数据：[数据]
- 用户行为：[行为]
- 优化资源：[资源]

请输出：
1. 漏斗诊断
2. 流失原因分析
3. 优化策略
4. 优先级排序
5. 效果预测
```

## 示例
### 示例1
- 输入：用户注册漏斗
- 输出：漏斗优化，含步骤简化、信息预填、进度展示、异常处理

## 注意事项
- 要分析流失
- 要找到瓶颈
- 要用户研究
- 要系统优化
