# 故障转移设计 (Failover Design)

## 触发条件
- 高可用
- 故障处理
- 业务连续
- 自动切换

## 输入
- 可用性目标
- 故障场景
- 切换时间
- 数据一致性

## 输出
- 故障转移架构
- 切换机制
- 健康检查
- 恢复流程

## Prompt模板
```
请设计故障转移。

背景：
- 可用性目标：[目标]
- 故障场景：[场景]
- 切换时间：[时间]
- 数据一致性：[一致性]

请输出：
1. 故障转移架构
2. 切换机制
3. 健康检查
4. 恢复流程
5. 验证测试
```

## 示例
### 示例1
- 输入：数据库高可用
- 输出：故障转移设计，主从切换、心跳检测、自动 failover

## 注意事项
- 要自动切换
- 要数据一致
- 要快速恢复
- 要定期演练
