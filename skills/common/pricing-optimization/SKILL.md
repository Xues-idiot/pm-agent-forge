# 价格优化 (Pricing Optimization)

## 触发条件
- 定价策略
- 价格测试
- 利润优化
- 竞争应对

## 输入
- 成本结构
- 客户支付意愿
- 竞争价格
- 价格敏感度

## 输出
- 价格策略
- 价格体系
- 测试方案
- 预期效果

## Prompt模板
```
请优化定价策略。

背景：
- 成本结构：[成本]
- 客户支付意愿：[意愿]
- 竞争价格：[竞争]
- 价格敏感度：[敏感度]

请输出：
1. 定价策略
2. 价格体系设计
3. 测试方案
4. 预期效果
5. 监控指标
```

## 示例
### 示例1
- 输入：在线课程，原价199元，转化率5%
- 输出：价格优化，含测试179/219/299三个价格点，考虑会员制和套餐制

## 注意事项
- 要平衡利润和转化
- 要进行价格测试
- 要考虑客户价值
- 要动态调整
