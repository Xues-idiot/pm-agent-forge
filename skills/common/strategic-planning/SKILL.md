# 战略规划 (Strategic Planning)

## 触发条件
- 中长期规划
- 战略制定
- 目标分解
- 执行跟踪

## 输入
- 企业愿景
- 市场环境
- 竞争态势
- 资源能力

## 输出
- 战略目标
- 战略路径
- 执行计划
- 评估机制

## Prompt模板
```
请制定战略规划。

背景：
- 企业愿景：[愿景]
- 市场环境：[环境]
- 竞争态势：[态势]
- 资源能力：[能力]

请输出：
1. 战略诊断
2. 战略目标
3. 战略路径
4. 执行计划
5. 资源配置
6. 评估机制
```

## 示例
### 示例1
- 输入：科技公司，3-5年战略规划
- 输出：战略规划，含愿景-战略-执行-评估

## 注意事项
- 要远见务实
- 要上下对齐
- 要动态调整
