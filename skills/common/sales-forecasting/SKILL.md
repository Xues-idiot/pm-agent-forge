# 销售预测 (Sales Forecasting)

## 触发条件
- 业绩预测
- 目标制定
- 资源规划
- 风险预警

## 输入
- 历史数据
- Pipeline数据
- 市场趋势
- 季节因素

## 输出
- 预测模型
- 季度预测
- 风险分析
- 调整建议

## Prompt模板
```
请进行销售预测。

背景：
- 历史数据：[数据]
- Pipeline数据：[Pipeline]
- 市场趋势：[趋势]
- 季节因素：[因素]

请输出：
1. 预测方法论
2. 预测结果
3. 置信区间
4. 风险分析
5. 调整建议
```

## 示例
### 示例1
- 输入：SaaS企业，年 ARR 1000万
- 输出：销售预测，含基于Pipeline的bottom-up和基于增长率的top-down两种方法

## 注意事项
- 要多方法验证
- 要考虑不确定性
- 要定期修正
- 要与实际对比
