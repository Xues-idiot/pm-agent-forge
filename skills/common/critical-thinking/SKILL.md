# 批判性思维 (Critical Thinking)

## Overview

批判性思维是系统化分析论证、识别逻辑谬误、形成独立判断的技能。涵盖论证分析、假设识别、逻辑检验、结论评估等核心环节。

## 何时使用

- 分析复杂问题
- 评估论证可靠性
- 识别逻辑漏洞
- 形成独立判断

## 流程

1. **论证识别**：识别论点和论据
2. **假设检验**：检验隐含假设
3. **逻辑分析**：分析论证结构
4. **证据评估**：评估证据质量
5. **谬误识别**：识别逻辑谬误
6. **结论形成**：形成自己的判断
7. **表达论证**：清晰表达观点

## 检查清单

- [ ] 论点论据区分清晰
- [ ] 隐含假设被识别和检验
- [ ] 逻辑结构分析完整
- [ ] 证据质量被评估
- [ ] 常见谬误被排查
- [ ] 结论有依据支撑

## 红牌警告

- 将观点当事实
- 忽视相反证据
- 逻辑推理有漏洞
- 证据来源不可靠
- 情绪化判断
