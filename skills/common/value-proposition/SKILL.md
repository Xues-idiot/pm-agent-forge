# 价值主张 (Value Proposition)

## 触发条件
- 价值定位
- 客户价值
- 差异化
- 价值沟通

## 输入
- 产品/服务
- 目标客户
- 竞争定位
- 价值驱动

## 输出
- 价值主张
- 价值框架
- 差异化
- 沟通策略

## Prompt模板
```
请为[产品/服务]设计价值主张。

背景：
- 产品/服务：[产品]
- 目标客户：[客户]
- 竞争定位：[定位]
- 价值驱动：[驱动]

请输出：
1. 客户需求分析
2. 价值主张框架
3. 核心价值主张
4. 价值要素
5. 差异化定位
6. 价值实现路径
7. 价值沟通
```

## 示例
### 示例1
- 输入：企业SaaS产品，差异化是AI功能
- 输出：价值主张，"AI驱动的效率提升"

## 注意事项
- 要客户视角
- 要具体可衡量
- 要差异化
