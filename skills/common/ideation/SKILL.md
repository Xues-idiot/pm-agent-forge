# 创意发散 (Ideation)

## 触发条件
- 创意生成
- 问题解决
- 头脑风暴
- 方案探索

## 输入
- 问题定义
- 参与人员
- 创意数量
- 评估标准

## 输出
- 创意清单
- 想法分类
- 初步筛选
- 发展方向

## Prompt模板
```
请进行创意发散。

背景：
- 问题定义：[问题]
- 参与人员：[人员]
- 创意数量：[数量]
- 评估标准：[标准]

请输出：
1. 创意清单
2. 想法分类
3. 初步筛选
4. 发展方向
5. 验证计划
```

## 示例
### 示例1
- 输入：产品创新
- 输出：创意发散，创意数量50+，分类归纳，重点方向

## 注意事项
- 要数量优先
- 要延迟评判
- 要跨界思维
- 要记录所有想法
