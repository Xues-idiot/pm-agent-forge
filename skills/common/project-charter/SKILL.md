# 项目章程 (Project Charter)

## 触发条件
- 项目立项
- 目标明确
- 授权明确
- 范围初步

## 输入
- 项目背景
- 目标目的
- 初步范围
- 资源假设

## 输出
- 项目章程
- 目标定义
- 角色职责
- 初步范围

## Prompt模板
```
请制定项目章程。

背景：
- 项目背景：[背景]
- 目标目的：[目标]
- 初步范围：[范围]
- 资源假设：[假设]

请输出：
1. 项目章程
2. 目标定义
3. 角色职责
4. 初步范围
5. 约束假设
```

## 示例
### 示例1
- 输入：新项目启动
- 输出：项目章程，含项目目的、目标、范围、组织、初步预算

## 注意事项
- 要高层授权
- 要目标清晰
- 要范围明确
- 要可操作
