# 安全运营 (Security Operations)

## 触发条件
- 信息安全
- 威胁防护
- 监控响应
- 合规要求

## 输入
- 安全态势
- 威胁情报
- 防护能力
- 合规需求

## 输出
- 安全策略
- 监控体系
- 响应流程
- 培训计划

## Prompt模板
```
请管理安全运营。

背景：
- 安全态势：[态势]
- 威胁情报：[情报]
- 防护能力：[能力]
- 合规需求：[需求]

请输出：
1. 安全策略
2. 监控体系
3. 响应流程
4. 应急处置
5. 培训计划
```

## 示例
### 示例1
- 输入：互联网企业
- 输出：安全运营，含SIEM部署、威胁狩猎、事件响应、安全意识培训

## 注意事项
- 要主动防御
- 要持续监控
- 要快速响应
- 要持续改进
