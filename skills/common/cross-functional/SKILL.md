# 跨职能协作 (Cross-Functional Collaboration)

## 触发条件
- 部门协作
- 资源共享
- 流程打通
- 目标对齐

## 输入
- 协作部门
- 协作目标
- 现有障碍
- 协调机制

## 输出
- 协作机制
- 沟通流程
- 资源共享
- 考核体系

## Prompt模板
```
请设计跨职能协作机制。

背景：
- 协作部门：[部门]
- 协作目标：[目标]
- 现有障碍：[障碍]
- 协调机制：[机制]

请输出：
1. 协作目标
2. 协作机制
3. 沟通流程
4. 资源共享
5. 考核体系
```

## 示例
### 示例1
- 输入：产研销协作不畅，影响交付
- 输出：协作机制，含联席会议、共享KPI、流程优化

## 注意事项
- 要目标对齐
- 要机制保障
- 要持续改进
