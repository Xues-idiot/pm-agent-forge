# 项目进度 (Project Schedule)

## 触发条件
- 进度规划
- 任务排序
- 时间估算
- 里程碑

## 输入
- 任务列表
- 依赖关系
- 资源约束
- 截止日期

## 输出
- 进度计划
- 关键路径
- 里程碑
- 资源分配

## Prompt模板
```
请制定项目进度。

背景：
- 任务列表：[任务]
- 依赖关系：[依赖]
- 资源约束：[约束]
- 截止日期：[日期]

请输出：
1. 进度计划
2. 任务排序
3. 关键路径
4. 里程碑
5. 资源分配
```

## 示例
### 示例1
- 输入：项目规划
- 输出：进度计划，甘特图、关键路径、里程碑、关键任务

## 注意事项
- 要合理估算
- 要关键路径
- 要留有缓冲
- 要动态调整
