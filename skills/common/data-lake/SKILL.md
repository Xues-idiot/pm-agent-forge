# 数据湖 (Data Lake)

## 触发条件
- 数据存储
- 原始数据
- 灵活分析
- 成本优化

## 输入
- 数据类型
- 存储需求
- 分析场景
- 成本预算

## 输出
- 湖仓架构
- 分区策略
- 格式选择
- 治理方案

## Prompt模板
```
请设计数据湖。

背景：
- 数据类型：[类型]
- 存储需求：[需求]
- 分析场景：[场景]
- 成本预算：[预算]

请输出：
1. 湖仓架构
2. 分区策略
3. 格式选择
4. 治理方案
5. 成本优化
```

## 示例
### 示例1
- 输入：机器学习数据
- 输出：数据湖架构，含Parquet存储、Delta Lake湖仓一体、分区管理

## 注意事项
- 要避免数据沼泽
- 要schema管理
- 要成本优化
- 要安全合规
