# ETL设计 (ETL Design)

## 触发条件
- 数据抽取
- 数据转换
- 数据加载
- 数据质量

## 输入
- 源系统
- 目标存储
- 转换规则
- 调度需求

## 输出
- ETL架构
- 抽取策略
- 转换规则
- 调度设计

## Prompt模板
```
请设计ETL流程。

背景：
- 源系统：[源]
- 目标存储：[目标]
- 转换规则：[规则]
- 调度需求：[需求]

请输出：
1. ETL架构
2. 抽取策略
3. 转换规则
4. 加载策略
5. 调度设计
```

## 示例
### 示例1
- 输入：CRM到数据仓库
- 输出：ETL设计，含全量增量抽取、清洗转换、数据校验、每日调度

## 注意事项
- 要增量优先
- 要数据校验
- 要失败重试
- 要血缘追踪
