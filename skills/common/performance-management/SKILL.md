# 绩效管理 (Performance Management)

## 触发条件
- 组织绩效设计
- 绩效指标
- 绩效评估
- 绩效改进

## 输入
- 战略目标
- 组织结构
- 现有考核
- 管理模式

## 输出
- 绩效体系
- 指标设计
- 评估机制
- 改进闭环

## Prompt模板
```
请为[组织/团队]设计绩效管理体系。

背景：
- 战略目标：[目标]
- 组织结构：[结构]
- 现有考核：[现状]
- 管理模式：[模式]

请输出：
1. 绩效管理原则
2. 目标体系设计
3. 指标设计（KPI/OKR）
4. 评估流程
5. 沟通反馈机制
6. 结果应用
7. 绩效改进
8. 系统支撑
```

## 示例
### 示例1
- 输入：集团公司，需要统一各子公司绩效管理
- 输出：绩效方案，含集团-子公司指标分解、差异化考核

## 注意事项
- 要战略对齐
- 要简单有效
- 要持续沟通
