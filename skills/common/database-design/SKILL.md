# 数据库设计 (Database Design)

## 触发条件
- 数据建模
- 性能优化
- 扩展性
- 数据一致性

## 输入
- 业务需求
- 数据量
- 性能要求
- 一致性要求

## 输出
- 数据模型
- 表结构
- 索引设计
- 分库分表

## Prompt模板
```
请设计数据库。

背景：
- 业务需求：[需求]
- 数据量：[数据量]
- 性能要求：[性能]
- 一致性要求：[一致性]

请输出：
1. 数据模型
2. 表结构
3. 索引设计
4. 分库分表
5. 备份策略
```

## 示例
### 示例1
- 输入：业务数据库
- 输出：数据库设计，ER图、表设计、索引优化

## 注意事项
- 要规范设计
- 要性能优先
- 要可扩展
- 要数据安全
