# 需求生成 (Demand Generation)

## 触发条件
- 获客增长
- 市场教育
- 品牌认知
- 销售赋能

## 输入
- 目标用户
- 产品价值
- 销售渠道
- 预算资源

## 输出
- 需求创造策略
- 内容策略
- 渠道组合
- 效果衡量

## Prompt模板
```
请制定需求生成策略。

背景：
- 目标用户：[用户]
- 产品价值：[价值]
- 销售渠道：[渠道]
- 预算资源：[预算]

请输出：
1. 需求洞察
2. 需求创造策略
3. 内容规划
4. 渠道组合
5. 效果衡量指标
```

## 示例
### 示例1
- 输入：B2B企业软件
- 输出：需求生成策略，含行业白皮书、在线研讨会、客户案例、ROI计算器

## 注意事项
- 要创造真实价值
- 要考虑用户旅程
- 要多触点协同
- 要持续优化
