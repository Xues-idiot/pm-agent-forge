# 企业合规 (Corporate Compliance)

## 触发条件
- 合规管理
- 法规遵循
- 内部控制
- 风险防范

## 输入
- 法规要求
- 业务流程
- 风险评估
- 现有控制

## 输出
- 合规框架
- 控制措施
- 培训计划
- 监控机制

## Prompt模板
```
请管理企业合规。

背景：
- 法规要求：[要求]
- 业务流程：[流程]
- 风险评估：[评估]
- 现有控制：[控制]

请输出：
1. 合规框架
2. 控制措施
3. 培训计划
4. 监控机制
5. 审计计划
```

## 示例
### 示例1
- 输入：上市公司
- 输出：企业合规，SOX合规、反腐败、数据保护

## 注意事项
- 要高层承诺
- 要全员参与
- 要持续监控
- 要及时改进
