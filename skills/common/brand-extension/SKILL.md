# 品牌延伸 (Brand Extension)

## 触发条件
- 品牌资产
- 产品延伸
- 市场拓展
- 风险控制

## 输入
- 品牌价值
- 延伸产品
- 品牌适配
- 市场接受

## 输出
- 延伸策略
- 产品选择
- 品牌适配
- 风险控制

## Prompt模板
```
请制定品牌延伸策略。

背景：
- 品牌价值：[价值]
- 延伸产品：[产品]
- 品牌适配：[适配]
- 市场接受：[接受]

请输出：
1. 延伸策略
2. 产品选择
3. 品牌适配
4. 市场测试
5. 风险控制
```

## 示例
### 示例1
- 输入：运动品牌
- 输出：品牌延伸，运动鞋→运动服饰→运动器材

## 注意事项
- 要品牌契合
- 要品质保证
- 要市场测试
- 要逐步推进
