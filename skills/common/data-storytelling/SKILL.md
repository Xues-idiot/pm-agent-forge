# 数据讲故事 (Data Storytelling)

## 触发条件
- 数据沟通
- 报告呈现
- 决策支持
- 说服力

## 输入
- 数据内容
- 受众特点
- 沟通目的
- 呈现场景

## 输出
- 故事框架
- 数据叙事
- 可视化建议
- 呈现技巧

## Prompt模板
```
请用数据讲故事。

背景：
- 数据内容：[数据]
- 受众特点：[受众]
- 沟通目的：[目的]
- 呈现场景：[场景]

请输出：
1. 核心信息
2. 故事框架
3. 数据呈现
4. 可视化建议
5. 呈现技巧
```

## 示例
### 示例1
- 输入：向CEO汇报季度业绩
- 输出：数据叙事方案，含关键洞察、故事线、数据支撑

## 注意事项
- 要受众导向
- 要逻辑清晰
- 要可视化辅助
