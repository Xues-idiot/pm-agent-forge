# 云架构 (Cloud Architecture)

## 触发条件
- 云迁移
- 架构设计
- 成本优化
- 性能要求

## 输入
- 业务需求
- 现有架构
- 云服务
- 成本约束

## 输出
- 架构设计
- 服务选型
- 网络设计
- 成本估算

## Prompt模板
```
请设计云架构。

背景：
- 业务需求：[需求]
- 现有架构：[架构]
- 云服务：[服务]
- 成本约束：[约束]

请输出：
1. 架构设计
2. 服务选型
3. 网络设计
4. 成本估算
5. 监控设计
```

## 示例
### 示例1
- 输入：Web应用上云
- 输出：云架构，ECS/RDS/S3/VPC，高可用设计

## 注意事项
- 要高可用
- 要成本优化
- 要安全设计
- 要可扩展
