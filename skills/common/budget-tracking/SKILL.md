# 预算跟踪 (Budget Tracking)

## 触发条件
- 成本控制
- 预算执行
- 偏差分析
- 决策支持

## 输入
- 预算计划
- 实际支出
- 预算科目
- 审批流程

## 输出
- 预算执行
- 偏差分析
- 预警机制
- 调整建议

## Prompt模板
```
请跟踪预算。

背景：
- 预算计划：[计划]
- 实际支出：[支出]
- 预算科目：[科目]
- 审批流程：[流程]

请输出：
1. 预算执行
2. 偏差分析
3. 预警机制
4. 调整建议
5. 审批建议
```

## 示例
### 示例1
- 输入：项目预算
- 输出：预算跟踪，执行率80%、偏差20%、超支预警、控制建议

## 注意事项
- 要及时准确
- 要定期跟踪
- 要预警机制
- 要审批控制
