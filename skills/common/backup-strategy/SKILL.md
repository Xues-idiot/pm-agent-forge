# 备份策略 (Backup Strategy)

## 触发条件
- 数据保护
- 灾难恢复
- 合规要求
- 成本控制

## 输入
- 数据类型
- 恢复需求
- 备份频率
- 保留期限

## 输出
- 备份方案
- 备份类型
- 存储位置
- 恢复测试

## Prompt模板
```
请制定备份策略。

背景：
- 数据类型：[类型]
- 恢复需求：[需求]
- 备份频率：[频率]
- 保留期限：[期限]

请输出：
1. 备份方案
2. 备份类型
3. 存储位置
4. 恢复测试
5. 成本评估
```

## 示例
### 示例1
- 输入：企业数据
- 输出：备份策略，全备/增量/差异、异地复制、定期测试

## 注意事项
- 要3-2-1原则
- 要定期测试
- 要加密保护
- 要成本优化
