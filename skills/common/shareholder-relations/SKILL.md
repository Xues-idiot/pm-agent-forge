# 股东关系 (Shareholder Relations)

## 触发条件
- 股东沟通
- 信息披露
- 投资者关系
- 股东大会

## 输入
- 股东结构
- 沟通需求
- 披露信息
- 活动计划

## 输出
- 沟通策略
- 披露计划
- 活动组织
- 反馈机制

## Prompt模板
```
请管理股东关系。

背景：
- 股东结构：[结构]
- 沟通需求：[需求]
- 披露信息：[信息]
- 活动计划：[计划]

请输出：
1. 沟通策略
2. 披露计划
3. 活动组织
4. 反馈机制
5. 危机应对
```

## 示例
### 示例1
- 输入：投资者关系
- 输出：股东关系，业绩发布、路演、股东大会

## 注意事项
- 要透明公正
- 要及时准确
- 要平等对待
- 要长期维护
