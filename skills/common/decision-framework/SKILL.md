# 决策框架 (Decision Framework)

## 触发条件
- 决策流程
- 决策标准
- 授权体系
- 决策质量

## 输入
- 决策类型
- 决策权限
- 信息质量
- 时间约束

## 输出
- 决策流程
- 决策标准
- 授权体系
- 质量评估

## Prompt模板
```
请设计决策框架。

背景：
- 决策类型：[类型]
- 决策权限：[权限]
- 信息质量：[质量]
- 时间约束：[约束]

请输出：
1. 决策分类
2. 决策流程
3. 决策标准
4. 授权体系
5. 决策质量
```

## 示例
### 示例1
- 输入：创业公司，需要明确决策权限
- 输出：决策框架，含分类分级、RACI、决策会

## 注意事项
- 要权责对等
- 要信息充分
- 要快速执行
