# 规模化 (Scaling Up)

## 触发条件
- 业务增长
- 组织扩张
- 流程标准化
- 效率保障

## 输入
- 增长目标
- 现有能力
- 市场机会
- 资源需求

## 输出
- 规模化路径
- 组织设计
- 流程优化
- 风险控制

## Prompt模板
```
请实现规模化增长。

背景：
- 增长目标：[目标]
- 现有能力：[能力]
- 市场机会：[机会]
- 资源需求：[需求]

请输出：
1. 规模化路径
2. 组织扩张
3. 流程标准化
4. 系统支持
5. 风险控制
```

## 示例
### 示例1
- 输入：创业公司扩张
- 输出：规模化，团队扩充、管理层级、流程标准化

## 注意事项
- 要有序扩张
- 要人才先行
- 要流程优化
- 要文化建设
