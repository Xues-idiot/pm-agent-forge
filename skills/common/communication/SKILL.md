# 沟通技巧 (Communication Skills)

## 触发条件
- 沟通策略设计
- 跨部门沟通
- 向上沟通
- 冲突沟通

## 输入
- 沟通对象
- 沟通目的
- 背景信息
- 关键信息

## 输出
- 沟通策略
- 信息结构
- 话术模板
- 注意事项

## Prompt模板
```
请为[沟通场景]设计沟通策略。

背景：
- 沟通对象：[对象]
- 沟通目的：[目的]
- 背景信息：[背景]
- 关键信息：[信息]

请输出：
1. 沟通目标定义
2. 对象分析（需求/顾虑/偏好）
3. 沟通策略选择
4. 信息结构设计（总-分-总）
5. 开场和切入方式
6. 关键信息传递
7. 可能的问题和回答
8. 后续跟进策略
```

## 示例
### 示例1
- 输入：向CEO汇报项目风险，需要争取资源支持
- 输出：沟通策略，含数据准备、结论先行、资源请求

## 注意事项
- 要了解对方
- 要逻辑清晰
- 要控制情绪
