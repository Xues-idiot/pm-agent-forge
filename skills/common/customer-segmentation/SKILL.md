# 客户细分 (Customer Segmentation)

## 触发条件
- 市场细分
- 客户分析
- 精准营销
- 差异化服务

## 输入
- 客户数据
- 行为特征
- 需求偏好
- 价值分层

## 输出
- 细分方案
- 客户画像
- 价值评估
- 策略建议

## Prompt模板
```
请进行客户细分。

背景：
- 客户数据：[数据]
- 行为特征：[特征]
- 需求偏好：[偏好]
- 价值分层：[分层]

请输出：
1. 细分方案
2. 客户画像
3. 价值评估
4. 策略建议
5. 优先级
```

## 示例
### 示例1
- 输入：零售客户
- 输出：客户细分，高价值、成长型、流失风险型

## 注意事项
- 要数据驱动
- 要动态更新
- 要可执行性
- 要差异化
