# 薪酬设计 (Compensation Design)

## 触发条件
- 薪酬策略
- 薪酬结构
- 激励体系
- 公平性

## 输入
- 薪酬理念
- 市场数据
- 岗位价值
- 成本预算

## 输出
- 薪酬策略
- 薪酬结构
- 激励方案
- 调整机制

## Prompt模板
```
请设计薪酬体系。

背景：
- 薪酬理念：[理念]
- 市场数据：[数据]
- 岗位价值：[价值]
- 成本预算：[预算]

请输出：
1. 薪酬策略
2. 薪酬结构
3. 绩效奖金方案
4. 福利体系
5. 调整机制
```

## 示例
### 示例1
- 输入：成长期科技公司
- 输出：薪酬设计，含基本工资70%+绩效20%+股权10%，市场75分位

## 注意事项
- 要外部竞争性
- 要内部公平性
- 要激励有效性
- 要成本可控性
