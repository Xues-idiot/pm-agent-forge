# 市场规模 (Market Sizing)

## 触发条件
- 市场评估
- 投资决策
- 机会识别
- 战略规划

## 输入
- 市场定义
- 数据来源
- 假设条件
- 分析方法

## 输出
- TAM总量
- SAM可服务
- SOM目标
- 增长预测

## Prompt模板
```
请评估市场规模。

背景：
- 市场定义：[定义]
- 数据来源：[来源]
- 假设条件：[假设]
- 分析方法：[方法]

请输出：
1. TAM总量
2. SAM可服务
3. SOM目标
4. 增长预测
5. 关键假设
```

## 示例
### 示例1
- 输入：中国咖啡市场
- 输出：市场规模，TAM 2000亿、SAM 800亿、SOM 100亿

## 注意事项
- 要明确定义
- 要数据支撑
- 要假设合理
- 要敏感性分析
