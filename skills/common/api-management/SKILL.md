# API管理 (API Management)

## 触发条件
- API设计
- 生命周期
- 安全管控
- 开发者生态

## 输入
- API清单
- 使用场景
- 安全要求
- 监控需求

## 输出
- API策略
- 安全设计
- 生命周期
- 监控机制

## Prompt模板
```
请管理API。

背景：
- API清单：[清单]
- 使用场景：[场景]
- 安全要求：[要求]
- 监控需求：[监控]

请输出：
1. API策略
2. 安全设计
3. 生命周期
4. 监控机制
5. 开发者门户
```

## 示例
### 示例1
- 输入：开放平台
- 输出：API管理，API网关、认证授权、流量控制

## 注意事项
- 要安全认证
- 要版本管理
- 要监控告警
- 要文档完善
