# GDPR合规 (GDPR Compliance)

## 触发条件
- 欧盟用户
- 数据跨境
- 合规审计
- 违规风险

## 输入
- 数据处理
- 法律基础
- 数据主体
- 跨境传输

## 输出
- 合规评估
- 措施清单
- 文档要求
- 违规处罚

## Prompt模板
```
请确保GDPR合规。

背景：
- 数据处理：[处理]
- 法律基础：[基础]
- 数据主体：[主体]
- 跨境传输：[传输]

请输出：
1. 合规评估
2. 措施清单
3. 文档要求
4. 违规处罚
5. 响应机制
```

## 示例
### 示例1
- 输入：欧盟业务
- 输出：GDPR合规，数据保护官、处理记录、影响评估

## 注意事项
- 要法律依据
- 要处理记录
- 要数据保护
- 要违规通知
