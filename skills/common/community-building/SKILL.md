# 社区建设 (Community Building)

## 触发条件
- 用户社区
- 活跃度提升
- 品牌忠诚
- UGC运营

## 输入
- 社区定位
- 目标用户
- 运营资源
- 商业模式

## 输出
- 社区策略
- 运营机制
- 内容规划
- 增长路径

## Prompt模板
```
请制定社区建设策略。

背景：
- 社区定位：[定位]
- 目标用户：[用户]
- 运营资源：[资源]
- 商业模式：[模式]

请输出：
1. 社区定位与价值
2. 运营机制设计
3. 内容规划
4. 增长路径
5. 商业变现
```

## 示例
### 示例1
- 输入：设计师工具产品
- 输出：社区策略，含教程分享、模板市场、设计师大赛、认证体系

## 注意事项
- 要明确价值主张
- 要培养核心用户
- 要持续运营
- 要平衡内容和商业
