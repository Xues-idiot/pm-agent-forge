# 增长飞轮 (Growth Loop)

## 触发条件
- 增长策略
- 复利增长
- 用户推荐
- 持续增长

## 输入
- 增长现状
- 用户行为
- 产品特点
- 激励机制

## 输出
- 飞轮设计
- 增长机制
- 激励机制
- 优化方向

## Prompt模板
```
请设计增长飞轮。

背景：
- 增长现状：[现状]
- 用户行为：[行为]
- 产品特点：[特点]
- 激励机制：[激励]

请输出：
1. 飞轮设计
2. 增长机制
3. 触发点
4. 激励机制
5. 优化方向
```

## 示例
### 示例1
- 输入：社交产品
- 输出：增长飞轮，更多用户→更多内容→更好体验→更多用户

## 注意事项
- 要自我强化
- 要复利效应
- 要持续优化
- 要用户体验
