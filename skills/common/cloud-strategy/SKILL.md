# 云战略 (Cloud Strategy)

## 触发条件
- 云迁移
- 云架构
- 成本优化
- 多云管理

## 输入
- 现有架构
- 迁移需求
- 成本目标
- 运维能力

## 输出
- 云战略
- 迁移规划
- 架构设计
- 成本模型

## Prompt模板
```
请制定云战略。

背景：
- 现有架构：[架构]
- 迁移需求：[需求]
- 成本目标：[目标]
- 运维能力：[能力]

请输出：
1. 云战略目标
2. 迁移策略
3. 架构设计
4. 成本模型
5. 风险评估
```

## 示例
### 示例1
- 输入：传统企业IT上云
- 输出：云战略，含渐进式迁移、云原生架构、FinOps成本管理

## 注意事项
- 要业务驱动
- 要成本优化
- 要安全合规
- 要持续治理
