# 团队效能 (Team Effectiveness)

## 触发条件
- 团队绩效
- 协作效率
- 冲突解决
- 目标达成

## 输入
- 团队构成
- 目标设定
- 协作模式
- 资源支持

## 输出
- 效能诊断
- 改进方案
- 协作机制
- 评估指标

## Prompt模板
```
请提升团队效能。

背景：
- 团队构成：[构成]
- 目标设定：[目标]
- 协作模式：[模式]
- 资源支持：[支持]

请输出：
1. 效能诊断
2. 改进方案
3. 协作机制
4. 评估指标
5. 责任分工
```

## 示例
### 示例1
- 输入：跨职能团队
- 输出：团队效能，沟通机制、决策流程、责任矩阵

## 注意事项
- 要目标清晰
- 要角色明确
- 要沟通顺畅
- 要持续改进
