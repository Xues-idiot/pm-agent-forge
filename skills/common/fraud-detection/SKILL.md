# 欺诈检测 (Fraud Detection)

## 触发条件
- 欺诈识别
- 异常交易
- 风险预警
- 调查处置

## 输入
- 交易数据
- 行为模式
- 风险指标
- 历史案例

## 输出
- 检测规则
- 异常预警
- 调查流程
- 防控建议

## Prompt模板
```
请检测欺诈。

背景：
- 交易数据：[数据]
- 行为模式：[模式]
- 风险指标：[指标]
- 历史案例：[案例]

请输出：
1. 检测规则
2. 异常预警
3. 调查流程
4. 防控建议
5. 监控指标
```

## 示例
### 示例1
- 输入：金融交易
- 输出：欺诈检测，规则引擎、机器学习、实时预警

## 注意事项
- 要及时检测
- 要减少误报
- 要调查闭环
- 要持续优化
