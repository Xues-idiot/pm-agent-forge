# 可持续发展报告 (Sustainability Reporting)

## 触发条件
- ESG披露
-  sustainability
- 合规要求
- 利益相关方

## 输入
- ESG指标
- 数据收集
- 报告框架
- 披露要求

## 输出
- 报告框架
- 指标体系
- 数据质量
- 披露策略

## Prompt模板
```
请编制可持续发展报告。

背景：
- ESG指标：[指标]
- 数据收集：[收集]
- 报告框架：[框架]
- 披露要求：[要求]

请输出：
1. 报告框架
2. 指标体系
3. 数据质量
4. 披露策略
5. 时间安排
```

## 示例
### 示例1
- 输入：ESG报告
- 输出：可持续发展报告，GRI标准、ESG评级、投资者披露

## 注意事项
- 要数据准确
- 要框架合规
- 要利益相关
- 要持续改进
