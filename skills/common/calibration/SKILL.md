# 绩效校准 (Calibration)

## 触发条件
- 绩效评估
- 评级标准
- 公平公正
- 团队平衡

## 输入
- 绩效数据
- 评级分布
- 评估标准
- 团队情况

## 输出
- 校准会议
- 评级结果
- 标准对齐
- 改进建议

## Prompt模板
```
请进行绩效校准。

背景：
- 绩效数据：[数据]
- 评级分布：[分布]
- 评估标准：[标准]
- 团队情况：[团队]

请输出：
1. 校准准备
2. 校准会议
3. 评级结果
4. 标准对齐
5. 反馈沟通
```

## 示例
### 示例1
- 输入：年度绩效评估
- 输出：绩效校准，强制分布、跨部门对齐、案例讨论

## 注意事项
- 要客观公正
- 要标准统一
- 要案例支撑
- 要及时反馈
