# 数据仓库 (Data Warehouse)

## 触发条件
- 数据分析
- 历史数据
- 报表生成
- 数据治理

## 输入
- 数据来源
- 分析需求
- 数据规模
- 查询模式

## 输出
- 仓库架构
- 模型设计
- 分层策略
- 优化方案

## Prompt模板
```
请设计数据仓库。

背景：
- 数据来源：[来源]
- 分析需求：[需求]
- 数据规模：[规模]
- 查询模式：[模式]

请输出：
1. 仓库架构
2. 数据模型
3. 分层设计
4. 调度策略
5. 优化方案
```

## 示例
### 示例1
- 输入：电商数据分析
- 输出：数据仓库，含ODS-DWD-DWS-ADS四层、Kimball维度建模

## 注意事项
- 要分层清晰
- 要模型规范
- 要ETL自动化
- 要查询性能
