# 软件许可 (Software License)

## 触发条件
- 许可证管理
- 合规审计
- 成本优化
- 使用监控

## 输入
- 软件类型
- 许可证数量
- 使用情况
- 续费成本

## 输出
- 许可证清单
- 合规状态
- 优化建议
- 采购计划

## Prompt模板
```
请管理软件许可。

背景：
- 软件类型：[类型]
- 许可证数量：[数量]
- 使用情况：[使用]
- 续费成本：[成本]

请输出：
1. 许可证清单
2. 合规状态
3. 优化建议
4. 采购计划
5. 风险提示
```

## 示例
### 示例1
- 输入：企业软件
- 输出：软件许可，正版授权、授权数量、续费管理

## 注意事项
- 要合规使用
- 要充分利用
- 要成本控制
- 要提前续费
