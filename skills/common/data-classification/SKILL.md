# 数据分类 (Data Classification)

## 触发条件
- 数据治理
- 敏感数据
- 保护级别
- 合规要求

## 输入
- 数据类型
- 敏感程度
- 业务价值
- 法规要求

## 输出
- 分类体系
- 标记规范
- 保护策略
- 处理流程

## Prompt模板
```
请制定数据分类标准。

背景：
- 数据类型：[类型]
- 敏感程度：[程度]
- 业务价值：[价值]
- 法规要求：[要求]

请输出：
1. 分类维度
2. 分类级别
3. 标记规范
4. 保护策略
5. 处理流程
```

## 示例
### 示例1
- 输入：金融机构
- 输出：数据分类，含公开、内部、机密、绝密四级及对应保护措施

## 注意事项
- 要明确标准
- 要易于执行
- 要与技术结合
- 要持续更新
