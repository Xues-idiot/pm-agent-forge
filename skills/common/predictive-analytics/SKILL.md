# 预测分析 (Predictive Analytics)

## 触发条件
- 趋势预测
- 需求预测
- 风险评估
- 决策优化

## 输入
- 预测目标
- 历史数据
- 影响因素
- 精度要求

## 输出
- 预测模型
- 预测结果
- 置信区间
- 应用建议

## Prompt模板
```
请进行预测分析。

背景：
- 预测目标：[目标]
- 历史数据：[数据]
- 影响因素：[因素]
- 精度要求：[要求]

请输出：
1. 预测模型
2. 预测结果
3. 置信区间
4. 模型评估
5. 应用建议
```

## 示例
### 示例1
- 输入：产品销量预测
- 输出：预测报告，含模型、结果、置信区间、业务建议

## 注意事项
- 要数据质量
- 要模型可解释
- 要持续验证
