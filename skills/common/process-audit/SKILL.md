# 过程审计 (Process Audit)

## 触发条件
- 流程检查
- 合规验证
- 效率评估
- 改进发现

## 输入
- 审计范围
- 流程文件
- 执行证据
- 历史问题

## 输出
- 审计发现
- 问题分类
- 整改建议
- 验证计划

## Prompt模板
```
请进行过程审计。

背景：
- 审计范围：[范围]
- 流程文件：[文件]
- 执行证据：[证据]
- 历史问题：[历史]

请输出：
1. 审计发现
2. 问题分类
3. 整改建议
4. 验证计划
5. 审计报告
```

## 示例
### 示例1
- 输入：流程合规
- 输出：过程审计，文件审核、现场验证、访谈确认

## 注意事项
- 要客观公正
- 要证据充分
- 要问题溯源
- 要整改跟踪
