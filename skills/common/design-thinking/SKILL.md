# 设计思维 (Design Thinking)

## 触发条件
- 产品创新
- 问题定义
- 用户研究
- 原型测试

## 输入
- 问题领域
- 用户群体
- 创新目标
- 资源情况

## 输出
- 用户洞察
- 解决方案
- 原型设计
- 测试反馈

## Prompt模板
```
请运用设计思维解决[问题]。

背景：
- 问题领域：[领域]
- 用户群体：[群体]
- 创新目标：[目标]
- 资源情况：[资源]

请用设计思维输出：
1. 共情理解（用户研究）
2. 问题定义（POV）
3.  ideation（创意发散）
4. 原型设计
5. 测试验证
6. 迭代优化
```

## 示例
### 示例1
- 输入：老年用户使用智能手机困难
- 输出：解决方案，含用户研究、痛点定义、简化界面原型

## 注意事项
- 要同理心
- 要快速原型
- 要用户验证
