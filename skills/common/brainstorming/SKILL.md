# 头脑风暴 (Brainstorming)

## 触发条件
- 创意生成
- 问题解决
- 团队创意
- 发散思维

## 输入
- 讨论主题
- 参与者
- 时间限制
- 创意方向

## 输出
- 创意发散
- 创意分类
- 创意评估
- 精选创意

## Prompt模板
```
请组织头脑风暴。

背景：
- 讨论主题：[主题]
- 参与者：[参与者]
- 时间限制：[时间]
- 创意方向：[方向]

请输出：
1. 规则说明
2. 创意发散
3. 创意分类
4. 创意评估
5. 精选结果
```

## 示例
### 示例1
- 输入：产品创新
- 输出：头脑风暴，疯狂想法、限制打破、可行性评估

## 注意事项
- 要延迟评判
- 要量在质先
- 要异想天开
- 要结合完善
