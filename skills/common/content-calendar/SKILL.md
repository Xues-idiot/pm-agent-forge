# 内容日历 (Content Calendar)

## 触发条件
- 内容规划
- 发布计划
- 团队协作
- 资源协调

## 输入
- 内容目标
- 内容类型
- 发布渠道
- 资源能力

## 输出
- 内容主题
- 发布计划
- 责任人
- 完成标准

## Prompt模板
```
请规划内容日历。

背景：
- 内容目标：[目标]
- 内容类型：[类型]
- 发布渠道：[渠道]
- 资源能力：[资源]

请输出：
1. 内容主题
2. 发布计划
3. 责任人
4. 完成标准
5. 效果追踪
```

## 示例
### 示例1
- 输入：内容营销
- 输出：内容日历，月度主题、周计划、节日热点

## 注意事项
- 要提前规划
- 要灵活调整
- 要团队协作
- 要效果导向
