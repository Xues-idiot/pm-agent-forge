# 事件响应计划 (Incident Response Plan)

## 触发条件
- 安全事件
- 应急响应
- 事件分类
- 处置流程

## 输入
- 事件类型
- 影响范围
- 响应资源
- 升级机制

## 输出
- 响应流程
- 角色职责
- 沟通计划
- 恢复步骤

## Prompt模板
```
请制定事件响应计划。

背景：
- 事件类型：[类型]
- 影响范围：[范围]
- 响应资源：[资源]
- 升级机制：[机制]

请输出：
1. 响应流程
2. 角色职责
3. 沟通计划
4. 恢复步骤
5. 事后复盘
```

## 示例
### 示例1
- 输入：安全事件
- 输出：事件响应，检测-遏制-根除-恢复-跟踪

## 注意事项
- 要快速响应
- 要职责清晰
- 要沟通顺畅
- 要持续改进
