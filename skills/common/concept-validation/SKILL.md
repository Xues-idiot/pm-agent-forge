# 概念验证 (Concept Validation)

## 触发条件
- 概念测试
- 市场验证
- 投资决策
- 风险识别

## 输入
- 概念描述
- 目标用户
- 验证方法
- 成功标准

## 输出
- 验证方案
- 用户反馈
- 市场反应
- 决策建议

## Prompt模板
```
请验证概念。

背景：
- 概念描述：[描述]
- 目标用户：[用户]
- 验证方法：[方法]
- 成功标准：[标准]

请输出：
1. 验证方案
2. 用户反馈
3. 市场反应
4. 决策建议
5. 风险提示
```

## 示例
### 示例1
- 输入：新功能概念
- 输出：概念验证，MVP测试、用户反馈、迭代方向

## 注意事项
- 要小步快跑
- 要用户验证
- 要数据支撑
- 要灵活调整
