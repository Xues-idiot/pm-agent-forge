# 主数据管理 (Master Data Management)

## 触发条件
- 主数据
- 数据整合
- 单一事实
- 数据共享

## 输入
- 主数据实体
- 数据源
- 共享需求
- 质量状况

## 输出
- 数据模型
- 整合策略
- 质量管理
- 共享机制

## Prompt模板
```
请管理主数据。

背景：
- 主数据实体：[实体]
- 数据源：[源]
- 共享需求：[需求]
- 质量状况：[质量]

请输出：
1. 数据模型
2. 整合策略
3. 质量管理
4. 共享机制
5. 维护流程
```

## 示例
### 示例1
- 输入：客户主数据
- 输出：主数据管理，客户档案、合并规则、共享服务

## 注意事项
- 要数据准确
- 要及时更新
- 要共享复用
- 要治理清晰
