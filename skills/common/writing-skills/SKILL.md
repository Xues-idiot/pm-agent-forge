# 写作技巧 (Writing Skills)

## 触发条件
- 商务写作
- 文档撰写
- 表达清晰
- 专业沟通

## 输入
- 写作目的
- 读者对象
- 内容类型
- 格式要求

## 输出
- 写作结构
- 内容组织
- 语言风格
- 格式规范

## Prompt模板
```
请提升写作技巧。

背景：
- 写作目的：[目的]
- 读者对象：[读者]
- 内容类型：[类型]
- 格式要求：[格式]

请输出：
1. 写作结构
2. 内容组织
3. 语言风格
4. 格式规范
5. 检查清单
```

## 示例
### 示例1
- 输入：商务邮件
- 输出：写作技巧，结论先行、要点清晰、专业简洁

## 注意事项
- 要目的明确
- 要读者导向
- 要结构清晰
- 要简洁专业
