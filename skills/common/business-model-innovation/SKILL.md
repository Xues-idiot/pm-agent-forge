# 商业模式创新 (Business Model Innovation)

## 触发条件
- 模式创新
- 价值创造
- 利润模式
- 可持续性

## 输入
- 现有模式
- 客户需求
- 技术变革
- 竞争威胁

## 输出
- 模式重构
- 价值主张
- 收入模式
- 创新路径

## Prompt模板
```
请创新商业模式。

背景：
- 现有模式：[模式]
- 客户需求：[需求]
- 技术变革：[变革]
- 竞争威胁：[威胁]

请输出：
1. 模式重构
2. 价值主张
3. 收入模式
4. 创新路径
5. 风险评估
```

## 示例
### 示例1
- 输入：传统制造
- 输出：商业模式创新，订阅服务、增值服务、生态平台

## 注意事项
- 要客户价值
- 要盈利可持续
- 要能力匹配
- 要渐进迭代
