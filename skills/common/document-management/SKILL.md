# 文档管理 (Document Management)

## 触发条件
- 文档组织
- 版本控制
- 协作共享
- 合规存档

## 输入
- 文档类型
- 文档结构
- 访问权限
- 存储需求

## 输出
- 文档体系
- 版本控制
- 权限管理
- 归档策略

## Prompt模板
```
请管理文档。

背景：
- 文档类型：[类型]
- 文档结构：[结构]
- 访问权限：[权限]
- 存储需求：[需求]

请输出：
1. 文档体系
2. 版本控制
3. 权限管理
4. 归档策略
5. 销毁机制
```

## 示例
### 示例1
- 输入：企业文档
- 输出：文档管理，集中存储、版本控制、权限分级

## 注意事项
- 要分类合理
- 要版本清晰
- 要权限严格
- 要合规存档
