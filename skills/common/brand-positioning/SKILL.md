# 品牌定位 (Brand Positioning)

## 触发条件
- 品牌策略
- 差异化竞争
- 市场定位
- 品牌感知

## 输入
- 品牌现状
- 竞争格局
- 目标用户
- 核心价值

## 输出
- 定位策略
- 品牌主张
- 差异化要素
- 传播策略

## Prompt模板
```
请制定品牌定位。

背景：
- 品牌现状：[现状]
- 竞争格局：[格局]
- 目标用户：[用户]
- 核心价值：[价值]

请输出：
1. 定位策略
2. 品牌主张
3. 差异化要素
4. 品牌故事
5. 传播策略
```

## 示例
### 示例1
- 输入：新茶饮品牌，主打健康
- 输出：品牌定位，"健康茶饮新选择"，天然+低糖+真茶

## 注意事项
- 要明确差异化
- 要契合用户需求
- 要可感知可体验
- 要长期一致
