# 度量体系 (Measurement System)

## 触发条件
- 指标设计
- 数据采集
- 分析模型
- 决策支持

## 输入
- 业务目标
- 现有数据
- 分析能力
- 可视化需求

## 输出
- 指标框架
- 数据定义
- 采集方案
- 可视化

## Prompt模板
```
请设计度量体系。

背景：
- 业务目标：[目标]
- 现有数据：[数据]
- 分析能力：[能力]
- 可视化需求：[需求]

请输出：
1. 度量目标
2. 指标框架
3. 指标定义
4. 数据采集
5. 分析模型
6. 可视化
```

## 示例
### 示例1
- 输入：运营团队，需要监控核心指标
- 输出：度量体系，含指标字典、仪表盘、周报

## 注意事项
- 要业务导向
- 要可操作性
- 要持续优化
