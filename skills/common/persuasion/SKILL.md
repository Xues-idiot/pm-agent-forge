# 说服技巧 (Persuasion)

## 触发条件
- 说服沟通
- 观点影响
- 行为改变
- 营销推广

## 输入
- 说服目标
- 对方诉求
- 论证材料
- 沟通方式

## 输出
- 说服策略
- 论证结构
- 表达方法
- 行动建议

## Prompt模板
```
请提升说服技巧。

背景：
- 说服目标：[目标]
- 对方诉求：[诉求]
- 论证材料：[材料]
- 沟通方式：[方式]

请输出：
1. 说服策略
2. 论证逻辑
3. 情感诉求
4. 社会认同
5. 行动引导
```

## 示例
### 示例1
- 输入：说服客户购买
- 输出：说服技巧，痛点挖掘、解决方案、案例证明、限时优惠

## 注意事项
- 要了解对方
- 要逻辑清晰
- 要情感共鸣
- 要信任建立
