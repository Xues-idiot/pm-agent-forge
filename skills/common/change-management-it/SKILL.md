# 变更管理 (Change Management)

## 触发条件
- 系统变更
- 发布管理
- 风险控制
- 稳定运营

## 输入
- 变更内容
- 影响范围
- 风险评估
- 资源准备

## 输出
- 变更计划
- 回滚方案
- 审批流程
- 发布流程

## Prompt模板
```
请管理IT变更。

背景：
- 变更内容：[内容]
- 影响范围：[范围]
- 风险评估：[风险]
- 资源准备：[准备]

请输出：
1. 变更计划
2. 风险评估
3. 回滚方案
4. 审批流程
5. 发布流程
```

## 示例
### 示例1
- 输入：核心系统升级
- 输出：变更管理，变更评审、灰度发布、回滚方案、监控告警

## 注意事项
- 要评估风险
- 要充分测试
- 要有回滚方案
- 要及时沟通
