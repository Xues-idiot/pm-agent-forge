# 用户旅程 (User Journey)

## 触发条件
- 体验优化
- 触点分析
- 流程改进
- 需求发现

## 输入
- 用户角色
- 使用场景
- 触点流程
- 情绪变化

## 输出
- 旅程地图
- 触点分析
- 情绪曲线
- 改进机会

## Prompt模板
```
请绘制用户旅程。

背景：
- 用户角色：[角色]
- 使用场景：[场景]
- 触点流程：[流程]
- 情绪变化：[情绪]

请输出：
1. 旅程地图
2. 触点分析
3. 情绪曲线
4. 改进机会
5. 优先级
```

## 示例
### 示例1
- 输入：App使用
- 输出：用户旅程，认知-下载-注册-使用-推荐各阶段

## 注意事项
- 要用户视角
- 要端到端
- 要情绪感知
- 要行动导向
