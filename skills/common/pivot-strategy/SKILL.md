# 转型策略 (Pivot Strategy)

## 触发条件
- 方向调整
- 市场变化
- 增长停滞
- 战略迭代

## 输入
- 现有策略
- 市场反馈
- 数据分析
- 团队能力

## 输出
- 转型方向
- 转型时机
- 转型风险
- 执行计划

## Prompt模板
```
请制定转型策略。

背景：
- 现有策略：[现有]
- 市场反馈：[反馈]
- 数据分析：[数据]
- 团队能力：[能力]

请输出：
1. 转型信号
2. 转型方向
3. 转型时机
4. 转型风险
5. 执行计划
```

## 示例
### 示例1
- 输入：B2C转B2B
- 输出：转型策略，目标客户转型、价值主张调整、销售模式变化

## 注意事项
- 要数据驱动
- 要果断决策
- 要团队共识
- 要快速执行
