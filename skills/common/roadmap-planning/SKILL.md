# 路线图规划 (Roadmap Planning)

## 触发条件
- 产品规划
- 迭代计划
- 资源分配
- 目标达成

## 输入
- 产品愿景
- 市场需求
- 技术约束
- 资源情况

## 输出
- 产品路线图
- 版本规划
- 里程碑
- 依赖关系

## Prompt模板
```
请规划产品路线图。

背景：
- 产品愿景：[愿景]
- 市场需求：[需求]
- 技术约束：[约束]
- 资源情况：[资源]

请输出：
1. 产品路线图
2. 版本规划
3. 里程碑
4. 依赖关系
5. 调整机制
```

## 示例
### 示例1
- 输入：年度规划
- 输出：产品路线图，Q1/Q2/Q3/Q4版本规划

## 注意事项
- 要愿景驱动
- 要市场拉动
- 要技术可行
- 要持续更新
