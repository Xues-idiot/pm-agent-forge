# 融资管理 (Capital Raising)

## 触发条件
- 融资策略
- 资金规划
- 投资人
- 交易完成

## 输入
- 融资阶段
- 资金需求
- 融资时间
- 投资人

## 输出
- 融资策略
- 材料准备
- 路演安排
- 交易结构

## Prompt模板
```
请管理融资。

背景：
- 融资阶段：[阶段]
- 资金需求：[需求]
- 融资时间：[时间]
- 投资人：[投资人]

请输出：
1. 融资策略
2. 材料准备
3. 路演安排
4. 交易结构
5. 时间规划
```

## 示例
### 示例1
- 输入：B轮融资
- 输出：融资管理，商业计划书、估值策略、投资人名单

## 注意事项
- 要准备充分
- 要估值合理
- 要选择合适
- 要未来规划
