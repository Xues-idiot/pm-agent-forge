# 废弃物管理 (Waste Management)

## 触发条件
- 固废处理
- 危废管理
- 资源回收
- 法规合规

## 输入
- 废弃物类型
- 产生量
- 处理方式
- 法规要求

## 输出
- 分类方案
- 处理流程
- 减量措施
- 合规检查

## Prompt模板
```
请管理废弃物。

背景：
- 废弃物类型：[类型]
- 产生量：[量]
- 处理方式：[方式]
- 法规要求：[法规]

请输出：
1. 分类方案
2. 处理流程
3. 减量措施
4. 合规检查
5. 应急预案
```

## 示例
### 示例1
- 输入：工业废弃物
- 输出：废弃物管理，分类收集、无害化、资源化

## 注意事项
- 要分类明确
- 要减量优先
- 要合规处置
- 要持续监测
