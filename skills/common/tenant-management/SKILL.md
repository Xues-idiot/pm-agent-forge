# 多租户管理 (Multi-tenancy Management)

## 触发条件
- SaaS架构
- 租户隔离
- 资源配额
- 计费管理

## 输入
- 租户类型
- 隔离需求
- 资源预算
- 计费模式

## 输出
- 租户架构
- 隔离策略
- 配额管理
- 计费方案

## Prompt模板
```
请管理多租户。

背景：
- 租户类型：[类型]
- 隔离需求：[需求]
- 资源预算：[预算]
- 计费模式：[模式]

请输出：
1. 租户架构
2. 隔离策略
3. 配额管理
4. 计费方案
5. SLA保障
```

## 示例
### 示例1
- 输入：企业SaaS服务
- 输出：多租户设计，含数据库隔离、租户配额、独立配置、计量计费

## 注意事项
- 要安全隔离
- 要资源公平
- 要灵活计费
- 要可扩展
