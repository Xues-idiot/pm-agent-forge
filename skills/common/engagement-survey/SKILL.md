# 敬业度调研 (Engagement Survey)

## 触发条件
- 员工敬业
- 文化评估
- 改进方向
- 跟踪趋势

## 输入
- 调研设计
- 响应数据
- 历史对比
- 基准对标

## 输出
- 敬业度分析
- 问题识别
- 改进建议
- 跟踪计划

## Prompt模板
```
请进行敬业度调研。

背景：
- 调研设计：[设计]
- 响应数据：[响应]
- 历史对比：[历史]
- 基准对标：[基准]

请输出：
1. 敬业度分析
2. 问题识别
3. 根因分析
4. 改进建议
5. 跟踪计划
```

## 示例
### 示例1
- 输入：年度敬业度调研
- 输出：敬业度分析，总体70分、对标分析、Top问题、改进计划

## 注意事项
- 要匿名保证
- 要主动行动
- 要持续跟踪
- 要管理层重视
