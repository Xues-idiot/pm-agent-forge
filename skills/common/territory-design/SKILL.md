# 区域规划 (Territory Design)

## 触发条件
- 区域划分
- 销售区域
- 服务区域
- 资源分配

## 输入
- 市场潜力
- 区域特征
- 现有资源
- 覆盖目标

## 输出
- 区域划分
- 潜力评估
- 资源配置
- 绩效目标

## Prompt模板
```
请规划区域。

背景：
- 市场潜力：[潜力]
- 区域特征：[特征]
- 现有资源：[资源]
- 覆盖目标：[目标]

请输出：
1. 区域划分
2. 潜力评估
3. 资源配置
4. 绩效目标
5. 时间计划
```

## 示例
### 示例1
- 输入：销售区域
- 输出：区域规划，按省份/城市、潜力分级、资源匹配

## 注意事项
- 要公平合理
- 要潜力匹配
- 要效率优先
- 要动态调整
