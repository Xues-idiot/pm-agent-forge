# 客户倡导 (Customer Advocacy)

## 触发条件
- 客户口碑
- 推荐营销
- 品牌大使
- 用户社群

## 输入
- 客户满意
- 客户画像
- 激励机制
- 传播渠道

## 输出
- 倡导计划
- 筛选机制
- 激励体系
- 传播策略

## Prompt模板
```
请制定客户倡导计划。

背景：
- 客户满意：[满意]
- 客户画像：[画像]
- 激励机制：[激励]
- 传播渠道：[渠道]

请输出：
1. 倡导计划
2. 筛选标准
3. 激励机制
4. 传播策略
5. 效果追踪
```

## 示例
### 示例1
- 输入：企业软件，高满意客户
- 输出：倡导计划，含MVP客户认证、案例合作、推荐奖励、口碑传播

## 注意事项
- 要选择合适客户
- 要提供真实价值
- 要自然不强迫
- 要追踪效果
