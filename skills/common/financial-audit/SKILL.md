# 财务审计 (Financial Audit)

## 触发条件
- 审计需求
- 合规检查
- 内部控制
- 风险评估

## 输入
- 财务报表
- 审计范围
- 合规要求
- 内控体系

## 输出
- 审计报告
- 问题清单
- 整改建议
- 风险评估

## Prompt模板
```
请进行财务审计。

背景：
- 财务报表：[报表]
- 审计范围：[范围]
- 合规要求：[要求]
- 内控体系：[体系]

请输出：
1. 审计报告
2. 问题清单
3. 整改建议
4. 风险评估
5. 内控建议
```

## 示例
### 示例1
- 输入：上市公司年报审计
- 输出：审计报告，无保留意见、内控完善

## 注意事项
- 要保持独立
- 要客观公正
- 要证据充分
- 要合规导向
