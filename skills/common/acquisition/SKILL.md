# 收购整合 (Acquisition Integration)

## 触发条件
- 收购决策
- 整合规划
- 价值创造
- 文化融合

## 输入
- 收购目标
- 整合范围
- 资源需求
- 风险识别

## 输出
- 整合策略
- 整合计划
- 文化融合
- 价值创造

## Prompt模板
```
请规划收购整合。

背景：
- 收购目标：[目标]
- 整合范围：[范围]
- 资源需求：[资源]
- 风险识别：[风险]

请输出：
1. 整合策略
2. 整合计划
3. 文化融合
4. 价值创造
5. 风险控制
```

## 示例
### 示例1
- 输入：科技收购
- 输出：收购整合，100天计划、产品整合、团队保留、文化融合

## 注意事项
- 要快速整合
- 要人才保留
- 要文化融合
- 要价值创造
