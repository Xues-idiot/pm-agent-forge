# 平衡计分卡 (Balanced Scorecard)

## 触发条件
- 战略管理
- 绩效评估
- 多维度
- 平衡视角

## 输入
- 财务视角
- 客户视角
- 内部流程
- 学习成长

## 输出
- 战略地图
- KPI体系
- 目标值
- 行动计划

## Prompt模板
```
请使用平衡计分卡。

背景：
- 财务目标：[财务]
- 客户目标：[客户]
- 内部流程：[流程]
- 学习成长：[学习]

请输出：
1. 战略地图
2. 四个维度KPI
3. 目标值设定
4. 行动计划
5. 评估机制
```

## 示例
### 示例1
- 输入：企业战略
- 输出：BSC设计，财务-收入增长、客户-满意度、内部-效率、学习-培训

## 注意事项
- 要战略导向
- 要四维平衡
- 要上下对齐
- 要持续改进
