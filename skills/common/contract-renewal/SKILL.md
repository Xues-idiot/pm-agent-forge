# 合同续签 (Contract Renewal)

## 触发条件
- 合同到期
- 供应商评估
- 谈判准备
- 决策分析

## 输入
- 合同历史
- 供应商表现
- 市场行情
- 替代方案

## 输出
- 续签评估
- 谈判策略
- 条款调整
- 决策建议

## Prompt模板
```
请评估合同续签。

背景：
- 合同历史：[历史]
- 供应商表现：[表现]
- 市场行情：[行情]
- 替代方案：[方案]

请输出：
1. 续签评估
2. 谈判策略
3. 条款调整
4. 决策建议
5. 时间计划
```

## 示例
### 示例1
- 输入：年度合同
- 输出：合同续签，绩效回顾、价格谈判、条款更新

## 注意事项
- 要绩效导向
- 要市场对比
- 要条款优化
- 要提前规划
