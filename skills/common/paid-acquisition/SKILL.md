# 付费获客 (Paid Acquisition)

## 触发条件
- 广告投放
- 付费流量
- 用户获取
- ROI优化

## 输入
- 广告平台
- 目标用户
- 预算分配
- 转化目标

## 输出
- 投放策略
- 创意方向
- 出价策略
- ROI预期

## Prompt模板
```
请优化付费获客。

背景：
- 广告平台：[平台]
- 目标用户：[用户]
- 预算分配：[预算]
- 转化目标：[目标]

请输出：
1. 投放策略
2. 创意方向
3. 出价策略
4. ROI预期
5. 监控指标
```

## 示例
### 示例1
- 输入：效果广告
- 输出：付费获客，Google Ads、Facebook Ads、信息流广告

## 注意事项
- 要精准定向
- 要创意优化
- 要持续测试
- 要ROI导向
