# 数据归档 (Data Archival)

## 触发条件
- 历史数据
- 存储成本
- 合规保留
- 数据访问

## 输入
- 归档数据
- 访问频率
- 保留要求
- 存储成本

## 输出
- 归档策略
- 分层存储
- 检索机制
- 销毁流程

## Prompt模板
```
请进行数据归档。

背景：
- 归档数据：[数据]
- 访问频率：[频率]
- 保留要求：[要求]
- 存储成本：[成本]

请输出：
1. 归档策略
2. 分层存储
3. 检索机制
4. 销毁流程
5. 合规检查
```

## 示例
### 示例1
- 输入：历史订单
- 输出：数据归档，热数据-冷数据-归档数据分层

## 注意事项
- 要分层存储
- 要合规保留
- 要可检索
- 要成本优化
