# 营销驱动增长 (Marketing-led Growth)

## 触发条件
- 营销策略
- 品牌建设
- 获客增长
- 营销ROI

## 输入
- 目标市场
- 产品定位
- 营销预算
- 增长目标

## 输出
- 营销策略
- 渠道组合
- 内容营销
- ROI评估

## Prompt模板
```
请实施营销驱动增长。

背景：
- 目标市场：[市场]
- 产品定位：[定位]
- 营销预算：[预算]
- 增长目标：[目标]

请输出：
1. 营销策略
2. 渠道组合
3. 内容营销
4. 获客漏斗
5. ROI评估
```

## 示例
### 示例1
- 输入：B2B营销
- 输出：MLG策略，内容营销、SEO、展会、ABM

## 注意事项
- 要品牌先行
- 要内容为王
- 要数据追踪
- 要持续优化
