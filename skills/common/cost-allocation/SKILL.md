# 成本分摊 (Cost Allocation)

## 触发条件
- 成本核算
- 部门预算
- 资源效率
- 责任归属

## 输入
- 成本项目
- 部门结构
- 资源使用
- 分摊规则

## 输出
- 分摊模型
- 账户结构
- 报告机制
- 优化流程

## Prompt模板
```
请制定成本分摊方案。

背景：
- 成本项目：[项目]
- 部门结构：[结构]
- 资源使用：[使用]
- 分摊规则：[规则]

请输出：
1. 分摊模型
2. 账户结构
3. 分摊规则
4. 报告机制
5. 优化流程
```

## 示例
### 示例1
- 输入：多产品线企业
- 输出：成本分摊，含资源标签、项目核算、部门预算、效益分析

## 注意事项
- 要公正透明
- 要简明易懂
- 要激励节约
- 要持续优化
