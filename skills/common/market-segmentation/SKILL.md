# 市场细分 (Market Segmentation)

## 触发条件
- 市场分析
- 目标客户
- 差异化策略
- 资源配置

## 输入
- 市场规模
- 客户特征
- 竞争格局
- 企业资源

## 输出
- 细分市场
- 优先级排序
- 进入策略
- 资源配置

## Prompt模板
```
请进行市场细分。

背景：
- 市场规模：[规模]
- 客户特征：[特征]
- 竞争格局：[格局]
- 企业资源：[资源]

请输出：
1. 细分维度
2. 各细分市场评估
3. 优先级排序
4. 进入策略
5. 资源配置建议
```

## 示例
### 示例1
- 输入：智能家居市场
- 输出：细分方案，含年龄代际×收入水平×住房类型×科技接受度

## 注意事项
- 要基于数据验证
- 要考虑可触及性
- 要匹配企业能力
- 要动态调整
