# 增购/交叉销售 (Upsell/Cross-sell)

## 触发条件
- 客单价提升
- 产品组合
- 精准推荐
- 价值最大化

## 输入
- 用户画像
- 购买历史
- 产品关联
- 场景时机

## 输出
- 推荐策略
- 产品组合
- 触发机制
- 效果评估

## Prompt模板
```
请设计增购交叉销售策略。

背景：
- 用户画像：[画像]
- 购买历史：[历史]
- 产品关联：[关联]
- 场景时机：[时机]

请输出：
1. 推荐目标
2. 用户分层
3. 产品推荐
4. 触发机制
5. 效果评估
```

## 示例
### 示例1
- 输入：电商平台，用户购买手机
- 输出：推荐策略，含手机壳-耳机-延保-配件组合推荐

## 注意事项
- 要精准推荐
- 要时机合适
- 要自然不打扰
