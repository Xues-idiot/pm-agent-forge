# 冲突解决 (Conflict Resolution)

## 触发条件
- 矛盾调解
- 利益冲突
- 团队协作
- 问题解决

## 输入
- 冲突类型
- 各方立场
- 利益分析
- 解决资源

## 输出
- 冲突分析
- 解决方案
- 调解策略
- 预防机制

## Prompt模板
```
请解决冲突。

背景：
- 冲突类型：[类型]
- 各方立场：[立场]
- 利益分析：[利益]
- 解决资源：[资源]

请输出：
1. 冲突分析
2. 各方利益
3. 解决方案
4. 调解策略
5. 预防机制
```

## 示例
### 示例1
- 输入：团队冲突
- 输出：冲突解决，倾听各方、找到共同利益、协商解决方案

## 注意事项
- 要及时介入
- 要中立公正
- 要聚焦利益
- 要建立机制
