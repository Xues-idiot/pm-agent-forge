# 数据目录 (Data Catalog)

## 触发条件
- 元数据管理
- 数据发现
- 数据血缘
- 数据治理

## 输入
- 数据资产
- 业务描述
- 技术属性
- 访问需求

## 输出
- 目录架构
- 元模型
- 分类体系
- 搜索机制

## Prompt模板
```
请建设数据目录。

背景：
- 数据资产：[资产]
- 业务描述：[描述]
- 技术属性：[属性]
- 访问需求：[需求]

请输出：
1. 目录架构
2. 元模型设计
3. 分类体系
4. 搜索机制
5. 治理流程
```

## 示例
### 示例1
- 输入：企业数据资产
- 输出：数据目录，含业务术语、技术元数据、血缘图谱、标签管理

## 注意事项
- 要持续更新
- 要多角色维护
- 要易于发现
- 要权限控制
