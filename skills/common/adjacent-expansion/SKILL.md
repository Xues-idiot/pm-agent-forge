# 相邻扩张 (Adjacent Expansion)

## 触发条件
- 业务延伸
- 新领域
- 能力复制
- 增长新曲线

## 输入
- 核心业务
- 相邻领域
- 能力匹配
- 市场潜力

## 输出
- 扩张策略
- 能力复制
- 市场机会
- 进入路径

## Prompt模板
```
请制定相邻扩张策略。

背景：
- 核心业务：[核心]
- 相邻领域：[领域]
- 能力匹配：[匹配]
- 市场潜力：[潜力]

请输出：
1. 扩张策略
2. 能力复制
3. 市场机会
4. 进入路径
5. 资源配置
```

## 示例
### 示例1
- 输入：电商平台
- 输出：相邻扩张，从实物电商→本地生活→旅游出行

## 注意事项
- 要能力复用
- 要客户协同
- 要资源投入
- 要快速验证
