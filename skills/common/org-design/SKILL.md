# 组织设计 (Organizational Design)

## 触发条件
- 组织架构
- 职责划分
- 汇报关系
- 效率优化

## 输入
- 战略目标
- 业务特点
- 人员现状
- 管理成本

## 输出
- 组织架构
- 职责划分
- 汇报关系
- 人员规划

## Prompt模板
```
请设计组织架构。

背景：
- 战略目标：[目标]
- 业务特点：[特点]
- 人员现状：[现状]
- 管理成本：[成本]

请输出：
1. 组织架构设计
2. 部门职责
3. 汇报关系
4. 关键岗位规划
5. 实施路径
```

## 示例
### 示例1
- 输入：创业公司从50人发展到200人
- 输出：组织设计，含职能制→事业部制过渡、授权体系、核心岗位招聘

## 注意事项
- 要匹配战略
- 要效率优先
- 要责权对等
- 要动态调整
