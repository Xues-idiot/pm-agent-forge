# 变革管理 (Change Management)

## 触发条件
- 变革规划
- 变革执行
- 变革阻力
- 变革评估

## 输入
- 变革类型
- 影响范围
- 时间要求
- 组织文化

## 输出
- 变革方案
- 沟通计划
- 阻力应对
- 评估机制

## Prompt模板
```
请为[变革项目]制定管理方案。

背景：
- 变革类型：[类型]
- 影响范围：[范围]
- 时间要求：[时间]
- 组织文化：[文化]

请输出：
1. 变革愿景和目标
2. 影响分析
3. 变革策略
4. 沟通和参与计划
5. 培训和准备
6. 阻力识别和应对
7. 早期胜利计划
8. 持续巩固
9. 效果评估
```

## 示例
### 示例1
- 输入：CRM系统上线，需要全员使用，打破原有习惯
- 输出：变革方案，含沟通策略、培训计划、激励措施

## 注意事项
- 要充分沟通
- 要关注人而非只是技术
- 要庆祝进步
