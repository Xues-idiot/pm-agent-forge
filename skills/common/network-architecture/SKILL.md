# 网络架构 (Network Architecture)

## 触发条件
- 网络设计
- 安全架构
- 性能优化
- 扩展性

## 输入
- 业务需求
- 用户规模
- 安全要求
- 成本预算

## 输出
- 架构设计
- 安全策略
- 扩展方案
- 成本评估

## Prompt模板
```
请设计网络架构。

背景：
- 业务需求：[需求]
- 用户规模：[规模]
- 安全要求：[要求]
- 成本预算：[预算]

请输出：
1. 网络架构设计
2. 安全策略
3. QoS设计
4. 扩展方案
5. 成本评估
```

## 示例
### 示例1
- 输入：企业园区网络
- 输出：网络架构，含核心-汇聚-接入三层、VLAN划分、安全域隔离

## 注意事项
- 要安全优先
- 要高可用
- 要可扩展
- 要成本合理
