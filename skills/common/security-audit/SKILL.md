# 安全审计 (Security Audit)

## 触发条件
- 安全合规
- 风险评估
- 控制验证
- 改进建议

## 输入
- 审计范围
- 合规要求
- 风险点
- 历史发现

## 输出
- 审计计划
- 发现问题
- 风险评估
- 整改建议

## Prompt模板
```
请进行安全审计。

背景：
- 审计范围：[范围]
- 合规要求：[要求]
- 风险点：[风险]
- 历史发现：[发现]

请输出：
1. 审计计划
2. 审计发现
3. 风险评估
4. 整改建议
5. 跟踪验证
```

## 示例
### 示例1
- 输入：企业安全审计
- 输出：安全审计，网络安全、数据安全、应用安全、物理安全

## 注意事项
- 要独立客观
- 要全面覆盖
- 要跟踪整改
- 要持续改进
