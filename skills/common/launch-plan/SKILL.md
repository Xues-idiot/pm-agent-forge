# 上线计划 (Launch Plan)

## 触发条件
- 产品发布
- 上线准备
- 推广计划
- 效果评估

## 输入
- 发布范围
- 目标用户
- 推广资源
- 时间节点

## 输出
- 上线计划
- 推广方案
- 监控机制
- 应急预案

## Prompt模板
```
请制定上线计划。

背景：
- 发布范围：[范围]
- 目标用户：[用户]
- 推广资源：[资源]
- 时间节点：[节点]

请输出：
1. 上线目标
2. 上线计划
3. 推广方案
4. 监控机制
5. 应急预案
```

## 示例
### 示例1
- 输入：新功能上线，需要快速推广
- 输出：上线计划，含分批上线、推广激励、监控告警

## 注意事项
- 要充分准备
- 要快速响应
- 要持续优化
