# 渠道管理 (Channel Management)

## 触发条件
- 渠道建设
- 渠道优化
- 渠道激励
- 渠道冲突

## 输入
- 渠道结构
- 渠道能力
- 激励政策
- 冲突情况

## 输出
- 渠道策略
- 渠道赋能
- 激励机制
- 冲突解决

## Prompt模板
```
请管理渠道。

背景：
- 渠道结构：[结构]
- 渠道能力：[能力]
- 激励政策：[政策]
- 冲突情况：[冲突]

请输出：
1. 渠道策略
2. 渠道赋能
3. 激励机制
4. 冲突解决
5. 评估体系
```

## 示例
### 示例1
- 输入：分销渠道
- 输出：渠道管理，渠道分级、赋能支持、激励机制

## 注意事项
- 要渠道平衡
- 要利益一致
- 要赋能提升
- 要持续优化
