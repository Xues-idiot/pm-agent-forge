# 漏斗转化 (Funnel Conversion)

## 触发条件
- 转化分析
- 流程优化
- 流失识别
- 增长黑客

## 输入
- 漏斗阶段
- 用户行为
- 转化路径
- 关键节点

## 输出
- 转化漏斗
- 流失节点
- 优化建议
- 预期提升

## Prompt模板
```
请分析漏斗转化。

背景：
- 漏斗阶段：[阶段]
- 用户行为：[行为]
- 转化路径：[路径]
- 关键节点：[节点]

请输出：
1. 转化漏斗
2. 流失节点
3. 优化建议
4. 预期提升
5. 测试方案
```

## 示例
### 示例1
- 输入：电商转化
- 输出：漏斗分析，浏览-加购-下单-支付-成交

## 注意事项
- 要全面覆盖
- 要细分维度
- 要假设验证
- 要持续优化
