# 环境合规 (Environmental Compliance)

## 触发条件
- 环境保护
- 法规要求
- 排放控制
- 可持续发展

## 输入
- 环保法规
- 排放数据
- 许可要求
- 监测记录

## 输出
- 合规评估
- 改进方案
- 监测计划
- 报告机制

## Prompt模板
```
请确保环境合规。

背景：
- 环保法规：[法规]
- 排放数据：[数据]
- 许可要求：[许可]
- 监测记录：[记录]

请输出：
1. 合规评估
2. 改进方案
3. 监测计划
4. 报告机制
5. 应急预案
```

## 示例
### 示例1
- 输入：制造业环保
- 输出：环境合规，ISO14001、废水废气排放、危废处理

## 注意事项
- 要法规合规
- 要持续监测
- 要员工意识
- 要持续改进
