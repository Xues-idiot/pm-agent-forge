# 流失分析 (Churn Analysis)

## 触发条件
- 用户流失
- 留存率下降
- 流失原因
- 预防策略

## 输入
- 流失用户特征
- 流失时间节点
- 用户行为路径
- 流失前兆信号

## 输出
- 流失原因分析
- 流失预警模型
- 预防策略
- 召回方案

## Prompt模板
```
请分析用户流失。

背景：
- 流失用户特征：[特征]
- 流失时间节点：[节点]
- 用户行为路径：[路径]
- 流失前兆信号：[信号]

请输出：
1. 流失原因分析
2. 流失预警指标
3. 预防策略
4. 召回方案
5. 效果预测
```

## 示例
### 示例1
- 输入：订阅制App，月流失率上升
- 输出：流失分析，含使用频次下降、付费功能使用降低、客服咨询增加

## 注意事项
- 要细分流失类型
- 要早发现早干预
- 要分析关键节点
- 要量化流失成本
