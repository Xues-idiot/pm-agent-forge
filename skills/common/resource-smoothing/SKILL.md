# 资源平衡 (Resource Smoothing)

## 触发条件
- 资源冲突
- 进度优化
- 能力规划
- 效率提升

## 输入
- 资源需求
- 资源供给
- 任务优先级
- 约束条件

## 输出
- 平衡方案
- 调整建议
- 优先级
- 风险应对

## Prompt模板
```
请进行资源平衡。

背景：
- 资源需求：[需求]
- 资源供给：[供给]
- 任务优先级：[优先]
- 约束条件：[约束]

请输出：
1. 资源分析
2. 冲突识别
3. 平衡方案
4. 调整建议
5. 风险应对
```

## 示例
### 示例1
- 输入：研发人力紧张
- 输出：资源平衡，含任务延期、外包调用、优先级排序、加班计划

## 注意事项
- 要全局优化
- 要沟通协调
- 要动态调整
- 要关注质量
