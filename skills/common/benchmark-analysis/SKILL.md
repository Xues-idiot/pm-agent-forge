# 基准分析 (Benchmark Analysis)

## 触发条件
- 竞争对标
- 行业标准
- 差距识别
- 改进方向

## 输入
- 对标对象
- 指标体系
- 数据来源
- 分析目标

## 输出
- 基准对比
- 差距分析
- 改进机会
- 行动计划

## Prompt模板
```
请进行基准分析。

背景：
- 对标对象：[对象]
- 指标体系：[体系]
- 数据来源：[来源]
- 分析目标：[目标]

请输出：
1. 基准对比
2. 差距分析
3. 改进机会
4. 行动计划
5. 预期效果
```

## 示例
### 示例1
- 输入：行业对标
- 输出：基准分析，行业领先、良好、平均水平对比

## 注意事项
- 要选对标杆
- 要数据准确
- 要客观分析
- 要行动计划
