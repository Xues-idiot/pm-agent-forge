# 决策方法论 (Decision Making)

## Overview

决策方法论是系统化分析和做出高质量决策的技能。涵盖问题定义、方案分析、风险评估、决策选择等核心环节。

## 何时使用

- 重要业务决策
- 战略选择
- 资源分配
- 方案对比

## 流程

1. **问题定义**：明确决策目标和约束
2. **信息收集**：收集决策所需信息
3. **方案生成**：列出可行方案
4. **方案分析**：分析各方案优劣势
5. **风险评估**：评估各方案风险
6. **决策选择**：选择最优方案
7. **执行复盘**：执行并复盘决策效果

## 检查清单

- [ ] 问题定义清晰
- [ ] 方案覆盖全面
- [ ] 分析客观公正
- [ ] 风险评估到位
- [ ] 决策有明确依据
- [ ] 有执行跟踪机制

## 红牌警告

- 问题定义模糊
- 方案考虑不全面
- 分析带有偏见
- 忽视重大风险
- 决策后不跟踪执行
