# 采购策略 (Procurement Strategy)

## 触发条件
- 采购优化
- 成本控制
- 供应链管理
- 供应商关系

## 输入
- 采购品类
- 市场需求
- 供应商市场
- 成本目标

## 输出
- 采购策略
- 供应商策略
- 成本优化
- 风险管理

## Prompt模板
```
请制定采购策略。

背景：
- 采购品类：[品类]
- 市场需求：[需求]
- 供应商市场：[市场]
- 成本目标：[目标]

请输出：
1. 采购策略
2. 供应商策略
3. 成本优化路径
4. 风险管理
5. 执行计划
```

## 示例
### 示例1
- 输入：制造企业原材料采购
- 输出：采购策略，含集中采购、供应商整合、长期协议、备选供应商

## 注意事项
- 要战略采购思维
- 要建立长期关系
- 要关注TCO
- 要合规透明
