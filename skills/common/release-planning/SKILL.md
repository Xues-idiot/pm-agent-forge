# 发布计划 (Release Planning)

## 触发条件
- 版本规划
- 功能发布
- 风险控制
- 交付节奏

## 输入
- 产品路线图
- 依赖关系
- 资源能力
- 质量目标

## 输出
- 发布版本
- 时间计划
- 功能范围
- 风险控制

## Prompt模板
```
请制定发布计划。

背景：
- 产品路线图：[路线图]
- 依赖关系：[依赖]
- 资源能力：[资源]
- 质量目标：[目标]

请输出：
1. 发布版本规划
2. 功能范围
3. 时间计划
4. 风险控制
5. 验收标准
```

## 示例
### 示例1
- 输入：产品版本规划
- 输出：发布计划，含V1.0-V2.0功能、发布时间、依赖关系、风险预案

## 注意事项
- 要滚动规划
- 要灵活调整
- 要质量优先
- 要持续集成
