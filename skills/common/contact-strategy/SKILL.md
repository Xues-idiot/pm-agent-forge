# 触达策略 (Contact Strategy)

## 触发条件
- 用户触达
- 沟通时机
- 渠道选择
- 内容策略

## 输入
- 用户特征
- 触达目的
- 可用渠道
- 合规要求

## 输出
- 触达策略
- 渠道组合
- 时机选择
- 内容规划

## Prompt模板
```
请制定触达策略。

背景：
- 用户特征：[用户]
- 触达目的：[目的]
- 可用渠道：[渠道]
- 合规要求：[要求]

请输出：
1. 触达目标
2. 渠道组合策略
3. 触达时机
4. 内容策略
5. 频率控制
```

## 示例
### 示例1
- 输入：电商大促前触达老客户
- 输出：触达策略，含短信+推送+邮件组合，节奏规划

## 注意事项
- 要精准触达
- 要频率适中
- 要合规第一
