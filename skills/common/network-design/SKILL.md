# 网络设计 (Network Design)

## 触发条件
- 网络架构
- 性能优化
- 安全设计
- 扩展规划

## 输入
- 业务需求
- 用户规模
- 带宽需求
- 安全要求

## 输出
- 网络架构
- 拓扑设计
- 设备选型
- 实施计划

## Prompt模板
```
请设计网络。

背景：
- 业务需求：[需求]
- 用户规模：[规模]
- 带宽需求：[带宽]
- 安全要求：[安全]

请输出：
1. 网络架构
2. 拓扑设计
3. 设备选型
4. 实施计划
5. 运维建议
```

## 示例
### 示例1
- 输入：企业网络
- 输出：网络设计，核心-汇聚-接入、VLAN划分、防火墙

## 注意事项
- 要高可用
- 要安全可靠
- 要可扩展
- 要成本合理
