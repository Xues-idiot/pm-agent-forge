# 渗透测试 (Penetration Testing)

## 触发条件
- 安全测试
- 漏洞发现
- 风险评估
- 合规要求

## 输入
- 测试范围
- 测试类型
- 安全要求
- 资源配合

## 输出
- 测试计划
- 测试结果
- 风险评估
- 修复建议

## Prompt模板
```
请进行渗透测试。

背景：
- 测试范围：[范围]
- 测试类型：[类型]
- 安全要求：[要求]
- 资源配合：[配合]

请输出：
1. 测试计划
2. 测试方法
3. 发现漏洞
4. 风险评估
5. 修复建议
```

## 示例
### 示例1
- 输入：Web应用
- 输出：渗透测试，含OWASP Top 10、系统性测试、漏洞利用演示

## 注意事项
- 要授权测试
- 要全面覆盖
- 要专业执行
- 要及时修复
