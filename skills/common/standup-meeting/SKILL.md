# 站会 (Standup Meeting)

## 触发条件
- 每日同步
- 进度共享
- 障碍识别
- 团队协调

## 输入
- 团队成员
- 工作进展
- 今日计划
- 障碍问题

## 输出
- 进度同步
- 障碍识别
- 协调决策
- 后续跟踪

## Prompt模板
```
请组织站会。

背景：
- 团队成员：[成员]
- 工作进展：[进展]
- 今日计划：[计划]
- 障碍问题：[障碍]

请输出：
1. 进度同步
2. 障碍识别
3. 协调决策
4. 后续跟踪
5. 责任分配
```

## 示例
### 示例1
- 输入：开发团队
- 输出：站会，昨日完成、今日计划、阻碍问题

## 注意事项
- 要简短高效
- 要站立形式
- 要聚焦障碍
- 要后续跟踪
