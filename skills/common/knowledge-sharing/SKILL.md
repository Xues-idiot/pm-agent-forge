# 知识共享 (Knowledge Sharing)

## 触发条件
- 知识管理
- 经验沉淀
- 信息透明
- 组织学习

## 输入
- 知识类型
- 共享文化
- 技术支撑
- 激励机制

## 输出
- 共享机制
- 知识库
- 分享平台
- 激励方案

## Prompt模板
```
请设计知识共享机制。

背景：
- 知识类型：[类型]
- 共享文化：[文化]
- 技术支撑：[支撑]
- 激励机制：[激励]

请输出：
1. 知识分类
2. 共享机制
3. 知识库
4. 分享平台
5. 激励机制
```

## 示例
### 示例1
- 输入：研发团队，需要沉淀技术经验
- 输出：共享机制，含技术分享会、知识库、代码评审

## 注意事项
- 要文化先行
- 要工具支撑
- 要持续运营
