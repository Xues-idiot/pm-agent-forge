# 云迁移 (Cloud Migration)

## 触发条件
- 上云规划
- 迁移策略
- 风险控制
- 成本优化

## 输入
- 迁移范围
- 现有架构
- 目标平台
- 时间计划

## 输出
- 迁移方案
- 风险评估
- 实施路径
- 成本测算

## Prompt模板
```
请规划云迁移。

背景：
- 迁移范围：[范围]
- 现有架构：[架构]
- 目标平台：[平台]
- 时间计划：[计划]

请输出：
1. 迁移方案
2. 风险评估
3. 实施路径
4. 成本测算
5. 运维规划
```

## 示例
### 示例1
- 输入：数据中心迁移
- 输出：云迁移，lift-and-shift、重构、混合云策略

## 注意事项
- 要充分评估
- 要分步实施
- 要业务连续
- 要成本优化
