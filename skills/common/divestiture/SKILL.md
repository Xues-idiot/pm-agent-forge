# 资产剥离 (Divestiture)

## 触发条件
- 业务优化
- 资源配置
- 战略聚焦
- 价值释放

## 输入
- 剥离资产
- 业务关联
- 市场环境
- 潜在买家

## 输出
- 剥离策略
- 剥离时机
- 交易结构
- 价值评估

## Prompt模板
```
请制定剥离策略。

背景：
- 剥离资产：[资产]
- 业务关联：[关联]
- 市场环境：[环境]
- 潜在买家：[买家]

请输出：
1. 剥离策略
2. 剥离时机
3. 交易结构
4. 价值评估
5. 风险控制
```

## 示例
### 示例1
- 输入：非核心业务
- 输出：资产剥离，业务分拆、单独上市或出售

## 注意事项
- 要战略聚焦
- 要时机合适
- 要估值合理
- 要有序退出
