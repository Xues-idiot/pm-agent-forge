# ITIL实施 (ITIL Implementation)

## 触发条件
- IT服务管理
- 流程标准化
- 服务质量
- 运维效率

## 输入
- 服务范围
- 成熟度现状
- 改进目标
- 资源投入

## 输出
- 实施路径
- 流程设计
- 工具平台
- 持续改进

## Prompt模板
```
请实施ITIL。

背景：
- 服务范围：[范围]
- 成熟度现状：[现状]
- 改进目标：[目标]
- 资源投入：[资源]

请输出：
1. 成熟度评估
2. 实施路径
3. 流程设计
4. 工具平台
5. 持续改进
```

## 示例
### 示例1
- 输入：IT部门
- 输出：ITIL实施，事件管理→问题管理→变更管理渐进实施

## 注意事项
- 要价值驱动
- 要渐进实施
- 要工具支持
- 要持续改进
