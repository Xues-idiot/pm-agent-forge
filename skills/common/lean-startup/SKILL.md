# 精益创业 (Lean Startup)

## 触发条件
- 创业方法
- 快速验证
- 学习迭代
- MVP开发

## 输入
- 商业假设
- 用户反馈
- 开发资源
- 学习目标

## 输出
- MVP开发
- 验证实验
- 迭代方向
- 学到/未学到

## Prompt模板
```
请使用精益创业方法。

背景：
- 商业假设：[假设]
- 用户反馈：[反馈]
- 开发资源：[资源]
- 学习目标：[目标]

请输出：
1. MVP设计
2. 验证实验
3. 指标定义
4. 迭代决策
5. 学到总结
```

## 示例
### 示例1
- 输入：新创项目
- 输出：精益创业，构建-测量-学习，MVP→用户反馈→迭代

## 注意事项
- 要快速验证
- 要数据驱动
- 要持续迭代
- 要团队学习
