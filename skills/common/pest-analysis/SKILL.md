# PEST分析 (PEST Analysis)

## 触发条件
- 宏观环境
- 市场洞察
- 战略规划
- 行业研究

## 输入
- 分析对象
- 政治因素
- 经济因素
- 社会因素
- 技术因素

## 输出
- PEST框架
- 关键因素
- 机会威胁
- 战略建议

## Prompt模板
```
请进行PEST分析。

背景：
- 分析对象：[对象]
- 政治因素：[政治]
- 经济因素：[经济]
- 社会因素：[社会]
- 技术因素：[技术]

请输出：
1. 政治(Political)
2. 经济(Economic)
3. 社会(Social)
4. 技术(Technological)
5. 关键发现
6. 战略建议
```

## 示例
### 示例1
- 输入：互联网行业政策环境分析
- 输出：PEST分析，含监管趋势、经济影响、用户变化、技术趋势

## 注意事项
- 要与行业相关
- 要关注趋势
- 要战略导向
