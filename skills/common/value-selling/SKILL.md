# 价值销售 (Value Selling)

## 触发条件
- 销售策略
- 价值呈现
- 价格谈判
- 竞争优势

## 输入
- 产品价值
- 客户需求
- 竞争格局
- 销售能力

## 输出
- 销售策略
- 价值呈现
- 谈判策略
- 工具支撑

## Prompt模板
```
请设计[产品]价值销售策略。

背景：
- 产品价值：[价值]
- 客户需求：[需求]
- 竞争格局：[格局]
- 销售能力：[能力]

请输出：
1. 价值主张
2. 销售策略
3. 价值呈现
4. 谈判策略
5. 销售工具
```

## 示例
### 示例1
- 输入：企业软件，需要溢价销售
- 输出：价值销售策略，含ROI计算、案例库、谈判指南

## 注意事项
- 要价值聚焦
- 要差异化
- 要持续训练
