# 定价策略 (Pricing Strategy)

## 触发条件
- 定价决策
- 价格优化
- 价值定价
- 竞争定价

## 输入
- 产品价值
- 成本结构
- 客户价值感知
- 竞争环境

## 输出
- 定价策略
- 价格结构
- 定价流程
- 价格调整机制

## Prompt模板
```
请制定[产品]定价策略。

背景：
- 产品价值：[价值]
- 成本结构：[成本]
- 客户感知：[感知]
- 竞争环境：[环境]

请输出：
1. 定价目标
2. 定价策略选择
3. 价格结构
4. 定价流程
5. 价格调整机制
```

## 示例
### 示例1
- 输入：企业软件，客制化需求强，竞品价格透明
- 输出：价值定价+分层定价+年付优惠

## 注意事项
- 要价值导向
- 要竞争适配
- 要持续优化
