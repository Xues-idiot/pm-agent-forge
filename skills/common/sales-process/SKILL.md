# 销售流程优化 (Sales Process Optimization)

## 触发条件
- 流程效率
- 转化提升
- 标准化复制
- 绩效改进

## 输入
- 当前流程
- 瓶颈环节
- 人员能力
- 工具支持

## 输出
- 流程优化方案
- 关键节点
- 工具建议
- 培训计划

## Prompt模板
```
请优化销售流程。

背景：
- 当前流程：[流程]
- 瓶颈环节：[瓶颈]
- 人员能力：[能力]
- 工具支持：[工具]

请输出：
1. 流程诊断
2. 优化方案
3. 关键节点定义
4. 工具/系统建议
5. 培训计划
```

## 示例
### 示例1
- 输入：销售周期过长，从线索到签单45天
- 输出：流程优化，含阶段拆分、缩短审批、增加自动化、明确责任人

## 注意事项
- 要以客户为中心
- 要消除浪费
- 要量化改进
- 要持续迭代
