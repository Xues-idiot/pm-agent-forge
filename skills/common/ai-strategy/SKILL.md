# AI战略 (AI Strategy)

## 触发条件
- AI应用规划
- AI能力建设
- AI转型
- AI伦理

## 输入
- 业务场景
- AI现状
- 技术能力
- 战略目标

## 输出
- AI战略
- 应用场景
- 能力建设
- 伦理框架

## Prompt模板
```
请制定[组织]AI战略。

背景：
- 业务场景：[场景]
- AI现状：[现状]
- 技术能力：[能力]
- 战略目标：[目标]

请输出：
1. AI愿景
2. 应用场景优先级
3. 技术架构
4. 数据准备
5. 算法选型
6. 团队建设
7. 合作伙伴
8. AI伦理
9. 实施路径
```

## 示例
### 示例1
- 输入：制造企业，想用AI提升质量检测效率
- 输出：AI战略，含视觉检测、预测维护、智能排产

## 注意事项
- 要场景驱动
- 要数据先行
- 要伦理合规
