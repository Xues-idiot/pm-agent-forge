# 线索评分 (Lead Scoring)

## 触发条件
- 线索分级
- 销售优先级
-资源分配
- 转化优化

## 输入
- 线索特征
- 行为数据
- 历史转化
- 成交客户画像

## 输出
- 评分模型
- 分级标准
- 分配规则
- 迭代策略

## Prompt模板
```
请设计线索评分体系。

背景：
- 线索特征：[特征]
- 行为数据：[行为]
- 历史转化：[转化]
- 成交客户画像：[画像]

请输出：
1. 评分维度设计
2. 权重分配
3. 分级标准
4. 分配规则
5. 迭代优化方向
```

## 示例
### 示例1
- 输入：SaaS产品，多渠道线索
- 输出：评分体系，含公司规模×职位匹配度×产品认知度×行为活跃度

## 注意事项
- 要基于数据驱动
- 要结合成交样本
- 要定期迭代
- 要平衡效率和公平
