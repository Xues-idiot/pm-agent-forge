# 教练技术 (Coaching Skills)

## 触发条件
- 员工辅导
- 绩效沟通
- 职业发展
- 领导力培养

## 输入
- 辅导对象
- 辅导目标
- 情境背景
- 关系阶段

## 输出
- 辅导计划
- 对话框架
- 跟进策略
- 效果评估

## Prompt模板
```
请设计[对象]的教练辅导方案。

背景：
- 辅导对象：[对象]
- 辅导目标：[目标]
- 情境背景：[背景]
- 关系阶段：[阶段]

请输出：
1. 辅导目标设定
2. 辅导关系建立
3. 辅导对话框架（GROW等）
4. 关键问题设计
5. 行动跟进
6. 反馈技巧
7. 效果评估
```

## 示例
### 示例1
- 输入：新晋升管理者，需要提升领导力
- 输出：教练方案，含辅导框架、关键对话、跟进计划

## 注意事项
- 要倾听
- 要启发
- 要行动导向
