# 利润率分析 (Margin Analysis)

## 触发条件
- 盈利能力
- 成本控制
- 产品策略
- 定价决策

## 输入
- 收入结构
- 成本组成
- 产品分部
- 渠道分部

## 输出
- 毛利率
- 净利率
- 贡献分析
- 改进机会

## Prompt模板
```
请分析利润率。

背景：
- 收入结构：[收入]
- 成本组成：[成本]
- 产品分部：[产品]
- 渠道分部：[渠道]

请输出：
1. 毛利率
2. 净利率
3. 贡献分析
4. 改进机会
5. 监控指标
```

## 示例
### 示例1
- 输入：多产品线
- 输出：利润率分析，各产品毛利率、渠道利润率、客户利润贡献

## 注意事项
- 要分解到最小
- 要找到驱动
- 要持续监控
- 要优化改进
