# 问题日志 (Issue Log)

## 触发条件
- 问题跟踪
- 任务管理
- 进度监控
- 团队协作

## 输入
- 问题描述
- 责任人
- 优先级
- 截止日期

## 输出
- 问题清单
- 处理状态
- 升级机制
- 统计分析

## Prompt模板
```
请管理问题日志。

背景：
- 问题描述：[描述]
- 责任人：[责任人]
- 优先级：[优先级]
- 截止日期：[日期]

请输出：
1. 问题分类
2. 处理状态
3. 升级机制
4. 统计分析
5. 趋势分析
```

## 示例
### 示例1
- 输入：项目管理问题
- 输出：问题日志，含问题描述、责任人、状态、解决时间

## 注意事项
- 要及时记录
- 要明确责任
- 要定期回顾
- 要持续改进
