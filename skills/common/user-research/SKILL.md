# 用户研究 (User Research)

## 触发条件
- 用户洞察
- 需求发现
- 体验优化
- 产品决策

## 输入
- 研究目标
- 用户群体
- 研究方法
- 资源约束

## 输出
- 研究计划
- 方法选择
- 洞察发现
- 建议输出

## Prompt模板
```
请进行用户研究。

背景：
- 研究目标：[目标]
- 用户群体：[群体]
- 研究方法：[方法]
- 资源约束：[约束]

请输出：
1. 研究计划
2. 方法选择
3. 样本设计
4. 洞察发现
5. 建议输出
```

## 示例
### 示例1
- 输入：新功能需求发现
- 输出：用户研究，含深度访谈、问卷调研、行为数据分析、用户画像

## 注意事项
- 要目标明确
- 要方法适配
- 要样本代表
- 要洞察落地
