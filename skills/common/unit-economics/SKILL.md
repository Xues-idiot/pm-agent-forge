# 单位经济 (Unit Economics)

## 触发条件
- 商业模式
- 盈利能力
- 规模效应
- 定价决策

## 输入
- 收入模型
- 成本结构
- 客户价值
- 获取成本

## 输出
- 单位经济模型
- LTV/CAC
- 盈亏平衡
- 优化方向

## Prompt模板
```
请分析单位经济。

背景：
- 收入模型：[收入]
- 成本结构：[成本]
- 客户价值：[价值]
- 获取成本：[成本]

请输出：
1. 单位经济模型
2. LTV计算
3. CAC计算
4. LTV/CAC比
5. 盈亏平衡
6. 优化方向
```

## 示例
### 示例1
- 输入：SaaS订阅模式
- 输出：单位经济分析，含LTV/CAC、回本周期、规模效应

## 注意事项
- 要真实数据
- 要长期视角
- 要规模效应
