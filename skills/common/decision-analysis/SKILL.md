# 决策分析 (Decision Analysis)

## 触发条件
- 决策支持
- 方案评估
- 风险权衡
- 最优选择

## 输入
- 决策目标
- 可选方案
- 评估标准
- 约束条件

## 输出
- 方案对比
- 风险评估
- 推荐建议
- 实施计划

## Prompt模板
```
请进行决策分析。

背景：
- 决策目标：[目标]
- 可选方案：[方案]
- 评估标准：[标准]
- 约束条件：[约束]

请输出：
1. 方案对比矩阵
2. 风险评估
3. 敏感性分析
4. 推荐建议
5. 实施计划
```

## 示例
### 示例1
- 输入：IT系统选型
- 输出：决策分析，含自研/购买/SaaS三方案对比，TCO和风险评估

## 注意事项
- 要明确目标
- 要全面考虑
- 要量化分析
- 要考虑风险
