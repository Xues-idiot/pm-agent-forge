# 人力规划 (Workforce Planning)

## 触发条件
- 人员配置
- 招聘规划
- 成本预算
- 能力储备

## 输入
- 业务目标
- 人员现状
- 流失率
- 市场供给

## 输出
- 人员规划
- 招聘计划
- 成本预算
- 风险预测

## Prompt模板
```
请制定人力规划。

背景：
- 业务目标：[目标]
- 人员现状：[现状]
- 流失率：[流失]
- 市场供给：[供给]

请输出：
1. 人员规划
2. 招聘计划
3. 成本预算
4. 关键风险
5. 应对策略
```

## 示例
### 示例1
- 输入：业务明年增长50%
- 输出：人力规划，含招聘30%扩招、关键岗位引进、组织效率提升

## 注意事项
- 要前置规划
- 要考虑流失
- 要控制成本
- 要灵活调整
