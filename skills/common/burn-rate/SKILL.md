# 烧钱率 (Burn Rate)

## 触发条件
- 成本控制
- 资金效率
- 融资决策
- 增长策略

## 输入
- 运营成本
- 收入
- 用户增长
- 目标

## 输出
- 烧钱率
- 效率指标
- 优化建议
- 融资时机

## Prompt模板
```
请分析烧钱率。

背景：
- 运营成本：[成本]
- 收入：[收入]
- 用户增长：[增长]
- 目标：[目标]

请输出：
1. 烧钱率计算
2. 效率指标
3. 行业对比
4. 优化建议
5. 融资时机
```

## 示例
### 示例1
- 输入：SaaS公司
- 输出：烧钱率，月成本200万、收入50万、净烧钱150万

## 注意事项
- 要效率优先
- 要优化成本
- 要增长合理
- 要融资节奏
