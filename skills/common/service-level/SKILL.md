# 服务级别协议 (Service Level Agreement)

## 触发条件
- SLA设计
- 服务标准
- 监控机制
- 持续改进

## 输入
- 服务类型
- 客户需求
- 服务能力
- 成本约束

## 输出
- SLA框架
- 服务目录
- 监控指标
- 改进机制

## Prompt模板
```
请设计服务级别协议。

背景：
- 服务类型：[类型]
- 客户需求：[需求]
- 服务能力：[能力]
- 成本约束：[约束]

请输出：
1. SLA框架
2. 服务目录
3. 服务级别
4. 响应时间
5. 监控机制
6. 报告机制
7. 持续改进
```

## 示例
### 示例1
- 输入：IT外包服务，需要与客户签订SLA
- 输出：SLA方案，含服务目录、响应时间、奖罚机制

## 注意事项
- 要可衡量
- 要合理可行
- 要持续监控
