# 燃尽图 (Burndown Chart)

## 触发条件
- 进度跟踪
- 趋势预测
- 风险预警
- 团队管理

## 输入
- 任务总量
- 已完成
- 剩余时间
- 理想趋势

## 输出
- 进度分析
- 趋势预测
- 风险识别
- 调整建议

## Prompt模板
```
请分析燃尽图。

背景：
- 任务总量：[总量]
- 已完成：[完成]
- 剩余时间：[时间]
- 理想趋势：[理想]

请输出：
1. 进度分析
2. 趋势预测
3. 风险识别
4. 调整建议
5. 完成预估
```

## 示例
### 示例1
- 输入：Sprint进度
- 输出：燃尽图分析，含剩余工作量、预测完成、偏差分析、调整建议

## 注意事项
- 要每日更新
- 要分析偏差
- 要及时调整
- 要团队共识
