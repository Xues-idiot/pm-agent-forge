# 客户反馈管理 (Customer Feedback Management)

## 触发条件
- 用户反馈
- 产品改进
- 服务优化
- 满意度提升

## 输入
- 反馈来源
- 反馈内容
- 反馈量级
- 处理能力

## 输出
- 反馈分类
- 处理流程
- 改进计划
- 闭环机制

## Prompt模板
```
请管理客户反馈。

背景：
- 反馈来源：[来源]
- 反馈内容：[内容]
- 反馈量级：[量级]
- 处理能力：[能力]

请输出：
1. 反馈分类体系
2. 处理流程
3. 优先级排序
4. 改进计划
5. 闭环机制
```

## 示例
### 示例1
- 输入：App Store评分下降，从4.5降到3.8
- 输出：反馈管理，含紧急修复Top问题、主动联系差评用户、每周复盘

## 注意事项
- 要及时响应
- 要分类处理
- 要闭环追踪
- 要转化为改进
