# 威胁检测 (Threat Detection)

## 触发条件
- 网络威胁
- 恶意行为
- 漏洞利用
- 攻击检测

## 输入
- 威胁类型
- 检测规则
- 历史攻击
- 情报来源

## 输出
- 检测方案
- 规则库
- 分析方法
- 响应策略

## Prompt模板
```
请检测威胁。

背景：
- 威胁类型：[类型]
- 检测规则：[规则]
- 历史攻击：[攻击]
- 情报来源：[来源]

请输出：
1. 检测方案
2. 规则库
3. 分析方法
4. 响应策略
5. 处置流程
```

## 示例
### 示例1
- 输入：网络入侵
- 输出：威胁检测，IDS/IPS、行为分析、威胁情报联动

## 注意事项
-要多维度检测
- 要行为分析
- 要情报协同
- 要快速处置
