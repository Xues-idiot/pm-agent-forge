# 特许经营 (Franchise)

## 触发条件
- 商业模式
- 快速扩张
- 品牌输出
- 风险分担

## 输入
- 商业模型
- 品牌价值
- 加盟商
- 标准化

## 输出
-  Franchise策略
- 加盟设计
- 标准化
- 支持体系

## Prompt模板
```
请制定特许经营策略。

背景：
- 商业模型：[模型]
- 品牌价值：[品牌]
- 加盟商：[加盟]
- 标准化：[标准]

请输出：
1. Franchise策略
2. 加盟设计
3. 标准化体系
4. 支持体系
5. 风险控制
```

## 示例
### 示例1
- 输入：餐饮连锁
- 输出：特许经营，单店模型→标准化运营→加盟复制

## 注意事项
- 要模型可复制
- 要标准化
- 要加盟商筛选
- 要持续支持
