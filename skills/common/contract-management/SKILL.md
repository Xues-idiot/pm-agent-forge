# 合同管理 (Contract Management)

## 触发条件
- 合同签订
- 合同执行
- 变更管理
- 风险控制

## 输入
- 合同条款
- 执行情况
- 变更请求
- 履约风险

## 输出
- 合同台账
- 执行跟踪
- 变更记录
- 风险预警

## Prompt模板
```
请管理合同。

背景：
- 合同条款：[条款]
- 执行情况：[执行]
- 变更请求：[变更]
- 履约风险：[风险]

请输出：
1. 合同台账
2. 执行跟踪
3. 变更记录
4. 风险预警
5. 关闭条件
```

## 示例
### 示例1
- 输入：采购合同
- 输出：合同管理，交付进度、付款节点、变更流程

## 注意事项
- 要合规执行
- 要及时跟踪
- 要变更受控
- 要到期提醒
