# 回顾会议 (Retrospectives)

## 触发条件
- 迭代复盘
- 过程改进
- 团队反思
- 持续改进

## 输入
- 迭代周期
- 团队表现
- 成功因素
- 改进机会

## 输出
- 回顾议程
- 问题识别
- 改进措施
- 行动项

## Prompt模板
```
请组织回顾会议。

背景：
- 迭代周期：[周期]
- 团队表现：[表现]
- 成功因素：[成功]
- 改进机会：[机会]

请输出：
1. 回顾议程
2. 问题识别
3. 改进措施
4. 行动项
5. 跟踪机制
```

## 示例
### 示例1
- 输入：敏捷团队
- 输出：回顾会议，继续/停止/开始、投票排序、改进计划

## 注意事项
- 要开放氛围
- 要聚焦改进
- 要可操作
- 要跟踪执行
