# 转移定价 (Transfer Pricing)

## 触发条件
- 关联交易
- 税负优化
- 内部结算
- 合规要求

## 输入
- 交易类型
- 定价方法
- 市场基准
- 合规要求

## 输出
- 定价策略
- 定价规则
- 结算机制
- 合规检查

## Prompt模板
```
请制定转移定价策略。

背景：
- 交易类型：[类型]
- 定价方法：[方法]
- 市场基准：[基准]
- 合规要求：[要求]

请输出：
1. 定价策略
2. 定价规则
3. 结算机制
4. 合规检查
5. 文档要求
```

## 示例
### 示例1
- 输入：集团内交易
- 输出：转移定价，独立交易原则、可比价格法、利润分割法

## 注意事项
- 要符合法规
- 要公平交易
- 要文档齐全
- 要风险可控
