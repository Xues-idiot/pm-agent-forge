# 成本优化 (Cost Optimization)

## 触发条件
- 成本控制
- 效率提升
- 资源浪费
- 预算管理

## 输入
- 成本结构
- 业务流程
- 资源配置
- 优化目标

## 输出
- 优化方案
- 优先级排序
- 实施路径
- 预期效果

## Prompt模板
```
请制定成本优化方案。

背景：
- 成本结构：[结构]
- 业务流程：[流程]
- 资源配置：[资源]
- 优化目标：[目标]

请输出：
1. 成本分析
2. 优化机会
3. 优先级排序
4. 实施路径
5. 预期效果
```

## 示例
### 示例1
- 输入：运营成本超支20%
- 输出：优化方案，含流程简化、采购议价、外包非核心、自动化替代

## 注意事项
- 要区分固定和可变
- 要避免牺牲价值
- 要分阶段实施
- 要持续监控
