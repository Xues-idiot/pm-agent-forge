# 平台战略 (Platform Strategy)

## 触发条件
- 平台定位
- 双边市场
- 网络效应
- 平台治理

## 输入
- 业务类型
- 用户规模
- 变现模式
- 竞争格局

## 输出
- 平台战略
- 生态设计
- 治理机制
- 增长策略

## Prompt模板
```
请制定[平台]战略。

背景：
- 业务类型：[类型]
- 用户规模：[规模]
- 变现模式：[模式]
- 竞争格局：[格局]

请输出：
1. 平台定位
2. 双边市场设计
3. 价值创造
4. 网络效应
5. 平台治理
6. 变现模式
7. 增长策略
8. 开放生态
```

## 示例
### 示例1
- 输入：交易平台，需要提升双边活跃度
- 输出：平台战略，含撮合机制、激励设计、治理规则

## 注意事项
- 要双边平衡
- 要网络效应
- 要生态治理
