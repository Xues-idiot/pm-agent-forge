# 破产重组 (Bankruptcy Restructuring)

## 触发条件
- 债务危机
- 破产重整
- 债务重组
- 资产清算

## 输入
- 债务状况
- 资产情况
- 债权人结构
- 重组目标

## 输出
- 债务方案
- 重组路径
- 债权处理
- 运营优化

## Prompt模板
```
请进行破产重组。

背景：
- 债务状况：[状况]
- 资产情况：[资产]
- 债权人结构：[结构]
- 重组目标：[目标]

请输出：
1. 债务方案
2. 重组路径
3. 债权处理
4. 运营优化
5. 时间表
```

## 示例
### 示例1
- 输入：企业债务危机
- 输出：债务重组，削债、转股、延期

## 注意事项
- 要依法合规
- 要平衡利益
- 要社会稳定
- 要恢复运营
