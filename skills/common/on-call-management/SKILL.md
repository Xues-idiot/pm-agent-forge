# On-Call管理 (On-Call Management)

## 触发条件
- 值班管理
- 故障响应
- 运维保障
- 团队协作

## 输入
- 服务重要性
- 团队规模
- 响应要求
- 激励机制

## 输出
- 值班制度
- 响应流程
- 升级机制
- 激励方案

## Prompt模板
```
请管理On-Call。

背景：
- 服务重要性：[重要]
- 团队规模：[规模]
- 响应要求：[要求]
- 激励机制：[激励]

请输出：
1. 值班制度
2. 响应SLA
3. 升级机制
4. 激励方案
5. 轮换规则
```

## 示例
### 示例1
- 输入：24小时互联网服务
- 输出：On-Call管理，含P1故障15分钟响应、周末双班制、值班补贴

## 注意事项
- 要合理轮换
- 要及时响应
- 要有Backup
- 要持续改进
