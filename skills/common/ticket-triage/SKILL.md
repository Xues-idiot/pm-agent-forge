# 工单分类 (Ticket Triage)

## 触发条件
- 问题处理
- 优先级
- 分类管理
- 响应效率

## 输入
- 工单内容
- 类型分类
- 优先级
- 处理能力

## 输出
- 分类结果
- 优先级
- 路由分配
- 处理建议

## Prompt模板
```
请进行工单分类。

背景：
- 工单内容：[内容]
- 类型分类：[类型]
- 优先级：[优先]
- 处理能力：[能力]

请输出：
1. 问题分类
2. 优先级评估
3. 路由分配
4. 处理建议
5. SLA预估
```

## 示例
### 示例1
- 输入：客服工单
- 输出：工单分类，含问题类型、紧急程度、路由到合适团队、SLA承诺

## 注意事项
- 要快速响应
- 要准确分类
- 要合理分配
- 要持续优化
