# 用户测试 (User Testing)

## 触发条件
- 用户验证
- 产品测试
- 反馈收集
- 体验优化

## 输入
- 测试目标
- 用户筛选
- 测试任务
- 评估指标

## 输出
- 测试方案
- 用户招募
- 任务设计
- 结果分析

## Prompt模板
```
请进行用户测试。

背景：
- 测试目标：[目标]
- 用户筛选：[筛选]
- 测试任务：[任务]
- 评估指标：[指标]

请输出：
1. 测试方案
2. 用户招募
3. 测试执行
4. 数据收集
5. 结果分析
```

## 示例
### 示例1
- 输入：App测试
- 输出：用户测试，5名用户、完成任务、观察记录、分析报告

## 注意事项
- 要明确目标
- 要真实场景
- 要及时记录
- 要深入分析
