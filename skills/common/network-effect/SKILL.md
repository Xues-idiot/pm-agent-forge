# 网络效应 (Network Effect)

## 触发条件
- 平台战略
- 用户价值
- 市场定位
- 增长策略

## 输入
- 用户基础
- 连接类型
- 价值创造
- 竞争壁垒

## 输出
- 网络效应类型
- 价值分析
- 发展阶段
- 增长策略

## Prompt模板
```
请分析网络效应。

背景：
- 用户基础：[基础]
- 连接类型：[类型]
- 价值创造：[价值]
- 竞争壁垒：[壁垒]

请输出：
1. 网络效应类型
2. 价值分析
3. 临界点
4. 增长策略
5. 竞争壁垒
```

## 示例
### 示例1
- 输入：社交网络
- 输出：网络效应，直接网络效应、间接网络效应、临界用户数

## 注意事项
- 要达到临界点
- 要促进连接
- 要体验一致
- 要持续增长
