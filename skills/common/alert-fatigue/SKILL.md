# 告警优化 (Alert Fatigue Prevention)

## 触发条件
- 告警过多
- 噪音过滤
- 响应效率
- 监控质量

## 输入
- 告警现状
- 处理能力
- 故障数据
- 响应时效

## 输出
- 告警评估
- 优化策略
- 分级标准
- 治理流程

## Prompt模板
```
请优化告警。

背景：
- 告警现状：[现状]
- 处理能力：[能力]
- 故障数据：[数据]
- 响应时效：[时效]

请输出：
1. 告警评估
2. 噪音分析
3. 优化策略
4. 分级标准
5. 治理流程
```

## 示例
### 示例1
- 输入：每日数千条告警
- 输出：告警优化，含重复告警合并、动态阈值、智能化分级、订阅机制

## 注意事项
- 要信号质量
- 要动态调整
- 要持续优化
- 要团队共识
