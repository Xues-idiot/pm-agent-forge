# SEO优化 (SEO Optimization)

## 触发条件
- 搜索引擎
- 自然流量
- 关键词排名
- 网站优化

## 输入
- 关键词策略
- 现有排名
- 网站结构
- 内容质量

## 输出
- 审计报告
- 优化策略
- 内容计划
- 预期效果

## Prompt模板
```
请优化SEO。

背景：
- 关键词策略：[策略]
- 现有排名：[排名]
- 网站结构：[结构]
- 内容质量：[质量]

请输出：
1. 审计报告
2. 优化策略
3. 内容计划
4. 预期效果
5. 监控机制
```

## 示例
### 示例1
- 输入：网站SEO
- 输出：SEO优化，技术SEO、内容SEO、外链建设

## 注意事项
- 要白帽SEO
- 要长期投入
- 要用户体验
- 要持续监测
