# 创新管理 (Innovation Management)

## 触发条件
- 创新机制设计
- 创新项目管理
- 创新文化培育
- 创新成果转化

## 输入
- 组织类型
- 创新目标
- 资源投入
- 风险偏好

## 输出
- 创新机制
- 创新流程
- 评估体系
- 转化路径

## Prompt模板
```
请为[组织]设计创新管理体系。

背景：
- 组织类型：[类型]
- 创新目标：[目标]
- 资源投入：[资源]
- 风险偏好：[风险]

请输出：
1. 创新战略定位
2. 创新类型（渐进/颠覆/突破）
3. 创新机制设计（创意收集/评审/孵化）
4. 创新流程
5. 资源配置
6. 激励机制
7. 风险控制
8. 成果转化路径
9. 效果评估
```

## 示例
### 示例1
- 输入：大型企业，需要激励员工创新，防止大企业病
- 输出：创新方案，含创意大赛、内部孵化、创新基金

## 注意事项
- 要宽容失败
- 要快速验证
- 要商业化导向
