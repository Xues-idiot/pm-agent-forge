# 恢复测试 (Recovery Testing)

## 触发条件
- 灾难恢复
- 故障演练
- 预案验证
- 团队能力

## 输入
- 恢复场景
- 测试目标
- 参与团队
- 评估标准

## 输出
- 测试计划
- 测试结果
- 问题发现
- 改进行动

## Prompt模板
```
请进行恢复测试。

背景：
- 恢复场景：[场景]
- 测试目标：[目标]
- 参与团队：[团队]
- 评估标准：[标准]

请输出：
1. 测试计划
2. 测试执行
3. 测试结果
4. 问题发现
5. 改进计划
```

## 示例
### 示例1
- 输入：数据中心故障切换
- 输出：恢复测试，RTO/RPO验证、切换流程、团队协作评估

## 注意事项
- 要定期测试
- 要全面覆盖
- 要记录问题
- 要持续改进
