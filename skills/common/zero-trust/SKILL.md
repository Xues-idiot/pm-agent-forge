# 零信任 (Zero Trust)

## 触发条件
- 零信任架构
- 持续验证
- 微分段
- 安全边界

## 输入
- 保护对象
- 信任模型
- 网络架构
- 监控能力

## 输出
- 零信任架构
- 验证策略
- 网络分段
- 实施路径

## Prompt模板
```
请实施零信任。

背景：
- 保护对象：[对象]
- 信任模型：[模型]
- 网络架构：[架构]
- 监控能力：[能力]

请输出：
1. 零信任架构
2. 验证策略
3. 网络分段
4. 实施路径
5. 评估标准
```

## 示例
### 示例1
- 输入：企业安全
- 输出：零信任，永不信任、持续验证、最小权限

## 注意事项
- 要身份中心
- 要微分段
- 要持续监控
- 要渐进实施
