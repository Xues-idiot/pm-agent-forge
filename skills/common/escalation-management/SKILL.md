# 升级管理 (Escalation Management)

## 触发条件
- 问题升级
- 决策升级
- 资源协调
- 风险上报

## 输入
- 问题类型
- 升级标准
- 决策权限
- 处理时效

## 输出
- 升级流程
- 升级标准
- 决策权限
- 反馈机制

## Prompt模板
```
请建立升级管理机制。

背景：
- 问题类型：[类型]
- 升级标准：[标准]
- 决策权限：[权限]
- 处理时效：[时效]

请输出：
1. 升级场景
2. 升级标准
3. 升级路径
4. 决策权限
5. 反馈机制
```

## 示例
### 示例1
- 输入：客户投诉升级
- 输出：升级机制，含一线处理→组长→经理→总监四级升级路径

## 注意事项
- 要明确标准
- 要及时升级
- 要信息同步
- 要闭环处理
