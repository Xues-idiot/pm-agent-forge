# 负载均衡 (Load Balancing)

## 触发条件
- 流量分发
- 高可用
- 性能优化
- 故障处理

## 输入
- 流量特点
- 性能要求
- 可用性目标
- 成本预算

## 输出
- 负载策略
- 算法选择
- 健康检查
- 故障处理

## Prompt模板
```
请设计负载均衡。

背景：
- 流量特点：[特点]
- 性能要求：[要求]
- 可用性目标：[目标]
- 成本预算：[预算]

请输出：
1. 负载策略
2. 算法选择
3. 健康检查
4. 故障处理
5. 监控告警
```

## 示例
### 示例1
- 输入：Web应用
- 输出：负载均衡，含Nginx/LVS七层/四层负载、健康检查、故障自动切换

## 注意事项
- 要高可用
- 要算法合适
- 要健康检查
- 要监控告警
