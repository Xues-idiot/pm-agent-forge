# 待完成工作 (Jobs To Be Done)

## 触发条件
- 用户洞察
- 产品创新
- 需求分析
- 价值定位

## 输入
- 用户群体
- 使用场景
- 目标任务
- 痛点需求

## 输出
- JTBD框架
- 用户任务
- 痛点分析
- 创新机会

## Prompt模板
```
请用JTBD框架分析用户需求。

背景：
- 用户群体：[用户]
- 使用场景：[场景]
- 目标任务：[任务]
- 痛点需求：[痛点]

请输出：
1. 功能性任务
2. 情感性任务
3. 社会性任务
4. 痛点分析
5. 创新机会
```

## 示例
### 示例1
- 输入：咖啡消费者行为研究
- 输出：JTBD分析，含提神-社交-习惯-犒劳等任务类型

## 注意事项
- 要用户视角
- 要深入洞察
- 要创新导向
