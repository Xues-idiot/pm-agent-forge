# 信用管理 (Credit Management)

## 触发条件
- 信用政策
- 客户信用
- 风险控制
- 应收账款

## 输入
- 信用政策
- 客户评估
- 账期
- 坏账

## 输出
- 信用评估
- 额度设定
- 监控机制
- 催收策略

## Prompt模板
```
请管理信用。

背景：
- 信用政策：[政策]
- 客户评估：[评估]
- 账期：[账期]
- 坏账：[坏账]

请输出：
1. 信用评估
2. 额度设定
3. 监控机制
4. 催收策略
5. 风险控制
```

## 示例
### 示例1
- 输入：B2B企业
- 输出：信用管理，客户分级、账期设定、信用额度

## 注意事项
- 要评估客户
- 要控制风险
- 要及时催收
- 要动态调整
