# 客户旅程 (Customer Journey)

## 触发条件
- 旅程设计
- 触点优化
- 体验提升
- 转化提升

## 输入
- 客户类型
- 购买决策
- 触点现状
- 优化目标

## 输出
- 旅程地图
- 触点分析
- 优化机会
- 实施方案

## Prompt模板
```
请设计[客户类型]客户旅程。

背景：
- 客户类型：[类型]
- 购买决策：[决策]
- 触点现状：[现状]
- 优化目标：[目标]

请输出：
1. 旅程阶段
2. 各阶段触点
3. 客户行为
4. 关键决策点
5. 痛点分析
6. 优化机会
```

## 示例
### 示例1
- 输入：B2B采购决策，决策链复杂
- 输出：旅程地图，含认知-评估-决策-采购全链路

## 注意事项
- 要客户视角
- 要全链路
- 要持续优化
