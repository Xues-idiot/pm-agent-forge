# 资金管理 (Treasury)

## 触发条件
- 资金集中
- 投资管理
- 利率风险
- 外汇风险

## 输入
- 资金池
- 投资标的
- 利率环境
- 外汇敞口

## 输出
- 资金策略
- 投资组合
- 风险管理
- 收益优化

## Prompt模板
```
请管理资金。

背景：
- 资金池：[池]
- 投资标的：[标的]
- 利率环境：[利率]
- 外汇敞口：[敞口]

请输出：
1. 资金策略
2. 投资组合
3. 风险管理
4. 收益优化
5. 流动性保障
```

## 示例
### 示例1
- 输入：集团资金
- 输出：资金管理，内部借贷、短期理财、外汇对冲

## 注意事项
- 要安全优先
- 要流动性
- 要收益平衡
- 要风险控制
