# 投资组合管理 (Investment Portfolio)

## 触发条件
- 资产管理
- 分散投资
- 风险收益
- 收益优化

## 输入
- 可投资产
- 风险偏好
- 收益目标
- 投资期限

## 输出
- 组合配置
- 风险管理
- 业绩评估
- 调整建议

## Prompt模板
```
请管理投资组合。

背景：
- 可投资产：[资产]
- 风险偏好：[偏好]
- 收益目标：[目标]
- 投资期限：[期限]

请输出：
1. 组合配置
2. 风险管理
3. 业绩评估
4. 调整建议
5. 监控指标
```

## 示例
### 示例1
- 输入：机构投资者
- 输出：组合配置，股票60%、债券30%、现金10%

## 注意事项
- 要分散风险
- 要流动性
- 要动态调整
- 要长期视角
