# 第三方风险管理 (Third-party Risk Management)

## 触发条件
- 供应商风险
- 外包管理
- 合作伙伴
- 供应链安全

## 输入
- 第三方列表
- 风险类型
- 合同条款
- 监控能力

## 输出
- 风险评估
- 分级管理
- 监控机制
- 应急计划

## Prompt模板
```
请管理第三方风险。

背景：
- 第三方列表：[列表]
- 风险类型：[类型]
- 合同条款：[条款]
- 监控能力：[能力]

请输出：
1. 风险评估
2. 分级管理体系
3. 监控机制
4. 应急计划
5. 退出策略
```

## 示例
### 示例1
- 输入：IT外包服务商
- 输出：风险管理，含数据安全审计、服务水平监控、业务连续性预案

## 注意事项
- 要全面评估
- 要持续监控
- 要合同保障
- 要准备替代
