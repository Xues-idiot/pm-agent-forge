# 经验教训 (Lessons Learned)

## 触发条件
- 经验沉淀
- 知识管理
- 流程优化
- 风险预防

## 输入
- 项目案例
- 成功经验
- 失败教训
- 最佳实践

## 输出
- 经验分类
- 教训总结
- 应用建议
- 分享机制

## Prompt模板
```
请沉淀经验教训。

背景：
- 项目案例：[案例]
- 成功经验：[经验]
- 失败教训：[教训]
- 最佳实践：[实践]

请输出：
1. 经验分类
2. 成功要素
3. 失败原因
4. 应用建议
5. 分享计划
```

## 示例
### 示例1
- 输入：多个项目复盘总结
- 输出：经验库，含需求变更处理、风险预警、沟通技巧、技术选型

## 注意事项
- 要及时沉淀
- 要分类管理
- 要易于检索
- 要定期回顾
