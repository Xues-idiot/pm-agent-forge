# 保险风险管理 (Insurance Risk Management)

## 触发条件
- 保险配置
- 风险转移
- 保障规划
- 保费优化

## 输入
- 风险敞口
- 保险需求
- 预算限制
- 现有保单

## 输出
- 保险方案
- 保费测算
- 保障分析
- 优化建议

## Prompt模板
```
请管理保险风险。

背景：
- 风险敞口：[敞口]
- 保险需求：[需求]
- 预算限制：[预算]
- 现有保单：[保单]

请输出：
1. 保险方案
2. 保费测算
3. 保障分析
4. 优化建议
5. 风险覆盖
```

## 示例
### 示例1
- 输入：企业风险
- 输出：保险方案，财产险、责任险、信用险

## 注意事项
- 要全面覆盖
- 要成本效益
- 要动态调整
- 要合规要求
