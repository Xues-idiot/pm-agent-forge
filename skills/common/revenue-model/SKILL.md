# 收入模式 (Revenue Model)

## 触发条件
- 商业模式
- 收入来源
- 盈利结构
- 增长策略

## 输入
- 产品价值
- 客户需求
- 定价方式
- 收入占比

## 输出
- 收入模式
- 收入来源
- 定价机制
- 优化方向

## Prompt模板
```
请设计收入模式。

背景：
- 产品价值：[价值]
- 客户需求：[需求]
- 定价方式：[方式]
- 收入占比：[占比]

请输出：
1. 收入模式
2. 收入来源
3. 定价机制
4. 收入结构
5. 优化方向
```

## 示例
### 示例1
- 输入：多元化企业
- 输出：收入模式，产品销售+服务收入+订阅收入+广告收入

## 注意事项
- 要多元收入
- 要可预见性
- 要客户价值
- 要持续增长
