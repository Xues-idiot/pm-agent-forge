# 实时处理 (Real-time Processing)

## 触发条件
- 实时分析
- 流处理
- 低延迟
- 事件驱动

## 输入
- 数据流
- 处理逻辑
- 延迟要求
- 规模需求

## 输出
- 流处理架构
- 处理模型
- 容错设计
- 监控机制

## Prompt模板
```
请设计实时处理。

背景：
- 数据流：[流]
- 处理逻辑：[逻辑]
- 延迟要求：[延迟]
- 规模需求：[规模]

请输出：
1. 流处理架构
2. 处理模型
3. 容错设计
4. 监控机制
5. 扩展策略
```

## 示例
### 示例1
- 输入：实时推荐
- 输出：实时处理，Flink流处理、状态管理、窗口计算

## 注意事项
- 要低延迟
- 要exactly-once
- 要状态管理
- 要可扩展
