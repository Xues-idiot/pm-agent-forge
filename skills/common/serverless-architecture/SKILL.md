# 无服务器架构 (Serverless Architecture)

## 触发条件
- Serverless
- 函数计算
- 成本优化
- 弹性扩展

## 输入
- 业务场景
- 负载特点
- 开发能力
- 成本目标

## 输出
- Serverless设计
- 函数划分
- 触发器设计
- 成本模型

## Prompt模板
```
请设计Serverless架构。

背景：
- 业务场景：[场景]
- 负载特点：[特点]
- 开发能力：[能力]
- 成本目标：[目标]

请输出：
1. Serverless设计
2. 函数划分
3. 触发器设计
4. 数据管理
5. 成本模型
```

## 示例
### 示例1
- 输入：事件处理
- 输出：Serverless设计，含阿里云函数计算、EventBridge触发器、日志存储

## 注意事项
- 要事件驱动
- 要函数独立
- 要冷启动优化
- 要成本监控
