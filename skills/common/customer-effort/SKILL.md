# 客户费力度 (Customer Effort Score)

## 触发条件
- 服务体验
- 便捷性
- 成本分析
- 体验优化

## 输入
- 任务难度
- 解决路径
- 时间成本
- 客户反馈

## 输出
- CES评估
- 体验分析
- 简化机会
- 优化方案

## Prompt模板
```
请分析客户费力度。

背景：
- 任务难度：[难度]
- 解决路径：[路径]
- 时间成本：[时间]
- 客户反馈：[反馈]

请输出：
1. CES计算
2. 体验分析
3. 高费力点
4. 简化机会
5. 优化方案
```

## 示例
### 示例1
- 输入：客服解决问题
- 输出：CES分析，平均3.2分，退款流程费力最高

## 注意事项
- 要低费力
- 要快速解决
- 要简化流程
- 要持续优化
