# 威胁情报 (Threat Intelligence)

## 触发条件
- 安全威胁
- 情报收集
- 风险预警
- 防御增强

## 输入
- 情报来源
- 威胁类型
- 行业背景
- 现有防御

## 输出
- 情报分析
- 威胁评估
- 防御建议
- 预警机制

## Prompt模板
```
请分析威胁情报。

背景：
- 情报来源：[来源]
- 威胁类型：[类型]
- 行业背景：[背景]
- 现有防御：[防御]

请输出：
1. 情报分析
2. 威胁评估
3. 防御建议
4. 预警机制
5. 响应预案
```

## 示例
### 示例1
- 输入：金融行业
- 输出：威胁情报，APT攻击、钓鱼攻击、行业威胁报告

## 注意事项
- 要多源情报
- 要专业分析
- 要及时预警
- 要主动防御
