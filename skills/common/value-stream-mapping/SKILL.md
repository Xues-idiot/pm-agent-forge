# 价值流映射 (Value Stream Mapping)

## 触发条件
- 流程优化
- 浪费识别
- 交付效率
- 精益改善

## 输入
- 业务流程序列
- 时间分析
- 价值判断
- 改进目标

## 输出
- 当前状态图
- 未来状态图
- 改善机会
- 实施计划

## Prompt模板
```
请绘制价值流映射。

背景：
- 业务流程序列：[流程]
- 时间分析：[时间]
- 价值判断：[价值]
- 改进目标：[目标]

请输出：
1. 当前状态图
2. 时间分析
3. 浪费识别
4. 未来状态图
5. 实施计划
```

## 示例
### 示例1
- 输入：订单履行流程
- 输出：价值流图，含各环节时间、增值vs非增值、改善机会

## 注意事项
- 要端到端
- 要数据支撑
- 要全员参与
- 要持续改善
