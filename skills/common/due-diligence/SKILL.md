# 尽职调查 (Due Diligence)

## 触发条件
- 投资决策
- 尽职调查
- 风险评估
- 交易完成

## 输入
- 调查标的
- 调查范围
- 时间要求
- 资源安排

## 输出
- 调查报告
- 风险发现
- 价值评估
- 投资建议

## Prompt模板
```
请进行尽职调查。

背景：
- 调查标的：[标的]
- 调查范围：[范围]
- 时间要求：[时间]
- 资源安排：[资源]

请输出：
1. 调查计划
2. 调查发现
3. 风险评估
4. 价值评估
5. 投资建议
```

## 示例
### 示例1
- 输入：投资标的
- 输出：尽调报告，财务、法务、业务、技术全面调查

## 注意事项
- 要全面深入
- 要数据验证
- 要关注风险
- 要专业独立
