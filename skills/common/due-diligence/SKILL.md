# 尽职调查 (Due Diligence)

## 触发条件
- 并购交易
- 合作伙伴
- 投资决策
- 供应商审核

## 输入
- 调查对象
- 调查范围
- 时间要求
- 资源安排

## 输出
- 调查报告
- 风险发现
- 价值评估
- 合作建议

## Prompt模板
```
请进行尽职调查。

背景：
- 调查对象：[对象]
- 调查范围：[范围]
- 时间要求：[要求]
- 资源安排：[安排]

请输出：
1. 调查计划
2. 调查发现
3. 风险评估
4. 价值评估
5. 合作建议
```

## 示例
### 示例1
- 输入：收购目标公司
- 输出：尽调报告，含财务、法务、技术、市场、人力资源全面调查

## 注意事项
- 要全面深入
- 要数据验证
- 要关注风险
- 要专业独立
