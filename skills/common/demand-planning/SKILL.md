# 需求计划 (Demand Planning)

## 触发条件
- 需求预测
- 计划协调
- 供应链协同
- 库存优化

## 输入
- 历史数据
- 市场趋势
- 促销活动
- 季节因素

## 输出
- 需求预测
- 计划协调
- 风险预警
- 优化建议

## Prompt模板
```
请进行需求计划。

背景：
- 历史数据：[历史]
- 市场趋势：[趋势]
- 促销活动：[活动]
- 季节因素：[季节]

请输出：
1. 需求预测
2. 计划协调
3. 风险预警
4. 优化建议
5. 监控指标
```

## 示例
### 示例1
- 输入：消费品需求
- 输出：需求计划，月度预测、节假日调整、安全库存

## 注意事项
- 要数据驱动
- 要协同沟通
- 要滚动预测
- 要持续改进
