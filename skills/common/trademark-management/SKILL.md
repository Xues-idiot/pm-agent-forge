# 商标管理 (Trademark Management)

## 触发条件
- 商标注册
- 品牌保护
- 侵权处理
- 续展维护

## 输入
- 品牌战略
- 商标清单
- 市场覆盖
- 侵权情况

## 输出
- 商标布局
- 注册策略
- 维权方案
- 续展计划

## Prompt模板
```
请管理商标。

背景：
- 品牌战略：[战略]
- 商标清单：[清单]
- 市场覆盖：[覆盖]
- 侵权情况：[侵权]

请输出：
1. 商标布局
2. 注册策略
3. 维权方案
4. 续展计划
5. 监测机制
```

## 示例
### 示例1
- 输入：品牌全球化
- 输出：商标管理，全球注册、防御注册、侵权监测

## 注意事项
- 要全面保护
- 要提前注册
- 要持续监测
- 要维权及时
