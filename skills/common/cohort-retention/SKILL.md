# 队列留存分析 (Cohort Retention)

## 触发条件
- 用户留存
- 行为分析
- 产品优化
- 生命周期

## 输入
- 用户队列
- 时间维度
- 行为定义
- 分组维度

## 输出
- 留存曲线
- 队列对比
- 流失分析
- 优化建议

## Prompt模板
```
请进行队列留存分析。

背景：
- 用户队列：[队列]
- 时间维度：[维度]
- 行为定义：[定义]
- 分组维度：[分组]

请输出：
1. 留存曲线
2. 队列对比
3. 流失分析
4. 优化建议
5. 监控指标
```

## 示例
### 示例1
- 输入：App用户
- 输出：留存分析，次日/7日/30日留存、同期群对比

## 注意事项
- 要定义清晰
- 要持续跟踪
- 要细分分析
- 要结合业务
