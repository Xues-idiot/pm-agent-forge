# SMART目标 (SMART Goals)

## 触发条件
- 目标制定
- 计划分解
- 绩效设定
- 个人发展

## 输入
- 愿景目标
- 时间范围
- 资源情况
- 能力水平

## 输出
- SMART目标
- 分解计划
- 里程碑
- 评估标准

## Prompt模板
```
请将目标制定为SMART目标。

背景：
- 愿景目标：[愿景]
- 时间范围：[时间]
- 资源情况：[资源]
- 能力水平：[能力]

请输出：
1. Specific（具体）
2. Measurable（可衡量）
3. Achievable（可达成）
4. Relevant（相关）
5. Time-bound（有时限）
6. 分解计划
7. 里程碑
```

## 示例
### 示例1
- 输入：销售目标"提升业绩"
- 输出：SMART目标，"Q3将销售额提升20%至500万"

## 注意事项
- 要具体清晰
- 要可衡量
- 要可达成
