# 创意评估 (Idea Evaluation)

## 触发条件
- 创意筛选
- 方案评估
- 投资决策
- 优先级

## 输入
- 创意描述
- 评估标准
- 市场数据
- 资源需求

## 输出
- 评估矩阵
- 评分分析
- 优先级
- 建议结论

## Prompt模板
```
请评估创意。

背景：
- 创意描述：[描述]
- 评估标准：[标准]
- 市场数据：[市场]
- 资源需求：[资源]

请输出：
1. 评估维度
2. 评估矩阵
3. 评分分析
4. 风险评估
5. 建议结论
```

## 示例
### 示例1
- 输入：产品创意
- 输出：创意评估，创新性、可行性、市场规模、竞争分析

## 注意事项
- 要多维度
- 要数据支撑
- 要团队共识
- 要动态调整
