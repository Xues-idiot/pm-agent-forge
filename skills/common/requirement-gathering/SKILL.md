# 需求收集 (Requirement Gathering)

## 触发条件
- 需求获取
- 业务调研
- 用户访谈
- 文档分析

## 输入
- 业务背景
- 相关干系人
- 现有系统
- 调研计划

## 输出
- 需求清单
- 需求优先级
- 业务规则
- 验收标准

## Prompt模板
```
请收集需求。

背景：
- 业务背景：[背景]
- 相关干系人：[干系人]
- 现有系统：[系统]
- 调研计划：[计划]

请输出：
1. 需求清单
2. 需求优先级
3. 业务规则
4. 验收标准
5. 假设约束
```

## 示例
### 示例1
- 输入：系统需求
- 输出：需求收集，用户访谈、问卷调查、原型确认

## 注意事项
- 要多方收集
- 要明确完整
- 要优先级
- 要可验证
