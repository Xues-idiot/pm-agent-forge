# 增长实验 (Growth Experiments)

## 触发条件
- 增长黑客
- 实验驱动
- 快速迭代
- 规模扩展

## 输入
- 增长目标
- 实验假设
- 测试资源
- 分析能力

## 输出
- 实验优先级
- 测试设计
- 数据分析
- 决策建议

## Prompt模板
```
请进行增长实验。

背景：
- 增长目标：[目标]
- 实验假设：[假设]
- 测试资源：[资源]
- 分析能力：[能力]

请输出：
1. 实验优先级
2. 测试设计
3. 数据分析
4. 决策建议
5. 扩展计划
```

## 示例
### 示例1
- 输入：用户增长
- 输出：增长实验，改版测试、文案测试、渠道测试

## 注意事项
- 要假设驱动
- 要快速测试
- 要数据分析
- 要规模化
