# 困难对话 (Difficult Conversation)

## 触发条件
- 敏感话题
- 负面反馈
- 冲突沟通
- 敏感协商

## 输入
- 对话目的
- 对方预期
- 敏感程度
- 情境把握

## 输出
- 对话策略
- 内容准备
- 表达方式
- 应对反应

## Prompt模板
```
请处理困难对话。

背景：
- 对话目的：[目的]
- 对方预期：[预期]
- 敏感程度：[敏感]
- 情境把握：[情境]

请输出：
1. 对话策略
2. 内容准备
3. 开场方式
4. 表达技巧
5. 应对反应
```

## 示例
### 示例1
- 输入：绩效批评
- 输出：困难对话，具体行为、影响说明、期望明确、支持提供

## 注意事项
- 要充分准备
- 要具体事实
- 要同理对方
- 要解决问题
