# 分拆上市 (Spin-off)

## 触发条件
- 业务分立
- 估值释放
- 股东回报
- 战略聚焦

## 输入
- 分拆业务
- 资本市场
- 股权结构
- 监管要求

## 输出
- 分拆方案
- 估值分析
- 股权设计
- 上市计划

## Prompt模板
```
请设计分拆上市。

背景：
- 分拆业务：[业务]
- 资本市场：[市场]
- 股权结构：[结构]
- 监管要求：[要求]

请输出：
1. 分拆方案
2. 估值分析
3. 股权设计
4. 上市计划
5. 时间表
```

## 示例
### 示例1
- 输入：业务分拆
- 输出：分拆上市，业务独立、估值重塑、股东分配

## 注意事项
- 要战略清晰
- 要估值合理
- 要股东支持
- 要合规披露
