# 边缘计算 (Edge Computing)

## 触发条件
- 边缘部署
- 低延迟
- 数据处理
- IoT场景

## 输入
- 业务场景
- 延迟要求
- 数据量
- 覆盖范围

## 输出
- 边缘架构
- 计算分布
- 数据同步
- 运维方案

## Prompt模板
```
请设计边缘计算架构。

背景：
- 业务场景：[场景]
- 延迟要求：[要求]
- 数据量：[数据]
- 覆盖范围：[范围]

请输出：
1. 边缘架构
2. 计算分布
3. 数据同步
4. 安全策略
5. 运维方案
```

## 示例
### 示例1
- 输入：自动驾驶
- 输出：边缘计算，含车端计算、路边单元、云边协同、低延迟通信

## 注意事项
- 要低延迟
- 要本地处理
- 要云边协同
- 要安全可靠
