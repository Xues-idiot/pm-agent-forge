# 定价打包 (Pricing & Packaging)

## 触发条件
- 定价策略
- 产品包设计
- 价值分层
- 客户选择

## 输入
- 产品价值
- 客户分层
- 竞争对手
- 成本结构

## 输出
- 定价策略
- 包设计
- 层级划分
- 转化策略

## Prompt模板
```
请设计定价和打包方案。

背景：
- 产品价值：[价值]
- 客户分层：[分层]
- 竞争对手：[竞品]
- 成本结构：[成本]

请输出：
1. 定价策略
2. 产品包设计
3. 价值层级
4. 价格锚定
5. 转化路径
```

## 示例
### 示例1
- 输入：SaaS产品
- 输出：定价方案，免费版/专业版/企业版，价格锚定设计

## 注意事项
- 要客户导向
- 要价值感知
- 要简化选择
- 要持续优化
