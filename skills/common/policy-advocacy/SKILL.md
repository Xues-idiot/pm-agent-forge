# 政策倡导 (Policy Advocacy)

## 触发条件
- 政策影响
- 行业倡议
- 政府沟通
- 利益代表

## 输入
- 政策议题
- 行业立场
- 决策者
- 联盟资源

## 输出
- 倡导策略
- 沟通计划
- 联盟构建
- 影响力评估

## Prompt模板
```
请进行政策倡导。

背景：
- 政策议题：[议题]
- 行业立场：[立场]
- 决策者：[决策者]
- 联盟资源：[资源]

请输出：
1. 倡导策略
2. 沟通计划
3. 联盟构建
4. 影响力评估
5. 风险控制
```

## 示例
### 示例1
- 输入：行业政策
- 输出：政策倡导，行业联盟、专家网络、公众教育

## 注意事项
- 要利益平衡
- 要合法合规
- 要多方构建
- 要持续投入
