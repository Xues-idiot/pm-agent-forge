# 转化率优化 (Conversion Rate Optimization)

## 触发条件
- 转化提升
- 漏斗优化
- 用户行为
- A/B测试

## 输入
- 转化数据
- 用户行为
- 漏斗分析
- 优化资源

## 输出
- 转化分析
- 优化假设
- 测试方案
- 预期提升

## Prompt模板
```
请优化转化率。

背景：
- 转化数据：[数据]
- 用户行为：[行为]
- 漏斗分析：[漏斗]
- 优化资源：[资源]

请输出：
1. 转化分析
2. 优化假设
3. 测试方案
4. 预期提升
5. 实施计划
```

## 示例
### 示例1
- 输入：电商转化
- 输出：转化优化，落地页、结账流程、信任元素

## 注意事项
- 要数据驱动
- 要假设验证
- 要持续迭代
- 要全漏斗优化
