# 文化建设 (Culture Building)

## 触发条件
- 文化塑造
- 价值观
- 行为规范
- 文化传承

## 输入
- 文化愿景
- 核心价值观
- 行为标准
- 传承机制

## 输出
- 文化体系
- 行为规范
- 传播机制
- 评估机制

## Prompt模板
```
请建设文化。

背景：
- 文化愿景：[愿景]
- 核心价值观：[价值观]
- 行为标准：[标准]
- 传承机制：[传承]

请输出：
1. 文化体系
2. 行为规范
3. 传播机制
4. 激励机制
5. 评估机制
```

## 示例
### 示例1
- 输入：企业文化塑造
- 输出：文化建设，使命愿景价值观、行为准则、文化活动

## 注意事项
- 要领导带头
- 要融入工作
- 要持续强化
- 要与时俱进
