# 档案管理 (Record Management)

## 触发条件
- 档案归档
- 档案存储
- 档案检索
- 档案销毁

## 输入
- 档案类型
- 保存期限
- 归档要求
- 存储条件

## 输出
- 归档方案
- 存储策略
- 检索机制
- 销毁计划

## Prompt模板
```
请管理档案。

背景：
- 档案类型：[类型]
- 保存期限：[期限]
- 归档要求：[要求]
- 存储条件：[条件]

请输出：
1. 归档方案
2. 存储策略
3. 检索机制
4. 销毁计划
5. 合规检查
```

## 示例
### 示例1
- 输入：人事档案
- 输出：档案管理，电子化归档、长期保存、合规销毁

## 注意事项
- 要合规优先
- 要长期保存
- 要便于检索
- 要安全销毁
