# 变更评审 (Change Review)

## 触发条件
- 变更评估
- 影响分析
- 决策评审
- 变更控制

## 输入
- 变更内容
- 影响范围
- 成本收益
- 风险评估

## 输出
- 变更评估
- 影响分析
- 决策建议
- 审批意见

## Prompt模板
```
请进行变更评审。

背景：
- 变更内容：[内容]
- 影响范围：[范围]
- 成本收益：[成本]
- 风险评估：[风险]

请输出：
1. 变更描述
2. 影响分析
3. 成本收益
4. 风险评估
5. 决策建议
```

## 示例
### 示例1
- 输入：项目范围变更
- 输出：变更评审，范围影响、进度影响、成本影响、风险评估

## 注意事项
- 要全面分析
- 要权衡决策
- 要记录备案
- 要控制变更
