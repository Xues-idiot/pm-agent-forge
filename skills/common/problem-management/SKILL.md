# 问题管理 (Problem Management)

## 触发条件
- 根因分析
- 问题解决
- 知识积累
- 风险预防

## 输入
- 问题类型
- 事件数据
- 根因分析
- 资源需求

## 输出
- 问题分类
- 根因分析
- 解决方案
- 预防措施

## Prompt模板
```
请进行问题管理。

背景：
- 问题类型：[类型]
- 事件数据：[数据]
- 根因分析：[分析]
- 资源需求：[资源]

请输出：
1. 问题分类
2. 根因分析
3. 解决方案
4. 预防措施
5. 知识库更新
```

## 示例
### 示例1
- 输入：重复发生的事件
- 输出：问题管理，识别已知错误、制定解决方案、预防再发

## 注意事项
- 要根因分析
- 要主动管理
- 要知识积累
- 要持续改进
