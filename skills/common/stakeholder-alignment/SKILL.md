# 干系人对齐 (Stakeholder Alignment)

## 触发条件
- 干系人管理
- 期望管理
- 冲突解决
- 沟通协调

## 输入
- 干系人
- 利益诉求
- 影响程度
- 沟通需求

## 输出
- 分析矩阵
- 沟通策略
- 冲突解决
- 期望管理

## Prompt模板
```
请进行干系人对齐。

背景：
- 干系人：[干系人]
- 利益诉求：[诉求]
- 影响程度：[程度]
- 沟通需求：[需求]

请输出：
1. 干系人分析
2. 利益诉求
3. 影响分析
4. 沟通策略
5. 冲突解决
6. 期望管理
```

## 示例
### 示例1
- 输入：项目涉及多个部门，需要协调
- 输出：对齐方案，含干系人地图、沟通计划、冲突处理

## 注意事项
- 要主动沟通
- 要平衡利益
- 要持续维护
