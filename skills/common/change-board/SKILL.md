# 变更委员会 (Change Board)

## 触发条件
- 变更审批
- 变更决策
- 变更评估
- 变更协调

## 输入
- 变更请求
- 影响分析
- 资源需求
- 紧急程度

## 输出
- 审批决策
- 实施条件
- 回滚方案
- 后续评估

## Prompt模板
```
请组织变更委员会。

背景：
- 变更请求：[请求]
- 影响分析：[分析]
- 资源需求：[需求]
- 紧急程度：[程度]

请输出：
1. 审批决策
2. 实施条件
3. 回滚方案
4. 后续评估
5. 教训总结
```

## 示例
### 示例1
- 输入：重大变更
- 输出：变更委员会，审批通过、附带条件、跟踪验证

## 注意事项
- 要规范流程
- 要评估影响
- 要集体决策
- 要跟踪执行
