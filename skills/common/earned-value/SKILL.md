# 挣值管理 (Earned Value Management)

## 触发条件
- 进度成本
- 绩效评估
- 预测分析
- 项目控制

## 输入
- 计划值PV
- 挣值EV
- 实际成本AC
- 完工预算

## 输出
- 进度偏差
- 成本偏差
- 预测完工
- 绩效评估

## Prompt模板
```
请进行挣值分析。

背景：
- 计划值PV：[PV]
- 挣值EV：[EV]
- 实际成本AC：[AC]
- 完工预算BAC：[BAC]

请输出：
1. SPI计算
2. CPI计算
3. 进度偏差
4. 成本偏差
5. 预测分析
```

## 示例
### 示例1
- 输入：项目数据
- 输出：挣值分析，SPI=0.9、CPI=1.1、预测ETC、完工日期

## 注意事项
- 要数据准确
- 要定期计算
- 要预测分析
- 要及时纠偏
