# 商业模式画布 (Business Model Canvas)

## 触发条件
- 模式设计
- 创业验证
- 业务转型
- 创新探索

## 输入
- 客户细分
- 价值主张
- 渠道通路
- 客户关系
- 收入来源
- 核心资源
- 关键业务
- 成本结构

## 输出
- 商业模式画布
- 关键假设
- 验证计划
- 迭代方向

## Prompt模板
```
请用商业模式画布设计。

背景：
- 客户细分：[客户]
- 价值主张：[价值]
- 渠道通路：[渠道]
- 客户关系：[关系]
- 收入来源：[收入]
- 核心资源：[资源]
- 关键业务：[业务]
- 成本结构：[成本]

请输出：
1. 商业模式画布
2. 关键假设
3. 验证计划
4. 迭代方向
```

## 示例
### 示例1
- 输入：订阅制SaaS产品
- 输出：商业模式画布，含客户-价值-渠道-收入等九宫格

## 注意事项
- 要简洁清晰
- 要假设验证
- 要持续迭代
