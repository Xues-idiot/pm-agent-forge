# MVP设计 (MVP Design)

## 触发条件
- 创业验证
- 产品开发
- 快速试错
- 资源最小化

## 输入
- 核心价值
- 用户需求
- 技术能力
- 资源限制

## 输出
- MVP定义
- 核心功能
- 验证计划
- 迭代方向

## Prompt模板
```
请设计MVP方案。

背景：
- 核心价值：[价值]
- 用户需求：[需求]
- 技术能力：[能力]
- 资源限制：[限制]

请输出：
1. MVP定义
2. 核心功能
3. 非功能范围
4. 验证计划
5. 迭代路线
```

## 示例
### 示例1
- 输入：在线教育平台，需要快速验证
- 输出：MVP方案，含最小功能集、验证指标、迭代计划

## 注意事项
- 要聚焦核心
- 要快速验证
- 要持续迭代
