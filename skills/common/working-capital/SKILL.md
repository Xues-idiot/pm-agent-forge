# 营运资本 (Working Capital)

## 触发条件
- 资金效率
- 库存管理
- 应收账款
- 应付账款

## 输入
- 库存
- 应收账款
- 应付账款
- 运营周期

## 输出
- 营运资本
- 周期分析
- 优化空间
- 效率指标

## Prompt模板
```
请管理营运资本。

背景：
- 库存：[库存]
- 应收账款：[应收]
- 应付账款：[应付]
- 运营周期：[周期]

请输出：
1. 营运资本
2. 周期分析
3. 优化空间
4. 效率指标
5. 改进建议
```

## 示例
### 示例1
- 输入：制造企业
- 输出：营运资本分析，现金周期60天、库存周转30天

## 注意事项
- 要提高周转
- 要控制库存
- 要优化账期
- 要持续监控
