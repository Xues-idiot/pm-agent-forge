# 告警规则 (Alerting Rules)

## 触发条件
- 告警配置
- 异常检测
- 故障预警
- 噪声控制

## 输入
- 监控指标
- 阈值设置
- 告警级别
- 升级策略

## 输出
- 告警规则
- 阈值配置
- 告警路由
- 抑制机制

## Prompt模板
```
请配置告警规则。

背景：
- 监控指标：[指标]
- 阈值设置：[阈值]
- 告警级别：[级别]
- 升级策略：[策略]

请输出：
1. 告警规则
2. 阈值配置
3. 告警路由
4. 抑制机制
5. 响应流程
```

## 示例
### 示例1
- 输入：服务监控
- 输出：告警规则，CPU/内存/响应时间/P99延迟

## 注意事项
- 要精准告警
- 要分级处理
- 要减少噪声
- 要及时升级
