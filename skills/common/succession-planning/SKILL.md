# 继任计划 (Succession Planning)

## 触发条件
- 人才储备
- 关键岗位
- 梯队建设
- 风险应对

## 输入
- 关键岗位
- 人才盘点
- 发展进度
- 风险识别

## 输出
- 继任地图
- 候选人
- 发展计划
- 时间表

## Prompt模板
```
请制定继任计划。

背景：
- 关键岗位：[岗位]
- 人才盘点：[盘点]
- 发展进度：[进度]
- 风险识别：[风险]

请输出：
1. 继任地图
2. 候选人识别
3. 发展计划
4. 时间表
5. 风险预案
```

## 示例
### 示例1
- 输入：高管继任
- 输出：继任计划，继任者画像、内部候选人、发展计划、过渡期

## 注意事项
- 要关键岗位
- 要人才储备
- 要发展计划
- 要定期更新
