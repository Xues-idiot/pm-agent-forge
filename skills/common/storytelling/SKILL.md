# 讲故事 (Storytelling)

## 触发条件
- 故事沟通
- 演讲技巧
- 品牌传播
- 说服影响

## 输入
- 故事目的
- 目标受众
- 故事素材
- 传播场景

## 输出
- 故事结构
- 情节设计
- 表达技巧
- 应用场景

## Prompt模板
```
请用讲故事的方式沟通。

背景：
- 故事目的：[目的]
- 目标受众：[受众]
- 故事素材：[素材]
- 传播场景：[场景]

请输出：
1. 故事结构
2. 情节设计
3. 情感触发
4. 行动号召
5. 应用场景
```

## 示例
### 示例1
- 输入：创业分享
- 输出：故事讲述，创始人故事、挑战困难、坚持梦想、团队凝聚

## 注意事项
- 要有感染力
- 要情感共鸣
- 要有细节
- 要有行动号召
