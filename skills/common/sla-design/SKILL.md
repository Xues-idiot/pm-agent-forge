# SLA设计 (SLA Design)

## 触发条件
- 服务承诺
- 可靠性目标
- 成本平衡
- 客户沟通

## 输入
- 服务类型
- 可靠性目标
- 技术能力
- 客户期望

## 输出
- SLA定义
- 指标设计
- 补偿机制
- 监控方案

## Prompt模板
```
请设计SLA。

背景：
- 服务类型：[类型]
- 可靠性目标：[目标]
- 技术能力：[能力]
- 客户期望：[期望]

请输出：
1. SLA定义
2. 指标设计
3. 测量方法
4. 补偿机制
5. 监控方案
```

## 示例
### 示例1
- 输入：云数据库服务
- 输出：SLA设计，含99.95%可用性、故障响应时间、补偿机制

## 注意事项
- 要可测量
- 要可达成
- 要成本平衡
- 要持续改进
