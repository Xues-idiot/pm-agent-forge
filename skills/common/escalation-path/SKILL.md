# 升级路径 (Escalation Path)

## 触发条件
- 问题升级
- 决策流程
- 权限管理
- 快速响应

## 输入
- 问题类型
- 严重程度
- 当前处理人
- 升级时机

## 输出
- 升级路径
- 责任人
- 响应时间
- 升级标准

## Prompt模板
```
请设计升级路径。

背景：
- 问题类型：[类型]
- 严重程度：[程度]
- 当前处理人：[处理人]
- 升级时机：[时机]

请输出：
1. 升级路径
2. 各级责任人
3. 响应时间要求
4. 升级标准
5. 降级规则
```

## 示例
### 示例1
- 输入：技术支持问题
- 输出：升级路径，一线客服→技术专家→技术经理→CTO

## 注意事项
- 要明确标准
- 要及时升级
- 要信息同步
- 要闭环处理
