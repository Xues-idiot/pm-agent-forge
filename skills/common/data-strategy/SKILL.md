# 数据战略 (Data Strategy)

## 触发条件
- 数据治理
- 数据资产
- 分析能力
- 数据驱动

## 输入
- 数据现状
- 业务需求
- 技术能力
- 资源投入

## 输出
- 数据战略
- 治理框架
- 平台架构
- 应用场景

## Prompt模板
```
请制定数据战略。

背景：
- 数据现状：[现状]
- 业务需求：[需求]
- 技术能力：[能力]
- 资源投入：[资源]

请输出：
1. 数据战略目标
2. 治理框架
3. 平台架构
4. 应用场景
5. 实施路线
```

## 示例
### 示例1
- 输入：零售企业数据分散在各系统
- 输出：数据战略，含数据中台建设、指标体系统一、数据资产盘点、BI赋能

## 注意事项
- 要业务导向
- 要渐进式建设
- 要标准化先行
- 要安全合规
