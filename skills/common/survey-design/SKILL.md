# 问卷设计 (Survey Design)

## 触发条件
- 问卷调研
- 定量研究
- 满意度调查
- 反馈收集

## 输入
- 调研目标
- 目标人群
- 问题类型
- 样本量

## 输出
- 问卷结构
- 问题设计
- 样本计划
- 分析框架

## Prompt模板
```
请设计问卷。

背景：
- 调研目标：[目标]
- 目标人群：[人群]
- 问题类型：[类型]
- 样本量：[样本]

请输出：
1. 问卷结构
2. 问题设计
3. 样本计划
4. 分析框架
5. 预期产出
```

## 示例
### 示例1
- 输入：用户满意度
- 输出：问卷设计，NPS量表、满意度维度、开放式问题

## 注意事项
- 要目标明确
- 要问题简洁
- 要逻辑清晰
- 要样本代表
