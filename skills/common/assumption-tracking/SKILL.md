# 假设跟踪 (Assumption Tracking)

## 触发条件
- 项目假设
- 风险识别
- 假设验证
- 决策依据

## 输入
- 项目假设
- 验证方法
- 证据来源
- 假设状态

## 输出
- 假设清单
- 验证计划
- 风险评估
- 调整决策

## Prompt模板
```
请跟踪假设。

背景：
- 项目假设：[假设]
- 验证方法：[方法]
- 证据来源：[来源]
- 假设状态：[状态]

请输出：
1. 假设清单
2. 验证方法
3. 验证结果
4. 风险评估
5. 决策调整
```

## 示例
### 示例1
- 输入：新产品开发
- 输出：假设跟踪，用户愿付费20元、月留存50%、获客成本10元

## 注意事项
- 要明确假设
- 要验证方法
- 要及时验证
- 要基于事实
