# 服务请求 (Service Request)

## 触发条件
- 客户请求
- 流程处理
- SLA管理
- 满意度

## 输入
- 请求类型
- 请求内容
- 处理流程
- SLA目标

## 输出
- 请求分类
- 处理流程
- SLA达标
- 满意度

## Prompt模板
```
请管理服务请求。

背景：
- 请求类型：[类型]
- 请求内容：[内容]
- 处理流程：[流程]
- SLA目标：[目标]

请输出：
1. 请求分类
2. 处理流程
3. SLA达标
4. 满意度分析
5. 优化建议
```

## 示例
### 示例1
- 输入：IT服务台
- 输出：服务请求，含密码重置、权限申请、故障报修等处理流程

## 注意事项
- 要标准化
- 要SLA达标
- 要持续优化
- 要客户满意
