# 采购审批 (Procurement Approval)

## 触发条件
- 采购申请
- 预算审批
- 权限控制
- 合规管理

## 输入
- 采购内容
- 预算金额
- 采购类型
- 申请人

## 输出
- 审批流程
- 审批权限
- 审批意见
- 完成记录

## Prompt模板
```
请进行采购审批。

背景：
- 采购内容：[内容]
- 预算金额：[金额]
- 采购类型：[类型]
- 申请人：[申请人]

请输出：
1. 审批流程
2. 审批权限
3. 审批意见
4. 完成记录
5. 归档要求
```

## 示例
### 示例1
- 输入：采购申请
- 输出：采购审批，金额分级、部门审批、财务审批

## 注意事项
- 要分级审批
- 要预算控制
- 要合规透明
- 要高效便捷
