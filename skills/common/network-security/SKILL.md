# 网络安全 (Network Security)

## 触发条件
- 网络防护
- 边界安全
- 流量分析
- 攻击防御

## 输入
- 网络架构
- 威胁类型
- 防护等级
- 监控需求

## 输出
- 安全架构
- 防护策略
- 监控机制
- 响应流程

## Prompt模板
```
请保障网络安全。

背景：
- 网络架构：[架构]
- 威胁类型：[威胁]
- 防护等级：[等级]
- 监控需求：[需求]

请输出：
1. 安全架构
2. 防护策略
3. 监控机制
4. 响应流程
5. 评估标准
```

## 示例
### 示例1
- 输入：企业网络
- 输出：网络安全，防火墙、IDS/IPS、网络分段

## 注意事项
- 要分层防御
- 要纵深防护
- 要实时监控
- 要快速响应
