# 保修管理 (Warranty Management)

## 触发条件
- 保修服务
- 售后管理
- 成本预测
- 客户满意度

## 输入
- 保修条款
- 保修期
- 维修数据
- 成本预算

## 输出
- 保修策略
- 服务流程
- 成本预测
- 改进建议

## Prompt模板
```
请管理保修。

背景：
- 保修条款：[条款]
- 保修期：[期限]
- 维修数据：[数据]
- 成本预算：[预算]

请输出：
1. 保修策略
2. 服务流程
3. 成本预测
4. 改进建议
5. 监控指标
```

## 示例
### 示例1
- 输入：电子产品
- 输出：保修管理，一年整机保修、核心部件两年

## 注意事项
- 要条款清晰
- 要服务便捷
- 要成本可控
- 要数据跟踪
