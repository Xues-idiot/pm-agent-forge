# 企业社会责任 (Corporate Social Responsibility)

## 触发条件
- CSR策略
- 社区贡献
- 环境保护
- 公益慈善

## 输入
- CSR领域
- 资源预算
- 项目选择
- 利益相关方

## 输出
- CSR策略
- 项目计划
- 资源配置
- 效果评估

## Prompt模板
```
请管理企业社会责任。

背景：
- CSR领域：[领域]
- 资源预算：[预算]
- 项目选择：[选择]
- 利益相关方：[相关方]

请输出：
1. CSR策略
2. 项目计划
3. 资源配置
4. 效果评估
5. 信息披露
```

## 示例
### 示例1
- 输入：企业CSR
- 输出：企业社会责任，教育捐赠、环保项目、员工志愿者

## 注意事项
- 要战略匹配
- 要实质议题
- 要持续投入
- 要透明披露
