# 董事会管理 (Board Management)

## 触发条件
- 董事会运作
- 战略决策
- 高管任命
- 监督问责

## 输入
- 董事会结构
- 会议计划
- 议程设置
- 决策事项

## 输出
- 会议管理
- 议程设计
- 决策支持
- 监督报告

## Prompt模板
```
请管理董事会。

背景：
- 董事会结构：[结构]
- 会议计划：[计划]
- 议程设置：[设置]
- 决策事项：[事项]

请输出：
1. 会议管理
2. 议程设计
3. 决策支持
4. 监督报告
5. 披露要求
```

## 示例
### 示例1
- 输入：董事会运作
- 输出：董事会管理，年度会议、战略审议、高管评估

## 注意事项
- 要规范运作
- 要充分准备
- 要高效决策
- 要有效监督
