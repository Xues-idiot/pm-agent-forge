# 首次解决率 (First Contact Resolution)

## 触发条件
- 问题解决
- 效率指标
- 客户体验
- 成本控制

## 输入
- 问题类型
- 解决时间
- 升级率
- 客户反馈

## 输出
- FCR计算
- 影响因素
- 改进机会
- 提升策略

## Prompt模板
```
请分析首次解决率。

背景：
- 问题类型：[类型]
- 解决时间：[时间]
- 升级率：[升级]
- 客户反馈：[反馈]

请输出：
1. FCR计算
2. 影响因素
3. 改进机会
4. 提升策略
5. 监控指标
```

## 示例
### 示例1
- 输入：客服数据
- 输出：FCR分析，首次解决率75%，产品问题解决率低需改进

## 注意事项
- 要高FCR
- 要根因解决
- 要信息充分
- 要持续提升
