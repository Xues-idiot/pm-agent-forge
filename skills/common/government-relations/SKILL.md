# 政府关系 (Government Relations)

## 触发条件
- 政府沟通
- 政策影响
- 危机应对
- 业务发展

## 输入
- 政府资源
- 政策环境
- 业务需求
- 关系网络

## 输出
- 关系策略
- 沟通计划
- 合作项目
- 危机预案

## Prompt模板
```
请管理政府关系。

背景：
- 政府资源：[资源]
- 政策环境：[环境]
- 业务需求：[需求]
- 关系网络：[网络]

请输出：
1. 关系策略
2. 沟通计划
3. 合作项目
4. 危机预案
5. 评估机制
```

## 示例
### 示例1
- 输入：政企合作
- 输出：政府关系，政府来访、政策支持、项目合作

## 注意事项
- 要长期经营
- 要互惠互利
- 要合规透明
- 要专人负责
