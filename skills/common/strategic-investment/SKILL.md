# 战略投资 (Strategic Investment)

## 触发条件
- 投资布局
- 生态构建
- 协同效应
- 财务回报

## 输入
- 投资目标
- 标的筛选
- 投资方式
- 协同预期

## 输出
- 投资策略
- 标的评估
- 投资方式
- 协同计划

## Prompt模板
```
请制定战略投资策略。

背景：
- 投资目标：[目标]
- 标的筛选：[标的]
- 投资方式：[方式]
- 协同预期：[协同]

请输出：
1. 投资策略
2. 标的筛选标准
3. 投资方式
4. 协同计划
5. 退出策略
```

## 示例
### 示例1
- 输入：战略投资布局
- 输出：战略投资，被投企业筛选、战略协同、少数股权

## 注意事项
- 要战略协同
- 要长期视角
- 要风险控制
- 要持续管理
