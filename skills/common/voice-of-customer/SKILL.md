# 客户之声 (Voice of Customer)

## 触发条件
- 用户反馈
- 需求洞察
- 产品改进
- 体验优化

## 输入
- 反馈来源
- 反馈内容
- 反馈量级
- 分析需求

## 输出
- 反馈分类
- 需求洞察
- 优先级
- 改进建议

## Prompt模板
```
请分析客户之声。

背景：
- 反馈来源：[来源]
- 反馈内容：[内容]
- 反馈量级：[量级]
- 分析需求：[需求]

请输出：
1. 反馈分类
2. 关键主题
3. 需求优先级
4. 改进建议
5. 跟踪机制
```

## 示例
### 示例1
- 输入：产品反馈
- 输出：VoC分析，功能需求60%、体验问题30%、其他10%

## 注意事项
- 要多渠道收集
- 要归类分析
- 要优先级排序
- 要闭环追踪
