# IT财务管理 (IT Financial Management)

## 触发条件
- IT预算
- 成本控制
- 投资回报
- 资源优化

## 输入
- 预算需求
- 成本结构
- 投资计划
- 效益预期

## 输出
- 预算方案
- 成本控制
- 投资分析
- 效益评估

## Prompt模板
```
请管理IT财务。

背景：
- 预算需求：[需求]
- 成本结构：[结构]
- 投资计划：[投资]
- 效益预期：[效益]

请输出：
1. 预算方案
2. 成本结构
3. 投资分析
4. 效益评估
5. 优化建议
```

## 示例
### 示例1
- 输入：IT部门预算
- 输出：IT财务管理，年度预算分配、成本监控、投资回报跟踪

## 注意事项
- 要合理预算
- 要控制成本
- 要追踪效益
- 要持续优化
