# 工作场所安全 (Workplace Safety)

## 触发条件
- 场所安全
- 隐患排查
- 防护设施
- 应急准备

## 输入
- 场所类型
- 危险识别
- 安全设施
- 法规要求

## 输出
- 安全评估
- 防护方案
- 检查计划
- 应急预案

## Prompt模板
```
请保障工作场所安全。

背景：
- 场所类型：[类型]
- 危险识别：[危险]
- 安全设施：[设施]
- 法规要求：[法规]

请输出：
1. 安全评估
2. 防护方案
3. 检查计划
4. 应急预案
5. 培训机制
```

## 示例
### 示例1
- 输入：办公室安全
- 输出：工作场所安全，消防设施、电气安全、紧急出口

## 注意事项
- 要隐患排查
- 要设施完备
- 要定期检查
- 要应急演练
