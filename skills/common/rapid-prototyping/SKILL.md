# 快速原型 (Rapid Prototyping)

## 触发条件
- 原型设计
- 快速验证
- 用户测试
- 迭代改进

## 输入
- 设计概念
- 复杂程度
- 资源限制
- 验证目标

## 输出
- 原型类型
- 制作计划
- 测试方案
- 迭代计划

## Prompt模板
```
请制作快速原型。

背景：
- 设计概念：[概念]
- 复杂程度：[程度]
- 资源限制：[限制]
- 验证目标：[目标]

请输出：
1. 原型类型选择
2. 制作计划
3. 测试方案
4. 反馈收集
5. 迭代改进
```

## 示例
### 示例1
- 输入：界面设计
- 输出：快速原型，纸质原型→线框图→高保真

## 注意事项
- 要快速低成本
- 要足够验证
- 要用户测试
- 要持续迭代
