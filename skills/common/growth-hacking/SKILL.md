# 增长黑客 (Growth Hacking)

## 触发条件
- 用户增长
- 低成本获客
- 增长策略
- 数据驱动

## 输入
- 增长目标
- 当前漏斗
- 可用渠道
- 资源投入

## 输出
- 增长策略
- 实验计划
- 渠道优先级
- 预期效果

## Prompt模板
```
请制定增长策略。

背景：
- 增长目标：[目标]
- 当前漏斗：[漏斗]
- 可用渠道：[渠道]
- 资源投入：[资源]

请输出：
1. 增长策略
2. 实验计划
3. 渠道优先级
4. 预期效果
5. 监控指标
```

## 示例
### 示例1
- 输入：互联网产品
- 输出：增长黑客，病毒传播、SEO优化、内容营销

## 注意事项
- 要数据驱动
- 要快速实验
- 要低成本
- 要持续迭代
