# 收入运营 (Revenue Operations)

## 触发条件
- 收入增长
- 运营效率
- 团队协作
- 指标体系

## 输入
- 商业模式
- 收入目标
- 组织架构
- 技术能力

## 输出
- RevOps策略
- 流程优化
- 指标体系
- 工具支撑

## Prompt模板
```
请设计[公司]收入运营体系。

背景：
- 商业模式：[模式]
- 收入目标：[目标]
- 组织架构：[架构]
- 技术能力：[能力]

请输出：
1. RevOps战略
2. 销售运营
3. 营销运营
4. 客户成功
5. 数据分析
6. 指标体系
7. 工具支撑
```

## 示例
### 示例1
- 输入：B2B公司，销售和营销协作不畅
- 输出：RevOps方案，含流程打通、数据统一、指标对齐

## 注意事项
- 要跨部门协作
- 要数据驱动
- 要持续优化
