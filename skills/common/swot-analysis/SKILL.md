# SWOT分析 (SWOT Analysis)

## 触发条件
- 战略分析
- 竞争分析
- 项目评估
- 决策支持

## 输入
- 分析对象
- 市场环境
- 内部能力
- 竞争态势

## 输出
- SWOT矩阵
- 战略建议
- 机会转化
- 风险应对

## Prompt模板
```
请进行SWOT分析。

背景：
- 分析对象：[对象]
- 市场环境：[环境]
- 内部能力：[能力]
- 竞争态势：[态势]

请输出：
1. 优势(Strengths)
2. 劣势(Weaknesses)
3. 机会(Opportunities)
4. 威胁(Threats)
5. SO策略
6. WO策略
7. ST策略
8. WT策略
```

## 示例
### 示例1
- 输入：创业公司战略规划
- 输出：SWOT分析报告，含内外部因素、策略组合

## 注意事项
- 要客观全面
- 要突出重点
- 要策略可行
