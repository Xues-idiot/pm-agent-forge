# 最后一公里 (Last Mile Delivery)

## 触发条件
- 配送效率
- 客户体验
- 成本控制
- 配送优化

## 输入
- 配送区域
- 订单量
- 配送资源
- 客户位置

## 输出
- 配送方案
- 路径规划
- 成本分析
- 时效保障

## Prompt模板
```
请优化最后一公里。

背景：
- 配送区域：[区域]
- 订单量：[订单]
- 配送资源：[资源]
- 客户位置：[位置]

请输出：
1. 配送方案
2. 路径规划
3. 成本分析
4. 时效保障
5. 客户体验
```

## 示例
### 示例1
- 输入：电商配送
- 输出：最后一公里，众包配送、自提点、智能调度

## 注意事项
- 要成本控制
- 要时效保障
- 要客户便利
- 要技术赋能
