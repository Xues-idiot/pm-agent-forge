# 石川图 (Ishikawa Diagram)

## 触发条件
- 原因分析
- 问题分解
- 质量工具
- 团队协作

## 输入
- 问题陈述
- 原因类别
- 具体因素
- 协作分析

## 输出
- 石川图
- 原因分类
- 关键因素
- 行动计划

## Prompt模板
```
请使用石川图分析。

背景：
- 问题陈述：[问题]
- 原因类别：[类别]
- 具体因素：[因素]
- 协作分析：[协作]

请输出：
1. 问题定义
2. 石川图结构
3. 原因分类
4. 关键因素
5. 行动计划
```

## 示例
### 示例1
- 输入：项目延期
- 输出：石川图，从人员、方法、机器、材料、环境、管理分析原因

## 注意事项
- 要全面系统
- 要团队参与
- 要层层分解
- 要行动导向
