# 卓越运营 (Operational Excellence)

## 触发条件
- 运营效率
- 流程优化
- 持续改进
- 精益管理

## 输入
- 运营现状
- 效率目标
- 改进空间
- 资源投入

## 输出
- 优化方案
- 实施路径
- 评估指标
- 持续机制

## Prompt模板
```
请设计卓越运营方案。

背景：
- 运营现状：[现状]
- 效率目标：[目标]
- 改进空间：[空间]
- 资源投入：[资源]

请输出：
1. 运营诊断
2. 改进机会
3. 优化方案
4. 实施路径
5. 评估指标
6. 持续改进
```

## 示例
### 示例1
- 输入：客服运营，成本高效率低
- 输出：卓越运营方案，含流程优化、自动化、人员增效

## 注意事项
- 要数据驱动
- 要聚焦价值
- 要持续改进
