# 价值创新 (Value Innovation)

## 触发条件
- 价值突破
- 差异化
- 成本优化
- 新市场

## 输入
- 现有价值
- 竞争焦点
- 买方需求
- 创新可能

## 输出
- 剔除因素
- 减少因素
- 增加因素
- 创造因素

## Prompt模板
```
请进行价值创新。

背景：
- 现有价值：[现有]
- 竞争焦点：[竞争]
- 买方需求：[需求]
- 创新可能：[可能]

请输出：
1. 剔除哪些
2. 减少哪些
3. 增加哪些
4. 创造哪些
5. 战略画布
```

## 示例
### 示例1
- 输入：酒店业
- 输出：价值创新，取消奢华大堂、增加简洁房间、创造自助入住

## 注意事项
- 要顾客导向
- 要同步差异化
- 要成本优势
- 要突破竞争
