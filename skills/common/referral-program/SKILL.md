# 推荐计划 (Referral Program)

## 触发条件
- 裂变增长
- 口碑营销
- 病毒传播
- 用户获取

## 输入
- 产品特点
- 目标用户
- 激励机制
- 成本预算

## 输出
- 推荐方案
- 激励设计
- 传播路径
- 效果评估

## Prompt模板
```
请设计推荐计划。

背景：
- 产品特点：[特点]
- 目标用户：[用户]
- 激励机制：[激励]
- 成本预算：[预算]

请输出：
1. 推荐目标
2. 激励设计
3. 传播路径
4. 分享机制
5. 效果评估
```

## 示例
### 示例1
- 输入：在线教育产品，客单价高
- 输出：推荐计划，含邀请得优惠券、试听课分享、学习成果分享

## 注意事项
- 激励要有吸引力
- 分享要便捷
- 要控制成本
