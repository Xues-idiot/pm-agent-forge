# 组织重构 (Organizational Restructuring)

## 触发条件
- 组织变革
- 效率提升
- 战略调整
- 成本优化

## 输入
- 业务战略
- 组织现状
- 变革目标
- 约束条件

## 输出
- 架构方案
- 职责调整
- 人员安排
- 实施计划

## Prompt模板
```
请进行组织重构。

背景：
- 业务战略：[战略]
- 组织现状：[现状]
- 变革目标：[目标]
- 约束条件：[约束]

请输出：
1. 架构方案
2. 职责调整
3. 人员安排
4. 实施计划
5. 风险控制
```

## 示例
### 示例1
- 输入：企业转型
- 输出：组织重构，事业部制、矩阵管理、阿米巴模式

## 注意事项
- 要战略导向
- 要平稳过渡
- 要沟通充分
- 要持续优化
