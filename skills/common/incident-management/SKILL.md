# 事件管理 (Incident Management)

## 触发条件
- 故障响应
- 问题解决
- 服务恢复
- 事后复盘

## 输入
- 服务类型
- 故障级别
- 影响范围
- 响应目标

## 输出
- 响应流程
- 升级机制
- 解决方案
- 预防措施

## Prompt模板
```
请设计[系统]事件管理方案。

背景：
- 服务类型：[类型]
- 故障级别：[级别]
- 影响范围：[范围]
- 响应目标：[目标]

请输出：
1. 事件分级
2. 响应流程
3. 升级机制
4. 沟通机制
5. 问题解决
6. 事后复盘
7. 预防措施
8. 工具支撑
```

## 示例
### 示例1
- 输入：电商平台，需要提升故障恢复速度
- 输出：事件方案，含分级响应、快速恢复、复盘机制

## 注意事项
- 要快速响应
- 要透明沟通
- 要持续改进
