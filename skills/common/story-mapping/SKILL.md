# 用户故事地图 (User Story Mapping)

## 触发条件
- 需求梳理
- 优先级
-  Releases
- 团队协作

## 输入
- 用户角色
- 用户活动
- 用户任务
-  发布计划

## 输出
- 故事地图
-  发布分组
-  优先级
-  迭代规划

## Prompt模板
```
请绘制用户故事地图。

背景：
- 用户角色：[角色]
- 用户活动：[活动]
- 用户任务：[任务]
- 发布计划：[计划]

请输出：
1. 故事地图
2. 用户活动流
3. 发布分组
4. 优先级排序
5. 迭代规划
```

## 示例
### 示例1
- 输入：电商用户
- 输出：故事地图，含浏览-搜索-比较-购买-支付-物流-售后各阶段

## 注意事项
- 要用户视角
- 要端到端
- 要持续完善
- 要指导发布
