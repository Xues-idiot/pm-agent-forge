# OKR设定 (OKR Setting)

## 触发条件
- 目标设定
- 绩效考核
- 战略对齐
- 团队协同

## 输入
- 战略目标
- 团队职能
- 周期长度
- 对齐关系

## 输出
- 目标(Objectives)
- 关键结果(Key Results)
- 评分标准
- 对齐关系

## Prompt模板
```
请设定OKR。

背景：
- 战略目标：[战略]
- 团队职能：[职能]
- 周期长度：[周期]
- 对齐关系：[对齐]

请输出：
1. 目标(Objectives)
2. 关键结果(Key Results)
3. 评分标准
4. 对齐关系
5. 跟踪机制
```

## 示例
### 示例1
- 输入：产品团队
- 输出：OKR，用户增长指标、功能交付指标、质量指标

## 注意事项
- 要挑战性
- 要可衡量
- 要透明公开
- 要周期回顾
