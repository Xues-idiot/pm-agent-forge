# 完成定义 (Definition of Done)

## 触发条件
- 质量标准
- 团队共识
- 交付规范
- 质量保障

## 输入
- 业务需求
- 技术要求
- 质量目标
- 团队能力

## 输出
- DoD定义
- 检查清单
- 质量门禁
- 持续改进

## Prompt模板
```
请定义完成标准。

背景：
- 业务需求：[需求]
- 技术要求：[要求]
- 质量目标：[目标]
- 团队能力：[能力]

请输出：
1. DoD定义
2. 检查清单
3. 质量门禁
4. 角色责任
5. 持续改进
```

## 示例
### 示例1
- 输入：软件交付
- 输出：DoD定义，含代码审查、单元测试、集成测试、文档更新、部署上线

## 注意事项
- 要团队共识
- 要可验证
- 要持续更新
- 要严格执行
