# 财务报告 (Financial Reporting)

## 触发条件
- 报告编制
- 合规披露
- 决策支持
- 绩效评估

## 输入
- 财务数据
- 会计准则
- 披露要求
- 报告使用者

## 输出
- 报表编制
- 指标分析
- 趋势解读
- 建议输出

## Prompt模板
```
请编制财务报告。

背景：
- 财务数据：[数据]
- 会计准则：[准则]
- 披露要求：[要求]
- 报告使用者：[使用者]

请输出：
1. 报表编制
2. 指标分析
3. 趋势解读
4. 建议输出
5. 合规检查
```

## 示例
### 示例1
- 输入：月度报告
- 输出：财务报告，资产负债表、利润表、现金流量表

## 注意事项
- 要准确及时
- 要合规披露
- 要分析深入
- 要建议可行
