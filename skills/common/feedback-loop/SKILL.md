# 反馈闭环 (Feedback Loop)

## 触发条件
- 反馈收集
- 分析处理
- 改进实施
- 效果验证

## 输入
- 反馈来源
- 反馈类型
- 处理流程
- 响应目标

## 输出
- 收集机制
- 分析流程
- 改进方案
- 闭环验证

## Prompt模板
```
请设计反馈闭环机制。

背景：
- 反馈来源：[来源]
- 反馈类型：[类型]
- 处理流程：[流程]
- 响应目标：[目标]

请输出：
1. 反馈收集
2. 分类处理
3. 分析诊断
4. 改进实施
5. 效果验证
```

## 示例
### 示例1
- 输入：产品团队，需要收集用户反馈并改进
- 输出：闭环机制，含反馈渠道、分析优先级、迭代改进

## 注意事项
- 要及时响应
- 要分类处理
- 要闭环验证
