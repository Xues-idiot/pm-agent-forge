# 成本预测 (Cost Forecasting)

## 触发条件
- 成本预测
- 项目控制
- 决策支持
- 资源规划

## 输入
- 实际成本
- 剩余工作
- 预算基线
- 消耗模式

## 输出
- 预测模型
- 完工估算
- 风险分析
- 调整建议

## Prompt模板
```
请预测成本。

背景：
- 实际成本：[实际]
- 剩余工作：[剩余]
- 预算基线：[基线]
- 消耗模式：[模式]

请输出：
1. 预测模型
2. EAC计算
3. 完工偏差
4. 风险分析
5. 调整建议
```

## 示例
### 示例1
- 输入：项目成本数据
- 输出：成本预测，EAC 120万、偏差10万、风险可控

## 注意事项
- 要基于数据
- 要考虑风险
- 要动态更新
- 要决策支持
