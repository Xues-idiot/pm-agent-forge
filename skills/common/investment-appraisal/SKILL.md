# 投资评估 (Investment Appraisal)

## 触发条件
- 投资决策
- 项目筛选
- 资本配置
- 价值创造

## 输入
- 投资项目
- 预期收益
- 风险水平
- 资本成本

## 输出
- 评估指标
- 风险分析
- 优先级排序
- 决策建议

## Prompt模板
```
请评估投资。

背景：
- 投资项目：[项目]
- 预期收益：[收益]
- 风险水平：[风险]
- 资本成本：[成本]

请输出：
1. 评估指标
2. 风险分析
3. 优先级排序
4. 决策建议
5. 监控机制
```

## 示例
### 示例1
- 输入：项目投资
- 输出：投资评估，NPV/IRR/回收期/ROI

## 注意事项
- 要多维度
- 要风险调整
- 要资本成本
- 要持续跟踪
