# 技术战略 (Technology Strategy)

## 触发条件
- 技术规划
- 技术选型
- 技术路线
- 研发投入

## 输入
- 业务战略
- 技术现状
- 发展趋势
- 竞争环境

## 输出
- 技术战略
- 技术路线图
- 能力规划
- 投资建议

## Prompt模板
```
请制定[组织]技术战略。

背景：
- 业务战略：[战略]
- 技术现状：[现状]
- 发展趋势：[趋势]
- 竞争环境：[环境]

请输出：
1. 技术愿景
2. 技术现状评估
3. 技术趋势分析
4. 技术战略目标
5. 技术路线图
6. 核心能力建设
7. 技术架构
8. 研发投资
```

## 示例
### 示例1
- 输入：互联网公司，需要规划未来3年技术方向
- 输出：技术战略，含云原生、AI优先、数据平台

## 注意事项
- 要匹配业务
- 要前瞻布局
- 要持续演进
