# Jobs-to-be-Done

## 触发条件
- 需求理解
- 用户动机
- 产品创新
- 市场洞察

## 输入
- 用户画像
- 使用场景
- 目标动机
- 痛点需求

## 输出
- JTBD框架
- 用户目标
- 需求洞察
- 创新机会

## Prompt模板
```
请分析Jobs-to-be-Done。

背景：
- 用户画像：[画像]
- 使用场景：[场景]
- 目标动机：[动机]
- 痛点需求：[痛点]

请输出：
1. JTBD框架
2. 功能性任务
3. 情感性任务
4. 社会性任务
5. 创新机会
```

## 示例
### 示例1
- 输入：咖啡消费者
- 输出：JTBD分析，咖啡不仅是提神，还有社交、习惯、自我犒劳

## 注意事项
- 要洞察动机
- 要超越功能
- 要关注情感
- 要持续研究
