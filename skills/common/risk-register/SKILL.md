# 风险登记 (Risk Register)

## 触发条件
- 风险管理
- 风险识别
- 风险监控
- 风险应对

## 输入
- 风险来源
- 风险类型
- 影响程度
- 发生概率

## 输出
- 风险清单
- 风险评估
- 应对策略
- 监控计划

## Prompt模板
```
请管理风险登记。

背景：
- 风险来源：[来源]
- 风险类型：[类型]
- 影响程度：[影响]
- 发生概率：[概率]

请输出：
1. 风险清单
2. 风险评估
3. 应对策略
4. 监控计划
5. 触发条件
```

## 示例
### 示例1
- 输入：项目风险
- 输出：风险登记，技术风险、资源风险、外部风险、应对措施

## 注意事项
- 要持续更新
- 要定期回顾
- 要及时应对
- 要触发监控
