# 销售漏斗 (Sales Pipeline)

## 触发条件
- 销售预测
- 过程管理
- 转化分析
- 资源分配

## 输入
- 销售阶段
- 商机数量
- 转化率
- 平均周期

## 输出
- 漏斗分析
- 转化率
- 预测金额
- 优化建议

## Prompt模板
```
请分析销售漏斗。

背景：
- 销售阶段：[阶段]
- 商机数量：[数量]
- 转化率：[转化]
- 平均周期：[周期]

请输出：
1. 漏斗分析
2. 转化率
3. 预测金额
4. 优化建议
5. 行动项
```

## 示例
### 示例1
- 输入：销售漏斗
- 输出：漏斗分析，线索-商机-报价-合同-回款各阶段转化

## 注意事项
- 要阶段清晰
- 要数据准确
- 要动态跟踪
- 要预测准确
