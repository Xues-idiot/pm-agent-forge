# 审计追踪 (Audit Trails)

## 触发条件
- 审计需求
- 操作记录
- 责任追溯
- 安全监控

## 输入
- 审计范围
- 记录要求
- 保存期限
- 访问需求

## 输出
- 审计策略
- 记录机制
- 分析报告
- 预警机制

## Prompt模板
```
请管理审计追踪。

背景：
- 审计范围：[范围]
- 记录要求：[要求]
- 保存期限：[期限]
- 访问需求：[需求]

请输出：
1. 审计策略
2. 记录机制
3. 分析报告
4. 预警机制
5. 归档策略
```

## 示例
### 示例1
- 输入：金融系统
- 输出：审计追踪，全操作记录、异常行为检测、合规审计

## 注意事项
- 要全面记录
- 要不可篡改
- 要便于分析
- 要长期保存
