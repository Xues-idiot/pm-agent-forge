# 领导力发展 (Leadership Development)

## 触发条件
- 人才培养
- 梯队建设
- 能力提升
- 继任计划

## 输入
- 领导力模型
- 人才盘点
- 发展资源
- 业务需求

## 输出
- 发展计划
- 培训方案
- 实践机会
- 评估机制

## Prompt模板
```
请发展领导力。

背景：
- 领导力模型：[模型]
- 人才盘点：[盘点]
- 发展资源：[资源]
- 业务需求：[需求]

请输出：
1. 发展计划
2. 培训方案
3. 实践机会
4. 评估机制
5. 晋升通道
```

## 示例
### 示例1
- 输入：高潜人才
- 输出：领导力发展，教练辅导、轮岗锻炼、挑战性任务

## 注意事项
- 要个性化
- 要实践导向
- 要持续反馈
- 要业务挂钩
