# 质量门禁 (Quality Gate)

## 触发条件
- 质量控制
- 发布标准
- 检查点
- 风险控制

## 输入
- 质量目标
- 流程阶段
- 检查标准
- 风险点

## 输出
- 质量门禁
- 检查标准
- 通过准则
- 责任人

## Prompt模板
```
请设计质量门禁。

背景：
- 质量目标：[目标]
- 流程阶段：[阶段]
- 检查标准：[标准]
- 风险点：[风险]

请输出：
1. 质量门禁
2. 各阶段标准
3. 通过准则
4. 检查清单
5. 责任人
```

## 示例
### 示例1
- 输入：软件开发，需要质量门禁
- 输出：门禁方案，含需求评审、代码审查、测试验收

## 注意事项
- 要标准明确
- 要严格执行
- 要持续改进
