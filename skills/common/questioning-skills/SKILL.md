# 提问技巧 (Questioning Skills)

## 触发条件
- 提问沟通
- 信息收集
- 启发思考
- 问题解决

## 输入
- 提问目的
- 提问对象
- 问题类型
- 沟通情境

## 输出
- 问题设计
- 提问方法
- 倾听回应
- 信息收集

## Prompt模板
```
请提升提问技巧。

背景：
- 提问目的：[目的]
- 提问对象：[对象]
- 问题类型：[类型]
- 沟通情境：[情境]

请输出：
1. 问题设计
2. 提问方法
3. 开放式问题
4. 封闭式问题
5. 追问技巧
```

## 示例
### 示例1
- 输入：需求调研
- 输出：提问技巧，开放式探索、5Why追问、假设验证

## 注意事项
- 要目的明确
- 要开放式
- 要善于追问
- 要倾听回应
