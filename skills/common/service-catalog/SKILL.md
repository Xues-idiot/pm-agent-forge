# 服务目录 (Service Catalog)

## 触发条件
- 服务定义
- 服务标准化
- 服务展示
- 服务管理

## 输入
- 服务类型
- 服务范围
- 客户需求
- 标准化程度

## 输出
- 服务目录
- 服务标准
- 申请流程
- 评估机制

## Prompt模板
```
请设计服务目录。

背景：
- 服务类型：[类型]
- 服务范围：[范围]
- 客户需求：[需求]
- 标准化程度：[程度]

请输出：
1. 服务目录
2. 服务标准
3. 服务流程
4. 申请方式
5. 评估指标
```

## 示例
### 示例1
- 输入：IT服务部门，需要对内部客户提供服务
- 输出：服务目录，含服务项、标准、流程、SLA

## 注意事项
- 要客户导向
- 要标准化
- 要持续更新
