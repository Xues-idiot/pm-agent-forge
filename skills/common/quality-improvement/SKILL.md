# 质量改进 (Quality Improvement)

## 触发条件
- 质量提升
- 缺陷减少
- 流程优化
- 成本降低

## 输入
- 质量问题
- 根本原因
- 改进目标
- 资源投入

## 输出
- 改进方案
- 实施计划
- 效果验证
- 标准化

## Prompt模板
```
请进行质量改进。

背景：
- 质量问题：[问题]
- 根本原因：[原因]
- 改进目标：[目标]
- 资源投入：[资源]

请输出：
1. 改进方案
2. 实施计划
3. 效果验证
4. 标准化
5. 经验总结
```

## 示例
### 示例1
- 输入：质量问题
- 输出：质量改进，六西格玛、PDCA循环、精益改善

## 注意事项
- 要数据说话
- 要团队参与
- 要持之以恒
- 要标准化固化
