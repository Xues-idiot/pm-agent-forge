# 竞争定位 (Competitive Positioning)

## 触发条件
- 差异化
- 市场定位
- 竞争优势
- 品牌策略

## 输入
- 竞争对手
- 市场细分
- 自身优势
- 客户认知

## 输出
- 定位策略
- 差异化要素
- 品牌主张
- 传播策略

## Prompt模板
```
请制定竞争定位。

背景：
- 竞争对手：[对手]
- 市场细分：[细分]
- 自身优势：[优势]
- 客户认知：[认知]

请输出：
1. 市场定位
2. 差异化策略
3. 价值主张
4. 品牌主张
5. 传播策略
```

## 示例
### 示例1
- 输入：智能手机
- 输出：定位，"拍照最强的旗舰机"，摄像+AI+设计

## 注意事项
- 要差异化
- 要客户认可
- 要可执行
- 要持续强化
