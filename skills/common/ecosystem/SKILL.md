# 生态系统 (Ecosystem)

## 触发条件
- 生态建设
- 合作伙伴
- 价值网络
- 共同进化

## 输入
- 核心企业
- 合作伙伴
- 价值流动
- 利益分配

## 输出
- 生态架构
- 合作伙伴
- 价值创造
- 治理机制

## Prompt模板
```
请构建生态系统。

背景：
- 核心企业：[核心]
- 合作伙伴：[伙伴]
- 价值流动：[流动]
- 利益分配：[分配]

请输出：
1. 生态架构
2. 合作伙伴定位
3. 价值创造机制
4. 利益分配机制
5. 治理机制
```

## 示例
### 示例1
- 输入：手机生态
- 输出：生态系统，手机厂商、开发者、运营商、配件商、用户

## 注意事项
- 要开放共赢
- 要价值创造
- 要可持续
- 要治理完善
