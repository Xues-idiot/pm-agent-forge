# 成熟度模型 (Maturity Model)

## 触发条件
- 能力成熟度
- 发展阶段
- 改进路径
- 目标设定

## 输入
- 评估维度
- 当前级别
- 目标级别
- 改进资源

## 输出
- 成熟度评估
- 差距分析
- 改进计划
- 升级路径

## Prompt模板
```
请评估成熟度。

背景：
- 评估维度：[维度]
- 当前级别：[当前]
- 目标级别：[目标]
- 改进资源：[资源]

请输出：
1. 成熟度评估
2. 差距分析
3. 改进计划
4. 升级路径
5. 关键里程碑
```

## 示例
### 示例1
- 输入：CMMI评估
- 输出：成熟度模型，初始级、管理级、定义级、定量管理级、优化级

## 注意事项
- 要循序渐进
- 要证据支撑
- 要持续改进
- 要结合实际
