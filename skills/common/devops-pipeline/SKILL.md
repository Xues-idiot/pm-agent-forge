# DevOps流水线 (DevOps Pipeline)

## 触发条件
- 持续集成
- 持续部署
- 自动化测试
- 快速交付

## 输入
- 技术栈
- 团队规模
- 交付需求
- 质量要求

## 输出
- 流水线设计
- 自动化方案
- 质量门禁
- 部署策略

## Prompt模板
```
请设计DevOps流水线。

背景：
- 技术栈：[栈]
- 团队规模：[规模]
- 交付需求：[需求]
- 质量要求：[要求]

请输出：
1. 流水线设计
2. 自动化范围
3. 质量门禁
4. 部署策略
5. 监控反馈
```

## 示例
### 示例1
- 输入：微服务项目
- 输出：流水线，含代码检查→单元测试→集成测试→镜像构建→灰度发布

## 注意事项
- 要渐进式推进
- 要工具链集成
- 要自动化为主
- 要持续改进
