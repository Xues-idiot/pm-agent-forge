#规模化敏捷 (Scaled Agile)

## 触发条件
- 大规模敏捷
- 多团队协作
- 跨组织协调
- 敏捷规模化

## 输入
- 团队数量
- 依赖关系
- 协调机制
- 治理模式

## 输出
- 规模化框架
- 团队协调
- 价值流
- 监控机制

## Prompt模板
```
请实施规模化敏捷。

背景：
- 团队数量：[数量]
- 依赖关系：[依赖]
- 协调机制：[机制]
- 治理模式：[模式]

请输出：
1. 规模化框架
2. 团队协调
3. 价值流
4. 监控机制
5. 改进计划
```

## 示例
### 示例1
- 输入：大型组织
- 输出：规模化敏捷，SAFe/LeSS框架、PI Planning

## 注意事项
- 要框架适配
- 要团队自治
- 要协调机制
- 要持续改进
