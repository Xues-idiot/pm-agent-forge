# 并购重组 (Merger & Acquisition)

## 触发条件
- 战略并购
- 股权收购
- 资产重组
- 业务整合

## 输入
- 标的资产
- 交易结构
- 估值区间
- 整合计划

## 输出
- 尽职调查
- 估值分析
- 交易方案
- 整合路径

## Prompt模板
```
请进行并购重组。

背景：
- 标的资产：[标的]
- 交易结构：[结构]
- 估值区间：[估值]
- 整合计划：[计划]

请输出：
1. 尽职调查
2. 估值分析
3. 交易方案
4. 整合路径
5. 风险控制
```

## 示例
### 示例1
- 输入：横向并购
- 输出：尽职调查，市场份额、协同效应、整合成本

## 注意事项
- 要尽职调查
- 要合理估值
- 要风险控制
- 要整合规划
