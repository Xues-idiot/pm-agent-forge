# 项目健康度 (Project Health)

## 触发条件
- 项目状态
- 风险预警
- 整体评估
- 决策支持

## 输入
- 进度数据
- 成本数据
- 质量数据
- 风险状态

## 输出
- 健康度评估
- 红黄绿状态
- 风险预警
- 改进建议

## Prompt模板
```
请评估项目健康度。

背景：
- 进度数据：[进度]
- 成本数据：[成本]
- 质量数据：[质量]
- 风险状态：[风险]

请输出：
1. 健康度模型
2. 进度健康
3. 成本健康
4. 质量健康
5. 综合评估
```

## 示例
### 示例1
- 输入：项目数据
- 输出：健康度评估，绿色-进展正常、黄色-需要关注、红色-需要干预

## 注意事项
- 要多维度
- 要客观真实
- 要及时预警
- 要果断行动
