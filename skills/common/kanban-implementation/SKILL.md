# 看板实施 (Kanban Implementation)

## 触发条件
- 流程可视化
- 限制在制品
- 流动优化
- 持续改进

## 输入
- 现有流程
- 团队构成
- 改进目标
- 可视化需求

## 输出
- 看板设计
- 流程映射
- WIP限制
- 改进计划

## Prompt模板
```
请实施看板。

背景：
- 现有流程：[流程]
- 团队构成：[构成]
- 改进目标：[目标]
- 可视化需求：[需求]

请输出：
1. 看板设计
2. 流程映射
3. WIP限制
4. 改进计划
5. 评估指标
```

## 示例
### 示例1
- 输入：服务团队
- 输出：看板实施，需求板、开发板、发布板

## 注意事项
- 要可视化
- 要限制WIP
- 要流动顺畅
- 要持续改进
