# 合规管理 (Regulatory Compliance)

## 触发条件
- 合规要求
- 法规变化
- 审计准备
- 风险防控

## 输入
- 业务范围
- 适用法规
- 违规风险
- 整改需求

## 输出
- 合规框架
- 差距分析
- 整改计划
- 监控机制

## Prompt模板
```
请管理合规风险。

背景：
- 业务范围：[范围]
- 适用法规：[法规]
- 违规风险：[风险]
- 整改需求：[需求]

请输出：
1. 合规框架
2. 法规要求
3. 差距分析
4. 整改计划
5. 监控机制
```

## 示例
### 示例1
- 输入：金融科技公司
- 输出：合规管理，含数据安全、隐私保护、反洗钱、监管报送

## 注意事项
- 要了解法规
- 要主动合规
- 要持续监控
- 要培训到位
