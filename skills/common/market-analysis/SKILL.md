# 市场分析 (Market Analysis)

## 触发条件
- 市场研究
- 机会识别
- 竞争分析
- 趋势判断

## 输入
- 市场范围
- 研究目的
- 数据来源
- 分析深度

## 输出
- 市场规模
- 趋势分析
- 竞争格局
- 机会建议

## Prompt模板
```
请进行[市场]分析。

背景：
- 市场范围：[范围]
- 研究目的：[目的]
- 数据来源：[来源]
- 分析深度：[深度]

请输出：
1. 市场规模
2. 市场结构
3. 增长趋势
4. 竞争格局
5. 客户需求
6. 机会识别
7. 风险分析
```

## 示例
### 示例1
- 输入：智能家居市场，需要判断机会
- 输出：分析报告，含规模、增速、竞争、机会

## 注意事项
- 要数据支撑
- 要逻辑清晰
- 要有洞察
