# 事件驱动架构 (Event Driven Architecture)

## 触发条件
- 异步通信
- 松耦合
- 实时响应
- 可扩展性

## 输入
- 业务事件
- 消费者
- 发布订阅
- 可靠性

## 输出
- 事件模型
- 拓扑设计
- 可靠性设计
- 监控机制

## Prompt模板
```
请设计事件驱动架构。

背景：
- 业务事件：[事件]
- 消费者：[消费者]
- 发布订阅：[订阅]
- 可靠性：[可靠性]

请输出：
1. 事件模型
2. 拓扑设计
3. 可靠性设计
4. 监控机制
5. 故障处理
```

## 示例
### 示例1
- 输入：实时系统
- 输出：事件驱动，消息队列、事件溯源、CQRS

## 注意事项
- 要松耦合
- 要最终一致
- 要幂等设计
- 要监控完备
