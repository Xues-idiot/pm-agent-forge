# 产品路线图 (Product Roadmap)

## 触发条件
- 产品规划
- 迭代计划
- 版本管理
- 资源协调

## 输入
- 产品愿景
- 市场需求
- 技术约束
- 资源情况

## 输出
- 路线图
- 版本规划
- 里程碑
- 评估机制

## Prompt模板
```
请制定产品路线图。

背景：
- 产品愿景：[愿景]
- 市场需求：[需求]
- 技术约束：[约束]
- 资源情况：[资源]

请输出：
1. 产品愿景
2. 战略主题
3. 路线图
4. 版本规划
5. 里程碑
6. 评估调整
```

## 示例
### 示例1
- 输入：移动App，需要规划年度版本
- 输出：路线图，含Q1-Q4主题、核心功能、迭代计划

## 注意事项
- 要愿景驱动
- 要灵活调整
- 要沟通清晰
