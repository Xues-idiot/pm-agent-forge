# 同理心 (Empathy)

## 触发条件
- 理解他人
- 共情沟通
- 客户理解
- 团队协作

## 输入
- 同理对象
- 情境理解
- 情绪感知
- 沟通目的

## 输出
- 同理表达
- 理解确认
- 共情回应
- 行动建议

## Prompt模板
```
请培养同理心。

背景：
- 同理对象：[对象]
- 情境理解：[情境]
- 情绪感知：[情绪]
- 沟通目的：[目的]

请输出：
1. 情境理解
2. 情绪感知
3. 同理表达
4. 共情回应
5. 行动建议
```

## 示例
### 示例1
- 输入：客户投诉
- 输出：同理心表达，理解情绪、确认感受、表明理解、解决问题

## 注意事项
- 要换位思考
- 要感知情绪
- 要表达理解
- 要具体行动
