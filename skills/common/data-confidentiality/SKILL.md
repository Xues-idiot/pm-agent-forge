# 数据保密处理 (Data Confidentiality Processing)

## 触发条件
- 数据包含敏感信息
- 报告需要对外分享
- 涉及个人信息
- 合规审查要求

## 输入
- 原始数据
- 敏感字段清单
- 保密级别要求
- 使用场景

## 输出
- 脱敏后数据
- 脱敏规则说明
- 保留字段映射
- 合规说明

## Prompt模板
```
请进行数据保密处理。

背景：
- 原始数据：[数据内容]
- 敏感字段：[字段清单]
- 保密级别：[级别]
- 使用场景：[场景]

请输出：
1. 脱敏后数据
2. 脱敏规则说明
3. 字段映射表
4. 合规声明
```

## 示例
### 示例1
- 输入：用户手机号
- 输出：138****5678

## 注意事项
- 脱敏不可逆
- 保留分析价值
- 符合法规
- 全流程记录
