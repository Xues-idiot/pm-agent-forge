# 成本削减 (Cost Reduction)

## 触发条件
- 成本控制
- 效率提升
- 预算优化
- 持续改进

## 输入
- 成本结构
- 削减目标
- 优先级
- 实施能力

## 输出
- 削减计划
- 优先级
- 实施路径
- 预期效果

## Prompt模板
```
请制定成本削减计划。

背景：
- 成本结构：[结构]
- 削减目标：[目标]
- 优先级：[优先级]
- 实施能力：[能力]

请输出：
1. 成本分析
2. 削减机会
3. 优先级
4. 实施路径
5. 预期效果
```

## 示例
### 示例1
- 输入：运营成本
- 输出：成本削减，外包非核心、流程简化、采购议价

## 注意事项
- 要避免牺牲价值
- 要分阶段
- 要持续跟踪
- 要激励参与
