# 沟通计划 (Communication Plan)

## 触发条件
- 沟通规划
- 干系人沟通
- 信息传递
- 反馈收集

## 输入
- 沟通目标
- 干系人
- 沟通内容
- 沟通渠道

## 输出
- 沟通策略
- 沟通计划
- 信息内容
- 反馈机制

## Prompt模板
```
请制定沟通计划。

背景：
- 沟通目标：[目标]
- 干系人：[干系人]
- 沟通内容：[内容]
- 沟通渠道：[渠道]

请输出：
1. 沟通策略
2. 沟通计划
3. 信息内容
4. 反馈机制
5. 评估改进
```

## 示例
### 示例1
- 输入：项目沟通
- 输出：沟通计划，周报、双周会、月度汇报、即时群

## 注意事项
- 要计划明确
- 要渠道合适
- 要信息清晰
- 要反馈及时
