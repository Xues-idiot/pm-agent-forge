# 蓝海战略 (Blue Ocean Strategy)

## 触发条件
- 新市场
- 差异化
- 价值创新
- 竞争规避

## 输入
- 现有市场
- 竞争要素
- 客户痛点
- 创新可能

## 输出
- 剔除因素
- 减少因素
- 增加因素
- 创造因素
- 战略画布

## Prompt模板
```
请制定蓝海战略。

背景：
- 现有市场：[市场]
- 竞争要素：[要素]
- 客户痛点：[痛点]
- 创新可能：[创新]

请输出：
1. 战略画布
2. 剔除哪些
3. 减少哪些
4. 增加哪些
5. 创造哪些
```

## 示例
### 示例1
- 输入：在线教育
- 输出：蓝海战略，剔除名师限制、增加AI互动、创造沉浸学习

## 注意事项
- 要价值创新
- 要同步差异化
- 要低成本
- 要易实施
