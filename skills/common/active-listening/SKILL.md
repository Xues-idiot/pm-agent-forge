# 积极倾听 (Active Listening)

## 触发条件
- 倾听沟通
- 理解确认
- 共情沟通
- 信息收集

## 输入
- 倾听目的
- 对方表达
- 背景信息
- 反馈方式

## 输出
- 倾听技巧
- 理解确认
- 共情回应
- 信息总结

## Prompt模板
```
请提升倾听技巧。

背景：
- 倾听目的：[目的]
- 对方表达：[表达]
- 背景信息：[背景]
- 反馈方式：[反馈]

请输出：
1. 倾听准备
2. 倾听技巧
3. 理解确认
4. 共情回应
5. 信息总结
```

## 示例
### 示例1
- 输入：员工反馈
- 输出：积极倾听，全神贯注、确认理解、共情回应、总结确认

## 注意事项
- 要全神贯注
- 要不打断
- 要确认理解
- 要共情回应
