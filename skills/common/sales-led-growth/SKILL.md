# 销售驱动增长 (Sales-led Growth)

## 触发条件
- 销售策略
- 渠道建设
- 团队管理
- 业绩增长

## 输入
- 产品特点
- 目标市场
- 销售能力
- 增长目标

## 输出
- 销售策略
- 渠道建设
- 团队管理
- 激励机制

## Prompt模板
```
请实施销售驱动增长。

背景：
- 产品特点：[特点]
- 目标市场：[市场]
- 销售能力：[能力]
- 增长目标：[目标]

请输出：
1. 销售策略
2. 渠道策略
3. 团队建设
4. 激励机制
5. 指标体系
```

## 示例
### 示例1
- 输入：企业软件
- 输出：SLG策略，SDR→AE→CS，客户分级，业绩提成

## 注意事项
- 要线索为王
- 要流程标准化
- 要团队专业
- 要持续培训
