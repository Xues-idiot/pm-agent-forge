# 服务台 (Service Desk)

## 触发条件
- 服务请求
- 问题咨询
- 故障报修
- 用户支持

## 输入
- 请求类型
- 用户信息
- 问题描述
- 紧急程度

## 输出
- 服务响应
- 处理分配
- 处理进度
- 满意度

## Prompt模板
```
请运营服务台。

背景：
- 请求类型：[类型]
- 用户信息：[信息]
- 问题描述：[描述]
- 紧急程度：[程度]

请输出：
1. 服务响应
2. 处理分配
3. 处理进度
4. 满意度
5. 改进建议
```

## 示例
### 示例1
- 输入：IT服务台
- 输出：服务台，一线支持、工单管理、满意度调查

## 注意事项
- 要快速响应
- 要专业服务
- 要持续改进
- 要数据驱动
