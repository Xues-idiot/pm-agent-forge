# 每日站会 (Daily Standup)

## 触发条件
- 团队同步
- 问题暴露
- 进度跟踪
- 协作沟通

## 输入
- 团队成员
- 任务状态
- 阻塞问题
- 计划安排

## 输出
- 进度同步
- 问题识别
- 协调安排
- 升级事项

## Prompt模板
```
请组织每日站会。

背景：
- 团队成员：[成员]
- 任务状态：[状态]
- 阻塞问题：[阻塞]
- 计划安排：[计划]

请输出：
1. 进度同步
2. 问题识别
3. 协调安排
4. 升级事项
5. 后续跟踪
```

## 示例
### 示例1
- 输入：Scrum团队
- 输出：站会内容，昨日完成、今日计划、阻塞问题、需协调事项

## 注意事项
- 要简短高效
- 要站立形式
- 要问题不过夜
- 要记录升级
