# 商业模式设计 (Business Model Design)

## 触发条件
- 模式创新
- 价值创造
- 收入来源
- 成本结构

## 输入
- 客户需求
- 价值主张
- 能力资源
- 利润逻辑

## 输出
- 商业模式画布
- 价值链
- 盈利模型
- 风险控制

## Prompt模板
```
请设计[业务]商业模式。

背景：
- 客户需求：[需求]
- 价值主张：[主张]
- 能力资源：[能力]
- 利润逻辑：[逻辑]

请输出：
1. 商业模式画布
2. 客户细分
3. 价值主张
4. 渠道触达
5. 客户关系
6. 收入来源
7. 核心资源
8. 关键活动
9. 成本结构
```

## 示例
### 示例1
- 输入：共享经济平台，需要设计双边商业模式
- 输出：平台模式，含撮合服务、交易抽佣、广告变现

## 注意事项
- 要价值创造
- 要可持续盈利
- 要风险可控
