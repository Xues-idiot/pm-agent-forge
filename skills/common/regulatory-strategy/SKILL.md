# 监管策略 (Regulatory Strategy)

## 触发条件
- 监管合规
- 市场准入
- 政策影响
- 战略决策

## 输入
- 监管环境
- 行业政策
- 竞争态势
- 商业目标

## 输出
- 监管分析
- 合规路径
- 风险评估
- 应对策略

## Prompt模板
```
请制定监管策略。

背景：
- 监管环境：[环境]
- 行业政策：[政策]
- 竞争态势：[态势]
- 商业目标：[目标]

请输出：
1. 监管分析
2. 合规路径
3. 风险评估
4. 应对策略
5. 沟通计划
```

## 示例
### 示例1
- 输入：医药行业
- 输出：监管策略，药品注册、GMP合规、监管沟通

## 注意事项
- 要政策敏感
- 要前瞻布局
- 要主动沟通
- 要持续跟踪
