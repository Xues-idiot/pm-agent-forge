# 合规审计 (Compliance Audit)

## 触发条件
- 合规检查
- 法规要求
- 内部控制
- 外部审计

## 输入
- 法规范围
- 审计标准
- 业务系统
- 历史问题

## 输出
- 审计计划
- 检查清单
- 问题记录
- 整改建议

## Prompt模板
```
请进行合规审计。

背景：
- 法规范围：[范围]
- 审计标准：[标准]
- 业务系统：[系统]
- 历史问题：[历史]

请输出：
1. 审计计划
2. 检查清单
3. 问题记录
4. 整改建议
5. 跟踪验证
```

## 示例
### 示例1
- 输入：数据合规审计
- 输出：合规审计，GDPR/CCPA/SOC2检查、问题整改

## 注意事项
- 要全面覆盖
- 要客观公正
- 要证据充分
- 要持续跟踪
