# 品牌管理 (Brand Management)

## 触发条件
- 品牌定位
- 品牌建设
- 品牌价值
- 品牌保护

## 输入
- 品牌现状
- 目标受众
- 品牌定位
- 竞争差异

## 输出
- 品牌策略
- 品牌识别
- 传播规划
- 评估体系

## Prompt模板
```
请管理品牌。

背景：
- 品牌现状：[现状]
- 目标受众：[受众]
- 品牌定位：[定位]
- 竞争差异：[差异]

请输出：
1. 品牌策略
2. 品牌识别
3. 传播规划
4. 评估体系
5. 保护机制
```

## 示例
### 示例1
- 输入：消费品品牌
- 输出：品牌管理，品牌定位、品牌故事、品牌体验

## 注意事项
- 要一致性
- 要差异化
- 要持续投入
- 要动态调整
