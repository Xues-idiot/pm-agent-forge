# 隐私管理 (Privacy Management)

## 触发条件
- 数据隐私
- 合规要求
- 用户权利
- 风险防控

## 输入
- 数据类型
- 隐私法规
- 用户场景
- 保护措施

## 输出
- 隐私策略
- 合规框架
- 用户权利
- 风险评估

## Prompt模板
```
请管理数据隐私。

背景：
- 数据类型：[类型]
- 隐私法规：[法规]
- 用户场景：[场景]
- 保护措施：[措施]

请输出：
1. 隐私策略
2. 合规框架
3. 用户权利保障
4. 技术措施
5. 风险评估
```

## 示例
### 示例1
- 输入：电商平台
- 输出：隐私管理，含用户知情同意、数据加密、权限控制、删除权保障

## 注意事项
- 要法规合规
- 要用户透明
- 要数据最小化
- 要持续审计
