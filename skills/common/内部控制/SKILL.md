# 内部控制 (Internal Control)

## 触发条件
- 内部控制
- 流程规范
- 风险防范
- 合规要求

## 输入
- 控制目标
- 业务流程
- 风险点
- 现有控制

## 输出
- 控制框架
- 控制措施
- 职责分离
- 监督机制

## Prompt模板
```
请设计内部控制。

背景：
- 控制目标：[目标]
- 业务流程：[流程]
- 风险点：[风险]
- 现有控制：[控制]

请输出：
1. 控制框架
2. 控制措施
3. 职责分离
4. 监督机制
5. 评估机制
```

## 示例
### 示例1
- 输入：财务流程
- 输出：内部控制，审批流程、四眼原则、定期审计

## 注意事项
- 要流程规范
- 要职责分离
- 要监督执行
- 要持续改进
