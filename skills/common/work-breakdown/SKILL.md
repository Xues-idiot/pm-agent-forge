# 工作分解结构 (Work Breakdown Structure)

## 触发条件
- 范围定义
- 任务分解
- 责任分配
- 进度规划

## 输入
- 项目范围
- 交付物
- 团队能力
- 分解标准

## 输出
- WBS结构
- 任务清单
- 责任矩阵
- 层级关系

## Prompt模板
```
请创建工作分解结构。

背景：
- 项目范围：[范围]
- 交付物：[交付]
- 团队能力：[能力]
- 分解标准：[标准]

请输出：
1. WBS结构
2. 任务清单
3. 责任矩阵
4. 层级关系
5. 编码体系
```

## 示例
### 示例1
- 输入：软件开发项目
- 输出：WBS分解，一级-阶段、二级-模块、三级-任务

## 注意事项
- 要100%原则
- 要层级清晰
- 要责任明确
- 要可管理
