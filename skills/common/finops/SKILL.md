# FinOps管理 (FinOps)

## 触发条件
- 云成本
- 成本优化
- 资源效率
- 预算控制

## 输入
- 云资源
- 账单数据
- 成本目标
- 团队规模

## 输出
- 成本模型
- 分摊机制
- 优化建议
- 监控告警

## Prompt模板
```
请管理云成本。

背景：
- 云资源：[资源]
- 账单数据：[账单]
- 成本目标：[目标]
- 团队规模：[规模]

请输出：
1. 成本模型
2. 分摊机制
3. 优化建议
4. 监控告警
5. 治理流程
```

## 示例
### 示例1
- 输入：多团队云环境
- 输出：FinOps管理，含资源打标、成本分摊、Reserved实例、优化建议

## 注意事项
- 要全员意识
- 要持续优化
- 要技术驱动
- 要右移左移
