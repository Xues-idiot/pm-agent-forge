# IPO准备 (IPO Preparation)

## 触发条件
- 上市准备
- 财务规范
- 公司治理
- 投资者关系

## 输入
- 上市目标
- 财务状况
- 公司治理
- 合规要求

## 输出
- IPO策略
- 财务规范
- 治理完善
- 投资者关系

## Prompt模板
```
请准备IPO。

背景：
- 上市目标：[目标]
- 财务状况：[财务]
- 公司治理：[治理]
- 合规要求：[合规]

请输出：
1. IPO策略
2. 财务规范
3. 治理完善
4. 投资者关系
5. 时间规划
```

## 示例
### 示例1
- 输入：创业公司上市
- 输出：IPO准备，财务审计、治理规范、券商选择

## 注意事项
- 要提前规划
- 要规范运营
- 要业绩支撑
- 要风险披露
