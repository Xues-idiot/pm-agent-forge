# 超速增长 (Hypergrowth)

## 触发条件
- 快速增长
- 规模扩张
- 组织变革
- 挑战应对

## 输入
- 增长目标
- 现有基础
- 市场机会
- 竞争态势

## 输出
- 超速增长策略
- 组织扩张
- 流程变革
- 风险控制

## Prompt模板
```
请实现超速增长。

背景：
- 增长目标：[目标]
- 现有基础：[基础]
- 市场机会：[机会]
- 竞争态势：[竞争]

请输出：
1. 超速增长策略
2. 组织扩张
3. 流程变革
4. 文化建设
5. 风险控制
```

## 示例
### 示例1
- 输入：高速增长创业公司
- 输出：超速增长，团队每月扩招、流程敏捷化、文化开放透明

## 注意事项
- 要快但稳健
- 要人才先行
- 要流程敏捷
- 要文化传承
