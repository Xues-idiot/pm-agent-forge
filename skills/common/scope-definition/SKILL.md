# 范围定义 (Scope Definition)

## 触发条件
- 项目范围
- 边界定义
- 交付物
- 排除项

## 输入
- 项目目标
- 干系人期望
- 约束条件
- 假设前提

## 输出
- 范围说明书
- 交付物清单
- 验收标准
- 范围边界

## Prompt模板
```
请定义项目范围。

背景：
- 项目目标：[目标]
- 干系人期望：[期望]
- 约束条件：[约束]
- 假设前提：[假设]

请输出：
1. 范围说明书
2. 交付物清单
3. 验收标准
4. 范围边界
5. 除外责任
```

## 示例
### 示例1
- 输入：项目启动
- 输出：范围定义，包含/不包含/交付物/验收标准

## 注意事项
- 要清晰边界
- 要干系人确认
- 要可交付
- 要可控
