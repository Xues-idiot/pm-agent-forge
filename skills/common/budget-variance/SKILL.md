# 预算差异 (Budget Variance)

## 触发条件
- 预算控制
- 差异分析
- 业绩评估
- 纠偏措施

## 输入
- 预算金额
- 实际金额
- 差异分类
- 责任归属

## 输出
- 差异分析
- 原因归类
- 责任归属
- 改进措施

## Prompt模板
```
请分析预算差异。

背景：
- 预算金额：[预算]
- 实际金额：[实际]
- 差异分类：[分类]
- 责任归属：[归属]

请输出：
1. 差异分析
2. 原因归类
3. 责任归属
4. 改进措施
5. 预警机制
```

## 示例
### 示例1
- 输入：部门预算
- 输出：预算差异，预算10万/实际12万/差异20%/原因分析

## 注意事项
- 要及时分析
- 要原因追溯
- 要责任清晰
- 要改进措施
