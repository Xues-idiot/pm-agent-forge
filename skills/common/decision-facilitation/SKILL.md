# 决策促进 (Decision Facilitation)

## 触发条件
- 决策引导
- 团队决策
- 方案评估
- 共识达成

## 输入
- 决策问题
- 候选方案
- 决策标准
- 参与人

## 输出
- 决策框架
- 评估方法
- 讨论引导
- 结论记录

## Prompt模板
```
请促进决策。

背景：
- 决策问题：[问题]
- 候选方案：[方案]
- 决策标准：[标准]
- 参与人：[参与]

请输出：
1. 决策框架
2. 评估矩阵
3. 讨论引导
4. 投票方法
5. 结论记录
```

## 示例
### 示例1
- 输入：团队方案选择
- 输出：决策促进， pros/cons、评估矩阵、决策树、共识法

## 注意事项
- 要中立引导
- 要确保参与
- 要聚焦目标
- 要明确结论
