# 多云策略 (Multi-cloud Strategy)

## 触发条件
- 多云管理
- 供应商选择
- 成本优化
- 风险分散

## 输入
- 业务需求
- 供应商选项
- 成本预算
- 运维能力

## 输出
- 多云架构
- 供应商策略
- 成本模型
- 运维方案

## Prompt模板
```
请制定多云策略。

背景：
- 业务需求：[需求]
- 供应商选项：[选项]
- 成本预算：[预算]
- 运维能力：[能力]

请输出：
1. 多云架构
2. 供应商选择
3. 成本模型
4. 数据管理
5. 运维方案
```

## 示例
### 示例1
- 输入：企业级应用
- 输出：多云策略，含AWS主用、Azure备份、跨云数据同步、统一管理平台

## 注意事项
- 要避免vendor-lock-in
- 要成本优化
- 要一致体验
- 要简化管理
