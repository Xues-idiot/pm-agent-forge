# 市场进入 (Market Entry)

## 触发条件
- 新市场进入
- 市场调研
- 进入策略
- 执行计划

## 输入
- 目标市场
- 竞争格局
- 产品适配
- 资源投入

## 输出
- 市场分析
- 进入策略
- 执行计划
- 风险评估

## Prompt模板
```
请制定[市场]进入策略。

背景：
- 目标市场：[市场]
- 竞争格局：[格局]
- 产品适配：[适配]
- 资源投入：[资源]

请输出：
1. 市场吸引力
2. 竞争分析
3. 客户需求
4. 进入模式
5. 产品策略
6. 渠道策略
7. 投资回报
8. 风险评估
```

## 示例
### 示例1
- 输入：国产品牌想进入东南亚市场
- 输出：进入策略，含市场选择、渠道合作、本地化

## 注意事项
- 要充分调研
- 要模式适配
- 要控制风险
