# 维护计划 (Maintenance Planning)

## 触发条件
- 设备维护
- 预防性维护
- 维修管理
- 效率提升

## 输入
- 设备状况
- 维护周期
- 维修资源
- 停机影响

## 输出
- 维护计划
- 资源安排
- 停机优化
- 成本控制

## Prompt模板
```
请制定维护计划。

背景：
- 设备状况：[状况]
- 维护周期：[周期]
- 维修资源：[资源]
- 停机影响：[影响]

请输出：
1. 维护计划
2. 资源安排
3. 停机优化
4. 成本控制
5. 效果评估
```

## 示例
### 示例1
- 输入：工厂设备
- 输出：维护计划，预防性维护、点检制度、备件准备

## 注意事项
- 要预防为主
- 要计划先行
- 要资源匹配
- 要持续改进
