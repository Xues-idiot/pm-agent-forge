# 工作分解结构 (Work Breakdown Structure)

## 触发条件
- 任务分解
- 项目规划
- 责任分配
- 成本估算

## 输入
- 项目范围
- 交付物
- 组织结构
- 分解原则

## 输出
- WBS结构
- 工作包
- 责任矩阵
- 层级关系

## Prompt模板
```
请创建工作分解结构。

背景：
- 项目范围：[范围]
- 交付物：[交付]
- 组织结构：[组织]
- 分解原则：[原则]

请输出：
1. WBS结构
2. 工作包
3. 责任矩阵
4. 层级关系
5. 编码体系
```

## 示例
### 示例1
- 输入：项目规划
- 输出：WBS，Phase-Deliverable-Work Package三层

## 注意事项
- 要100%覆盖
- 要层次清晰
- 要工作包独立
- 要责任明确
