# 组织健康 (Organizational Health)

## 触发条件
- 组织效能
- 文化健康
- 领导力
- 持续改进

## 输入
- 组织指标
- 员工反馈
- 领导行为
- 文化评估

## 输出
- 健康度评估
- 问题诊断
- 改进建议
- 跟踪机制

## Prompt模板
```
请评估组织健康。

背景：
- 组织指标：[指标]
- 员工反馈：[反馈]
- 领导行为：[领导]
- 文化评估：[文化]

请输出：
1. 健康度评估
2. 问题诊断
3. 根因分析
4. 改进建议
5. 跟踪机制
```

## 示例
### 示例1
- 输入：组织诊断
- 输出：组织健康，员工敬业、目标清晰、持续改进、创新文化

## 注意事项
- 要多维度
- 要客观数据
- 要行动导向
- 要持续跟踪
