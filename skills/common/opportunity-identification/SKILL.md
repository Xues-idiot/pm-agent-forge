# 机会识别 (Opportunity Identification)

## 触发条件
- 市场机会
- 业务突破
- 创新方向
- 增长点

## 输入
- 市场趋势
- 客户需求
- 竞争空白
- 自身能力

## 输出
- 机会清单
- 优先级
- 进入策略
- 资源需求

## Prompt模板
```
请识别业务机会。

背景：
- 市场趋势：[趋势]
- 客户需求：[需求]
- 竞争空白：[空白]
- 自身能力：[能力]

请输出：
1. 机会清单
2. 吸引力评估
3. 可行性评估
4. 优先级
5. 进入策略
6. 资源需求
```

## 示例
### 示例1
- 输入：消费品市场，寻找新增长点
- 输出：机会识别报告，含市场空白、可行性、优先级

## 注意事项
- 要市场导向
- 要能力匹配
- 要快速验证
