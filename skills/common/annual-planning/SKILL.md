# 年度规划 (Annual Planning)

## 触发条件
- 年度目标
- 资源配置
- 策略规划
- 预算制定

## 输入
- 战略目标
- 市场预测
- 资源需求
- 历史数据

## 输出
- 年度目标
- 策略举措
- 预算规划
- 时间表

## Prompt模板
```
请制定年度规划。

背景：
- 战略目标：[战略]
- 市场预测：[预测]
- 资源需求：[资源]
- 历史数据：[历史]

请输出：
1. 年度目标
2. 市场分析
3. 策略举措
4. 预算规划
5. 时间表
```

## 示例
### 示例1
- 输入：公司战略目标
- 输出：年度规划，收入增长30%、重点产品、关键举措、预算分配

## 注意事项
- 要战略导向
- 要资源匹配
- 要上下对齐
- 要动态调整
