# 销售漏斗 (Sales Funnel)

## 触发条件
- 销售流程
- 转化率优化
- 客户获取
- 漏斗监控

## 输入
- 漏斗阶段
- 各阶段数据
- 流失原因
- 优化目标

## 输出
- 漏斗模型
- 转化策略
- 优化建议
- 监控指标

## Prompt模板
```
请设计销售漏斗。

背景：
- 漏斗阶段：[阶段]
- 各阶段数据：[数据]
- 流失原因：[原因]
- 优化目标：[目标]

请输出：
1. 漏斗模型设计
2. 各阶段转化策略
3. 流失预防措施
4. 监控指标体系
5. 优化优先级
```

## 示例
### 示例1
- 输入：B2B软件销售，从线索到签单
- 输出：漏斗模型，含EDM获客→SDR筛选→AE跟进→方案演示→商务谈判→签单

## 注意事项
- 要明确各阶段定义
- 要追踪关键指标
- 要分析流失原因
- 要持续优化迭代
