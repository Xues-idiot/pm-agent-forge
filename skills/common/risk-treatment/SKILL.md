# 风险应对 (Risk Treatment)

## 触发条件
- 风险评估
- 应对策略
- 资源分配
- 优先级

## 输入
- 风险清单
- 风险等级
- 应对选项
- 资源约束

## 输出
- 应对策略
- 行动计划
- 责任人
- 时间表

## Prompt模板
```
请制定风险应对。

背景：
- 风险清单：[清单]
- 风险等级：[等级]
- 应对选项：[选项]
- 资源约束：[约束]

请输出：
1. 应对策略
2. 行动计划
3. 责任人
4. 时间表
5. 监控机制
```

## 示例
### 示例1
- 输入：项目风险
- 输出：风险应对，规避/转移/减轻/接受策略

## 注意事项
- 要成本效益
- 要切实可行
- 要责任人明确
- 要定期评估
