# CI/CD管道 (CI/CD Pipeline)

## 触发条件
- 持续集成
- 持续部署
- 自动化
- 快速交付

## 输入
- 构建流程
- 部署环境
- 自动化程度
- 发布策略

## 输出
- 管道设计
- 阶段定义
- 自动化配置
- 回滚策略

## Prompt模板
```
请设计CI/CD管道。

背景：
- 构建流程：[构建]
- 部署环境：[环境]
- 自动化程度：[程度]
- 发布策略：[策略]

请输出：
1. 管道设计
2. 阶段定义
3. 自动化配置
4. 回滚策略
5. 监控告警
```

## 示例
### 示例1
- 输入：应用发布
- 输出：CI/CD，代码构建、测试、部署、监控

## 注意事项
- 要自动化
- 要快速反馈
- 要可追溯
- 要安全合规
