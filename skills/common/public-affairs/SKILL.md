# 公共事务 (Public Affairs)

## 触发条件
- 公共关系
- 舆论影响
- 危机管理
- 形象建设

## 输入
- 舆论环境
- 利益相关方
- 议题设置
- 传播渠道

## 输出
- 议题策略
- 传播计划
- 危机预案
- 评估机制

## Prompt模板
```
请管理公共事务。

背景：
- 舆论环境：[环境]
- 利益相关方：[相关方]
- 议题设置：[议题]
- 传播渠道：[渠道]

请输出：
1. 议题策略
2. 传播计划
3. 危机预案
4. 评估机制
5. 资源调配
```

## 示例
### 示例1
- 输入：企业公关
- 输出：公共事务，媒体关系、危机公关、舆情监测

## 注意事项
- 要主动透明
- 要快速响应
- 要多方沟通
- 要持续建设
