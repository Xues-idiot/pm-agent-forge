# 情感分析 (Sentiment Analysis)

## 触发条件
- 客户反馈
- 舆情监控
- 体验分析
- 预警系统

## 输入
- 反馈内容
- 数据来源
- 分析维度
- 应用场景

## 输出
- 情感分类
- 关键主题
- 趋势分析
- 预警机制

## Prompt模板
```
请进行情感分析。

背景：
- 反馈内容：[内容]
- 数据来源：[来源]
- 分析维度：[维度]
- 应用场景：[场景]

请输出：
1. 情感分类
2. 主题提取
3. 趋势分析
4. 预警机制
5. 行动建议
```

## 示例
### 示例1
- 输入：社交媒体评论
- 输出：情感分析，正面65%、负面15%、中性20%、趋势监控

## 注意事项
- 要多渠道
- 要实时分析
- 要深入洞察
- 要快速响应
