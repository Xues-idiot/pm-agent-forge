# 能力评估 (Capability Assessment)

## 触发条件
- 组织诊断
- 差距分析
- 培训需求
- 战略匹配

## 输入
- 战略目标
- 能力要求
- 现有能力
- 评估方法

## 输出
- 能力框架
- 差距分析
- 发展路径
- 优先级

## Prompt模板
```
请评估组织能力。

背景：
- 战略目标：[目标]
- 能力要求：[要求]
- 现有能力：[现有]
- 评估方法：[方法]

请输出：
1. 能力框架
2. 差距分析
3. 发展路径
4. 优先级排序
5. 资源投入
```

## 示例
### 示例1
- 输入：企业数字化转型
- 输出：能力评估，含技术能力、数据能力、组织能力三维度差距分析

## 注意事项
- 要基于战略
- 要客观评估
- 要分清缓急
- 要持续跟踪
