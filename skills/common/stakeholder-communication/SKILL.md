# 利益相关者沟通 (Stakeholder Communication)

## 触发条件
- 干系人管理
- 高层沟通
- 汇报机制
- 关系维护

## 输入
- 干系人地图
- 沟通需求
- 汇报关系
- 沟通频率

## 输出
- 沟通策略
- 汇报计划
- 关系维护
- 冲突处理

## Prompt模板
```
请制定利益相关者沟通策略。

背景：
- 干系人地图：[地图]
- 沟通需求：[需求]
- 汇报关系：[关系]
- 沟通频率：[频率]

请输出：
1. 干系人分析
2. 沟通策略
3. 汇报计划
4. 关系维护
5. 冲突处理
```

## 示例
### 示例1
- 输入：大型项目，多个部门参与
- 输出：沟通策略，含周报机制、双周例会、关键里程碑汇报、即时问题升级

## 注意事项
- 要了解各方需求
- 要主动沟通
- 要信息对齐
- 要建立信任
