# 持续改进 (Continuous Improvement)

## 触发条件
- 质量提升
- 效率优化
- 成本降低
- 文化建立

## 输入
- 改进领域
- 现状痛点
- 改进目标
- 参与意愿

## 输出
- 改进计划
- 实施方案
- 评估机制
- 激励方案

## Prompt模板
```
请设计持续改进机制。

背景：
- 改进领域：[领域]
- 现状痛点：[痛点]
- 改进目标：[目标]
- 参与意愿：[意愿]

请输出：
1. 改进文化
2. 改进机制
3. 改进方法
4. 实施方案
5. 评估机制
6. 激励方案
```

## 示例
### 示例1
- 输入：车间生产效率有提升空间
- 输出：持续改进方案，含Kaizen、提案制度、激励政策

## 注意事项
- 要全员参与
- 要小步快跑
- 要持续坚持
