# 人体工学 (Ergonomics)

## 触发条件
- 工作效率
- 职业健康
- 设备设计
- 舒适度

## 输入
- 工作任务
- 人员能力
- 设备工具
- 环境因素

## 输出
- 人机界面
- 作业设计
- 健康建议
- 改善方案

## Prompt模板
```
请应用人体工学。

背景：
- 工作任务：[任务]
- 人员能力：[能力]
- 设备工具：[工具]
- 环境因素：[环境]

请输出：
1. 人机界面
2. 作业设计
3. 健康建议
4. 改善方案
5. 评估标准
```

## 示例
### 示例1
- 输入：办公室
- 输出：人体工学，工位设计、显示器高度、键盘鼠标

## 注意事项
- 要以人为本
- 要效率健康
- 要持续改善
- 要培训意识
