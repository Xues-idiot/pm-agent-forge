# 供应链优化 (Supply Chain Optimization)

## 触发条件
- 供应链效率
- 成本控制
- 交付能力
- 风险应对

## 输入
- 供应链结构
- 成本分布
- 交付周期
- 瓶颈环节

## 输出
- 优化方案
- 成本分析
- 交付提升
- 风险应对

## Prompt模板
```
请优化供应链。

背景：
- 供应链结构：[结构]
- 成本分布：[分布]
- 交付周期：[周期]
- 瓶颈环节：[瓶颈]

请输出：
1. 优化方案
2. 成本分析
3. 交付提升
4. 风险应对
5. 实施路径
```

## 示例
### 示例1
- 输入：制造业供应链
- 输出：供应链优化，供应商整合、库存优化、物流优化

## 注意事项
- 要端到端视角
- 要协同合作
- 要数字化
- 要持续改进
