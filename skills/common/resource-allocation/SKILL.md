# 资源配置 (Resource Allocation)

## 触发条件
- 资源优化
- 投资决策
- 优先级
- 效率提升

## 输入
- 资源类型
- 业务优先级
- 约束条件
- 优化目标

## 输出
- 配置方案
- 优先级
- 预期产出
- 评估机制

## Prompt模板
```
请优化资源配置。

背景：
- 资源类型：[类型]
- 业务优先级：[优先级]
- 约束条件：[约束]
- 优化目标：[目标]

请输出：
1. 资源现状
2. 业务优先级
3. 配置方案
4. 预期产出
5. 评估机制
```

## 示例
### 示例1
- 输入：研发团队50人，多个项目并行
- 输出：资源配置方案，含项目优先级、人员分配、产出预期

## 注意事项
- 要聚焦重点
- 要动态调整
- 要效能评估
