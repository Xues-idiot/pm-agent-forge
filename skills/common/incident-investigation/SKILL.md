# 事故调查 (Incident Investigation)

## 触发条件
- 事故分析
- 原因追溯
- 责任认定
- 预防措施

## 输入
- 事故经过
- 现场证据
- 证人访谈
- 历史数据

## 输出
- 调查报告
- 原因分析
- 责任认定
- 预防建议

## Prompt模板
```
请调查事故。

背景：
- 事故经过：[经过]
- 现场证据：[证据]
- 证人访谈：[证人]
- 历史数据：[历史]

请输出：
1. 调查报告
2. 原因分析
3. 责任认定
4. 预防建议
5. 整改跟踪
```

## 示例
### 示例1
- 输入：安全事故
- 输出：事故调查，现场勘查、原因分析、责任划分

## 注意事项
- 要客观公正
- 要证据充分
- 要原因分析
- 要预防为主
