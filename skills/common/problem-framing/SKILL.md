# 问题定义 (Problem Framing)

## 触发条件
- 问题分析
- 范围定义
- 需求理解
- 解决方向

## 输入
- 问题表象
- 相关方
- 约束条件
- 成功标准

## 输出
- 问题陈述
- 范围边界
- 相关方分析
- 解决方向

## Prompt模板
```
请定义问题。

背景：
- 问题表象：[表象]
- 相关方：[相关]
- 约束条件：[约束]
- 成功标准：[标准]

请输出：
1. 问题陈述
2. 范围边界
3. 相关方分析
4. 假设条件
5. 解决方向
```

## 示例
### 示例1
- 输入：销售下降
- 输出：问题定义，销售下降15%、定义为用户获取问题、排除产品因素

## 注意事项
- 要准确描述
- 要界定范围
- 要明确相关方
- 要验证假设
