# 商业论证 (Business Case)

## 触发条件
- 项目立项
- 投资回报
- 预算审批
- 战略匹配

## 输入
- 项目背景
- 成本收益
- 风险收益
- 资源需求

## 输出
- 商业论证
- 财务模型
- 风险分析
- 建议结论

## Prompt模板
```
请制定商业论证。

背景：
- 项目背景：[背景]
- 成本收益：[收益]
- 风险收益：[风险]
- 资源需求：[需求]

请输出：
1. 商业论证
2. 财务模型
3. 风险与收益
4. 替代方案
5. 建议结论
```

## 示例
### 示例1
- 输入：工厂自动化改造项目
- 输出：商业论证，含投资300万、年节省100万、3年回本、风险可控

## 注意事项
- 要客观真实
- 要数据支撑
- 要考虑风险
- 要定期复盘
