# 员工敬业度 (Employee Engagement)

## 触发条件
- 员工满意度
- 文化建设
- 团队氛围
- 留存率

## 输入
- 调研数据
- 流失分析
- 文化现状
- 管理风格

## 输出
- 提升策略
- 文化活动
- 沟通机制
- 效果追踪

## Prompt模板
```
请提升员工敬业度。

背景：
- 调研数据：[数据]
- 流失分析：[分析]
- 文化现状：[现状]
- 管理风格：[风格]

请输出：
1. 关键发现
2. 提升策略
3. 文化活动
4. 沟通机制
5. 效果追踪
```

## 示例
### 示例1
- 输入：年度敬业度调查，满意度下降
- 输出：提升策略，含职业发展透明化、工作生活平衡、认可文化、福利升级

## 注意事项
- 要基于数据
- 要关注管理者
- 要持续投入
- 要衡量ROI
