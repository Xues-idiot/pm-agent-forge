# 价值链分析 (Value Chain Analysis)

## 触发条件
- 竞争优势
- 成本优化
- 流程再造
- 战略定位

## 输入
- 业务活动
- 成本结构
- 价值创造
- 竞争态势

## 输出
- 价值链图
- 成本分析
- 优势机会
- 优化建议

## Prompt模板
```
请进行价值链分析。

背景：
- 业务活动：[活动]
- 成本结构：[成本]
- 价值创造：[价值]
- 竞争态势：[态势]

请输出：
1. 价值链分解
2. 成本分析
3. 价值评估
4. 竞争优势识别
5. 优化机会
```

## 示例
### 示例1
- 输入：制造业价值链，需要寻找降本点
- 输出：价值链分析，含采购-生产-物流-销售各环节优化

## 注意事项
- 要全局视角
- 要聚焦关键
- 要持续优化
