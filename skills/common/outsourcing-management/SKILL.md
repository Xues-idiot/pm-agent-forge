# 外包管理 (Outsourcing Management)

## 触发条件
- 外包策略
- 供应商选择
- 合同管理
- 外包监控

## 输入
- 外包范围
- 业务重要性
- 供应商市场
- 管理能力

## 输出
- 外包策略
- 供应商管理
- 合同方案
- 监控机制

## Prompt模板
```
请设计[业务]外包管理方案。

背景：
- 外包范围：[范围]
- 业务重要性：[重要性]
- 供应商市场：[市场]
- 管理能力：[能力]

请输出：
1. 外包策略选择
2. 供应商筛选标准
3. 合同设计
4. 服务水平协议（SLA）
5. 定价模式
6. 沟通机制
7. 绩效监控
8. 风险控制
9. 退出策略
```

## 示例
### 示例1
- 输入：IT部门，想把运维业务外包
- 输出：外包方案，含供应商评估、合同设计、监控指标

## 注意事项
- 要明确责任边界
- 要持续监控
- 要有退出机制
