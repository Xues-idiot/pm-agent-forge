# 问题解决 (Problem Solving)

## 触发条件
- 问题分析诊断
- 根本原因分析
- 解决方案设计
- 决策支持

## 输入
- 问题描述
- 背景信息
- 约束条件
- 相关方

## 输出
- 问题分析报告
- 根因分析
- 解决方案
- 实施计划

## Prompt模板
```
请分析和解决[问题]。

问题背景：
- 问题描述：[描述]
- 影响范围：[范围]
- 紧急程度：[程度]
- 约束条件：[约束]

请输出：
1. 问题定义（5W1H）
2. 问题影响分析
3. 现状调查和数据收集
4. 根因分析（鱼骨图/5Why）
5. 解决方案选项
6. 方案评估和选择
7. 实施方案
8. 效果验证计划
```

## 示例
### 示例1
- 输入：电商平台退货率上升，从5%上升到8%
- 输出：问题分析报告，含数据诊断、根因分析、解决方案

## 注意事项
- 要准确定义问题
- 根因分析要深入
- 方案要可行
