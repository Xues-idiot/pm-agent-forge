# KPI设计 (KPI Design)

## 触发条件
- 指标体系
- 目标分解
- 绩效考核
- 行为引导

## 输入
- 战略目标
- 业务重点
- 组织层级
- 考核周期

## 输出
- KPI体系
- 目标值
- 权重设计
- 评估标准

## Prompt模板
```
请设计KPI体系。

背景：
- 战略目标：[目标]
- 业务重点：[重点]
- 组织层级：[层级]
- 考核周期：[周期]

请输出：
1. 指标框架
2. KPI清单
3. 目标值
4. 权重设计
5. 评估标准
6. 采集方式
```

## 示例
### 示例1
- 输入：销售团队，需要设计KPI
- 输出：KPI体系，含业绩-回款-客户-能力四维度

## 注意事项
- 要战略对齐
- 要可衡量
- 要行为导向
