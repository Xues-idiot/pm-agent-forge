# 产品战略 (Product Strategy)

## 触发条件
- 产品规划
- 产品路线
- 投资决策
- 生命周期

## 输入
- 市场机会
- 竞争格局
- 资源能力
- 商业目标

## 输出
- 产品战略
- 路线图
- 优先级
- 衡量指标

## Prompt模板
```
请制定[产品线]产品战略。

背景：
- 市场机会：[机会]
- 竞争格局：[格局]
- 资源能力：[能力]
- 商业目标：[目标]

请输出：
1. 产品愿景
2. 市场分析
3. 竞争定位
4. 产品组合
5. 路线图
6. 优先级框架
7. 成功指标
```

## 示例
### 示例1
- 输入：企业软件产品线，需要规划3年发展
- 输出：产品战略，含市场聚焦、平台化、生态合作

## 注意事项
- 要市场驱动
- 要资源聚焦
- 要持续迭代
