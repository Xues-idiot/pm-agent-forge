# 危机沟通 (Crisis Communication)

## 触发条件
- 危机应对
- 声誉管理
- 媒体沟通
- 内部沟通

## 输入
- 危机类型
- 影响程度
- 利益相关者
- 应对资源

## 输出
- 沟通策略
- 声明口径
- 媒体策略
- 内部沟通

## Prompt模板
```
请制定危机沟通策略。

背景：
- 危机类型：[类型]
- 影响程度：[程度]
- 利益相关者：[相关者]
- 应对资源：[资源]

请输出：
1. 危机评估
2. 沟通策略
3. 核心信息
4. 媒体策略
5. 内部沟通
```

## 示例
### 示例1
- 输入：产品安全事故
- 输出：危机沟通，含第一时间道歉、主动承担责任、公布整改措施、持续透明沟通

## 注意事项
- 要快速响应
- 要坦诚负责
- 要统一口径
- 要持续沟通
