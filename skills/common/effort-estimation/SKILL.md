# 工作量估算 (Effort Estimation)

## 触发条件
- 资源规划
- 成本预算
- 进度计划
- 承诺决策

## 输入
- 工作范围
- 历史数据
- 团队能力
- 风险因素

## 输出
- 估算结果
- 估算方法
- 置信区间
- 调整因素

## Prompt模板
```
请估算工作量。

背景：
- 工作范围：[范围]
- 历史数据：[数据]
- 团队能力：[能力]
- 风险因素：[风险]

请输出：
1. 估算结果
2. 估算方法
3. 置信区间
4. 调整因素
5. 敏感性分析
```

## 示例
### 示例1
- 输入：功能开发
- 输出：工作量估算，类比估算、参数估算、三点估算

## 注意事项
- 要多种方法
- 要历史数据
- 要团队参与
- 要风险预留
