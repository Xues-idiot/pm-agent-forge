# 退货政策 (Return Policy)

## 触发条件
- 退货处理
- 售后政策
- 客户体验
- 成本控制

## 输入
- 退货原因
- 退货流程
- 成本影响
- 客户诉求

## 输出
- 退货政策
- 处理流程
- 成本分析
- 改进建议

## Prompt模板
```
请制定退货政策。

背景：
- 退货原因：[原因]
- 退货流程：[流程]
- 成本影响：[影响]
- 客户诉求：[诉求]

请输出：
1. 退货政策
2. 处理流程
3. 成本分析
4. 改进建议
5. 预防措施
```

## 示例
### 示例1
- 输入：电商退货
- 输出：退货政策，7天无理由、15天质量问题退换

## 注意事项
- 要客户友好
- 要流程清晰
- 要成本可控
- 要预防欺诈
