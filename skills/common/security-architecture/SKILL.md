# 安全架构 (Security Architecture)

## 触发条件
- 安全设计
- 防护体系
- 纵深防御
- 合规要求

## 输入
- 业务特点
- 威胁模型
- 防护需求
- 合规要求

## 输出
- 安全架构
- 防护层次
- 监控体系
- 应急响应

## Prompt模板
```
请设计安全架构。

背景：
- 业务特点：[特点]
- 威胁模型：[模型]
- 防护需求：[需求]
- 合规要求：[要求]

请输出：
1. 安全架构
2. 防护层次
3. 监控体系
4. 应急响应
5. 合规保障
```

## 示例
### 示例1
- 输入：金融系统
- 输出：安全架构，含网络隔离、加密传输、身份认证、审计日志、安全运营

## 注意事项
- 要纵深防御
- 要零信任
- 要持续监控
- 要快速响应
