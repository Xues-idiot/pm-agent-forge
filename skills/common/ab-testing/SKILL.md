# A/B测试 (A/B Testing)

## 触发条件
- 实验设计
- 方案验证
- 决策支持
- 效果评估

## 输入
- 测试假设
- 分组方案
- 样本量
- 评估指标

## 输出
- 测试设计
- 分组策略
- 样本计算
- 结果分析

## Prompt模板
```
请设计A/B测试。

背景：
- 测试假设：[假设]
- 分组方案：[方案]
- 样本量：[样本]
- 评估指标：[指标]

请输出：
1. 测试设计
2. 分组策略
3. 样本量计算
4. 测试周期
5. 结果分析框架
```

## 示例
### 示例1
- 输入：按钮颜色对点击率影响
- 输出：AB测试设计，含分组、样本量、统计显著性

## 注意事项
- 要样本充足
- 要随机分组
- 要统计显著
