# 实验设计 (Experiment Design)

## 触发条件
- 实验规划
- 假设验证
- 因果推断
- 决策支持

## 输入
- 实验目标
- 变量设计
- 样本规划
- 评估标准

## 输出
- 实验方案
- 变量定义
- 样本要求
- 分析计划

## Prompt模板
```
请设计实验。

背景：
- 实验目标：[目标]
- 变量设计：[变量]
- 样本规划：[样本]
- 评估标准：[标准]

请输出：
1. 实验方案
2. 变量定义
3. 样本要求
4. 分析计划
5. 风险控制
```

## 示例
### 示例1
- 输入：营销实验
- 输出：实验设计，对照组实验、因子设计、AB测试

## 注意事项
- 要随机对照
- 要变量控制
- 要样本代表
- 要结论可推广
