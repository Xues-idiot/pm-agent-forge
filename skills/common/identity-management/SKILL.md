# 身份管理 (Identity Management)

## 触发条件
- 用户身份
- 账号管理
- 身份认证
- 权限管理

## 输入
- 用户群体
- 认证需求
- 账号生命周期
- 合规要求

## 输出
- 身份策略
- 账号管理
- 认证方案
- 治理机制

## Prompt模板
```
请管理身份。

背景：
- 用户群体：[群体]
- 认证需求：[需求]
- 账号生命周期：[周期]
- 合规要求：[要求]

请输出：
1. 身份策略
2. 账号管理
3. 认证方案
4. 治理机制
5. 合规审计
```

## 示例
### 示例1
- 输入：企业用户
- 输出：身份管理，SSO联合认证、账号生命周期、权限自动回收

## 注意事项
- 要安全认证
- 要账号合规
- 要自动化管理
- 要持续监控
