# 留存策略 (Retention Strategy)

## 触发条件
- 用户留存
- 流失预防
- 价值提升
- 忠诚度

## 输入
- 用户行为
- 流失原因
- 生命周期
- 价值分层

## 输出
- 留存策略
- 预防机制
- 召回方案
- 激励体系

## Prompt模板
```
请制定留存策略。

背景：
- 用户行为：[行为]
- 流失原因：[原因]
- 生命周期：[周期]
- 价值分层：[分层]

请输出：
1. 流失预警
2. 预防机制
3. 召回方案
4. 激励体系
5. 效果评估
```

## 示例
### 示例1
- 输入：App用户30日留存率低
- 输出：留存策略，含流失预警、Push策略、会员体系

## 注意事项
- 要早期干预
- 要价值导向
- 要持续优化
