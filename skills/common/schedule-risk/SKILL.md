# 进度风险 (Schedule Risk)

## 触发条件
- 风险识别
- 进度评估
- 缓冲管理
- 应急计划

## 输入
- 风险来源
- 工期估算
- 浮动时间
- 风险偏好

## 输出
- 风险评估
- 缓冲计算
- 应急预案
- 监控机制

## Prompt模板
```
请评估进度风险。

背景：
- 风险来源：[来源]
- 工期估算：[估算]
- 浮动时间：[浮动]
- 风险偏好：[偏好]

请输出：
1. 风险识别
2. 风险评估
3. 缓冲计算
4. 应急预案
5. 监控机制
```

## 示例
### 示例1
- 输入：项目进度
- 输出：进度风险，关键路径风险、浮动时间不足、缓冲不够

## 注意事项
- 要识别风险
- 要量化评估
- 要合理缓冲
- 要应急预案
