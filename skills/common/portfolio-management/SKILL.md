# 项目组合管理 (Portfolio Management)

## 触发条件
- 投资组合
- 优先级排序
- 资源配置
- 收益最大化

## 输入
- 项目池
- 战略优先级
- 资源容量
- 风险承受

## 输出
- 组合策略
- 筛选标准
- 资源配置
- 监控机制

## Prompt模板
```
请优化[组织]项目组合管理。

背景：
- 项目池：[池]
- 战略优先级：[优先级]
- 资源容量：[容量]
- 风险承受：[承受]

请输出：
1. 组合目标
2. 评估标准
3. 筛选模型
4. 优先级排序
5. 资源配置
6. 组合监控
7. 动态调整
8. 收益分析
```

## 示例
### 示例1
- 输入：研发中心，每年100个项目，预算有限
- 输出：组合方案，含筛选模型、优先级矩阵、资源分配

## 注意事项
- 要对齐战略
- 要平衡风险
- 要动态调整
