# 需求预测 (Demand Forecasting)

## 触发条件
- 需求计划
- 库存优化
- 生产安排
- 资源配置

## 输入
- 历史数据
- 市场趋势
- 促销活动
- 季节因素

## 输出
- 预测模型
- 预测结果
- 置信区间
- 调整建议

## Prompt模板
```
请进行需求预测。

背景：
- 历史数据：[数据]
- 市场趋势：[趋势]
- 促销活动：[活动]
- 季节因素：[季节]

请输出：
1. 预测模型
2. 预测结果
3. 置信区间
4. 调整建议
5. 监控指标
```

## 示例
### 示例1
- 输入：零售需求
- 输出：需求预测，时间序列、机器学习、组合预测

## 注意事项
- 要数据质量
- 要多方法验证
- 要持续优化
- 要结合业务
