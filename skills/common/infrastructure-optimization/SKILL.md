# 基础设施优化 (Infrastructure Optimization)

## 触发条件
- 成本优化
- 性能提升
- 资源利用率
- 自动化

## 输入
- 现有资源
- 利用率
- 成本分析
- 性能瓶颈

## 输出
- 优化方案
- 成本节约
- 性能提升
- 自动化建议

## Prompt模板
```
请优化基础设施。

背景：
- 现有资源：[资源]
- 利用率：[利用率]
- 成本分析：[分析]
- 性能瓶颈：[瓶颈]

请输出：
1. 优化方案
2. 成本节约
3. 性能提升
4. 自动化建议
5. 实施计划
```

## 示例
### 示例1
- 输入：云资源优化
- 输出：基础设施优化，Reserved Instances、右-sizing、自动化伸缩

## 注意事项
- 要监控数据
- 要持续优化
- 要弹性设计
- 要成本效益
