# 专利组合 (Patent Portfolio)

## 触发条件
- 专利申请
- 专利维护
- 专利运营
- 侵权风险

## 输入
- 技术领域
- 专利数量
- 申请质量
- 维护成本

## 输出
- 专利组合
- 申请策略
- 维护决策
- 运营计划

## Prompt模板
```
请管理专利组合。

背景：
- 技术领域：[领域]
- 专利数量：[数量]
- 申请质量：[质量]
- 维护成本：[成本]

请输出：
1. 专利组合
2. 申请策略
3. 维护决策
4. 运营计划
5. 成本控制
```

## 示例
### 示例1
- 输入：研发产出
- 输出：专利组合，发明专利、实用新型、外观设计

## 注意事项
- 要质量优先
- 要布局核心
- 要成本效益
- 要运营变现
