# 外包战略 (Outsourcing Strategy)

## 触发条件
- 外包决策
- 供应商选择
- 合同管理
- 绩效管理

## 输入
- 业务类型
- 外包范围
- 供应商市场
- 成本目标

## 输出
- 外包策略
- 供应商评估
- 合同设计
- 绩效管理

## Prompt模板
```
请制定[业务]外包战略。

背景：
- 业务类型：[类型]
- 外包范围：[范围]
- 供应商市场：[市场]
- 成本目标：[目标]

请输出：
1. 外包可行性
2. 供应商选择
3. 合同策略
4. SLA设计
5. 定价模式
6. 绩效管理
7. 风险管理
8. 退出策略
```

## 示例
### 示例1
- 输入：IT运维外包，需要降低成本
- 输出：外包战略，含供应商评估、合同设计、绩效管理

## 注意事项
- 要明确责任
- 要激励约束
- 要持续管理
