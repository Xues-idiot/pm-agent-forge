# 变革曲线 (Change Curve)

## 触发条件
- 变革管理
- 变革阶段
- 阻力应对
- 变革推动

## 输入
- 变革类型
- 个人反应
- 变革阶段
- 支持机制

## 输出
- 变革阶段
- 应对策略
- 支持计划
- 沟通策略

## Prompt模板
```
请应用变革曲线。

背景：
- 变革类型：[类型]
- 个人反应：[反应]
- 变革阶段：[阶段]
- 支持机制：[支持]

请输出：
1. 变革阶段识别
2. 各阶段特征
3. 应对策略
4. 支持计划
5. 成功指标
```

## 示例
### 示例1
- 输入：组织重组
- 输出：变革曲线，否认→抗拒→探索→承诺各阶段策略

## 注意事项
- 要理解规律
- 要耐心推动
- 要及时支持
- 要持续沟通
