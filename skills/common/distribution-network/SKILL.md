# 分销网络 (Distribution Network)

## 触发条件
- 网络布局
- 仓储规划
- 配送效率
- 成本优化

## 输入
- 销售区域
- 客户分布
- 物流成本
- 服务水平

## 输出
- 网络设计
- 仓库选址
- 配送模式
- 成本模型

## Prompt模板
```
请设计分销网络。

背景：
- 销售区域：[区域]
- 客户分布：[分布]
- 物流成本：[成本]
- 服务水平：[服务]

请输出：
1. 网络设计
2. 仓库选址
3. 配送模式
4. 成本模型
5. 评估指标
```

## 示例
### 示例1
- 输入：全国分销
- 输出：分销网络，全国仓、区域仓、前置仓组合

## 注意事项
- 要覆盖目标
- 要成本最优
- 要配送时效
- 要灵活扩展
