# 基础设施即代码 (Infrastructure as Code)

## 触发条件
- 环境一致性
- 自动化部署
- 版本控制
- 快速扩展

## 输入
- 基础设施
- 模板定义
- 环境类型
- 版本管理

## 输出
- 代码结构
- 模板设计
- 部署流程
- 变更管理

## Prompt模板
```
请实施基础设施即代码。

背景：
- 基础设施：[基础设施]
- 模板定义：[模板]
- 环境类型：[类型]
- 版本管理：[版本]

请输出：
1. 代码结构
2. 模板设计
3. 部署流程
4. 变更管理
5. 审计追溯
```

## 示例
### 示例1
- 输入：云环境
- 输出：IaC，Terraform模板、环境分层、版本控制

## 注意事项
- 要代码化管理
- 要版本控制
- 要可重复
- 要安全配置
