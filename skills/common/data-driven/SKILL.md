# 数据驱动决策 (Data-Driven Decision)

## 触发条件
- 数据决策
- 分析文化
- 指标体系
- 决策流程

## 输入
- 数据资产
- 分析能力
- 决策场景
- 文化基础

## 输出
- 数据战略
- 指标体系
- 分析工具
- 决策机制

## Prompt模板
```
请建立数据驱动决策体系。

背景：
- 数据资产：[资产]
- 分析能力：[能力]
- 决策场景：[场景]
- 文化基础：[文化]

请输出：
1. 数据战略
2. 指标体系
3. 分析工具
4. 决策流程
5. 组织文化
```

## 示例
### 示例1
- 输入：传统企业，想转型数据驱动
- 输出：方案含指标体系、仪表盘、周会决策机制

## 注意事项
- 要业务导向
- 要工具有效
- 要文化支撑
