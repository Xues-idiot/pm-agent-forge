# 利益相关方参与 (Stakeholder Engagement)

## 触发条件
- 利益相关方
- 沟通管理
- 期望管理
- 社会责任

## 输入
- 相关方识别
- 期望分析
- 沟通计划
- 资源投入

## 输出
- 参与策略
- 沟通计划
- 反馈机制
- 关系评估

## Prompt模板
```
请管理利益相关方。

背景：
- 相关方识别：[识别]
- 期望分析：[分析]
- 沟通计划：[计划]
- 资源投入：[资源]

请输出：
1. 参与策略
2. 沟通计划
3. 反馈机制
4. 关系评估
5. 冲突管理
```

## 示例
### 示例1
- 输入：企业社会责任
- 输出：利益相关方参与，社区沟通、环保组织、员工代表

## 注意事项
- 要识别全面
- 要分类管理
- 要主动沟通
- 要持续参与
