# 需求管理 (Requirements Management)

## 触发条件
- 需求收集
- 需求分析
- 优先级
- 变更控制

## 输入
- 业务需求
- 用户反馈
- 技术约束
- 资源限制

## 输出
- 需求文档
- 优先级
- 迭代计划
- 验收标准

## Prompt模板
```
请设计需求管理流程。

背景：
- 业务需求：[需求]
- 用户反馈：[反馈]
- 技术约束：[约束]
- 资源限制：[限制]

请输出：
1. 需求收集
2. 需求分析
3. 需求优先级
4. 需求文档
5. 变更控制
6. 验收标准
```

## 示例
### 示例1
- 输入：产品迭代，需求多资源少
- 输出：需求管理，含评分模型、优先级、迭代规划

## 注意事项
- 要聚焦价值
- 要持续优化
- 要透明沟通
