# 条款清单 (Term Sheet)

## 触发条件
- 投资条款
- 融资谈判
- 股权设计
- 保护条款

## 输入
- 融资阶段
- 估值预期
- 投资人要求
- 创始团队诉求

## 输出
- 条款设计
- 估值
- 股权结构
- 保护条款

## Prompt模板
```
请设计条款清单。

背景：
- 融资阶段：[阶段]
- 估值预期：[估值]
- 投资人要求：[要求]
- 创始团队诉求：[诉求]

请输出：
1. 估值
2. 股权结构
3. 董事会
4. 保护条款
5. 退出机制
```

## 示例
### 示例1
- 输入：A轮融资
- 输出：条款清单，估值、优先股、反稀释、董事会

## 注意事项
- 要保护创始人
- 要合理估值
- 要风险平衡
- 要未来融资
