# 对标管理 (Benchmarking)

## 触发条件
- 行业对标
- 最佳实践
- 差距分析
- 改进方向

## 输入
- 对标对象
- 指标体系
- 数据来源
- 改进目标

## 输出
- 对标报告
- 差距分析
- 最佳实践
- 改进建议

## Prompt模板
```
请进行对标分析。

背景：
- 对标对象：[对象]
- 指标体系：[体系]
- 数据来源：[来源]
- 改进目标：[目标]

请输出：
1. 对标指标
2. 数据收集
3. 差距分析
4. 最佳实践
5. 改进建议
```

## 示例
### 示例1
- 输入：电商运营，对标行业top3
- 输出：对标报告，含流量-转化-客单-复购各维度对比

## 注意事项
- 要选对对象
- 要数据准确
- 要落地可行
