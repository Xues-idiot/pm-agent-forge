# 阻力管理 (Resistance Management)

## 触发条件
- 变革阻力
- 负面情绪
- 沟通障碍
- 利益冲突

## 输入
- 阻力来源
- 阻力表现
- 利益相关者
- 应对资源

## 输出
- 阻力分析
- 应对策略
- 沟通计划
- 监控机制

## Prompt模板
```
请管理变革阻力。

背景：
- 阻力来源：[来源]
- 阻力表现：[表现]
- 利益相关者：[相关者]
- 应对资源：[资源]

请输出：
1. 阻力分析
2. 应对策略
3. 沟通计划
4. 激励措施
5. 监控机制
```

## 示例
### 示例1
- 输入：新系统上线
- 输出：阻力管理，培训支持、奖励机制、渐进推行、反馈渠道

## 注意事项
- 要理解原因
- 要尊重员工
- 要参与式
- 要及时回应
