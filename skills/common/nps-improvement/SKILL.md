# NPS提升 (NPS Improvement)

## 触发条件
- 客户忠诚
- 推荐意愿
- 满意度
- 净推荐值

## 输入
- NPS分数
- 反馈分析
- 竞争对手
- 改进资源

## 输出
- NPS分析
- 驱动因素
- 改进行动
- 跟踪机制

## Prompt模板
```
请提升NPS。

背景：
- NPS分数：[分数]
- 反馈分析：[分析]
- 竞争对手：[竞品]
- 改进资源：[资源]

请输出：
1. NPS分析
2. 驱动因素
3. 改进行动
4. 跟踪机制
5. 目标设定
```

## 示例
### 示例1
- 输入：NPS调研
- 输出：NPS提升，贬损者转化、被动者促进 promoter

## 注意事项
- 要深入分析
- 要行动导向
- 要持续跟踪
- 要全员参与
