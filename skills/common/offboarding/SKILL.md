# 离职管理 (Offboarding)

## 触发条件
- 员工离职
- 知识传承
- 关系维护
- 风险控制

## 输入
- 离职原因
- 工作交接
- 资产回收
- 离职面谈

## 输出
- 交接清单
- 知识沉淀
- 关系维护
- 风险评估

## Prompt模板
```
请管理离职流程。

背景：
- 离职原因：[原因]
- 工作交接：[交接]
- 资产回收：[资产]
- 离职面谈：[面谈]

请输出：
1. 交接清单
2. 知识沉淀
3. 关系维护
4. 风险评估
5. 改进建议
```

## 示例
### 示例1
- 输入：员工离职
- 输出：离职管理，工作交接、资产回收、离职面谈、校友会

## 注意事项
- 要知识传承
- 要手续完备
- 要关系维护
- 要风险控制
