# 资源效率 (Resource Efficiency)

## 触发条件
- 资源节约
- 成本降低
- 环境保护
- 可持续发展

## 输入
- 资源消耗
- 效率指标
- 优化空间
- 技术方案

## 输出
- 效率分析
- 优化方案
- 目标设定
- 实施路径

## Prompt模板
```
请提升资源效率。

背景：
- 资源消耗：[消耗]
- 效率指标：[指标]
- 优化空间：[空间]
- 技术方案：[方案]

请输出：
1. 效率分析
2. 优化方案
3. 目标设定
4. 实施路径
5. 评估机制
```

## 示例
### 示例1
- 输入：能源效率
- 输出：资源效率，节能改造、余热回收、智能控制

## 注意事项
- 要数据驱动
- 要技术可行
- 要经济效益
- 要持续改进
