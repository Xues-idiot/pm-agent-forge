# 开发者关系 (Developer Relations)

## 触发条件
- 开发者社区
- 技术布道
- 生态运营
- 反馈收集

## 输入
- 开发者群体
- 技术产品
- 社区目标
- 运营资源

## 输出
- DevRel策略
- 社区运营
- 技术内容
- 反馈闭环

## Prompt模板
```
请管理开发者关系。

背景：
- 开发者群体：[群体]
- 技术产品：[产品]
- 社区目标：[目标]
- 运营资源：[资源]

请输出：
1. DevRel策略
2. 社区运营
3. 技术布道
4. 反馈收集
5. 生态支持
```

## 示例
### 示例1
- 输入：开放平台
- 输出：开发者关系，技术博客、线下Meetup、开发者大奖

## 注意事项
- 要真诚服务
- 要技术价值
- 要社区文化
- 要持续运营
