# 市场进入战略 (Market Entry Strategy)

## 触发条件
- 新市场
- 进入决策
- 风险评估
- 资源配置

## 输入
- 目标市场
- 竞争态势
- 自身能力
- 风险偏好

## 输出
- 市场评估
- 进入模式
- 风险分析
- 资源配置

## Prompt模板
```
请制定市场进入战略。

背景：
- 目标市场：[市场]
- 竞争态势：[态势]
- 自身能力：[能力]
- 风险偏好：[风险]

请输出：
1. 市场吸引力评估
2. 进入模式选择
3. 风险与应对
4. 资源配置
5. 时间规划
```

## 示例
### 示例1
- 输入：消费品进入新地区
- 输出：进入战略，先试点再扩展，电商+线下组合

## 注意事项
- 要谨慎验证
- 要资源聚焦
- 要风险可控
- 要灵活调整
