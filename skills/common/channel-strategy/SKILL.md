# 渠道策略 (Channel Strategy)

## 触发条件
- 渠道规划
- 渠道管理
- 资源分配
- 渠道创新

## 输入
- 产品特点
- 目标用户
- 渠道类型
- 竞争格局

## 输出
- 渠道策略
- 渠道选择
- 管理机制
- 激励机制

## Prompt模板
```
请制定渠道策略。

背景：
- 产品特点：[特点]
- 目标用户：[用户]
- 渠道类型：[类型]
- 竞争格局：[格局]

请输出：
1. 渠道策略
2. 渠道选择标准
3. 管理机制
4. 激励机制
5. 评估指标
```

## 示例
### 示例1
- 输入：新品饮料进入市场
- 输出：渠道策略，含超市便利店优先、电商平台同步、特殊渠道（健身房）拓展

## 注意事项
- 要匹配产品特点
- 要精准触达用户
- 要渠道利益平衡
- 要持续优化
