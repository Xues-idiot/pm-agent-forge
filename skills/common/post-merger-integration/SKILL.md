# 并购后整合 (Post Merger Integration)

## 触发条件
- 整合规划
- 协同实现
- 文化融合
- 价值创造

## 输入
- 整合范围
- 协同目标
- 组织设计
- 变革管理

## 输出
- 整合计划
- 协同追踪
- 风险管理
- 价值实现

## Prompt模板
```
请进行并购后整合。

背景：
- 整合范围：[范围]
- 协同目标：[目标]
- 组织设计：[设计]
- 变革管理：[变革]

请输出：
1. 整合计划
2. 协同追踪
3. 风险管理
4. 价值实现
5. 成功标准
```

## 示例
### 示例1
- 输入：并购整合
- 输出：整合计划，业务整合、人员整合、系统整合

## 注意事项
- 要快速整合
- 要协同优先
- 要文化融合
- 要持续沟通
