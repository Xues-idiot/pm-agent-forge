# 福利方案 (Benefits Package)

## 触发条件
- 员工福利
- 吸引人才
- 成本效益
- 员工需求

## 输入
- 福利需求
- 成本预算
- 员工偏好
- 行业标准

## 输出
- 福利结构
- 成本分摊
- 选择机制
- 沟通方案

## Prompt模板
```
请设计福利方案。

背景：
- 福利需求：[需求]
- 成本预算：[预算]
- 员工偏好：[偏好]
- 行业标准：[标准]

请输出：
1. 福利结构
2. 成本分摊
3. 选择机制
4. 沟通方案
5. 评估机制
```

## 示例
### 示例1
- 输入：员工福利
- 输出：福利方案，保险、体检、培训、弹性福利

## 注意事项
- 要员工需求
- 要成本效益
- 要差异化
- 要合规要求
