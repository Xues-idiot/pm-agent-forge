# 业务连续性计划 (Business Continuity Plan)

## 触发条件
- 业务中断
- 灾难应对
- 恢复优先
- 危机管理

## 输入
- 关键业务
- 恢复目标
- 资源需求
- 切换策略

## 输出
- 连续性策略
- 业务恢复
- 资源保障
- 沟通机制

## Prompt模板
```
请制定业务连续性计划。

背景：
- 关键业务：[业务]
- 恢复目标：[目标]
- 资源需求：[需求]
- 切换策略：[策略]

请输出：
1. 连续性策略
2. 业务恢复
3. 资源保障
4. 沟通机制
5. 演练计划
```

## 示例
### 示例1
- 输入：核心业务
- 输出：业务连续性，RTO/RPO、恢复优先级、资源调配

## 注意事项
- 要关键优先
- 要切实可行
- 要定期演练
- 要持续更新
