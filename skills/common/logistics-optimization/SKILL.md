# 物流优化 (Logistics Optimization)

## 触发条件
- 物流效率
- 配送成本
- 仓储管理
- 运输优化

## 输入
- 物流网络
- 仓储分布
- 运输方式
- 配送需求

## 输出
- 优化方案
- 网络设计
- 成本优化
- 效率提升

## Prompt模板
```
请优化物流。

背景：
- 物流网络：[网络]
- 仓储分布：[分布]
- 运输方式：[方式]
- 配送需求：[需求]

请输出：
1. 优化方案
2. 网络设计
3. 成本优化
4. 效率提升
5. 实施计划
```

## 示例
### 示例1
- 输入：电商物流
- 输出：物流优化，分仓布局、智能调度、最后一公里

## 注意事项
- 要网络规划
- 要技术赋能
- 要成本效益
- 要客户体验
