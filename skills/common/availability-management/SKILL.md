# 可用性管理 (Availability Management)

## 触发条件
- 服务可用性
- 故障预防
- 恢复能力
- SLA保障

## 输入
- 可用性目标
- 故障场景
- 恢复能力
- 成本约束

## 输出
- 可用性设计
- 监控方案
- 恢复策略
- 评估机制

## Prompt模板
```
请管理可用性。

背景：
- 可用性目标：[目标]
- 故障场景：[场景]
- 恢复能力：[能力]
- 成本约束：[约束]

请输出：
1. 可用性目标
2. 高可用设计
3. 监控方案
4. 恢复策略
5. 评估机制
```

## 示例
### 示例1
- 输入：核心业务系统
- 输出：可用性管理，99.99%可用性、主备切换、自动故障转移

## 注意事项
- 要目标明确
- 要设计可靠
- 要持续监控
- 要定期演练
