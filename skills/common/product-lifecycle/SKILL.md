# 产品生命周期 (Product Lifecycle)

## 触发条件
- 产品规划
- 阶段策略
- 更新迭代
- 退出决策

## 输入
- 产品阶段
- 市场反馈
- 竞争态势
- 盈利能力

## 输出
- 生命周期阶段
- 阶段策略
- 转型时机
- 退出规划

## Prompt模板
```
请管理产品生命周期。

背景：
- 产品阶段：[阶段]
- 市场反馈：[反馈]
- 竞争态势：[态势]
- 盈利能力：[盈利]

请输出：
1. 当前阶段评估
2. 阶段策略
3. 转型信号
4. 更新计划
5. 退出策略
```

## 示例
### 示例1
- 输入：成熟期产品
- 输出：生命周期策略，优化体验、降低成本、寻找新场景、准备替代品

## 注意事项
- 要提前规划
- 要关注信号
- 要有序退出
- 要客户导向
