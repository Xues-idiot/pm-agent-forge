# 供应商风险 (Vendor Risk)

## 触发条件
- 供应商评估
- 风险识别
- 合同风险
- 合规要求

## 输入
- 供应商信息
- 服务依赖
- 财务状况
- 过往表现

## 输出
- 风险评估
- 风险等级
- 控制措施
- 监控计划

## Prompt模板
```
请评估供应商风险。

背景：
- 供应商信息：[信息]
- 服务依赖：[依赖]
- 财务状况：[状况]
- 过往表现：[表现]

请输出：
1. 风险评估
2. 风险等级
3. 控制措施
4. 监控计划
5. 退出策略
```

## 示例
### 示例1
- 输入：IT供应商
- 输出：供应商风险，技术能力、财务稳定、安全合规

## 注意事项
- 要全面评估
- 要持续监控
- 要备份方案
- 要合同保护
