# 坚定表达 (Assertiveness)

## 触发条件
- 坚定沟通
- 立场表达
- 边界设定
- 自信沟通

## 输入
- 表达目的
- 沟通对象
- 情境因素
- 自信程度

## 输出
- 表达策略
- 沟通技巧
- 边界设定
- 应对方式

## Prompt模板
```
请提升坚定表达能力。

背景：
- 表达目的：[目的]
- 沟通对象：[对象]
- 情境因素：[情境]
- 自信程度：[自信]

请输出：
1. 表达策略
2. 语言技巧
3. 边界设定
4. 应对挑战
5. 实践练习
```

## 示例
### 示例1
- 输入：拒绝不合理要求
- 输出：坚定表达，清晰表达、说明理由、提出替代、坚定而尊重

## 注意事项
- 要清晰直接
- 要尊重对方
- 要说明理由
- 要提出替代
