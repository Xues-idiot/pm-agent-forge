# 测试策略 (Testing Strategy)

## 触发条件
- 测试规划
- 质量保障
- 测试覆盖
- 风险控制

## 输入
- 测试范围
- 测试类型
- 资源限制
- 质量目标

## 输出
- 测试计划
- 测试金字塔
- 自动化策略
- 发布标准

## Prompt模板
```
请制定测试策略。

背景：
- 测试范围：[范围]
- 测试类型：[类型]
- 资源限制：[资源]
- 质量目标：[目标]

请输出：
1. 测试计划
2. 测试金字塔
3. 自动化策略
4. 发布标准
5. 风险评估
```

## 示例
### 示例1
- 输入：敏捷开发
- 输出：测试策略，单元测试、集成测试、端到端测试

## 注意事项
- 要测试分层
- 要自动化优先
- 要持续集成
- 要风险导向
