# 劳动力分析 (Workforce Analytics)

## 触发条件
- 人效分析
- 人员洞察
- 决策支持
- 趋势预测

## 输入
- 人员数据
- 绩效数据
- 流动数据
- 业务关联

## 输出
- 人效指标
- 流动分析
- 趋势预测
- 建议输出

## Prompt模板
```
请进行劳动力分析。

背景：
- 人员数据：[人员]
- 绩效数据：[绩效]
- 流动数据：[流动]
- 业务关联：[关联]

请输出：
1. 人效指标
2. 流动分析
3. 趋势预测
4. 风险预警
5. 建议输出
```

## 示例
### 示例1
- 输入：HR数据
- 输出：劳动力分析，人效比、流失率、人才密度、趋势预测

## 注意事项
- 要数据驱动
- 要业务关联
- 要预测分析
- 要决策支持
