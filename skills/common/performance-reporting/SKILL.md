# 绩效报告 (Performance Reporting)

## 触发条件
- 绩效评估
- 目标追踪
- 进度报告
- 决策支持

## 输入
- KPI指标
- 目标值
- 实际数据
- 分析维度

## 输出
- 绩效分析
- 差距报告
- 趋势分析
- 改进建议

## Prompt模板
```
请编制绩效报告。

背景：
- KPI指标：[指标]
- 目标值：[目标]
- 实际数据：[数据]
- 分析维度：[维度]

请输出：
1. 绩效分析
2. 差距报告
3. 趋势分析
4. 改进建议
5. 奖惩建议
```

## 示例
### 示例1
- 输入：销售绩效
- 输出：绩效报告，目标达成率、同比环比、区域排名

## 注意事项
- 要数据准确
- 要分析深入
- 要及时报告
- 要行动导向
