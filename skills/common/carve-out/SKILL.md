# 业务剥离 (Carve-out)

## 触发条件
- 业务分立
- 估值定价
- 法律架构
- 过渡服务

## 输入
- 业务范围
- 财务数据
- 法律实体
- 客户合同

## 输出
- 剥离方案
- 财务模型
- 法律架构
- 过渡计划

## Prompt模板
```
请设计业务剥离。

背景：
- 业务范围：[范围]
- 财务数据：[数据]
- 法律实体：[实体]
- 客户合同：[合同]

请输出：
1. 剥离方案
2. 财务模型
3. 法律架构
4. 过渡计划
5. 税务筹划
```

## 示例
### 示例1
- 输入：业务分立
- 输出：业务剥离，资产分离、人员转移、合同承接

## 注意事项
- 要法律合规
- 要税务优化
- 要客户同意
- 要平稳过渡
