# 转型或坚持 (Pivot or Persist)

## 触发条件
- 战略调整
- 业务转型
- 坚持验证
- 决策点

## 输入
- 验证结果
- 市场反馈
- 核心假设
- 资源状态

## 输出
- 评估分析
- 转型选项
- 坚持理由
- 决策建议

## Prompt模板
```
请评估转型或坚持。

背景：
- 验证结果：[结果]
- 市场反馈：[反馈]
- 核心假设：[假设]
- 资源状态：[资源]

请输出：
1. 假设验证回顾
2. 市场反馈分析
3. 转型选项
4. 坚持理由
5. 决策建议
```

## 示例
### 示例1
- 输入：创业项目用户增长不及预期
- 输出：评估报告，含转型方向、坚持依据、决策建议

## 注意事项
- 要数据驱动
- 要果断决策
- 要资源匹配
