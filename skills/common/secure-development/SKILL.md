# 安全开发生命周期 (Secure Development Lifecycle)

## 触发条件
- 安全开发
- 流程规范
- 代码安全
- 质量门禁

## 输入
- 开发流程
- 安全需求
- 培训计划
- 工具链

## 输出
- SDL流程
- 安全活动
- 检查清单
- 工具集成

## Prompt模板
```
请实施安全开发。

背景：
- 开发流程：[流程]
- 安全需求：[需求]
- 培训计划：[计划]
- 工具链：[工具]

请输出：
1. SDL流程
2. 安全活动
3. 检查清单
4. 工具集成
5. 度量指标
```

## 示例
### 示例1
- 输入：DevSecOps
- 输出：安全开发生命周期，威胁建模、代码扫描、渗透测试

## 注意事项
- 要左移安全
- 要自动化
- 要持续培训
- 要文化培养
