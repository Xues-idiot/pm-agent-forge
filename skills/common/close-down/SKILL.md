# 项目收尾 (Project Close-down)

## 触发条件
- 项目结束
- 交付验收
- 经验总结
- 资源释放

## 输入
- 项目成果
- 验收标准
- 未完成项
- 经验教训

## 输出
- 收尾清单
- 交付物
- 经验总结
- 资源释放

## Prompt模板
```
请进行项目收尾。

背景：
- 项目成果：[成果]
- 验收标准：[标准]
- 未完成项：[未完成]
- 经验教训：[教训]

请输出：
1. 收尾清单
2. 交付验收
3. 未完成项处理
4. 经验总结
5. 资源释放
```

## 示例
### 示例1
- 输入：项目完成
- 输出：项目收尾，交付物验收、合同关闭、团队解散、经验归档

## 注意事项
- 要完整收尾
- 要经验沉淀
- 要资源释放
- 要及时归档
