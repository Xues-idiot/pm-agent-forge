# NPS分析 (Net Promoter Score)

## 触发条件
- 用户忠诚
- 推荐意愿
- 满意度
- 增长预测

## 输入
- 推荐评分
- 贬损者
- 被动着
- 推荐者

## 输出
- NPS值
- 用户分类
- 驱动因素
- 改进建议

## Prompt模板
```
请分析NPS。

背景：
- 推荐评分：[评分]
- 贬损者：[贬损]
- 被动着：[被动]
- 推荐者：[推荐]

请输出：
1. NPS计算
2. 用户分类
3. 推荐驱动因素
4. 贬损原因
5. 改进建议
```

## 示例
### 示例1
- 输入：用户调研
- 输出：NPS分析，NPS值45，推荐者65%、被动者25%、贬损者10%

## 注意事项
- 要定期测量
- 要分析驱动
- 要行动闭环
- 要持续跟踪
