# 业务影响分析 (Business Impact Analysis)

## 触发条件
- 风险评估
- 业务连续性
- 恢复优先级
- 资源规划

## 输入
- 业务过程
- 中断影响
- 恢复需求
- 合规要求

## 输出
- 业务优先级
- 中断影响
- 恢复目标
- 资源需求

## Prompt模板
```
请进行业务影响分析。

背景：
- 业务过程：[过程]
- 中断影响：[影响]
- 恢复需求：[需求]
- 合规要求：[合规]

请输出：
1. 业务识别
2. 影响评估
3. 恢复优先级
4. 恢复目标
5. 资源需求
```

## 示例
### 示例1
- 输入：企业核心业务
- 输出：BIA分析，识别关键业务、影响程度、恢复优先级

## 注意事项
- 要全面识别
- 要量化影响
- 要明确优先级
- 要定期更新
