# 代码审查 (Code Review)

## 触发条件
- 代码质量
- 团队协作
- 知识共享
- 风险控制

## 输入
- 代码变更
- 审查标准
- 团队规范
- 风险点

## 输出
- 审查意见
- 问题分级
- 批准意见
- 改进建议

## Prompt模板
```
请进行代码审查。

背景：
- 代码变更：[变更]
- 审查标准：[标准]
- 团队规范：[规范]
- 风险点：[风险]

请输出：
1. 审查意见
2. 问题分级
3. 批准意见
4. 改进建议
5. 后续跟踪
```

## 示例
### 示例1
- 输入：代码合并
- 输出：代码审查，代码规范、安全漏洞、设计问题

## 注意事项
- 要及时审查
- 要客观公正
- 要重点突出
- 要建设性反馈
