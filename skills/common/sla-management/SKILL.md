# SLA管理 (SLA Management)

## 触发条件
- 服务承诺
- 响应时效
- 质量标准
- 违约处理

## 输入
- 服务范围
- 服务级别
- 资源能力
- 客户期望

## 输出
- SLA定义
- 监控机制
- 报告体系
- 改进流程

## Prompt模板
```
请管理SLA。

背景：
- 服务范围：[范围]
- 服务级别：[级别]
- 资源能力：[能力]
- 客户期望：[期望]

请输出：
1. SLA定义
2. 监控指标
3. 报告机制
4. 违约处理
5. 持续改进
```

## 示例
### 示例1
- 输入：IT运维服务
- 输出：SLA管理，含P1故障1小时响应、P2 4小时、P3 24小时

## 注意事项
- 要切实可行
- 要可监控
- 要定期回顾
- 要持续提升
