# 估算技术 (Estimation Techniques)

## 触发条件
- 工作量估算
- 资源规划
- 项目报价
- 进度制定

## 输入
- 任务描述
- 团队能力
- 历史数据
- 估算方法

## 输出
- 估算方法
- 估算结果
- 置信区间
- 调整因素

## Prompt模板
```
请进行工作量估算。

背景：
- 任务描述：[描述]
- 团队能力：[能力]
- 历史数据：[数据]
- 估算方法：[方法]

请输出：
1. 估算方法选择
2. 估算过程
3. 估算结果
4. 置信区间
5. 风险调整
```

## 示例
### 示例1
- 输入：功能开发任务
- 输出：估算方案，含故事点、功能点估算、三点估算法、风险调整

## 注意事项
- 要基于数据
- 要团队参与
- 要留有缓冲
- 要持续校准
