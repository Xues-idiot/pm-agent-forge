# 订阅模式 (Subscription Model)

## 触发条件
- 商业模式
- 收入预测
- 客户留存
- 续费率

## 输入
- 订阅价值
- 定价层级
- 客户画像
- 续订策略

## 输出
- 订阅设计
- 定价方案
- 留存策略
- 续订优化

## Prompt模板
```
请设计订阅模式。

背景：
- 订阅价值：[价值]
- 定价层级：[层级]
- 客户画像：[画像]
- 续订策略：[策略]

请输出：
1. 订阅设计
2. 定价方案
3. 客户分层
4. 留存策略
5. 续订优化
```

## 示例
### 示例1
- 输入：会员服务
- 输出：订阅模式，月/年订阅、自动续订、会员权益体系

## 注意事项
- 要持续价值
- 要简化体验
- 要续订便利
- 要持续优化
