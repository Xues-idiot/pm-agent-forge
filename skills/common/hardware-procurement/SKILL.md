# 硬件采购 (Hardware Procurement)

## 触发条件
- 设备采购
- 规格确定
- 供应商选择
- 成本控制

## 输入
- 设备需求
- 技术规格
- 预算范围
- 交付时间

## 输出
- 采购方案
- 规格说明
- 供应商评估
- 成本分析

## Prompt模板
```
请进行硬件采购。

背景：
- 设备需求：[需求]
- 技术规格：[规格]
- 预算范围：[预算]
- 交付时间：[时间]

请输出：
1. 采购方案
2. 规格说明
3. 供应商评估
4. 成本分析
5. 实施计划
```

## 示例
### 示例1
- 输入：服务器采购
- 输出：硬件采购，配置要求、供应商比价、交货安装

## 注意事项
- 要规格明确
- 要性价比
- 要交货及时
- 要质量保证
