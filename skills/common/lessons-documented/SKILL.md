# 经验文档化 (Lessons Documented)

## 触发条件
- 经验沉淀
- 知识管理
- 持续改进
- 团队学习

## 输入
- 项目案例
- 成功经验
- 失败教训
- 最佳实践

## 输出
- 文档结构
- 内容模板
- 审核机制
- 共享计划

## Prompt模板
```
请文档化经验。

背景：
- 项目案例：[案例]
- 成功经验：[经验]
- 失败教训：[教训]
- 最佳实践：[实践]

请输出：
1. 文档结构
2. 内容模板
3. 审核机制
4. 共享计划
5. 更新维护
```

## 示例
### 示例1
- 输入：年度项目复盘
- 输出：经验文档，项目概况、成功因素、问题分析、改进建议

## 注意事项
- 要结构化
- 要实用性
- 要易获取
- 要持续更新
