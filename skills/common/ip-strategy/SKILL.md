# 知识产权战略 (IP Strategy)

## 触发条件
- 知识产权
- 创新保护
- 商业化
- 竞争壁垒

## 输入
- 技术创新
- 市场布局
- 竞争对手
- 商业目标

## 输出
- IP布局
- 保护策略
- 商业化路径
- 风险防控

## Prompt模板
```
请制定知识产权战略。

背景：
- 技术创新：[创新]
- 市场布局：[布局]
- 竞争对手：[竞品]
- 商业目标：[目标]

请输出：
1. IP布局
2. 保护策略
3. 商业化路径
4. 风险防控
5. 投资规划
```

## 示例
### 示例1
- 输入：科技企业
- 输出：IP战略，专利布局、商标保护、商业秘密

## 注意事项
- 要商业导向
- 要提前布局
- 要全球视野
- 要持续运营
