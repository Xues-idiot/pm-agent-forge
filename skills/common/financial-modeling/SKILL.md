# 财务建模 (Financial Modeling)

## 触发条件
- 财务预测
- 投资分析
- 预算规划
- 决策支持

## 输入
- 假设条件
- 历史数据
- 预测期间
- 模型类型

## 输出
- 财务模型
- 收入预测
- 成本预测
- 现金流

## Prompt模板
```
请建立财务模型。

背景：
- 假设条件：[假设]
- 历史数据：[历史]
- 预测期间：[期间]
- 模型类型：[类型]

请输出：
1. 财务模型
2. 收入预测
3. 成本预测
4. 现金流预测
5. 敏感性分析
```

## 示例
### 示例1
- 输入：5年规划
- 输出：财务模型，3年盈亏平衡、IRR 25%

## 注意事项
- 要假设合理
- 要数据准确
- 要敏感性分析
- 要持续更新
