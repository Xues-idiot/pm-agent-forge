# 许可交易 (Licensing Deals)

## 触发条件
- 知识产权
- 技术许可
- 收入变现
- 合作扩张

## 输入
- 许可资产
- 潜在被许可方
- 许可条款
- 市场机会

## 输出
- 许可策略
- 条款设计
- 谈判方案
- 合同管理

## Prompt模板
```
请进行许可交易。

背景：
- 许可资产：[资产]
- 潜在被许可方：[潜在]
- 许可条款：[条款]
- 市场机会：[机会]

请输出：
1. 许可策略
2. 条款设计
3. 谈判方案
4. 合同管理
5. 收益分成
```

## 示例
### 示例1
- 输入：技术许可
- 输出：许可交易，专利许可、软件授权、联合开发

## 注意事项
- 要价值评估
- 要条款清晰
- 要风险控制
- 要收益合理
