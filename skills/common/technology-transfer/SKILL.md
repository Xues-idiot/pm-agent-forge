# 技术转让 (Technology Transfer)

## 触发条件
- 技术商业化
- 成果转化
- 产学研合作
- 技术输出

## 输入
- 技术成果
- 受让方
- 转让方式
- 交易条件

## 输出
- 转让策略
- 估值方法
- 合同条款
- 实施计划

## Prompt模板
```
请进行技术转让。

背景：
- 技术成果：[成果]
- 受让方：[受让方]
- 转让方式：[方式]
- 交易条件：[条件]

请输出：
1. 转让策略
2. 估值方法
3. 合同条款
4. 实施计划
5. 风险控制
```

## 示例
### 示例1
- 输入：高校成果
- 输出：技术转让，专利转让、许可转让、合作开发

## 注意事项
- 要价值评估
- 要尽职调查
- 要条款明确
- 要支持到位
