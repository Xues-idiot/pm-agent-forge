# 漏斗分析 (Funnel Analysis)

## 触发条件
- 转化优化
- 用户旅程
- 流失诊断
- 运营提升

## 输入
- 漏斗阶段
- 转化数据
- 用户行为
- 优化目标

## 输出
- 漏斗图
- 转化分析
- 流失原因
- 优化建议

## Prompt模板
```
请进行漏斗分析。

背景：
- 漏斗阶段：[阶段]
- 转化数据：[数据]
- 用户行为：[行为]
- 优化目标：[目标]

请输出：
1. 漏斗图
2. 各阶段转化率
3. 流失分析
4. 流失原因假设
5. 优化建议
```

## 示例
### 示例1
- 输入：电商购买漏斗
- 输出：漏斗分析，含浏览-加购-下单-支付各环节转化

## 注意事项
- 要数据驱动
- 要归因分析
- 要假设验证
