# 区域规划 (Territory Planning)

## 触发条件
- 区域划分
- 销售分配
- 资源优化
- 绩效管理

## 输入
- 市场潜力
- 销售团队
- 客户分布
- 历史绩效

## 输出
- 区域划分方案
- 配额分配
- 资源调配
- 评估机制

## Prompt模板
```
请规划销售区域。

背景：
- 市场潜力：[潜力]
- 销售团队：[团队]
- 客户分布：[分布]
- 历史绩效：[绩效]

请输出：
1. 区域划分方案
2. 配额分配
3. 资源调配
4. 评估机制
5. 动态调整规则
```

## 示例
### 示例1
- 输入：全国性消费品企业，50名销售
- 输出：区域规划，含7个大区、按GDP和渠道数量分配、每季度动态调整

## 注意事项
- 要公平透明
- 要考虑潜力差异
- 要动态平衡
- 要激励与约束并存
