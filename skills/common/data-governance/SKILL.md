# 数据治理 (Data Governance)

## 触发条件
- 数据资产管理
- 数据标准
- 数据质量
- 数据安全

## 输入
- 数据类型
- 数据现状
- 业务需求
- 合规要求

## 输出
- 治理框架
- 数据标准
- 质量管理
- 安全策略

## Prompt模板
```
请制定[组织]数据治理方案。

背景：
- 数据类型：[类型]
- 数据现状：[现状]
- 业务需求：[需求]
- 合规要求：[要求]

请输出：
1. 治理框架
2. 数据资产目录
3. 数据标准
4. 数据质量管理
5. 元数据管理
6. 主数据管理
7. 数据安全
8. 数据共享
9. 组织保障
```

## 示例
### 示例1
- 输入：金融机构，需要建立统一数据治理
- 输出：治理方案，含数据标准、质量规则、安全分级

## 注意事项
- 要业务驱动
- 要逐步推进
- 要持续运营
