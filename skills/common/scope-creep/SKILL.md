# 范围蔓延 (Scope Creep)

## 触发条件
- 范围控制
- 变更管理
- 需求管理
- 进度影响

## 输入
- 原始范围
- 新增需求
- 影响评估
- 变更控制

## 输出
- 蔓延识别
- 影响分析
- 控制措施
- 决策建议

## Prompt模板
```
请控制范围蔓延。

背景：
- 原始范围：[原始]
- 新增需求：[需求]
- 影响评估：[评估]
- 变更控制：[控制]

请输出：
1. 蔓延识别
2. 影响分析
3. 控制措施
4. 决策建议
5. 预防机制
```

## 示例
### 示例1
- 输入：项目范围变化
- 输出：范围蔓延，识别蔓延、分析影响、评估成本、拒绝或纳入

## 注意事项
- 要严格变更
- 要评估影响
- 要区分价值
- 要预防为主
