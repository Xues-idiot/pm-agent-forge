# SLA监控 (SLA Monitoring)

## 触发条件
- 服务质量
- 合同履行
- 绩效评估
- 改进依据

## 输入
- SLA指标
- 监控数据
- 报告周期
- 违约情况

## 输出
- 监控报告
- 合规状态
- 偏差分析
- 改进建议

## Prompt模板
```
请监控SLA。

背景：
- SLA指标：[指标]
- 监控数据：[数据]
- 报告周期：[周期]
- 违约情况：[违约]

请输出：
1. 监控报告
2. 合规状态
3. 偏差分析
4. 改进建议
5. 处罚申请
```

## 示例
### 示例1
- 输入：外包服务
- 输出：SLA监控，每月达标率、违约记录、改进要求

## 注意事项
- 要数据准确
- 要及时监控
- 要奖惩分明
- 要持续改进
