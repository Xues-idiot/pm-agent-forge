# 灾难恢复计划 (Disaster Recovery Plan)

## 触发条件
- 灾难场景
- 系统恢复
- 数据恢复
- RTO/RPO

## 输入
- 灾难类型
- 恢复目标
- 系统架构
- 备份策略

## 输出
- 恢复方案
- 切换流程
- 验证步骤
- 人员安排

## Prompt模板
```
请制定灾难恢复计划。

背景：
- 灾难类型：[类型]
- 恢复目标：[目标]
- 系统架构：[架构]
- 备份策略：[策略]

请输出：
1. 恢复方案
2. 切换流程
3. 验证步骤
4. 人员安排
5. 演练计划
```

## 示例
### 示例1
- 输入：数据中心
- 输出：灾难恢复，冷备切换、热备同步、RTO 4小时

## 注意事项
- 要恢复目标
- 要定期演练
- 要自动切换
- 要数据完整
