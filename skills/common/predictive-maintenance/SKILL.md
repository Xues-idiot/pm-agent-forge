# 预测性维护 (Predictive Maintenance)

## 触发条件
- 设备维护
- 故障预测
- 备件管理
- 运维优化

## 输入
- 设备数据
- 故障历史
- 监测指标
- 维护成本

## 输出
- 预测模型
- 维护策略
- 阈值设置
- 实施方案

## Prompt模板
```
请制定预测性维护。

背景：
- 设备数据：[数据]
- 故障历史：[历史]
- 监测指标：[指标]
- 维护成本：[成本]

请输出：
1. 预测模型
2. 维护策略
3. 阈值设置
4. 实施方案
5. 效果评估
```

## 示例
### 示例1
- 输入：工业设备
- 输出：预测维护，含振动分析、温度监测、寿命预测、维护计划优化

## 注意事项
- 要数据积累
- 要模型迭代
- 要阈值优化
- 要运维协同
