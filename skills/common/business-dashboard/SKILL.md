# 业务看板 (Business Dashboard)

## 触发条件
- 数据监控
- 运营管理
- 决策支持
- 目标追踪

## 输入
- 业务指标
- 数据来源
- 受众需求
- 更新频率

## 输出
- 看板设计
- 指标体系
- 可视化
- 运维机制

## Prompt模板
```
请设计业务看板。

背景：
- 业务指标：[指标]
- 数据来源：[来源]
- 受众需求：[受众]
- 更新频率：[频率]

请输出：
1. 看板目标
2. 指标体系
3. 布局设计
4. 可视化
5. 刷新机制
6. 运维保障
```

## 示例
### 示例1
- 输入：电商运营团队需要监控核心指标
- 输出：业务看板，含DAU、GMV、转化、库存等核心指标

## 注意事项
- 要聚焦关键
- 要更新及时
- 要分层清晰
