# 特性优先级 (Features Prioritization)

## 触发条件
- 需求排序
- 迭代规划
- 资源分配
- 价值最大化

## 输入
- 需求列表
- 价值评估
- 开发成本
- 依赖关系

## 输出
- 优先级排序
- 迭代计划
- 资源分配
- 风险提示

## Prompt模板
```
请排列功能优先级。

背景：
- 需求列表：[列表]
- 价值评估：[价值]
- 开发成本：[成本]
- 依赖关系：[依赖]

请输出：
1. 优先级排序
2. 迭代计划
3. 资源分配
4. 风险提示
5. 决策依据
```

## 示例
### 示例1
- 输入：需求池
- 输出：功能优先级，MoSCoW法则、RICE评分

## 注意事项
- 要价值导向
- 要成本效益
- 要依赖关系
- 要持续调整
