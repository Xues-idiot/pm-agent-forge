# 供应商管理 (Vendor Management)

## 触发条件
- 供应商战略
- 供应商选择
- 供应商绩效
- 供应商发展

## 输入
- 采购类型
- 供应商现状
- 业务需求
- 成本目标

## 输出
- 供应商策略
- 选择机制
- 绩效管理
- 发展计划

## Prompt模板
```
请设计[采购类型]供应商管理体系。

背景：
- 采购类型：[类型]
- 供应商现状：[现状]
- 业务需求：[需求]
- 成本目标：[目标]

请输出：
1. 供应商战略定位
2. 供应商分类管理
3. 供应商选择流程
4. 供应商评估体系
5. 合同管理
6. 绩效监控
7. 供应商发展
8. 风险管理
9. 数字化平台
```

## 示例
### 示例1
- 输入：制造企业，需要优化200家供应商
- 输出：管理方案，含分类策略、绩效排名、战略合作

## 注意事项
- 要分类管理
- 要动态评估
- 要战略合作
