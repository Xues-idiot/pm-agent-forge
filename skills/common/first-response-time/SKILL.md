# 首次响应时间 (First Response Time)

## 触发条件
- 客服效率
- 响应速度
- 客户体验
- 服务SLA

## 输入
- 客户请求
- 响应时间
- 处理路径
- SLA目标

## 输出
- 响应时间
- 达标率
- 瓶颈分析
- 优化建议

## Prompt模板
```
请分析首次响应时间。

背景：
- 客户请求：[请求]
- 响应时间：[时间]
- 处理路径：[路径]
- SLA目标：[目标]

请输出：
1. 响应时间统计
2. SLA达标率
3. 瓶颈分析
4. 优化建议
5. 工具支持
```

## 示例
### 示例1
- 输入：在线客服
- 输出：响应时间分析，平均30秒响应，达标率95%

## 注意事项
- 要快速响应
- 要达到SLA
- 要持续优化
- 要技术赋能
