# 债务管理 (Debt Management)

## 触发条件
- 负债融资
- 杠杆使用
- 偿债能力
- 信用管理

## 输入
- 债务结构
- 利率水平
- 偿债计划
- 融资能力

## 输出
- 债务策略
- 融资方案
- 偿债计划
- 风险控制

## Prompt模板
```
请管理债务。

背景：
- 债务结构：[结构]
- 利率水平：[利率]
- 偿债计划：[计划]
- 融资能力：[能力]

请输出：
1. 债务策略
2. 融资方案
3. 偿债计划
4. 风险控制
5. 成本优化
```

## 示例
### 示例1
- 输入：企业负债
- 输出：债务管理，银行贷款、债券发行、融资租赁

## 注意事项
- 要杠杆适度
- 要成本最低
- 要期限匹配
- 要持续监控
