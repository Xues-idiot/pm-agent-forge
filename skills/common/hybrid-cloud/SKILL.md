# 混合云架构 (Hybrid Cloud)

## 触发条件
- 混合云
- 统一管理
- 数据合规
- 成本优化

## 输入
- 业务需求
- 现有架构
- 合规要求
- 成本目标

## 输出
- 混合云架构
- 网络设计
- 数据管理
- 运维方案

## Prompt模板
```
请设计混合云架构。

背景：
- 业务需求：[需求]
- 现有架构：[架构]
- 合规要求：[要求]
- 成本目标：[目标]

请输出：
1. 混合云架构
2. 网络设计
3. 数据管理
4. 身份认证
5. 运维方案
```

## 示例
### 示例1
- 输入：金融企业
- 输出：混合云架构，含公有云弹性、业务私有云核心、数据专线连接

## 注意事项
- 要安全合规
- 要统一管理
- 要网络可靠
- 要成本合理
