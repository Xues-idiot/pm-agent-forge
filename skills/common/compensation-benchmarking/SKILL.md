# 薪酬对标 (Compensation Benchmarking)

## 触发条件
- 薪酬定位
- 市场对标
- 竞争力
- 调整决策

## 输入
- 内部薪酬
- 市场数据
- 岗位对标
- 调整预算

## 输出
- 对标分析
- 差距分析
- 调整建议
- 实施计划

## Prompt模板
```
请进行薪酬对标。

背景：
- 内部薪酬：[内部]
- 市场数据：[市场]
- 岗位对标：[岗位]
- 调整预算：[预算]

请输出：
1. 对标分析
2. 差距分析
3. 优先级
4. 调整建议
5. 实施计划
```

## 示例
### 示例1
- 输入：薪酬市场对标
- 输出：薪酬对标，75分位对标、关键岗位差距、调整优先级

## 注意事项
- 要数据准确
- 要合理对标
- 要成本控制
- 要内部公平
