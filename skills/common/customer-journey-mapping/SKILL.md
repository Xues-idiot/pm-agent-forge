# 用户旅程地图 (Customer Journey Mapping)

## 触发条件
- 用户旅程
- 触点分析
- 体验优化
- 跨部门协同

## 输入
- 用户角色
- 业务场景
- 触点数据
- 优化目标

## 输出
- 旅程阶段
- 触点分析
- 痛点识别
- 优化机会

## Prompt模板
```
请绘制用户旅程地图。

背景：
- 用户角色：[角色]
- 业务场景：[场景]
- 触点数据：[数据]
- 优化目标：[目标]

请输出：
1. 旅程阶段
2. 触点梳理
3. 痛点识别
4. 情绪曲线
5. 优化机会
```

## 示例
### 示例1
- 输入：保险购买旅程
- 输出：旅程地图，含认知→比较→投保→理赔各阶段触点、痛点、机会

## 注意事项
- 要用户视角
- 要基于数据
- 要跨渠道
- 要持续迭代
