# 报表自动化 (Report Automation)

## 触发条件
- 报表生成
- 数据汇总
- 定时发送
- 效率提升

## 输入
- 报表需求
- 数据来源
- 发布周期
- 接收人

## 输出
- 报表模板
- 数据抽取
- 自动化流程
- 分发机制

## Prompt模板
```
请设计报表自动化。

背景：
- 报表需求：[需求]
- 数据来源：[来源]
- 发布周期：[周期]
- 接收人：[接收人]

请输出：
1. 报表模板
2. 数据抽取
3. 自动化流程
4. 分发机制
5. 异常处理
```

## 示例
### 示例1
- 输入：月度报表
- 输出：报表自动化，数据同步、模板渲染、定时发送

## 注意事项
- 要数据准确
- 要及时更新
- 要格式规范
- 要异常监控
