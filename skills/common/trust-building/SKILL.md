# 建立信任 (Trust Building)

## 触发条件
- 信任关系
- 诚信沟通
- 可靠性
- 团队信任

## 输入
- 信任对象
- 信任基础
- 历史关系
- 信任障碍

## 输出
- 信任评估
- 建立策略
- 行动建议
- 维护机制

## Prompt模板
```
请建立信任。

背景：
- 信任对象：[对象]
- 信任基础：[基础]
- 历史关系：[历史]
- 信任障碍：[障碍]

请输出：
1. 信任评估
2. 信任基础
3. 建立策略
4. 行动建议
5. 维护机制
```

## 示例
### 示例1
- 输入：跨部门合作
- 输出：建立信任，言行一致、专业能力、主动沟通、兑现承诺

## 注意事项
- 要言行一致
- 要专业能力
- 要主动沟通
- 要兑现承诺
