# 归因分析 (Attribution Modeling)

## 触发条件
- 营销效果
- 渠道评估
- 投资优化
- 转化路径

## 输入
- 转化类型
- 触点数据
- 归因模型
- 业务场景

## 输出
- 归因模型
- 渠道贡献
- ROI分析
- 优化建议

## Prompt模板
```
请进行归因分析。

背景：
- 转化类型：[类型]
- 触点数据：[数据]
- 归因模型：[模型]
- 业务场景：[场景]

请输出：
1. 归因模型选择
2. 各渠道贡献
3. ROI分析
4. 优化建议
```

## 示例
### 示例1
- 输入：多渠道营销归因
- 输出：归因报告，含首次触点、末次触点、多触点模型对比

## 注意事项
- 要模型匹配
- 要数据准确
- 要业务理解
