# 用户访谈 (User Interview)

## 触发条件
- 用户研究
- 需求挖掘
- 痛点发现
- 方案验证

## 输入
- 访谈目标
- 受访对象
- 访谈大纲
- 访谈方式

## 输出
- 访谈记录
- 需求洞察
- 痛点分析
- 改进建议

## Prompt模板
```
请进行用户访谈。

背景：
- 访谈目标：[目标]
- 受访对象：[对象]
- 访谈大纲：[大纲]
- 访谈方式：[方式]

请输出：
1. 访谈记录
2. 需求洞察
3. 痛点分析
4. 改进建议
5. 后续计划
```

## 示例
### 示例1
- 输入：产品调研
- 输出：用户访谈，使用场景、痛点需求、期望功能

## 注意事项
- 要充分准备
- 要开放提问
- 要深度挖掘
- 要客观记录
