# 保留策略 (Retention Policy)

## 触发条件
- 数据管理
- 合规要求
- 存储优化
- 生命周期

## 输入
- 数据类型
- 法规要求
- 业务需求
- 存储成本

## 输出
- 保留策略
- 分级保留
- 销毁机制
- 合规审计

## Prompt模板
```
请制定保留策略。

背景：
- 数据类型：[类型]
- 法规要求：[法规]
- 业务需求：[需求]
- 存储成本：[成本]

请输出：
1. 保留策略
2. 分级保留
3. 销毁机制
4. 合规审计
5. 例外处理
```

## 示例
### 示例1
- 输入：企业数据
- 输出：保留策略，根据数据类型设定不同的保留期和销毁方式

## 注意事项
- 要法规合规
- 要业务需求
- 要成本效益
- 要定期审计
