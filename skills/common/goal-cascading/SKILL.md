# 目标分解 (Goal Cascading)

## 触发条件
- 目标分解
- 责任落实
- 执行对齐
- 效果评估

## 输入
- 上级目标
- 下级能力
- 资源条件
- 时间约束

## 输出
- 目标分解
- 子目标
- 责任分工
- 协同要求

## Prompt模板
```
请进行目标分解。

背景：
- 上级目标：[上级]
- 下级能力：[能力]
- 资源条件：[资源]
- 时间约束：[约束]

请输出：
1. 目标分解
2. 子目标设定
3. 责任分工
4. 协同要求
5. 评估标准
```

## 示例
### 示例1
- 输入：公司目标分解到部门
- 输出：目标分解，公司→事业部→部门→团队→个人

## 注意事项
- 要SMART目标
- 要可衡量
- 要责任明确
- 要协同清晰
