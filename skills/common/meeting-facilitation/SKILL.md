# 会议引导 (Meeting Facilitation)

## 触发条件
- 设计会议议程
- 主持重要会议
- 推动团队决策
- 协调多方讨论

## 输入
- 会议目的
- 参与者
- 时间约束
- 决策事项

## 输出
- 会议设计方案
- 议程安排
- 引导流程
- 会议纪要模板

## Prompt模板
```
请设计[会议]的引导方案。

背景：
- 会议目的：[目的]
- 参与者：[人员]
- 时间安排：[时间]
- 决策事项：[事项]

请输出：
1. 会议目标定义
2. 参会者角色分析
3. 会议议程设计
4. 各环节引导方法
5. 讨论工具选择
6. 决策机制设计
7. 风险预案（偏离话题等）
8. 会议纪要模板
9. 跟进事项跟踪
```

## 示例
### 示例1
- 输入：战略规划会议，高管团队，半天时间，需确定明年战略重点
- 输出：战略会议引导方案，含团队共创方法、决策流程

## 注意事项
- 议程要合理
- 要让所有人参与
- 要达成明确结论
