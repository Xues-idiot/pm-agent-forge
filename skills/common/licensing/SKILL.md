# 授权经营 (Licensing)

## 触发条件
- 品牌输出
- 技术变现
- 轻资产
- 快速扩张

## 输入
- 授权资产
- 被授权方
- 授权范围
- 商业模式

## 输出
- 授权策略
- 协议设计
- 质量控制
- 收益模式

## Prompt模板
```
请制定授权策略。

背景：
- 授权资产：[资产]
- 被授权方：[被授权]
- 授权范围：[范围]
- 商业模式：[模式]

请输出：
1. 授权策略
2. 授权协议
3. 质量控制
4. 收益模式
5. 风险控制
```

## 示例
### 示例1
- 输入：品牌授权
- 输出：授权经营，品牌标准、品质要求、销售分成

## 注意事项
- 要资产保护
- 要品质控制
- 要互利共赢
- 要持续监督
