# 差距分析 (Gap Analysis)

## 触发条件
- 现状评估
- 目标差距
- 改进需求
- 路径规划

## 输入
- 当前状态
- 目标状态
- 差距维度
- 优先级

## 输出
- 差距清单
- 原因分析
- 弥合路径
- 资源需求

## Prompt模板
```
请进行差距分析。

背景：
- 当前状态：[现状]
- 目标状态：[目标]
- 差距维度：[维度]
- 优先级：[优先级]

请输出：
1. 差距清单
2. 原因分析
3. 弥合路径
4. 资源需求
5. 时间计划
```

## 示例
### 示例1
- 输入：能力差距
- 输出：差距分析，技能差距、工具差距、流程差距

## 注意事项
- 要客观评估
- 要具体量化
- 要分清主次
- 要可操作性
