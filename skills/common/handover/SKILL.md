# 交接管理 (Handover Management)

## 触发条件
- 岗位交接
- 项目交接
- 知识传递
- 平稳过渡

## 输入
- 交接内容
- 接替人员
- 知识范围
- 时间要求

## 输出
- 交接计划
- 知识文档
- 过渡期
- 确认机制

## Prompt模板
```
请管理交接。

背景：
- 交接内容：[内容]
- 接替人员：[人员]
- 知识范围：[范围]
- 时间要求：[要求]

请输出：
1. 交接清单
2. 知识文档
3. 过渡计划
4. 确认机制
5. 后续支持
```

## 示例
### 示例1
- 输入：核心员工离职
- 输出：交接管理，文档整理、操作手册、交接期培训、一对一辅导

## 注意事项
- 要充分准备
- 要文档齐全
- 要平稳过渡
- 要持续支持
