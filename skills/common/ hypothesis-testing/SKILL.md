# 假设检验 (Hypothesis Testing)

## 触发条件
- 商业假设
- 实验设计
- 决策验证
- 快速验证

## 输入
- 商业假设
- 验证方法
- 资源约束
- 风险承受

## 输出
- 假设清单
- 实验设计
- 验证标准
- 决策建议

## Prompt模板
```
请设计假设验证方案。

背景：
- 商业假设：[假设]
- 验证方法：[方法]
- 资源约束：[资源]
- 风险承受：[风险]

请输出：
1. 假设清单
2. 优先级
3. 实验设计
4. 样本计算
5. 验证标准
6. 决策建议
```

## 示例
### 示例1
- 输入：价格调整假设"提价10%不影响销量"
- 输出：验证方案，含A/B测试设计、样本量、评估指标

## 注意事项
- 要可证伪
- 要成本可控
- 要快速验证
