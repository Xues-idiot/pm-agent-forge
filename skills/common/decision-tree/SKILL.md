# 决策树 (Decision Tree)

## 触发条件
- 决策分析
- 方案选择
- 风险评估
- 逻辑推理

## 输入
- 决策目标
- 可选方案
- 评估标准
- 不确定因素

## 输出
- 决策树结构
- 期望值计算
- 敏感性分析
- 推荐方案

## Prompt模板
```
请进行决策树分析。

背景：
- 决策目标：[目标]
- 可选方案：[方案]
- 评估标准：[标准]
- 不确定因素：[因素]

请输出：
1. 决策树结构
2. 期望值计算
3. 敏感性分析
4. 推荐方案
5. 风险提示
```

## 示例
### 示例1
- 输入：投资决策
- 输出：决策树，最大期望收益、风险偏好、敏感性

## 注意事项
- 要明确定义
- 要量化概率
- 要敏感分析
- 要考虑风险
