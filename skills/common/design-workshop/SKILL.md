# 设计工作坊 (Design Workshop)

## 触发条件
- 共创设计
- 问题解决
- 团队协作
- 快速原型

## 输入
- 设计问题
- 参与者
- 时间安排
- 材料准备

## 输出
- 工作坊设计
- 活动流程
- 产出定义
- 后续计划

## Prompt模板
```
请设计工作坊。

背景：
- 设计问题：[问题]
- 参与者：[参与者]
- 时间安排：[时间]
- 材料准备：[材料]

请输出：
1. 工作坊设计
2. 活动流程
3. 分组安排
4. 产出定义
5. 后续计划
```

## 示例
### 示例1
- 输入：App设计
- 输出：设计工作坊，自我介绍、痛点挖掘、方案构思、原型制作

## 注意事项
- 要明确目标
- 要充分准备
- 要鼓励参与
- 要产出明确
