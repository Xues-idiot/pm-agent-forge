# 并购战略 (Merger Strategy)

## 触发条件
- 并购机会
- 标的评估
- 整合规划
- 价值创造

## 输入
- 战略目标
- 标的筛选
- 估值预期
- 整合能力

## 输出
- 标的筛选
- 估值方法
- 整合规划
- 风险评估

## Prompt模板
```
请制定并购战略。

背景：
- 战略目标：[目标]
- 标的筛选：[标的]
- 估值预期：[估值]
- 整合能力：[整合]

请输出：
1. 并购目标
2. 标的筛选标准
3. 估值方法
4. 整合规划
5. 风险评估
```

## 示例
### 示例1
- 输入：横向并购
- 输出：并购战略，标的筛选标准、估值模型、100天计划

## 注意事项
- 要战略驱动
- 要充分尽调
- 要重视整合
- 要文化匹配
