# 知识管理 (Knowledge Management)

## 触发条件
- 知识体系构建
- 知识库建设
- 知识共享
- 知识变现

## 输入
- 业务类型
- 知识类型
- 现有知识
- 人员规模

## 输出
- 知识管理方案
- 知识库设计
- 共享机制
- 评估体系

## Prompt模板
```
请为[组织]设计知识管理方案。

背景：
- 组织类型：[类型]
- 知识类型：[类型]
- 现有知识：[现状]
- 人员规模：[规模]

请输出：
1. 知识管理目标和愿景
2. 知识分类体系
3. 知识库架构设计
4. 知识获取和沉淀
5. 知识共享机制
6. 知识应用场景
7. 知识管理工具
8. 知识贡献激励
9. 知识安全管理
10. 效果评估和迭代
```

## 示例
### 示例1
- 输入：咨询公司，需要沉淀项目经验
- 输出：知识管理方案，含知识库分类、最佳实践库、知识地图

## 注意事项
- 要与业务结合
- 要持续运营
- 要激励贡献
