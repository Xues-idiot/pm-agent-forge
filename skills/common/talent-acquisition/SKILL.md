# 人才获取 (Talent Acquisition)

## 触发条件
- 招聘需求
- 人才竞争
- 雇主品牌
- 招聘效率

## 输入
- 岗位需求
- 人才市场
- 招聘渠道
- 雇主品牌

## 输出
- 招聘策略
- 渠道选择
- 甄选流程
- 雇主品牌

## Prompt模板
```
请优化人才获取。

背景：
- 岗位需求：[需求]
- 人才市场：[市场]
- 招聘渠道：[渠道]
- 雇主品牌：[品牌]

请输出：
1. 招聘策略
2. 渠道选择
3. 甄选流程
4. 雇主品牌
5. 成本控制
```

## 示例
### 示例1
- 输入：技术招聘
- 输出：人才获取，校园招聘、社会招聘、内部推荐

## 注意事项
- 要雇主品牌
- 要精准渠道
- 要高效流程
- 要候选人体验
