# 数据素养 (Data Literacy)

## 触发条件
- 数据能力培训
- 数据驱动决策
- 数据分析方法
- 数据可视化

## 输入
- 受众群体
- 能力水平
- 业务场景
- 学习目标

## 输出
- 培训方案
- 学习路径
- 实践案例
- 评估体系

## Prompt模板
```
请为[组织/团队]设计数据素养提升方案。

背景：
- 受众群体：[群体]
- 能力水平：[水平]
- 业务场景：[场景]
- 学习目标：[目标]

请输出：
1. 数据素养框架
2. 学习路径设计
3. 培训内容（数据分析/可视化/解读）
4. 实践案例设计
5. 学习方式（培训/实践/认证）
6. 在岗练习计划
7. 效果评估方法
8. 持续学习机制
```

## 示例
### 示例1
- 输入：销售团队20人，需要用数据做客户分析
- 输出：数据素养方案，含基础统计、数据工具、客户分析案例

## 注意事项
- 要结合业务场景
- 要循序渐进
- 要学以致用
