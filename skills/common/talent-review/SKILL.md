# 人才盘点 (Talent Review)

## 触发条件
- 人才评估
- 盘点分析
- 发展建议
- 规划决策

## 输入
- 人才数据
- 能力评估
- 绩效数据
- 发展意愿

## 输出
- 人才地图
- 分类分析
- 发展建议
- 规划决策

## Prompt模板
```
请进行人才盘点。

背景：
- 人才数据：[数据]
- 能力评估：[能力]
- 绩效数据：[绩效]
- 发展意愿：[意愿]

请输出：
1. 人才地图
2. 九宫格分析
3. 分类分析
4. 发展建议
5. 规划建议
```

## 示例
### 示例1
- 输入：年度人才盘点
- 输出：人才盘点，高绩效高潜力、高绩效低潜力、九宫格分布

## 注意事项
- 要数据客观
- 要多维度
- 要发展导向
- 要持续跟踪
