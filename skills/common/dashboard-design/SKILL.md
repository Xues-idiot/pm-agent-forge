# 数据看板 (Dashboard Design)

## 触发条件
- 数据可视化
- 实时监控
- 业务洞察
- 移动端

## 输入
- 监控场景
- 数据指标
- 用户角色
- 访问设备

## 输出
- 看板架构
- 布局设计
- 刷新策略
- 交互设计

## Prompt模板
```
请设计数据看板。

背景：
- 监控场景：[场景]
- 数据指标：[指标]
- 用户角色：[角色]
- 访问设备：[设备]

请输出：
1. 看板架构
2. 布局设计
3. 指标定义
4. 刷新策略
5. 交互设计
```

## 示例
### 示例1
- 输入：运营监控中心
- 输出：数据看板，含实时数据大屏、KPI卡片、趋势图、异常告警

## 注意事项
- 要信息层次
- 要实时准确
- 要响应式
- 要性能优化
