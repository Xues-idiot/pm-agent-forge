# 知识库管理 (Knowledge Base)

## 触发条件
- 知识沉淀
- 信息共享
- 客服支持
- 培训材料

## 输入
- 知识来源
- 知识类型
- 使用场景
- 更新频率

## 输出
- 知识结构
- 内容组织
- 维护机制
- 检索优化

## Prompt模板
```
请管理知识库。

背景：
- 知识来源：[来源]
- 知识类型：[类型]
- 使用场景：[场景]
- 更新频率：[频率]

请输出：
1. 知识结构
2. 内容组织
3. 维护机制
4. 检索优化
5. 质量控制
```

## 示例
### 示例1
- 输入：客服知识库
- 输出：知识库，含FAQ、产品知识、故障排查、话术指南

## 注意事项
- 要结构清晰
- 要易于检索
- 要持续更新
- 要质量保证
