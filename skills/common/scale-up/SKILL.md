# 规模化 (Scale Up)

## 触发条件
- 业务增长
- 组织扩张
- 流程复制
- 风险控制

## 输入
- 增长目标
- 当前规模
- 瓶颈问题
- 资源能力

## 输出
- 规模化策略
- 关键里程碑
- 组织规划
- 风险预案

## Prompt模板
```
请制定规模化方案。

背景：
- 增长目标：[目标]
- 当前规模：[规模]
- 瓶颈问题：[瓶颈]
- 资源能力：[能力]

请输出：
1. 规模化策略
2. 关键里程碑
3. 组织规划
4. 流程复制
5. 系统支撑
6. 风险预案
```

## 示例
### 示例1
- 输入：SaaS产品从1亿到10亿规模
- 输出：规模化方案，含组织调整、流程优化、系统升级

## 注意事项
- 要有序扩张
- 要能力跟上
- 要风险控制
