# 职业路径 (Career Path)

## 触发条件
- 职业规划
- 发展通道
- 能力提升
- 人才保留

## 输入
- 职业兴趣
- 能力现状
- 发展目标
- 机会识别

## 输出
- 职业规划
- 发展路径
- 能力差距
- 行动计划

## Prompt模板
```
请设计职业路径。

背景：
- 职业兴趣：[兴趣]
- 能力现状：[现状]
- 发展目标：[目标]
- 机会识别：[机会]

请输出：
1. 职业志向
2. 发展路径
3. 能力差距
4. 发展计划
5. 时间规划
```

## 示例
### 示例1
- 输入：员工职业发展
- 输出：职业路径，专业通道、管理通道、双通道发展

## 注意事项
- 要个性化
- 要双向沟通
- 要持续跟踪
- 要与业务结合
