# 缓存策略 (Cache Strategy)

## 触发条件
- 缓存设计
- 性能优化
- 一致性
- 成本控制

## 输入
- 访问特点
- 数据特性
- 性能要求
- 成本预算

## 输出
- 缓存架构
- 淘汰策略
- 一致性方案
- 监控运维

## Prompt模板
```
请制定缓存策略。

背景：
- 访问特点：[特点]
- 数据特性：[特性]
- 性能要求：[要求]
- 成本预算：[预算]

请输出：
1. 缓存架构
2. 缓存策略
3. 淘汰机制
4. 一致性方案
5. 监控运维
```

## 示例
### 示例1
- 输入：热点数据
- 输出：缓存策略，含Redis多级缓存、LRU淘汰、Cache Aside模式

## 注意事项
- 要数据分层
- 要策略合理
- 要一致性平衡
- 要容量规划
