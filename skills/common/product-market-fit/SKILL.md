# 产品市场匹配 (Product-Market Fit)

## 触发条件
- 产品验证
- 市场进入
- 增长策略
- 投资决策

## 输入
- 产品价值
- 用户反馈
- 市场反应
- 竞争态势

## 输出
- PMF评估
- 信号指标
- 达成路径
- 下一阶段

## Prompt模板
```
请评估产品市场匹配。

背景：
- 产品价值：[价值]
- 用户反馈：[反馈]
- 市场反应：[反应]
- 竞争态势：[态势]

请输出：
1. PMF定义
2. 信号指标
3. 当前状态
4. 达成路径
5. 下一阶段
```

## 示例
### 示例1
- 输入：新产品早期用户反馈好
- 输出：PMF评估，含信号指标、达成评估、下一步建议

## 注意事项
- 要用户认可
- 要市场验证
- 要持续优化
