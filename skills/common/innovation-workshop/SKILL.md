# 创新工作坊 (Innovation Workshop)

## 触发条件
- 创新思维
- 问题解决
- 团队创意
- 创新文化

## 输入
- 创新主题
- 参与者
- 时间安排
- 创新方法

## 输出
- 工作坊设计
- 创新方法
- 产出定义
- 后续计划

## Prompt模板
```
请设计创新工作坊。

背景：
- 创新主题：[主题]
- 参与者：[参与]
- 时间安排：[时间]
- 创新方法：[方法]

请输出：
1. 工作坊设计
2. 创新方法应用
3. 活动流程
4. 产出定义
5. 后续计划
```

## 示例
### 示例1
- 输入：业务创新
- 输出：创新工作坊，SCAMPER、类比思考、原型验证

## 注意事项
- 要开放心态
- 要跨界融合
- 要快速行动
- 要持续迭代
