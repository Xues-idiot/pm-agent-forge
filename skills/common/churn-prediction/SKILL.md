# 流失预测 (Churn Prediction)

## 触发条件
- 客户流失
- 风险预警
- 留存分析
- 挽留策略

## 输入
- 流失定义
- 历史数据
- 行为特征
- 流失阶段

## 输出
- 流失模型
- 风险评分
- 流失因素
- 挽留策略

## Prompt模板
```
请预测客户流失。

背景：
- 流失定义：[定义]
- 历史数据：[历史]
- 行为特征：[特征]
- 流失阶段：[阶段]

请输出：
1. 流失模型
2. 风险评分
3. 流失因素
4. 挽留策略
5. 监控指标
```

## 示例
### 示例1
- 输入：订阅服务
- 输出：流失预测，高风险客户、流失预警、挽留活动

## 注意事项
- 要早期预警
- 要精准识别
- 要主动干预
- 要成本效益
