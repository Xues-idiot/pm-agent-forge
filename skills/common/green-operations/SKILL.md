# 绿色运营 (Green Operations)

## 触发条件
- 节能减排
- 环境保护
- 成本降低
- ESG绩效

## 输入
- 能耗数据
- 排放数据
- 环保措施
- 成本效益

## 输出
- 现状分析
- 优化方案
- 减排目标
- 实施路径

## Prompt模板
```
请推进绿色运营。

背景：
- 能耗数据：[能耗]
- 排放数据：[排放]
- 环保措施：[措施]
- 成本效益：[效益]

请输出：
1. 现状分析
2. 优化方案
3. 减排目标
4. 实施路径
5. 评估指标
```

## 示例
### 示例1
- 输入：绿色工厂
- 输出：绿色运营，节能改造、清洁能源、循环经济

## 注意事项
- 要经济效益
- 要技术可行
- 要持续改进
- 要全员参与
