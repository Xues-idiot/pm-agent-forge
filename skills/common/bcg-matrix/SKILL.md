# BCG矩阵 (BCG Matrix)

## 触发条件
- 产品组合
- 投资决策
- 业务规划
- 资源配置

## 输入
- 产品/业务
- 市场增长率
- 市场份额
- 盈利情况

## 输出
- BCG矩阵定位
- 策略建议
- 投资优先级
- 资源配置

## Prompt模板
```
请用BCG矩阵分析业务组合。

背景：
- 产品/业务：[业务]
- 市场增长率：[增长率]
- 市场份额：[份额]
- 盈利情况：[盈利]

请输出：
1. BCG矩阵定位
2. 问题业务
3. 明星业务
4. 金牛业务
5. 瘦狗业务
6. 策略建议
7. 投资优先级
```

## 示例
### 示例1
- 输入：公司有4条产品线
- 输出：BCG分析，含各产品定位、策略建议、资源配置

## 注意事项
- 要数据准确
- 要动态视角
- 要资源平衡
