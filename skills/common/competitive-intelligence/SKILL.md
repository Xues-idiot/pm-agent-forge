# 竞争情报 (Competitive Intelligence)

## 触发条件
- 竞争对手分析
- 市场监控
- 竞争策略
- 早期预警

## 输入
- 竞争对手
- 信息来源
- 分析目的
- 行动周期

## 输出
- 情报报告
- 竞争分析
- 策略建议
- 预警机制

## Prompt模板
```
请进行[竞争对手]竞争情报分析。

背景：
- 竞争对手：[对手]
- 信息来源：[来源]
- 分析目的：[目的]
- 行动周期：[周期]

请输出：
1. 竞争对手概况
2. 产品分析
3. 定价策略
4. 渠道策略
5. 营销策略
6. 优劣势
7. 动向监控
8. 预警机制
```

## 示例
### 示例1
- 输入：主要竞品动态监控，需要月度报告
- 输出：情报报告，含产品更新、定价变化、市场份额

## 注意事项
- 要合法合规
- 要持续收集
- 要行动导向
