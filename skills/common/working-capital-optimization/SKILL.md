# 营运资本优化 (Working Capital Optimization)

## 触发条件
- 资金效率
- 流动性
- 成本控制
- 运营改善

## 输入
- 存货水平
- 应收账款
- 应付账款
- 现金周期

## 输出
- 优化方案
- 目标指标
- 实施路径
- 监控机制

## Prompt模板
```
请优化营运资本。

背景：
- 存货水平：[存货]
- 应收账款：[应收]
- 应付账款：[应付]
- 现金周期：[周期]

请输出：
1. 优化方案
2. 目标指标
3. 实施路径
4. 监控机制
5. 预期效果
```

## 示例
### 示例1
- 输入：企业运营
- 输出：营运资本优化，存货压缩、账期优化、现金周期缩短

## 注意事项
- 要流动性
- 要效率提升
- 要成本降低
- 要持续改进
