# 营运资本优化 (Working Capital Optimization)

## 触发条件
- 资金效率
- 流动性
- 成本控制
- 经营效率

## 输入
- 存货水平
- 应收账款
- 应付账款
- 现金管理

## 输出
- 现状分析
- 优化方案
- 目标设定
- 实施路径

## Prompt模板
```
请优化营运资本。

背景：
- 存货水平：[存货]
- 应收账款：[应收]
- 应付账款：[应付]
- 现金管理：[现金]

请输出：
1. 现状分析
2. 优化方案
3. 目标设定
4. 实施路径
5. 预期效果
```

## 示例
### 示例1
- 输入：企业营运资本
- 输出：营运资本优化，存货周转、账期优化、现金管理

## 注意事项
- 要流动性
- 要效率优先
- 要平衡关系
- 要持续改进
