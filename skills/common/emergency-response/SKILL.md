# 应急响应 (Emergency Response)

## 触发条件
- 紧急事件
- 人员安全
- 财产保护
- 快速处置

## 输入
- 紧急类型
- 影响程度
- 响应资源
- 处置权限

## 输出
- 响应指令
- 处置措施
- 疏散路线
- 协调机制

## Prompt模板
```
请进行应急响应。

背景：
- 紧急类型：[类型]
- 影响程度：[程度]
- 响应资源：[资源]
- 处置权限：[权限]

请输出：
1. 响应指令
2. 处置措施
3. 疏散路线
4. 协调机制
5. 事后处理
```

## 示例
### 示例1
- 输入：火灾事件
- 输出：应急响应，疏散撤离、灭火处置、医疗救护

## 注意事项
- 要人员优先
- 要快速响应
- 要统一指挥
- 要预案执行
