# 组织设计 (Organizational Design)

## 触发条件
- 组织架构
- 职责分配
- 流程优化
- 效率提升

## 输入
- 战略要求
- 业务流程
- 人员能力
- 管控模式

## 输出
- 架构方案
- 职责划分
- 管控界面
- 实施路径

## Prompt模板
```
请设计组织架构。

背景：
- 战略要求：[要求]
- 业务流程：[流程]
- 人员能力：[能力]
- 管控模式：[模式]

请输出：
1. 架构方案
2. 职责划分
3. 管控界面
4. 实施路径
5. 风险评估
```

## 示例
### 示例1
- 输入：集团管控
- 输出：组织设计，战略管控、财务管控、运营管控

## 注意事项
- 要战略导向
- 要流程贯通
- 要权责清晰
- 要高效协同
