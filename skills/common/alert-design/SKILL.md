# 告警设计 (Alert Design)

## 触发条件
- 异常监控
- 阈值设置
- 告警分级
- 响应流程

## 输入
- 监控指标
- 正常范围
- 影响程度
- 响应资源

## 输出
- 告警规则
- 阈值设置
- 分级策略
- 响应流程

## Prompt模板
```
请设计告警。

背景：
- 监控指标：[指标]
- 正常范围：[范围]
- 影响程度：[程度]
- 响应资源：[资源]

请输出：
1. 告警规则
2. 阈值设置
3. 分级策略
4. 响应流程
5. 升级机制
```

## 示例
### 示例1
- 输入：系统监控
- 输出：告警设计，P1/P2/P3/P4分级，自动告警+人工确认

## 注意事项
- 要精准告警
- 要分级合理
- 要及时响应
- 要减少噪音
