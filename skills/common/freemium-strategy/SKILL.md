# 免费增值策略 (Freemium Strategy)

## 触发条件
- 获客模式
- 转化策略
- 用户基数
- 盈利模式

## 输入
- 免费价值
- 付费价值
- 转化率
- 成本结构

## 输出
- Freemium设计
- 转化漏斗
- 转化策略
- 单位经济

## Prompt模板
```
请制定Freemium策略。

背景：
- 免费价值：[免费]
- 付费价值：[付费]
- 转化率：[转化]
- 成本结构：[成本]

请输出：
1. Freemium设计
2. 免费功能
3. 付费功能
4. 转化漏斗
5. 单位经济
```

## 示例
### 示例1
- 输入：在线工具
- 输出：Freemium设计，免费基础功能，高级功能付费，5%转化率

## 注意事项
- 要免费有价值
- 要付费有理由
- 要转化自然
- 要单位经济正
