# 客户体验管理 (Customer Experience)

## 触发条件
- 体验优化
- 满意度提升
- 触点管理
- 体验设计

## 输入
- 客户旅程
- 触点数据
- 反馈声音
- 行业标杆

## 输出
- 体验策略
- 触点优化
- 监测体系
- 改进计划

## Prompt模板
```
请管理客户体验。

背景：
- 客户旅程：[旅程]
- 触点数据：[数据]
- 反馈声音：[反馈]
- 行业标杆：[标杆]

请输出：
1. 体验策略
2. 关键触点优化
3. 体验监测体系
4. 改进计划
5. 预期效果
```

## 示例
### 示例1
- 输入：电商平台，NPS下降
- 输出：体验管理，含购物流程简化、退换货优化、客服响应提速

## 注意事项
- 要端到端视角
- 要数据驱动
- 要跨部门协同
- 要持续监测
