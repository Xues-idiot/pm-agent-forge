# 用户分群 (User Segmentation)

## 触发条件
- 用户洞察
- 精准运营
- 个性化
- 价值分层

## 输入
- 用户特征
- 行为数据
- 分群目的
- 应用场景

## 输出
- 分群方案
- 群组特征
- 运营策略
- 效果评估

## Prompt模板
```
请进行用户分群。

背景：
- 用户特征：[特征]
- 行为数据：[数据]
- 分群目的：[目的]
- 应用场景：[场景]

请输出：
1. 分群方案
2. 群组特征
3. 运营策略
4. 效果评估
```

## 示例
### 示例1
- 输入：电商用户分群
- 输出：分群报告，含RFM分群、各群特征、运营策略

## 注意事项
- 要业务导向
- 要可操作性
- 要持续优化
