# 敏捷工作 (Agile Work)

## 触发条件
- 敏捷转型
- Scrum实施
- 敏捷项目管理
- 团队敏捷化

## 输入
- 组织规模
- 当前模式
- 转型目标
- 团队特征

## 输出
- 敏捷转型方案
- Scrum流程
- 角色设计
- 评估机制

## Prompt模板
```
请为[团队/组织]设计敏捷转型方案。

背景：
- 组织规模：[规模]
- 当前模式：[模式]
- 转型目标：[目标]
- 团队特征：[特征]

请输出：
1. 敏捷转型路径
2. Scrum流程设计
3. 角色和职责（Scrum Master/PO）
4. 仪式设计（Sprint/Planning/Daily/Review/Retro）
5. 度量体系
6. 团队结构设计
7. 规模化方案（如需要）
8. 常见问题和应对
9. 转型效果评估
```

## 示例
### 示例1
- 输入：50人研发团队，从瀑布模式转向敏捷
- 输出：转型方案，含Scrum流程、角色转换、度量指标

## 注意事项
- 要渐进式转型
- 要领导支持
- 要持续改进
