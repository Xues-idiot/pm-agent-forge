# ESG战略 (ESG Strategy)

## 触发条件
- ESG战略制定
- 可持续发展
- 碳中和
- 社会责任

## 输入
- 行业类型
- ESG现状
- 利益相关方
- 目标市场

## 输出
- ESG战略
- 目标设定
- 实施路径
- 评估机制

## Prompt模板
```
请为[组织]制定ESG战略。

背景：
- 行业类型：[行业]
- ESG现状：[现状]
- 利益相关方：[相关方]
- 目标市场：[市场]

请输出：
1. ESG现状评估
2. 实质性议题识别
3. ESG战略目标
4. 环境（E）目标和措施
5. 社会（S）目标和措施
6. 治理（G）目标和措施
7. 实施路径
8. 资源投入
9. 信息披露
10. 效果评估
```

## 示例
### 示例1
- 输入：制造企业，出口欧美，需要满足ESG披露要求
- 输出：ESG战略，含碳减排目标、供应链责任、治理结构

## 注意事项
- 要实质性议题
- 要利益相关方参与
- 要持续改进
