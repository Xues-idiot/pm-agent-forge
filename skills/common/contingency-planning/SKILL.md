# 应急计划 (Contingency Planning)

## 触发条件
- 风险应对
- 业务连续性
- 灾难恢复
- 预案制定

## 输入
- 潜在风险
- 业务影响
- 恢复需求
- 可用资源

## 输出
- 应急场景
- 响应流程
- 资源调配
- 恢复计划

## Prompt模板
```
请制定应急计划。

背景：
- 潜在风险：[风险]
- 业务影响：[影响]
- 恢复需求：[需求]
- 可用资源：[资源]

请输出：
1. 应急场景
2. 触发条件
3. 响应流程
4. 资源调配
5. 恢复计划
```

## 示例
### 示例1
- 输入：电商平台大促
- 输出：应急计划，含服务器扩容、应急预案、值班安排、监控告警

## 注意事项
- 要切合实际
- 要定期演练
- 要简洁可操作
- 要持续更新
