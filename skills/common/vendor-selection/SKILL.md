# 供应商选择 (Vendor Selection)

## 触发条件
- 供应商评估
- 采购决策
- 招投标
- 战略合作

## 输入
- 采购需求
- 供应商名单
- 评估标准
- 招标要求

## 输出
- 供应商评估
- 评分排名
- 推荐建议
- 谈判策略

## Prompt模板
```
请选择供应商。

背景：
- 采购需求：[需求]
- 供应商名单：[名单]
- 评估标准：[标准]
- 招标要求：[要求]

请输出：
1. 供应商评估
2. 评分排名
3. 推荐建议
4. 谈判策略
5. 合同条款
```

## 示例
### 示例1
- 输入：IT采购
- 输出：供应商选择，技术能力、价格、服务、历史业绩

## 注意事项
- 要全面评估
- 要公平公正
- 要综合考量
- 要风险控制
