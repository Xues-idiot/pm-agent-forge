# 全渠道运营 (Omnichannel Strategy)

## 触发条件
- 渠道整合
- 客户体验
- 数据打通
- 运营协同

## 输入
- 渠道现状
- 客户旅程
- 技术能力
- 组织结构

## 输出
- 全渠道策略
- 渠道整合
- 技术方案
- 运营机制

## Prompt模板
```
请设计全渠道运营策略。

背景：
- 渠道现状：[现状]
- 客户旅程：[旅程]
- 技术能力：[能力]
- 组织结构：[结构]

请输出：
1. 全渠道目标
2. 渠道角色定位
3. 客户旅程整合
4. 数据打通
5. 技术架构
6. 运营协同
7. KPI体系
```

## 示例
### 示例1
- 输入：零售品牌，线上线下割裂
- 输出：全渠道方案，含订单统一、库存共享、会员一体

## 注意事项
- 要客户中心
- 要数据打通
- 要组织协同
