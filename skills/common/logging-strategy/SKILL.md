# 日志策略 (Logging Strategy)

## 触发条件
- 日志收集
- 问题排查
- 合规审计
- 安全分析

## 输入
- 日志来源
- 日志类型
- 存储需求
- 保留策略

## 输出
- 日志规范
- 收集方案
- 存储设计
- 分析工具

## Prompt模板
```
请制定日志策略。

背景：
- 日志来源：[来源]
- 日志类型：[类型]
- 存储需求：[需求]
- 保留策略：[策略]

请输出：
1. 日志规范
2. 收集方案
3. 存储设计
4. 分析工具
5. 成本控制
```

## 示例
### 示例1
- 输入：应用系统
- 输出：日志策略，结构化日志、集中存储、ELK分析

## 注意事项
- 要结构化
- 要集中化
- 要保留合规
- 要成本优化
