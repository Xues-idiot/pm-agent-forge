# 用户画像 (Persona Development)

## 触发条件
- 用户分类
- 需求理解
- 精准营销
- 产品设计

## 输入
- 用户数据
- 行为特征
- 业务场景
- 画像目的

## 输出
- 画像维度
- 群体划分
- 标签体系
- 应用场景

## Prompt模板
```
请开发用户画像。

背景：
- 用户数据：[数据]
- 行为特征：[特征]
- 业务场景：[场景]
- 画像目的：[目的]

请输出：
1. 画像维度
2. 群体划分
3. 标签体系
4. 画像描述
5. 应用场景
```

## 示例
### 示例1
- 输入：电商平台
- 输出：用户画像，含基础属性、购买行为、偏好特征、价值分层

## 注意事项
- 要数据驱动
- 要动态更新
- 要业务导向
- 要隐私合规
