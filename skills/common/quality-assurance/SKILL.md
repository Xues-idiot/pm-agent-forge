# 质量保障 (Quality Assurance)

## 触发条件
- 质量控制
- 流程规范
- 缺陷预防
- 持续改进

## 输入
- 质量标准
- 工作流程
- 质量问题
- 资源投入

## 输出
- 质量框架
- 流程规范
- 检测机制
- 改进计划

## Prompt模板
```
请建立质量保障体系。

背景：
- 质量标准：[标准]
- 工作流程：[流程]
- 质量问题：[问题]
- 资源投入：[资源]

请输出：
1. 质量框架
2. 质量标准
3. 检测机制
4. 缺陷管理
5. 持续改进
```

## 示例
### 示例1
- 输入：软件开发流程
- 输出：QA体系，含代码规范、Code Review、自动化测试、上线检查

## 注意事项
- 要预防为主
- 要全流程覆盖
- 要数据驱动
- 要持续改进
