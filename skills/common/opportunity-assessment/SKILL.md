# 机会评估 (Opportunity Assessment)

## 触发条件
- 商业机会
- 市场洞察
- 投资决策
- 优先级排序

## 输入
- 机会描述
- 市场潜力
- 竞争态势
- 资源需求

## 输出
- 评估框架
- 市场规模
- 风险分析
- 投资建议

## Prompt模板
```
请评估商业机会。

背景：
- 机会描述：[描述]
- 市场潜力：[潜力]
- 竞争态势：[竞争]
- 资源需求：[需求]

请输出：
1. 机会评估框架
2. 市场规模
3. 竞争分析
4. 风险评估
5. 投资建议
```

## 示例
### 示例1
- 输入：新产品线机会
- 输出：机会评估，含TAM/SAM/SOM、竞争门槛、盈利模型、风险因素

## 注意事项
- 要基于数据
- 要考虑时机
- 要评估能力
- 要决策果断
