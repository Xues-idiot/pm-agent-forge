# 帕累托分析 (Pareto Analysis)

## 触发条件
- 问题优先级
- 资源分配
- 重点突破
- 效果最大化

## 输入
- 问题列表
- 发生频率
- 影响程度
- 解决成本

## 输出
- 排列图
- 分类分析
- 优先级
- 资源建议

## Prompt模板
```
请进行帕累托分析。

背景：
- 问题列表：[列表]
- 发生频率：[频率]
- 影响程度：[影响]
- 解决成本：[成本]

请输出：
1. 排列图
2. 80/20分析
3. 问题分类
4. 优先级建议
5. 资源分配
```

## 示例
### 示例1
- 输入：客户投诉分析
- 输出：帕累托分析，80%投诉来自20%问题类型，重点解决Top3

## 注意事项
- 要数据准确
- 要客观分类
- 要重点突破
- 要持续跟踪
