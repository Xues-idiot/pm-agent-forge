# 资产追踪 (Asset Tracking)

## 触发条件
- 资产管理
- 资产盘点
- 资产折旧
- 资产优化

## 输入
- 资产清单
- 资产位置
- 使用状况
- 折旧政策

## 输出
- 资产台账
- 位置追踪
- 折旧计算
- 处置建议

## Prompt模板
```
请追踪资产。

背景：
- 资产清单：[清单]
- 资产位置：[位置]
- 使用状况：[状况]
- 折旧政策：[政策]

请输出：
1. 资产台账
2. 位置追踪
3. 折旧计算
4. 处置建议
5. 优化方案
```

## 示例
### 示例1
- 输入：企业固定资产
- 输出：资产追踪，RFID标签、位置可视化、折旧管理

## 注意事项
- 要实时准确
- 要责任到人
- 要合规折旧
- 要优化配置
