# 客户获取成本 (Customer Acquisition Cost)

## 触发条件
- 获客效率
- 营销ROI
- 增长策略
- 预算分配

## 输入
- 营销支出
- 新客数量
- 渠道效果
- 客户价值

## 输出
- CAC计算
- 渠道分析
- 优化策略
- LTV/CAC

## Prompt模板
```
请分析客户获取成本。

背景：
- 营销支出：[支出]
- 新客数量：[新客]
- 渠道效果：[渠道]
- 客户价值：[价值]

请输出：
1. CAC计算
2. 渠道CAC对比
3. CAC趋势
4. LTV/CAC比率
5. 优化建议
```

## 示例
### 示例1
- 输入：电商平台
- 输出：CAC分析，总CAC 150元，搜索200元、社媒80元、转介绍50元

## 注意事项
- 要全成本
- 要渠道细分
- 要动态跟踪
- 要优化投入
