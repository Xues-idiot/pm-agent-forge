# 市场进入策略 (Go-to-Market)

## 触发条件
- 新产品上市
- 新市场进入
- 扩张策略
- 首发节奏

## 输入
- 产品特点
- 目标市场
- 竞争态势
- 资源预算

## 输出
- GTM策略
- 渠道选择
- 定价方案
- 时间节奏

## Prompt模板
```
请制定市场进入策略。

背景：
- 产品特点：[特点]
- 目标市场：[市场]
- 竞争态势：[态势]
- 资源预算：[预算]

请输出：
1. GTM策略
2. 渠道策略
3. 定价策略
4. 推广计划
5. 时间节奏
```

## 示例
### 示例1
- 输入：企业级SaaS产品进入新行业
- 输出：GTM策略，含标杆客户优先、行业活动渗透、合作伙伴赋能

## 注意事项
- 要找准切入点
- 要集中资源突破
- 要快速验证市场
- 要灵活调整
