# 监管报告 (Regulatory Reporting)

## 触发条件
- 监管报送
- 合规披露
- 数据质量
- 截止日期

## 输入
- 报告要求
- 数据来源
- 报送频率
- 监管机构

## 输出
- 报告模板
- 数据提取
- 审核流程
- 报送机制

## Prompt模板
```
请管理监管报告。

背景：
- 报告要求：[要求]
- 数据来源：[来源]
- 报送频率：[频率]
- 监管机构：[机构]

请输出：
1. 报告模板
2. 数据提取
3. 审核流程
4. 报送机制
5. 质量控制
```

## 示例
### 示例1
- 输入：金融监管
- 输出：监管报告，人行报表、银保监报表、定期报送

## 注意事项
- 要准确及时
- 要数据可溯
- 要流程规范
- 要持续改进
