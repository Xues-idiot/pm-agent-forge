# 投诉处理 (Complaint Handling)

## 触发条件
- 客户投诉
- 问题解决
- 满意度
- 口碑管理

## 输入
- 投诉内容
- 投诉原因
- 处理历史
- 客户期望

## 输出
- 处理方案
- 解决时限
- 补偿方案
- 预防措施

## Prompt模板
```
请处理客户投诉。

背景：
- 投诉内容：[内容]
- 投诉原因：[原因]
- 处理历史：[历史]
- 客户期望：[期望]

请输出：
1. 问题诊断
2. 处理方案
3. 解决时限
4. 补偿方案
5. 预防措施
```

## 示例
### 示例1
- 输入：产品质量投诉
- 输出：投诉处理，退换货+补偿+原因分析+改善措施

## 注意事项
- 要及时响应
- 要同理沟通
- 要彻底解决
- 要闭环追踪
