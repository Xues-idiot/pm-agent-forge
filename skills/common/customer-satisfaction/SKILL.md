# 客户满意度 (Customer Satisfaction)

## 触发条件
- 用户体验
- 服务质量
- 满意度
- 改进方向

## 输入
- 满意因素
- 不满意
- 期望差距
- 行业基准

## 输出
- 满意度评估
- 关键因素
- 改进机会
- 行动方案

## Prompt模板
```
请分析客户满意度。

背景：
- 满意因素：[满意]
- 不满意：[不满]
- 期望差距：[期望]
- 行业基准：[基准]

请输出：
1. 满意度评估
2. 关键驱动因素
3. 不满意原因
4. 改进机会
5. 行动方案
```

## 示例
### 示例1
- 输入：酒店服务
- 输出：满意度分析，整体4.2分，清洁满意、响应速度不满

## 注意事项
- 要多维度
- 要对比基准
- 要持续跟踪
- 要改进闭环
