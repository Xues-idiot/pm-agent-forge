# 未来研究 (Future Studies)

## 触发条件
- 趋势预测
- 情景规划
- 技术预见
- 战略前瞻

## 输入
- 研究领域
- 时间范围
- 关注焦点
- 战略需求

## 输出
- 趋势分析
- 情景方案
- 技术路线
- 战略建议

## Prompt模板
```
请进行[领域]未来研究。

背景：
- 研究领域：[领域]
- 时间范围：[范围]
- 关注焦点：[焦点]
- 战略需求：[需求]

请输出：
1. 趋势识别
2. 驱动因素分析
3. 关键不确定因素
4. 情景方案（2-4个）
5. 技术路线图
6. 对战略的影响
7. 早期信号监测
8. 战略建议
```

## 示例
### 示例1
- 输入：未来10年教育行业
- 输出：未来研究，含技术趋势、情景方案、战略建议

## 注意事项
- 要多角度
- 要有想象力
- 要基于证据
