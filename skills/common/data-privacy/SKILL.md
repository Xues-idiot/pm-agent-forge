# 数据隐私 (Data Privacy)

## 触发条件
- 隐私保护
- 合规要求
- 数据治理
- 用户信任

## 输入
- 数据类型
- 隐私法规
- 处理目的
- 用户权利

## 输出
- 隐私策略
- 合规清单
- 控制措施
- 响应机制

## Prompt模板
```
请管理数据隐私。

背景：
- 数据类型：[类型]
- 隐私法规：[法规]
- 处理目的：[目的]
- 用户权利：[权利]

请输出：
1. 隐私策略
2. 合规清单
3. 控制措施
4. 响应机制
5. 培训计划
```

## 示例
### 示例1
- 输入：用户数据
- 输出：数据隐私，收集告知、使用授权、访问权限、删除权

## 注意事项
- 要合法合规
- 要用户告知
- 要权限控制
- 要持续监控
