# 责任矩阵 (Responsibility Matrix)

## 触发条件
- 角色明确
- 责任分工
- 协作分工
- 沟通依据

## 输入
- 任务列表
- 团队成员
- 角色定义
- 责任类型

## 输出
- 责任矩阵
- 角色职责
- 协作关系
- 沟通机制

## Prompt模板
```
请制定责任矩阵。

背景：
- 任务列表：[任务]
- 团队成员：[成员]
- 角色定义：[角色]
- 责任类型：[类型]

请输出：
1. 责任矩阵
2. 角色职责
3. RACI定义
4. 协作关系
5. 沟通机制
```

## 示例
### 示例1
- 输入：项目任务分配
- 输出：RACI矩阵，R执行、A负责、C咨询、I知会

## 注意事项
- 要明确责任
- 要避免空白
- 要动态更新
- 要沟通顺畅
