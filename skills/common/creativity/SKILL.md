# 创新思维 (Creative Thinking)

## 触发条件
- 创新方案设计
- 问题解决方案
- 产品创新
- 商业模式创新

## 输入
- 创新领域
- 约束条件
- 现有方案
- 资源情况

## 输出
- 创新方向
- 方案设计
- 评估框架
- 实施路径

## Prompt模板
```
请为[问题/领域]设计创新方案。

背景：
- 问题/领域：[领域]
- 现有方案：[方案]
- 约束条件：[约束]
- 资源情况：[资源]

请用创新思维方法输出：
1. 问题重新定义（5Why/TRIZ）
2. 创新方向发散（类比/反向/跨界）
3. 创新方案构思
4. 方案评估（可行性/影响/风险）
5. 方案细化设计
6. 试点计划
7. 规模化路径
```

## 示例
### 示例1
- 输入：银行网点客户流失问题，传统营销效果差
- 输出：创新方案，含场景金融、社交营销、生态合作

## 注意事项
- 要敢于打破常规
- 要多方借鉴
- 要小步验证
