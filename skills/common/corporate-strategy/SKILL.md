# 公司战略 (Corporate Strategy)

## 触发条件
- 战略方向
- 业务组合
- 资源配置
- 竞争优势

## 输入
- 业务组合
- 市场地位
- 资源能力
- 增长目标

## 输出
- 战略选择
- 业务布局
- 资源配置
- 协同效应

## Prompt模板
```
请制定公司战略。

背景：
- 业务组合：[组合]
- 市场地位：[地位]
- 资源能力：[能力]
- 增长目标：[目标]

请输出：
1. 战略选择
2. 业务布局
3. 资源配置
4. 协同效应
5. 退出策略
```

## 示例
### 示例1
- 输入：多元化集团
- 输出：公司战略，有限多元化、核心业务、新兴业务

## 注意事项
- 要聚焦核心
- 要资源配置
- 要协同效应
- 要动态平衡
