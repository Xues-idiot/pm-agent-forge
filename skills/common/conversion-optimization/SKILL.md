# 转化优化 (Conversion Optimization)

## 触发条件
- 转化率提升
- 用户体验
- 漏斗分析
- A/B测试

## 输入
- 转化漏斗
- 用户行为
- 痛点识别
- 优化目标

## 输出
- 优化假设
- 实验设计
- 实施方案
- 效果评估

## Prompt模板
```
请优化转化率。

背景：
- 转化漏斗：[漏斗]
- 用户行为：[行为]
- 痛点识别：[痛点]
- 优化目标：[目标]

请输出：
1. 转化分析
2. 优化假设
3. 实验设计
4. 实施方案
5. 效果评估
```

## 示例
### 示例1
- 输入：电商下单转化
- 输出：转化优化，含 checkout步骤简化、信任标识、支付方式优化

## 注意事项
- 要数据驱动
- 要小步快跑
- 要用户视角
- 要持续迭代
