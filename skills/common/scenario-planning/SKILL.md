# 情景规划 (Scenario Planning)

## 触发条件
- 战略预判
- 不确定性
- 长期规划
- 韧性建设

## 输入
- 关键趋势
- 不确定性
- 时间范围
- 决策影响

## 输出
- 情景框架
- 情景描述
- 应对策略
- 触发指标

## Prompt模板
```
请进行情景规划。

背景：
- 关键趋势：[趋势]
- 不确定性：[不确定]
- 时间范围：[范围]
- 决策影响：[影响]

请输出：
1. 关键驱动因素
2. 情景框架
3. 各情景描述
4. 应对策略
5. 触发指标
```

## 示例
### 示例1
- 输入：企业5年战略规划
- 输出：情景规划，含快速增长、行业整合、监管收紧三种情景及应对

## 注意事项
- 要多角度
- 要可能性与影响平衡
- 要定性与定量结合
- 要定期更新
