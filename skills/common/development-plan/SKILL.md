# 发展计划 (Development Plan)

## 触发条件
- 人才培养
- 能力提升
- 职业发展
- 继任规划

## 输入
- 能力评估
- 发展需求
- 职业目标
- 资源条件

## 输出
- 发展目标
- 培训计划
- 实践安排
- 评估机制

## Prompt模板
```
请制定发展计划。

背景：
- 能力评估：[评估]
- 发展需求：[需求]
- 职业目标：[目标]
- 资源条件：[资源]

请输出：
1. 发展目标
2. 能力差距
3. 培训计划
4. 实践安排
5. 评估时间
```

## 示例
### 示例1
- 输入：高潜力员工
- 输出：发展计划，能力提升目标、IDP制定、轮岗计划、导师辅导

## 注意事项
- 要个性化
- 要实战锻炼
- 要持续跟踪
- 要与业务结合
