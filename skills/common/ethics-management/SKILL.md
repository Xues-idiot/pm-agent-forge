# 伦理管理 (Ethics Management)

## 触发条件
- 商业伦理
- 道德规范
- 员工行为
- 声誉保护

## 输入
- 伦理标准
- 行业规范
- 风险场景
- 举报机制

## 输出
- 伦理框架
- 行为准则
- 培训计划
- 调查流程

## Prompt模板
```
请管理伦理。

背景：
- 伦理标准：[标准]
- 行业规范：[规范]
- 风险场景：[场景]
- 举报机制：[机制]

请输出：
1. 伦理框架
2. 行为准则
3. 培训计划
4. 调查流程
5. 改进措施
```

## 示例
### 示例1
- 输入：企业伦理
- 输出：伦理管理，行为准则、合规培训、举报热线

## 注意事项
- 要高层承诺
- 要全员培训
- 要举报保护
- 要调查独立
