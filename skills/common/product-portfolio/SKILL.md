# 产品组合管理 (Product Portfolio)

## 触发条件
- 产品矩阵
- 资源分配
- 产品退出
- 产品规划

## 输入
- 产品列表
- 市场表现
- 战略目标
- 资源约束

## 输出
- 组合策略
- 产品分级
- 资源配置
- 退出计划

## Prompt模板
```
请管理产品组合。

背景：
- 产品列表：[列表]
- 市场表现：[表现]
- 战略目标：[目标]
- 资源约束：[约束]

请输出：
1. 组合分析
2. 产品分级
3. 资源配置
4. 产品路线图
5. 退出/收敛计划
```

## 示例
### 示例1
- 输入：多元化企业，多个产品线
- 输出：组合策略，含明星产品重点投入、金牛产品保持利润、问题产品评估去留

## 注意事项
- 要基于数据决策
- 要考虑战略一致性
- 要平衡短期和长期
- 要有序收敛
