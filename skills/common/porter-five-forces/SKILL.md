# 波特五力 (Porter's Five Forces)

## 触发条件
- 行业分析
- 竞争战略
- 投资决策
- 市场进入

## 输入
- 行业类型
- 竞争格局
- 替代威胁
- 新进入者

## 输出
- 五力分析
- 行业吸引力
- 竞争强度
- 战略建议

## Prompt模板
```
请进行五力分析。

背景：
- 行业类型：[行业]
- 竞争格局：[格局]
- 替代威胁：[替代]
- 新进入者：[进入]

请输出：
1. 供应商议价能力
2. 买方议价能力
3. 新进入者威胁
4. 替代品威胁
5. 行业内竞争
6. 行业吸引力
7. 战略建议
```

## 示例
### 示例1
- 输入：在线教育行业竞争分析
- 输出：五力分析，含师资-平台-替代-进入-竞争各维度

## 注意事项
- 要数据支撑
- 要动态视角
- 要战略导向
