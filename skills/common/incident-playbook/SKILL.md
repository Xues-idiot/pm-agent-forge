# 事件处置手册 (Incident Playbook)

## 触发条件
- 故障处理
- 应急响应
- 标准化流程
- 知识沉淀

## 输入
- 故障类型
- 处理步骤
- 责任人
- 工具支持

## 输出
- 处置流程
- 角色职责
- 检查清单
- 升级机制

## Prompt模板
```
请制定事件处置手册。

背景：
- 故障类型：[类型]
- 处理步骤：[步骤]
- 责任人：[责任人]
- 工具支持：[工具]

请输出：
1. 处置流程
2. 角色职责
3. 检查清单
4. 常用命令
5. 升级机制
```

## 示例
### 示例1
- 输入：数据库故障
- 输出：处置手册，含故障识别→初步诊断→紧急处理→问题修复→复盘总结

## 注意事项
- 要结构清晰
- 要检查清单化
- 要定期更新
- 要演练验证
