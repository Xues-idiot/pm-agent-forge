# 供应计划 (Supply Planning)

## 触发条件
- 供应协调
- 产能规划
- 物料需求
- 交付保障

## 输入
- 需求预测
- 供应来源
- 产能限制
- 交付周期

## 输出
- 供应计划
- 物料需求
- 产能安排
- 风险预案

## Prompt模板
```
请制定供应计划。

背景：
- 需求预测：[预测]
- 供应来源：[来源]
- 产能限制：[限制]
- 交付周期：[周期]

请输出：
1. 供应计划
2. 物料需求
3. 产能安排
4. 风险预案
5. 监控机制
```

## 示例
### 示例1
- 输入：制造供应
- 输出：供应计划，MPS、MRP、供应商协同

## 注意事项
- 要供需平衡
- 要成本效益
- 要供应商协同
- 要持续优化
