# 鱼骨图 (Fishbone Diagram)

## 触发条件
- 原因分析
- 问题分解
- 头脑风暴
- 系统思考

## 输入
- 问题陈述
- 分类维度
- 可能原因
- 团队参与

## 输出
- 鱼骨图
- 原因分类
- 关键因素
- 分析结论

## Prompt模板
```
请使用鱼骨图分析。

背景：
- 问题陈述：[问题]
- 分类维度：[维度]
- 可能原因：[原因]
- 团队参与：[团队]

请输出：
1. 鱼骨图结构
2. 原因分类
3. 关键因素
4. 分析结论
5. 后续行动
```

## 示例
### 示例1
- 输入：产品质量问题
- 输出：鱼骨图，从人机料法环测六方面分析原因

## 注意事项
- 要全面系统
- 要团队参与
- 要层层深入
- 要落地行动
