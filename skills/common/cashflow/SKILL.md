# 现金流管理 (Cash Flow Management)

## 触发条件
- 现金管理
- 流动性
- 资金效率
- 风险控制

## 输入
- 现金流入
- 现金流出
- 账期
- 融资能力

## 输出
- 现金流预测
- 缺口预警
- 优化策略
- 融资安排

## Prompt模板
```
请管理现金流。

背景：
- 现金流入：[流入]
- 现金流出：[流出]
- 账期：[账期]
- 融资能力：[能力]

请输出：
1. 现金流预测
2. 缺口分析
3. 优化策略
4. 融资安排
5. 风险控制
```

## 示例
### 示例1
- 输入：企业现金流
- 输出：现金流管理，月度预测、资金缺口预警、账期优化

## 注意事项
- 要预测准确
- 要控制风险
- 要优化效率
- 要灵活融资
