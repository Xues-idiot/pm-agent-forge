# 健康安全 (Health & Safety)

## 触发条件
- 职业健康
- 安全法规
- 风险评估
- 事故预防

## 输入
- 法规要求
- 工作环境
- 健康风险
- 培训需求

## 输出
- 安全政策
- 风险评估
- 控制措施
- 培训计划

## Prompt模板
```
请管理健康安全。

背景：
- 法规要求：[法规]
- 工作环境：[环境]
- 健康风险：[风险]
- 培训需求：[培训]

请输出：
1. 安全政策
2. 风险评估
3. 控制措施
4. 培训计划
5. 监控机制
```

## 示例
### 示例1
- 输入：工厂安全
- 输出：健康安全，危险识别、安全作业、应急响应

## 注意事项
- 要法规合规
- 要预防为主
- 要全员参与
- 要持续改进
