# IT资产管理 (IT Asset Management)

## 触发条件
- 资产盘点
- 成本控制
- 许可证管理
- 生命周期管理

## 输入
- 资产清单
- 采购记录
- 使用状态
- 成本数据

## 输出
- 资产台账
- 生命周期管理
- 成本分析
- 处置计划

## Prompt模板
```
请管理IT资产。

背景：
- 资产清单：[清单]
- 采购记录：[记录]
- 使用状态：[状态]
- 成本数据：[成本]

请输出：
1. 资产台账
2. 生命周期管理
3. 成本分析
4. 采购计划
5. 处置计划
```

## 示例
### 示例1
- 输入：企业IT资产
- 输出：资产管理，含硬件登记、软件许可证、维保到期提醒、报废流程

## 注意事项
- 要完整准确
- 要动态更新
- 要成本优化
- 要合规处置
