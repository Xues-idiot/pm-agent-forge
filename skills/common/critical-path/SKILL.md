# 关键路径分析 (Critical Path Analysis)

## 触发条件
- 进度管理
- 风险管理
- 资源优化
- 进度压缩

## 输入
- 任务网络
- 工期估算
- 依赖关系
- 资源限制

## 输出
- 关键路径
- 浮动时间
- 进度风险
- 优化建议

## Prompt模板
```
请进行关键路径分析。

背景：
- 任务网络：[网络]
- 工期估算：[估算]
- 依赖关系：[依赖]
- 资源限制：[限制]

请输出：
1. 关键路径
2. 浮动时间
3. 进度风险
4. 优化建议
5. 压缩方案
```

## 示例
### 示例1
- 输入：项目网络图
- 输出：关键路径分析，关键路径、浮动时间、进度风险、压缩方案

## 注意事项
- 要准确建模
- 要动态更新
- 要关注风险
- 要资源平衡
