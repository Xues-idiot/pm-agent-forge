# 转型时刻 (Pivot)

## 触发条件
- 方向调整
- 市场验证
- 增长停滞
- 战略迭代

## 输入
- 当前指标
- 学习数据
- 验证假设
- 团队共识

## 输出
- 转型方向
- 转型时机
- 转型风险
- 执行计划

## Prompt模板
```
请决定是否转型。

背景：
- 当前指标：[指标]
- 学习数据：[学习]
- 验证假设：[假设]
- 团队共识：[共识]

请输出：
1. 数据分析
2. 转型信号
3. 转型方向
4. 风险评估
5. 执行计划
```

## 示例
### 示例1
- 输入：B2C失败
- 输出：转型方向，转向B2B、目标客户、技术复用

## 注意事项
- 要数据驱动
- 要果断决策
- 要团队共识
- 要快速执行
