# 自助服务 (Self-service)

## 触发条件
- 客户支持
- 自动化
- 成本效率
- 用户体验

## 输入
- 常见问题
- 用户能力
- 工具支持
- 成本考量

## 输出
- 自助方案
- 知识库
- 自动化
- 升级路径

## Prompt模板
```
请设计自助服务。

背景：
- 常见问题：[问题]
- 用户能力：[能力]
- 工具支持：[工具]
- 成本考量：[成本]

请输出：
1. 自助场景
2. 知识库设计
3. 自动化流程
4. 引导设计
5. 升级机制
```

## 示例
### 示例1
- 输入：在线客服
- 输出：自助服务，FAQ+智能客服+操作指南+视频教程

## 注意事项
- 要用户友好
- 要易于使用
- 要持续完善
- 要平滑升级
