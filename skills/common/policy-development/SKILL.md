# 政策制定 (Policy Development)

## 触发条件
- 制度建立
- 合规要求
- 流程规范
- 员工行为

## 输入
- 政策目标
- 法规要求
- 业务实际
- 利益相关方

## 输出
- 政策草案
- 影响分析
- 审批流程
- 培训计划

## Prompt模板
```
请制定政策。

背景：
- 政策目标：[目标]
- 法规要求：[法规]
- 业务实际：[实际]
- 利益相关方：[相关方]

请输出：
1. 政策草案
2. 影响分析
3. 审批流程
4. 培训计划
5. 评估机制
```

## 示例
### 示例1
- 输入：信息安全
- 输出：政策制定，安全政策、acceptable使用、事件报告

## 注意事项
- 要法规合规
- 要业务适配
- 要清晰明确
- 要持续更新
