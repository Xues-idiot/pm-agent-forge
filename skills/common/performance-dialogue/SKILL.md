# 绩效对话 (Performance Dialogue)

## 触发条件
- 绩效沟通
- 反馈辅导
- 目标跟踪
- 发展讨论

## 输入
- 绩效数据
- 目标进展
- 行为表现
- 发展需求

## 输出
- 绩效反馈
- 辅导建议
- 发展计划
- 共识确认

## Prompt模板
```
请进行绩效对话。

背景：
- 绩效数据：[数据]
- 目标进展：[进展]
- 行为表现：[表现]
- 发展需求：[需求]

请输出：
1. 绩效反馈
2. 成功认可
3. 改进建议
4. 发展计划
5. 共识确认
```

## 示例
### 示例1
- 输入：季度绩效面谈
- 输出：绩效对话，成就认可、改进点讨论、下季目标、发展规划

## 注意事项
- 要双向沟通
- 要具体事实
- 要发展导向
- 要记录确认
