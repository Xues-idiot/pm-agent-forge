# 入职流程 (Onboarding Process)

## 触发条件
- 新员工
- 入职培训
- 融入团队
- 岗位适应

## 输入
- 岗位要求
- 团队结构
- 培训计划
- 工作流程

## 输出
- 入职计划
- 培训安排
- 导师匹配
- 目标设定

## Prompt模板
```
请设计入职流程。

背景：
- 岗位要求：[要求]
- 团队结构：[结构]
- 培训计划：[计划]
- 工作流程：[流程]

请输出：
1. 入职计划
2. 培训安排
3. 导师匹配
4. 目标设定
5. 评估机制
```

## 示例
### 示例1
- 输入：新员工入职
- 输出：入职流程，第一周/第一个月/第三个月计划

## 注意事项
- 要系统全面
- 要导师指导
- 要目标清晰
- 要持续跟踪
