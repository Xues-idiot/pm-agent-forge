# 关键链管理 (Critical Chain)

## 触发条件
- 项目管理
- 进度控制
- 资源优化
- 风险管理

## 输入
- 任务分解
- 依赖关系
- 资源约束
- 缓冲设置

## 输出
- 关键链识别
- 缓冲设置
- 执行监控
- 动态调整

## Prompt模板
```
请进行关键链管理。

背景：
- 任务分解：[分解]
- 依赖关系：[关系]
- 资源约束：[约束]
- 缓冲设置：[缓冲]

请输出：
1. 关键链识别
2. 缓冲设置
3. 执行计划
4. 监控机制
5. 动态调整
```

## 示例
### 示例1
- 输入：复杂项目
- 输出：关键链管理，含关键路径、feed缓冲、project缓冲、执行监控

## 注意事项
- 要资源约束
- 要设置缓冲
- 要持续监控
- 要动态调整
