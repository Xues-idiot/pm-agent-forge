# 最佳实践 (Best Practices)

## 触发条件
- 经验总结
- 方法论
- 流程优化
- 标准建立

## 输入
- 业务场景
- 成功案例
- 行业标准
- 实施条件

## 输出
- 实践总结
- 适用条件
- 实施指南
- 效果评估

## Prompt模板
```
请总结最佳实践。

背景：
- 业务场景：[场景]
- 成功案例：[案例]
- 行业标准：[标准]
- 实施条件：[条件]

请输出：
1. 实践总结
2. 适用条件
3. 实施指南
4. 效果评估
5. 迭代建议
```

## 示例
### 示例1
- 输入：敏捷开发
- 输出：最佳实践，每日站会、迭代评审、持续集成

## 注意事项
- 要来源于实践
- 要可复制
- 要本地化
- 要持续迭代
