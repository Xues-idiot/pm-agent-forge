# 应急响应 (Incident Response)

## 触发条件
- 故障处理
- 问题解决
- 服务恢复
- 事后复盘

## 输入
- 故障现象
- 影响范围
- 资源状态
- 沟通需求

## 输出
- 响应策略
- 处理流程
- 沟通计划
- 改进措施

## Prompt模板
```
请制定应急响应方案。

背景：
- 故障现象：[现象]
- 影响范围：[范围]
- 资源状态：[状态]
- 沟通需求：[需求]

请输出：
1. 影响评估
2. 响应策略
3. 处理流程
4. 沟通计划
5. 改进措施
```

## 示例
### 示例1
- 输入：电商平台支付故障
- 输出：应急响应，含快速止血、临时关闭支付、通知用户、修复后验证

## 注意事项
- 要快字当先
- 要及时沟通
- 要记录过程
- 要复盘改进
