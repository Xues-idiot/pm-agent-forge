# MVP开发 (MVP Development)

## 触发条件
- 最小产品
- 快速验证
- 资源优化
- 市场测试

## 输入
- 核心功能
- 用户需求
- 开发资源
- 验证目标

## 输出
- MVP范围
- 功能优先级
- 开发计划
- 验证指标

## Prompt模板
```
请开发MVP。

背景：
- 核心功能：[功能]
- 用户需求：[需求]
- 开发资源：[资源]
- 验证目标：[目标]

请输出：
1. MVP范围
2. 功能优先级
3. 开发计划
4. 验证指标
5. 迭代计划
```

## 示例
### 示例1
- 输入：新产品
- 输出：MVP开发，核心功能3个、验证指标、迭代计划

## 注意事项
- 要核心价值
- 要快速交付
- 要验证学习
- 要持续改进
