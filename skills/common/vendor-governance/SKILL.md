# 供应商治理 (Vendor Governance)

## 触发条件
- 供应商关系
- 服务质量
- 合规管理
- 价值创造

## 输入
- 供应商列表
- 服务协议
- 绩效数据
- 风险等级

## 输出
- 治理框架
- 监控机制
- 沟通机制
- 改进计划

## Prompt模板
```
请治理供应商。

背景：
- 供应商列表：[列表]
- 服务协议：[协议]
- 绩效数据：[数据]
- 风险等级：[风险]

请输出：
1. 治理框架
2. 监控机制
3. 沟通机制
4. 改进计划
5. 退出机制
```

## 示例
### 示例1
- 输入：IT服务商
- 输出：供应商治理，月度审查、季度评估、年度战略回顾

## 注意事项
- 要关系管理
- 要绩效驱动
- 要风险管控
- 要持续改进
