# 战略分析 (Strategic Analysis)

## 触发条件
- 战略制定
- 竞争分析
- 市场洞察
- 决策支持

## 输入
- 行业环境
- 竞争格局
- 自身资源
- 发展机会

## 输出
- 分析框架
- 环境评估
- 竞争洞察
- 战略建议

## Prompt模板
```
请进行战略分析。

背景：
- 行业环境：[环境]
- 竞争格局：[格局]
- 自身资源：[资源]
- 发展机会：[机会]

请输出：
1. 分析框架选择
2. PEST分析
3. 五力分析
4. SWOT分析
5. 战略建议
```

## 示例
### 示例1
- 输入：进入新市场
- 输出：战略分析，全面的外部+内部+竞争分析

## 注意事项
- 要系统全面
- 要数据支撑
- 要多角度
- 要转化为行动
