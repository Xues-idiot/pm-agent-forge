# 计划扑克 (Planning Poker)

## 触发条件
- 敏捷估算
- 团队共识
- 故事点
- Sprint规划

## 输入
- 用户故事
- 团队成员
- 估算标准
- 历史参考

## 输出
- 估算讨论
- 故事点
- 分歧解决
- 团队共识

## Prompt模板
```
请使用计划扑克估算。

背景：
- 用户故事：[故事]
- 团队成员：[成员]
- 估算标准：[标准]
- 历史参考：[参考]

请输出：
1. 估算讨论
2. 估算结果
3. 分歧解决
4. 团队共识
5. 记录归档
```

## 示例
### 示例1
- 输入：产品待办列表
- 输出：计划扑克估算，含各团队成员估算、讨论分歧、最终共识

## 注意事项
- 要独立估算
- 要讨论分歧
- 要基于经验
- 要持续校准
