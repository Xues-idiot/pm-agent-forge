# 回顾会议 (Retrospective)

## 触发条件
- 团队复盘
- 持续改进
- 协作优化
- 经验沉淀

## 输入
- Sprint情况
- 团队反馈
- 问题列表
- 改进建议

## 输出
- 问题分类
- 改进措施
- 行动计划
- 责任分配

## Prompt模板
```
请组织回顾会议。

背景：
- Sprint情况：[情况]
- 团队反馈：[反馈]
- 问题列表：[问题]
- 改进建议：[建议]

请输出：
1. 问题分类
2. 根因分析
3. 改进措施
4. 行动计划
5. 责任分配
```

## 示例
### 示例1
- 输入：Sprint复盘
- 输出：回顾会议，含继续/停止/开始三列、Action items、责任到人

## 注意事项
- 要开放坦诚
- 要聚焦改进
- 要可落地
- 要跟踪闭环
