# OKR框架 (OKR Framework)

## 触发条件
- 目标管理
- 组织对齐
- 绩效考核
- 敏捷落地

## 输入
- 战略目标
- 组织结构
- 团队特点
- 实施周期

## 输出
- OKR设计
- 对齐机制
- 评估方式
- 运营机制

## Prompt模板
```
请设计OKR框架。

背景：
- 战略目标：[目标]
- 组织结构：[结构]
- 团队特点：[特点]
- 实施周期：[周期]

请输出：
1. OKR设计原则
2. O设计
3. KR设计
4. 对齐机制
5. 评估方式
6. 运营机制
```

## 示例
### 示例1
- 输入：互联网公司，想推行OKR
- 输出：OKR框架，含设计原则、评分标准、对齐机制

## 注意事项
- 要聚焦重点
- 要上下对齐
- 要透明公开
