# 影响力 (Influence Skills)

## 触发条件
- 说服沟通
- 领导影响
- 跨部门协作
- 推动执行

## 输入
- 影响目标
- 影响力来源
- 对象特征
- 情境因素

## 输出
- 影响策略
- 沟通方法
- 关系建立
- 推动机制

## Prompt模板
```
请提升影响力。

背景：
- 影响目标：[目标]
- 影响力来源：[来源]
- 对象特征：[对象]
- 情境因素：[情境]

请输出：
1. 影响策略
2. 沟通方法
3. 关系建立
4. 利益分析
5. 推动机制
```

## 示例
### 示例1
- 输入：跨部门推动
- 输出：影响力，寻找共同利益、建立信任、借助数据、寻求支持

## 注意事项
- 要了解对方
- 要建立信任
- 要展示价值
- 要双赢思维
