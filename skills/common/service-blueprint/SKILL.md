# 服务蓝图 (Service Blueprint)

## 触发条件
- 服务设计
- 客户旅程
- 触点优化
- 服务标准化

## 输入
- 客户行为
- 服务触点
- 前台后台
- 痛点需求

## 输出
- 服务蓝图
- 触点设计
- 流程优化
- 标准定义

## Prompt模板
```
请设计服务蓝图。

背景：
- 客户行为：[行为]
- 服务触点：[触点]
- 前台后台：[前后台]
- 痛点需求：[痛点]

请输出：
1. 服务蓝图
2. 各触点设计
3. 流程优化
4. 标准定义
5. 改进优先级
```

## 示例
### 示例1
- 输入：酒店入住服务
- 输出：服务蓝图，含网上预订→大堂等候→入住登记→房卡提供→引导入房

## 注意事项
- 要以客户视角
- 要线上线下打通
- 要前后台协同
- 要持续迭代
