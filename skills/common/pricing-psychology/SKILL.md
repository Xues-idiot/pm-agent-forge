# 定价心理学 (Pricing Psychology)

## 触发条件
- 定价策略
- 心理定价
- 价格展示
- 促单技巧

## 输入
- 产品价值
- 目标用户
- 竞争价格
- 促销场景

## 输出
- 定价策略
- 心理技巧
- 展示方式
- 促单方案

## Prompt模板
```
请运用定价心理学。

背景：
- 产品价值：[价值]
- 目标用户：[用户]
- 竞争价格：[竞争]
- 促销场景：[场景]

请输出：
1. 定价策略
2. 心理定价技巧
3. 价格展示方式
4. 促单技巧
```

## 示例
### 示例1
- 输入：新品上市定价
- 输出：定价方案，含锚定价格、分割定价、限时优惠

## 注意事项
- 要研究用户心理
- 要对比感知价值
- 要促进购买决策
