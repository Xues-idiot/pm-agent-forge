# 人才发展 (Talent Development)

## 触发条件
- 人才培养
- 能力提升
- 职业规划
- 继任计划

## 输入
- 人才盘点
- 能力模型
- 发展资源
- 业务需求

## 输出
- 发展计划
- 培训体系
- 职业路径
- 继任计划

## Prompt模板
```
请制定人才发展计划。

背景：
- 人才盘点：[盘点]
- 能力模型：[模型]
- 发展资源：[资源]
- 业务需求：[需求]

请输出：
1. 发展框架
2. 培训体系
3. 职业路径
4. 继任计划
5. 效果评估
```

## 示例
### 示例1
- 输入：高潜力员工，需要加速发展
- 输出：发展计划，含轮岗体验、导师辅导、挑战性任务、外部培训

## 注意事项
- 要个性化设计
- 要实战锻炼
- 要持续跟踪
- 要与业务结合
