# 用户生命周期运营 (User Lifecycle Operations)

## 触发条件
- 设计用户运营体系
- 提升用户价值
- 降低用户流失
- 用户分层运营

## 输入
- 用户数据
- 业务场景
- 现有运营策略
- 资源情况

## 输出
- 用户生命周期地图
- 分层运营策略
- 触达策略
- 转化提升方案

## Prompt模板
```
请为[产品/平台]设计用户生命周期运营方案。

背景：
- 产品类型：[产品类型]
- 用户规模：[规模]
- 当前痛点：[痛点]
- 可用资源：[资源]

请输出：
1. 用户生命周期阶段定义（获取-激活-留存-变现-推荐）
2. 各阶段核心指标和目标
3. 分层运营策略（高价值/成长/流失预警等）
4. 关键触达策略和时机
5. 各阶段转化提升方案
6. 运营节奏建议
```

## 示例
### 示例1
- 输入：在线教育平台，用户从注册到付费转化率低
- 输出：用户生命周期运营方案，含新手引导-付费转化-续费激励策略

## 注意事项
- 要识别关键决策点
- 预防流失比召回更重要
- 分层运营要动态调整
