# Channel Operations

渠道运营，管理营销渠道和投放运营。

## 何时使用

- 渠道策略制定
- 投放计划管理
- 渠道效果优化
- ROI提升
- 多渠道协调

## 渠道基础

### 渠道类型

```markdown
## 渠道类型

### 付费渠道

## 投放管理

### 投放流程

```markdown
## 投放流程

### 标准流程
```
需求确认 → 方案制定 → 素材准备 → 账户搭建 → 投放执行 → 效果监测 → 优化迭代
```

### 各环节要点
| 环节 | 内容 | 产出 |
|------|------|------|
| 需求确认 | 目标/预算/时间 | 需求文档 |
| 方案制定 | 渠道/定向/出价 | 投放方案 |
| 素材准备 | 图文/视频/落地页 | 广告素材 |
| 账户搭建 | 账户结构/定向 | 投放设置 |
| 投放执行 | 监控/调价 | 投放中 |
| 效果监测 | 数据收集 | 数据报告 |
| 优化迭代 | 策略调整 | 优化方案 |
```

### 投放准备
```markdown
## 投放准备

### 基础设置
| 设置项 | 内容 | 要点 |
|--------|------|------|
| 账户 | 账户开通/充值 | 资质 |
| 资质 | 行业资质/授权 | 合规 |
| 白名单 | 特殊类目 | 提前申请 |

### 定向设置
| 维度 | 选项 | 说明 |
|------|------|------|
| 地域 | 省/市/区 | 目标区域 |
| 年龄 | 18-24/25-34 | 目标人群 |
| 性别 | 男/女/不限 | 性别定向 |
| 兴趣 | 关键词/类目 | 兴趣定向 |
| 行为 | 搜索/购买 | 行为定向 |
```
```

### 账户结构

```markdown
## 账户结构

### 结构层级
```
账户 → 推广计划 → 推广组 → 广告创意
```

### 结构设计
```markdown
## 账户设计

### 按产品
- 计划：产品A/产品B/产品C
- 组：产品A-关键词A/产品A-关键词B

### 按渠道
- 计划：搜索/信息流/社交
- 组：各渠道细分

### 按地域
- 计划：华北/华东/华南
- 组：各省市

### 按素材
- 计划：图片/视频/文案
- 组：各素材类型
```

### 命名规范
- 计划：产品-渠道-日期
- 组：人群-定向-素材
- 创意：产品-卖点-版本
```

## 投放优化

### 核心指标

```markdown
## 核心指标

### 曝光指标
| 指标 | 计算 | 优化方向 |
|------|------|----------|
| 展现量 | 广告展示次数 | 提价/扩定向 |
| CPM | 1000展现成本 | 降低CPM |

### 点击指标
| 指标 | 计算 | 优化方向 |
|------|------|----------|
| 点击量 | 广告点击次数 | 优化素材 |
| CTR | 点击/展现 | 提创意 |
| CPC | 点击成本 | 优化出价 |

### 转化指标
| 指标 | 计算 | 优化方向 |
|------|------|----------|
| 转化量 | 完成目标数 | 优化落地页 |
| CVR | 转化/点击 | 优化转化 |
| CPA | 转化成本 | 综合优化 |
| ROI | 收入/成本 | 终极目标 |
```

### 优化策略

```markdown
## 优化策略

### 素材优化
| 优化点 | 方法 | 测试 |
|--------|------|------|
| 创意 | 痛点/卖点/差异化 | AB测试 |
| 配图 | 高质量/场景化 | 多版本 |
| 标题 | 数字/疑问/利益 | 多版本 |
| CTA | 明确行动 | 多版本 |

### 出价优化
```markdown
## 出价优化

### 出价策略
| 策略 | 适用 | 说明 |
|------|------|------|
| 手动出价 | 精细化运营 | 控制成本 |
| 自动出价 | 起步阶段 | 探索最优 |
| 目标转化 | 稳定阶段 | 优化转化 |
| 放量优先 | 冲量阶段 | 扩大规模 |

### 调价原则
1. 数据积累后调价
2. 小幅多次调整
3. 观察3-5天后调整
4. 参考行业均值
```
```

### 定向优化

```markdown
## 定向优化

### 定向调整
| 问题 | 原因 | 优化 |
|------|------|------|
| 展现少 | 定向过窄 | 放宽定向 |
| 点击贵 | 竞争激烈 | 调整出价 |
| 转化差 | 人群不匹配 | 重新定向 |

### 人群包
```markdown
## 人群包

### 人群类型
| 类型 | 来源 | 用途 |
|------|------|------|
| 基础包 | 地域/年龄/性别 | 基础定向 |
| 兴趣包 | 平台兴趣分类 | 扩大覆盖 |
| 行为包 | 搜索/购买行为 | 精准投放 |
| 转化包 | 已转化用户 | 再营销 |
| lookalike | 种子用户放大 | 相似扩展 |
```

### 再营销
- 访问未转化：召回
- 加购未购买：促转化
- 已购买用户：复购
- 高价值用户：维护
```

```markdown
## 渠道协同

### 协同模式

### 联动策略
```markdown
## 联动策略

### 引流转私域
1. 投放广告 → 加微信/进群
2. 私域运营 → 深度转化
3. 老客复购 → 口碑传播

### 跨渠道协同
1. 抖音：种草/引流
2. 小红书：种草/口碑
3. 搜索：收割/转化
4. 信息流：品牌/再营销
```

### 预算分配

```markdown
## 预算分配

### 分配原则
| 原则 | 说明 | 适用 |
|------|------|------|

### 分配比例
```markdown
## 预算分配

### 常规分配
- 主力渠道：50-60%
- 测试渠道：20-30%
- 机动预算：10-20%

### 阶段分配
- 冷启动：测试为主
- 成长期：效果渠道放量
- 稳定期：优化ROI
- 衰退期：开拓新渠道
```
