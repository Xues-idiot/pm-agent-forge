# 增长策略 (Growth Strategy)

## 触发条件
- 制定增长目标
- 增长瓶颈分析
- 增长策略设计
- 增长实验规划

## 输入
- 增长现状
- 业务数据
- 资源情况
- 竞品情况

## 输出
- 增长策略报告
- 增长实验计划
- 优先级排序
- 资源需求

## Prompt模板
```
请为[产品/业务]制定增长策略。

现状分析：
- 现有指标：[DAU/MAU/GMV等]
- 增长目标：[目标]
- 资源限制：[资源]
- 竞品动态：[竞品]

请用AARRR模型分析并输出：
1. Acquisition（获客）：获客渠道、获客成本、获客效率
2. Activation（激活）：激活率、关键行为、新手转化
3. Retention（留存）：留存曲线、流失分析、留存策略
4. Revenue（变现）：ARPU、付费率、变现模式
5. Referral（推荐）：推荐率、K因子、裂变机制
6. 增长实验优先级和资源需求
```

## 示例
### 示例1
- 输入：社区产品，DAU 100万，目标是500万
- 输出：增长策略，含获客优化、留存提升、变现模式设计

## 注意事项
- 要找到最大杠杆点
- 实验要小步快跑
- 短期和长期平衡
