# 采购管理 (Procurement Management)

## 触发条件
- 采购策略
- 供应商管理
- 采购流程
- 成本控制

## 输入
- 采购类型
- 支出结构
- 供应商现状
- 优化目标

## 输出
- 采购策略
- 流程优化
- 供应商管理
- 成本节省

## Prompt模板
```
请优化[采购类别]采购管理。

背景：
- 采购类型：[类型]
- 支出结构：[结构]
- 供应商现状：[现状]
- 优化目标：[目标]

请输出：
1. 采购战略定位
2. 支出分析
3. 采购流程优化
4. 供应商策略
5. 合同管理
6. 价格管理
7. 采购数字化
8. 采购风险管理
9. 绩效评估
```

## 示例
### 示例1
- 输入：集团企业，年采购额10亿，需要整合供应商
- 输出：采购方案，含供应商整合、集中采购、战略合作

## 注意事项
- 要战略采购
- 要总成本最低
- 要合规透明
