# 客户运营 (Customer Operations)

## 触发条件
- 客户分层运营
- 客户生命周期管理
- 客户价值提升
- 客户流失预防

## 输入
- 客户数据
- 业务场景
- 运营目标
- 资源情况

## 输出
- 客户分层策略
- 运营方案
- 触达策略
- 效果评估

## Prompt模板
```
请为[业务]设计客户运营方案。

背景：
- 业务类型：[类型]
- 客户规模：[规模]
- 运营目标：[目标]
- 资源情况：[资源]

请输出：
1. 客户分层模型设计
2. 各层客户特征和需求
3. 分层运营策略
4. 客户生命周期运营
5. 关键触达点和时机
6. 客户价值提升方案
7. 流失预防和召回
8. 运营效果评估指标
```

## 示例
### 示例1
- 输入：电商平台，用户1000万，需要提升复购率
- 输出：客户运营方案，含RFM分层、精准营销、会员体系

## 注意事项
- 要数据驱动
- 要个性化
- 要持续优化
