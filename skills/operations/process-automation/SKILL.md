# 流程自动化 (Process Automation)

## 触发条件
- 自动化机会
- RPA应用
- 流程优化
- 实施路径

## 输入
- 流程现状
- 自动化需求
- 技术能力
- 投资预算

## 输出
- 自动化方案
- 流程选择
- 技术设计
- 实施计划

## Prompt模板
```
请设计[流程]自动化方案。

背景：
- 流程现状：[现状]
- 自动化需求：[需求]
- 技术能力：[能力]
- 投资预算：[预算]

请输出：
1. 流程诊断
2. 自动化机会
3. 技术选型
4. 流程设计
5. 实施路径
6. 投资回报
```

## 示例
### 示例1
- 输入：财务报销流程，需要减少人工处理
- 输出：自动化方案，含RPA+OCR+审批流

## 注意事项
- 要ROI合理
- 要循序渐进
- 要持续运营
