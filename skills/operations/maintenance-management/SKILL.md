# 维护管理 (Maintenance Management)

## 触发条件
- 设备维护策略
- 预防性维护
- 预测性维护
- 成本优化

## 输入
- 设备类型
- 设备规模
- 维护现状
- 目标要求

## 输出
- 维护策略
- 计划制定
- 资源配置
- 效果评估

## Prompt模板
```
请优化[设备/设施]维护管理。

背景：
- 设备类型：[类型]
- 设备规模：[规模]
- 维护现状：[现状]
- 目标要求：[目标]

请输出：
1. 设备资产梳理
2. 维护现状诊断
3. 维护策略选择（事后/预防/预测）
4. 维护计划
5. 备件管理
6. 人员配置
7. 成本控制
8. 效果评估
9. 持续改进
```

## 示例
### 示例1
- 输入：工厂设备，故障频繁，维护成本高
- 输出：维护方案，含预防性维护、预测性维护、备件优化

## 注意事项
- 要预防为主
- 要数据驱动
- 要持续改进
