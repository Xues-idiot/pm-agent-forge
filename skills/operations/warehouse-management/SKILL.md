# 仓储管理 (Warehouse Management)

## 触发条件
- 仓储优化
- 库位规划
- 作业效率
- 库存准确

## 输入
- 仓库类型
- 货品特点
- 作业量
- 自动化水平

## 输出
- 仓储方案
- 库位设计
- 作业流程
- 效率指标

## Prompt模板
```
请优化[仓库]管理。

背景：
- 仓库类型：[类型]
- 货品特点：[特点]
- 作业量：[量]
- 自动化：[水平]

请输出：
1. 仓库布局
2. 库位规划
3. 作业流程
4. 库存管理
5. 盘点管理
6. 安全管理
7. 效率指标
8. 自动化
```

## 示例
### 示例1
- 输入：电商仓库，日出库10万单，需要提效
- 输出：仓储方案，含库位优化、作业流程、自动化设备

## 注意事项
- 要效率优先
- 要库存准确
- 要持续改进
