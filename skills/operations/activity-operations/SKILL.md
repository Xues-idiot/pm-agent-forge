# 活动运营 (Activity Operations)

## 触发条件
- 设计活动运营方案
- 策划用户活动
- 活动效果分析
- 活动复盘优化

## 输入
- 活动目标
- 目标用户
- 活动预算
- 时间周期

## 输出
- 活动策划方案
- 活动执行手册
- 活动复盘报告
- 活动迭代建议

## Prompt模板
```
请为[业务场景]设计一场[主题]活动运营方案。

活动背景：
- 活动目标：[目标]
- 目标用户：[用户]
- 预算范围：[预算]
- 活动时间：[周期]

请输出：
1. 活动主题和核心创意
2. 活动规则和参与流程
3. 奖励机制设计
4. 传播机制（分享/邀请）
5. 风险点和备选方案
6. 数据监测指标
7. 活动复盘维度
```

## 示例
### 示例1
- 输入：电商平台年货节活动，预算200万，目标提升GMV 30%
- 输出：年货节活动策划，含预热/正式/返场三阶段

## 注意事项
- 活动规则要简单易懂
- 奖励要有吸引力且可控
- 要有传播裂变机制
