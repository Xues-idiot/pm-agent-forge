# 精益管理 (Lean Management)

## 触发条件
- 精益转型
- 浪费消除
- 效率提升
- 持续改进

## 输入
- 业务类型
- 改进目标
- 精益基础
- 人员能力

## 输出
- 精益方案
- 改善项目
- 标准作业
- 评估机制

## Prompt模板
```
请推进[组织]精益管理。

背景：
- 业务类型：[类型]
- 改进目标：[目标]
- 精益基础：[基础]
- 人员能力：[能力]

请输出：
1. 精益成熟度
2. 价值流分析
3. 改善项目
4. 浪费消除
5. 标准作业
6. 可视化管理
7. 持续改进
```

## 示例
### 示例1
- 输入：制造企业，需要降低制造成本
- 输出：精益方案，含价值流、现场改善、标准化

## 注意事项
- 要领导支持
- 要全员参与
- 要持续改进
