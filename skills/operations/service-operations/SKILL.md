# 服务运营 (Service Operations)

## 触发条件
- 服务体系设计
- 服务流程优化
- 服务质量提升
- 服务成本控制

## 输入
- 服务类型
- 服务规模
- 用户期望
- 成本目标

## 输出
- 服务体系方案
- 服务流程
- 质量标准
- 成本模型

## Prompt模板
```
请为[业务]设计服务运营体系。

背景：
- 服务类型：[类型]
- 服务规模：[规模]
- 用户期望：[期望]
- 成本目标：[目标]

请输出：
1. 服务定位和目标
2. 服务体系架构
3. 服务流程设计
4. 服务标准建立
5. 服务质量控制
6. 服务团队管理
7. 服务成本优化
8. 服务技术创新
9. 服务效果评估
```

## 示例
### 示例1
- 输入：快递公司，需要提升时效和满意度，控制成本
- 输出：服务运营方案，含时效标准、网点管理、成本优化

## 注意事项
- 要平衡体验和成本
- 要标准化流程
- 要持续改进
