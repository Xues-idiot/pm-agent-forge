# 现场管理 (Site Management)

## 触发条件
- 现场运营
- 标准化
- 5S管理
- 目视化管理

## 输入
- 现场类型
- 运营规模
- 管理目标
- 人员能力

## 输出
- 管理方案
- 标准建立
- 监控机制
- 持续改进

## Prompt模板
```
请优化[现场]管理。

背景：
- 现场类型：[类型]
- 运营规模：[规模]
- 管理目标：[目标]
- 人员能力：[能力]

请输出：
1. 现场诊断
2. 标准化
3. 5S管理
4. 目视化管理
5. 异常管理
6. 安全环境
7. 持续改进
```

## 示例
### 示例1
- 输入：工厂车间，需要提升现场管理水平
- 输出：管理方案，含5S、标准作业、异常响应

## 注意事项
- 要标准化
- 要可视化
- 要持续改进
