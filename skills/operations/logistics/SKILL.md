# 物流管理 (Logistics Management)

## 触发条件
- 物流网络规划
- 配送优化
- 仓储管理
- 成本控制

## 输入
- 业务模式
- 物流现状
- 覆盖范围
- 成本目标

## 输出
- 物流方案
- 网络设计
- 配送优化
- 成本模型

## Prompt模板
```
请优化[业务]物流管理。

背景：
- 业务模式：[模式]
- 物流现状：[现状]
- 覆盖范围：[范围]
- 成本目标：[目标]

请输出：
1. 物流诊断
2. 网络规划
3. 仓储策略
4. 配送路径优化
5. 配送模式选择
6. 运输管理
7. 成本控制
8. 数字化方案
9. 应急方案
```

## 示例
### 示例1
- 输入：电商平台，要覆盖全国，物流成本占比15%
- 输出：物流方案，含仓网规划、智能调度、成本优化

## 注意事项
- 要网络优化
- 要技术赋能
- 要持续降本
