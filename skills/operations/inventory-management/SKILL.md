# 库存管理 (Inventory Management)

## 触发条件
- 库存策略
- 补货优化
- 库存周转
- 库存成本

## 输入
- 产品类型
- 供应链现状
- 库存目标
- 资金约束

## 输出
- 库存策略
- 补货模型
- 成本优化
- 监控体系

## Prompt模板
```
请优化[产品]库存管理。

背景：
- 产品类型：[类型]
- 供应链现状：[现状]
- 库存目标：[目标]
- 资金约束：[约束]

请输出：
1. 库存诊断
2. 库存分类（ABC）
3. 补货策略
4. 安全库存设计
5. 呆滞库存处理
6. 库存周转优化
7. 成本控制
8. 仓库布局
9. 系统支撑
```

## 示例
### 示例1
- 输入：服装品牌，库存周转90天，目标60天
- 输出：优化方案，含补货模型、快反机制、促销清仓

## 注意事项
- 要平衡服务和成本
- 要数据驱动
- 要持续优化
