# 质量保证 (Quality Assurance)

## 触发条件
- QA体系建立
- 测试策略
- 质量标准
- 缺陷管理

## 输入
- 产品类型
- 开发模式
- 质量目标
- 团队能力

## 输出
- QA体系
- 测试策略
- 质量标准
- 改进方案

## Prompt模板
```
请为[产品/团队]建立质量保证体系。

背景：
- 产品类型：[类型]
- 开发模式：[模式]
- 质量目标：[目标]
- 团队能力：[能力]

请输出：
1. QA成熟度评估
2. 测试策略设计
3. 测试用例设计
4. 自动化测试方案
5. 缺陷管理
6. 发布标准
7. 质量度量
8. 持续改进
```

## 示例
### 示例1
- 输入：SaaS产品，敏捷开发，需要提升发布质量
- 输出：QA方案，含自动化测试、灰度发布、质量门禁

## 注意事项
- 要预防为主
- 要自动化
- 要持续改进
