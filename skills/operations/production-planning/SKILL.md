# 生产计划 (Production Planning)

## 触发条件
- 主生产计划
- 物料需求
- 产能匹配
- 计划优化

## 输入
- 需求预测
- 生产能力
- 物料供应
- 交期目标

## 输出
- MPS方案
- MRP运算
- 产能平衡
- 排产计划

## Prompt模板
```
请优化[生产]计划体系。

背景：
- 需求预测：[预测]
- 生产能力：[能力]
- 物料供应：[供应]
- 交期目标：[目标]

请输出：
1. 计划体系设计
2. 主生产计划
3. 物料需求计划
4. 产能规划
5. 排产优化
6. 异常处理
7. 持续改进
```

## 示例
### 示例1
- 输入：离散制造，需要缩短交付周期
- 输出：计划方案，含MPS、MRP、APS排产

## 注意事项
- 要平衡供需
- 要持续优化
- 要快速响应
