# 需求预测 (Demand Forecasting)

## 触发条件
- 需求预测
- 计划制定
- 库存优化
- 供应链协同

## 输入
- 历史数据
- 预测范围
- 影响因素
- 精度目标

## 输出
- 预测模型
- 预测结果
- 置信区间
- 建议措施

## Prompt模板
```
请进行[产品/业务]需求预测。

背景：
- 历史数据：[数据]
- 预测范围：[范围]
- 影响因素：[因素]
- 精度目标：[目标]

请输出：
1. 数据分析
2. 预测模型选择
3. 预测结果
4. 置信区间
5. 敏感性分析
6. 建议措施
```

## 示例
### 示例1
- 输入：服装品类未来3个月销售预测
- 输出：预测报告，含预测模型、结果、备货建议

## 注意事项
- 要考虑影响因素
- 要定期修正
- 要与业务结合
