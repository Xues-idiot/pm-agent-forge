## 内容生产

### 生产流程

```markdown
## 内容生产流程

### 标准流程
```
选题 → 策划 → 创作 → 审核 → 发布 → 复盘
```

### 各环节职责
| 环节 | 负责人 | 输出 | 时长 |
|------|--------|------|------|
| 选题 | 内容主编 | 选题表 | 1天 |
| 策划 | 策划 | 策划案 | 1天 |
| 创作 | 编辑/作者 | 初稿 | 2-3天 |
| 审核 | 主编/专家 | 审核意见 | 1天 |
| 发布 | 运营 | 已发布内容 | 1天 |
| 复盘 | 运营 | 复盘报告 | 发布后3天 |

### 紧急流程
1. 快速选题（0.5天）
2. 简化策划（0.5天）
3. 快速创作（1天）
4. 快速审核（0.5天）
5. 发布
```
