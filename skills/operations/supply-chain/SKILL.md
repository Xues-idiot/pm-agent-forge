# 供应链运营 (Supply Chain Operations)

## 触发条件
- 供应链优化
- 库存管理
- 供应商管理
- 成本控制

## 输入
- 业务模式
- 供应链现状
- 优化目标
- 约束条件

## 输出
- 供应链优化方案
- 库存策略
- 供应商管理
- 成本模型

## Prompt模板
```
请为[业务]优化供应链运营。

背景：
- 业务模式：[模式]
- 供应链现状：[现状]
- 优化目标：[目标]
- 约束条件：[约束]

请输出：
1. 供应链诊断
2. 供应链网络设计
3. 库存策略（安全库存/补货策略）
4. 供应商管理策略
5. 采购流程优化
6. 物流体系设计
7. 成本分析和优化
8. 风险应对
9. 数字化方案
```

## 示例
### 示例1
- 输入：生鲜电商，需要降低损耗，提升配送效率
- 输出：供应链优化方案，含产地直采、冷链优化、预售模式

## 注意事项
- 要端到端优化
- 要控制库存
- 要管理风险
