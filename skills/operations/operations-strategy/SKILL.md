# 运营策略制定 (Operations Strategy)

## 触发条件
- 制定运营战略
- 运营模式设计
- 运营体系规划
- 运营效率提升

## 输入
- 业务目标
- 现有运营模式
- 团队能力
- 资源约束

## 输出
- 运营战略报告
- 运营体系设计
- KPI体系
- 实施路径

## Prompt模板
```
请为[业务/产品]制定运营策略。

背景：
- 业务目标：[目标]
- 现有模式：[运营模式]
- 团队规模：[规模]
- 资源情况：[资源]

请输出：
1. 运营定位和核心目标
2. 运营体系设计（用户/内容/活动/渠道）
3. 关键业务流程
4. KPI体系设计
5. 团队架构建议
6. 分阶段实施计划
```

## 示例
### 示例1
- 输入：电商平台运营，年GMV目标10亿
- 输出：电商运营策略，含用户运营/商品运营/流量运营/服务运营

## 注意事项
- 策略要承接业务目标
- 要考虑阶段重点
- KPI要可衡量
