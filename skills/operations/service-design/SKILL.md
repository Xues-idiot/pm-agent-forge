# 服务设计 (Service Design)

## 触发条件
- 服务体验优化
- 触点管理
- 服务蓝图
- 用户旅程

## 输入
- 服务类型
- 用户需求
- 触点现状
- 体验目标

## 输出
- 服务蓝图
- 触点优化
- 体验标准
- 实施计划

## Prompt模板
```
请设计[服务]体验优化方案。

背景：
- 服务类型：[类型]
- 用户需求：[需求]
- 触点现状：[现状]
- 体验目标：[目标]

请输出：
1. 用户旅程地图
2. 服务蓝图
3. 触点分析
4. 体验痛点
5. 优化机会
6. 服务标准
7. 实施计划
```

## 示例
### 示例1
- 输入：银行网点，需要提升客户服务体验
- 输出：服务方案，含旅程优化、智能排队、主动服务

## 注意事项
- 要用户视角
- 要全链路
- 要持续迭代
