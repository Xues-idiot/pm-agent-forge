# 业务运营 (Business Operations)

## 触发条件
- 业务流程
- 运营效率
- 成本优化
- 持续改进

## 输入
- 业务类型
- 流程现状
- 效率目标
- 资源约束

## 输出
- 运营方案
- 流程优化
- 效率指标
- 实施路径

## Prompt模板
```
请优化[业务]运营体系。

背景：
- 业务类型：[类型]
- 流程现状：[现状]
- 效率目标：[目标]
- 资源约束：[约束]

请输出：
1. 运营诊断
2. 流程优化
3. 效率指标
4. 成本模型
5. 自动化机会
6. 人员配置
7. 工具支撑
8. 持续改进
```

## 示例
### 示例1
- 输入：客服业务，需要降低运营成本
- 输出：运营方案，含流程优化、智能客服、人员效率

## 注意事项
- 要效率优先
- 要数据驱动
- 要持续改进
