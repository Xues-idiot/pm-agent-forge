# 社群运营 (Community Operations)

## 触发条件
- 社群搭建
- 社群活跃
- KOC培育
- 社群变现

## 输入
- 社群定位
- 目标用户
- 运营目标
- 团队资源

## 输出
- 社群策略
- 内容规划
- 活动方案
- 评估体系

## Prompt模板
```
请为[品牌/产品]设计社群运营方案。

背景：
- 社群定位：[定位]
- 目标用户：[用户]
- 运营目标：[目标]
- 团队资源：[资源]

请输出：
1. 社群定位和价值主张
2. 社群结构设计（主群/分群/层级）
3. 用户招募策略
4. 内容规划（日常/专题/活动）
5. 互动机制设计
6. KOC培育计划
7. 变现路径设计
8. 数据监测和评估
9. 运营团队配置
```

## 示例
### 示例1
- 输入：美妆品牌，目标用户是年轻女性，期望通过社群提升复购
- 输出：社群运营方案，含小红书矩阵、微信社群、会员体系

## 注意事项
- 价值要持续提供
- 互动要自然
- 要培育核心用户
