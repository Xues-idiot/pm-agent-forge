# Campaign Operation

营销活动运营。

## 何时使用

- 营销活动策划
- 促销活动执行
- 节日活动运营
- 活动效果分析
- 活动复盘优化

## 活动类型

| 促活活动 | 激活老用户 | DAU/回访 |

## 活动玩法

### 常见玩法

```markdown
## 活动玩法

### 折扣类
| 玩法 | 说明 | 效果 |
|------|------|------|
| 满减 | 满X减Y | 提升客单价 |
| 折扣 | 直接打折 | 快速转化 |
| 限时特价 | 限量优惠 | 紧迫感 |
| 会员价 | 专属优惠 | 会员价值 |

### 红包类
| 玩法 | 说明 | 效果 |
|------|------|------|
| 现金红包 | 直接返现 | 强刺激 |
| 优惠券 | 满X可用 | 复购 |
| 积分 | 积分奖励 | 粘性 |
| 金币 | 虚拟货币 | 留存 |

### 互动类
| 玩法 | 说明 | 效果 |
|------|------|------|
| 抽奖 | 概率中奖 | 参与感 |
| 签到 | 连续奖励 | 习惯 |
| 任务 | 完成任务 | 深度 |
| 分享 | 社交裂变 | 传播 |
```

### 玩法设计原则

```markdown
## 玩法设计

### 好的玩法
- 参与门槛低
- 规则简单易懂
- 奖励感知明显
- 有社交传播点
- 可持续运营

### 避免的坑
- 规则过于复杂
- 奖励感知弱
- 漏洞被套利
- 虚假宣传
- 过度打扰
```

## 红牌警告

- 目标不清晰
- 规则有漏洞
- 技术有风险
- 推广无渠道
- 客服不知情
- 无数据监控
- 不做复盘
