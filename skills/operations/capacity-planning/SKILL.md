# 产能规划 (Capacity Planning)

## 触发条件
- 产能评估
- 扩张规划
- 资源调度
- 成本优化

## 输入
- 业务预测
- 现有产能
- 扩张周期
- 投资预算

## 输出
- 产能方案
- 投资计划
- 风险评估
- 实施路径

## Prompt模板
```
请制定[业务]产能规划。

背景：
- 业务预测：[预测]
- 现有产能：[产能]
- 扩张周期：[周期]
- 投资预算：[预算]

请输出：
1. 需求预测
2. 产能现状评估
3. 缺口分析
4. 扩建方案
5. 投资估算
6. 建设周期
7. 风险评估
8. 实施路径
```

## 示例
### 示例1
- 输入：芯片封装，需求增长50%，需要扩产
- 输出：产能方案，含产线扩建、设备选型、风险评估

## 注意事项
- 要适度超前
- 要控制风险
- 要分步实施
