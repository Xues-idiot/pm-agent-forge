# 运营分析 (Operations Analysis)

## 触发条件
- 运营数据诊断
- 业务问题分析
- 运营效果评估
- 优化方向建议

## 输入
- 业务数据
- 运营指标
- 历史趋势
- 问题描述

## 输出
- 数据诊断报告
- 问题根因分析
- 优化建议
- 行动方案

## Prompt模板
```
请对[业务/产品]的运营数据进行分析诊断。

数据背景：
- 核心指标：[DAU/GMV/转化率等]
- 近期变化：[变化情况]
- 问题现象：[问题描述]
- 历史数据：[数据范围]

请输出：
1. 数据概览和趋势分析
2. 问题定位和根因分析
3. 与行业基准对比
4. 优化机会点
5. 优先级排序和行动建议
```

## 示例
### 示例1
- 输入：电商平台转化率下降，近期从3%降到2.5%
- 输出：转化率下降诊断，含流量/商品/体验多维度分析

## 注意事项
- 要基于数据而非主观判断
- 要找到根本原因
- 建议要可执行
