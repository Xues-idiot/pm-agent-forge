### 关键词布局

```markdown
## 关键词布局

### 布局原则
- 首页：核心词（3-5个）
- 栏目页：次级词（2-3个）
- 内容页：长尾词（每个1-2个）
- 专题页：时效词/热门词

### 布局位置
| 位置 | 密度 | 注意 |
|------|------|------|
| 标题 | 1个核心词 | 前置 |
| description | 自然融入 | 吸引点击 |
| H1 | 核心词 | 仅1个 |
| H2/H3 | 扩展词 | 2-4个 |
| 正文 | 3-5次 | 自然 |
| 图片ALT | 描述词 | 1个 |
```

## 页面优化

### 标题优化

```markdown
## 标题优化

### 标题写法
| 类型 | 格式 | 示例 |
|------|------|------|
| 直觉式 | 核心词+修饰 | 产品名称+评测 |
| 疑问式 | 疑问+回答 | xxx怎么样 |
| 数字式 | 数字+内容 | 10个技巧 |
| 对比式 | A vs B | xxx和xxx哪个好 |
| 权威式 | 品牌/专家 | xxx官方指南 |

### 标题要求
- 长度：30字以内
- 核心词前置
- 包含品牌词
- 避免关键词堆砌

### 示例
```markdown
## 标题示例

### 好标题
- 「产品名」评测：2024年最值得买的xxx
- xxx怎么样？深度使用30天告诉你答案
- 2024年xxx选购指南（附10大品牌对比）

### 坏标题
- xxx|xxx|xxx（堆砌）
- 点击查看xxx（无意义）
- 最全最强的xxx在这里（夸大）
```
```

## 链接建设

### 内链建设

```markdown
## 内链建设

### 内链原则
- 逻辑相关
- 锚文本自然
- 层级清晰
- 不要过度

### 内链策略
```markdown
## 内链策略

### 策略类型
| 策略 | 方法 | 效果 |
|------|------|------|
| 锚文本 | 关键词做链接 | 传递权重 |
| 相关推荐 | 底部推荐相关 | 增加PV |
| 面包屑 | 层级导航 | 增强结构 |
| 导航 | 顶部导航 | 重要页面 |

### 注意事项
1. 不要过度内链
2. 不要用相同锚文本
3. 不要链接到错误页面
4. 不要忽视nofollow
```
```
