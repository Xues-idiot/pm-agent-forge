# 服务水平管理 (Service Level Management)

## 触发条件
- SLA设计
- 服务监控
- 服务报告
- 服务改进

## 输入
- 服务类型
- 客户需求
- 服务能力
- 成本约束

## 输出
- SLA设计
- 监控方案
- 报告机制
- 改进计划

## Prompt模板
```
请设计[服务]SLA方案。

背景：
- 服务类型：[类型]
- 客户需求：[需求]
- 服务能力：[能力]
- 成本约束：[约束]

请输出：
1. SLA框架
2. 服务目录
3. 响应时间
4. 解决时间
5. 可用性指标
6. 监控机制
7. 报告机制
8. 持续改进
```

## 示例
### 示例1
- 输入：IT服务外包，需要与客户签订SLA
- 输出：SLA方案，含响应时间、解决时间、可用性保证

## 注意事项
- 要合理可行
- 要可衡量
- 要持续改进
