# 环境管理 (Environmental Management)

## 触发条件
- 环境合规
- 污染治理
- 绿色转型
- 碳排放

## 输入
- 行业类型
- 排放情况
- 法规要求
- 减排目标

## 输出
- 环境方案
- 合规清单
- 减排措施
- 认证计划

## Prompt模板
```
请优化[组织]环境管理。

背景：
- 行业类型：[行业]
- 排放情况：[排放]
- 法规要求：[要求]
- 减排目标：[目标]

请输出：
1. 环境现状评估
2. 合规要求清单
3. 污染防治措施
4. 资源节约
5. 碳减排路径
6. 绿色技术
7. 认证和报告
8. 持续改进
```

## 示例
### 示例1
- 输入：制造企业，需要达到碳达峰要求
- 输出：环境方案，含节能改造、清洁能源、碳交易

## 注意事项
- 要合规优先
- 要绿色发展
- 要持续改进
