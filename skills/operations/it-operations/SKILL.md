# IT运维 (IT Operations)

## 触发条件
- 运维体系
- 监控告警
- 自动化运维
- 运维效率

## 输入
- 系统规模
- 服务级别
- 团队能力
- 工具现状

## 输出
- 运维方案
- 监控体系
- 自动化
- 效率指标

## Prompt模板
```
请优化[组织]IT运维体系。

背景：
- 系统规模：[规模]
- 服务级别：[SLA]
- 团队能力：[能力]
- 工具现状：[现状]

请输出：
1. 运维成熟度评估
2. 监控体系
3. 告警机制
4. 自动化运维
5. 变更管理
6. 容量管理
7. 成本控制
8. 效率指标
```

## 示例
### 示例1
- 输入：互联网公司，100台服务器，需要提升运维效率
- 输出：运维方案，含监控、自动化、容量管理

## 注意事项
- 要自动化优先
- 要可观测
- 要持续改进
