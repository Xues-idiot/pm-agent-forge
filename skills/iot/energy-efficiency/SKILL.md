# 能源管理 (Energy Management)

## 触发条件
- 能源管理体系
- 能耗监控
- 节能改造
- 碳排放管理

## 输入
- 能源类型
- 用能现状
- 节能目标
- 投资能力

## 输出
- 能源管理方案
- 监控体系
- 节能措施
- 碳减排路径

## Prompt模板
```
请为[组织]设计能源管理方案。

背景：
- 能源类型：[类型]
- 用能现状：[现状]
- 节能目标：[目标]
- 投资能力：[能力]

请输出：
1. 能源管理目标和原则
2. 能源监测体系设计
3. 能源数据分析
4. 节能机会识别
5. 节能技术措施
6. 行为节能
7. 新能源应用
8. 碳排放管理
9. 投资回报分析
10. 实施计划
```

## 示例
### 示例1
- 输入：工业园区，年用电1亿度，想要降低15%能耗
- 输出：能源方案，含智能电表、节能改造、分布式光伏

## 注意事项
- 要全面诊断
- 要技术+管理结合
- 要持续优化
