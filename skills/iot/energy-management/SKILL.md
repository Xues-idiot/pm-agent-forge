# Energy Management Skill

管理能源消耗，优化能源效率，监控物联网环境中的能源系统。

## 何时使用

- 能耗监控
- 节能优化
- 能源审计
- 碳排放管理
- 电费优化
- 设备能效管理

## 能源管理流程

```
1. 能源监测 → 2. 数据分析 → 3. 节能优化 → 4. 效果验证 → 5. 持续改进
```

## 红牌警告

- 监测数据造假
- 能耗严重超标
- 设备带病运行
- 节能措施无效
- 投资回报不达
- 碳排放超标
- 安全隐患未改
- 管理责任缺失
