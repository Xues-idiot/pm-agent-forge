## 边缘基础

### 边缘计算概念

```markdown
## 边缘计算

### 计算架构
| 架构 | 位置 | 特点 | 适用 |
|------|------|------|------|
| 云计算 | 云端数据中心 | 资源丰富 | 复杂处理 |
| 边缘计算 | 近用户节点 | 低延迟 | 实时处理 |
| 终端计算 | 设备本地 | 最快 | 简单推理 |

### 边缘优势
| 优势 | 说明 | 应用 |
|------|------|------|
| 低延迟 | 网络一跳 | 实时控制 |
| 大带宽 | 减少传输 | 视频处理 |
| 数据省 | 减少上云 | 带宽敏感 |
| 可靠性 | 离线可用 | 关键业务 |
| 隐私 | 本地处理 | 敏感数据 |

### 边缘 vs 云计算
```markdown
## 对比

| 维度 | 边缘 | 云 |
|------|------|---|
| 延迟 | <10ms | 50-200ms |
| 成本 | 设备成本 | 按量付费 |
| 能力 | 有限 | 无限 |
| 可靠性 | 离线可用 | 依赖网络 |
| 运维 | 分散 | 集中 |
```
```
