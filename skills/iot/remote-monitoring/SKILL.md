# 远程监控 (Remote Monitoring)

## 触发条件
- 设备监控
- 状态诊断
- 预警通知
- 远程运维

## 输入
- 设备类型
- 监控参数
- 响应要求
- 运维能力

## 输出
- 监控方案
- 阈值设计
- 预警机制
- 运维流程

## Prompt模板
```
请设计[设备]远程监控方案。

背景：
- 设备类型：[类型]
- 监控参数：[参数]
- 响应要求：[要求]
- 运维能力：[能力]

请输出：
1. 监控架构
2. 数据采集
3. 参数阈值
4. 预警机制
5. 诊断分析
6. 运维流程
7. 远程控制
8. 系统集成
```

## 示例
### 示例1
- 输入：工业设备，需要远程监控运行状态
- 输出：监控方案，含数据采集、阈值告警、远程诊断

## 注意事项
- 要实时准确
- 要及时预警
- 要闭环处理
