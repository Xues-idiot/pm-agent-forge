# 安全设计 (Security Design)

## 触发条件
- 安全架构设计
- 物联网安全
- 数据保护
- 合规设计

## 输入
- 系统类型
- 威胁模型
- 保护资产
- 合规要求

## 输出
- 安全架构
- 防护措施
- 监控体系
- 合规方案

## Prompt模板
```
请设计[系统]安全方案。

背景：
- 系统类型：[类型]
- 威胁模型：[威胁]
- 保护资产：[资产]
- 合规要求：[要求]

请输出：
1. 安全目标和原则
2. 威胁建模
3. 安全架构设计
4. 身份认证
5. 访问控制
6. 数据加密
7. 网络安全
8. 应用安全
9. 安全监控
10. 应急响应
```

## 示例
### 示例1
- 输入：工业控制系统，需要符合等保2.0
- 输出：安全方案，含网络隔离、纵深防御、安全审计

## 注意事项
- 要纵深防御
- 要持续监控
- 要持续改进
