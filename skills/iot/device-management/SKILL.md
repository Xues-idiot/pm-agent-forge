# IoT Device Management Skill

管理物联网设备，覆盖设备全生命周期。

## 何时使用

- 新设备注册和配置
- 设备固件更新
- 设备分组管理
- 设备状态监控
- 设备退役和替换
- 批量设备操作

## 设备管理流程

```
1. 设备接入 → 2. 设备配置 → 3. 设备监控 → 4. 设备维护 → 5. 设备退役
```

```markdown

| 步骤 | 操作 | 验证 |
```

## 红牌警告

- 设备未经测试直接入网
- 固件更新无回滚方案
- 批量操作无验证机制
- 设备状态异常未处理
- 配置变更未记录
- 设备证书过期未更新
- 敏感配置未加密
