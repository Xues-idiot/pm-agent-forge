# 资产追踪 (Asset Tracking)

## 触发条件
- 资产管理
- 位置追踪
- 维护管理
- 资产利用

## 输入
- 资产类型
- 资产规模
- 追踪需求
- 技术能力

## 输出
- 追踪方案
- 资产管理
- 维护计划
- 利用分析

## Prompt模板
```
请设计[资产]追踪方案。

背景：
- 资产类型：[类型]
- 资产规模：[规模]
- 追踪需求：[需求]
- 技术能力：[能力]

请输出：
1. 资产管理
2. 追踪技术
3. 位置管理
4. 使用监控
5. 维护管理
6. 资产盘点
7. 利用分析
```

## 示例
### 示例1
- 输入：医院医疗设备，需要追踪位置和使用
- 输出：追踪方案，含RFID追踪、位置地图、维护提醒

## 注意事项
- 要精准追踪
- 要实时监控
- 要数据分析
