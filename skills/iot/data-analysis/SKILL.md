# IoT数据分析 (IoT Data Analytics)

## 触发条件
- IoT数据处理
- 时序数据分析
- 异常检测
- 预测性维护

## 输入
- 数据类型
- 数据规模
- 分析需求
- 实时性要求

## 输出
- 数据分析方案
- 算法设计
- 实时处理架构
- 应用场景

## Prompt模板
```
请为[IoT场景]设计数据分析方案。

背景：
- 设备类型：[类型]
- 数据类型：[类型]
- 数据规模：[规模]
- 分析需求：[需求]

请输出：
1. 数据分析架构
2. 数据采集和预处理
3. 时序数据库选型
4. 实时分析设计
5. 离线分析设计
6. 异常检测算法
7. 预测性维护方案
8. 数据可视化
9. 分析结果应用
```

## 示例
### 示例1
- 输入：工业设备监控，需要预测故障，数据量TB级
- 输出：分析方案，含时序数据库、异常检测、预测模型

## 注意事项
- 要考虑实时性
- 算法要轻量化
- 结果要可解释
