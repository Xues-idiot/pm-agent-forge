# IoT系统架构 (IoT System Architecture)

## 触发条件
- IoT系统规划
- 设备连接架构
- 边缘计算设计
- 云平台架构

## 输入
- 应用场景
- 设备规模
- 数据需求
- 可靠性要求

## 输出
- 系统架构设计
- 设备接入方案
- 数据处理流程
- 技术选型

## Prompt模板
```
请为[IoT应用场景]设计系统架构。

背景：
- 应用场景：[场景]
- 设备规模：[规模]
- 数据需求：[需求]
- 可靠性要求：[要求]

请输出：
1. 系统架构概览
2. 设备接入层设计（协议/网关）
3. 边缘计算架构
4. 云平台架构（计算/存储/网络）
5. 数据处理流程（采集-传输-处理-分析）
6. 应用服务层设计
7. 安全架构
8. 可靠性设计
9. 技术选型和成本估算
```

## 示例
### 示例1
- 输入：智慧工厂场景，万级设备接入，需要实时控制
- 输出：IoT架构，含边缘计算、TSN网络、实时数据处理

## 注意事项
- 要考虑规模扩展
- 协议要标准化
- 安全要贯穿全栈
