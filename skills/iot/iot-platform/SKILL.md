# IoT平台设计 (IoT Platform Design)

## 触发条件
- IoT平台架构
- 设备接入
- 数据处理
- 应用支撑

## 输入
- 应用场景
- 设备规模
- 功能需求
- 性能要求

## 输出
- 平台架构
- 功能设计
- 技术选型
- 实施路径

## Prompt模板
```
请设计[IoT应用]平台方案。

背景：
- 应用场景：[场景]
- 设备规模：[规模]
- 功能需求：[需求]
- 性能要求：[要求]

请输出：
1. 平台定位
2. 系统架构
3. 设备接入
4. 数据处理
5. 应用支撑
6. 安全设计
7. 运维监控
8. 技术选型
9. 实施路径
```

## 示例
### 示例1
- 输入：智慧城市IoT平台，千万级设备接入
- 输出：平台方案，含设备管理、数据中台、应用支撑

## 注意事项
- 要高扩展
- 要高可靠
- 要安全第一
