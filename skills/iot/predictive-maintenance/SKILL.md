# Predictive Maintenance Skill

监控设备健康，预测故障，规划维护计划。

## 何时使用

- 设备健康监测
- 故障预测
- 维护计划制定
- 备件管理
- 设备寿命管理

## 预测性维护流程

```
1. 数据采集 → 2. 状态监测 → 3. 故障预测 → 4. 维护决策 → 5. 效果评估
```

### 综合健康指数
HI = Σ(参数得分 × 权重) / Σ权重

| HI范围 | 健康等级 | 状态 |
|--------|----------|------|
| 90-100 | 优秀 | 正常运行 |
| 70-89 | 良好 | 注意观察 |
| 50-69 | 一般 | 计划维护 |
| 30-49 | 较差 | 尽快维护 |
| 0-29 | 危险 | 立即停机 |
```

### 3. 故障预测

**故障预测模型：**

## 故障预测

### 预测方法
| 方法 | 适用场景 | 精度 | 数据需求 |
|------|----------|------|----------|
| 阈值告警 | 已知故障模式 | 中 | 少 |

### 故障模式库
```markdown
## 故障模式

| 故障类型 | 典型特征 | 预测指标 | 提前预警时间 |
|----------|----------|----------|--------------|
| 轴承磨损 | 振动增大 | 振动值 | 1-2周 |
| 密封泄漏 | 压力下降 | 压力差 | 3-7天 |
| 电机过热 | 温度上升 | 温度+电流 | 1-3天 |
| 润滑不良 | 摩擦增大 | 振动+温度 | 1-2周 |
| 电气故障 | 电流异常 | 电流波形 | 1-7天 |
```

### 剩余寿命预测（RUL）
```markdown
## RUL 预测

### RUL 计算公式
RUL = (当前阈值 - 当前值) / 退化速率

### 示例
```
轴承当前振动：6.5mm/s
危险阈值：12mm/s
退化速率：0.3mm/s/天

RUL = (12 - 6.5) / 0.3 = 18天
```
```

### 预警规则：

```markdown
## 预警规则

| 预警级别 | 条件 | 通知方式 | 响应时间 |
|----------|------|----------|----------|
| P1-紧急 | HI<50或RUL<7天 | 短信+电话 | 立即 |
| P2-警告 | HI 50-70或RUL<30天 | 企微/钉钉 | 24小时 |
| P3-注意 | HI 70-85或异常趋势 | 系统通知 | 72小时 |
| P4-正常 | HI>85 | 仅记录 | - |

### 4. 维护决策

**维护策略选择：**

```markdown
## 维护策略

### 策略对比
| 策略 | 触发条件 | 优点 | 缺点 |
|------|----------|------|------|
| 事后维修 | 故障后 | 维护成本低 | 停机损失大 |
| 定期维修 | 按计划周期 | 可预测 | 可能过度维护 |
| 预测维修 | 故障前兆 | 最优效率 | 技术要求高 |
| 主动维护 | 根因分析 | 根本解决问题 | 成本高 |

### 维护决策树
```
设备报警？
    ↓
HI < 50? → 是 → 立即维护（P1）
    ↓否
HI < 70? → 是 → 计划维护（本周内）
    ↓否
RUL < 30天? → 是 → 计划维护（本月内）
    ↓否
继续监测
```
```
