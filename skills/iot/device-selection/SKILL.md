# IoT Device Selection Skill

选择物联网设备，评估硬件选项，规划物联网设备采购。

## 何时使用

- 设备选型
- 供应商评估
- 技术方案制定
- 设备采购规划
- 成本评估
- 兼容性分析

## 设备选型流程

```
1. 需求分析 → 2. 技术评估 → 3. 供应商选择 → 4. 成本分析 → 5. 采购实施
```

### TCO计算
```markdown
## TCO = 采购成本 + 运营成本 × 使用年限

示例：
采购成本：100万
年运营成本：20万
使用年限：5年

TCO = 100 + 20×5 = 200万
年化成本 = 200/5 = 40万/年
```
