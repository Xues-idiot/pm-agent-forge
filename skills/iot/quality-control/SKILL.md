# 质量控制 (Quality Control)

## 触发条件
- 质量管理体系
- 质量检测
- 质量改进
- 质量数据分析

## 输入
- 行业类型
- 质量现状
- 质量目标
- 检测能力

## 输出
- 质量体系
- 检测方案
- 改进方法
- 评估指标

## Prompt模板
```
请为[制造企业]设计质量控制方案。

背景：
- 行业类型：[类型]
- 质量现状：[现状]
- 质量目标：[目标]
- 检测能力：[能力]

请输出：
1. 质量管理体系设计
2. 质量标准建立
3. 检测方案设计
4. 过程质量控制
5. 来料质量控制
6. 出货质量控制
7. 质量数据分析
8. 质量改进方法（Six Sigma等）
9. 供应商质量管理
10. 质量成本分析
```

## 示例
### 示例1
- 输入：电子制造厂，客诉率2%，目标降到0.5%
- 输出：质量方案，含SPC控制、质量追溯、8D分析

## 注意事项
- 要预防为主
- 要全员参与
- 要持续改进
