# 智能制造 (Smart Manufacturing)

## 触发条件
- 智能制造规划
- 数字化工厂
- 产线自动化
- 工业4.0

## 输入
- 制造类型
- 现状评估
- 转型目标
- 投资预算

## 输出
- 转型规划
- 技术架构
- 实施路径
- 投资回报

## Prompt模板
```
请为[制造企业]规划智能制造转型。

背景：
- 制造类型：[类型]
- 现状评估：[评估]
- 转型目标：[目标]
- 投资预算：[预算]

请输出：
1. 智能制造成熟度评估
2. 转型目标和路径
3. 技术架构设计
4. 数据采集和联网
5. MES/ERP集成
6. 质量管理智能化
7. 预测性维护
8. 能源管理
9. 实施路径和里程碑
10. 投资回报分析
```

## 示例
### 示例1
- 输入：汽车零部件厂，需要提升生产效率，降低次品率
- 输出：智能制造方案，含产线数字化、质量追溯、预测维护

## 注意事项
- 要渐进式推进
- 要标准先行
- 要注重实效
