# Go-to-Market Strategy Skill

制定产品上市策略，规划市场进入，执行推广计划。

## 何时使用

- 新产品发布
- 市场进入
- 产品上市规划
- 渠道策略
- 定价上市
- 竞品上市应对

## GTM 流程

```
1. 市场分析 → 2. 定位策略 → 3. 渠道设计 → 4. 营销计划 → 5. 执行上市 → 6. 效果复盘
```

## 红牌警告

- 产品未ready就上市
- 定价脱离市场
- 渠道选择错误
- 目标用户不清晰
- 竞品反应预估不足
- 上市时间延误
- 部门协作不畅
- 效果追踪缺失
