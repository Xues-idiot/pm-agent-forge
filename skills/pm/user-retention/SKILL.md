## 红牌警告

- 只关注拉新不关注留存
- 无流失预警机制
- 新用户引导差
- 过度打扰用户
- 忠诚度没有价值

## 好/坏对比

**好**：
```
留存运营：
- 建立完善的留存指标
- 早期引导体验核心价值
- 活跃机制丰富
- 流失预警+召回
- 忠诚度体系有价值
→ 留存率高、用户价值大
```

**坏**：
```
留存运营：
- 只管拉新不管留存
- 没有流失预警
- 新手引导缺失
- 过度推送骚扰
- 会员体系鸡肋
→ 流失率高、价值低
```
