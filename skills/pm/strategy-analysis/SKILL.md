# Strategy Analysis

战略分析，制定长期规划和做出战略决策。

## 何时使用

- 战略规划
- 市场分析
- 商业决策
- 资源配置
- 竞争优势建立

## 战略框架

### SWOT分析

```markdown
## SWOT分析

### 矩阵
| | 正面 | 负面 |
|--|-----|-----|
| **内部** | Strengths优势 | Weaknesses劣势 |
| **外部** | Opportunities机会 | Threats威胁 |

### 分析模板
```markdown
## SWOT分析

### Strengths (优势)
- 内部资源/能力
- 竞争优势

### Weaknesses (劣势)
- 内部不足
- 资源缺口

### Opportunities (机会)
- 市场趋势
- 政策变化
- 技术变革

### Threats (威胁)
- 竞争对手
- 市场风险
- 替代品
```

### 战略选择
| 战略 | 说明 | 适用条件 |
|------|------|----------|
| SO进攻 | 利用优势抓住机会 | 强势+机会 |
| WO改善 | 利用机会弥补劣势 | 机会+劣势 |
| ST防御 | 利用优势回避威胁 | 强势+威胁 |
| WT防御 | 弥补劣势回避威胁 | 劣势+威胁 |

## 红牌警告

- 分析流于形式
- 战略不落地
- 忽视竞争对手
- 资源不匹配
- 不做动态调整

## 好/坏对比

**好**：
```
战略规划：
- 系统分析市场和竞争
- 明确商业模式
- 选择差异化定位
- 制定可执行计划
- 定期复盘调整
→ 战略清晰、执行有力
```

**坏**：
```
战略规划：
- 拍脑袋决策
- 竞品分析缺失
- 战略假大空
- 计划不执行
→ 战略迷失、执行混乱
```
