- 资源协调

## 红牌警告

- 干系人遗漏
- 沟通不及时
- 冲突不处理
- 资源浪费
- 信息不对称
- 各自为战

## 好/坏对比

**好**：
```
跨部门协作：
- 干系人管理到位
- 沟通机制明确
- 冲突及时处理
- 资源合理分配
- 信息充分共享
→ 高效协作、目标达成
```

**坏**：
```
跨部门协作：
- 沟通不畅
- 责任不清
- 冲突升级
- 资源争抢
- 信息孤岛
→ 效率低下、项目失败
```
