# Pricing Strategy Skill

制定定价策略，分析定价模型，优化产品定价。

## 何时使用

- 新产品定价
- 价格调整
- 定价模型选择
- 促销活动设计
- 竞品价格分析

## 定价流程

```
1. 成本分析 → 2. 市场研究 → 3. 定价模型选择 → 4. 价格测试 → 5. 监控优化
```

### 3. 定价模型选择

**定价模式：**

```markdown
## 定价模式

### 常用模式对比
| 模式 | 适用场景 | 优点 | 缺点 |
|------|---------|------|------|
| 成本加成 | B2B/制造业 | 简单保底 | 忽视市场 |
| 竞品定价 | 成熟市场 | 快速定位 | 被动跟随 |
| 价值定价 | 差异化产品 | 利润最大化 | 难量化价值 |
| 渗透定价 | 新市场开拓 | 快速获客 | 利润低 |
| 撇脂定价 | 创新产品 | 早期利润高 | 容易模仿 |
| 订阅制 | SaaS/会员 | 稳定收入 | 需持续价值 |
| 用量定价 | API/资源 | 匹配成本 | 波动大 |

### 套餐设计
```markdown
## 套餐结构

### 三层套餐
| 层级 | 价格 | 目标用户 | 核心功能 | 功能数 |
|------|------|----------|----------|--------|
| 基础版 | X元/月 | 个人/小微 | 核心功能 | 10 |
| 专业版 | Y元/月 | 中小企业 | 高级功能 | 30 |
| 企业版 | Z元/月 | 大型企业 | 全部功能+定制 | 不限 |

### 套餐设计原则
- 基础版：满足基本需求，降低试用门槛
- 专业版：最受欢迎，利润主力
- 企业版：高价高价值，定制化
```

**定价心理技巧：**

```markdown
## 定价心理学

### 价格呈现
- [ ] 399 → 400以下（心理阈值）
- [ ] 99 → 100以下（数字位数）
- [ ] 月付/年付切换（降低感知成本）

### 锚定效应
- 高价SKU在前，引导选择次高价
- 原价划掉，现价突出

### 套餐捆绑
- [ ] 功能捆绑（增加感知价值）
- [ ] 时间捆绑（年付折扣）
- [ ] 跨产品捆绑（生态锁定）
```

### 5. 监控优化

**价格监控看板：**

```markdown
## 监控指标

### 核心指标
| 指标 | 定义 | 每日监控 |
|------|------|---------|
| 转化率 | 付费/访问 | ✅ |
| 客单价 | 收入/订单数 | ✅ |
| ARPU | 收入/活跃用户 | ✅ |
| 付费率 | 付费用户/总用户 | ✅ |

### 长期指标
| 指标 | 定义 | 每周监控 |
|------|------|---------|
| LTV | 用户生命周期价值 | ✅ |
| CAC | 获客成本 | ✅ |
| LTV/CAC |  ratio | ✅ |
| ROI | 投入产出比 | ✅ |

### 调价触发条件
- [ ] 成本变化 >10%
- [ ] 竞品价格大调整
- [ ] 转化率持续下降
- [ ] 用户反馈价格过高
```

## 定价策略检查清单

- [ ] 成本分析完整
- [ ] 竞品分析全面
- [ ] 定价模式合理
- [ ] 用户价值可量化
- [ ] 测试验证假设
- [ ] 监控指标到位
- [ ] 调价预案清晰
- [ ] 法律合规（明码标价）

## 红牌警告

- 定价低于成本
- 忽视竞品定价
- 价格体系混乱
- 未经测试大调价
- 违反价格法规
- 只看短期转化忽视LTV
- 定价与产品定位不一致
- 促销活动无底线
