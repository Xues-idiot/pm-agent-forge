# Market Analysis

市场分析，研究市场机会和竞争格局。

## 何时使用

- 市场机会评估
- 市场规模测算
- 竞争格局分析
- 市场趋势研究
- 投资决策支持

## 市场分析基础

### 分析框架

```markdown
## 市场分析框架

### 常用模型
| 模型 | 适用 | 要点 |
|------|------|------|
| PEST | 宏观环境 | 政治/经济/社会/技术 |
| 波特五力 | 行业结构 | 供应商/买家/新进入者/替代品/竞争对手 |
| SWOT | 战略分析 | 优势/劣势/机会/威胁 |
| 3C | 竞争策略 | 公司/竞争对手/客户 |
| STP | 定位分析 | 细分/目标/定位 |

### 分析层次
| 层次 | 内容 | 方法 |
|------|------|------|
| 宏观分析 | 政策/经济/社会/技术 | PEST |
| 行业分析 | 规模/结构/趋势 | 波特五力 |
| 市场分析 | 细分/规模/增长 | 市场规模模型 |
| 竞争分析 | 格局/对手/策略 | 竞品分析框架 |
```

## 市场规模

### 市场规模测算

```markdown
## 市场规模测算

### 测算方法
| 方法 | 适用 | 公式 |
|------|------|------|
| 自上而下 | 有公开数据 | 总量×渗透率 |
| 自下而上 | 基础数据 | 单元×数量×价格 |
| 市场契合法 | 创新产品 | 深度访谈+估算 |

### 关键概念
```markdown
## 市场规模概念

### TAM (Total Addressable Market)
- 全部潜在市场
- 理论最大值

### SAM (Serviceable Available Market)
- 可服务市场
- 目标区域/产品

### SOM (Serviceable Obtainable Market)
- 可获得市场
- 实际可获得
```

### 增长分析

```markdown
## 市场增长分析

### 增长指标
| 指标 | 计算 | 含义 |
|------|------|------|
| 增长率 | (本期-上期)/上期 | 市场规模变化 |
| CAGR | (末值/初值)^(1/n)-1 | 年复合增长率 |
| 渗透率 | 用户数/潜在用户 | 市场成熟度 |

### 增长阶段
```markdown
## 市场增长阶段

### 阶段特征
| 阶段 | 特征 | 增长率 | 竞争 |
|------|------|--------|------|
| 导入期 | 早期用户 | <20% | 少 |
| 成长期 | 快速扩张 | 20-50% | 增加 |
| 成熟期 | 稳定增长 | 5-15% | 激烈 |
| 衰退期 | 负增长 | <0% | 退出 |

### 阶段判断
1. 政策支持度
2. 技术成熟度
3. 用户接受度
4. 资本投入度
```
```

### 趋势判断
```markdown
## 趋势判断

### 判断要素
1. 持续性：趋势维持时间
2. 普遍性：影响范围大小
3. 可逆性：是否可逆转
4. 显著性：影响程度大小

### 信号识别
- 早期信号：苗头/试探
- 中期信号：规模应用
- 晚期信号：全面爆发
```

### 可行性评估

```markdown
## 可行性评估

### 评估维度
| 维度 | 关键问题 | 输出 |
|------|----------|------|
| 市场吸引力 | 规模/增长/盈利 | 是/否 |
| 竞争地位 | 差异化/能力 | 优势/劣势 |
| 财务回报 | ROI/回收期 | 预期 |
| 风险可控 | 技术/政策/执行 | 风险清单 |

### 最终结论
- 强烈推荐：各项评估优秀
- 推荐：多项评估良好
- 谨慎推荐：有明显风险
- 不推荐：风险大于机会
```

## 红牌警告

- 数据来源不可靠
- 市场规模乱拍脑袋
- 竞品分析浅尝辄止
- 趋势判断缺乏依据
- 忽略政策监管风险
- 报告结论模棱两可
