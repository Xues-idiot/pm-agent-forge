# PM Market Insight Skill

评估市场规模、理解市场趋势、识别市场机会。

## 何时使用

- 评估新业务方向
- 制定年度产品规划
- 融资市场分析
- 验证产品市场匹配度

## 市场洞察流程

```
1. 明确分析目的 → 2. 收集市场数据 → 3. 市场规模估算 → 4. 趋势分析 → 5. 机会评估
```

### 1. 明确分析目的

| 目的 | 需要的分析深度 | 数据精度要求 |
|------|---------------|-------------|
| 战略规划 | 深度 | 高 |
| 产品方向 | 中等 | 中 |
| 融资pitch | 深度+故事 | 中高 |
| 内部汇报 | 中等 | 中 |

### 2. 收集市场数据

**数据来源：**
| 来源 | 适用场景 | 可靠性 |
|------|---------|--------|
| 行业协会报告 | 整体市场规模 | 高 |
| 上市公司财报 | 细分市场数据 | 高 |
| 咨询公司报告 | 趋势分析 | 中高 |
| 政府统计数据 | 宏观数据 | 高 |
| 第三方平台 | 电商/流量数据 | 中 |
| 专家访谈 | 定性验证 | 中 |

### 3. 市场规模估算

**TAM/SAM/SOM 框架：**

```
        TAM (Total Addressable Market)
       ┌─────────────────────────────┐
       │                             │
       │     SAM (Serviceable        │
       │     Addressable Market)     │
       │   ┌───────────────────┐     │
       │   │                   │     │
       │   │   SOM (Serviceable │
       │   │   Obtainable      │
       │   │   Market)         │     │
       │   └───────────────────┘     │
       └─────────────────────────────┘
```

**计算方法：**

**方法一：自上而下**
```markdown
## TAM 估算
目标用户群体：全国中小企业（约5000万家）
产品渗透率假设：10%（5年内）
客单价：¥299/年
TAM = 5000万 × 10% × ¥299 = ¥149.5亿
```

**方法二：自下而上**
```markdown
## SOM 估算
第一年目标：1000家企业客户
第二年目标：5000家
第三年目标：20000家
客单价：¥299/年
第三年SOM = 20000 × ¥299 = ¥598万
市占率 = ¥598万 / ¥149.5亿 = 0.04%
```

### 4. 趋势分析

**趋势分析框架：**

| 趋势类型 | 分析问题 | 数据来源 |
|---------|---------|---------|
| 技术趋势 | 技术变革如何影响市场？ | 技术报告、专利 |
| 监管趋势 | 政策变化带来什么机会？ | 政府文件、行业政策 |
| 消费趋势 | 用户偏好如何变化？ | 调研报告、社交媒体 |
| 经济趋势 | 宏观经济如何影响？ | 财经报告 |
| 社会趋势 | 人口/文化变化？ | 人口统计、趋势报告 |

### 5. 机会评估

**机会评估矩阵：**

```
                    市场规模
                       ↑
                       │    ★ 战略机会      ● 明星业务
                       │    （大市场+弱能力）│（大市场+强能力）
                       │
                       │
  ─────────────────────┼────────────────────→
                       │    ○ 维持现状       △ 放弃
                       │   （小市场+弱能力） │（小市场+强能力）
                       │
                       └──────────────────────→
                              竞争激烈程度
```

**评估维度：**

| 维度 | 评估问题 | 权重 |
|------|---------|------|
| 市场吸引力 | 规模、增长、盈利性？ | 30% |
| 竞争强度 | 进入壁垒、对手实力？ | 25% |
| 技术可行性 | 我们能做吗？ | 20% |
| 战略契合 | 符合公司方向吗？ | 15% |
| 组织能力 | 我们有团队执行吗？ | 10% |

## 市场分析方法

### PEST 分析

| 维度 | 要点 |
|------|------|
| Political | 政策、法规、政府支持 |
| Economic | 经济周期、消费能力、成本 |
| Social | 人口、习惯、价值观 |
| Technological | 技术创新、研发投入 |

### 波特五力

| 力 | 分析要点 |
|----|---------|
| 现有竞争 | 竞争者数量、集中度 |
| 新进入者 | 壁垒高低、品牌认知 |
| 替代品 | 替代威胁、转换成本 |
| 上游议价 | 供应商集中度、替代性 |
| 下游议价 | 买家集中度、价格敏感度 |

### 市场生命周期

```
    渗透率
      ↑        成长期       成熟期       衰退期
      │     ↗            ↗            ↗
      │   ↗              │            │
      │ ↗                │            │
  ────┼──────────────────┼────────────┼────→
      │                  │            │
      └──────────────────┴────────────┘
           导入期           时间
```

## 市场洞察检查清单

- [ ] 数据来源可靠
- [ ] 市场定义清晰
- [ ] 规模估算有方法论
- [ ] 趋势有数据支撑
- [ ] 机会评估多维度
- [ ] 风险识别全面
- [ ] 结论与数据一致

## 红牌警告

- 市场规模没有来源
- 假设市场会高速增长
- 忽略竞争对手的反应
- 只看总量不看结构
- 趋势判断没有数据支撑
- 过于乐观的机会评估

## 好/坏对比

**好分析：**
```
私域运营工具市场TAM约500亿，年增长率15%。
我们聚焦的SMB私域运营细分市场SAM约50亿。
基于当前产品能力和团队，我们3年内可获得5%市场，即SOM约2.5亿。
```

**坏分析：**
```
市场很大，有几千亿空间，我们要做到第一。
```
