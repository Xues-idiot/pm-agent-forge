# Competitive Intelligence

竞品情报分析，跟踪竞品和制定战略洞察。

## 何时使用

- 竞品分析
- 竞争情报收集
- 战略决策支持
- 市场定位
- 产品差异化

## 战略应用

### 差异化策略

```markdown
## 差异化策略

### 差异化维度
| 维度 | 竞品A | 竞品B | 我们 |
|------|--------|--------|------|
| 价格 | 高 | 中 | 低 |
| 功能 | 全 | 专 | 精 |
| 体验 | 复杂 | 一般 | 简单 |
| 服务 | 差 | 中 | 好 |

### 差异化方向
1. 功能聚焦 vs 功能全面
2. 高端 vs 性价比
3. 体验简洁 vs 功能强大
4. 服务优先 vs 工具优先
```

## 竞品分析报告

```markdown
# 竞品分析报告

## 竞品概览
| 竞品 | 定位 | 定价 | 市场份额 |
|------|------|------|----------|
| 竞品A | 高端市场 | ¥999/月 | 30% |
| 竞品B | 中端市场 | ¥299/月 | 40% |
| 我们 | 性价比 | ¥99/月 | 10% |

## 核心发现
1. 竞品A功能最全但价格高
2. 竞品B价格中庸但体验一般
3. 我们有机会在性价比市场

## 战略建议
1. 强化"便宜又好用"定位
2. 快速跟进竞品A的核心功能
3. 在体验上超越竞品B
```

## 检查清单

- [ ] 竞品选择合理
- [ ] 信息来源多元
- [ ] 数据定期更新
- [ ] SWOT分析完整
- [ ] 战略建议可行

## 红牌警告

- 只看公开信息
- 不跟踪竞品动态
- 盲目跟随竞品
- 忽视用户声音

## 好/坏对比

**好**：
```
竞品分析：
- 定期跟踪
- 多渠道收集
- 深度SWOT
- 战略建议落地
→ 形成竞争壁垒
```

**坏**：
```
竞品分析：
- 偶尔看一下
- 只看官网
- 没有分析
- 竞品跟了但没意义
→ 被动挨打
```
