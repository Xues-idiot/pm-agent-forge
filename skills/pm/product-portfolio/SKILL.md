# Product Portfolio

产品组合管理，优化产品线和做产品决策。

## 何时使用

- 产品线管理
- 产品组合分析
- 产品生命周期管理
- 资源分配决策
- 产品淘汰/退市

## 组合分析

### 组合框架

```markdown
## 产品组合

### 组合类型
| 类型 | 说明 | 示例 |
|------|------|------|
| 现金牛 | 高份额低增长 | 成熟产品 |
| 明星 | 高份额高增长 | 增长产品 |
| 问题 | 低份额高增长 | 新兴产品 |
| 瘦狗 | 低份额低增长 | 衰退产品 |

### 波士顿矩阵
```markdown
## 波士顿矩阵

高增长
    ↑
    │  明星      问题
    │   ↑          ↑
    │   │          │
低份额│──────────────│高份额
    │   │          │
    │   ↓          ↓
    │  现金牛     瘦狗
    └──────────────────→ 高增长
```
```

## 生命周期管理

### 产品生命周期

```markdown
## 产品生命周期

### 阶段特征
| 阶段 | 时间 | 特征 | 策略 |
|------|------|------|------|
| 开发期 | 0-6月 | 投入无收入 | 保密 |
| 导入期 | 6-12月 | 市场验证 | 推广 |
| 增长期 | 1-2年 | 快速增长 | 扩张 |
| 成熟期 | 2-5年 | 稳定增长 | 优化 |
| 衰退期 | 5年+ | 下滑 | 收缩/退出 |

### 各阶段指标
```markdown
## 生命周期指标

### 导入期
- 用户数：<10万
- 增长率：<20%
- 收入：<盈亏平衡

### 增长期
- 用户数：10-100万
- 增长率：>50%
- 收入：快速增长

### 成熟期
- 用户数：>100万
- 增长率：10-20%
- 收入：稳定盈利

### 衰退期
- 用户数：下降
- 增长率：<5%
- 收入：下降
```
```

### 生命周期策略

```markdown
## 生命周期策略

### 导入期策略
- 快速验证
- 收集反馈
- 迭代优化

### 增长期策略
- 加速增长
- 市场扩张
- 品牌建设

### 成熟期策略
- 维护份额
- 提升效率
- 成本控制

### 衰退期策略
```markdown
## 衰退期决策

### 选项
1. 维持：保持现状
2. 收缩：削减成本
3. 转型：寻找新方向
4. 退出：停止产品

### 决策因素
- 财务贡献
- 战略协同
- 用户价值
- 退出成本
```
```

## 产品决策

### 新产品决策

```markdown
## 新产品决策

### 评估框架
| 维度 | 问题 | 权重 |
|------|------|------|
| 市场吸引力 | 市场规模、增长、竞争 | 30% |
| 能力匹配 | 技术、资源、经验 | 25% |
| 战略协同 | 与现有产品互补 | 20% |
| 财务回报 | LTV、CAC、ROI | 25% |

### Go/No-Go决策
```markdown
## 决策标准

### Go条件
- [ ] 市场足够大
- [ ] 我们能赢
- [ ] 财务可接受
- [ ] 战略有意义

### No-Go条件
- [ ] 市场太小
- [ ] 竞争太激烈
- [ ] 能力不匹配
- [ ] 财务不划算
```
```

### 产品淘汰

```markdown
## 产品淘汰

### 淘汰标准
| 标准 | 阈值 | 说明 |
|------|------|------|
| 市场份额 | <5%且持续下降 | 边缘化 |
| 利润率 | 持续亏损 | 财务黑洞 |
| 用户数 | <1万且下降 | 失血 |
| 战略价值 | 无协同 | 分散资源 |

### 淘汰流程
```markdown
## 淘汰流程

1. 识别：发现衰退信号
2. 评估：分析是否值得救
3. 决策：继续/收缩/退出
4. 通知：用户沟通
5. 执行：平稳过渡
6. 复盘：经验总结
```
```

## 检查清单

- [ ] 产品组合清晰
- [ ] 生命周期管理到位

- [ ] 淘汰机制健全

## 红牌警告

- 产品线混乱
- 现金牛被忽视
- 问题产品太多
- 淘汰决策拖延
- 资源平均分配

## 好/坏对比

**好**：
```
产品组合：
- 结构清晰
- 生命周期分明
- 决策有标准
- 资源有重点
→ 健康发展、持续盈利
```

**坏**：
```
产品组合：
- 产品线混乱
- 瘦狗太多
- 明星缺乏
- 决策拍脑袋
→ 资源浪费、增长乏力
```
