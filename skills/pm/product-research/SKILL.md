# Product Research

产品研究方法论。

## 何时使用

- 用户研究
- 市场研究
- 竞品研究
- 需求调研
- 研究报告

## 研究类型

### 研究分类

```markdown
## 研究类型

### 定性研究
| 方法 | 特点 | 适用 |
|------|------|------|
| 用户访谈 | 深度探索 | 需求挖掘 |
| 焦点小组 | 群体讨论 | 创意激发 |
| 观察法 | 行为观察 | 使用场景 |
| 民族志 | 沉浸观察 | 深层洞察 |

### 定量研究
| 方法 | 特点 | 适用 |
|------|------|------|
| 问卷调查 | 大样本 | 验证假设 |
| 数据分析 | 现有数据 | 行为分析 |
| A/B测试 | 实验 | 方案验证 |
| 可用性测试 | 任务测试 | 体验评估 |

### 混合研究
```markdown
## 混合研究

### 研究流程
1. 定性探索：发现洞察
2. 定量验证：验证假设
3. 迭代深入：持续优化

### 时机选择
- 新产品：定性为主
- 迭代改进：定量为主
- 重大决策：混合研究
```
```

## 用户研究

### 用户访谈

```markdown
## 用户访谈

### 访谈设计
| 环节 | 内容 | 时间 |
|------|------|------|
| 开场 | 建立信任 | 5min |
| 基本信息 | 背景了解 | 10min |
| 核心问题 | 深入探索 | 30min |
| 验证问题 | 假设验证 | 10min |
| 收尾 | 感谢/补充 | 5min |

### 访谈技巧
```markdown
## 访谈技巧

### 提问技巧
- 开放式提问：怎么了/为什么
- 追问：能详细说说吗
- 假设提问：如果...会怎样
- 排序提问：哪个更重要

### 避免问题
- 引导性提问
- 是非题
- 过于开放
```
```

### 问卷调查

```markdown
## 问卷调查

### 问卷设计
| 环节 | 内容 | 原则 |
|------|------|------|
| 筛选 | 基本条件 | 必答 |
| 基本 | 人口统计 | 精简 |
| 核心 | 主要问题 | 聚焦 |
| 开放式 | 补充建议 | 可选 |

### 问题类型
| 类型 | 题目 | 分析 |
|------|------|------|
| 选择题 | 单选/多选 | 频次 |
| 量表题 | 李克特5/7级 | 均值 |
| 排序题 | 优先级 | 排序 |
| 开放式 | 自由回答 | 归类 |

### 样本设计
| 要点 | 说明 | 要求 |
|------|------|------|
| 样本量 | 统计显著 | >100 |
| 抽样方式 | 随机/分层 | 代表性 |
| 用户配额 | 性别/年龄 | 匹配用户 |
```

## 研究报告

### 报告结构

```markdown
## 研究报告

### 标准结构
```markdown
## 研究报告

1. 执行摘要
   - 研究背景
   - 核心发现
   - 建议行动

2. 研究方法
   - 研究类型
   - 样本信息
   - 数据来源

3. 核心发现
   - 发现1
   - 发现2
   - 发现3

4. 洞察与建议
   - 战略建议
   - 执行建议

5. 附录
   - 原始数据
   - 问卷内容
```

### 呈现技巧
- 结论先行
- 数据可视化
- 引用原话
- 清晰逻辑
```

### 研究质量

```markdown
## 研究质量

### 质量标准
| 标准 | 说明 | 检查 |
|------|------|------|
| 有效性 | 测量准确 | 信效度 |
| 可靠性 | 结果一致 | 多次验证 |
| 代表性 | 样本合理 | 配额检验 |
| 客观性 | 不带偏见 | 方法论 |

### 常见偏差
| 偏差 | 表现 | 避免 |
|------|------|------|
| 选择偏差 | 样本偏差 | 随机抽样 |
| 确认偏差 | 只看支持 | 否定假设 |
| 幸存者偏差 | 只看成功 | 全量分析 |
| 权威偏差 | 迷信权威 | 多源验证 |
```

## 检查清单

- [ ] 研究目的明确
- [ ] 方法选择合适
- [ ] 样本设计合理
- [ ] 数据收集规范
- [ ] 分析方法正确
- [ ] 报告呈现清晰
- [ ] 建议可执行

## 红牌警告

- 研究目的不明确
- 样本无代表性
- 数据造假
- 主观臆断
- 报告空洞
- 建议不可行
- 不做持续研究
