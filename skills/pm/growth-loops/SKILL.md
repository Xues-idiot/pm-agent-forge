# Growth Loops

增长飞轮分析，用Elena Verna的框架识别增长机制。

## 何时使用

- 分析增长策略
- 设计增长机制
- 评估产品增长潜力
- 优化现有增长飞轮
- 从线性获客转型到复合增长

## 核心哲学

> "增长不是漏斗，是会复合的飞轮。" — Elena Verna

**线性获客**：
```
花钱 → 获取用户 → 希望留存 → 烧钱循环
（贵，不复合）
```

**增长飞轮**：
```
用户创造价值 → 价值吸引新用户 → 新用户创造更多价值
（复合，良性循环）
```

---

## 四种增长飞轮

| 飞轮类型 | 机制 | 例子 |
|----------|------|------|
| **Viral** | 用户邀请用户 | Dropbox推荐、Slack团队邀请 |
| **Content** | 用户生成内容吸引流量 | 抖音、B站、知乎 |
| **Network Effect** | 用户越多价值越大 | 微信、Airbnb |
| **Paid** | 收入资助获客 | SaaS订阅模式 |

---

## Viral Loop (病毒式)

### 机制
用户使用产品 → 分享给朋友 → 朋友成为用户 → 循环

### 结构
```
输入 → 行动 → 输出 → 新输入
用户注册 → 分享邀请 → 朋友注册 → 双方获得奖励
```

### 成功要素
- 分享动机强（奖励有价值）
- 门槛低（点击即用）
- 产品真的好（用户愿意分享）

### 评估问题
- 用户会主动分享吗？
- 分享的奖励够吸引人吗？
- 邀请体验流畅吗？

---

## Content Loop (内容飞轮)

### 机制
用户创造内容 → 内容吸引流量 → 更多用户 → 创造更多内容

### 结构
```
创作者发布 → 内容被搜索/推荐 → 新用户看到 → 注册成为创作者
```

### 成功要素
- 内容能被搜索引擎收录
- 有推荐算法放大好内容
- 创作者有成就感/收益

### 评估问题
- 用户创造的内容有价值吗？
- 内容能吸引目标用户吗？
- 创作者为什么要创造内容？

---

## Network Effect (网络效应)

### 机制
更多用户加入 → 产品价值增加 → 吸引更多用户

### 类型
| 类型 | 例子 | 价值来源 |
|------|------|----------|
| 直接网络效应 | 微信 | 用户直接与其他用户互动 |
| 间接网络效应 | Airbnb | 房东多→房客多→房东更多 |
| 双边网络效应 | 淘宝 | 买家卖家互相吸引 |

### 成功要素
- 达到最低网络规模
- 核心用户互动密集
- 跨边价值传导

### 评估问题
- 产品有什么网络效应？
- 需要多少用户才能形成效应？
- 用户怎么从网络中获取价值？

---

## Paid Loop (付费飞轮)

### 机制
收入 → 投入获客 → 用户增长 → 更多收入

### 有效条件
- LTV > CAC（有正向单元经济）
- CAC < 收入的可持续比例

### 结构
```
收入 → 营销投入 → 新用户 → 收入增加
```

### 评估问题
- 单位经济是否健康？
- CAC能在规模下降低吗？
- LTV能提升吗？

## 红牌警告

- 只有付费获客，没有飞轮
- 有飞轮但速度太慢
- 飞轮依赖单一渠道
- 忽视单元经济

## 好/坏对比

**好**：
```
分析产品发现：
- 用户喜欢分享设计
- Figma分享 → 设计师看到 → 注册
→ 设计Content Loop
→ 投入内容SEO和创作者激励
```

**坏**：
```
只看获客渠道：
- 搜索引擎广告
- 社交媒体广告
→ 没有飞轮
→ 停止投钱就没有增长
```
