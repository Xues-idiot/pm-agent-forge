# Online Course Design Skill

设计在线课程，规划电子学习内容，管理虚拟课堂运营。

## 何时使用

- 在线课程设计
- 视频课程规划
- 直播教学管理
- 课件制作
- 在线学习平台
- 混合式学习

## 在线课程流程

```
1. 课程设计 → 2. 内容制作 → 3. 平台搭建 → 4. 教学运营 → 5. 效果评估
```

### 1. 课程设计

**课程体系：**

## 课程体系

### 体系规划
```markdown
## 课程体系

### 层级结构
- 入门课程：基础知识，小白用户
- 初级课程：核心技能，入门用户
- 中级课程：实战应用，有经验用户
- 高级课程：深度专长，专业用户

### 课程类型
| 类型 | 时长 | 形式 | 特点 |
|------|------|------|------|
| 微课 | 5-15分钟 | 视频 |碎片化学习 |
| 录播课 | 30-60分钟 | 视频+练习 | 自定进度 |
| 直播课 | 60-120分钟 | 实时互动 | 高互动 |
| 系列课 | 5-20节 | 混合 | 系统学习 |
| 实战营 | 7-30天 | 作业+点评 | 强实践 |
```

### 2. 内容制作

**视频制作：**

```markdown
## 视频制作

### 视频类型
| 类型 | 制作方式 | 适用场景 | 成本 |
|------|----------|----------|------|
| 录屏讲解 | 屏幕录制+讲解 | 软件操作 | 低 |
| 真人出镜 | 拍摄+后期 | 强调IP | 中 |
| 动画演示 | MG动画 | 抽象概念 | 高 |
| 情景剧 | 拍摄制作 | 场景模拟 | 高 |
| 混合式 | 真人+动画 | 综合 | 中 |

### 制作规范
```markdown
## 视频规范

### 技术参数
| 参数 | 要求 |
|------|------|
| 分辨率 | 1920×1080 (最低1280×720) |
| 帧率 | 30fps (运动内容60fps) |
| 格式 | MP4 (H.264编码) |
| 码率 | 3-8Mbps (1080P) |
| 音频 | 44100Hz, 128kbps, 双声道 |

### 教学设计
- [ ] 3分钟法则：每3分钟有互动
- [ ] 案例贯穿：1个核心案例
- [ ] 练习穿插：每10分钟1个小练习
```
```

### 5. 效果评估

**学习效果评估：**

```markdown
## 效果评估

### 评估模型（Kirkpatrick）
| 级别 | 内容 | 评估方式 |
|------|------|----------|
| 1-反应 | 学员满意度 | 课后调研 |
| 2-学习 | 知识技能掌握 | 考试/作业 |
| 3-行为 | 知识应用 | 跟踪观察 |
| 4-结果 | 业务影响 | 绩效数据 |

### 核心指标
| 指标 | 定义 | 目标值 |
|------|------|--------|
| 完课率 | 完成课程人数/报名人数 | >60% |
| 通过率 | 通过考试/参加考试 | >80% |
| 学习时长 | 人均学习时长 | 达到课程时长80% |
| 作业提交率 | 提交作业/布置作业 | >70% |
| NPS | 推荐指数 | >50 |
```

**课程迭代：**

```markdown
## 课程迭代

### 数据分析
| 数据 | 分析内容 | 改进方向 |
|------|----------|----------|
| 完课率低 | 哪个章节流失 | 内容/节奏优化 |
| 作业正确率低 | 哪个知识点 | 加强讲解 |
| 负面评价 | 哪些方面 | 改进相应模块 |
| 学习时长短 | 视频哪些时段退出 | 内容优化 |

### 迭代优先级
| 优先级 | 问题类型 | 迭代方式 |
|--------|----------|----------|
| P0 | 严重错误 | 立即修复 |
| P1 | 体验问题 | 下一期优化 |
| P2 | 细节问题 | 持续优化 |
```

## 红牌警告

- 课程内容过期
- 视频质量差
- 平台体验差
- 无人运营服务
- 虚假学习数据
- 学员隐私泄露
- 侵权内容
- 虚假宣传
