# Student Retention Skill

分析学员留存，降低流失率，提升学习参与度。

## 何时使用

- 学员流失分析
- 留存率提升
- 学员预警
- 流失干预
- 学习粘性设计

## 留存管理流程

```
1. 数据监控 → 2. 流失预警 → 3. 原因分析 → 4. 干预措施 → 5. 效果评估
```

### 1. 数据监控

**留存指标体系：**

```markdown
## 留存指标

### 核心指标
| 指标 | 定义 | 计算 | 健康值 |
|------|------|------|--------|
| 完课率 | 完成课程/报名 | 完成数/报名数 | >60% |
| 作业提交率 | 提交作业/布置作业 | 提交数/布置数 | >80% |
| 活跃天数 | 30天内活跃天数 | SUM(活跃天数) | >15天 |
| 回归率 | 流失后回归/流失用户 | 回归数/流失数 | >20% |

### 留存漏斗
```markdown
## 留存漏斗

报名 → 首次学习 → 持续学习 → 完成课程 → 推荐他人
 100%      70%         50%         30%         10%
```

### 行为指标
| 指标 | 定义 | 预警值 |
|------|------|--------|
| 登录间隔 | 最近登录-上次登录 | >7天 |
| 学习时长 | 每次学习时长 | <10分钟 |
| 视频完成率 | 已看视频/全部视频 | <30% |
| 互动率 | 参与互动/机会 | <20% |
```

### 3. 原因分析

**流失原因分析：**

```markdown
## 流失原因分析

### 常见原因分类
| 原因类别 | 占比 | 典型表现 |
|----------|------|----------|
| 时间不足 | 30% | 工作忙/没时间 |
| 难度不适 | 25% | 太难/太简单 |
| 内容不满意 | 20% | 不实用/不感兴趣 |
| 服务问题 | 15% | 反馈慢/答疑差 |
| 价格因素 | 10% | 觉得不值 |

### 调研模板
```markdown
## 流失调研

### 流失前兆
- [ ] 学习进度停滞
- [ ] 登录间隔变长
- [ ] 互动减少
- [ ] 负面反馈

### 流失时机
- [ ] 刚开始学习（课程不适应）
- [ ] 中期（难度增加）
- [ ] 后期（缺乏动力）

### 流失原因访谈
1. 什么原因让你决定不再继续？
2. 如果再给你一次机会，什么会让你留下？
3. 你会推荐这门课给朋友吗？
```
```

### 4. 干预措施

**分层干预策略：**

```markdown
## 干预策略

### 早期干预（未流失）
| 措施 | 触发条件 | 形式 | 效果目标 |
|------|----------|------|----------|
| 邮件提醒 | 48小时未登录 | 自动化邮件 | 回归率+15% |
| 学习计划 | 新学员 | 个性化计划 | 完课率+20% |
| 社群激活 | 沉默用户 | 社群活动 | 活跃+30% |

### 中期干预（轻度流失风险）
| 措施 | 触发条件 | 形式 | 效果目标 |
|------|----------|------|----------|
| 1对1沟通 | 7天未登录 | 班主任电访 | 回归率+25% |
| 激励活动 | 进度<30% | 打卡激励 | 学习时长+50% |
| 学习伙伴 | 自学用户 | 配对学习 | 完成率+15% |

### 深度干预（高流失风险）
| 措施 | 触发条件 | 形式 | 效果目标 |
|------|----------|------|----------|
| 深度访谈 | 14天未登录 | 专人沟通 | 了解根因 |
| 课程调整 | 难度问题 | 降级/换课 | 继续学习 |
| 退款安抚 | 要求退款 | 部分退款/延期 | 保留口碑 |
```

## 留存管理检查清单

- [ ] 监控指标全面

- [ ] 干预策略分层

## 红牌警告

- 流失预警失效
- 干预时机过晚
- 干预方式扰民
- 原因分析错误
- 激励机制无效
- 成本高于收益
- 负面口碑扩散
- 学员数据泄露
