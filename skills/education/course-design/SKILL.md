# Education Course Design Skill

设计课程和教学大纲，构建系统性的学习路径。

## 何时使用

- 设计新的课程体系
- 规划学习路径
- 制定教学大纲
- 课程迭代优化
- 创建培训计划

## 课程设计流程

```
1. 需求分析 → 2. 目标设定 → 3. 内容规划 → 4. 结构设计 → 5. 资源准备 → 6. 评估设计
```

### 1. 需求分析

**学习者分析：**

```markdown
## 学习者画像

### 基本信息
- 目标人群：
- 年龄范围：
- 背景知识：
- 学习目的：

### 学习特征
- 学习习惯：
- 偏好方式：
- 时间投入：
- 约束条件：
```

**需求类型：**

| 需求类型 | 描述 | 设计重点 |
|---------|------|---------|
| 技能培训 | 掌握特定技能 | 实践为主 |
| 知识传授 | 构建知识体系 | 系统性 |
| 认证考试 | 通过资格考试 | 考点覆盖 |
| 在职提升 | 提升工作能力 | 场景化 |

### 2. 目标设定

**学习目标分类（Bloom's Taxonomy）：**

| 层级 | 认知层次 | 行为动词 |
|------|---------|---------|
| L1 | 记忆 | 识别、回忆 |
| L2 | 理解 | 解释、总结 |
| L3 | 应用 | 使用、计算 |
| L4 | 分析 | 分析、比较 |
| L5 | 评价 | 评价、判断 |
| L6 | 创造 | 设计、构建 |

**目标书写：**

```markdown
## 学习目标

### 课程目标
完成本课程后，学习者能够：

1. [L3] 使用 [工具/方法] 解决 [问题类型]
2. [L4] 分析 [场景] 中的 [要素]
3. [L5] 评价 [内容] 的 [维度]
```

### 3. 内容规划

**内容模块划分：**

```markdown
## 内容模块

### 模块1：[模块名称]
- 知识点1
- 知识点2
- 知识点3
- 预计时长：X小时

### 模块2：[模块名称]
- 知识点1
- 知识点2
- 预计时长：X小时
```

**知识点分解：**

```markdown
## 知识点清单

| 知识点 | 类型 | 难度 | 优先级 | 依赖 |
|--------|------|------|--------|------|
| 概念A | 理论 | L2 | P0 | - |
| 技能B | 实践 | L3 | P0 | 概念A |
| 方法C | 实践 | L4 | P1 | 技能B |
```

### 4. 结构设计

**学习路径设计：**

```markdown
## 学习路径

```
模块1 → 模块2 → 模块3 → 模块4
  ↓        ↓       ↓       ↓
 基础    进阶    实践    综合
```

### 学习顺序
1. 先理论后实践
2. 先简单后复杂
3. 先局部后整体
```

**课程结构模板：**

```markdown
## 课程结构

### 第1章：入门（约2小时）
#### 1.1 介绍（30min）
- 学习目标
- 课程概览
- 前置要求

#### 1.2 基础概念（90min）
- 概念1
- 概念2
- 练习1
```

### 5. 资源准备

**教学资源清单：**

| 资源类型 | 内容 | 形式 |
|---------|------|------|
| 视频 | 讲解演示 | 录屏/真人 |
| 文档 | 知识内容 | Markdown/PDF |
| 练习 | 巩固知识 | 编程/问答 |
| 项目 | 综合实践 | 案例/作业 |
| 测试 | 效果评估 | 选择/实操 |

### 6. 评估设计

**评估方式：**

| 评估类型 | 目的 | 方式 |
|---------|------|------|
| 形成性评估 | 学习过程中检验 | 随堂测验 |
| 诊断性评估 | 了解起点状态 | 前测 |
| 总结性评估 | 检验学习成果 | 考试/项目 |

**评估标准：**

```markdown
## 评估标准

### 过程评估（40%）
- 参与度：10%
- 练习完成：15%
- 小组讨论：15%

### 结果评估（60%）
- 单元测验：20%
- 综合项目：25%
- 期末考试：15%

### 及格标准
总分 ≥ 60% 且期末考试 ≥ 50%
```

## 课程设计检查清单

- [ ] 学习者分析完整
- [ ] 学习目标具体可衡量
- [ ] 内容覆盖目标
- [ ] 学习路径合理
- [ ] 时间分配恰当
- [ ] 评估方式匹配目标
- [ ] 资源准备充分

## 红牌警告

- 目标过于笼统不可衡量
- 内容与目标不匹配
- 跳过基础直接进阶
- 评估方式与目标脱节
- 忽略学习者差异
- 时间规划不现实
- 缺少实践环节
