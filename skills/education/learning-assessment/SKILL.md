# Education Learning Assessment Skill

设计评估题目和测验，衡量学习效果。

## 何时使用

- 设计课程测验
- 创建考核题目
- 评估学习效果
- 出题组卷
- 评估学习者能力

## 评估设计流程

```
1. 明确评估目的 → 2. 确定评估类型 → 3. 设计题目 → 4. 组卷审核 → 5. 评分标准 → 6. 结果分析
```

### 1. 明确评估目的

**评估目的分类：**

| 目的 | 描述 | 题目特点 |
|------|------|---------|
| 诊断 | 了解起点状态 | 覆盖面广 |
| 形成 | 检验学习过程 | 针对性强 |
| 总结 | 评价学习成果 | 综合应用 |
| 认证 | 判断是否达标 | 标准明确 |

### 2. 确定评估类型

**题目类型对比：**

| 类型 | 优点 | 缺点 | 适用场景 |
|------|------|------|---------|
| 选择题 | 客观、自动批改 | 难以考查深度 | 基础知识 |
| 判断题 | 简洁、快速 | 猜测概率高 | 概念辨析 |
| 填空题 | 考察记忆 | 答案不唯一 | 关键词 |
| 简答题 | 表达思考过程 | 批改主观 | 理解应用 |
| 论述题 | 深度分析 | 批改主观 | 综合应用 |
| 实践题 | 真实能力 | 评估成本高 | 技能考核 |

### 3. 设计题目

**选择题设计：**

```markdown
## 选择题模板

### 单选题
题目：[情境/问题]
A. [选项1]
B. [选项2]
C. [选项3]
D. [选项4]

正确答案：B
难度：L2
知识点：

### 多选题
题目：[情境/问题]
A. [选项1]
B. [选项2]
C. [选项3]
D. [选项4]

正确答案：AB
难度：L3
知识点：
```

**选择题设计原则：**
- 选项长度相近
- 正确答案位置随机
- 无重复选项
- 避免绝对词汇（"总是"、"从不"）

**简答题设计：**

```markdown
## 简答题模板

### 题目
[问题描述]

### 答题要点
- [要点1]
- [要点2]
- [要点3]

### 参考答案
[参考答案]

### 评分标准
| 等级 | 分值 | 描述 |
|------|------|------|
| 优秀 | 90-100% | 要点完整、表达清晰 |
| 良好 | 75-89% | 要点基本完整 |
| 合格 | 60-74% | 关键要点正确 |
| 不合格 | <60% | 要点缺失或错误 |

### 难度：L4
### 预计答题时间：X分钟
```

**编程题设计：**

```markdown
## 编程题模板

### 题目
[问题描述]

### 输入说明
[输入格式]

### 输出说明
[输出格式]

### 示例
输入：
```
[输入示例]
```

输出：
```
[输出示例]
```

### 评测点
| 评测点 | 分值 |
|--------|------|
| 语法正确 | 20% |
| 通过基本测试 | 30% |
| 通过边界测试 | 30% |
| 代码规范 | 20% |

### 难度：L4-L5
### 预计完成时间：X分钟
```

### 4. 组卷审核

**试卷结构：**

```markdown
## 试卷结构

### 基本信息
- 课程：
- 总分：100分
- 时间：90分钟

### 题目分布
| 题型 | 题量 | 每题分值 | 总分 |
|------|------|---------|------|
| 单选题 | 20 | 2 | 40 |
| 多选题 | 5 | 4 | 20 |
| 简答题 | 3 | 8 | 24 |
| 实践题 | 1 | 16 | 16 |

### 知识点分布
| 知识点 | 题目数 | 分值 |
|--------|--------|------|
| 模块1 | 10 | 40 |
| 模块2 | 8 | 32 |
| 模块3 | 7 | 28 |
```

**组卷检查：**
- [ ] 覆盖所有重点知识点
- [ ] 难度分布合理
- [ ] 题目不重复
- [ ] 无错误选项
- [ ] 答案正确

### 5. 评分标准

**评分方式：**

| 方式 | 适用 | 优缺点 |
|------|------|--------|
| 客观评分 | 选择/判断 | 客观但浅层 |
| 人工评分 | 简答/论述 | 深度但主观 |
| 自动评分 | 编程/填空 | 准确但限制多 |
| 混合评分 | 综合 | 取长补短 |

### 6. 结果分析

**试卷分析：**

```markdown
## 试卷分析

### 基本统计
- 参加人数：
- 平均分：
- 最高分：
- 最低分：
- 及格率：

### 题目分析
| 题号 | 正确率 | 难度 | 区分度 |
|------|--------|------|--------|
| 1 | 85% | 0.85 | 0.42 |
| 2 | 72% | 0.72 | 0.38 |

### 知识点分析
| 知识点 | 掌握率 | 建议 |
|--------|--------|------|
| 概念A | 90% | 已掌握 |
| 概念B | 65% | 需要加强 |
```

**区分度标准：**
| 区分度 | 含义 | 处理建议 |
|--------|------|---------|
| ≥ 0.4 | 优秀 | 保留 |
| 0.3-0.39 | 良好 | 保留 |
| 0.2-0.29 | 一般 | 改进 |
| < 0.2 | 差 | 替换 |

## 评估设计检查清单

- [ ] 评估目的明确
- [ ] 题目类型选择合适
- [ ] 覆盖知识点完整
- [ ] 难度分布合理
- [ ] 无错误题目
- [ ] 答案正确
- [ ] 评分标准明确

## 红牌警告

- 题目有明显错误
- 答案不唯一但无标准
- 题目与知识点不匹配
- 难度过高或过低
- 题目泄露答案
- 评分标准不明确
- 评估方式与目标不符
