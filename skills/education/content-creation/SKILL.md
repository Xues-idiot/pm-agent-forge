# Education Content Creation Skill

创建教育内容，包括课程材料、练习题、解说词等。

## 何时使用

- 编写教学材料
- 创建练习题
- 撰写知识讲解
- 设计学习卡片
- 制作教学视频脚本

## 内容创作流程

```
1. 确定目标 → 2. 收集素材 → 3. 内容编写 → 4. 审核优化 → 5. 格式整理 → 6. 发布准备
```

### 1. 确定目标

**内容规划：**

```markdown
## 内容规划

### 基本信息
- 内容类型：[视频/文档/练习/交互]
- 目标受众：
- 所属课程：

### 学习目标
完成本内容后，学习者能够：
1.
2.

### 内容规格
- 预计时长/篇幅：
- 难度等级：L1-L6
- 知识点数量：
```

### 2. 收集素材

**素材来源：**

| 类型 | 来源 | 备注 |
|------|------|------|
| 官方文档 | 产品文档、API文档 | 权威准确 |
| 社区内容 | 博客、教程 | 需核实 |
| 实践经验 | 工作总结、案例 | 真实可信 |
| 外部资源 | 书籍、课程 | 需授权 |

**素材整理：**

```markdown
## 素材清单

### 核心素材
1. [素材1] - 来源：[]
2. [素材2] - 来源：[]

### 辅助素材
1. [素材1] - 来源：[]
2. [素材2] - 来源：[]

### 待核实
1. [待核实项]
```

### 3. 内容编写

**知识讲解模板：**

```markdown
## 知识讲解编写

### 标题：[主题]

### 开头（吸引注意）
[用一个故事/问题/数据吸引学习者]

### 核心内容
#### 要点1
[概念/原理说明]

**示例：**
```[代码/案例]
[示例内容]
```

**图示：**
[图示说明]

#### 要点2
...

### 总结
[1-2句话总结核心要点]

### 思考题
[1-2个引导思考的问题]
```

### 4. 审核优化

**内容审核清单：**

```markdown
## 内容审核

### 准确性
- [ ] 知识点正确无误
- [ ] 示例代码可运行
- [ ] 数据/事实有来源

### 完整性
- [ ] 覆盖所有要点
- [ ] 解释清晰易懂
- [ ] 有实例支撑

### 可读性
- [ ] 语言流畅
- [ ] 结构清晰
- [ ] 无错别字

### 教学性
- [ ] 有学习目标
- [ ] 有练习巩固
- [ ] 难度适中
```

### 5. 格式整理

**格式规范：**

```markdown
## 格式规范

### Markdown 格式
- 标题层级：# ## ### ####
- 代码块：```语言
- 强调：**粗体**、*斜体*
- 列表：- 或 1.

### 多媒体规范
- 图片：PNG/JPG，清晰度 ≥ 300dpi
- 视频：MP4，分辨率 ≥ 1080P
- 音频：MP3，采样率 ≥ 44.1kHz
```

### 6. 发布准备

**发布检查：**

```markdown
## 发布准备

### 内容检查
- [ ] 内容审核通过
- [ ] 格式符合规范
- [ ] 元信息完整

### 资源准备
- [ ] 图片已上传
- [ ] 视频已转码
- [ ] 附件已准备

### 平台适配
- [ ] 平台格式兼容
- [ ] 播放/显示正常
```

## 内容创作检查清单

- [ ] 学习目标明确
- [ ] 内容准确无误
- [ ] 结构清晰合理
- [ ] 语言流畅易懂
- [ ] 例子生动贴切
- [ ] 练习有针对性
- [ ] 格式规范统一

## 红牌警告

- 内容存在错误
- 知识点与目标不符
- 语言晦涩难懂
- 缺少实例支撑
- 练习与内容脱节
- 版权素材未授权
- 格式不统一不规范
