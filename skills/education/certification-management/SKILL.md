# Certification Management Skill

管理认证项目，颁发证书，处理证书验证和合规。

## 何时使用

- 证书体系设计
- 认证项目管理
- 证书颁发管理
- 证书验证
- 合规管理
- 证书续期

## 证书管理流程

```
1. 体系设计 → 2. 项目创建 → 3. 考核评估 → 4. 证书颁发 → 5. 验证管理
```

### 1. 体系设计

**证书类型分类：**

```markdown
## 证书类型

### 按颁发主体
| 类型 | 说明 | 权威性 | 适用场景 |
|------|------|--------|----------|
| 政府类证书 | 政府机构颁发 | 最高 | 职业资格/学历 |
| 行业类证书 | 行业协会颁发 | 高 | 行业专业认证 |
| 企业类证书 | 企业内部颁发 | 中 | 内部认证/培训 |
| 平台类证书 | 平台机构颁发 | 中低 | 在线学习认证 |

### 按用途分类
| 类型 | 示例 | 证明内容 |
|------|------|----------|
| 职业资格 | 教师资格证/CPA | 具备从业资格 |
| 技能认证 | PMP/阿里云认证 | 具备特定技能 |
| 培训结业 | 内部培训结业 | 完成培训 |
| 学历学位 | 大专/本科/硕士 | 教育程度 |
```

### 4. 证书颁发

**证书制作：**

```markdown
## 证书制作

### 证书内容
```markdown
## 证书模板

### 正面
[机构Logo]
[证书名称]

兹证明：
[姓名]
[身份证号后四位：****]
于[日期]完成[项目名称]培训
考核成绩合格，特发此证。

证书编号：
有效期至：

[颁发机构签章]      [日期]
```

### 证书要素
| 要素 | 要求 |
|------|------|
| 证书编号 | 唯一性，可查验 |
| 姓名 | 与证件一致 |
| 项目名称 | 准确完整 |
| 颁发日期 | 实际颁发日 |
| 有效期 | 按规定 |
| 签章 | 公章/专用章 |
| 二维码 | 防伪+验证 |
```

### 5. 验证管理

**验证系统：**

```markdown
## 证书验证

### 验证方式
| 方式 | 适用场景 | 实现方式 |
|------|----------|----------|
| 官网查询 | 公开验证 | 数据库查询 |
| 二维码验证 | 快速验证 | 扫码跳转 |
| API验证 | 企业对接 | 接口调用 |
| 人工验证 | 特殊情况 | 电话/邮件 |

### 验证信息
```markdown
## 验证返回信息

### 查询成功
- 证书状态：有效/已撤销/已过期
- 持有人信息：姓名（如公开）
- 项目名称：[项目名称]
- 颁发日期：YYYY-MM-DD
- 有效期至：YYYY-MM-DD

### 查询失败
- 未查询到该证书
- 证书已被撤销
- 证书已过期
```
```

## 红牌警告

- 证书颁发不合规
- 考核舞弊
- 证书伪造
- 虚假宣传认证
- 资质过期继续颁发
- 信息泄露
- 证书撤销不公示
- 违规收费
