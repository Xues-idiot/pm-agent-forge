# 考试设计 (Exam Design)

## Overview

考试设计是科学设计各类测评考试，确保考试信度和效度的技能。涵盖题目设计、难度控制、试卷生成、评分标准等核心环节。

## 何时使用

- 需要设计正式测评
- 考试结果影响重大
- 需要确保公平公正
- 在线考试系统建设

## 流程

1. **需求分析**：明确考试目的和对象
2. **大纲制定**：确定考核知识点
3. **题目设计**：编写各类题型
4. **难度校准**：校准题目难度
5. **试卷生成**：智能组卷
6. **评分标准**：制定评分细则
7. **质量分析**：分析信度效度

## 模板

### 考试设计规划模板
```
## 考试概览
- 考试目的: [选拔/诊断/水平测试]
- 考试对象: [人群特征]
- 考试时长: [X分钟]
- 总分/及格线: [X分/X分]

## 题目类型分布
| 题型 | 题量 | 分值 | 考核目标 |
|------|------|------|----------|
| 单选题 | 20 | 2分 | 基础知识 |
| 多选题 | 10 | 2分 | 综合理解 |
| 简答题 | 5 | 10分 | 深度分析 |
| 实践题 | 2 | 15分 | 应用能力 |

## 难度分布
- 容易题: [X%] - 区分度<0.3
- 中等题: [X%] - 区分度0.3-0.5
- 困难题: [X%] - 区分度>0.5

## 质量指标
- 信度: >[X] (Cronbach's α)
- 效度: >[X]
- 难度指数: 0.3-0.7
- 区分度: >[X]
```

## 检查清单

- [ ] 考核目标与题目匹配
- [ ] 题目难度分布合理
- [ ] 评分标准明确可操作
- [ ] 有防作弊措施
- [ ] 考试信度效度达标
- [ ] 题目有版权授权

## 红牌警告

- 题目泄漏导致泄密
- 评分标准不一致
- 题目难度不均衡
- 区分度不足无法选拔
- 侵犯题目版权
