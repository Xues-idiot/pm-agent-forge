# 自适应学习 (Adaptive Learning)

## Overview

自适应学习是通过AI技术根据学习者能力动态调整学习内容和路径的技能。涵盖学情诊断、内容适配、路径规划、效果评估等核心环节。

## 何时使用

- 学生个体差异大
- 需要个性化学习路径
- 传统教学效率低
- 在线教育平台优化

## 流程

1. **学情诊断**：评估学生初始能力
2. **目标设定**：确定学习目标
3. **内容适配**：推荐适合难度的内容
4. **路径规划**：规划最优学习路径
5. **学习反馈**：实时反馈学习效果
6. **动态调整**：根据表现调整难度
7. **效果评估**：评估学习效果

## 模板

### 自适应学习规划模板
```
## 学习者画像
- 初始水平: [测评结果]
- 学习风格: [视觉/听觉/动手]
- 学习偏好: [时间/节奏]
- 知识薄弱点: [列表]

## 知识图谱
- 知识点数量: [X个]
- 依赖关系: [前置/后置]
- 难度分级: [L1-L5]

## 自适应策略
| 维度 | 策略 |
|------|------|
| 内容难度 | 答对X题提升一级 |
| 推送频率 | 根据遗忘曲线推送 |
| 学习路径 | 最短路径优先 |

## 核心指标
- 学习效率提升: >[X]%
- 知识点掌握率: >[X]%
- 学习时长节省: >[X]%
```

## 检查清单

- [ ] 知识图谱覆盖全面
- [ ] 学情诊断准确
- [ ] 内容难度分级合理
- [ ] 学习路径优化有效
- [ ] 反馈及时有效
- [ ] 效果评估科学

## 红牌警告

- 知识图谱不完整
- 学情诊断不准确
- 难度跳跃过大
- 过度依赖系统忽视教师作用
- 学生产生学习依赖
