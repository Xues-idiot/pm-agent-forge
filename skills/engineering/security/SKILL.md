## 安全监控

### 日志审计

```markdown
## 安全日志

### 日志内容
| 事件 | 内容 | 保留 |
|------|------|------|
| 登录 | 用户/时间/IP | 1年 |
| 敏感操作 | 操作/数据 | 6月 |
| 权限变更 | 变更前后 | 1年 |
| 异常事件 | 告警/阻断 | 1年 |

### 日志规范
- 完整性：关键信息不漏
- 不可篡改：日志保护
- 时效性：及时记录
- 可查询：索引优化
```

### 告警响应

```markdown
## 安全告警

### 告警场景
| 场景 | 阈值 | 响应 |
|------|------|------|
| 暴力破解 | N次失败 | 封禁IP |
| 异常登录 | 异地/时间 | MFA验证 |
| 权限滥用 | 超权限访问 | 审计 |
| 数据导出 | 大量导出 | 审批 |
| API滥用 | QPS超限 | 限流 |

### 响应流程
```markdown
## 响应流程

1. 检测：发现异常
2. 验证：确认威胁
3. 阻断：立即止血
4. 调查：根因分析
5. 恢复：恢复正常
6. 复盘：改进防护
```
