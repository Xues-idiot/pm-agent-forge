# Deep Research

深度研究，系统性研究和分析复杂问题。

## 何时使用

- 技术选型研究
- 市场分析
- 竞品研究
- 方案评估
- 学术研究

## 研究流程

### 研究框架

```markdown
## 研究流程

### 标准流程
1. 明确问题 → 2. 文献回顾 → 3. 数据收集 → 4. 分析综合 → 5. 结论建议

### 迭代流程
```
假设 → 验证 → 发现 → 新假设 → ...
```
```

### 研究类型

```markdown
## 研究类型

### 探索性研究
- 目的：发现新见解
- 方法：访谈、案例、文献
- 输出：假设、洞察

### 验证性研究
- 目的：验证假设
- 方法：数据分析、实验
- 输出：结论、证据

### 评估性研究
- 目的：评估方案
- 方法：多维度对比
- 输出：建议、推荐
```

### SMART目标
- Specific：具体明确
- Measurable：可衡量
- Achievable：可实现
- Relevant：相关性强
- Time-bound：有时间限制

### 示例
在4周内，通过分析10家竞品，总结出3个差异化机会

| 方法 | 说明 |
|------|------|
| 重测信度 | 同一方法重复测量 |

## 红牌警告

- 问题定义模糊
- 信息来源单一
- 样本不具代表性
- 分析方法错误
- 结论过度推断
- 引用不规范
