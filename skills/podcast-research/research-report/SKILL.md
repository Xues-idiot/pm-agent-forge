# 播客研究报告生成 (Research Report Generation)

## Overview

播客研究报告生成是将播客内容整理成结构化研究报告的技能。

## 何时使用

- 研究某个话题
- 整理某个播客系列
- 生成学习笔记
- 制作分享内容

## 流程

1. **需求分析**：确定报告需求
2. **内容收集**：收集相关播客
3. **内容分析**：分析内容提取信息
4. **结构设计**：设计报告结构
5. **内容撰写**：撰写各章节
6. **审核校对**：审核内容准确性
7. **格式输出**：输出格式化报告

## 模板

### 播客研究报告模板
```markdown
# [主题] 播客研究报告

## 研究概要
- 研究主题: [主题]
- 涉及播客: [列表]
- Episode数: [X]
- 总时长: [X小时]
- 研究日期: [日期]

## 核心发现
[3-5个核心发现]

## 详细内容

### 主题1
**来源**: [播客名] EP[X]
**时间戳**: [XX:XX]
**关键内容**:
- [要点1]
- [要点2]

### 主题2
...

## 嘉宾观点

### 嘉宾A ([播客名])
**背景**: [简介]
**核心观点**:
- [观点1]
- [观点2]

## 关键引用
> "[引用内容]" - [嘉宾], [播客名], [时间]

## 行动计划
- [ ] 行动1
- [ ] 行动2

## 相关资源
- [播客链接]
- [相关阅读]
```

## 检查清单

- [ ] 可操作
