# 房地产行业分析 (Real Estate Analysis)

## Overview

房地产行业分析是评估房地产公司和REITs投资价值的技能。

## 何时使用

- 房地产股票投资
- REITs投资
- 产业链研究
- 宏观经济研究

## 流程

1. **行业分析**：分析房地产市场
2. **政策分析**：分析调控政策
3. **公司分析**：分析房企基本面
4. **土地储备**：分析土地储备
5. **财务分析**：分析财务健康
6. **估值分析**：进行估值
7. **投资建议**：给出建议

## 检查清单

- [ ] 政策解读准确
- [ ] 土地储备分析
- [ ] 财务分析
- [ ] 现金流分析
- [ ] 估值合理
- [ ] 风险揭示

## 红牌警告

- 政策收紧
- 现金流恶化
- 债务风险
- 库存积压
- 地价下跌
