# 业绩电话会议分析 (Earnings Call Analysis)

## Overview

业绩电话会议分析是分析上市公司财报电话会议的技能。提取管理层语气、战略信息、市场判断等关键内容。

## 何时使用

- 财报季跟踪
- 判断管理层信心
- 挖掘前瞻指引
- 评估执行力

## 流程

1. **会议收集**：获取电话会议记录
2. **角色识别**：识别发言人角色
3. **内容分段**：按议题分段
4. **情感分析**：分析语气态度
5. **关键提取**：提取关键信息
6. **对比分析**：与历史会议对比
7. **投资应用**：应用到投研

## 检查清单

- [ ] 记录完整

- [ ] 情感分析准确
- [ ] 信息提取完整
- [ ] 对比分析到位

## 红牌警告

- 记录遗漏
- 情感判断错误
- 管理层画饼误读
- 指引过于乐观
- 忽视负面信号
