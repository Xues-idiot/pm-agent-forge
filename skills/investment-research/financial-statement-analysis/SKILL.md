# 财务报表分析 (Financial Statement Analysis)

## Overview

财务报表分析是通过分析上市公司财务报表评估公司经营状况和投资价值的技能。涵盖利润表、资产负债表、现金流量表分析及财务指标计算。

## 何时使用

- 评估上市公司盈利能力
- 分析公司资产负债状况
- 判断公司现金流质量
- 筛选财务指标优秀的公司

## 流程

1. **数据获取**：获取三表原始数据
2. **数据清洗**：标准化报表格式
3. **指标计算**：计算盈利能力、偿债能力、运营能力指标
4. **趋势分析**：分析多年数据趋势
5. **横向对比**：与行业均值、竞争对手对比
6. **异常识别**：识别财务异常信号
7. **结论输出**：输出分析报告

## 检查清单

- [ ] 三表数据完整无缺失
- [ ] 财务指标计算正确
- [ ] 与行业数据对比合理
- [ ] 识别常见财务造假特征
- [ ] 多年趋势分析到位
- [ ] 风险信号明确标注

## 红牌警告

- 利润虚增但现金流差
- 应收账款远超营收增长
- 存货周转天数异常
- 关联交易异常频繁
- 商誉占净资产比例过高
