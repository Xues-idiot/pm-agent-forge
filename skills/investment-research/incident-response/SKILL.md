# 故障应急响应 (Incident Response)

## Overview

故障应急响应是建立故障处理机制的技能。确保系统稳定运行。

## 何时使用

- 系统故障
- 服务异常
- 数据问题
- 安全事件

## 流程

1. **故障检测**：检测故障
2. **故障定级**：定级故障
3. **问题定位**：定位问题
4. **应急处理**：处理故障
5. **用户通知**：通知用户
6. **复盘总结**：复盘总结
7. **预防措施**：预防措施

## 检查清单

- [ ] 快速检测
- [ ] 准确定级
- [ ] 快速定位
- [ ] 有效处理
- [ ] 及时通知
- [ ] 复盘改进

## 红牌警告

- 故障遗漏
- 定级不当
- 处理不当
- 用户未通知
- 复盘不彻底
