# RAG评估框架 (RAG Evaluation)

## Overview

RAG评估框架是评估检索增强生成效果的技能。包括相关性、准确性、完整性等指标。

## 何时使用

- RAG效果评估
- 组件诊断
- 优化方向
- 基准建立

## 流程

1. **指标设计**：设计评估指标
2. **数据集构建**：构建测试集
3. **评估执行**：执行评估
4. **结果分析**：分析结果
5. **问题诊断**：诊断问题
6. **优化建议**：给出建议
7. **持续监控**：持续监控

## 检查清单

- [ ] 指标合理
- [ ] 数据集代表性
- [ ] 评估准确
- [ ] 分析到位
- [ ] 建议可行
- [ ] 监控有效

## 红牌警告

- 指标不全面
- 数据集偏差
- 评估不准确
- 分析浅显
- 建议无效
