# 元学习 (Meta Learning)

## Overview

元学习是让Agent学会学习的技能。通过历史经验优化学习策略。

## 何时使用

- 策略优化
- 快速适应
- 经验复用
- 能力提升

## 流程

1. **经验收集**：收集经验
2. **策略学习**：学习策略
3. **快速适应**：快速适应
4. **泛化验证**：验证泛化
5. **策略更新**：更新策略
6. **效果评估**：评估效果
7. **持续迭代**：持续迭代

## 检查清单

- [ ] 经验收集
- [ ] 策略学习
- [ ] 适应快速
- [ ] 泛化有效
- [ ] 效果提升
- [ ] 迭代持续

## 红牌警告

- 经验偏差
- 过拟合
- 泛化失败
- 效果下降
- 迭代停滞
