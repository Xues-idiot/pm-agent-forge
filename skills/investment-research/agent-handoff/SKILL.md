# Agent交接 (Agent Handoff)

## Overview

Agent交接是设计多Agent间上下文传递和任务交接的技能。

## 何时使用

- Agent协作
- 任务交接
- 上下文传递
- 流程编排

## 流程

1. **交接设计**：设计交接协议
2. **状态打包**：打包状态
3. **上下文传递**：传递上下文
4. **验证接收**：验证接收
5. **继续执行**：继续执行
6. **回退机制**：回退机制
7. **日志记录**：记录日志

## 检查清单

- [ ] 交接协议
- [ ] 状态完整
- [ ] 传递准确
- [ ] 验证有效
- [ ] 回退可行
- [ ] 日志完整

## 红牌警告

- 信息丢失
- 状态不一致
- 死循环
- 责任不清
- 难以调试
