# Agent群体智能 (Agent Swarm)

## Overview

Agent群体智能是设计多Agent自主协作的技能。参考自然界蜂群智能，实现分布式问题解决。

## 何时使用

- 复杂问题分解
- 并行求解
- 群体决策
- 自组织系统

## 流程

1. **任务分解**：分解任务
2. **Agent角色**：定义Agent角色
3. **通信协议**：定义通信协议
4. **协作机制**：设计协作机制
5. **决策聚合**：聚合决策
6. **反馈机制**：反馈机制
7. **演化优化**：演化优化

## 检查清单

- [ ] 分解合理
- [ ] 角色明确
- [ ] 通信有效
- [ ] 协作高效
- [ ] 决策优化
- [ ] 系统稳定

## 红牌警告

- 死锁风险
- 通信混乱
- 决策冲突
- 系统振荡
- 难以调试
