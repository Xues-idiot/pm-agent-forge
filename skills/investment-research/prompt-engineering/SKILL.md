# 投研Agent提示词工程 (Research Agent Prompting)

## Overview

投研Agent提示词工程是为投研场景设计和优化LLM提示词的技能。提升Agent的分析质量和效率。

## 何时使用

- 设计新的投研Agent提示词
- 优化现有提示词效果
- 调试Agent输出质量
- 构建提示词模板库

## 流程

1. **任务分析**：分析任务需求
2. **角色设计**：设计Agent角色
3. **指令编写**：编写系统指令
4. **示例设计**：设计few-shot示例
5. **测试验证**：测试提示词效果
6. **迭代优化**：优化提示词
7. **模板归档**：归档优化后模板

## 模板

### 投研Agent提示词模板
```python
# 系统提示词模板
SYSTEM_PROMPT = """你是一位资深券商分析师，专注于A股投资研究。

你的职责：
1. 分析宏观经济和行业趋势
2. 研究上市公司基本面
3. 评估股票投资价值
4. 生成投资研究报告

分析框架：
- 定量分析：财务数据、估值指标
- 定性分析：商业模式、竞争壁垒、管理层
- 宏观分析：政策环境、市场情绪

输出要求：
1. 数据准确，来源可追溯
2. 逻辑清晰，论证充分
3. 风险揭示，客观全面
4. 投资建议，明确可执行

输出格式：
- 核心观点（3点以内）
- 详细分析
- 数据支撑
- 风险提示
- 投资建议
"""

# 用户请求模板
USER_PROMPT_TEMPLATE = """请分析{stock_code}({stock_name})：

研究维度：
1. 公司主营业务和盈利模式
2. 行业竞争格局和公司地位
3. 近年财务表现和关键指标
4. 当前估值水平和投资价值
5. 主要风险因素
6. 投资建议

请给出详细的分析报告。
"""
```
