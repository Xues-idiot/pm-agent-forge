# 自然语言生成 (Natural Language Generation)

## Overview

自然语言生成是自动生成投研报告、摘要等文本的技能。

## 何时使用

- 报告自动生成
- 要点总结
- 事件描述
- 风险提示

## 流程

1. **内容理解**：理解内容
2. **结构规划**：规划结构
3. **文本生成**：生成文本
4. **事实核查**：核查事实
5. **风格优化**：优化风格
6. **质量评估**：评估质量
7. **发布输出**：发布输出

## 检查清单

- [ ] 内容准确
- [ ] 结构清晰
- [ ] 风格专业
- [ ] 事实核查
- [ ] 逻辑连贯
- [ ] 可读性强

## 红牌警告

- 事实错误
- 逻辑混乱
- 风格不一致
- 幻觉产生
- 重复啰嗦
