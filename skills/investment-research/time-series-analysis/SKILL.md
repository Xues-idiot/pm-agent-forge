# 时间序列分析 (Time Series Analysis)

## Overview

时间序列分析是分析股价、经济指标等时序数据的技能。包括ARIMA、Prophet、LSTM等方法。

## 何时使用

- 趋势预测
- 季节性分析
- 异常检测
- 周期识别

## 流程

1. **数据准备**：准备时序数据
2. **探索分析**：探索模式
3. **模型选择**：选择模型
4. **预测分析**：预测分析
5. **模型评估**：评估模型
6. **结果解释**：解释结果
7. **应用输出**：应用到投研

## 检查清单

- [ ] 数据平稳
- [ ] 模型选择
- [ ] 参数调优
- [ ] 预测准确
- [ ] 解释合理
- [ ] 应用有效

## 红牌警告

- 非平稳数据
- 过度拟合
- 忽视季节性
- 预测过远
- 结果误读
