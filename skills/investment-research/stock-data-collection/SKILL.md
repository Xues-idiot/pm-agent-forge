# 股票数据采集 (Stock Data Collection)

## Overview

股票数据采集是从多种数据源获取A股市场原始数据的技能。涵盖行情数据、财务数据、公告数据、资金流向等核心数据的采集和清洗。

## 何时使用

- 需要获取股票实时/历史行情
- 需要采集上市公司财务数据
- 需要获取公告和新闻数据
- 需要获取资金流向数据

## 流程

1. **需求分析**：确定需要的数据类型和时间范围
2. **数据源选择**：选择合适的数据源（akshare/tushare/东方财富等）
3. **接口调用**：编写数据采集代码
4. **数据清洗**：处理缺失值、异常值
5. **数据存储**：存储到数据库或文件
6. **增量更新**：设计增量更新策略
7. **质量校验**：验证数据完整性

## 模板

### 数据采集配置模板
```python
# 采集配置
COLLECTION_CONFIG = {
    "stock_ list": "全量/指数成分/自选",
    "data_type": ["行情", "财务", "公告", "资金流"],
    "time_range": "历史/增量",
    "update_frequency": "实时/日更"
}

# 数据源配置
DATA_SOURCES = {
    "akshare": {
        "use_cases": ["行情", "基金", "宏观"],
        "rate_limit": "10次/秒"
    },
    "tushare": {
        "use_cases": ["财务", "公告", "资金流"],
        "need_token": True
    }
}
```

## 检查清单

- [ ] 数据源稳定性可靠
- [ ] API调用有错误处理和重试
- [ ] 采集频率符合数据源限制
- [ ] 数据清洗逻辑完善
- [ ] 增量更新策略正确
- [ ] 数据质量有校验机制

## 红牌警告

- 未处理数据缺失导致分析偏差
- 采集频率过高被封IP
- 使用未授权数据源
- 存储未做去重导致数据重复
- 时区处理错误导致数据错位
