# 函数调用设计 (Function Calling Design)

## Overview

函数调用设计是设计Agent工具调用的技能。包括工具Schema设计、错误处理、调用优化等。

## 何时使用

- 工具定义
- 函数调用
- 错误处理
- 性能优化
