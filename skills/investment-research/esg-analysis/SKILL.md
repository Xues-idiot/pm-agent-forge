# ESG分析 (ESG Analysis)

## Overview

ESG分析是评估公司环境、社会、治理表现的技能。越来越重要的投资维度。

## 何时使用

- ESG投资筛选
- 风险评估
- 可持续发展评估
- 责任投资

## 流程

1. **ESG评级获取**：获取ESG评级
2. **E分析**：分析环境表现
3. **S分析**：分析社会表现
4. **G分析**：分析治理表现
5. **争议事件**：分析争议事件
6. **趋势分析**：分析ESG趋势
7. **投资应用**：应用到投资

## 检查清单

- [ ] 评级来源可靠
- [ ] 分析全面
- [ ] 争议事件关注
- [ ] 趋势分析到位
- [ ] 投资建议合理
- [ ] 定期更新

## 红牌警告

- 评级不一致
- "洗绿"风险
- 数据过时
- 忽视争议
- 过于主观
