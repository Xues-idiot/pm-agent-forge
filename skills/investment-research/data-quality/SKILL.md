# 数据质量治理 (Data Quality Governance)

## Overview

数据质量治理是确保投研数据质量的技能。包括准确性、完整性、一致性等。

## 何时使用

- 数据采集
- 数据清洗
- 数据监控
- 质量报告

## 流程

1. **质量规则**：定义质量规则
2. **数据探查**：探查数据
3. **数据清洗**：清洗数据
4. **质量监控**：监控质量
5. **问题追踪**：追踪问题
6. **质量报告**：报告质量
7. **持续改进**：持续改进

## 检查清单

- [ ] 规则完整
- [ ] 探查准确
- [ ] 清洗有效
- [ ] 监控完善
- [ ] 问题追踪
- [ ] 报告清晰

## 红牌警告

- 规则不完整
- 质量问题遗漏
- 清洗不当
- 监控缺失
- 问题重复
