# Agent记忆系统 (Agent Memory System)

## Overview

Agent记忆系统是设计Agent长期记忆和上下文管理的技能。提升Agent的连续对话能力。

## 何时使用

- 多轮对话
- 用户偏好
- 研究上下文
- 历史记录

## 流程

1. **记忆设计**：设计记忆结构
2. **短期记忆**：管理短期记忆
3. **长期记忆**：管理长期记忆
4. **记忆检索**：检索记忆
5. **记忆更新**：更新记忆
6. **遗忘机制**：设计遗忘
7. **效果评估**：评估效果

## 检查清单

- [ ] 记忆结构合理
- [ ] 检索准确
- [ ] 更新及时
- [ ] 遗忘有效
- [ ] 性能良好
- [ ] 隐私保护

## 红牌警告

- 记忆过载
- 检索不准确
- 隐私泄露
- 存储膨胀
- 遗忘过早
