# 输出验证 (Output Validation)

## Overview

输出验证是验证Agent输出正确性、完整性的技能。

## 何时使用

- 质量控制
- 错误检测
- 格式检查
- 事实核查

## 流程

1. **验证规则**：定义验证规则
2. **验证执行**：执行验证
3. **错误处理**：处理错误
4. **反馈循环**：反馈循环
5. **优化规则**：优化规则
6. **监控统计**：监控统计
7. **持续改进**：持续改进

## 检查清单

- [ ] 规则完备
- [ ] 执行准确
- [ ] 处理得当
- [ ] 反馈有效
- [ ] 监控完善
- [ ] 改进持续

## 红牌警告

- 规则不全面
- 验证误判
- 处理不当
- 反馈无效
- 监控缺失
