# Prompt链式调用 (Prompt Chaining)

## Overview

Prompt链式调用是设计多步Prompt串联完成复杂任务的技能。

## 何时使用

- 复杂任务分解
- 逐步推理
- 质量提升
- 调试追踪

## 流程

1. **任务分解**：分解任务
2. **Prompt设计**：设计各步Prompt
3. **链式串联**：串联Prompt
4. **输出传递**：传递输出
5. **质量检查**：检查质量
6. **错误处理**：处理错误
7. **优化迭代**：优化链式

## 检查清单

- [ ] 分解合理

- [ ] 错误处理
- [ ] 性能达标
- [ ] 效果提升
