# 竞品对比分析 (Peer Comparison)

## Overview

竞品对比分析是对同行业公司进行多维度对比的技能。帮助识别行业内最优标的，发现估值洼地。

## 何时使用

- 寻找行业最优标的
- 横向对比估值
- 筛选投资标的
- 验证投资逻辑

## 流程

1. **公司筛选**：确定可比公司池
2. **维度设计**：设计对比维度
3. **数据收集**：收集各维度数据
4. **对比分析**：进行横向对比
5. **评分排序**：给各公司打分
6. **深度分析**：对重点公司深入分析
7. **投资建议**：给出推荐排序

## 检查清单

- [ ] 可比公司选择合理
- [ ] 对比维度全面
- [ ] 数据准确可靠
- [ ] 评分权重合理
- [ ] 结论有数据支撑
- [ ] 风险揭示充分

## 红牌警告

- 可比公司选择不当
- 数据错误导致对比失真
- 忽视公司间本质差异
- 估值对比忽视成长性差异
- 单一维度选股
