# 知识蒸馏 (Knowledge Distillation)

## Overview

知识蒸馏是将大模型能力蒸馏到小模型的技能。用于降低推理成本。

## 何时使用

- 模型压缩
- 成本优化
- 边缘部署
- 延迟敏感

## 流程

1. **教师模型**：选择教师模型
2. **学生模型**：选择学生模型
3. **蒸馏策略**：设计蒸馏策略
4. **训练蒸馏**：训练学生
5. **效果评估**：评估效果
6. **部署应用**：部署模型
7. **持续优化**：持续优化

## 检查清单

- [ ] 教师模型合适
- [ ] 学生模型精简
- [ ] 蒸馏有效
- [ ] 效果达标
- [ ] 成本降低
- [ ] 延迟降低

## 红牌警告

- 知识丢失
- 效果下降
- 过度压缩
- 训练不稳定
- 泛化能力下降
