# 另类数据分析 (Alternative Data Analysis)

## Overview

另类数据分析是利用非传统数据源（如卫星图像、信用卡数据、社交媒体）进行投资研究的技能。

## 何时使用

- 寻找信息优势
- 验证基本面数据
- 早期趋势识别
- 竞争优势构建

## 流程

1. **数据源选择**：选择另类数据源
2. **数据获取**：获取数据
3. **数据清洗**：清洗数据
4. **特征工程**：构建投资特征
5. **信号生成**：生成投资信号
6. **回测验证**：回测验证
7. **实盘应用**：应用到实盘

## 检查清单

- [ ] 数据源合法
- [ ] 数据质量可靠
- [ ] 特征设计合理
- [ ] 回测验证通过
- [ ] 容量评估
- [ ] 合规审查

## 红牌警告

- 数据不合法
- 隐私问题
- 过拟合风险
- 容量限制
- 合规风险
