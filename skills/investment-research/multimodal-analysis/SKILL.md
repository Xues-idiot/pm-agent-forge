# 多模态分析 (Multimodal Analysis)

## Overview

多模态分析是利用视觉语言模型分析K线图、财务报表图片等投研资料的技能。

## 何时使用

- K线图分析
- 财报图片识别
- 图表理解
- 文档OCR

## 流程

1. **图像获取**：获取图像
2. **图像预处理**：预处理图像
3. **多模态分析**：分析图像
4. **结果提取**：提取结果
5. **结果整合**：整合结果
6. **验证校对**：验证准确性
7. **应用输出**：应用到投研

## 检查清单

- [ ] 图像质量
- [ ] 分析准确
- [ ] 结果可靠
- [ ] 验证有效
- [ ] 成本可控
- [ ] 延迟可接受

## 红牌警告

- 图像不清晰
- 识别错误
- 幻觉产生
- 成本过高
- 延迟过高
