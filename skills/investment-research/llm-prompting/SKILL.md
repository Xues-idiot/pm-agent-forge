# LLM提示词优化 (LLM Prompt Optimization)

## Overview

LLM提示词优化是优化大语言模型提示词提升投研分析质量的技能。

## 何时使用

- 提升分析质量
- 减少幻觉
- 优化输出格式
- 成本优化

## 流程

1. **需求分析**：分析需求
2. **Prompt编写**：编写Prompt
3. **测试验证**：测试效果
4. **问题识别**：识别问题
5. **优化迭代**：优化Prompt
6. **效果评估**：评估效果
7. **模板固化**：固化为模板

- [ ] 角色设定清晰

- 指令不一致
