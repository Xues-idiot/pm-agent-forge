# 机器学习流水线 (ML Pipeline)

## Overview

机器学习流水线是构建完整ML工作流的技能。从数据到模型的自动化流程。

## 何时使用

- 量化因子开发
- 价格预测
- 文本分类
- 推荐系统

## 流程

1. **数据准备**：准备训练数据
2. **特征工程**：构建特征
3. **模型选择**：选择模型
4. **训练验证**：训练和验证
5. **超参调优**：调优超参数
6. **模型部署**：部署模型
7. **监控维护**：监控和维护

## 检查清单

- [ ] 数据质量
- [ ] 特征有效
- [ ] 模型选择
- [ ] 超参合理
- [ ] 部署稳定
- [ ] 监控完善

## 红牌警告

- 数据泄漏
- 过拟合
- 训练-serving skew
- 模型漂移
- 性能下降
