# RAG检索优化 (RAG Retrieval Optimization)

## Overview

RAG检索优化是提升检索增强生成效果的技能。包括检索质量、上下文压缩、混合检索等。

## 何时使用

- 提升RAG效果
- 减少token消耗
- 提升准确性
- 优化延迟

## 流程

1. **检索评估**：评估当前检索
2. **策略设计**：设计优化策略
3. **混合检索**：实现混合检索
4. **重排优化**：实现重排
5. **上下文压缩**：压缩上下文
6. **效果验证**：验证效果
7. **持续优化**：持续迭代

## 检查清单

- [ ] 检索质量
- [ ] 准确性
- [ ] 延迟
- [ ] 成本
- [ ] 覆盖度
- [ ] 稳定性

## 红牌警告

- 检索不准确
- 上下文遗漏
- 延迟过高
- 成本过高
- 幻觉产生
