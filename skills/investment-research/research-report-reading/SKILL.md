# 研报阅读与提炼 (Research Report Reading)

## Overview

研报阅读与提炼是从券商研报中快速提取关键信息的技能。识别有价值的研报和关键投资逻辑。

## 何时使用

- 快速阅读大量研报
- 提炼核心投资观点
- 识别研报价值
- 验证投资逻辑

## 流程

1. **研报筛选**：筛选相关研报
2. **快速浏览**：快速浏览摘要
3. **重点阅读**：精读核心章节
4. **信息提取**：提取关键数据
5. **观点验证**：验证逻辑正确性
6. **价值评估**：评估研报价值
7. **归档整理**：归档整理

- 数据来源: [可靠/一般/存疑]

## 检查清单

- [ ] 研报来源可靠
- [ ] 核心观点提取准确
- [ ] 数据验证核实
- [ ] 逻辑推演检验
- [ ] 风险提示关注
- [ ] 多研报对比

## 红牌警告

- 盲目相信券商评级
- 忽视利益冲突
- 数据不经核实
- 逻辑不严谨
- 忽视风险提示
- 过于依赖单篇研报
