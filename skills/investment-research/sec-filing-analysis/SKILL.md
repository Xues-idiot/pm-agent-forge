# SEC Filing分析 (SEC Filing Analysis)

## Overview

SEC Filing分析是解析和提取美国证监会 filings 信息的技能。涵盖10-K年报、10-Q季报、8-K重大事件、14A委托声明书等的分析。

## 何时使用

- 研究美股/中概股
- 分析公司治理
- 挖掘风险因素
- 理解战略规划

## 流程

1. **Filing获取**：从EDGAR获取Filing
2. **Filing分类**：分类Filing类型
3. **Section解析**：解析各章节
4. **关键信息提取**：提取关键数据
5. **时间线分析**：分析Filing时间线
6. **变更检测**：检测重大变更
7. **投资应用**：应用到投研

## 模板

### SEC Filing分析模板
```

- Filing类型: [10-K/10-Q/8-K/14A]
- Filing日期: [YYYY-MM-DD]

|----------|------|------|

| 业务变更 | X | 高 |
| 风险因素 | X | 中 |

- 业绩驱动因素: [X]

## 风险因素
1. [风险1]
2. [风险2]

## 历史对比
- vs 上期: [变化]
- vs 去年同期: [变化]
```

## 检查清单

- [ ] Filing来源可靠
- [ ] 解析准确
- [ ] 变更检测到位
- [ ] 影响评估合理
- [ ] 投资应用有效
- [ ] 时效性关注

## 红牌警告

- Filing信息获取不完整
- 解析错误
- 变更误判
- 影响高估或低估
- 忽视时效性
