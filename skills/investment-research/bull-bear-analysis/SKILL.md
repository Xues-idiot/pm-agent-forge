# 多空分析 (Bull/Bear Analysis)

## Overview

多空分析是同时分析上行和下行情景的技能。参考investment-thesis-agent的Bull/Bear Case Generator，提供平衡的投资视角。

## 何时使用

- 评估投资风险
- 情景规划
- 决策支持
- 组合分析

## 流程

1. **情景设定**：设定分析情景
2. **Bull Case**：分析乐观情景
3. **Bear Case**：分析悲观情景
4. **Base Case**：分析基准情景
5. **概率加权**：概率加权预期
6. **风险回报**：计算风险回报比
7. **投资建议**：综合建议

## 模板

### 多空分析模板
```
## 多空分析
### 股票: [X] 当前价: [X元]

## Bull Case (乐观情景)
- 假设条件:
  - [假设1]
  - [假设2]
- 目标价: [X元]
- 上涨空间: [+X%]
- 概率: [X%]

## Bear Case (悲观情景)
- 假设条件:
  - [假设1]
  - [假设2]
- 目标价: [X元]
- 下跌空间: [-X%]
- 概率: [X%]

## Base Case (基准情景)
- 假设条件:
  - [假设1]
  - [假设2]
- 目标价: [X元]
- 潜在涨跌: [±X%]
- 概率: [X%]

## 概率加权预期
- 概率加权价格: [X元]
- 预期回报: [X%]
- 风险回报比: [X:1]

## 投资建议
- 操作: [买入/持有/卖出]
- 理由: [X]
- 关键催化剂: [X]
- 关键风险: [X]
```

## 检查清单

- [ ] 情景假设合理
- [ ] 概率判断客观
- [ ] 分析平衡
- [ ] 风险回报清晰
- [ ] 建议可执行
- [ ] 定期更新

## 红牌警告

- 过于乐观
- 忽视尾部风险
- 概率判断主观
- 情景不全面
- 建议不明确
