# 上下文窗口管理 (Context Window Management)

## Overview

上下文窗口管理是高效管理LLM上下文长度的技能。包括压缩、摘要、选择性保留等。

## 何时使用

- 长对话
- 大文档
- 成本控制
- 性能优化

## 流程

1. **窗口分析**：分析上下文
2. **策略选择**：选择策略
3. **压缩实现**：实现压缩
4. **摘要生成**：生成摘要
5. **选择性保留**：选择性保留
6. **效果验证**：验证效果
7. **成本评估**：评估成本

## 检查清单

- [ ] 分析准确
- [ ] 策略合适
- [ ] 压缩有效
- [ ] 保留关键
- [ ] 效果良好
- [ ] 成本降低

## 红牌警告

- 关键信息丢失
- 压缩过度
- 语义丢失
- 成本未降
- 性能下降
