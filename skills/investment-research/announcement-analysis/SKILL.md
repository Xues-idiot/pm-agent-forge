# 公告解读 (Announcement Analysis)

## Overview

公告解读是分析上市公司公告信息，提取关键投资信息的技能。涵盖业绩公告、并购重组、股权变动、重要合同等类型公告的解读。

## 何时使用

- 跟踪持仓股公告
- 寻找超预期投资机会
- 避雷识别风险公告
- 把握事件驱动机会

## 流程

1. **公告收集**：从指定渠道获取公告
2. **类型分类**：判断公告类型
3. **信息提取**：提取关键数据和条款
4. **影响分析**：分析对公司的影响
5. **市场反应**：分析市场历史类似反应
6. **投资建议**：给出操作建议
7. **归档跟踪**：归档并持续跟踪

## 检查清单

- [ ] 公告来源权威
- [ ] 关键信息提取完整
- [ ] 影响分析客观
- [ ] 参考历史案例准确
- [ ] 投资建议有依据
- [ ] 及时跟踪后续公告

## 红牌警告

- 误读公告关键条款
- 忽视公告隐藏风险
- 类比案例不恰当
- 投资建议与公告内容不符
- 公告解读超出披露信息过度推测
