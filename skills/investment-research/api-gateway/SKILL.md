# API网关设计 (API Gateway)

## Overview

API网关设计是构建统一API网关的技能。提供认证、限流、监控等。

## 何时使用

- 多服务统一入口
- 认证授权
- 限流保护
- 监控统计

## 流程

1. **网关选型**：选择网关
2. **路由配置**：配置路由
3. **认证配置**：配置认证
4. **限流配置**：配置限流
5. **监控配置**：配置监控
6. **测试验证**：测试验证
7. **部署上线**：部署上线

## 检查清单

- [ ] 路由正确
- [ ] 认证安全
- [ ] 限流合理
- [ ] 监控完善
- [ ] 性能达标
- [ ] 文档完整

- 路由错误
- 安全漏洞
- 限流过严
- 性能瓶颈
