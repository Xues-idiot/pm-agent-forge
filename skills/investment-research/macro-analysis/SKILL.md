# 宏观分析 (Macro Analysis)

## Overview

宏观分析是分析国内外宏观经济环境对A股市场影响的技能。涵盖经济增长、通胀、货币政策、财政政策、汇率等核心维度的分析。

## 何时使用

- 判断大盘走势
- 把握政策主线
- 进行资产配置
- 预测市场风格

## 流程

1. **数据收集**：收集宏观数据
2. **经济增长**：分析GDP和PMI
3. **通胀分析**：分析CPI和PPI
4. **政策分析**：分析货币和财政政策
5. **汇率分析**：分析汇率对市场影响
6. **环境对比**：对比国内外环境
7. **综合判断**：给出宏观研判
