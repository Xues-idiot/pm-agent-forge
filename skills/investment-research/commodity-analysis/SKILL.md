# 大宗商品分析 (Commodity Analysis)

## Overview

大宗商品分析是分析黄金、原油、农产品等大宗商品价格走势和投资机会的技能。

## 何时使用

- 商品期货投资
- 通胀对冲
- 产业链研究
- 通胀预期

## 流程

1. **供需分析**：分析供需基本面
2. **宏观分析**：分析宏观影响
3. **美元分析**：分析美元影响
4. **季节性分析**：分析季节性规律
5. **技术分析**：技术面分析
6. **相关性分析**：与其他资产相关性
7. **投资策略**：制定投资策略
